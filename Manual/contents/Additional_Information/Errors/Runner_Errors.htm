Errores del corredor

Incluso después de la comprobación de la sintaxis en el editor de código y de que el compilador compruebe su código en busca de errores de compilación, todavía hay ocasiones en las que algo puede ir mal. En la mayoría de los casos esto arrojará un error de ejecución de la máquina virtual (VM ) (también llamado excepción de tiempo de ejecución) que se parece a esto:

Los errores de ejecución son generalmente más serios que los de compilación, ya que significan que hay algo seriamente incorrecto en su código que ni el editor de código ni el compilador han podido detectar, y como tal debe prestar atención a todos esos errores. Cuando se produce uno, puede utilizar el botón Copiar de la ventana emergente para copiar el error en el portapapeles, que luego puede pegar en un archivo de texto (o donde sea) para futuras referencias.

La estructura de este error es la siguiente:


  • Muestra el evento en el que se ha producido el error, así como la dirección Object (si procede).

  • Esto muestra el error real. Por lo tanto, muestra el alcance del error y el tipo de error que es. El primer valor entre los paréntesis será el ID de la instancia que estaba ejecutando el código - o podría ser uno de los valores especiales listados en la tabla de abajo - y el segundo es un valor interno que GameMaker utiliza para identificar errores y puede ser ignorado. En el caso de la imagen de arriba, está diciendo que la variable _image no ha sido establecida (es decir, no existe antes de ser accedida) en el object obj_EA_DEMO_Control.

  • Esto muestra el número de línea en la que se produce el error, así como un fragmento de código que muestra la posición. Ten en cuenta que esto no siempre es 100% preciso debido a la forma en que se compila el juego, pero debería ser suficiente para ayudarte a encontrar exactamente dónde está el problema.

Como se ha mencionado anteriormente, algunos mensajes de error no identificarán el ámbito por un valor de ID de instancia, sino por un valor negativo . Estos valores pueden ser utilizados para precisar la naturaleza exacta del error y a qué se refiere con los siguientes valores posibles:

Prefijo Alcance
-1 Auto
-2 Otros
-3 Todo
-4 Nadie
-5 Global
-6 No especificado
-7 Local

Los posibles errores del VM runner son los siguientes:

Error Mensaje Operación Descripción
DoSet Tipo de comparación inválido Tipos de datos Esto denota que el corredor ha tratado de comparar dos incompatibles data types, como un número real con un string.
DoConv Error de ejecución Tipos de datos Esto denota un error en la conversión de un tipo de datos en otro.
DoAdd Error de ejecución Matemáticas Algo ha fallado al utilizar la expresión de adición (+).
DoMul Error de ejecución Matemáticas Algo ha fallado al utilizar la expresión de multiplicación (*).
DoSub Error de ejecución Matemáticas Algo ha fallado al utilizar la expresión de la resta (-).
DoSub Motor de ejecución - No puede operar en string tipo Matemáticas Estás intentando restar el tipo de variables equivocado (por ejemplo, restar un string de un real).
DoDiv Error de ejecución Matemáticas Algo ha fallado al utilizar la expresión de división (/ o div).
DoDiv Motor de ejecución - No puede operar en string tipo Matemáticas Estás intentando dividir el tipo de variables equivocado (por ejemplo, dividir un string por un real).
DoDiv Dividir por cero Matemáticas Está intentando dividir por 0 (tenga en cuenta que esto sólo ocurrirá cuando se utilice la división de enteros, dividir un real (distinto de cero) por 0 dará como respuesta el infinito, dividir cero por cero dará como respuesta NaN). Puedes comprobar estos valores con (is_infinity) y (is_nan)
DoMod Error de ejecución Matemáticas Algo ha fallado al utilizar la expresión modulo (mod).
DoMod Motor de ejecución - No puede operar en string tipo Matemáticas Estás intentando utilizar el módulo (mod) en un tipo de variables equivocado (por ejemplo mod un string por un real).
DoAnd Error de ejecución Bitwise Algo ha fallado al utilizar la expresión "and" a nivel de bits (&).
DoAnd Motor de ejecución - No puede operar en string tipo Bitwise Estás tratando de usar "y" a nivel de bits (&) en el tipo de variables equivocado (por ejemplo, tratando de "y" un string con un real).
DoOr Error de ejecución Bitwise Algo ha fallado al utilizar la expresión "or" a nivel de bits (|).
DoOr Motor de ejecución - No puede operar en string tipo Bitwise Estás tratando de usar "o" (|) en el tipo de variables equivocado (por ejemplo, tratando de "o" un string con un real).
DoXor Error de ejecución Bitwise Algo ha fallado al utilizar la expresión "xor" a nivel de bits (^).
DoXor Motor de ejecución - No puede operar en string tipo Bitwise Estás intentando utilizar "xor" (^) en el tipo de variables equivocado (por ejemplo, intentando "xor" un string con un real).
DoShl Error de ejecución Bitwise Algo ha salido mal al desplazar a la izquierda (<<) un valor.
DoShl Motor de ejecución - No puede operar en string tipo Bitwise Está intentando desplazar a la izquierda (<<) el tipo de variables equivocado (por ejemplo, intentando desplazar a la izquierda un string).
DoShr Error de ejecución Bitwise Algo ha ido mal al desplazar a la derecha (>>) un valor.
DoShr Motor de ejecución - No puede operar en string tipo Bitwise Está intentando desplazar a la derecha (>>) el tipo de variables equivocado (por ejemplo, intentando desplazar a la derecha un string).
DoNeg Error de ejecución Negar Estás intentando convertir un tipo de variable en negativo cuando este tipo no permite tal operación.
DoNot Error de ejecución Negar Estás tratando de "no" un tipo de variable cuando este tipo no permite tal operación.
Push Error de ejecución - Índice de la variable fuera de rango (var) Pila La variable a la que se accede está fuera del rango establecido para el corredor.
Push Variable Get (var) Pila La variable dada no ha sido definida o es desconocida.
Pop Variable Índice fuera de rango (var) Pila La variable a la que se accede está fuera del rango establecido para el corredor.
Pop Variable Get (var) Pila La variable dada no ha sido definida o es desconocida.
With No se puede utilizar global en la declaración con Con Has intentado utilizar "global" como variable dentro de una sentencia "with", es decir:
with (global)
   {
   //do something;
   }
With No se puede utilizar local en la declaración con Con Has intentado utilizar "local" como variable dentro de una sentencia "with", es decir:
with (local)
   {
   //do something;
   }
DoCall Error de tipo de motor de ejecución Motor Se trata de un error de undefined en la máquina virtual. Si esto ocurriera, debería presentar un informe de error (véase: el menú de ayuda para saber cómo hacerlo).
Stack Overflow - Motor Un desbordamiento de la pila se produce cuando se utiliza demasiada memoria en la pila de llamadas y cuando su juego intenta utilizar más espacio del disponible en la pila de llamadas (es decir, cuando intenta acceder a la memoria más allá de los límites de la pila de llamadas, lo que es esencialmente un desbordamiento de buffer ), se dice que la pila se desborda, lo que resulta en un fallo del programa. Reinicie su ordenador y GameMaker y, si el error persiste, póngase en contacto con el servicio de asistencia y/o presente un error (como se ha explicado anteriormente).
