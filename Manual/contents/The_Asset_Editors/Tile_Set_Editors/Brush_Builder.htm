El Tile Set Brush Builder

Por defecto, cuando se "pinta" tiles en una capa de mapa de azulejos en el editor room, se selecciona un solo tile y se pinta con él. Sin embargo, tile sets está casi siempre diseñado para tener secciones que encajan de diferentes maneras para formar áreas o elementos completos. Por ejemplo, un juego de rol tile set puede tener una característica de paisaje tiles que puede ser conectada para crear características más grandes o más pequeñas - como árboles y edificios - dependiendo del número de azulejos utilizadas. Ahora bien, la colocación de múltiples características como esta en una capa room requeriría que usted fuera adelante y atrás muchas veces para cambiar tile, lo que no es bueno para su flujo de trabajo. Para resolver esto, GameMaker le permite crear tile Pinceles en el editor Tile Set, disponible al hacer clic en en el botón Brush Builder:

En el Brush Builder tienes el original tile set a la izquierda y un "lienzo" en blanco a la derecha. Ahora puede seleccionar cualquier tile de la izquierda y pintarlo en la derecha para crear "pinceles" personalizados que luego podrá utilizar en el editor room. Tenga en cuenta que puede hacer clic y mantener pulsado el botón izquierdo del ratón y luego arrastrar en el tile set para seleccionar múltiples tiles para pintar en el lienzo del pincel y también puede mantener pulsado / + para añadir tiles individuales específicos o mantener pulsado + para eliminar tiles específicos.

La siguiente imagen muestra nuestras características paisajísticas mencionadas anteriormente como ejemplo de tres pinceles personalizados realizados a partir de un único conjunto de azulejos:

A la derecha puede ver tres características que hemos realizado (resaltadas con un recuadro naranja en la imagen). Fíjese en que hemos dejado un espacio de un tile entre cada rasgo - esto se debe a que cualquier grupo de tiles que se toque será tratado como un solo pincel en el editor room, así que dejamos un espacio de un tile para mostrar que cada conjunto es un pincel distinto pincel que queremos crear. Mientras creamos los pinceles, pintamos con el botón izquierdo del ratón y borramos con el botón derecho . También puede ampliar la hoja tile o el lienzo de los pinceles utilizando / y la rueda del ratón cuando el ratón cuando el ratón está sobre una de las ventanas, o puede desplazarse utilizando + o el botón central del ratón y arrastrando. Tenga en cuenta que también puede utilizar las herramientas de zoom en la caja de herramientas dentro de las ventanas del editor para cambiar los niveles de zoom de cualquiera de las ventanas, así como para cambiar la visibilidad y el color de la cuadrícula.

En la parte superior derecha puedes ver la herramienta actualmente seleccionada, y también puedes establecer el tamaño del pincel con el que quieres pintar. El tamaño por defecto es 1, que es un solo tile, pero si lo ajusta a valores más altos entonces puede pintar (y borrar) con un pincel más grande compuesto por el tile seleccionado repetido, como se muestra en la imagen siguiente:

La caja de herramientas es donde puede seleccionar la herramienta a utilizar para muchas tareas diferentes en el editor Tile Set, algunas de las cuales dependerán de si de si tiene algo definido en su biblioteca de mosaicos automáticos. A continuación se ofrece un breve resumen de cada herramienta (tenga en cuenta que cuando haya seleccionado una capa de tile en el editor de Room, esta caja de herramientas se mostrará en la parte superior de la sala workspace junto con otras herramientas de dibujo):

Lápiz Esta es la herramienta lápiz. Utiliza el tile seleccionado para pintar en la ventana de Pinceles con el botón izquierdo del ratón y puede borrar con el botón derecho del ratón .
Borrador Con la herramienta de borrado puede utilizar el botón izquierdo del ratón para borrar un determinado tile en la ventana de pinceles. Esencialmente, todo lo que esto hace es poner el índice de tile a 0, que es el reservado "vacío" tile.
Selección Esta es la herramienta de selección, que se puede utilizar para definir un área de la ventana de Pinceles para trabajar en ella. Puede pulsar el botón izquierdo del ratón y luego arrastrar el ratón para crear un área rectangular, o puede pulsar / + para añadir selecciones múltiples y + para borrar una parte de la selección. Para borrar toda la selección puede pulsar . Cuando tienes un área de la ventana de Pinceles seleccionada, el resto de las herramientas (Lápiz, Voltear, Rotar, etc...) sólo funcionarán dentro del área seleccionada. Tenga en cuenta que también puede seleccionar tiles en la ventana y luego copiar ( / + "C"), cortar ( / + "X") y pegar ( / + " V"), lo que hará que la herramienta switch se convierta en el Lápiz y le permita "pintar" con ellos.
AutoTile Al hacer clic en esta herramienta se activa el estilo de pintura Auto tiling. Cuando esto está activo, puede seleccionar cualquier tile de la pestaña de Bibliotecas de Auto mosaico, y luego pintarlo en la ventana de Pinceles y GameMaker lo cambiará automáticamente para que coincida con el tiles circundante, siempre y cuando haya configurado correctamente la pestaña Auto Tile. Tenga en cuenta que la selección de un tile de la tile set que no es parte de las bibliotecas de autotiles restablecerá la herramienta de dibujo a la herramienta estándar Lápiz.
Flip Haciendo clic en la herramienta Voltear con el botón izquierdo del ratón (o utilizando el atajo de teclado "X" ) se volteará el tile o tiles actualmente seleccionado para dibujar a lo largo del eje horizontal (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel se volteará). seleccionados para dibujar a lo largo del eje horizontal , sin cambiar de herramienta de dibujo (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel se volteará). Si no tiene ningún tile seleccionado para dibujar, pero tiene un grupo de tiles seleccionado en la ventana del pincel, entonces la herramienta de volteo volteará el tiles seleccionado.
Espejo Haciendo clic en la herramienta Espejo con el botón izquierdo del ratón (o utilizando el atajo de teclado " Y" ) se reflejará la(s) baldosa(s) actualmente seleccionada(s) para dibujar a lo largo del eje vertical. seleccionado para dibujar a lo largo del eje vertical , sin cambiar la herramienta de dibujo (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel se reflejará). Si no tiene ningún tile seleccionado para dibujar, pero tiene un grupo de tiles seleccionado en la ventana del pincel, entonces la herramienta Espejo reflejará el tiles seleccionado.
Girar Haciendo clic en la herramienta Girar con el botón izquierdo del ratón (o utilizando el atajo de teclado " Z" ) girará la(s) baldosa(s) actualmente seleccionada(s) para dibujar seleccionado para dibujar 90° en el sentido de las agujas del reloj, sin cambiar de herramienta de dibujo (si tiene un pincel personalizado seleccionado para dibujar, todo el pincel girará). Si no tiene seleccionado ningún tile para dibujar, pero sí tiene seleccionado un grupo de tiles en la ventana de ventana del pincel, entonces la herramienta Rotar rotará el tiles seleccionado.

Debajo de las herramientas, puede encontrar dos secciones diferentes para seleccionar cualquier auto azulejo o azulejosanim ados que han sido creados usando la imagen actual de tile set. Un solo sprite que se utiliza para un tile set puede tener muchas, muchas, imágenes de celdas individuales en él, y estas pueden ser combinadas en el editor de Animación o de Auto mosaico para crear pinceles personalizados que se mostrarán en estas secciones y pueden ser utilizados en conjunto con tiles estático normal para crear pinceles (tenga en cuenta que un tile animado se animará independientemente de si lo ha seleccionado de la biblioteca o de la base tile set).

Una vez que haya configurado todos los pinceles que necesite, podrá utilizarlos para pintar tiles en cualquier capa del mapa tile dentro del Editor de espacios.
