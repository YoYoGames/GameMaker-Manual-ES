Nueve rodajas

El Nine Slicing (también conocido como 9-slicing) es una técnica utilizada para escalar imágenes rectangulares con el fin de preservar los detalles, haciendo que conserven su forma original tras el escalado.

Esto se consigue dividiendo la imagen en nueve partes separadas o "rodajas". Las rodajas de las esquinas se desplazan durante el escalado y se dibujan a la misma escala, luego las rodajas de los bordes y del centro se estiran o se repiten (dependiendo de su configuración) para llenar el espacio.

Se colocan cuatro guías en un sprite para dividir la imagen en nueve cortes. El siguiente gráfico muestra cómo las guías permanecen en su sitio después de que sprite se escale con Nine Slice:

El modo de mosaico de una rebanada determina cómo se muestra esa rebanada cuando se escala la sprite. Sólo se puede establecer para las rebanadas del borde y del centro (ya que las rebanadas de las esquinas no se escalan).

La siguiente imagen muestra cada uno de los modos de Tile cuando se aplica a la rebanada central:Si se habilita la rebanada nueve para un sprite, se utilizará para cambiar el tamaño de sprite dondequiera que aparezca, y funcionará en cualquier lugar existente donde se haya utilizado el sprite sin necesidad de realizar ningún otro cambio.

NOTA: Nine Slicing no funcionará con funciones que dibujen sólo una parte de la sprite o la distorsionen, como draw_sprite_part() o draw_sprite_pos().

Junto con las secuencias, esta función puede utilizarse para construir interfaces de usuario, y puede utilizarse, por ejemplo, para crear paredes rectangulares en un nivel de juego. La memoria de las texturas también puede ahorrarse utilizando Nueve rebanadasya que se puede utilizar una imagen más pequeña para cubrir un área mayor en la pantalla.

Nine Slicing puede ser activado en Sprites en el IDE. Para más información, consulte el editor Sprite .
