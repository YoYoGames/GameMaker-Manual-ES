Ejemplos de fuentes

Esta página contiene ejemplos e información sobre el código fuente de extension para Android y iOS, para ayudarle a crear su propio extensions nativo para esas plataformas.

Android (ejemplo básico)

Nuestro ejemplo extension para Android contendrá funciones que hacen lo siguiente:

  • Devuelve dos números sumados
  • Devuelve un string formado por una entrada string y un valor
  • Devuelve un string formado por dos cadenas de entrada
  • Activar el evento Async - Social y rellenar los valores en su mapa async_load

Este extension utilizará un archivo .java, cuyo contenido se indica a continuación (haga clic para abrirlo):

GenericTest.java

Para añadir un archivo de este tipo a su extension, necesita ir a sus Propiedades de Android y primero introducir un nombre de clase (para este ejemplo, "GenericTest"), luego hacer clic en "Añadir Fuente" y seleccionar el archivo .java. Esto añadirá el archivo seleccionado al directorio de su proyecto -- puede hacer clic con el botón derecho en su extension, seleccionar "Abrir en el Explorador" y ver los archivos fuente allí.

Ahora puede añadir un nuevo archivo de marcador de posición a su extension y crear funciones dentro de él con las siguientes propiedades:

  • función / nombre externo: AddTwoNumbers
    ayuda: AddTwoNumbers(value1, value2)
    argumentos: double, double
    tipo de retorno: double
  • función / nombre externo: HowManyObjects
    ayuda: HowManyObjects(value1, value2, string)
    argumentos: double, double, string
    tipo de retorno: string
  • función / nombre externo: BuildAString
    ayuda: BuildAString(string1, string2)
    argumentos: string, string
    tipo de retorno: string
  • función / nombre externo: ReturnAsync
    ayuda: ReturnAsync(value1, value2)
    argumentos: double, double
    tipo de retorno: double

Este es un ejemplo de la función AddTwoNumbers():

Ahora puedes llamar a estas funciones desde tu código de juego y utilizarlas como cualquier otra función incorporada:

var result1 = string(AddTwoNumbers(irandom(100), 50));
var result2 = BuildAString("Hello", "World");
var result3 = HowManyObjects(irandom(1000), irandom(1000), "Targets");
ReturnAsync(irandom(1000), irandom(1000));

La función ReturnAsync() desencadenará el evento Async - Social, por lo que tendrá que añadir ese evento a su object y leer los datos de su mapa async_load (que se rellena con el código fuente de la extensión dado anteriormente).

var type = async_load[? "type"];

if (type == "finished")
{
    var result4 = "value1: " + string(async_load[? "argument0"]) + ", ";
    result4 += "value2: " + string(async_load[? "argument1"]);
}

iOS (ejemplo básico)

El ejemplo proporcionado para iOS es el mismo que el ejemplo de Android dado anteriormente; sólo los archivos fuente difieren, lo cual cubriremos en esta sección.

iOS extensions utiliza un archivo .h y un archivo .mm, y el contenido de ambos se indica a continuación (haga clic para abrirlo):

GenericTest iOS Archivos fuente

Para añadir estos archivos a su extension, primero tiene que ir a sus Propiedades de iOS e introducir un nombre de clase (para este ejemplo, "GenericTest"). A continuación, puede utilizar la opción "Enter framework path" (si los archivos están presentes en su máquina macOS utilizada para la construcción) o la opción "Add Source" (si están en su máquina de desarrollo); independientemente de la opción que utilice, tendrá que dirigirlos al directorio que contiene sus archivos .h y .mm.

Puede añadir estos archivos fuente al mismo extension que se configuró en la sección anterior para Android, lo que significa que cualquier función ya definida en ese extension funcionará ahora también en iOS (dado que los nombres externos de todas las funciones en Android y iOS son los mismos). Si está creando un nuevo extension, puede simplemente crear las funciones de la misma manera que se ha descrito anteriormente.

Android (Callbacks)

public void onRestart()
public void onStop()
public void onDestroy()
public void onPause()
public void onResume()
public void onActivityResult(int requestCode, int resultCode, Intent data)
public void onRequestPermissionsResult(int requestCode, String permissions[], int[] grantResults)
public void onNewIntent(android.content.Intent newIntent)
public void onWindowFocusChanged(boolean hasFocus)
public void onConfigurationChanged(Configuration newConfig)
