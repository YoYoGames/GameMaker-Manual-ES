¿Cómo desactivar Extensions?

GameMaker tiene extensions para algunas plataformas de destino, por ejemplo, Steamworks y GDK.

Si está utilizando ambos extensions, querrá desactivar el GDK extension cuando construya para Steam, y viceversa.

Este problema no se limita a estos dos extensions, y se aplica a cualquier extension que esté destinado a un objetivo específico.

Desactivación selectiva de extensions

En primer lugar, establezca una configuración para cada una de sus plataformas de destino, si aún no lo ha hecho.

Este ejemplo sólo utiliza Steamworks y GDK.

Seleccione la configuración para una plataforma de destino. Supongamos que has seleccionado la configuración del GDK.

Abra un extension que no deba utilizarse para el GDK. Para este ejemplo, abra el Steamworks extension.

Desactive todos los objetivos en Copias a. Esto asegurará que el Steamworks extension no se ejecute cuando se utilice la configuración del GDK.

Ahora, switch a su configuración de Steamworks. Abre el GDK extension, y desactiva todos sus objetivos.

¡Hecho!

Ahora, cuando construyas para GDK, utiliza la configuración de GDK; cuando construyas para Steam, utiliza la configuración de Steamworks, y así sucesivamente.

Utilizar las configuraciones de este modo permite controlar qué extensions se utiliza para construir qué plataformas.

Si tienes más extensions, dales el mismo tratamiento: desactiva sus exportaciones en las configuraciones que no las necesitan.

Si tiene más plataformas de destino, añada configuraciones para ellas y modifique qué extensions se utiliza para ellas.
