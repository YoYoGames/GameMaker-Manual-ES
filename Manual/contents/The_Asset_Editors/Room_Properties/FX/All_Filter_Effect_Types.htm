Tipos y parámetros de FX

Esta página enumera todos los tipos de Filtros/Efectos proporcionados en GameMaker, junto con la siguiente información sobre cada FX:

  • Nombre interno (por ejemplo, "_filter_tintfilter") que se puede pasar a fx_create() para crear un FX en tiempo de ejecución
  • Descripción
  • Ejemplo de imagen
  • Parámetros del IDE, que pueden ser editados a través de las Propiedades de la Capa
  • Parámetros de tiempo de ejecución, que pueden utilizarse para modificar los FX en runtime, utilizando las funciones de filtro y efecto
    • Los parámetros de Runtime también mencionan sus tipos, para que sepa qué tipo de datos aplicar a cada parámetro.
