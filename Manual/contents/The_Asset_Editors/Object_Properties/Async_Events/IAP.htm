Compra en la aplicación

Este evento sólo puede ser activado por los diferentes In App Purchase (IAP) extensions para las diferentes plataformas de destino disponibles en el Marketplace. Este evento siempre generará un mapa DS exclusivo para este evento y almacenado en la variable especial async_load. Este mapa DS tendrá diferentes pares de claves/valores dependiendo de la función extension que lo haya desencadenado, pero siempre habrá una clave "id" que puede utilizarse para identificar el tipo de evento In App Purchase que es. Para conocer los detalles exactos de los posibles valores de retorno y las funciones que generan este evento, consulte los manuales individuales en PDF que se incluyen con cada uno de los extensions.
