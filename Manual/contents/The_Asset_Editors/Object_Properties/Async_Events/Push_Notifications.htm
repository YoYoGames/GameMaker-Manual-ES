Notificaciones Push

El evento de notificación push se activa mediante la devolución de llamada de notificaciones push en el SO del dispositivo, ya sea desde una fuente local o remota. Genera un Mapa DS que es exclusivo de este evento y se almacena en la variable especial async_load.

Este evento es utilizado exclusivamente por extensions (como la mensajería en la nube de Firebase extension), así que consulte la documentación de su extensión para obtener instrucciones sobre el uso de este evento.
