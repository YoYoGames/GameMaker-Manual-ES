El editor de Room

El editor room es el lugar donde se crean las salas de juego. Cada juego requiere al menos un room para funcionar, y en el room puedes colocar instancias, sprites, tiles, paths y fondos. Cada uno de estos diferentes assets puede ser colocado en su propia y única capa que luego puede ser ordenada como quieras en el Editor de Capas. Debido a la complejidad del editor de room, le daremos primero un breve resumen de las características más importantes, y luego podrá encontrar detalles más profundos en los enlaces de las secciones que aparecen a continuación.

Cuando cree un recurso room, puede hacer clic con el botón derecho del ratón en en el Navegador Asset para abrir el menú contextual, que le permitirá crear hijos rooms (véase la página sobre Herencia para más información), abrir el room para editarlo, añadir un nuevo grupo de recursos para organizar mejor el rooms, cambiar el nombre del room o eliminarlo. Tenga en cuenta que para cambiar el orden y/o la herencia de room debe utilizar el Gestor de Salas, que puede abrir utilizando el menú de la parte superior derecha del Navegador de Asset.

El editor de room es en sí mismo un espacio de trabajo y, como tal, puede hacer clic en en la pestaña y arrastrarla fuera de la ventana principal a una nueva ventana propia, quizás en otra pantalla, por ejemplo. También puedes volver a colocarla en la ventana principal arrastrando la pestaña a la parte superior de IDE y soltando el botón del ratón.

La interfaz de usuario del editor room es fácil de navegar y está dividida en varias secciones discretas. Las partes del editor que están acopladas -las propiedades de la sala y el editor de capas, así como las diferentes secciones de propiedades de las capas- también pueden eliminarse del acoplamiento simplemente arrastrándolas hacia fuera en el workspace, y pueden volver a añadirse al acoplamiento arrastrándolas a los lados o al fondo del workspace.

A continuación se ofrece un breve resumen de cada una de las secciones del editor de Room:

Reglas / Guías

Las reglas a lo largo del borde del lienzo le muestran la posición de las cosas que se colocan dentro de él, y están marcadas desde (0, 0) que es el centro del lienzo y el origen para el room. Puede hacer clic y arrastrar en las reglas para tirar de una guía horizontal o vertical en el room, y esta guía puede entonces ser utilizada para posicionar con precisión los diferentes assets que se están utilizando, ya que al mover un asset cerca de uno se "ajustará" a la guía. Mientras se posiciona assets dentro de room, también se mostrarán temporalmente guías inteligentes que indican la distancia entre assets, así como la distancia desde el límite o punto central de room.

La distancia a la que assets se "ajusta" a las guías, así como otras propiedades, se pueden establecer en las Preferencias del Editor de Salas.

Editor de capas

El editor room coloca las cosas en capas dentro de room. Cada capa tiene una "profundidad" discreta, donde las que aparecen al final de la lista en la ventana de capas se dibujarán debajo de las que aparecen cerca de la parte superior.

¡IMPORTANTE! Hay una profundidad de capa mínima y máxima de -16000 a 16000. Todo lo que se coloque en una capa fuera de ese rango no se dibujará aunque todos los eventos se seguirán ejecutando normalmente.

Las capas se crean haciendo clic en el botón apropiado para el tipo de capa que se desea crear, que son:

Instancias Esta capa es para colocar las instancias que se utilizarán en el room.
Mapas de azulejos Esta capa sirve para colocar tiles de conjuntos de azulejos comomapas de azulejos.
Caminos Esta capa sirve para mostrar y añadir caminos en la página web room.
Activos Esta capa es para añadir visualmente assets a la room, como sprites y secuencias.
Filtro/efecto Este tipo de capa define un filtro/efecto que se aplica a todas las capas por debajo de ella.
Antecedentes Este tipo de capa define un fondo, que es esencialmente una imagen de un solo color o sprite que puede estar en mosaico sobre toda la room.
UI This creates a global UI layer that is shared among rooms.

También puede crear una carpeta de capas utilizando el botón de carpeta donde puede agrupar las capas seleccionadas, así como eliminar las capas seleccionadas con el botón de eliminación . Cada uno de estos tipos de capas se discute aquí con más detalle.

Tenga en cuenta que puede alternar la herencia para el editor de capas, lo que afectará al orden y la visibilidad de las mismas. La visibilidad en sí misma puede establecerse haciendo clic en el icono del ojo al lado de cada una de las capas, o puede hacer clic en el de la parte superior para activar/desactivar todas las capas a la vez. También tiene la posibilidad de "bloquear" una capa (o todas ellas) para que no puedan ser modificadas haciendo clic en el icono del candado . Ten en cuenta que las capas marcadas como invisibles no se mostrarán cuando se ejecute el juego.

Propiedades de las capas

La ventana de propiedades de la capa room cambiará en función de la capa actualmente seleccionada en el editor de capas.

Cada ventana tendrá las diferentes propiedades y listas asociadas con el tipo de capa actualmente seleccionada, y usted podrá editar los detalles fundamentales de cómo se muestra la capa y qué hay realmente en la capa. Consulte aquí para obtener más detalles.

Propiedades de las habitaciones

Esta sección del editor de room es donde puede definir todas las propiedades básicas de room, como su tamaño, si utiliza la física, o las cámaras y ventanas que requiere. Esta sección se discute con más detalle aquí.

Caja de herramientas de capas

Ciertos tipos de capas tendrán herramientas adicionales añadidas a la parte superior de la IDE en la caja de herramientas de capas (por ejemplo, las capas tile o las capas path ). Las herramientas exactas cambiarán en función del tipo de capa que se esté editando, y así se explica en la página que explica cada capa, aquí.

Caja de herramientas de la habitación

En la parte superior del lienzo principal de Room Editor tienes unos cuantos controles para manejar cómo se muestran las cosas. Estos son:

  • Activar la rejilla del lienzo: Esto activará/desactivará la cuadrícula del lienzo del Editor Room. Esta es una rejilla que GameMaker dibuja sobre el lienzo principal para dividirlo en secciones, y por defecto se establece en 32x32px de tamaño. Sin embargo, si haces clic en el icono del menú de la rejilla abrirás las opciones de la rejilla:Estas opciones te permiten establecer el color y el alfa de la rejilla, así como los valores de las celdas para la rejilla a lo largo de los ejes X e Y. También tiene la opción de activar o desactivar el ajuste de la rejilla aquí (activado por defecto). Puedes utilizar los atajos de teclado "G" y + "G" para cambiar la visibilidad de la rejilla y el ajuste de la misma, respectivamente.
  • Controles de zoom del lienzo: Estos botones controlan el nivel de zoom del lienzo actual. Puede acercar o alejar el zoom y al hacer clic en el botón se restablecerá el lienzo para que sea 1:1 con el room que se está editando. También puede hacer clic en el botón de ajuste de la ventana para hacer que todo el lienzo room encaje dentro del editor actual workspace (esto ampliará o reducirá el zoom según corresponda para que encaje). Tenga en cuenta que también puede acercarse y alejarse utilizando los botones / y la rueda del ratón , y pulsando / + se ajustará el lienzo room a 1:1 con la configuración room.
  • Mostrar vistas: Al hacer clic en esta opción se activará o desactivará el rectángulo de vista. Cuando está habilitado, verá un área resaltada que significa las diferentes vistas que están habilitadas para el room.
  • Reproducir animaciones: Cuando añada sprites, sequences o tiles animado, es posible que quiera hacerse una idea de cómo quedarán dentro del propio room sin tener que compilar todo el proyecto, por lo que puede hacer clic en esta opción para iniciar la reproducción de las diferentes animaciones. Al hacer clic de nuevo, se detendrá la animación.
  • Lienzo Seleccionar desde cualquier capa: Por defecto, cuando haga clic en un elemento en el lienzo room sólo podrá seleccionar aquellos assets que estén en la capa que se esté editando en ese momento, sin embargo, si habilita esta opción podrá hacer clic en cualquier elemento y se seleccionará, cambiando la capa de destino a aquella en la que se encuentre el elemento. Puede utilizar el atajo de teclado " P" para activar temporalmente esta opción (mantenga pulsado "P" para activar y suelte para desactivar)

Lienzo de la habitación

El centro de la ventana del editor room está ocupado por el área donde se realiza toda la edición real. Aquí es donde colocarás tus instancias y assets, dibujarás tu tiles, o posicionarás tu paths. Puedes acercarte y alejarte usando la rueda del ratón o los controles de room en la parte superior, y puedes desplazarte manteniendo el botón central del ratón presionado y moviendo el ratón alrededor o usando + mientras te mueves.

Al crear capas de asset o capas de instancia, puede colocar la asset o la instancia simplemente arrastrándola desde el Navegador de Asset y luego soltándola donde quiera colocarla. También puede seleccionar un asset o una instancia del navegador Asset y mantener pulsado para previsualizar el recurso "in situ", y si además pulsa el botón izquierdo del ratón puede añadir la instancia e incluso "pintarla" en la capa room manteniendo el botón pulsado y arrastrando el ratón. Tenga en cuenta que, por defecto, las instancias y sprites que se añadan a un room de esta manera se ajustarán a la configuración de la cuadrícula definida en room, pero puede mantener pulsado / para que la instancia siga el ratón con precisión y no se ajuste (si suelta / podrá volver a ajustarse).

Para paths puede crear una nueva capa path y un recurso path directamente desde el editor Room - o crear un nuevo recurso path y luego arrastrarlo a la ventana principal del editor como lo haría para una instancia - y luego editar el path y sus conexiones en la ventana del editor también. En el caso de tile sets, puedes "pintarlos" desde el editor de tile set.

Tenga en cuenta que puede seleccionar y mover o eliminar varios assets de la misma capa manteniendo pulsados / + en los que desee, y la mayoría de los recursos del editor room le permitirán rotarlos y escalarlos (individualmente o como grupo) pulsando y manteniendo a lo largo de los bordes o esquinas exteriores y arrastrando el ratón. También puede hacer doble clic en sobre cualquier asset (tiles, instancias, sprites) para que el editor se desplace hasta él y abra la ventana de propiedades del asset seleccionado.

Barra de estado

La barra de estado se utiliza para mostrar información específica del contexto. La barra de estado siempre le mostrará en qué lugar de room se encuentra el cursor del ratón, pero también mostrará información adicional basada en la capa que se está editando, la herramienta que se está utilizando y el estado de esa herramienta.

Menú de la habitación

Además de las herramientas del editor, también verá que se ha añadido un nuevo menú desplegable en la parte superior de IDE.

Estos menús se explican a continuación:

  • Vista de capas: Esto reabrirá la ventana del Editor de Capas si la ha cerrado en algún momento.
  • Propiedadesde la sala: Esto reabrirá la ventana de propiedades de Room si la ha cerrado en algún momento.
  • Propiedades de la capa: Esto reabrirá la ventana de propiedades de la capa si la ha cerrado en algún momento.
  • Orden de Creación de Instancias: Esto abrirá la ventana de orden de creación de instancias (que también puede abrirse usando el botón en la ventana de propiedades de la sala ):
    Esta ventana enumera todas las instancias en el room en el orden en que serán creadas (de arriba a abajo). Si necesita que una instancia específica se cree antes que otra, puede simplemente hacer clic en y arrastrarla a la posición que desee.
  • Restablecer Windows en el escritorio actual: Esto restablecerá el diseño de la ventana del editor room a sus valores por defecto para el escritorio workspace actualmente enfocado.

  • Edición de mosaicos: Cuando se trabaja con una capa de tile, esta opción aparecerá resaltada en el menú desplegable. Tiene el siguiente submenú:
    Cuando edite la capa tile puede seleccionar múltiples tiles haciendo clic, manteniendo pulsado y arrastrando el botón izquierdo del ratón , o puede seleccionar y agrupar tiles individuales utilizando / + , o eliminar tiles individuales de la selección utilizando + . Desde este menú puede borrar la selección o invertirla, así como cortar, copiar o pegar la selección en otra capa (o incluso en la misma capa pero en otra posición). La última opción reabrirá la ventana acoplada del editor tile set en caso de que la haya cerrado.
  • Convertir imagen en mapa Tile : La opción "Convertir imagen en mapa Tile " es una potente herramienta que permite importar una sola imagen y, a continuación, extraer el tiles utilizado de la imagen y recrearlo como una capa de mapa tile en el editor room, creando los tile set y sprite necesarios como parte del proceso. Cuando seleccione esta herramienta, se le pedirá que proporcione un archivo de imagen sin comprimir (PNG, GIF o BMP) que se cargará. Al cargarlo, se le presentará la herramienta de importación de imágenes:
    Aquí se le ofrecen opciones sobre cómo dividir el tile al crear el tile set, y puede establecer la anchura y la altura de las celdas, así como cualquier desplazamiento necesario alrededor de los bordes. También puede establecer la anchura aproximada (en píxeles) para el sprite final que se crea. Por ejemplo, si el tile set se compone de 64x64 tile celdas y usted establece un ancho aquí de 200, el sprite final que se crea para su uso en el tile set será de 192 píxeles de ancho (es decir: 3 tile celdas). Si lo deja en el valor por defecto de 0, entonces GameMaker intentará hacer un sprite lo más "cuadrado" posible con aproximadamente el mismo número de celdas horizontales y verticales tile.

    Después de establecer la forma en que la imagen debe ser dividida, al hacer clic en la muestra al lado de la opción Quitar Color se abrirá el selector de color y le permitirá seleccionar un color que debe ser eliminado de la sprite final. Por lo general, se trata de un color de fondo que se desea eliminar, y si se establece esta muestra en cualquier cosa que no sea 100% transparente (alfa 0), se eliminará el color seleccionado en la importación.

    Por último, tiene la opción de nombrar la capa del mapa sprite, tile set y tile que se creará para usted con esta herramienta. Al finalizar la importación, se creará un sprite con todas las imágenes dispuestas en una cuadrícula, así como un tile set a partir de este sprite. En el room, se creará una nueva capa de mapa tile y se reproducirá la imagen utilizando el tile set generado. Tenga en cuenta que la herramienta no duplicará tiles y que, en cambio, reconocerá cuando una celda tenga datos de imagen que coincidan con otra celda (esto incluye tiles rotada).
