Características de la pluma

Feather proporciona una terminación de código inteligente y una comprobación de sintaxis mejorada en su GML Code scripts, junto con opciones de refactorización inteligentes.

Esta página enumera las funciones que Feather ofrece en el editor de código GML para los scripts y los eventos de los objetos. Estas características sólo estarán disponibles si Feather es enabled.

Información sobre Hover

Al pasar el ratón por encima de una función, sprite, un color o cualquier otro valor especial, se mostrará su información. Puedes ver algunos ejemplos a continuación.

Refactorización de activos

Si cambias el nombre de un asset en tu Asset Browser, Feather actualizará automáticamente todas las menciones del mismo en el código de tu proyecto, para que esas referencias no se rompan. Esto se puede activar o desactivar en la configuración de Feather.

También puede cambiar el nombre de un asset a través del Editor de Código. Coloque su cursor de texto en la referencia asset, y presione CTRL/CMD + SHIFT + R.

Introduzca un nuevo nombre y pulse "Aplicar". El propio asset, y todas las referencias a él en su proyecto, serán renombrados.

Soluciones rápidas

Cuando obtengas un error o una advertencia en tu código, coloca el cursor de texto en la línea reportada y presiona CTRL/CMD + Q para que aparezca el menú de Arreglos Rápidos. También puede hacer clic en el icono de la pluma en el canalón.

Este menú le muestra una solución rápida si es posible, y le permite desactivar la regla para esa línea, o cambiar su gravedad.

El ejemplo anterior muestra una solución rápida utilizada para arreglar un evento en el que el objetivo de superficie no se restablecía después de ser cambiado.

Al crear una función, el menú Quick Fix le permite generar automáticamente comentarios JSDoc para la función.

Buscar todas las referencias

Feather le permite encontrar todas las referencias a una variable en todo su proyecto.

Coloque el cursor del texto en una variable y pulse F3. Verás una pequeña barra de búsqueda en la parte superior de tu espacio de trabajo:

Navegue hasta la referencia anterior o siguiente pulsando la flecha correspondiente, o ciérrela.

Esta búsqueda es inteligente, por lo que en lugar de buscar simplemente una cadena de texto, busca referencias de la variable seleccionada desde su ámbito declarado. Esto significa que dos variables que tengan el mismo nombre pero que existan por separado en dos objects diferentes, no coincidirán.

Lista de todas las referencias

Coloque el cursor de texto en una variable y pulse SHIFT + F3. Esto abrirá la ventana de resultados de búsqueda y mostrará todas las referencias de la variable seleccionada en una lista.
