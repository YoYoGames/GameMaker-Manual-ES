Fragmentos de código

Una herramienta muy práctica que tiene a su disposición cuando edita su código es el uso de fragmentos de código. Al utilizar el editor script, puede pulsar F4 para abrir la ventana emergente de fragmentos de código, que le permite seleccionar de una lista uno de los métodos de código más utilizados:

Desde esta ventana puede seleccionar con el ratón el fragmento a utilizar, o puede pulsar la tecla de acceso directo asociada que aparece a la derecha. Esto añadirá el fragmento a su código para su edición.

También puede definir sus propios fragmentos de código si lo desea. Antes de hacer esto, tendrá que crear un archivo llamado "snippets.txt" en uno de los siguientes directorios:

  • Ventanas: %ProgramData%\GameMaker\User\
  • macOS: /Users/Shared/GameMaker/User/

Esta carpeta no se modifica durante ninguna actualización de GameMaker, por lo que su archivo editado permanecerá intacto (pero cualquier edición del archivo de fragmentos base en el directorio de instalación se revertirá).

Una vez creado el archivo en el directorio de usuario, puede editarlo con cualquier editor de texto siguiendo estas reglas:

  • Cada fragmento tiene que estar en una línea separada (no debe haber líneas en blanco)
  • Cada fragmento debe comenzar con la tecla de acceso directo que se va a utilizar, seguida de "-" y, a continuación, el nombre del fragmento (que es el que se muestra en el menú), seguido de dos puntos ":", por ejemplo:

I - Creación de instancias:

Después de los dos puntos se añade el fragmento de código

I - Creación de instancias:instance_create_layer(x, y, |layer|, object);

El código que añadas también debe seguir un formato específico en el que:

  • Todo el código está en una sola línea
  • Se utiliza "$" para una nueva línea
    • NOTA Cuando se crean varias líneas nuevas, debe haber un espacio entre cada signo "$". Por ejemplo, escribiría $ $ $ para añadir tres líneas nuevas.
  • Ponga el texto que necesita ser resaltado entre los símbolos "|". El Editor de Código seleccionará automáticamente este texto para editarlo cuando se añada el fragmento.

Vea los ejemplos que ya están en el archivo base de fragmentos para ver cómo se configura siguiendo las reglas anteriores. Puedes encontrar el archivo base en el directorio de instalación de GameMaker:

  • Windows: GameMaker\TextEditor\\\Nsnippets.txt
  • macOS: /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
