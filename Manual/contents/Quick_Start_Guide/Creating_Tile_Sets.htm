Creación de conjuntos de baldosas

Losconjuntos de baldosas se crean a partir de sprites, pero también se clasifican como un tipo separado de asset, ya que la forma en que GameMaker los maneja es un poco diferente a la de sprites y objects. Así, un tile set es una colección de tiles creada juntos en un sprite, y un tile es simplemente un recurso gráfico que se añade a un room de GameMaker como parte de un mapa de azulejos. Estos mapas de tile se dibujan en la pantalla pantalla con muy poco procesamiento de la CPU/GPU, lo que los hace ideales para diseñar cualquier elemento estático en los niveles.

Para que el procesamiento de tiles sea tan rápido, tienen ciertas restricciones, siendo la más básica que sólo pueden ser rectangulares o cuadradas (si quieres algo distinto a esto, entonces tendrás que crear un sprite normal y utilizar la Capa de Activos en el editor room, pero eso lo veremos más adelante en la sección de habitaciones). Otras restricciones incluyen el hecho de que sólo pueden ser girados 90°, o reflejados o volteados, y sólo pueden sólo pueden ser animados con fotogramas de una potencia de 2 (es decir: 2, 4, 8, 16, etc...).

Observe que el primer tile está en blanco en esa imagen. La forma en que el mapa tile para el room se almacena en GameMaker significa que el primer tile siempre tendrá que estar en blanco, ya que es esencialmente el "borrar" tile o el "espacio vacío" tile (este "blanco" tile tendrá un valor de índice de 0 - tiles están indexados con un valor que comienza en 0 y cuenta de izquierda a derecha, de arriba a abajo). Por lo tanto, al crear su tile sets, la esquina superior izquierda tile siempre tendrá que estar vacía de esta manera. Tenga en cuenta también que, aunque la imagen anterior muestra una pared completa tile set con 47 tiles, es posible que usted no necesita tantos, ya que puede girar y voltear tiles cuando los coloque en el editor room más adelante.

Con el sprite creado, podemos seguir adelante y definir el propio tile set, lo que significa que primero hay que crear un nuevo recurso tile set haciendo clic con el botón derecho del ratón en la carpeta de activos Tile Set y seleccionando Crear. Hemos fijado el tamaño de tile set en 64px, y hemos dejado el resto de las opciones en sus valores por defecto, ya que no tenemos necesidad de desplazamientos ni nada parecido (los valores de desplazamiento son útiles para cualquier tile set sprite que tenga espacios alrededor de cada uno de los tiles). Querrá nombrar el tile set asset usando un prefijo apropiado como " ts_" para poder identificarlo fácilmente más tarde, y tenga en cuenta que se aplican las mismas restricciones que para sprites, es decir: el nombre no puede empezar con un número, y sólo puede utilizar el guión bajo "_" y caracteres alfanuméricos.
