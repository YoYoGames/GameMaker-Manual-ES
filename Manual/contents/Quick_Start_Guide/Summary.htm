Resumen

Hemos llegado al final de esta Guía de Inicio Rápido y esperamos que hayas adquirido suficientes conocimientos y confianza para empezar a utilizar GameMaker para crear tus propios proyectos:

  • Cómo crear Sprites y Tile Sets y editarlos en el Editor de Imágenes
  • Cómo crear objetos y salas
  • La diferencia entre un objeto y una instancia
  • Cómo dibuja GameMaker y cómo utilizar el evento Draw
  • La diferencia entre el evento Draw y el evento Draw GUI
  • Cómo mover las instancias en una habitación
  • Conceptos básicos de programación como variables, funciones y expresiones

Has recorrido un largo camino en poco tiempo, pero ese no es el final del viaje. Este manual tiene dos extensas secciones de referencia que cubren todos los aspectos de IDE y cómo programar utilizando GML Visual o GML, y le recomendamos encarecidamente que vuelva aquí a esas secciones cada vez que no esté seguro de algo o quiera probar algo nuevo:

  • Referencia IDE
  • Referencia visual de GML
  • Referencia del código GML
  • Información adicional

También puedes obtener una serie de tutoriales de GameMaker en la página de inicio que utilizan los conceptos básicos que has aprendido aquí para crear pequeños y divertidos juegos, como Space Rocks.

Para obtener más ayuda técnica, como la configuración de las diferentes plataformas de destino, debes consultar el servicio de ayuda de GameMaker, y GameMaker también tiene una serie de blogs técnicos que cubren todo tipo de aspectos diferentes del uso de GameMaker, desde los conceptos más básicos hasta las técnicas más avanzadas, por lo que hay algo para todos. En particular, echa un vistazo al siguiente artículo:

  • Seis recursos que enseñaron a los profesionales a codificar con GameMaker

También nos gustaría señalar que GameMaker tiene sus propios foros llamados GMC (" Game Maker Community") que están dirigidos por la comunidad y llenos de miembros útiles que están dispuestos a dar algo de su tiempo para ayudar con cualquier problema de programación o técnico que pueda tener.

Por último, hay una gran cantidad de recursos de GameMaker en Internet, así que intenta siempre buscar información en tu navegador cuando estés atascado o busques tutoriales, y también hay varias comunidades activas en sitios como Reddit o Discord donde puedes registrarte y participar.

Hagas lo que hagas, esperamos que te diviertas con GameMaker y, si (cuando) creas un juego y lo publicas, no olvides escribirnos para contárnoslo en el GameMaker Showcase.

¡Feliz creación de juegos!
