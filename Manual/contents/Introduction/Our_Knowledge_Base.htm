Nuestra base de conocimientos

El manual de GameMaker que estás leyendo ahora tiene toda la información que necesitas para utilizar el producto y empezar a crear juegos. Sin embargo, no incluye información sobre la configuración para las diferentes plataformas para las que GameMaker puede crear juegos, ni tampoco incluye ninguna guía o ayuda para los diferentes extensions que están disponibles para ampliar la funcionalidad para algunas plataformas. Tampoco contiene información de última hora que pueda ser relevante para el producto o para plataformas específicas que no haya podido ser incluida en este manual.
