El microservidor web

La función principal de esta pequeña herramienta es conectar y servir archivos a algunas de las plataformas de destino disponibles para GameMaker (como el objetivo HTML5). También sirve archivos para el manual de GameMaker cuando se utiliza sin conexión.

Cuando ejecutes un juego en un objetivo que requiera la Micro Web Server, verás que se abre una nueva ventana mientras el juego se sirve en el dispositivo elegido:

NOTA La primera vez que ejecute un juego en un objetivo que requiera la Micro Web Server, es posible que reciba una alerta del cortafuegos. Debe permitir que la web server se comunique a través del cortafuegos si esto ocurre.

Cuando la Micro Web Server está funcionando, puedes acceder a tu juego desde cualquier dispositivo local (e incluso de forma remota, si el reenvío de puertos está bien configurado). Simplemente accede a la dirección IP local de tu ordenador a través del puerto mostrado en esta ventana, y podrás jugar a tu juego desde otro dispositivo, como un teléfono móvil, u otro ordenador.

Por ejemplo, digamos que la dirección IP local de tu ordenador es 192 .168.0.110(lee esto para saber cómo encontrarla), y el puerto que aparece en la ventana de Micro Web Server es 51264. Después de ejecutar el juego en tu ordenador, te conectarías a 192 .168.0.110:51264 desde un navegador en tu móvil, y jugarías tu juego.

La web Server mostrará una cierta cantidad de información de depuración relacionada con los archivos que se han servido, y tiene un botón para borrar esto en la parte inferior. Al lado del botón "Clear" está el path al archivo JS raíz para el proyecto, mientras que debajo de eso tienes la opción de Velocidad de Red. Esta opción está configurada como "Sin límite" por defecto, pero al hacer clic en ella se abrirá una lista de diferentes opciones de red:

Al cambiar esta configuración también cambiará la opción de Tamaño del paquete a la derecha. También puedes editar esto manualmente, en cuyo caso la Velocidad de la Red se establecerá en "Personalizada". Si lo ajustas a cualquier otro valor que no sea el máximo, se reducirá la velocidad de transferencia de la red y podrás probar los juegos en un entorno con poco ancho de banda.

También puede ver la lista de IPs permitidas junto con el número de puerto que se está utilizando, pero estos no pueden ser editados en la Micro Web Server. En su lugar, deben establecerse en la configuración del Administrador de Dispositivos para el objetivo del dispositivo dado, o en las Preferencias de HTML5 cuando se utiliza el dispositivo "por defecto".
