Generador de imágenes de proyectos

El generador de imágenes del proyecto se accede desde el menú Herramientas en en la parte superior de GameMaker IDE. Esta herramienta está diseñada para crear rápidamente todas las diferentes pantallas de bienvenida, imágenes de promoción e iconos que se requieren para todas las diferentes plataformas de destino a partir de un único conjunto de imágenes. Se suministra la imagen base y el icono base y luego haces clic en el botón Generar , y tus Opciones de Juego para cada plataforma se actualizarán con las nuevas imágenes.

Para que esto funcione correctamente es necesario proporcionar una pantalla de inicio de alta resolución y una imagen de icono de alta resolución. Los tamaños recomendados serían:

  • Pantalla de inicio: al menos 1920x1080px, o 1080x1920px para retrato, y con autoría de 24bit *.png sin transparencias.
  • Iconos: al menos 1024x1024px (que es el tamaño del icono más grande que se requiere), y con autoría de 24bit *.png que puede tener transparencias

Una vez que haya suministrado estas imágenes, se mostrarán en las ventanas de vista previa y podrá establecer el tipo de "ajuste" que se utilizará al aplicarlas a las diferentes plataformas. Puede elegir entre los siguientes:

  • Mantener relación de aspecto - esto escalará la imagen para que se ajuste manteniendo la relación de aspecto y añadiendo un color de fondo para rellenar los lados si el ajuste no es 1:1
  • Estirar para ajustar - esto estirará las imágenes para que se ajusten al tamaño completo de la imagen/icono.
  • Recortar para ajustar - esto tomará la imagen base y la hará encajar mejor a lo largo del eje horizontal o vertical y luego recortará el resto. Por ejemplo, si usted tiene una imagen de 640x480 que tiene que ser utilizado como una pantalla de inicio para 480x800, entonces la imagen se escalará para ajustarse al eje vertical (480 -> 800) y el resto de la imagen se recortará. Del mismo modo, si tiene una imagen de 1080x1920 que tiene que ajustarse a 800x800, la imagen se reducirá para ajustarse al eje horizontal y el resto se recortará. y el resto se recortará.

NOTA: La excepción a la afirmación anterior son los iconos adaptativos de Android. Estos no se generarán con esta herramienta debido a la forma en que están estructurados.

¡IMPORTANTE! Aunque esta herramienta generará todas las imágenes que necesites, ¡no significa que sea perfecta! Intentará hacer todo lo posible para que las imágenes se adapten a los diferentes tamaños y resoluciones de cada plataforma, pero a veces esto puede no pero a veces puede que no se vea como usted quiere, ya que no hay una solución única para todos. Por lo tanto, después de ejecutar esta herramienta, le recomendamos que revise todas las imágenes en las Opciones de Juego de cada plataforma y modifique las que no se vean exactamente como usted quiere. como quieres manualmente.
