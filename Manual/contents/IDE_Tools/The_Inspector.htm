El Inspector

La ventana del Inspector muestra todas las propiedades asociadas a los elementos seleccionados en la página web IDE.

Puede seleccionar uno o varios elementos en cualquier editor, y ver o cambiar sus diversas propiedades:

El Inspector puede abrirse desde el menú Windows en la parte superior de la ventana GameMaker:

Seleccione esta opción para abrir una nueva ventana del Inspector -- puede tener más de una abierta al mismo tiempo.

Después de abrir una ventana del Inspector, puede anclarla en cualquier lugar de IDE, o utilizarla como ventana flotante:

Uso del Inspector

El Inspector le permite ver y editar assets sin tener que depender únicamente de los espacios de trabajo.

Por ejemplo, es posible que edites el código de tu evento en pantalla completa, y que vuelvas a navegar con frecuencia a un Workspace para editar las propiedades de ese objeto.

Con el Inspector, puede ver los detalles de un objeto mientras permanece en la vista de pantalla completa:

Esto hace posible modificar las propiedades de un objeto y eliminar/cambiar eventos mientras se edita su código en pantalla completa, y también configurar la crianza de los hijos, las definiciones de las variables, y más.

Aunque este ejemplo en particular sólo cubre objects, el Inspector puede hacer mucho más y permitirle editar muchos tipos diferentes de assets (por ejemplo, Sprites, Fonts, Curvas de animación, etc.).

Elementos del editor

Además de assets, también puede inspeccionar elementos en un editor, como las capas en un room, las pistas en un Sequence, etc.

El Inspector le permite aplicar un filtro o efecto a una capa de room, lo que no puede hacer a través del propio Editor de Room:

NOTA Cuando se aplica un filtro/efecto a una capa que no es de efectos utilizando el Inspector, siempre se aplicará como un efecto de "capa única", es decir, su efecto sólo se aplicará en esa capa en particular y no en otras capas por debajo de ella.

El botón "Abrir el editor"

Si tiene seleccionado un asset, puede hacer clic en "Abrir Editor" en el Inspector para abrir su editor asset.

Por ejemplo, al hacer esto con un object se abrirá el Editor de Object, donde podrá modificar sus propiedades, añadir eventos, etc.

Sin embargo, cuando se edita cualquier cosa dentro de un room, como una instancia o una capa, el Inspector sólo se centrará en los editores dentro del Editor Room.

Como se muestra más arriba, si se selecciona una instancia en room y se hace clic en "Open Editor" en el inspector, se abrirá su editor de instancias.

Del mismo modo, si se selecciona una capa de Path y se hace clic en "Abrir editor", el foco se desplazará al panel Capas, donde se pueden editar las propiedades de la capa, en lugar de abrir el editor de Path asset.

Multi-Select

El Inspector le permite ver y editar las propiedades de varios assets y elementos (como las capas de room y las pistas de sequence ) al mismo tiempo.

Seleccione su assets o elementos deseados manteniendo pulsada la tecla CTRL / CMD y haciendo clic con el botón izquierdo del ratón. Los elementos seleccionados se mostrarán en una lista de "selección múltiple" en el Inspector:

Pulse la X junto a un elemento para eliminarlo de su selección.

Edición de valores

Si los elementos seleccionados comparten el mismo valor para una propiedad, ese valor compartido aparecerá en ese campo.

Sin embargo, si los elementos tienen valores diferentes para una propiedad, el editor simplemente mostrará - en lugar de ese valor.

Puede cambiar cualquiera de estos campos. El cambio de un valor se aplicará a todos los elementos seleccionados.

Selección dentro de la lista

Puede seleccionar cualquier elemento dentro de su lista de selección múltiple para ver y editar sus propiedades, sin tener que restablecer su selección.

Haga clic en el área vacía de la lista para volver a la vista de selección múltiple.

Tenga en cuenta que no puede seleccionar varios elementos dentro de esta lista.

Selección destacada

Cuando tenga un elemento seleccionado dentro de la lista de selección múltiple, aparecerá en el editor con un contorno rojo:

Puede cambiar el color de este contorno en las Preferencias de la sala y en las Preferencias de la secuencia.

Múltiples inspectores

Puede abrir varias ventanas del Inspector en cualquier momento, y cada una de ellas puede estar vinculada a un elemento específico en la página web IDE.

Para garantizar que cada vez que haga clic en un elemento no cambien todos los inspectores abiertos para mostrar la nueva selección, primero debe bloquear cada ventana de inspector una por una.

Para ello, debe abrir un inspector, seleccionar un elemento de IDE para inspeccionarlo y, a continuación, hacer clic en el botón de bloqueo de en el inspector para bloquearlo en el elemento (se muestra en la imagen siguiente). Una vez que hayas bloqueado un inspector, puedes abrir otro y repetir el proceso.

Tenga en cuenta que una ventana de inspección bloqueada no bloquea la edición de ningún valor del elemento que se está inspeccionando. Simplemente bloquea el inspector en el elemento seleccionado y usted puede seguir editándolo en cualquier momento, incluso cuando el elemento original no es el foco principal de la IDE.
