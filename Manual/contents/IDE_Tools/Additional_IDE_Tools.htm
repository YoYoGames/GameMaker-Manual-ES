Herramientas adicionales IDE

En esta sección del manual se describen las herramientas adicionales de IDE que están disponibles para ayudarte a crear tus juegos y que no se tratan en otras partes del manual, así como aquellas herramientas que son accesorios útiles para las otras secciones de IDE.

  • El Debugger
  • La Micro Web Server
  • La ventana del inspector
  • Local Asset Paquetes
  • Generador de imágenes de proyectos
  • Mezclador de sonido
  • Source Control
