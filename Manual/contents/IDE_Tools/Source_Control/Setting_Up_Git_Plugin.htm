Configuración del plugin Git

Antes de poder utilizar el plugin Git, tenemos que configurar el ejecutable Git para utilizarlo yendo a la sección Plugins - Source Control (Git) de las Preferencias:

El ejecutable Git path debería rellenarse automáticamente si lo encuentra GameMaker, sin embargo, si el campo está vacío, puede hacer clic en el botón y seleccionar manualmente el archivo git.exe incluido en la carpeta cmd de su instalación Git, que suele estar dentro de C:\Program Files\Git\.

Para el resto de las opciones de Git, consulte la página de Preferencias del Plugin.
