Opciones de repositorio

Una vez configurado el ejecutable Git y habilitado Source Control para su proyecto, ahora necesita enlazar el GameMaker IDE a un repositorio, lo que se hace de dos maneras:

  • Crear un repositorio de proyectos
  • Clonar un repositorio existente

En el primer caso, la creación de un repositorio de proyecto, esto creará un repositorio Git directamente donde está su proyecto. En la jerga de Git este es un "repositorio no desnudo", por lo que Push/Pull no tiene sentido en este caso, pero todavía puede confirmar y mantener un seguimiento de los cambios del proyecto, así como revertir los cambios y revisiones si es necesario. Para configurarlo, sólo tiene que hacer clic en Control de Fuentes > Crear Repositorio de Proyecto en el menú principal.

Como función de conveniencia, también puede Clonar un repositorio externo a través de IDE. De nuevo, necesitará el nombre de usuario y la contraseña del repositorio al que se va a conectar configurados en sus preferencias de identidad, y entonces es sólo un caso de hacer clic en Control de Origen > Clonar Repositorio.
