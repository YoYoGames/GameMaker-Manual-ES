Opciones de juego en Ubuntu

En esta sección se describen las diferentes opciones disponibles que controlan cómo se compilarán sus proyectos de juegos en Ubuntu (Linux). Las diferentes secciones son:

General

En esta pestaña se establece el nombre del juego tal y como se mostrará mientras se ejecuta, y también se puede proporcionar un correo electrónico de contacto y una descripción (corta y larga), así como una URL para la página de inicio del juego o del desarrollador.

Aquí también tienes la opción de habilitar Steam en tu juego. Si activas esta opción, asegúrate de que tienes el SDK de Steamworks instalado (ver aquí), y que has añadido el ID de la aplicación de Steam en las Opciones Generales del juego.

Finalmente tienes la opción de Desactivar el sandbox del sistema de archivos. Marcando esta opción se desactivará el sandboxing de GameMaker para guardar y cargar archivos, permitiendo que tus juegos accedan a los archivos desde cualquier parte del sistema que ejecuta el juego. Esta opción está desmarcada por defecto, ya que recomendamos mantener el GameMaker sandbox para los archivos en todos los sistemas y cuando se marca, algunas áreas de guardado/carga pueden seguir sin estar disponibles dependiendo de los permisos a nivel de sistema operativo. Para más información sobre el sandbox, consulte las páginas sobre el Sistema de Archivos GameMaker.

Gráficos

Las opciones gráficas son las que debes configurar para determinar cómo tu juego utilizará la tarjeta gráfica de tu máquina Ubuntu de destino. Se incluyen las siguientes opciones para que las modifiques:

  • Iniciar en pantalla completa: Si esta opción está marcada, el juego se iniciará en modo de pantalla completa, de lo contrario, se iniciará en modo de ventana. Esta opción está desactivada por defecto.
  • Permitir el cambio de pantalla completa: Con esta opción marcada, el usuario puede switch pasar de pantalla completa a ventana y viceversa utilizando los atajos estándar del sistema operativo Ubuntu. Esta opción está desactivada por defecto.
  • Interpolar los colores entre píxeles: Activa la interpolación lineal, que básicamente "suaviza" los píxeles. Para gráficos de píxeles nítidos, debería estar desactivado, pero si tiene buenas mezclas alfa y gráficos con bordes suavizados es mejor dejarlo activado.Está desactivado por defecto.
  • Mostrar Cursor: Cuando esta opción está activada, el cursor del sistema operativo por defecto se mostrará en su juego. Si está desactivada, mientras el ratón esté sobre la ventana del juego, no habrá ningún cursor visible a menos que hayas creado uno como parte del proyecto del juego (está activado por defecto).
  • Utiliza la sincronización para evitar el tearing: Esta opción activa o desactiva la sincronización v. Ten en cuenta que si tienes un juego con una velocidad de room de 120 y el jugador tiene un monitor con una tasa de refresco de 60, al activar esta opción también se bloqueará la velocidad del juego a 60. Esta opción está desactivada por defecto.
  • Permitir el cambio de tamaño de la ventana: Marcar esta opción permite al usuario cambiar el tamaño de la ventana del juego (la opción Ventana sin bordes debe estar desactivada para que esto funcione). Esta opción está desactivada por defecto.
  • Escala: Aquí puede elegir mantener la relación de aspecto (de modo que un room de 4:3 será "letter boxed" en un 16:9) o escalar completamente (estirando la imagen para que se ajuste a la pantalla completa).

    ¡ATENCIÓN! Al apagar la superficie de la aplicación se deshabilitarán todas las opciones de escala establecidas en las opciones de juego de Windows hasta que se vuelva a encender. Ver La superficie de aplicación para más detalles.

Por último, existe la opción de establecer el tamaño de la página de la textura. El tamaño por defecto (y más compatible) es 2048x2048, pero puedes elegir entre 256x256 y 8192x8192. También hay un botón marcado como Vista previa que generará las páginas de texturas para esta plataforma y luego abrirá una ventana para que puedas ver cómo quedan. Esto puede ser muy útil si quieres ver cómo están estructuradas las páginas de textura y evitar tener páginas de textura más grandes (o más pequeñas) de lo necesario. Para obtener más información sobre las páginas de textura, consulte aquí.

NOTA: Ten en cuenta que cuanto mayor sea el tamaño de la página de texturas, menos compatible será tu juego en los PC con especificaciones inferiores.

Imágenes

Aquí puedes añadir una pantalla de bienvenida para tu juego, y establecer el icono del juego que se utilizará. El icono debe ser una imagen de 64x64 píxeles en el formato .png, y la pantalla de bienvenida debe ser al menos del mismo tamaño que el primer room en tu juego (o su puerto de vista) y ser también del formato .png. La pantalla de inicio se mostrará mientras se carga el juego, y puede ser activada o desactivada marcando la opción Usar pantalla de inicio (que está desactivada por defecto).

Vale la pena señalar que GameMaker tiene una herramienta generadora de imágenes del proyecto que se puede utilizar para crear automáticamente todas las imágenes necesarias para todas las diferentes plataformas de destino para las que se está compilando el juego. Si utiliza esta herramienta, debe revisar las imágenes creadas para asegurarse de que son las que necesita.
