Información sobre la plantilla

Esta sección le permite establecer la información de su proyecto que se utiliza cuando se exporta como plantilla.

Aquí puede asignar una imagen, un icono y una descripción para el proyecto de la plantilla, aunque sólo se utilizará cuando se exporte como plantilla. Puede hacerlo abriendo el menú File Menuy haciendo clic en "Exportar proyecto" y seleccionando "Como plantilla".
