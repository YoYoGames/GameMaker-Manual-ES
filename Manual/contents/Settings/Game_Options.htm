Opciones de juego

Las opciones del juego se dividen en dos secciones, las opciones principales y la configuración de la plataforma. Las opciones principales comienzan con las opciones generales, que es donde se pueden establecer ciertas cosas que afectarán al juego independientemente de la plataforma en la que se está diseñando para funcionar.

La ventana tiene las siguientes secciones:

  • Lenguaje de scripting por defecto: Elija si los eventos de Objects y Scripts en el proyecto actual deben utilizar GML Code o GML Visual, o si debe preguntar siempre al crear un nuevo script.
  • Fotogramas de juego por segundo: Este valor establece el límite en el número de fotogramas por segundo que GameMaker debe tratar de mantener sus juegos cuando se ejecutan. Este valor representa el número de veces que el juego loop se ejecutará en un segundo (un solo juego loop se considera un solo "marco") y se establece en un valor por defecto de 60, aunque 30 se utiliza a menudo también. Hay muchos argumentos sobre qué velocidad de room es la ideal, pero la respuesta real depende del tipo de juego que estés haciendo, el tamaño de room, el número de instancias y lo intensivo que sea el código. Por ejemplo, si tienes un room grande y una velocidad de room de 30, el juego puede parecer "entrecortado", ya que cuanto más grande sea el área que estás viendo, más fácil es para el ojo percibir las velocidades bajas de room, pero si el juego es intensivo, entonces establecer la velocidad de room a 120 puede hacer que el juego se "retrase", ya que los fps caen debido a la cantidad de procesamiento que se está haciendo. Así que ten cuidado con este valor y trata de ajustarlo siempre a lo que tu juego necesita.
  • Color de dibujo por defecto: El color por defecto para dibujar texto o foros (y otras cosas) cuando no se ha establecido ningún color mediante código.
  • Color de dibujo por defecto fuera de la región room: El color por defecto que se utilizará para limpiar el fondo de cualquier área fuera del juego room.
  • Generar imágenes del proyecto: Al hacer clic en esta opción se abrirá la herramienta de generación de imágenes del proyecto.
  • Fecha de inicio del proyecto: La fecha en que se inició el proyecto.
  • Tiempo de uso del proyecto: El tiempo que ha pasado desde que inició el proyecto.
  • GUID del proyecto: Este es el identificador único que se crea para cada proyecto dentro de GameMaker. Si desea cambiarlo, puede hacer clic en el botón Crear nuevo GUID para generar uno nuevo.
  • Generar mipmaps para páginas de textura separadas: Esta opción hará que GameMaker cree mipmaps para aquellos sprites que hayan marcado la opción de Página de Textura Separada en los ajustes de Textura del Editor Sprite. Activar el mipmapping significa que la página de textura original para el sprite será escalada y filtrada en múltiples resoluciones dentro del archivo de textura. Esto está normalmente desactivado, pero si estás usando vistas para acercar o alejar grandes áreas del juego room, o estás trabajando con un juego 3D, entonces puedes querer activar esto y luego usar las Funciones de Mipmapping para ajustar cómo el juego trata los diferentes niveles de mipmap. Tenga en cuenta que si usted no está marcando sprites como el uso de las páginas de textura individual, todavía se puede habilitar el mipmapping en una base de grupo de textura desde el Administrador de Grupo de Textura. Esta opción está desactivada por defecto.
  • Habilitar el control de fuentes: Si desea que el proyecto esté bajo SCM (Source Control Management) entonces necesita marcar esto. Está desactivado por defecto (ver aquí para más información sobre source control).
  • Autor: El nombre del autor del juego (o editor o estudio).
  • Última modificación del proyecto: Esta sección simplemente muestra la fecha/hora de la última vez que se modificó el proyecto (no es editable).
  • Aceptar la licencia de Spine: Aquí puedes marcar la casilla para que se te muestre la licencia de Spine Runtime y aceptarla. Esto te permitirá usar Spine sprites en tu juego (esta licencia también se mostrará la primera vez que intentes añadir un Spine sprite a tu proyecto y no hayas marcado la licencia aquí de antemano).
  • Modo de compatibilidad de colisiones: Esto le dice a GameMaker que use el sistema de colisión heredado usado antes de las versiones 2022 (que usaba valores enteros en lugar de flotantes). Lee el Modo de Compatibilidad de Colisiones para más información. Esto está desactivado por defecto y se recomienda mantenerlo desactivado para los nuevos proyectos.
  • Habilitar el comportamiento de copia en escritura para las matrices: Esto habilita el comportamiento obsoleto de Copiar en Escritura para el proyecto actualmente abierto. Lea Copiar al escribir para más información. Esta opción está desactivada por defecto y se recomienda mantenerla desactivada para los nuevos proyectos.

La sección de opciones principales también contiene un menú para la información de la plantilla.

El Navegador de Activos también tendrá otras Opciones de Juego disponibles para usted dependiendo de las plataformas de destino que están disponibles para la licencia que usted tiene y no todos ellos pueden estar disponibles o visibles.

Configuración de la plataforma

Existen las siguientes opciones de juego específicas para cada plataforma:

  • Opera GX
  • Windows
  • macOS
  • Ubuntu
  • Windows UWP
  • HTML5
  • Android
  • iOS
  • tvOS

NOTA Las diferentes opciones de juegos de consola no aparecen aquí debido a las diferentes restricciones legales que mantienen las diferentes compañías. Si tienes una licencia de consola, puedes obtener esta información del servicio de asistencia de YoYo Games, pero primero debes contactar con YoYo Games con el correo electrónico de tu ID de consola y solicitar permiso para ver las diferentes secciones.

Algunos proyectos de GameMaker también pueden mostrar opciones de juego para el objetivo de Amazon Fire. Se trata de opciones heredadas y de sólo lectura, ya que GameMaker admite ahora la plataforma Fire directamente mediante las opciones de juego de Android. Si su proyecto sigue mostrando Amazon Fire, puede encontrar información sobre las opciones en la siguiente página:
