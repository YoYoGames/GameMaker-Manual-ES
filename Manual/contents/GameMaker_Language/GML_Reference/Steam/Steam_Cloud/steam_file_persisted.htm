steam_file_persisted

Con esta función puede comprobar el archivo dado para ver si se ha sincronizado con la nube Steam. Un valor de retorno de true significa que lo está, mientras que false significa que no lo está.

Sintaxis:

steam_file_persisted(filename);

Argumento Descripción
filename El nombre del archivo a comprobar.

Devuelve:

Boolean

Ejemplo:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

El código anterior comprobará si un archivo ha sido almacenado en la nube Steam, y si no lo ha hecho lo sincronizará.
