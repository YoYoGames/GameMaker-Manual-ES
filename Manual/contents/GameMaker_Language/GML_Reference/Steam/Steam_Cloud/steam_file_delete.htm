steam_file_delete

Esta función eliminará el archivo dado de la nube Steam. La función devolverá un valor 0 si falla por cualquier motivo y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_file_delete(filename);

Argumento Descripción
filename El nombre de la eliminación del archivo.

Devuelve:

Real

Ejemplo:

if steam_file_exists("Save.txt")
{
    steam_file_delete("Save.txt");
}

El código anterior comprobará si el archivo existe, y si lo hace, lo eliminará de la nube Steam.
