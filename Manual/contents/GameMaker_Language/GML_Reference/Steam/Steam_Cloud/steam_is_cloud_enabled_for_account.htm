steam_is_cloud_enabled_for_account

Con esta función puedes comprobar que el servicio Steam Cloud está habilitado por el usuario en su configuración de Steam Client. Devolverá true si lo está y false en caso contrario. Ten en cuenta que esto no significa automáticamente que puedas almacenar datos en la Nube, ya que también tendrá que haber sido habilitado para tu juego (puedes comprobarlo usando la función steam_is_cloud_enabled_for_app()). Si la Nube Steam está habilitada para tu juego, pero el usuario la tiene desactivada localmente, puedes seguir utilizando las funciones de la Nube para almacenar y recuperar datos de una copia local de todos los archivos, sólo que no los subirá a la Nube al final del juego, ni se sincronizará al inicio del mismo.

Sintaxis:

steam_is_cloud_enabled_for_account();

Devuelve:

Boolean

Ejemplo:

if steam_is_cloud_enabled_for_account
{
    steam_file_share("Save.txt");
}

El código anterior comprueba si el usuario tiene activada la nube Steam y, si devuelve true, sincronizará el archivo dado.
