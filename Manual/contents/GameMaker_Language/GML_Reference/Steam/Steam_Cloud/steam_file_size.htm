tamaño_de_archivo_de_vapor

Con esta función puede comprobar el tamaño de un archivo almacenado en la nube Steam. El número real devuelto es el tamaño, en bytes, del archivo.

Sintaxis:

steam_file_size(filename);

Argumento Descripción
filename El nombre del archivo para comprobar el tamaño.

Devuelve:

Real

Ejemplo:

file_bytes = steam_file_size("Save.txt");

El código anterior almacena el tamaño de un archivo de la nube Steam en la variable "file_bytes".
