steam_file_write_file

Con esta función puede copiar el contenido de un archivo guardado localmente a un archivo sincronizado con la nube Steam. El archivo local debe existir antes de usar esta función, y devolverá un valor de 0 si falla por cualquier razón y un valor superior a 0 si tiene éxito.

Sintaxis:

steam_file_write_file(steam_filename, local_filename);

Argumento Descripción
steam_filename El archivo de Steam Cloud para copiar.
local_filename El archivo local que se utilizará para copiar.

Devuelve:

real

Ejemplo:

steam_file_write_file("rm_koala.png", "Koala2.png");

El código anterior copiará el contenido del archivo "Koala2.png" al archivo "rm_koala.png" de Steam Cloud.
