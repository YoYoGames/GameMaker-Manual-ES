steam_is_cloud_enabled_for_app

Con esta función puedes comprobar que el servicio Steam Cloud está habilitado para tu juego. Devolverá true si lo está y false en caso contrario. Ten en cuenta que esto no significa automáticamente que puedas utilizar las funciones de la Nube, ya que el usuario puede switch desactivar la sincronización de la Nube desde su cliente Steam. Puede comprobarlo con la función steam_is_cloud_enabled_for_account(), pero, aunque esté desactivada para el usuario (y activada para el juego), las funciones seguirán funcionando para almacenar y recuperar datos de una copia local de todos los archivos, sólo que no los subirá a la nube al finalizar el juego, ni los sincronizará al iniciarlo.

Sintaxis:

steam_is_cloud_enabled_for_app();

Devuelve:

Boolean

Ejemplo:

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_total();
}

El código anterior comprueba si la nube steam está habilitada para el juego y, si es así, obtiene el tamaño de la cuota de almacenamiento y lo almacena en una variable.
