steam_get_quota_total

Al utilizar la Nube Steam para almacenar y sincronizar archivos, debe configurar la cuota de espacio que necesitará su juego. Esta cuota se aplica a cada juego habilitado para la Nube, por usuario y por juego, así que, por ejemplo, si la cuota para el Juego X es de 1 megabyte, entonces cada cuenta de Steam que posea el Juego X podrá almacenar, como máximo, 1 megabyte de datos asociados a ese juego en la Nube. Cualquier otro juego habilitado para la Nube que posea el usuario (por ejemplo, el Juego Y) no se verá afectado por los datos almacenados por el Juego X. La cuota por defecto para los nuevos juegos de Steamworks es de un gigabyte, pero puede cambiarla desde el panel de control de Steamworks para su juego.

NOTA: Una vez que la cuota se agota , las escrituras de archivos fallarán. Si crees que es posible que la cuota se agote para el usuario de tu juego, debes crear un código para manejarlo, ya que al no hacer nada dejas a los usuarios en una situación en la que no pueden arreglar las cosas y eso llevará a una mala experiencia de tu juego.

Sintaxis:

steam_get_quota_total();

Devuelve:

Real

Ejemplo:

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_total();
}

El código anterior comprueba si la nube steam está habilitada para el juego y, si es así, obtiene el tamaño de la cuota de almacenamiento y lo almacena en una variable.
