steam_file_share

Con esta función puede forzar a su juego a sincronizar el archivo dado con la nube Steam. Esto no es normalmente necesario debido a que el juego se sincronizará automáticamente al final de la sesión del jugador, ni es recomendado por Steam, pero puede ser útil para asegurar que la información sensible se sincronice inmediatamente. La función devolverá un valor 0 si falla por cualquier razón y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_file_share(filename);

Argumento Descripción
filename El nombre de la sincronización del archivo.

Devuelve:

Real

Ejemplo:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

El código anterior comprobará si un archivo ha sido almacenado en la nube Steam, y si no lo ha hecho lo sincronizará.
