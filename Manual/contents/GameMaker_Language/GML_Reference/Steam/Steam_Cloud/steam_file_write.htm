steam_file_write

Puede utilizar esta función para escribir datos en un archivo, que luego se sincronizará con la nube Steam cuando el usuario salga del juego. si el archivo no existe, esta función lo creará por usted, y si ya existe, sobrescribirá cualquier dato que ya esté almacenado en el archivo con los nuevos datos string. La función devolverá un valor de 0 si falla por cualquier razón y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_file_write(filename, data, size);

Argumento Descripción
filename El nombre del archivo a escribir.
data Los datos a escribir (a string).
size el tamaño de los datos a escribir.

Devuelve:

Real

Ejemplo:

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

El código anterior preparará una serie de variables locales y luego las utilizará para escribir en (o crear) un archivo que luego se sincronizará con la nube Steam.
