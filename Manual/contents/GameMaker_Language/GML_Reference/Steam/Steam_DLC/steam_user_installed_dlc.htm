steam_user_installed_dlc

Si tu juego tiene DLC creado para él, puedes usar esta función para comprobar si el usuario lo ha instalado antes de acceder a cualquier archivo asociado a él. La función devuelve true si el jugador tiene el contenido instalado, y false si el usuario no lo tiene, pero tenga en cuenta que el usuario también debe poseer el DLC, por lo que debe utilizar la función adicional de steam_user_owns_dlc() para comprobar que también es de su propiedad antes de utilizarla. Tenga en cuenta que los IDs de Steam pueden ser números grandes, por lo que es posible que tenga que lanzar el valor de su ID como un int64() antes de suministrarlo a la función.

Sintaxis:

steam_user_installed_dlc(dlc_id);

Argumento Descripción
dlc_id El identificador único del DLC que se va a comprobar (un número entero).

Devuelve:

Boolean

Ejemplo:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

El código anterior comprobará si el usuario ha comprado, e instalado, el DLC con el id 10354, y si es así establecerá una variable global con un valor diferente.
