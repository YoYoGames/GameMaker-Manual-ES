steam_user_owns_dlc

Si tu juego tiene DLC creado para él, puedes utilizar esta función para comprobar si el usuario lo ha comprado antes de acceder a cualquier archivo asociado a él. La función devolverá true (1) si el jugador posee el contenido, false (0) si no lo posee o el ID de DLC dado no es válido, o -1 si no ha iniciado sesión en Steam.

Tenga en cuenta que aunque el usuario posea el DLC, puede que no lo tenga instalado, por lo que debe utilizar adicionalmente la función steam_user_installed_dlc() para asegurarse de que está instalado antes de utilizarlo. Tenga en cuenta que los IDs de Steam pueden ser números grandes, por lo que es posible que tenga que convertir el valor de su ID en un int64() antes de suministrarlo a la función.

Sintaxis:

steam_user_owns_dlc(dlc_id);

Argumento Descripción
dlc_id El identificador único del DLC que se va a comprobar (un entero int64).

Devuelve:

Integer

Ejemplo:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

El código anterior comprobará si el usuario ha comprado, e instalado, el DLC con el id 10354, y si es así establecerá una variable global con un valor diferente.
