steam_set_achievement

Con esta función puedes indicar a la API Steam que conceda ("establezca") un logro para el jugador. Estos logros deben haber sido definidos en la página de cuentas del panel de control de Steamworks para su juego y el string que se pasa a la función debe coincidir con el utilizado como nombre de la API en el panel de control. La superposición de juego Steam mostrará un panel de notificación al usuario informándole del logro que ha recibido, a menos que el logro ya haya sido concedido, en cuyo caso no ocurrirá nada.

Sintaxis:

steam_set_achievement(ach_name);

Argumento Descripción
ach_name El nombre del logro a establecer (string).

Devuelve:

N/A

Ejemplo:

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

El código anterior premiará al jugador con un logro si la variable global "Muertes" es igual a 10 y si el logro no ha sido ya concedido.
