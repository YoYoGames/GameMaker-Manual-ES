steam_set_stat_int

Con esta función puedes establecer una estadística específica a un nuevo valor entero con signo. La estadística debe haber sido definida previamente en la página de cuentas del panel de control de Steamworks para tu juego y el string que se pasa a la función debe coincidir con el utilizado como nombre de la API en el panel de control. Ejemplos de cuando podrías usar esto son para rastrear cuántas veces muere el jugador o para rastrear el progreso hacia un logro.

Sintaxis:

steam_set_stat_int(stat_name, value);

Argumento Descripción
stat_name El nombre de la estadística a establecer (un string).
value El valor (un entero de 32 bits) para establecer la estadística.

Devuelve:

N/A

Ejemplo:

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

El código anterior establece una estadística y luego comprueba el valor final de la misma para decidir si se concede un logro o no.
