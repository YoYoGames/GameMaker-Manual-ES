steam_get_stat_int

Con esta función puedes obtener el valor de una estadística entera con signo específica. La estadística debe haber sido definida previamente en la página de cuentas del panel de control de Steamworks para tu juego y el string que se pasa a la función debe coincidir con el utilizado como Nombre de la API en el panel de control.

Sintaxis:

steam_get_stat_int(stat_name);

Argumento Descripción
stat_name El nombre de la estadística a obtener (un string).

Devuelve:

Real

Ejemplo:

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

El código anterior establece una estadística y luego comprueba el valor final de la misma para decidir si se concede un logro o no.
