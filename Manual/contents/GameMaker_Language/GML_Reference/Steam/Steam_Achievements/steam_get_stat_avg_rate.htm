steam_get_stat_avg_rate

Con esta función puedes obtener el valor de una estadística media específica. La estadística debe haber sido definida previamente en la página de cuentas del panel de control de Steamworks para tu juego y el string que se pasa a la función debe coincidir con con el utilizado como nombre de la API en el panel de control.

Sintaxis:

steam_get_stat_avg_rate(stat_name);

Argumento Descripción
stat_name El nombre de la estadística a obtener (un string).

Devuelve:

Real

Ejemplo:

var avg = steam_get_stat_avg_rate("PointsPerHour");
draw_text(8, 8, "PPH = " + string(avg);

El código anterior obtiene el valor actual de la estadística media "PointsPerHour" y lo dibuja en la pantalla.
