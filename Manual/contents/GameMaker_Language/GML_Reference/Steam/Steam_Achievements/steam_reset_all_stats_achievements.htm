steam_reset_all_stats_achievements

Con esta función puedes restablecer todas las estadísticas y logros del usuario actual a sus valores por defecto (tal y como están definidos en el panel de control de Steamworks para tu juego). Se recomienda utilizar esta función únicamente como herramienta de depuración cuando desarrolles tu juego.

Sintaxis:

steam_reset_all_stats_achievements();

Devuelve:

N/A

Ejemplo:

ini_open("Save.ini");
if global.Version != ini_read_real("Data", "Version", 0)
{
    ini_write_real("Data", "Version", global.Version);
    steam_reset_all_stats_achievements();
}
ini_close();

El código anterior comprueba un valor almacenado en un archivo ini contra el de una variable global y si son diferentes, restablece las estadísticas y los logros del juego.
