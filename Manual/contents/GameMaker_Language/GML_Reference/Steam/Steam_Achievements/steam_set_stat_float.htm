steam_set_stat_float

Con esta función puedes establecer una estadística específica a un nuevo valor de punto flotante. La estadística debe haber sido definida previamente en la página de cuentas del panel de control de Steamworks para tu juego y el string que se pasa a la función debe coincidir con el utilizado como nombre de la API en el panel de control. Ejemplos de cuando podrías usar esto son para rastrear qué tan lejos ha viajado tu jugador, o qué porcentaje del juego está completo.

Sintaxis:

steam_set_stat_float(stat_name, value);

Argumento Descripción
stat_name El nombre de la estadística a establecer (un string).
value El valor (un float de 32 bits) para establecer la estadística.

Devuelve:

N/A

Ejemplo:

var dist_pc = (dist / dist_max) * 100;
steam_set_stat_float("Travelled", dist_pc);

El código anterior calcula un porcentaje basado en la variable de distancia recorrida "dist" y la distancia máxima que puedes recorrer "dist_max" y luego establece la estadística "Travelled" al nuevo valor.
