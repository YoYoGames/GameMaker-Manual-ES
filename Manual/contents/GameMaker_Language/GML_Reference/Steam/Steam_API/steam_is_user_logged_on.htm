steam_is_user_logged_on

Esta función devolverá true si el cliente Steam tiene actualmente una conexión activa con el Steam servers. Si devuelve false, significa que no hay una conexión activa debido a un problema de red en la máquina local, o a que Steam server está caído u ocupado.

Sintaxis:

steam_is_user_logged_on();

Devuelve:

String

Ejemplo:

if steam_is_user_logged_on()
{
    global.user_id = steam_get_user_account_id();
}

El código anterior comprobará si el usuario está conectado a la Steam server y si almacena el ID de usuario en una variable global.
