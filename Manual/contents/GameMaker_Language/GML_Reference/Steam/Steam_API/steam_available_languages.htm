steam_available_languages

Esta función puede utilizarse para obtener una lista de todos los idiomas disponibles para Steam. El valor devuelto es simplemente una lista de idiomas separada por comas.

Sintaxis:

steam_available_languages();

Devuelve:

String

Ejemplo:

language = steam_available_languages();

Lo anterior obtiene los idiomas disponibles para Steam como string y lo almacena en una variable.
