steam_current_game_language

Esta función se utiliza para recuperar el idioma actual que está utilizando Steam (como string), por ejemplo "english".

Sintaxis:

steam_current_game_language();

Devuelve:

String

Ejemplo:

language = steam_current_game_language();

El código anterior obtiene el idioma utilizado para el juego actual.
