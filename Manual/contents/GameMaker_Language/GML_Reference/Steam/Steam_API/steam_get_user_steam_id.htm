steam_get_user_steam_id

Puede utilizar esta función para devolver el identificador de usuario único Steam del usuario actualmente conectado al cliente Steam. Se trata de un valor único de 64 bits que puede utilizarse para identificar a los usuarios individuales y no su nombre de usuario en pantalla (éste puede encontrarse utilizando la función steam_get_persona_name()). Como se trata de un int64, es posible que tenga que convertirlo en tal cuando lo utilice en funciones posteriores usando la función int64() función.

Sintaxis:

steam_get_user_steam_id();

Devuelve:

Integer (int64)

Ejemplo:

if steam_initialised()
{
    global.u_id = steam_get_user_steam_id();
}

El código anterior establecerá una variable global con el ID único del usuario actual Steam si la API del cliente Steam está correctamente inicializada.
