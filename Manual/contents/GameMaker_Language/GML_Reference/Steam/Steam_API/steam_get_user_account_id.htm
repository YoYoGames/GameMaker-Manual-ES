steam_get_user_account_id

Esta función se utiliza para recuperar el ID de usuario único que Steam asigna a cada usuario, que es necesario para utilizar algunas de las funciones de contenido generado por el usuario.

Sintaxis:

steam_get_user_account_id();

Devuelve:

Real

Ejemplo:

global.user_id = steam_get_user_account_id();

El código anterior obtiene el ID de usuario único de la persona que posee el juego y lo almacena en una variable global.
