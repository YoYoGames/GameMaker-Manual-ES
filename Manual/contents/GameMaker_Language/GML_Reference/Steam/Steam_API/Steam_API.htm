General Steam API

El siguiente conjunto de funciones son todas para comprobar la disponibilidad de ciertos aspectos del cliente Steam o de la API server, y como tal devuelven true o false dependiendo de factores como que server esté ocupado o que el usuario esté conectado, etc. Esto significa que estas funciones deben ser utilizadas antes de cualquier otra llamada a la función de la API Steam para asegurar que la configuración del cliente/servidor es correcta y se comunica con tu juego:

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
