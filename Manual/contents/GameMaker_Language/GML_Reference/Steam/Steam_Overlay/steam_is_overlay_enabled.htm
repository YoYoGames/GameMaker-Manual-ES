steam_is_overlay_enabled

Cuando se utiliza la API Steam, se puede llamar a esta función para comprobar que la API del cliente Steam tiene activada la funcionalidad de superposición.

Sintaxis:

steam_is_overlay_enabled();

Devuelve:

Boolean

Ejemplo:

global.steam_api = false;
if steam_initialised()
{
    if steam_stats_ready() && steam_is_overlay_enabled()
    {
        global.steamapi = true;
    }
}

El código anterior establecerá una variable global como verdadera si la API del cliente Steam está correctamente inicializada, junto con las estadísticas y la funcionalidad de superposición de Steam, o establecerá la variable como falsa en caso contrario.
