steam_activate_overlay_user

Esta función abrirá la ventana superpuesta Steam a uno de los diálogos elegidos relacionados con el ID de usuario dado. Los posibles diálogos strings disponibles son:

  • "steamid" - abre el navegador web de la Comunidad Steam a la página del usuario o grupo
  • "chat" - abre una ventana de chat al usuario especificado

Tenga en cuenta que los ID de Steam pueden ser números grandes, por lo que es posible que tenga que convertir el valor de su ID en un int64() antes de suministrarlo a la función.

Sintaxis:

steam_activate_overlay_user(dialog_name, steamid);

Argumento Descripción
dialog_name El diálogo en el que se abre la superposición (véase más abajo).
steamid El ID de usuario o el ID de grupo de Steam que se va a utilizar.

Devuelve:

N/A

Ejemplo:

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

El código anterior sondea la última tecla del teclado presionada y si es la tecla de función 1 o la tecla de función 2, abrirá el overlay Steam para ver el grupo Steam almacenado en la variable global "GamegroupID", o abrirá la ventana de chat para chatear con el usuario almacenado en la variable global "FriendID".
