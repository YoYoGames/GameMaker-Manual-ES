La superposición de Steam es la pantalla visual que puede aparecer para mostrar la interfaz de Steam al usuario. Esto lo hace normalmente el propio usuario mediante una combinación de teclas, pero también tienes la posibilidad de hacerlo desde tu juego, de manera que puedes mapear un botón o un evento para mostrar la superposición, utilizando las siguientes funciones:

  • steam_is_overlay_enabled
  • steam_is_overlay_activated
  • steam_activate_overlay
  • steam_activate_overlay_browser
  • steam_activate_overlay_store
  • steam_activate_overlay_user
