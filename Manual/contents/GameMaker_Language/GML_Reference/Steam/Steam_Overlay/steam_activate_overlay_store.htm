steam_activate_overlay_store

Con esta función puedes abrir el overlay Steam en la página de la tienda de un juego para que los usuarios puedan comprar o descargar DLC (por ejemplo). Tienes que proporcionar el identificador único de la aplicación del juego o del DLC que obtendrás en el panel de control de Steam cuando lo configures.

Sintaxis:

steam_activate_overlay_store(app_id);

Argumento Descripción
app_id El identificador único de la aplicación para su juego.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_store(global.DLC_id);
}

El código anterior sondea el teclado en busca de la tecla F1 y si es así, la superposición Steam se abrirá en la página para el contenido del juego utilizando el ID de la aplicación almacenado en la variable global.
