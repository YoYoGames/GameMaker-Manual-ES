steam_activate_overlay_browser

Con esta función puedes abrir la superposición del juego Steam a su navegador web y luego hacer que cargue la URL especificada. necesitas usar la URL completa como string para que esto se resuelva correctamente, por ejemplo: "http://www.steamgames.com/".

Sintaxis:

steam_activate_overlay(url);

Argumento Descripción
url La URL (completa) de la superposición para abrir.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_browser("http://www.steamgames.com/");
}

El código anterior sondea el teclado en busca de la tecla F1 y si es así, se abrirá la superposición Steam y se resolverá la URL dada.
