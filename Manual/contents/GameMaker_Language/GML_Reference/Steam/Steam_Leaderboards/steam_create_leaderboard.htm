steam_create_leaderboard

Con esta función puedes crear una nueva tabla de clasificación para tu juego. El primer argumento es un string que define el nombre de su tabla de clasificación, y este nombre debe ser utilizado en cualquier otra llamada a la función relacionada con la tabla de clasificación que se está creando. A continuación, puede definir el método de clasificación (ver las constantes listadas abajo) así como la forma en que se muestra la información (de nuevo, una de las constantes listadas abajo). La función devolverá un valor de -1 si no consigue crear la tabla de clasificación o un valor mayor que -1 si ha tenido éxito.

NOTA: Si has creado previamente una tabla de clasificación con el mismo nombre (ya sea a través de código o a través de tu página Steam para el juego) ¡esta función será ignorada!

Ordenar Constante Descripción
lb_sort_none No hay clasificación. La información se mostrará "tal cual".
lb_sort_ascending Ordena la tabla de posiciones en orden ascendente.
lb_sort_descending Ordena la tabla de posiciones en orden descendente.

Visualización de la constante Descripción
lb_disp_none Mostrar la tabla de clasificación "tal cual".
lb_disp_numeric Mostrar la tabla de clasificación como una pantalla numérica.
lb_disp_time_sec Mostrar los valores de la tabla de clasificación como tiempos, siendo el valor base los segundos.
lb_disp_time_ms Mostrar los valores de la tabla de clasificación como tiempos, con el valor base de milisegundos.

Sintaxis:

steam_create_leaderboard(lb_name, sort_method, display_type);

Argumento Descripción
lb_name El nombre de la tabla de clasificación que está creando (un string).
sort_method El método para ordenar las entradas de la tabla de clasificación (ver las constantes listadas abajo).
display_type La forma de mostrar la tabla de posiciones al usuario (ver las constantes que se indican a continuación).

Devuelve:

Real

Ejemplo:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

El código anterior creará una tabla de clasificación llamada "Tiempos de Juego", y la configurará para mostrar los resultados en orden ascendente y con una indicación en segundos.
