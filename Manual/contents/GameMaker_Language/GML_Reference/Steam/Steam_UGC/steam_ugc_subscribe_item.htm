steam_ugc_subscribe_item

Esta función puede utilizarse para suscribirse a un elemento UGC. Se suministra el ID del archivo publicado y luego la función devuelve un valor de ID asíncrono único que puede utilizarse para comprobar cuándo se ha descargado el elemento suscrito en el evento Steam Async. Este evento tendrá el mapa async_load mapa DS que se rellenará con los siguientes pares clave/valor:

  • "id" - El ID asíncrono devuelto por la función que desencadenó el evento
  • "resultado" - El resultado de la operación (un valor real). Será la constante GML ugc_result_success o algún otro número real. Por lo tanto, debe comprobar esta constante para asegurarse de que la llamada ha tenido éxito, y si de lo contrario algo no ha funcionado correctamente. El resto de los posibles valores devueltos se muestran como resultado del valor "EResult" de Steam y deberías ver steamclientpublic.h en las cabeceras del SDK para los 89 valores posibles.
  • "event_type" - El string "ugc_subscribe_item".
  • "published_file_id" - El ID único del elemento publicado.

Sintaxis:

steam_ugc_subscribe_item(published_file_id);

Argumento Descripción
published_file_id El ID de archivo único para el UGC a suscribir.

Devuelve:

Async ID

Ejemplo:

steam_sub = steam_ugc_subscribe_item(global.pubFileID);

El código anterior suscribirá (y descargará) el elemento con el ID de archivo almacenado en la variable global "pubFileID".
