steam_ugc_get_item_update_progress

Esta función puede utilizarse para seguir el estado de actualización de un elemento. Se le da el manejador del elemento (devuelto por la función steam_ugc_start_item_update()) y un mapa DS vacío que se rellenará con los siguientes pares clave/valor:

  • "status_code" - El código de estado de Steam
  • "status_string" - Un string para el estado actual
  • "bytes_processed" - Los bytes procesados hasta el momento
  • "bytes_total" - El número total de bytes en la actualización

Si hay un error la función devolverá false y el mapa estará vacío, de lo contrario la función devuelve true.

Sintaxis:

steam_ugc_get_item_update_progress(ugc_update_handle, info_map);

Argumento Descripción
ugc_update_handle El identificador único para el UGC que se va a actualizar.
info_map Un índice de mapa DS (creado previamente).

Devuelve:

Boolean

Ejemplo:

var uploadMap = ds_map_create();
steam_ugc_get_item_update_progress(global.itemHandle, uploadMap);
var statusCode = uploadMap[? "status_code"];
var status = uploadMap[? "status_string"];
var processed = uploadMap[? "bytes_processed"];
var total = uploadMap[? "bytes_total"];
draw_text(32, 0, "Upload info for item:" + string(global.itemHandle));
draw_text(32, 15, "status code:" + string(statusCode));
draw_text(32, 30, "status:" + string(status));
draw_text(32, 45, "bytes processed:" +string(processed));
draw_text(32, 60, "bytes total:" + string( total));
ds_map_destroy(uploadMap);

El código anterior consultará el estado de carga del elemento indexado en la variable global "itemHandle", utilizando un mapa DS para almacenar la información. A continuación, se analiza y los valores resultantes se dibujan en la pantalla.
