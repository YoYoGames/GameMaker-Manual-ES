steam_ugc_get_subscribed_items

Esta función rellenará una lista DS con todos los IDs de archivos publicados para los artículos a los que el usuario está suscrito actualmente. Primero debe crear la lista y almacenar el índice en una variable, luego pasarla a la función. La función devolverá true si todo es correcto y la API Steam está inicializada, o false si hay un error.

Sintaxis:

steam_ugc_get_subscribed_items(item_list);

Argumento Descripción
item_list Un índice de lista DS (creado previamente).

Devuelve:

Boolean

Ejemplo:

steam_list = ds_list_create();
steam_ugc_get_subscribed_items(steam_list);

El código anterior creará una lista DS vacía y luego la rellenará con los IDs de los archivos de todos los elementos suscritos por el usuario.
