steam_ugc_query_add_excluded_tag

Esta función puede utilizarse para filtrar aún más cualquier consulta de CGU, concretamente para excluir un CGU determinado de la solicitud de consulta. El manejador de la consulta es el valor devuelto cuando se creó la consulta (usando, por ejemplo, steam_ugc_create_query_user()) y el segundo argumento es un string que desea utilizar como etiqueta a excluir. La función devolverá true si el filtro de consulta se ha establecido correctamente, o false en caso contrario.

Sintaxis:

steam_ugc_query_add_excluded_tag(ugc_query_handle , tag_name);

Argumento Descripción
ugc_query_handle El mango de la consulta que se va a utilizar.
tag_name El nombre de la etiqueta a excluir (a string).

Devuelve:

Boolean

Ejemplo:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_add_excluded_tag(query_handle, "walking simulator");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

El código anterior crea una solicitud de consulta y almacena su manejador en una variable local para su uso futuro en el resto de las funciones que definen aún más la solicitud de consulta antes de enviarla.
