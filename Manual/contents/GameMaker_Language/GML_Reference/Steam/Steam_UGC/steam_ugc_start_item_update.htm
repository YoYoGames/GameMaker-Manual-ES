steam_ugc_start_item_update

Esta función debe llamarse antes de añadir o actualizar la información de un elemento UGC. Debe proporcionar el ID de aplicación único de su juego en Steam, junto con el ID de archivo publicado único que se devolvió para el elemento cuando lo creó utilizando la función steam_ugc_create_item(). La función devolverá un manejador de actualización único para el elemento, que luego puede utilizar en las funciones del elemento UGC para actualizar (o añadir) información para la carga.

Sintaxis:

steam_ugc_start_item_update(consumer_app_id, published_file_id);

Argumento Descripción
consumer_app_id El identificador único de la aplicación para su juego en Steam.
published_file_id El valor único del ID del archivo publicado para el artículo.

Devuelve:

Handle

Ejemplo:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

El código anterior obtiene el ID del juego y lo utiliza junto con el ID del archivo publicado previamente para generar un controlador de actualización para el elemento. Este manejador se utiliza entonces para actualizar varias piezas de información antes de que la actualización sea enviada al Taller servers .
