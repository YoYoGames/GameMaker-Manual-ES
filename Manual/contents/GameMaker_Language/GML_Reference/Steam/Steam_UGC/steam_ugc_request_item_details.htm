steam_ugc_request_item_details

Esta función puede utilizarse para recuperar información sobre un ID de archivo determinado. Se da el ID del archivo y se suministra una edad máxima para la comprobación (ver los documentos de Steam para más información), entonces la función devuelve un valor único de ID asíncrono que puede ser utilizado para comprobar los detalles correctamente en el evento Steam Async. Este evento tendrá el async_load mapa DS que se rellenará con los siguientes pares clave/valor:

  • "id" - El ID asíncrono devuelto por la función que desencadenó el evento
  • "resultado" - El resultado de la operación (un valor real). Será la constante GML ugc_result_success o algún otro número real. Por lo tanto, debe comprobar esta constante para asegurarse de que la llamada ha tenido éxito, y si por el contrario algo no ha funcionado correctamente. El resto de los posibles valores devueltos se muestran como resultado del valor "EResult" de Steam y deberías ver steamclientpublic.h en las cabeceras del SDK para los 89 valores posibles.
  • "event_type" - El string "ugc_item_details"
  • "cached_data" - Será true si los detalles devueltos provienen del local cache o false si se toman del servidor
  • "published_file_id"" -Contiene el identificador único de archivo publicado para el elemento
  • "file_type" - El tipo de archivo utilizado
  • "creator_app_id" - El ID de Steam del creador del artículo
  • "consumer_app_id" - El ID de Steam del consumidor del artículo
  • "título" - El título del artículo
  • "descripción" - La descripción del artículo
  • "steam_id_owner" - El ID de Steam del propietario del artículo
  • "time_created" - La hora en que se creó el elemento por primera vez
  • "time_updated" - La última vez que se actualizó el elemento
  • "time_added_to_user_list" - La hora a la que se suscribió el elemento
  • "visibilidad" - La visibilidad del elemento (véase steam_ugc_set_item_visibility() para las constantes devueltas)
  • "prohibido" - Si el artículo ha sido prohibido(verdadero) o no(falso)
  • "accepted_for_use" - Si el elemento ha sido aceptado para su uso(true) o no(false)
  • "tags_truncated" - Versión abreviada de las etiquetas en forma de matriz
  • "tags" - Una matriz de las etiquetas del artículo
  • "handle_file" - El único manejador de archivo para el elemento
  • "handle_preview_file" - El único manejador de la imagen de vista previa del artículo (puede usarse como argumento con steam_ugc_download() para descargar una imagen de vista previa)
  • "filename" - El nombre del archivo del artículo
  • "file_size" - El tamaño del archivo del artículo
  • "preview_file_size" - El tamaño de la imagen de vista previa
  • "url" - La URL completa del artículo
  • "votes_up" - El número de votos positivos recibidos
  • "votes_down" - El número de votos negativos recibidos
  • "puntuación" - La puntuación global del elemento
  • "account_id_owner" - El ID de la cuenta del propietario del ID de Steam (se puede utilizar en la función steam_ugc_create_query_user_ex())

Sintaxis:

steam_ugc_request_item_details(published_file_id, max_age_seconds);

Argumento Descripción
published_file_id El ID de archivo único para el UGC que se va a comprobar.
max_age_seconds La "edad" de los datos a comprobar (se recomienda entre 30 y 60 segundos).

Devuelve:

Async ID

Ejemplo ampliado:

En este ejemplo enviamos una solicitud de detalles para un artículo y luego analizamos el mapa async_load DS resultante para establecer algunas variables. Primero enviamos la solicitud:

steam_details = steam_ugc_request_item_details(global.fileID, 60);

El código anterior solicitará detalles sobre el elemento con el ID del archivo almacenado en la variable global y lanzará un evento Steam Async con la información devuelta. En este evento podemos entonces analizar el mapa y almacenar algunos de los valores en variables que luego pueden ser utilizadas para mostrar la información al usuario:

var map_id = async_load[? "id"];
var result = async_load[? "result"];
if (map_id == steam_details) && (result == ugc_result_success)
{
    mTitle = async_load[? "title"];
    mDesc = async_load[? "description"];
    mTags = async_load[? "tags"];
    m_hPreviewFile = async_load[? "handle_preview_file"];
    m_hOwnerSteamId = async_load[? "steam_id_owner"];
    mOwnerAccountId = async_load[? "account_id_owner"];
    mPubFileId = async_load[? "published_file_id"];
    mScore = async_load[? "score"];
}
