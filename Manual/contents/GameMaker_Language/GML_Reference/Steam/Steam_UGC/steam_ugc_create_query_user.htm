steam_ugc_create_query_user

Esta función puede utilizarse para consultar la base de datos de UGC. La función utiliza automáticamente el ID por defecto de la aplicación, el usuario y asume que la consulta la realiza el consumidor (en lugar del creador). La función requiere que se utilicen las siguientes constantes para el tipo de datos a consultar, el tipo de elemento a consultar y el orden en que se clasificarán los elementos devueltos, así como un número de página - tenga en cuenta que una consulta devolverá un número máximo de 50 elementos.

ugc_list_Publicado Consulte la documentación de Steam.
ugc_list_VotedOn -
ugc_list_VotedUp -
ugc_list_VotedDown -
ugc_list_WillVoteLater -
ugc_list_Favorited -
ugc_list_Subscribed -
ugc_list_UsedOrPlayed -
ugc_list_Followed -

Ordenación constante Descripción

La función devuelve un valor único de manejo de la consulta que debe ser almacenado en una variable para su uso en las otras funciones de consulta. Tenga en cuenta que esta función sólo prepara la consulta pero no la envía realmente - para ello debe llamar a la función steam_ugc_send_query() - y puede utilizar otras funciones steam_ugc_query_*() para refinar la solicitud de búsqueda antes de enviarla.

Sintaxis:

steam_ugc_create_query_user(list_type, match_type, sort_order, page);

Argumento Descripción
list_type El tipo de lista de datos que se va a crear (ver las constantes listadas a continuación).
match_type El tipo de elementos UGC a consultar (ver las constantes listadas a continuación).
sort_order La forma en que los datos deben ser ordenados (ver las constantes listadas abajo).
page El número de página a consultar.

Devuelve:

Query Handle

Ejemplo:

query_handle = steam_ugc_create_query_user(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, 1);

El código anterior crea una solicitud de consulta y almacena su manejo en una variable para su uso futuro.
