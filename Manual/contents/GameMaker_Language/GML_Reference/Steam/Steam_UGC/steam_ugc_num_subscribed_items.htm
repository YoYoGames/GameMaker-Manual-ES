steam_ugc_num_subscribed_items

Esta función puede utilizarse para obtener el número de artículos a los que se ha suscrito el usuario actual.

Sintaxis:

steam_ugc_num_subscribed_items();

Devuelve:

Real

Ejemplo:

numSub = steam_ugc_num_subscribed_items();

El código anterior almacenará el número de elementos suscritos en una variable.
