steam_ugc_submit_item_update

Esta función enviará el elemento UGC inexed por el handle dado al Steam Workshop servers, añadiendo las notas de cambio que se utilizarán para el elemento dado. Las notas de cambio deben ser un string y el manejador de actualización es el valor devuelto cuando se llamó a la función steam_ugc_start_item_update(). Cuando se utiliza esta función devolverá un valor de ID asíncrono que puede ser analizado cuando el evento asíncrono de Steam se dispara para informar de la creación del elemento. El evento contendrá los siguientes valores de clave/mapa en el async_load mapa DS:

  • "id" - El ID asíncrono devuelto por la función de llamada
  • "resultado" - El resultado de la operación (un valor real). Será la constante GML ugc_result_success o algún otro número real. Por lo tanto, debe comprobar esta constante para asegurarse de que la llamada ha tenido éxito, y si de lo contrario algo no ha funcionado correctamente. El resto de los posibles valores devueltos se muestran como resultado del valor "EResult" de Steam y deberías ver steamclientpublic.h en las cabeceras del SDK para los 89 valores posibles.
  • "event_type" - Esta clave tendrá el valor "ugc_update_item"
  • "legal_agreement_required" - Será true o false (consulte los documentos de Steam para más detalles)

Sintaxis:

steam_ugc_submit_item_update(ugc_update_handle, change_note);

Argumento Descripción
ugc_update_handle El identificador único para el UGC que se va a actualizar.
change_note Las notas de cambio (a string) que se utilizarán para el artículo.

Devuelve:

Async ID

Ejemplo:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

El código anterior obtiene el ID del juego y lo utiliza junto con un ID de archivo publicado previamente para generar un controlador de actualización para el elemento. Este manejador se utiliza entonces para actualizar varias piezas de información antes de que la actualización sea enviada al Taller servers .
