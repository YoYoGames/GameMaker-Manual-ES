steam_send_screenshot

Con esta función puedes subir una captura de pantalla a la página de perfil de la Comunidad Steam del usuario que ha iniciado la sesión. El nombre del archivo que proporcione es el nombre del archivo local que se creó cuando tomó la captura de pantalla utilizando las funciones GameMaker Studio 2 screen_save() o screen_save_part(). La anchura y la altura definen el tamaño de la imagen, y la función devolverá un valor de 0 si falla por cualquier razón y un valor mayor que 0 si tiene éxito.

Sintaxis:

steam_send_screenshot(filename, width, height);

Argumento Descripción
filename El nombre del archivo de imagen a cargar (un string).
width El ancho de la imagen (un entero).
height La altura de la imagen (un entero).

Devuelve:

Real

Ejemplo:

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

El código anterior sondeará la API Steam en busca de una solicitud de captura de pantalla y, si se ha producido, se generará un nombre único para el archivo de imagen, se tomará una captura de pantalla y se enviará el archivo a la página de la Comunidad Steam para el usuario.
