steam_is_screenshot_requested

Esta función sondea la API Steam para ver si la tecla para tomar una captura de pantalla del juego ha sido presionada. La función sólo devolverá true durante un paso (tick del juego) cuando se pulse la tecla, y devolverá false en todos los demás momentos.

Tenga en cuenta que si se pulsa la tecla de captura de pantalla, esta función sólo devolverá true una vez por cada paso que se pulse, y devolverá false para cualquier llamada posterior dentro del mismo paso. Por ejemplo, si se solicita una captura de pantalla en el fotograma actual y se llama a esta función en el evento Step para averiguarlo, se obtendrá true; sin embargo, si se llama de nuevo en Draw GUI para comprobar si se ha solicitado una captura de pantalla, se obtendrá false ya que la función ya había sido "utilizada" en el evento Step. Para utilizar el valor de retorno de la función varias veces dentro del mismo marco, se recomienda almacenarlo en una variable y leerla en lugar de llamar de nuevo a la función.

NOTA: Esta función no toma una captura de pantalla por usted. Sólo señala que se ha pulsado la tecla y debe utilizar las funciones GameMaker Studio 2 screen_save() o screen_save_part() para guardar una copia local del archivo a cargar.

Sintaxis:

steam_is_screenshot_requested();

Devuelve:

Boolean

Ejemplo:

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

El código anterior sondeará la API Steam en busca de una solicitud de captura de pantalla y, si se ha producido, se generará un nombre único para el archivo de imagen, se tomará una captura de pantalla y se enviará el archivo a la página de la Comunidad Steam para el usuario.
