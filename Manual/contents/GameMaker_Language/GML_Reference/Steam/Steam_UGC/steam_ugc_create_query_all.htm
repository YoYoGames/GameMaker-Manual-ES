steam_ugc_create_query_all

Esta función puede utilizarse para consultar la base de datos UGC utilizando algunos tipos de consulta predefinidos. La función requiere las siguientes constantes para el tipo de consulta a crear, el tipo de elemento a consultar y el número de página a consultar - tenga en cuenta que una consulta devolverá un número máximo de 50 elementos.

Tipo de consulta Constante Descripción
ugc_query_RankedByVote Consulte la documentación de Steam.
ugc_query_RankedByPublicationDate -
ugc_query_AcceptedForGameRankedByAcceptanceDate -
ugc_query_RankedByTrend -
ugc_query_FavoritedByFriendsRankedByPublicationDate -
ugc_query_CreatedByFriendsRankedByPublicationDate -
ugc_query_RankedByNumTimesReported -
ugc_query_CreatedByFollowedUsersRankedByPublicationDate -
ugc_query_NotYetRated -
ugc_query_RankedByTotalVotesAsc -
ugc_query_RankedByVotesUp -
ugc_query_RankedByTextSearch -

Tipo de partido Constante Descripción
ugc_match_Items Consulte la documentación de Steam.
ugc_match_Items_Mtx -
ugc_match_Items_ReadyToUse -
ugc_match_Collections -
ugc_match_Artwork -
ugc_match_Videos -
ugc_match_Screenshots -
ugc_match_AllGuides -
ugc_match_WebGuides -
ugc_match_IntegratedGuides -
ugc_match_UsableInGame -
ugc_match_ControllerBindings -



La función devuelve un valor único de manejo de la consulta que debe ser almacenado en una variable para su uso en las otras funciones de consulta. Tenga en cuenta que esta función sólo prepara la consulta pero no la envía realmente - para ello debe llamar a la función steam_ugc_send_query() - y puede utilizar otras funciones steam_ugc_query_*() para refinar la solicitud de búsqueda antes de enviarla.

Sintaxis:

steam_ugc_create_query_all(query_type, match_type, page);

Argumento Descripción
query_type El tipo de consulta que se va a crear (ver las constantes que aparecen a continuación).
match_type El tipo de elementos UGC a consultar (ver las constantes listadas a continuación).
page El número de página a consultar.

Devuelve:

Query Handle

Ejemplo:

query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);

El código anterior crea una solicitud de consulta general y almacena su manejo en una variable para su uso futuro.
