display_get_gui_height

Con esta función puede obtener la altura (en píxeles) de la GUI tal y como se utiliza en el evento Draw GUI.

Sintaxis:

display_get_gui_height();

Devuelve:

Real

Ejemplo:

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

El código anterior establecerá un anuncio que se mostrará en la esquina inferior izquierda de la pantalla.
