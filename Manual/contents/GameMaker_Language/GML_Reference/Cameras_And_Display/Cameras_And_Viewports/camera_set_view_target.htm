camera_set_view_target

Puede utilizar esta función para establecer el objetivo de seguimiento de la cámara de visualización dentro de la página web room. Debe dar el valor de ID de cámara único (devuelto por las diferentes funciones camera_create() funciones) y, a continuación, la instancia de destino o el ID de object en el que desea establecer la vista de la cámara. Tenga en cuenta que si elige un ID de object y hay más de una instancia de ese object en el room, no hay forma de que GameMaker sepa qué instancia desea seguir, por lo que podría ser cualquiera de ellas.

Sintaxis:

camera_set_view_target(camera_id, instance_id/object_id)

Argumento Tipo Descripción
camera_id Identificación de la cámara The unique camera ID value returned when you created the camera
instance_id/object_id Activo del objeto o ID de la instancia Instance or object to have the camera target for following

Devuelve:

N/A

Ejemplo:

camera_set_view_target(view_camera[0], id);

El código anterior establecerá el objetivo de la cámara de vista para la cámara asignada al puerto de vista[0] a la de la instancia que ejecuta el código.
