camera_get_view_speed_y

Esta función se puede utilizar para recuperar la velocidad de movimiento de la cámara dada a lo largo del eje y (movimiento vertical). El valor devuelto estará en píxeles por fotograma del juego.

Sintaxis:

camera_get_view_speed_y(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
    camera_set_view_speed(view_camera[0], 5, 5);
}

El código anterior recupera la velocidad x y la velocidad y de la cámara asignada al puerto de vista[0] y luego comprueba si coincide con el valor dado. Si no es así, la cámara de la vista se ajusta a esa velocidad.
