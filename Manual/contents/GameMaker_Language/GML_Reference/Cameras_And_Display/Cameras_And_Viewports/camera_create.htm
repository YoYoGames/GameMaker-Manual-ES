camera_create

Esta función creará una cámara "vacía", es decir, una cámara sin ajustes de vista o proyección definidos. Una vez creada la cámara puede utilizar las diferentes funciones de matriz para establecer las matrices de vista y proyección. También puede utilizar esta función para crear la cámara y luego utilizar las diferentes funciones de camera_set_* para configurar la vista, sin embargo tenga en cuenta que debe proporcionar al menos una posición y un tamaño de vista, de lo contrario la cámara resultante no será válida y dará resultados extraños si se aplica a un puerto de vista (si no establece el object siguiente, los bordes o la velocidad de movimiento estos se establecerán en los valores por defecto para una cámara de vista en el Editor Room ).

Tenga en cuenta que las cámaras son como todos los recursos dinámicos y ocuparán memoria cuando se creen. Esto significa que debes tener cuidado de no crear cámaras innecesariamente y de destruir las que no necesites llamando a la función camera_destroy(). Esto también significa que las cámaras persistirán de room a room si se asignan a una variable global (también persistirán cuando se asignen a variables de instancia o locales, pero no podrás acceder a ellas después de cambiar room, de ahí la pérdida de memoria), de modo que puedes -por ejemplo- crear una nueva cámara, asignarla a una variable global, y luego al comienzo de cada room establecer las matrices de vista y proyección necesarias y el puerto de vista que necesitas utilizar.

Sintaxis:

camera_create();

Devuelve:

Camera ID

Ejemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

El código anterior crea una nueva cámara y asigna su ID a view[0]. A continuación, se crean las matrices de vista y proyección y se almacenan en variables locales antes de utilizarlas para configurar la cámara.
