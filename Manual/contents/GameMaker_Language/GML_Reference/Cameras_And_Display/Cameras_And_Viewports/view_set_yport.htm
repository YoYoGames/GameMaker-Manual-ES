view_set_yport

Esta función puede utilizarse para fijar la posición y del puerto de vista dado. Se da el índice del puerto de vista (de 0 a 7) y la nueva posición en la que se va a colocar.

Sintaxis:

view_set_yport(view_port, y)

ArgumentoTipo Descripción
view_port The view port to target (0 - 7)
y The new y position

Devuelve:

N/A

Ejemplo:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
    view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

El código anterior comprobará la posición y del puerto de la vista[0] y si no está donde tiene que estar se pone en esa posición.
