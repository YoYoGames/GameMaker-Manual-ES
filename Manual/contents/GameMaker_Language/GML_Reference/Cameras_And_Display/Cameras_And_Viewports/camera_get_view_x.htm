camera_get_view_x

Esta función se puede utilizar para recuperar la posición x de la vista para una cámara determinada. Tenga en cuenta que esta función sólo es válida para las cámaras creadas con camera_create_view() o para las añadidas en el room editor.

Sintaxis:

camera_get_view_x(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

El código anterior recupera la posición de la cámara asignada al puerto de vista[0] y luego dibuja el texto relativo a esa posición.
