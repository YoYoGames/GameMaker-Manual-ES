ver_cámara

Este array contiene el ID de cámara único asignado al puerto de vista dado, y puede ser establecido a una nueva cámara o leído para obtener la cámara actual, devolviendo -1 si no hay ninguna cámara asignada. Puede tener hasta 8 puertos de vista activos en un room (valores de matriz 0 a 7), y puede asignar una cámara a cualquiera de ellos simplemente ajustando esta variable al valor del ID de la cámara (tal y como lo devuelven las funciones camera_create() o camera_create_view()). Si has establecido la cámara a un puerto de vista a través del Editor Room, esta variable mantendrá el ID de la cámara para ese puerto.

Sintaxis:

view_camera[0 ... 7];

Devuelve:

Camera ID or -1 (if no camera is assigned)

Ejemplo:

if view_camera[0] == -1
{
    view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

El código anterior comprueba si el puerto de vista [0] tiene una cámara adjunta y si no la tiene entonces se crea una y se le asigna.
