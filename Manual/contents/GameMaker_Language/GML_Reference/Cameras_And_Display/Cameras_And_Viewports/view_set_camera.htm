view_set_camera

Esta función establece una cámara en un puerto de vista específico. Se indica el puerto de visualización que se va a establecer (de 0 a 7), y se proporciona el valor único del ID de la cámara (devuelto por las funciones camera_create() o cuando se utiliza view_get_camera()). Si das un valor de -1 como ID de la cámara, entonces estás eliminando una cámara del puerto de vista y ten en cuenta que si ese puerto de vista está habilitado y visible puedes obtener algunos resultados impredecibles.

Sintaxis:

view_set_camera(view_port, camera_id)

ArgumentoTipo Descripción
view_port The view port to target (0 - 7)
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

El código anterior creará una nueva cámara y le asignará el puerto de vista[0].
