view_set_hport

Esta función se puede utilizar para establecer la altura del puerto de vista dado. Se da el índice del puerto de vista (de 0 a 7) y la nueva altura a utilizar.

Sintaxis:

view_set_hport(view_port, h)

ArgumentoTipo Descripción
view_port The view port to target (0 - 7)
h The new height (in pixels)

Devuelve:

Ejemplo:

if view_get_hport(0) != (display_get_height () / 2)
{
    view_set_hport(0, display_get_height() / 2);
}

El código anterior comprobará la altura del puerto de la vista[0] y si no es la mitad de la altura de la pantalla se establece en ese valor.
