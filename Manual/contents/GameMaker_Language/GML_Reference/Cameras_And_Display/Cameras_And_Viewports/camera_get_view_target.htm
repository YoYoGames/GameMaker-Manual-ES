camera_get_view_target

Esta función puede utilizarse para recuperar el objetivo de seguimiento de la cámara dada. El valor de retorno puede ser un índice object, un ID de instancia o -1 si no se ha establecido ningún objetivo de seguimiento.

Sintaxis:

camera_get_view_target(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Object Asset, Instance ID or -1

Ejemplo:

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_player);
}

El código anterior recupera el objetivo de seguimiento de la cámara asignada al puerto de vista[0] y luego comprueba si coincide con el índice "obj_Player" de object. Si no es así, la cámara de la vista se configura para seguir una instancia de ese object.
