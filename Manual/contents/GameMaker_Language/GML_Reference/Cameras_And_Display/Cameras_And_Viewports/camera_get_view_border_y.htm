camera_get_view_border_y

Esta función se puede utilizar para recuperar el valor del borde del objeto/instancia siguiente de la cámara dada a lo largo del eje y (borde vertical). El valor devuelto estará en píxeles.

Sintaxis:

camera_get_view_border_y(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

El código anterior recupera los valores xborder e yborder de la cámara asignada al puerto de vista[0] y luego comprueba si coincide con el valor dado. Si no lo hace, la cámara de la vista se establece en el valor dado.
