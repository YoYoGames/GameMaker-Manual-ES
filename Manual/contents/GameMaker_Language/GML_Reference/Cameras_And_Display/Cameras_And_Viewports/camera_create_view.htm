camera_create_view

Esta función creará una nueva cámara y luego establecerá su vista en el room, antes de devolver el valor de ID único para identificar la cámara en futuras llamadas a la función. La cámara se creará utilizando los valores que establezcas en esta función para la posición, el tamaño y otros detalles que se correlacionan con los valores que establezcas en el Editor Room (ver aquí). Tenga en cuenta que esta función puede tomar un número variable de argumentos, siendo sólo obligatorios la posición x/y y la anchura/altura. Cualquier argumento no suministrado utilizará sus argumentos por defecto.

Tenga en cuenta que las cámaras son como todos los recursos dinámicos y ocuparán memoria cuando se creen. Esto significa que debes tener cuidado de no crear cámaras innecesariamente y de destruir las que no necesites llamando a la función camera_destroy(). Esto también significa que las cámaras persistirán de room a room si se asignan a una variable global (también persistirán cuando se asignen a variables de instancia o locales, pero no podrás acceder a ellas después de cambiar room, de ahí la pérdida de memoria), de modo que puedes -por ejemplo- crear una nueva cámara, asignarla a una variable global, y luego al comienzo de cada room utilizar las funciones camera_set_* para establecer las propiedades antes de asignarla a un puerto de vista.

Sintaxis:

camera_create_view(room_x, room_y, width, height, [angle, object, x_speed, y_speed, x_border, y_border])

Argumento Tipo Descripción
room_x Real The x position (within the room) of the left side of the area the camera view captures.
room_y Real The y position (within the room) of the top of the area the camera view captures.
width Real The width of the area the view captures.
height Real The height of the area the view captures.
angle Real OPTIONAL El ángulo a establecer para la vista de la cámara (de 0° a 360°, por defecto es 0).
object Activo del objeto o ID de la instancia OPTIONAL El ID del objeto (o el ID de la instancia) para la vista de la cámara a seguir (por defecto es -1, sin object).
x_speed Real OPTIONAL La velocidad horizontal máxima a la que se puede mover la vista (se establece en -1 para el seguimiento "instantáneo", por defecto es -1).
y_speed Real OPTIONAL La velocidad vertical máxima a la que puede moverse la vista (se establece en -1 para el seguimiento "instantáneo", por defecto es -1).
x_border Real OPTIONAL El espacio horizontal mínimo entre el borde del área de la vista y la instancia que está configurada para seguir (antes de que la vista comience a moverse y si está configurada para seguir una instancia, por defecto es 0).
y_border Real OPTIONAL El espacio vertical mínimo entre el borde del área de la vista y la instancia que está configurada para seguir (antes de que la vista comience a moverse y si está configurada para seguir una instancia, por defecto es 0).

Devuelve:

Camera ID

Ejemplo:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

El código anterior elimina la cámara actual asignada a viewport[0] y luego crea una nueva cámara, define sus propiedades de vista y luego la asigna a viewport[0].
