camera_destroy

Con esta función puedes destruir cualquier cámara. Cuando se llama a la función se suministra el valor único del ID de la cámara, que se obtiene de las funciones de camera_create_* o del view_camera si se utiliza el Editor Room para configurar el puerto de vista y la vista. Nunca debes destruir una cámara que esté actualmente asignada a una vista visible, a menos que estés asignando una nueva cámara a esa vista en el mismo paso, y siempre debes destruir cualquier cámara que hayas creado a través de código cuando ya no sea necesaria para tu juego para evitar fugas de memoria, y también puedes destruir las cámaras por defecto si tienes alguna asignada en el Editor de Room, pero debes asignar una nueva cámara al puerto de vista (o desactivarla) de lo contrario obtendrás resultados extraños.

Sintaxis:

camera_destroy(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.

Devuelve:

N/A

Ejemplo:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

El código anterior destruye la cámara actualmente asignada al puerto de vista [0] y luego crea una nueva cámara y asigna su ID a ese puerto.
