camera_get_view_height

Esta función se puede utilizar para recuperar la altura (en píxeles) de la cámara dada. Tenga en cuenta que esta función sólo es válida para las cámaras creadas con camera_create_view() o para las añadidas en el editor de salas editor.

Sintaxis:

camera_get_view_height(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

El código anterior recupera la anchura y la altura de la cámara asignada al puerto de vista[0] y luego establece su posición relativa al centro.
