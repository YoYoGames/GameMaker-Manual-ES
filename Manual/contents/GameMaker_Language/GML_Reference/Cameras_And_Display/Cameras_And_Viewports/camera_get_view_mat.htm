camera_get_view_mat

Esta función se puede utilizar para recuperar la matriz de vista. La función devuelve la matriz que puede ser utilizada en otras funciones de matriz o para establecer la matriz de vista de otra cámara (utilizando camera_set_view_mat()). Tenga en cuenta que esta función se utiliza generalmente con cámaras creadas con camera_create()pero también puede utilizarse con las creadas en el editor room (o con camera_create_view()) - en cuyo caso la matriz devuelta sólo será válida después de que la cámara dada se utilice en un puerto de vista por primera vez.

Sintaxis:

camera_get_view_mat(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Matrix Array

Ejemplo:

mat = camera_get_view_mat(camera_view[0]);

El código anterior obtiene la matriz de vista para la cámara asignada al puerto de vista[0].
