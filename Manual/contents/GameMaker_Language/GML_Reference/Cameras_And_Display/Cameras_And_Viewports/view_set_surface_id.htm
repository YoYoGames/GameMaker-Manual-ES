view_set_surface_id

Con esta variable se puede establecer el contenido de un puerto de vista para dibujar a una superficie. Cuando se trabaja con superficies, a menudo se requiere capturar toda la región visible de la pantalla a la superficie, por lo que se asignaría a un puerto de vista utilizando esta función. Esto significa que todo lo que se muestra en el puerto de vista elegido se dibujará ahora a la superficie asignada y el contenido de ese puerto de vista ya no se mostrará, lo que significa que tendrás que

  • Habilitar una nueva vista y dibujar la superficie sólo en esa vista (utilizando view_current para comprobar qué vista se está dibujando)
  • Dibuja la superficie en el evento Draw GUI o Post Draw de una instancia, ya que estos eventos son independientes de las vistas.

Cuando se utiliza esta función se da el índice del puerto de la vista (de 0 a 7) y un índice de superficie (ya sea el application_surface o el valor del índice único devuelto por la función surface_create()) o, si se ha asignado previamente una superficie y se desea eliminarla, un valor de -1. Para más ejemplos sobre la asignación del puerto de vista a una superficie, véase la variable view_surface_id.

Sintaxis:

view_set_surface_id(view_port, surf)

Argumento Tipo Descripción
view_port Real The view port to target (0 - 7)
surf Identificación de la superficie The surface to apply to the view

Devuelve:

Real

Ejemplo:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

El código anterior comprobará si se ha asignado una superficie al puerto de la vista[0] y, si no lo ha hecho, se asignará una.
