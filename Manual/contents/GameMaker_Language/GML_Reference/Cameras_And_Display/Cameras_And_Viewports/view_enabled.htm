view_enabled

Esta variable controla si los puertos de vista que son visibles dentro de room están habilitados o no. Si tienes los puertos de vista configurados como visibles y luego desactivas esta opción, todo el room se dibujará en la pantalla escalado al tamaño de la ventana en lugar de que las diferentes cámaras se dibujen a través de los puertos de vista.

Sintaxis:

view_enabled;

Devuelve:

(true: enabled, false: disabled)

Ejemplo:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

El código anterior comprueba si las vistas están habilitadas para el room, y si no lo están, se asegura de que el puerto de vista[0] es visible y entonces habilita las vistas para el room.
