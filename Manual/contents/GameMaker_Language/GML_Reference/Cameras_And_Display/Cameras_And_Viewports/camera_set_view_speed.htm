camera_set_view_speed

Puede utilizar esta función para actualizar la velocidad de la cámara de visión dentro de la página web room. Debe dar el valor único de identificación de la cámara (tal y como lo devuelven las diferentes funciones camera_create() funciones) y luego la velocidad x e y (horizontal y vertical) y vertical) que debe moverse cuando se configura para seguir una instancia determinada. La velocidad se calcula como píxeles por paso y puede establecerse en "-1" para que la cámara se mueva instantáneamente, pero si la cámara no está configurada para seguir ninguna instancia, los valores establecidos aquí no tendrán ningún efecto visible. valores establecidos aquí no tendrán ningún efecto visible.

Sintaxis:

camera_set_view_speed(camera_id, xspeed, yspeed)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.
xspeed The speed (number of pixels per game frame) that the view should move on the horizontal (x) axis
yspeed The speed (number of pixels per game frame) that the view should move on the vertical (y) axis

Devuelve:

Ejemplo:

camera_set_view_speed(view_camera[0], 5, 5);

El código anterior establecerá la velocidad de la cámara de vista para la cámara asignada al puerto de vista[0].
