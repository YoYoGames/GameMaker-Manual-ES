camera_set_update_script

Esta función puede ser usada para establecer una función de script que será llamada cada frame del juego que la cámara sea asignada a un puerto de vista visible y activo. Se da el valor único del ID de la cámara (tal y como lo devuelven las diferentes funciones de camera_create ) y el ID de la función a llamar. El orden en el que las funciones adjuntas a las cámaras y el renderizado real de la vista de la cámara es el siguiente:

  • Las cámaras de todos los puertos de vista visibles y activos tienen su función de actualización llamada
  • Entonces, para cada vista:
    • Se llama a la función de inicio de la cámara para esa vista
    • Se ejecutan los eventos de dibujo para esa vista
    • Se llama a la función de finalización de la cámara
  • Pasar a la siguiente vista visible activa y repetir...

Aplicar una función de actualización personalizada a una cámara anula su comportamiento de actualización por defecto. Por ejemplo, si configuras tu cámara para que siga un object (digamos, obj_player) al llamar a camera_create_view(), o lo estableces así en las propiedades de Room, dejará de seguir ese object una vez que se establezca una actualización personalizada script.

Sintaxis:

camera_set_update_script(camera_id, script)

Argumento Tipo Descripción
camera_id Identificación de la cámara The unique camera ID value returned when you created the camera.
script Función del guión The script function to run each game frame

Devuelve:

N/A

Ejemplo:

camera_set_update_script(view_camera[0], updateCamera);

El código anterior establece la función de actualización para la cámara asignada al puerto de vista [0].
