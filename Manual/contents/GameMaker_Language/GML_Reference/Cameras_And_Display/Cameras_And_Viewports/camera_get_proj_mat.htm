camera_get_proj_mat

Esta función puede utilizarse para recuperar la matriz de proyección. La función devuelve la matriz que puede ser utilizada en otras funciones matriciales o para establecer la matriz de proyección de otra cámara (utilizando camera_set_proj_mat()).

Sintaxis:

camera_get_proj_mat(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Matrix Array

Ejemplo:

mat = camera_get_proj_mat(camera_view[0]);

El código anterior obtiene la matriz de proyección para la cámara asignada al puerto de vista[0].
