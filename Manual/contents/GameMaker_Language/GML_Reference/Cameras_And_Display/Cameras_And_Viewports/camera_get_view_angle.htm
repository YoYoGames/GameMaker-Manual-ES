camera_get_view_angle

Esta función se puede utilizar para recuperar el ángulo de la cámara dada. El valor de retorno estará entre 0 y 360. Tenga en cuenta que esta función sólo es válida para las cámaras creadas con camera_create_view() o para las añadidas en el editor room.

Sintaxis:

camera_get_view_angle(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var ang = camera_get_view_angle(view_camera[0]);
if (ang != 0)
{
    camera_set_view_angle(view_camera[0], 0);
}

El código anterior recupera el ángulo de la cámara asignada al puerto de vista[0] y luego lo comprueba para ver si coincide con el valor dado. Si no lo hace, la cámara de la vista se ajusta a ese ángulo.
