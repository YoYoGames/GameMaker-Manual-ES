camera_set_proj_mat

Esta función establece la matriz de proyección para una cámara determinada. Se le da el valor único del ID de la cámara (devuelto por las diferentes funciones camera_create() funciones) y una matriz de proyección a aplicar. Puede encontrar más información sobre la creación de matrices de proyección en la sección Funciones de Matriz, específicamente en matrix_build_projection_perspective() y matrix_build_projection_ortho().

Hay algunas cosas que debes tener en cuenta al utilizar esta función:

  • Si su cámara hace un seguimiento automático de object - es decir: ha sido creada usando camera_create_view() con un índice/identificación de instancia de object que no sea -1, o estás configurando una cámara definida en el editor de Room, o estás configurando la cámara por defecto - entonces esta matriz se sobrescribirá en el siguiente fotograma del juego.
  • Si utiliza esta función para switch a una proyección en perspectiva (en lugar de la proyección ortográfica por defecto), el dibujo automático de sprites en mosaico dejará de funcionar y tendrá que ser manejado manualmente. Esto incluye cualquier capa de fondo que utilice mosaicos horizontales o verticales y las funciones draw_sprite_tiled() y draw_sprite_tiled_ext(), que sólo dibujarán el sprite una vez en lugar de embaldosarlo si se está utilizando una proyección en perspectiva.

Sintaxis:

camera_set_proj_mat(camera_id, matrix)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Devuelve:

Ejemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

El código anterior crea una nueva cámara y la asigna al puerto de vista[0]. Esta cámara tiene entonces sus matrices de vista y proyección establecidas.
