camera_set_default

Esta función puede utilizarse para configurar la cámara por defecto para utilizar una cámara personalizada que haya creado previamente utilizando una de las camera_create() funciones. Cuando se crea un room sin puertos de vista activos o cámaras de vista, GameMaker sigue utilizando una cámara para mostrar la acción en el juego. Esta cámara se llama cámara por defecto y puede ser configurada y manipulada (e incluso destruida) como cualquier otra cámara.

Sintaxis:

camera_set_default(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Ejemplo:

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

El código anterior crea una nueva cámara y luego la asigna como la cámara por defecto a utilizar.
