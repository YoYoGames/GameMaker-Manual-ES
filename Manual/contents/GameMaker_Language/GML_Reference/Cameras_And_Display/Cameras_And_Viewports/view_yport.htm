ver_puerto

Con esta matriz incorporada puedes obtener o establecer la posición y del puerto de vista dado. El puerto de vista es el área de la pantalla donde se dibuja la vista, y puedes tener hasta 8 activos en cualquier momento (el array tiene valores de 0 a 7 inclusive para dar 8 puertos). Ahora, por defecto para GameMaker es que la ventana del juego (o lienzo de fondo) es del mismo tamaño que el room, sin embargo cuando activas los puertos de vista y las cámaras, este comportamiento cambia y se utiliza el tamaño total del cuadro delimitador para todos los puertos de vista. Así, si tienes dos puertos de vista diferentes en dos posiciones distintas, el área total que cubren define el tamaño de la ventana de juego. La siguiente imagen ilustra esto:

Si sólo tiene un puerto de vista, ajustar el valor x o y del puerto puede tener algunos efectos interesantes, pero normalmente no es práctico (ver la imagen de abajo) y por eso es mejor mantener la posición x e y como (0,0).

Sintaxis:

view_yport[0 .. 7];

Devuelve:

Ejemplo:

view_xport[0] = 0;
view_yport[0] = 0;

El código anterior restablece la posición del puerto de vista[0] a la posición (0,0) (esquina superior izquierda) de la pantalla.
