camera_set_view_border

Puede utilizar esta función para establecer el borde de seguimiento de la cámara de visión dentro de la página web room. Debe dar el valor único del ID de la cámara (tal y como lo devuelven las diferentes funciones de camera_create ) y luego dar el tamaño del borde x y el tamaño del borde y (horizontal y vertical). Estos valores sólo se utilizarán cuando a la cámara de visión se le haya asignado un objetivo de instancia a seguir (ya sea en el Editor Room o utilizando la función camera_set_view_target()) y se refieren a a la distancia del borde de la vista que debe tener la instancia antes de que la vista actualice su posición para seguirla.

Sintaxis:

camera_set_view_border(camera_id, x_border, y_border)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

Devuelve:

Ejemplo:

camera_set_view_border(view_camera[0], 64, 64);

El código anterior establecerá el borde de la cámara de vista para la cámara asignada al puerto de vista[0].
