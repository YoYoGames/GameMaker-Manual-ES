camera_set_view_mat

Esta función establecerá la matriz de vista para una cámara determinada. Se le da el valor único del ID de la cámara (devuelto por las diferentes funciones camera_create() funciones) y una matriz de vista a aplicar. Puede encontrar más información sobre la creación de matrices de vista en la sección Funciones de Matriz, específicamente matrix_build_lookat().

Tenga en cuenta que si su cámara hace un seguimiento automático de object - es decir: ha sido creada usando camera_create_view() con un índice object / ID de instancia que no es -1, o está configurando una cámara definida en el Editor Room, o está configurando la cámara por defecto - entonces esta matriz se sobrescribirá en el siguiente fotograma del juego.

Sintaxis:

camera_set_view_mat(camera_id, matrix)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Devuelve:

Ejemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

El código anterior crea una nueva cámara y la asigna al puerto de vista[0]. Esta cámara tiene entonces sus matrices de vista y proyección establecidas.
