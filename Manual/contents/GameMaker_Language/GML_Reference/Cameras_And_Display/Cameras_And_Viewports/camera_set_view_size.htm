camera_set_view_size

Puede utilizar esta función para actualizar el tamaño de la cámara de visión dentro de la página web room. Debe dar el valor único del ID de la cámara (devuelto por las diferentes funciones camera_create() funciones) y luego da la anchura y la altura (en píxeles) para ajustar la cámara.

Sintaxis:

camera_set_view_size(camera_id, width, height)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.
width The width of the camera view in pixels
height The height of the camera view in pixels

Devuelve:

Ejemplo:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

El código anterior establecerá el tamaño de la cámara de visualización para la cámara asignada al puerto de visualización[0].
