camera_get_default

Esta función se puede utilizar para recuperar el valor único del ID de la cámara por defecto (la cámara que GameMaker utiliza cuando no hay vistas de cámara o puertos activos en un juego room).

Sintaxis:

camera_get_default()

Devuelve:

Camera ID

Ejemplo:

var def = camera_get_default();
view_camera[0] = def;

El código anterior obtiene el ID de la cámara por defecto y establece la cámara de la vista para el puerto[0] para usarla.
