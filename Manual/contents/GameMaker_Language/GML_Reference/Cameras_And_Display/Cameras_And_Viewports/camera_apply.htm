aplicación_de_cámara

Esta función aplicará los ajustes de cámara dados a la vista de cámara actual que se está renderizando inmediatamente. Mientras que puedes establecer cualquier vista de cámara usando las funciones específicas de camera_set_*, los nuevos ajustes no serán usados para el renderizado hasta el siguiente frame del juego, pero con esta función puedes hacer esos cambios inmediatamente. Si estás usando múltiples vistas, entonces la función debe ser usada en el Evento de Dibujo y puedes usar la variable view_current para aplicar selectivamente la cámara dada sólo a la vista de cámara que se está renderizando actualmente. Tenga en cuenta que esto NO cambia el valor del ID de la cámara, sólo la configuración de esa cámara, y que si se utiliza esta función en la cámara por defecto (es decir, la cámara interna GameMaker utiliza cuando no hay cámaras de vista están activos) los cambios se sobrescribirán el próximo marco de juego por la configuración interna para el dibujo de la room.

Sintaxis:

camera_apply(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera.

Devuelve:

Ejemplo:

if view_current == 0
{
    camera_apply(cutscene_cam);
}

El código anterior (en un Evento de Dibujo) comprueba qué vista de cámara se está renderizando actualmente y si es la vista de cámara[0] entonces se le aplican los ajustes de la cámara referenciada por la variable "cutscene_cam".
