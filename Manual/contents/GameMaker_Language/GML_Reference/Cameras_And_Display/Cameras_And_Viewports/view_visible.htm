ver_visible

Esta variable se puede utilizar para averiguar si un puerto de vista en particular es actualmente visible o no. También puede establecer esta variable para activar o desactivar una vista estableciendo el valor en true (visible) o false (invisible). Tenga en cuenta que incluso si tiene un puerto de vista establecido como visible, si los puertos de vista no están habilitados (utilizando la variable incorporada view_enabled o habilitándolos en el Editor Room ) entonces no se dibujarán en la pantalla.

Sintaxis:

view_visible[0 ... 7];

Devuelve:

(true: enabled, false: disabled)

Ejemplo:

if !view_visible[0]
{
    view_visible[0] = true;
}

El código anterior comprueba si la vista[0] es visible o no y, si no lo es, la establece como visible.
