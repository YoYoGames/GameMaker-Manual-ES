camera_get_update_script

Esta función puede utilizarse para recuperar el ID de la función de script asignada como actualización script para la cámara dada. Si no se asigna script, la función devolverá -1.

Sintaxis:

camera_get_update_script(camera_id)

ArgumentoTipo Descripción
camera_id The unique camera ID value returned when you created the camera

Devuelve:

Script Function or -1 if no function is assigned

Ejemplo:

var scr = camera_get_update_script(camera_view[0]);
if (scr != scr_cutscene)
{
    camera_set_update_script(camera_view[0], scr_cutscene);
}

El código anterior comprueba la función script asignada como función de actualización para la cámara asignada al puerto de vista[0] y si no es "cutScene" se establece en ese script.
