camera_get_active

Esta función se puede utilizar para recuperar el valor único de identificación de la cámara actualmente activa.

Sintaxis:

camera_get_active()

Devuelve:

Camera ID

Ejemplo:

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = _active;
}

El código anterior obtiene el ID de la cámara activa y establece la cámara de la vista para el puerto[0] para usarla.
