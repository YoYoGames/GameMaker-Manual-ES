mostrar_obtener_frecuencia

Esta función devuelve la frecuencia (o tasa de refresco) de la pantalla en la que se está jugando. Devolverá un valor real como fotogramas por segundo, así que por ejemplo si tu monitor es de 60hz obtendrás 60, si está funcionando a 144hz entonces obtendrás 144, y así sucesivamente.

NOTA Esta función no es compatible con HTML5.

Sintaxis:

display_get_frequency();

Devuelve:

Ejemplo:

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

El código anterior obtiene la frecuencia de la pantalla, y ejecuta algunas condiciones para que el juego se ejecute a 240, 120, 60 o 30 fps. Esto significa que si tu pantalla es de 90hz el juego correrá a 60 fps, si es de 144hz entonces correrá a 120 fps, etc. Por supuesto, puedes pasar la frecuencia de la pantalla directamente a game_set_speed() para que se utilice como velocidad de fotogramas del juego.
