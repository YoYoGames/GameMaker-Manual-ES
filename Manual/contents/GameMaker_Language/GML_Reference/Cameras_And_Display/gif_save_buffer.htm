gif_save_buffer

Con esta función puedes guardar una animación GIF. Se suministra el índice del GIF (devuelto por la función gif_open()) y la función devolverá un grow de 1 byte alineado buffer con los datos del GIF. Tenga en cuenta que los datos GIF finales habrán sido paletizados utilizando la Paleta Universal 884 (ver aquí para más información).

Sintaxis:

gif_save_buffer(gif_index);

ArgumentoTipo Descripción
gif_index The ID of gif to save

Devuelve:

Buffer ID

Ejemplo:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        global.capture_buff = gif_save_buffer(gif_image);
        count = 0;
        save_gif = false;
    }
    count++;
}

El código anterior creará un archivo de imagen GIF con 30 fotogramas tomados de la superficie de la aplicación y luego lo guardará en un buffer.
