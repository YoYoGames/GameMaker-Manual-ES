gif_save

Con esta función puedes guardar una animación GIF. Debe proporcionar el índice del GIF (devuelto por la función gif_open()) así como un nombre de archivo para guardarlo. Tenga en cuenta que GameMaker no añade automáticamente el archivo .gif extension , por lo que deberá incluirlo como parte del nombre del archivo string si desea que el archivo guardado sea identificado como un GIF. El GIF creado se paletizará utilizando la Paleta Universal 884 (ver aquí para más información).

Tenga en cuenta que si la función tiene éxito y el GIF se guarda correctamente, entonces devolverá 0, de lo contrario devolverá -1.

Sintaxis:

gif_save(gif_index, fname);

ArgumentoTipo Descripción
gif_index The ID of gif to save
fname The filename to use for the gif

Devuelve:

Ejemplo:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

El código anterior creará un archivo de imagen GIF con 30 fotogramas tomados de la superficie de la aplicación y luego lo guardará.
