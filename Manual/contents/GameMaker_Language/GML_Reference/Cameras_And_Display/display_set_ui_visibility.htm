display_set_ui_visibility

Esta función se puede utilizar para mostrar u ocultar la interfaz de usuario del sistema sólo en Android. La función requiere que proporcione uno o más indicadores del sistema como un valor entero. Cuando se utiliza más de un flag, estos deben ser fusionados utilizando el bitwise "o", como se muestra en el ejemplo siguiente. Puede encontrar una lista de Android sistema flags aquí.

Sintaxis:

display_set_ui_visibility(flags);

Argumento Tipo banderas
flags Real The Android SYSTEM flags to use (as an integer value).

Devuelve:

N/A

Ejemplo:

var flags = 1024|4096;
display_set_ui_visibility(flags);

El código anterior utilizará el sistema Android flags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" y "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" para establecer la visibilidad de la pantalla.
