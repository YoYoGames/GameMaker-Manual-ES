ventana_obtener_ancho

Con esta función puedes obtener el ancho actual (en píxeles) de la ventana del juego.

Sintaxis:

window_get_width();

Devuelve:

Ejemplo:

ww = window_get_width();

El código anterior almacenará el ancho actual de la ventana en la variable "ww".
