ventana_puesta_de_la_puerta_de_la_ventana

Con esta función puedes fijar la ventana del juego en una posición específica dentro de la pantalla (en macOS, Linux(Ubuntu) y Windows) o dentro del navegador (HTML5).

NOTA: Si su juego HTML5 utiliza un index.html personalizado y que establece el lienzo a una posición fija, entonces esta función no tendrá efecto en la posición de la ventana.

Sintaxis:

window_set_position(x, y);

ArgumentoTipo Descripción
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

Devuelve:

Ejemplo:

window_set_position(0, 0);

El código anterior posicionará la ventana del juego en la esquina superior izquierda del navegador o de la pantalla (dependiendo del módulo de destino que se utilice).
