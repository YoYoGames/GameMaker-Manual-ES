display_set_sleep_margin

Esta función se puede utilizar para establecer el valor del margen de sueño utilizado para el tiempo de renderizado de su juego, y requiere un valor de milisegundos. Los valores por defecto son los siguientes:

Plataforma Milisegundos
Windows 10
macOS 10
Ubuntu 10
HTML5 10
Android 4
iOS 4
Windows UWP 10
Xbox 10
PS4 10
Nintendo Switch 10

Ten en cuenta que aunque el método de temporización esté configurado para utilizar la temporización vsync, el margen de reposo tendrá algún efecto sobre el renderizado del juego, por lo que debes tener cuidado al establecer este valor. Para más información sobre la temporización de la pantalla, consulta display_set_timing_method().

NOTA: Además del margen de reposo, puede controlar aún más su rendimiento y consumo de energía en Windows ajustando la resolución del programador de hilos en runtime.

Sintaxis:

display_set_sleep_margin(milliseconds);

Argumento Tipo Descripción
milliseconds Real The number of milliseconds for the sleep margin

Devuelve:

N/A

Ejemplo:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

El código anterior comprobará el método de temporización y, si no está establecido en tm_sleep, se establecerá en ese valor y el margen de reposo se establecerá en 20.
