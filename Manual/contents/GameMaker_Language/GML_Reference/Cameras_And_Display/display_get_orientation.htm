mostrar_obtener_orientación

Esta función devolverá una de las cuatro constantes que tiene GameMaker para indicarle si el dispositivo que ejecuta el juego está siendo sostenido en modo horizontal o vertical (ver la tabla de abajo).

Tenga en cuenta que esta función puede no detectar correctamente la orientación del dispositivo cuando se utiliza en el módulo de destino HTML5. Sin embargo, esto se puede imitar fácilmente mediante el uso del siguiente script:

return (browser_width < browser_height);

Esta función devolvería true para el retrato y false para el paisaje.

Sintaxis:

display_get_orientation()

Devuelve:

Display Orientation Constant

Constante de orientación de la pantalla
Constante Descripción
display_landscape El dispositivo se sostiene horizontalmente, es decir, el borde más largo es de izquierda a derecha, y el botón de menú está a la derecha.
display_landscape_flipped Como arriba, sólo que ahora el botón del menú está a la izquierda.
display_portrait El dispositivo se sujeta verticalmente, es decir, el borde más largo va de arriba a abajo y el botón de menú está en la parte inferior.
display_portrait_flipped Como en el caso anterior, sólo que ahora el botón del menú está en la parte superior.

Ejemplo:

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

El código anterior comprueba la orientación del dispositivo y establece una variable global en función del valor devuelto.
