gif_add_surface

Con esta función puedes guardar un fotograma en un archivo GIF desde una superficie. Se suministra el ID del archivo GIF (devuelto por la función gif_open()) y un ID de superficie (devuelto por la función surface_create()), así como un tiempo de retardo entre fotogramas. El tiempo de retardo se calcula en 1/100 de segundo, pero tenga en cuenta que una vez creado el GIF, si lo utiliza con fines promocionales, por ejemplo, los distintos navegadores pueden interpretar el retardo de los fotogramas de forma ligeramente diferente - vea aquí para más detalles. Puede (opcionalmente) suministrar valores de desplazamiento X e Y para la superficie que se utiliza para añadir al GIF, en cuyo caso la parte dibujada en el GIF comenzará desde el desplazamiento y no desde la posición por defecto (0,0), permitiéndole seleccionar sólo una sección de la superficie a añadir. También puede suministrar un valor de cuantificación, que es un número entero entre 0 y 3, donde 0 es cuantificación completa y 3 es cuantificación cero (los valores más bajos tendrán una degradación posterior en la calidad, pero crean un GIF más pequeño). Tenga en cuenta que no hay mucha diferencia entre la calidad por defecto de 2, y la más alta de 3, y el uso de 3 ralentizará significativamente la creación del GIF, por lo que se debe tener cuidado al utilizar el valor máximo.

Tenga en cuenta que no hay límites incorporados en el número de fotogramas o en el tamaño de esos fotogramas en su GIF, pero si utiliza demasiada memoria entonces la función fallará y devolverá -1, de lo contrario devolverá 0.

Sintaxis:

gif_add_surface(gif_index, surface, delay_time, [xoffset], [yoffset]);

Argumento Tipo Descripción
gif_index IDENTIFICACIÓN DEL GIF The ID of gif to add the surface data to
surface Identificación de la superficie The ID of the surface to use as the added frame
delay_time Real The delay time for the frame (in 100ths of a second)
[xoffset] Real OPTIONAL El desplazamiento a lo largo del eje X de la superficie a añadir desde
[yoffset] Real OPTIONAL El desplazamiento a lo largo del eje Y de la superficie a añadir desde
[quantization] Real OPTIONAL La cantidad de cuantificación de 0 a 3 (0 calidad mínima, 3 calidad máxima, por defecto es 2)

Devuelve:

Real

Ejemplo:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

El código anterior creará un archivo de imagen GIF con 30 fotogramas tomados de la superficie de la aplicación y luego lo guardará.
