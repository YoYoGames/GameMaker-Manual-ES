display_mouse_set

Con esta función puedes cambiar o fijar la posición del ratón dentro de la pantalla del juego, lo que puede ser útil para juegos FPS, por ejemplo. La función sólo funcionará mientras el juego esté enfocado y al usar ALT + Tab se desbloqueará el ratón.

Sintaxis:

display_mouse_set(x, y);

ArgumentoTipo Descripción
x The x coordinate in the display.
y The y coordinate in the display.

Devuelve:

Ejemplo:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

El código anterior centraría el ratón en la pantalla del juego.
