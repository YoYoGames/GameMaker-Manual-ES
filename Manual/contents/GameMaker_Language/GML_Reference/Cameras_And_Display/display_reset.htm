display_reset

Esta función restablece la configuración de la pantalla a la que se estableció cuando se inició el juego, pero también permite cambiar el nivel actual de antialiasing a pantalla completa que se está utilizando y si se utiliza la sincronización vertical. Los niveles de antialiasing disponibles son 0, 2, 4 y 8, siendo el valor de inicio por defecto 0, y el ajuste por defecto de v-sync es falso (desactivado). Activar v-sync puede dar una experiencia de juego más suave, pero también necesitará más potencia de procesamiento, por lo que su impacto debe ser considerado cuidadosamente antes de su uso, y lo mismo ocurre con el anti-aliasing, donde cuanto más alto sea el número, más procesamiento se requiere.

Como no todos los dispositivos de destino son iguales, algunos pueden no soportar el antialiasing 8x o 4x, por ejemplo, y por eso hay una variable de sólo lectura disponible para obtener los diferentes niveles de AA que el dispositivo que ejecuta el juego puede mostrar:

display_aa

Esta variable devolverá un valor entero basado en la configuración de bits para los diferentes niveles. Así que para sólo 2xAA, esto reportará 2, para la disponibilidad de 2x y 4x reportará 6. Para 8 y 4 informará 12. Para los 3 niveles (2,4 y 8) informará 14.

Sintaxis:

display_reset(aa, vsync);

Argumento Tipo Descripción
aa Real The level of anti-aliasing filtering (0, 2, 4 or 8).
vsync Booleano Toggle vertical synchronisation to on (true) or off (false).

Devuelve:

Real

Ejemplo:

if (display_aa > 12)
{
    display_reset(8, true);
}

El código anterior establecerá el nivel de antialiasing a 8 si es compatible y switch v-sync a on.
