ventana_de_color

Esta función permite establecer el color de fondo de la ventana de juego. Este color representa el que se utilizará para aquellas zonas de la ventana de juego que no estén ocupadas por ninguna vista. La siguiente imagen lo ilustra:

La imagen de arriba tiene dos vistas con dos puertos de vista, cada uno dibujado en diferentes posiciones. Esto estira la ventana del juego para acomodar ambos puertos y utiliza el color de la ventana para colorear el fondo donde no se muestra ninguna vista.

Sintaxis:

window_set_colour(colour);

ArgumentoTipo Descripción
colour The colour to set the region.

Devuelve:

Ejemplo:

if window_get_colour() != c_black
{
    window_set_colour(c_black);
}

El código anterior comprobará el color de la ventana para ver si está establecido como negro o no, y si no lo está lo establece como negro.
