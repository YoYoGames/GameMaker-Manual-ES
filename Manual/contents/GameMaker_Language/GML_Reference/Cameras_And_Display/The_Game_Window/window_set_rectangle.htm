window_set_rectangle

Con esta función puede establecer la posición de la ventana del juego dentro del navegador (HTML5) o de la pantalla (Windows, Ubuntu (Linux) o macOS) y establecer también la escala de la ventana. Para más información sobre la posición y el tamaño de la ventana, consulte window_set_position() y window_set_size().

Sintaxis:

window_set_rectangle(x, y, w, h);

ArgumentoTipo Descripción
x The new x coordinate of the window.
y The new y coordinate of the window.
w The new width of the window.
h The new height of the window.

Devuelve:

Ejemplo:

window_set_rectangle(0, 0, display_get_width(), display_get_height());

El código anterior hará que la ventana del juego ocupe toda el área de visualización (el navegador o la pantalla, dependiendo de la plataforma de destino).
