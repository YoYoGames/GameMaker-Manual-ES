window_set_size

Con esta función puedes cambiar el tamaño de la ventana del juego. En las plataformas de escritorio, esto suele referirse al área del cliente de la ventana, excluyendo la barra de título y los bordes:

Si usas esta función para cambiar el tamaño de la ventana del juego, el contenido de la ventana se escalará para ajustarse con una proporción 1:1 al lienzo de fondo, lo que mantendrá la imagen nítida y clara (aunque escalada, si la ventana es diferente del tamaño de room o de la vista). Por ejemplo, si tienes una ventana de 200 x 150 píxeles, y el jugador estira la ventana arrastrando los bordes, el tamaño final de la ventana puede ser de 400 x 300 píxeles, pero el lienzo de dibujo sigue siendo de 200 x 150, lo que da una imagen borrosa ya que la proporción de píxeles es ahora de 1:2. Sin embargo, si se utiliza esta función para cambiar el tamaño de la ventana, la relación 1:1 se mantiene, aunque la imagen se escala para ajustarse al nuevo tamaño de la ventana.

Sintaxis:

window_set_size( w, h );

ArgumentoTipo Descripción
w The new width of the window.
h The new height of the window.

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

El código anterior cambiará el tamaño de la ventana en función del valor de la variable "global.scale" cuando se pulse el botón del ratón.
