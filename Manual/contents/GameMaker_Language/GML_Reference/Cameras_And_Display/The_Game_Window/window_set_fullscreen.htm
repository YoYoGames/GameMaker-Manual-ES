window_set_fullscreen

Con esta función puede establecer que la ventana del juego sea a pantalla completa (true) o no (false). Tenga en cuenta que para el objetivo de macOS, debe haber desmarcado la opción "Iniciar en pantalla completa" y marcado la opción "Permitir al jugador cambiar el tamaño de la ventana del juego" en las Opciones del juego, de lo contrario esta función fallará. También tenga en cuenta que esta función no funcionará en HTML5 a menos que se añada como una devolución de llamada "clicable" (ver aquí para más detalles).

Sintaxis:

window_set_fullscreen(full);

ArgumentoTipo Descripción
full Whether to set the screen to fullscreen (true) or not (false).

Devuelve:

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

El código anterior comprueba si se ha pulsado el botón del ratón y, a continuación, establece la ventana a pantalla completa si aún no lo está, o la establece a ventana si ya lo está.
