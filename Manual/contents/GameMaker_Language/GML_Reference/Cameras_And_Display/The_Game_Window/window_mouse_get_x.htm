window_mouse_get_x

Con esta función puedes obtener la posición x del cursor del ratón (en píxeles) dentro del navegador si es un juego HTML5 o dentro de la pantalla si es un juego Windows, Ubuntu (Linux) o macOS.

NOTA: Para las funciones habituales del ratón, consulte la sección sobre la entrada del ratón

Sintaxis:

window_mouse_get_x();

Devuelve:

Ejemplo:

wx = window_mouse_get_x();

El código anterior almacena la posición actual de la ventana del eje x del ratón en la variable "wx".
