ventana_set_min_width

Esta función se puede utilizar para establecer un ancho mínimo de ventana para su juego. Si activas la opción de redimensionamiento de la ventana en las Opciones de Juego para la plataforma de destino, entonces el jugador puede redimensionar la ventana del juego a cualquier tamaño que desee, sin embargo usando esta función puedes limitar el ancho mínimo al tamaño que especifiques. Si desea volver al comportamiento por defecto (es decir, sin mínimo), entonces utilice un valor de -1.

NOTA: Esta función sólo está disponible en las plataformas Windows, macOS y Linux.

Sintaxis:

window_set_min_width(width);

ArgumentoTipo Descripción
width The minimum width in pixels for the game window.

Devuelve:

Ejemplo:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

El código anterior establecerá la anchura y la altura mínimas y máximas de la ventana del juego.
