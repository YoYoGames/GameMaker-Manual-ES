window_get_fullscreen

Esta función devuelve si la ventana del juego está en modo de pantalla completa (true) o no (false).

Sintaxis:

window_get_fullscreen();

Devuelve:

Ejemplo:

if window_get_fullscreen()
{
    draw_text(32, 32, "Fullscreen is ON");
}
else
{
    draw_text(32, 32, "Fullscreen is OFF");
}

El código anterior comprobará si la ventana está en modo de pantalla completa o no y dibujará un mensaje de texto dependiendo del valor devuelto.
