ventana_centro

Con esta función puedes centrar la ventana del juego en la pantalla cuando el módulo de destino es Windows, Ubuntu (Linux) o macOS, o puedes centrarla en el navegador si el módulo de destino es HTML5. Esta función no tiene efecto en ningún otro dispositivo.

NOTA Si has cambiado el tamaño de la ventana del juego en el paso actual (por ejemplo, cambiando de pantalla completa a ventana o usando window_set_size()), y deseas centrar la nueva ventana, esta función debe ser llamada al menos un paso después (por ejemplo, usando un alarm), de lo contrario no funcionará correctamente.

Sintaxis:

window_center();

Devuelve:

Ejemplo:

if !window_get_fullscreen()
{
    window_center();
}

El código anterior centrará la ventana del juego dentro de la pantalla si no está en modo de pantalla completa.
