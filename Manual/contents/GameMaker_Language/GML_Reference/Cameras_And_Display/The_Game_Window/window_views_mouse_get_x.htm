window_views_mouse_get_x

Esta función devuelve la coordenada x del ratón con respecto a todas las vistas activas y devuelve el mismo valor mouse_x.

NOTA: Para las funciones habituales del ratón, véase la sección sobre la entrada del ratón.

Sintaxis:

window_views_mouse_get_x();

Devuelve:

Ejemplo:

myx = window_views_mouse_get_x();

Esto establecería mix a la coordenada x del ratón en la ventana relativa a todas las vistas activas.
