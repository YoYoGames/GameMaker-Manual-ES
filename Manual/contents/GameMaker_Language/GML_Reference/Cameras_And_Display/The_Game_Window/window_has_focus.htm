window_has_focus

Con esta función puedes sondear el estado de la ventana (o pestaña) y si pierde el foco la función devolverá false de lo contrario devolverá true. En la mayoría de los casos puedes simplemente usar la función os_is_paused() para probar esto, pero en algunos casos muy específicos (por ejemplo, juegos en Chrome Apps) esa función no se activará, en cuyo caso deberías usar esta función en su lugar.

NOTA: Esta función sólo es válida para el HTML5, Windows, y MacOS.

Sintaxis:

window_has_focus();

Devuelve:

Ejemplo:

if !window_has_focus()
{
    pause_game();
}

El código anterior comprobará si la ventana del juego está enfocada o no, y si la función devuelve false se llamará a una función.
