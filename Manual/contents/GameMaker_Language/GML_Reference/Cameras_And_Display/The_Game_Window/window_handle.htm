window_handle

Con esta función se puede obtener el valor del id interno de Windows (el HWND, un puntero). Esta función es realmente útil sólo para los escritores de extension que necesitan el window handle para llamar a las APIs de Windows en el código DLL (el puntero devuelto debe ser fundido en un string y luego en el C++ sólo fundirlo en un HWND). La siguiente tabla muestra las plataformas soportadas junto con lo que devuelven:

Plataforma Devuelve
Windows Ventana HWND
macOS Clase NSWindow
Ubuntu (Linux) Mango de XWindow
HTML5 Lienzo ID

Sintaxis:

window_handle();

Devuelve:

Pointer

Ejemplo:

win_id = windows_handle();

El código anterior almacenará el id de la ventana del juego en una variable.
