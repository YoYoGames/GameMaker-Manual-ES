window_set_caption

Con esta función puedes cambiar o establecer el título de la ventana de room en la que te encuentras. Esta leyenda aparece en la parte superior de la ventana, junto al icono del juego, cuando éste no está en modo de pantalla completa.

Sintaxis:

window_set_caption(caption);

ArgumentoTipo Descripción
caption The new caption.

Devuelve:

Ejemplo:

if window_get_caption() != ""
{
    window_set_caption("");
}

El código anterior comprobará el título de la ventana para ver si tiene texto o no, y si lo tiene lo pone en un string vacío para que no se muestre ningún título.
