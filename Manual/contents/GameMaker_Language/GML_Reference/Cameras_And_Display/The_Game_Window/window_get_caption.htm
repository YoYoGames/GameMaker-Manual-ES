window_get_caption

Esta función devuelve el título de la ventana (es el texto que aparece en la parte superior de la ventana, junto a su icono) y, por defecto, muestra el título de la página room en la que se encuentra.

Sintaxis:

window_get_caption();

Devuelve:

Ejemplo:

if window_get_caption() != ""
{
    window_set_caption("");
}

El código anterior comprobará el título de la ventana para ver si tiene texto o no, y si lo tiene lo pone en un string vacío para que no se muestre ningún título.
