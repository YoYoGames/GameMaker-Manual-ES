ventana_get_x

Con esta función puedes obtener la posición x (en píxeles) dentro del navegador si es un juego HTML5 o dentro de la pantalla si es un juego Windows, Ubuntu (Linux) o macOS.

Sintaxis:

window_get_x();

Devuelve:

Ejemplo:

wx = window_get_x();

El código anterior almacena la posición actual de la ventana del eje x en la variable "wx".
