La ventana de juego

El juego que creas ocurre en una ventana (incluso cuando es a pantalla completa), y esta ventana tiene una serie de propiedades, como la posición, el tamaño, si es a pantalla completa, etc... Estos detalles se establecen normalmente de forma automática para ti, basándose en el tamaño de room y los puertos de vista habilitados en combinación con los ajustes de las Opciones de Juego para la plataforma de destino, pero puedes cambiarlos durante el juego usando las funciones listadas en esta página.

NOTA Estas funciones son sólo para Windows, Ubuntu (Linux), macOS y módulos de destino HTML5 y pueden no funcionar en cualquier otro dispositivo.

La siguiente imagen ilustra cómo se relacionan e interactúan algunas funciones generales de las ventanas:

Ten en cuenta que la imagen de arriba muestra la ventana del juego dibujada en un navegador, pero para los otros objetivos, puedes sustituir el navegador por la pantalla, por lo que una función como window_get_y() devolverá la posición de la parte superior de la ventana del juego en relación con la pantalla.
