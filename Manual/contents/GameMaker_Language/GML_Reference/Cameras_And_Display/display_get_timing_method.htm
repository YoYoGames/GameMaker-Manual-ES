display_get_timing_method

Esta función se puede utilizar para recuperar el método de cronometraje que se utilizará para renderizar su juego. El método puede ser una de las constantes enumeradas a continuación.

Para más información sobre los diferentes métodos de cronometraje, consulte display_set_timing_method().

Sintaxis:

display_get_timing_method();

Devuelve:

Display Timing Method Constant

Método de temporización de la pantalla Constante
Constante Descripción
tm_sleep El valor del margen de sueño es el principal método de temporización
tm_countvsyncs La sincronización Vsync es el método principal de sincronización (por defecto para todas las plataformas compatibles)

Ejemplo:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

El código anterior comprobará el método de temporización y, si no está establecido en tm_sleep, se establecerá en ese valor y el margen de reposo se establecerá en 20.
