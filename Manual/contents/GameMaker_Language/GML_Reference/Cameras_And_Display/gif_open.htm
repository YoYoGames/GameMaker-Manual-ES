gif_open

Con esta función puede crear una imagen vacía en formato GIF, lista para que se le añadan datos. Cuando llame a la función, tendrá que especificar la anchura y la altura del GIF (en píxeles, y no hay límite superior de tamaño excepto para la memoria disponible), y la función devolverá el valor de identificación único utilizado para identificar el regalo en las funciones posteriores, o devolverá -1 si el gif no se ha podido inicializar (por ejemplo, si la anchura/altura son demasiado grandes para la memoria disponible). También puede especificar un argumento opcional para establecer el "color claro" del GIF. Se trata de un valor RGB (sin componente alfa), que borrará el GIF a este color antes de que se le añada nada. Si no se proporciona un argumento de color claro, se utilizará el color negro por defecto.

Tenga en cuenta que al utilizar esta función debe llamar a gif_save() para finalizar la creación del GIF antes de abrir otro archivo para la grabación (por lo que para cada gif_open() debe haber un acompañamiento gif_save()).

Sintaxis:

gif_open(width, height, [clear_colour]);

Argumento Tipo Descripción
width Real The width of the gif to create
height Real The height of the gif to create
[clear_colour] Color OPTIONAL El color (RGB) para borrar el gif a

Devuelve:

GIF ID

Ejemplo:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

El código anterior creará un archivo de imagen GIF con 30 fotogramas tomados de la superficie de la aplicación y luego lo guardará.
