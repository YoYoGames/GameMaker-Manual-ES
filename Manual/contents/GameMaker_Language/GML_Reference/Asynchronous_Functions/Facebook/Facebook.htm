Facebook

Facebook es una de las redes sociales más importantes de Internet y muchos juegos y desarrolladores tienen actualmente sus propias páginas para publicar juegos y medios. GameMaker tiene un extension que permite conectarse con Facebook y realizar ciertas acciones y tareas que su API permite. Es recomendable que visites la página de desarrolladores de Facebook antes de empezar a utilizar la extension, ya que es esencial que conozcas bien el funcionamiento de Facebook Graph y las llamadas y permisos disponibles para que puedas sacar el máximo partido a estas funciones.

En el siguiente enlace puedes encontrar cómo configurar y utilizar Facebook extension para todas las plataformas disponibles:

  • Android, iOS y HTML5: Integración de Facebook

Las siguientes son funciones obsoletas que sólo eran válidas para la antigua API de Facebook. Por lo tanto, ya no puedes utilizar estas funciones en tus proyectos y se enumeran aquí en caso de que hayas importado un proyecto antiguo con ellas. Para poder utilizar las funciones actuales de Facebook, debes haber activado la API de Facebook en las opciones de juego correspondientes a la plataforma de destino, lo que te pedirá que descargues e instales un extension. El extension viene con un "mini-manual" en formato *.pdf en el que se explica cómo utilizarlo y se muestran todas las definiciones de las funciones y los ejemplos. Todas las referencias a las antiguas funciones de facebook_*() deberán ser sustituidas (y actualizadas) para utilizar los equivalentes de extension fb_*().

Las funciones que se indican a continuación sirven para integrar el juego con Facebook:

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
