HTTP

Esta sección enumera todas las diferentes funciones HTTP asíncronas disponibles en GameMaker. Estas funciones generarán un Evento HTTP Asíncrono en todas las instancias que lo tengan:

  • http_request
  • http_get
  • http_get_file
  • http_post_string

Tenga en cuenta que las funciones anteriores de http_* pueden no funcionar como se espera debido a problemas de seguridad entre dominios. Esto significa que las solicitudes a su server o los intentos de cargar recursos desde otros dominios son bloqueados y pueden aparecer resultados en blanco o errores 404. Una de las formas de evitar esto es tener algún PHP del lado de server que permita a ciertos dominios acceder a su server (esta es también una forma de proteger sus recursos y bloquear servers que no están incluidos en la lista de permitidos de PHP).

El siguiente es un ejemplo de código PHP que puede utilizar para esto:

$http_origin = $_SERVER['HTTP_ORIGIN'];
if ($http_origin == "http://127.0.0.1:51268")
{
    header("Access-Control-Allow-Origin: *");
}

Para las solicitudes de carga de imágenes en las que es importante determinar o establecer su tipo de origen cruzado, existen las siguientes funciones:

  • http_get_request_crossorigin
  • http_set_request_crossorigin

En la mayoría de los casos no debería ser necesario utilizar estas funciones, pero si el juego está almacenado en un server seguro - donde ciertos assets pueden requerir autenticación básica para ser accedidos y están generando errores de seguridad cuando se cargan - establecer el tipo de origen cruzado para las solicitudes de imágenes a "usar credenciales" puede ser necesario (en lugar de la configuración por defecto "anónimo").
