achievement_login_status

Esta función actualmente sólo es útil para Google Play en la plataforma Android. Comprobará si el usuario ya ha iniciado sesión en los servicios de Google, en cuyo caso devolverá true, o no, devolviendo false. Si devuelve verdadero no es necesario llamar a la función achievement_login() función.

Sintaxis:

achievement_login_status();

Devuelve:

Boolean

Ejemplo:

if !achievement_login_status()
    {
    achievement_login();
    }

El código anterior te registrará en el servicio de clasificación y logros elegido.
