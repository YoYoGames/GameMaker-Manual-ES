puntuación_posterior_de_logros

Puedes utilizar esta función para enviar tu puntuación al servicio de tablas de clasificación y logros elegido. En Android, se envía el ID de la tabla de puntuaciones como string (es el ID único de la tabla de clasificación que se asignó cuando se configuró la tabla de clasificación), mientras que en todas las demás plataformas se proporciona el nombre de la tabla de clasificación definida como string, y luego se da el valor real de la puntuación. La función desencadenará un Evento Social Asíncrono en el que el mapa async_load DS devuelto tendrá los siguientes pares clave/valor:

  • "type" - Es el tipo de evento que se ha disparado, que será el string "achievement_post_score_result" para esta función.
  • "id" - Es el ID del evento y, para esta función, devolverá una constante GML GooglePlayServices_EVENT_ID_POST_SCORE.
  • "leaderboard_id" - Es el ID único de la tabla de clasificación que se ha actualizado.
  • "Estado" - Será 1 si las puntuaciones se han actualizado correctamente, o 0 si la solicitud ha fallado
  • "daily_new_best" - Será 1 si la puntuación enviada fue la mejor del usuario en el último día, o 0 en caso contrario.
  • "daily_best" - Mantendrá la mejor puntuación del usuario del último día de juego.
  • "weekly_new_best" - Será 1 si la puntuación enviada fue la mejor del usuario en la última semana, o 0 en caso contrario.
  • "weekly_best" - Contiene la mejor puntuación del usuario de la última semana.
  • "all_time_new_best" - Será 1 si la puntuación enviada es la mejor de todos los tiempos del usuario, o 0 en caso contrario.
  • "all_time_best" - Contiene la mejor puntuación del usuario de todos los tiempos.

Sintaxis:

achievement_post_score(leaderboard, score)

Argumento Descripción
leaderboard El nombre/ID de la tabla de clasificación.
score La puntuación a enviar.

Devuelve:

N/A

Ejemplo ampliado:

if achievement_available()
    {
    achievement_post_score(global.LeaderboardID, 1000);
    }

El código anterior comprueba si el servicio de tabla de clasificación y logros elegido está disponible y, si es así, publica una puntuación en la tabla de puntuación llamada "Tabla de clasificación". Esto desencadenará un evento social asíncrono que puede ser analizado así:

if async_load[? "id"] == GooglePlayServices_EVENT_ID_POST_SCORE
    {
    if async_load[? "type"] == "achievement_post_score_result"
        {
        if async_load[? "status"] == 1
            {
            global.Best_Day = async_load[? "daily_best"];
            global.Best_Week = async_load[? "weekly_best"];
            global.Best = async_load[? "all_time_best"];
            }
        }
    }

El código anterior comprueba el mapa DS async_load para las claves "id" y "type" que corresponden a esta función, y luego comprueba el estado del evento. Si el estado es 1, entonces algunos valores del mapa DS se almacenan en variables globales globales.
