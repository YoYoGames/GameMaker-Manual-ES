incremento_de_logros

Los juegos pueden tener logros sin valor de finalización, por lo que puedes utilizar esta función para incrementar esos logros en una cantidad determinada. En Android, se envía el ID del logro como string (es el ID de logro único que se asignó cuando cuando se configura el logro), mientras que en todas las demás plataformas se suministra el nombre del logro definido como string, y luego se da el valor real del logro para incrementarlo.

La función desencadenará un evento social asíncrono en el que el mapa async_load devuelto tendrá los siguientes pares clave/valor:

  • "tipo" - Es el tipo de evento que se ha disparado, que será el string "achievement_increment_result" para esta función.
  • "id" - Es el ID del evento y, para esta función, devolverá una constante GML GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT.
  • "achievement_id" - Es el ID único del logro que se ha incrementado.
  • "estado" - Será 1 si el logro se incrementó con éxito, o 0 si la solicitud falló
  • "completo" - Será 1 si el logro ha sido completado por este incremento, o 0 si la solicitud falló

Sintaxis:

achievement_increment(name, value)

Argumento Descripción
name El nombre/identificación del logro.
value El valor que se enviará como incremento.

Devuelve:

N/A

Ejemplo:

achievement_increment(global.Achievement[0], score);

El código anterior añade la puntuación actual del jugador al logro con el ID almacenado en la matriz global.
