achievement_load_progress

Esta función enviará una solicitud a server para obtener información sobre todos los logros disponibles. Activará una llamada de retorno Social Asynchronous Event que contiene el mapa async_load mapa rellenado con los pares clave/valor relevantes. La clave de identificación de este mapa DS se utiliza para identificar la devolución de llamada correcta (puede haber puede haber más de una función de activación para cualquier evento asíncrono), y se emparejará con la constante achievement_achievement_info, así como con una serie de otros pares clave/valor para cada jugador. El contenido exacto del mapa se muestra a continuación:

NOTA: Esta función es sólo para iOS.

  • "id" - Para esta función debe ser achievement_achievement_info.
  • "numentries" - El número de logros en la lista.
  • "AchN" - El nombre del logro, donde "N" es un valor entero correspondiente a su lugar en la lista de entradas.
  • "AchCompN" - Cómo de completo es el logro "N" como valor porcentual de 0 a 100 (un string).

Sintaxis:

achievement_load_progress()

Devuelve:

N/A

Ejemplo ampliado:

El siguiente código solicitaría el progreso de los logros cuando el usuario está en un dispositivo iOS:

if os_type == os_ios
    {
    achievement_load_progress();
    }

Esto enviará una solicitud de información sobre los logros y generará una devolución de llamada asíncrona con el mapa especial async_load DS que contiene los siguientes datos:

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_achievement_info
    {
    var numentries = ds_map_find_value(async_load, "numentries");
    for(var i = 0; i < numentries; i++;)
        {
        ach_id[i] = ds_map_find_value(async_load, "Ach" + string(i));
        comp[i] = ds_map_find_value(async_load, "AchComp" + string(i));
        }
    }

El código anterior comprueba el mapa DS devuelto en el evento social asíncrono y si su "id" coincide con la constante que se está comprobando, entonces loops a través de las entradas en el mapa para cada uno de los logros y almacena su nombre y el porcentaje de finalización.
