logro_mostrar_logros

Esta función abrirá la página de logros de la plataforma elegida. Tenga en cuenta que se trata de una función asíncrona, es decir, su juego seguirá ejecutándose en segundo plano mientras se muestra la página de logros. Por lo tanto, debe tener cuidado cuidado con el uso de esta función y asegúrese de poner en pausa el juego o de permitir que se muestre sólo en las zonas del juego en las que no interfiera con la jugabilidad.

Sintaxis:

achievement_show_achievements()

Devuelve:

N/A

Ejemplo:

if achievement_available()
    {
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    achievement_show_achievements();
    }

El código anterior comprueba si el servicio de tabla de clasificación y logros elegido está disponible y, en caso afirmativo, establece una variable global y crea una instancia de controlador "pause" antes de llamar a la función para mostrar los logros en pantalla.
