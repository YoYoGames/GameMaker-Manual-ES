post_de_logros

Puedes utilizar esta función para enviar tus logros a la tabla de clasificación y al servicio de logros elegidos. En Android necesitarás el ID del logro como string (este es el ID de logro único que se asignó cuando configuraste el logro), mientras que en el resto de plataformas deberás proporcionar el nombre del logro como string, y luego dar el porcentaje que has completado para conseguir el logro (0 - Ninguno, 100 - Completado). La función desencadenará un evento social asíncrono en el que el async_load DS devuelto tendrá los siguientes pares clave/valor:

  • "type" - Es el tipo de evento que se ha disparado, que será el string "achievement_post_result" para esta función.
  • "id" - Es el ID del evento y, para esta función, devolverá una constante GML GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT.
  • "achievement_id" - Es el ID único del logro que se ha actualizado.
  • "Estado" - Será 1 si el logro se ha actualizado con éxito, o 0 si la solicitud ha fallado

Sintaxis:

achievement_post(achievement, percent)

Argumento Descripción
achievement El nombre/ID del logro.
percent El porcentaje del logro completado.

Devuelve:

N/A

Ejemplo:

if achievement_available()
    {
    achievement_post(global.Achievement[0], 100);
}

El código anterior comprueba si la tabla de clasificación y el servicio de logros elegidos están disponibles y, si es así, publica un logro.
