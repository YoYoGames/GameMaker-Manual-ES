logro_disponible

Esta función devolverá true si el usuario está actualmente conectado a Internet y el sistema de clasificación y logros elegido está disponible, de lo contrario devolverá false.

Sintaxis:

achievement_available()

Devuelve:

Boolean

Ejemplo:

if achievement_available()
    {
    achievement_post("ach_Level1", 100);
    }

El código anterior comprobará que el servicio de tabla de clasificación está disponible y, si es así, publicará en él.
