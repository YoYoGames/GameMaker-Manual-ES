reajuste_de_los_logros

Esta función restablecerá todos los logros a sus valores iniciales para el juego. Esta función se proporciona como una función de depuración y no se recomienda que permita al usuario final hacer esto en sus juegos.

Sintaxis:

achievement_reset()

Devuelve:

N/A

Ejemplo:

if debug_mode
    {
    if keyboard_check_pressed(vk_space)
        {
        achievement_reset()
        }
    }

El código anterior comprueba si el juego está en modo de depuración, y si lo está pulsando "espacio" se reiniciarán todos los logros.
