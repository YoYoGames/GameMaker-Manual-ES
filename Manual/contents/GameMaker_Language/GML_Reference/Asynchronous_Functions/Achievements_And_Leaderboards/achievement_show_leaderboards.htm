achievement_show_leaderboards

Esta función abrirá la página de tablas de clasificación de la plataforma elegida. Por favor, ten en cuenta que esta es una función asíncrona, es decir, tu juego continuará ejecutándose en segundo plano mientras se muestra la página de tablas de clasificación. Por lo tanto, debe tener cuidado con el uso de esta función y asegúrate de poner en pausa el juego o permitir que se muestre sólo en áreas de tu juego donde no interfiera con el juego.

Sintaxis:

achievement_show_leaderboards()

Devuelve:

N/A

Ejemplo:

if achievement_available()
    {
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    achievement_show_leaderboards();
    }

El código anterior comprueba si el servicio de tablas de clasificación y logros elegido está disponible y, en caso afirmativo, establece una variable global y crea una instancia de "pausa" del controlador antes de llamar a la función para mostrar las tablas de clasificación en pantalla.
