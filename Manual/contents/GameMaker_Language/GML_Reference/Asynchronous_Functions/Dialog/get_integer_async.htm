get_integer_async

Esta función abre una ventana y muestra un mensaje así como un espacio para que el usuario introduzca un valor (que contendrá el valor por defecto suministrado para empezar). Esta es una función asíncrona, y como tal GameMaker no bloquea el dispositivo en el que se está ejecutando mientras espera una respuesta, sino que sigue ejecutando los eventos de forma normal. Una vez que el usuario ha escrito su string y ha pulsado el botón "Okay", se activa un evento asíncrono Diálogo que, sólo mientras dure el evento, tendrá un mapa DS almacenado en la variable async_load.

Este mapa contendrá las tres claves, "id", "status", y "value". "id" es el valor que fue devuelto por la función cuando fue llamada, el "estado" será true para el botón "Okay" que fue presionado, o false si el mensaje fue cancelado (cuando sea aplicable ya que no todas las plataformas de destino permiten que los mensajes sean cancelados). Por último, "value" devolverá el número entero que el usuario haya introducido como string (se devolverá un string "" vacío si no se ha introducido nada).

NOTA Esta función es sólo para uso de depuración y pruebasy no debe ser utilizada en juegos lanzados. Para ello, debes crear una interfaz de usuario personalizada para recibir las entradas de los jugadores, de modo que tengas un control total sobre el aspecto y el comportamiento de los diálogos.

Sintaxis:

get_integer_async(string, default);

Argumento Tipo Descripción
string Cadena The message to show in the dialog.
default Real The default value.

Devuelve:

Async Request ID

Ejemplo ampliado:

El evento de pulsación del botón izquierdo del ratón (por ejemplo) del object que está mostrando el mensaje tendría el siguiente código:

msg = get_integer_async("How old are you?", 0);

Lo anterior mostrará un mensaje solicitando que el usuario ingrese un string y presione "Okay". El id de la función se almacena en la variable "msg" y se utilizará en el evento asíncrono Dialogs como se muestra a continuación:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
{
    if ds_map_find_value(async_load, "status")
    {
        global.Age = ds_map_find_value(async_load, "value");
    }
}

El código anterior comprueba la clave "id" del DS Map devuelto contra el valor almacenado en la variable "msg". Si son iguales, entonces comprueba si se ha pulsado "Okay" (en lugar de cerrar/cancelar la ventana) y si devuelve true entonces establece una variable global a partir del entero devuelto.
