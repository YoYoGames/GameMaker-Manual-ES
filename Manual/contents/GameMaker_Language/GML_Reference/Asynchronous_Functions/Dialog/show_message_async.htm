show_message_async

Esta función abre una ventana y muestra al usuario el mensaje que usted defina en la función. Esta es una función asíncrona, y como tal GameMaker no bloquea el dispositivo en el que se está ejecutando mientras espera una respuesta, sino que sigue ejecutando los eventos de forma normal. Una vez que el usuario ha pulsado el botón "Okay", se activa un evento asíncrono Diálogo que, sólo mientras dure el evento, tendrá un ds_map almacenado en la variable async_load.

Este mapa contendrá las dos claves, "id" y "status", siendo "id" el valor que fue devuelto por la función cuando fue llamada, y el "status" es true para el botón "Okay" que fue presionado, o false si el mensaje fue cancelado (donde esté disponible ya que no todas las plataformas de destino permiten la cancelación de diálogos).

NOTA Esta función es sólo para uso de depuración y pruebasy no debe ser utilizada en juegos lanzados. Para ello, debe crear una interfaz de usuario personalizada para recibir las entradas de los jugadores, de modo que tenga un control completo sobre el aspecto y el comportamiento de los diálogos.

Sintaxis:

show_message_async(string);

Argumento Tipo Descripción
string Cadena The message to show to the user.

Devuelve:

Async Request ID

Ejemplo ampliado:

El evento de pulsación del botón izquierdo del ratón (por ejemplo) del object que está mostrando el mensaje tendría el siguiente código:

msg = show_message_async("You will now be taken to the store");

Lo anterior mostrará un mensaje con el string dado. El identificador del mensaje se almacena en la variable "msg" y se utilizará en el evento asíncrono Dialogs como se muestra a continuación:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if (i_d == msg)
{
    if (ds_map_find_value(async_load, "status"))
    {
        url_open("https://play.google.com/store");
    }
}

El código anterior comprueba la clave "id" del DS Map devuelto con el valor almacenado en la variable "msg". Si son iguales, entonces comprueba si se ha pulsado el botón "Okay" (en lugar de cerrar/cancelar la ventana) y si devuelve true abre una url.
