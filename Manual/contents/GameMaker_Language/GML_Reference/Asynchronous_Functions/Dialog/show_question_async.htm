show_question_async

Esta función abre una ventana y muestra al usuario la pregunta que se define en la función. Esta es una función asíncrona, y como tal GameMaker no bloquea el dispositivo en el que se está ejecutando mientras espera una respuesta, sino que sigue ejecutando eventos de forma normal. La función tiene dos botones que muestran "Sí" y "No", y una vez que el usuario ha pulsado uno, se dispara un evento asíncrono de Diálogo que, sólo mientras dure ese evento, tendrá un mapa DS almacenado en la variable async_load.

Este mapa contendrá las dos claves, "id", y "status". "id" es el valor que fue devuelto por la función cuando se llamó, mientras que el "status" será true o false para "Sí" y "No" respectivamente.

NOTA Esta función es sólo para uso de depuración y pruebasy no debe ser utilizada en juegos lanzados. Para ello, debe crear una interfaz de usuario personalizada para recibir las entradas de los jugadores, de modo que tenga un control completo sobre el aspecto y el comportamiento de los diálogos.

Sintaxis:

show_question_async(string);

Argumento Tipo Descripción
string Cadena The question to ask to the user.

Devuelve:

Async Request ID

Ejemplo ampliado:

El evento de pulsación del botón izquierdo del ratón (por ejemplo) del object que está mostrando el mensaje tendría el siguiente código:

msg = show_question_async("Do you want to buy some armour for " + string(armour[0, 5]) + "coins?");

Lo anterior mostrará una pregunta con el string dado, solicitando que el usuario presione "sí" o "no". El id de la función se almacena en la variable "msg" y se utilizará en el evento de diálogo asíncrono como se muestra a continuación:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if (i_d == msg)
{
    if (ds_map_find_value(async_load, "status"))
    {
        coins -= armour[0, 5];
        global.protection += armour[0, 0];
    }
}

El código anterior comprueba la clave "id" del DS Map devuelto con el valor almacenado en la variable "msg". Si son iguales, entonces comprueba si uno de los dos botones fue pulsado y si devuelve true entonces deducirá un valor de una variable y añadirá un valor a una variable global también.
