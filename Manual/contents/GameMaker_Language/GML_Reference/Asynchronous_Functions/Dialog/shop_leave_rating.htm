shop_leave_rating

Esta función abre un diálogo dependiente del sistema operativo en el que puede pedir al usuario que publique una valoración o un comentario en una página determinada. Puede definir el texto que debe aparecer en el diálogo, así como el texto que desea que aparezca en los dos botones y la URL donde debe publicarse el comentario.

NOTA: Esta función es sólo para los objetivos Android y iOS.

¡IMPORTANTE! En iOS, cuando se utiliza TestFlight, en el desarrollo se mostrará el aviso pero no podrá hacer nada realmente. Una vez que su aplicación está en la App Store, este botón se activa y el usuario tiene la opción de escribir una reseña o dejar una calificación. Ten en cuenta que si distribuyes el juego a través de TestFlight, es posible que esta ventana ni siquiera se muestre.

Sintaxis:

shop_leave_rating(text, yes_string, no_string, url)

Argumento Tipo Descripción
text Cadena Text that appears on the dialog.
yes_string Cadena Text that appears on the "yes" button.
no_string Cadena Text that appears on the "no" button.
url Cadena The full URL where the comment has to be sent.

Devuelve:

N/A

Ejemplo:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

El código anterior pedirá al usuario que deje un comentario si la variable "timer" ha llegado a 0.
