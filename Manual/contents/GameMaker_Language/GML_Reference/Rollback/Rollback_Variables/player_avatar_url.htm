player_avatar_url

Esta variable almacena la URL del avatar del jugador, que puede cargarse mediante sprite_add().

Puede utilizar player_avatar_sprite que ya tiene cargado el sprite.

Esta variable sólo existe en las instancias de jugadores creadas por rollback_define_player()y es diferente para cada jugador.

Losjugadores invitados mostrarán un marcador de posición sprite.

Sintaxis:

player_avatar_url;

Devuelve:

String
