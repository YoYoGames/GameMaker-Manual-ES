rollback_confirmed_frame

Esta variable global almacena el último número de fotograma en el que se sincronizaron y confirmaron las entradas de todos los jugadores.

Puede comparar esto con rollback_current_frame para comprobar si el marco actual tiene entradas confirmadas para todos los jugadores.

También puede utilizar la función rollback_sync_on_frame() para comprobar si la trama actual está sincronizada.

Sintaxis:

rollback_confirmed_frame

Devuelve:

Real

Ejemplo:

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

Este código comprueba si la trama actual tiene entradas confirmadas de todos los jugadores, comprobando si el valor de la trama actual es igual al valor de la trama confirmada.

Si eso es cierto, ejecuta una función para comprobar si la condición de finalización del juego se ha cumplido, de modo que puede terminar el juego.

Esto se hace para evitar que un estado basado en la predicción termine el juego, y espera una trama totalmente sincronizada antes de intentar comprobar las condiciones de finalización del juego.
