player_avatar_sprite

Esta variable almacena el sprite de la instancia actual del jugador.

Esta variable sólo existe en las instancias de jugadores creadas por rollback_define_player()y es diferente para cada jugador.

Losjugadores invitados mostrarán un marcador de posición sprite.

Sintaxis:

player_avatar_sprite;

Devuelve:

Sprite Asset

Ejemplo:

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Esto muestra el avatar de la instancia del jugador que ejecuta el código, en un área de 64x64, 256px por encima de la instancia.
