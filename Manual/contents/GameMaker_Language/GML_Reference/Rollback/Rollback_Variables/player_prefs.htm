player_prefs

Esta variable almacena las preferencias establecidas para el jugador actual. Se asigna a las instancias de jugador creadas con rollback_define_player().

Por favor, lea: Preferencias del jugador

Sintaxis:

player_prefs;

Devuelve:

Any

Ejemplo:

image_index = player_prefs.character;

El código anterior lee una preferencia y establece una variable de instancia en el reproductor.
