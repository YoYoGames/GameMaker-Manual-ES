tipo_jugador

Esta variable indica si la instancia del jugador es un usuario de GX.games que ha iniciado sesión o un invitado. Puede devolver cualquiera de las siguientes cadenas:

Cadena Descripción
"Guest" El jugador no está registrado en GX.games
"User" El jugador se registra en GX.games

Los usuarios invitados obtienen avatares y nombres de reserva.

Esta variable sólo existe en las instancias de jugadores creadas por rollback_define_player()y es diferente para cada jugador.

Sintaxis:

player_type;

Devuelve:

String
