rollback_frame_actual

Es una variable global que almacena el número de fotogramas que han pasado desde que se inició la partida multijugador.

Debe utilizarse en lugar de current_time y otras variables similares, como se explica en Variables temporales.

Sintaxis:

rollback_current_frame

Devuelve:

Real

Ejemplo:

image_angle = dsin(rollback_current_frame) * 90;

El código anterior utiliza rollback_current_frame con dsin() para calcular un valor oscilante entre 1 y -1, que se multiplica por 90 y se aplica al ángulo de la instancia.
