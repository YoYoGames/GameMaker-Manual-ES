Multijugador Rollback

El multijugador de Rollback en GameMaker le permite configurar y crear rápidamente un juego multijugador en red para el objetivo Opera GX.

Esta funcionalidad utiliza los servidores GX para conectar a los jugadores y sincronizar la entrada, por lo que el desarrollador no necesita configurar server para iniciar una nueva partida en línea.

Mientras que la funcionalidad en línea sólo se puede utilizar con el objetivo Opera GX, puede utilizar "Sync Test" para probar su juego multijugador localmente, con entradas aleatorias o simuladas que se aplican a los otros jugadores. La prueba de sincronización es funcional en los objetivos Windows y macOS además de Opera GX.
