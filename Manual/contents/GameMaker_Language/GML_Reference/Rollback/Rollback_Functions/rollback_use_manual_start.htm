rollback_use_manual_start

Por defecto, una partida multijugador comienza automáticamente cuando todos los jugadores se han unido.

Llamar a esta función desactiva ese comportamiento de inicio automático, por lo que el juego espera a que lo inicies manualmente.

Si se utiliza, esta función debe ejecutarse antes de la función join/create.

Sintaxis:

rollback_use_manual_start();

Devuelve:

N/A

Ejemplo:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

El código anterior permite el inicio manual, y luego se une o crea una nueva partida multijugador.
