rollback_get_info

Esta función devuelve un struct que contiene información sobre un jugador. Sólo debe utilizarse cuando se gestionen manualmente las instancias de los jugadores. Si se utiliza rollback_define_player()entonces cada instancia de jugador recibirá automáticamente esta información en forma de variables.

La función toma el ID del jugador como argumento, y devuelve un struct con las siguientes variables:

Variable Tipo Descripción
player_avatar_url Cadena URL que apunta a la imagen del avatar del jugador
player_avatar_sprite Activo Sprite Sprite que contiene el avatar del jugador
player_name Cadena El nombre del jugador
player_type Cadena Indica si el jugador está registrado en GX.games o no

Será "Invitado" para los usuarios invitados, y "Usuario" para los usuarios registrados
player_user_id Cadena El ID del jugador en GX.games
player_prefs Cualquier Las preferencias establecidas para el jugador

Sintaxis:

rollback_get_info(player_id);

Argumento Tipo Descripción
player_id Real The ID of the player for which information will be returned.

Devuelve:

Struct
