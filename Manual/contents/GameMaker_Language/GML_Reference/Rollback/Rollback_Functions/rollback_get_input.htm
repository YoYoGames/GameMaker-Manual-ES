rollback_get_input

Esta función devuelve un struct que contiene los valores de entrada de un jugador. Toma un argumento opcional que especifica el ID del jugador para el que se devolverán las entradas.

Cuando se utiliza en una instancia de jugador creada por rollback_define_player()no es necesario especificar un ID de jugador, por lo que se puede llamar a esta función sin ningún argumento.

Cuando se utiliza en otras instancias, o en instancias de jugadores creadas manualmente, se debe especificar un ID de jugador.

Para obtener información sobre las entradas que se devuelven y cómo se pueden modificar, consulte Definición de entradas.

Sintaxis:

rollback_get_input(player_id);

Argumento Tipo Descripción
player_id Real OPTIONAL El ID del jugador para el que se devolverán las entradas.

Devuelve:

Struct

Ejemplo:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

El código anterior obtiene la entrada struct en una instancia de jugador que fue creada automáticamente por el sistema Rollback. Basándose en los valores devueltos, mueve la instancia 2 píxeles en cada dirección.

Para ver un ejemplo ampliado, consulte Crear una partida multijugador.
