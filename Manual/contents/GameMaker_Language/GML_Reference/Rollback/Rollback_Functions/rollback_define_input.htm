rollback_define_input

Esta función se utiliza para definir entradas personalizadas para todos los jugadores. Toma un struct que debe contener todas las entradas con sus constantes de entrada asignadas.

Para saber cómo asignar entradas personalizadas, consulte Controles personalizados.

Si se utiliza, esta función debe ejecutarse antes de la función join/create.

Sintaxis:

rollback_define_input(input_struct);

Argumento Tipo Descripción
input_struct Struct A struct containing input names as variables, and their assigned inputs as values.

Devuelve:

N/A

Ejemplo:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

El código anterior define entradas personalizadas para todos los jugadores. Las dos últimas entradas utilizan múltiples controles, enumerándolos en un array.

Para ver un ejemplo ampliado, consulte Definición de entradas.
