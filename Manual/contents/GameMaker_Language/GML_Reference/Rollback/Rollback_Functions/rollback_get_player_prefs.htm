rollback_get_player_prefs

Esta función devuelve el conjunto de preferencias para el ID de jugador dado.

Si no se especifica un ID de jugador, la función debe llamarse dentro de una instancia de jugador, en cuyo caso se devolverán las preferencias de ese jugador.

También puede utilizar la variable de instancia player_prefs.

Esta función devuelve las últimas preferencias que se sincronizaron para el jugador dado, por lo que la configuración de las preferencias no actualizará inmediatamente las preferencias que devuelve esta función.

Por favor, lea: Preferencias del jugador

Sintaxis:

rollback_get_player_prefs([player_id]);

Argumento Tipo Descripción
player_id Real OPTIONAL El ID del jugador para el que se devuelven las preferencias.

Devuelve:

Any

Ejemplo:

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

El código anterior obtiene las preferencias establecidas para el jugador actual, y aplica una preferencia a una variable de instancia.
