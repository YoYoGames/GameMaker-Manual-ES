rollback_use_player_prefs

Esta función activa las preferencias de los jugadores y permite establecer las preferencias por defecto para todos los jugadores.

Las preferencias por defecto pueden ser de cualquier tipo. Si no se especifican las preferencias por defecto, entonces todos los jugadores deben tener las preferencias establecidas antes de que el juego pueda comenzar.

Al llamar a esta función se desactiva el inicio automático, por lo que hay que iniciar el juego manualmente. Ver: Arranque manual vs. automático

Si se utiliza, esta función debe ejecutarse antes de la función join/create.

Por favor, lea: Preferencias del jugador

Sintaxis:

rollback_use_player_prefs([default]);

Argumento Tipo Descripción
default Cualquier OPTIONAL Las preferencias por defecto a establecer para todos los jugadores.

Devuelve:

N/A

Ejemplo:

rollback_use_player_prefs({
carácter: 0,
color: c_white
});

El código anterior habilita las preferencias del jugador, y establece las preferencias por defecto en un struct.
