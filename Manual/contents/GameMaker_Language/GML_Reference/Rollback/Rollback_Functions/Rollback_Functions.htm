Funciones de reversión

Las siguientes funciones se utilizan para iniciar, unirse o abandonar una partida multijugador:

  • rollback_join_game
  • rollback_create_game
  • rollback_start_game
  • volver a dejar el juego

Las siguientes funciones se utilizan para definir y obtener información para su juego:

  • Set
    • rollback_define_player
    • rollback_define_input
    • rollback_define_mock_input
    • rollback_define_input_frame_delay
    • rollback_use_manual_start
    • rollback_use_player_prefs
    • rollback_set_player_prefs
  • Obtenga
    • rollback_get_input
    • rollback_get_info
    • rollback_get_player_prefs

Las siguientes funciones se proporcionan para su uso durante un juego activo:

  • rollback_chat
  • rollback_sync_on_frame

Las siguientes funciones se utilizan para realizar pruebas, o para activar/desactivar características relacionadas con las pruebas:

  • rollback_use_random_input
  • rollback_display_events
