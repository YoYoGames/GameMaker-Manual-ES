rollback_chat

Esta función se utiliza para enviar un mensaje string a otros jugadores.

Se especifica un string para enviar, con un tamaño máximo de 128 bytes.

Cuando no se especifica el segundo argumento, el mensaje se envía a todos los jugadores, incluido el remitente.

Puedes especificar un ID de jugador en el segundo argumento, que es el jugador que recibirá el mensaje.

¿Cómo se usa?

Esta función está pensada para ser utilizada "localmente", por lo que sería el resultado de una entrada local, no de una entrada Rollback.

Vea el ejemplo de uso en la parte inferior. Escucha los mensajes recibidos en el evento Rollback Event.

Sintaxis:

rollback_chat(message, [to]);

Argumento Tipo Descripción
message Cadena The message to send, as a string.
to Real OPTIONAL El ID del jugador al que se enviará el mensaje. Si no se especifica este argumento, o si se especifica -1, se envía el mensaje a todos los jugadores.

Devuelve:

N/A

Ejemplo:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Si se pulsa la tecla Intro localmente, el keyboard_string valor se envía a todos los jugadores. Ese string se borra entonces para poder escribir un nuevo mensaje.

Esto desencadenará el evento Rollback Event cuando el mensaje sea recibido por todos los clientes (incluido el remitente).

Evento de retroceso

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
