rollback_start_game

Esta función fuerza el inicio de la partida multijugador actual, antes de que todos los jugadores se hayan unido. Primero hay que crear una partida multijugador con rollback_create_game().

El sistema normalmente espera a que todos los jugadores se unan antes de iniciar automáticamente la partida. Esta función puede utilizarse para iniciar la partida antes de que todos los jugadores se hayan unido.

Puede llamar a rollback_use_manual_start() para hacer que Rollback espere esta función incluso después de que todos los jugadores se hayan unido.

Al llamar a esta función se activará el evento Rollback Start.

Sintaxis:

rollback_start_game();

Devuelve:

N/A

Ejemplo:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Si el juego Rollback no está en marcha, y se pulsa Enter, esto iniciará el juego.

Esto permite al jugador iniciar la partida manualmente antes de que se unan todos los jugadores.
