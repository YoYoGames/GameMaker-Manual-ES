os_lock_orientation

Con esta función puede "bloquear" su dispositivo en la orientación actual hasta el momento en que lo "libere" para permitir de nuevo todas las orientaciones habilitadas en las Opciones de Juego para esa plataforma de destino. Tenga en cuenta que probablemente quiera confirmar que la orientación es la deseada antes de bloquearla.

Sintaxis:

os_lock_orientation(flag)

ArgumentoTipo Descripción
flag Set to true o false para activar o desactivar el bloqueo de la orientación.

Devuelve:

Ejemplo:

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

El código anterior comprueba el tipo de SO y si es Android o un iOS entonces el bloqueo de la orientación se marca como true.
