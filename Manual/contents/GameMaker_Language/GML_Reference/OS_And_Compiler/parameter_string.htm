cadena_de_parámetro

Los parámetros de la línea de comandos son esos comandos extra que puedes añadir a un exe para cambiar la forma en que se ejecuta el programa y con esta función puedes obtener el parámetro de la línea de comandos elegido como string. Puedes encontrar el número de parámetros para el juego actual usando la función parameter_count(), donde el primer parámetro tiene el índice 1 y el último tiene el índice devuelto por la función (un valor de 0 es especial ya que es el nombre de archivo del ejecutable del juego, incluyendo el path). Cabe destacar que esta función funcionará para en la plataforma HTML5, recuperando los parámetros de la url.

Sintaxis:

parameter_string(n);

Devuelve:

Ejemplo:

var p_num;
p_num = parameter_count();
if (p_num > 0)
{
    var i;
    for (i = 0; i < p_num; i += 1)
    {
        p_string[i] = parameter_string(i + 1);
    }
}

El código anterior obtendrá el número de parámetros de la línea de comandos, y si hay 1 o más, loop a través de ellos y los almacenará como strings en una matriz.
