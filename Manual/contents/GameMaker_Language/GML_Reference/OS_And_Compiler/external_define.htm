definición_externa

Esta función puede utilizarse para definir una llamada a una función externa a una dll específica (para Windows) o a una dylib (para Mac). Este archivo puede ser un archivo incluido o parte de un extension.

Se proporciona el nombre (y path) del archivo, y luego el nombre de la función que se desea definir. A continuación, debe definir la convención de llamada que se utilizará (véase la lista de constantes más abajo), así como el tipo de resultado que se espera (también una constante, como se indica más abajo).

Por último, debe indicar el número de argumentos que puede tomar la función (de 0 a 15) y para cada uno de los argumentos debe especificar también su tipo. Tenga en cuenta que para las funciones con 4 o más argumentos, todos ellos deben ser del tipo ty_real.

NOTA: Esto es sólo para dll o dylib que han sido añadidos como "Archivos Incluidos" al GameMaker IDE. No funcionará con aquellos archivos añadidos como extensions, ya que aquellos requieren que se definan las funciones en el propio paquete extension.

Constantes de tipo de llamada externa
Constante Descripción
dll_cdecl Esta es la llamada por defecto de C, C++
dll_stdcall Esta es la llamada estándar de WinAPI (Windows dll solamente)
Exteriores Data Type Constantes
Constante Descripción
ty_real Un argumento de número real
ty_string Un argumento terminado en cero string

Sintaxis:

external_define(dll, name, calltype, restype, argnumb, argtype[0], argtype[1], ...argtype[10]);

Argumento Tipo Descripción
dll Cadena The name of the DLL file (string)
name Cadena The name of the function (string)
calltype Constante de tipo de llamada externa (dll_*) The calling convention used
restype Data Type Constante externa (ty_*) The type of the result to expect
argnumb Real The number of arguments (0 - 10)
argtype[0 ... 10] Data Type Constante externa (ty_*) The types of the arguments being used

Devuelve:

External Function

Ejemplo:

my_funcion = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);

El código de ejemplo anterior definirá una función externa llamada "MyMin" con dos argumentos.
