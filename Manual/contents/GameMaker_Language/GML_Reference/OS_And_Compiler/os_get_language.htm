os_get_language

Esta función devuelve un string con el código de idioma de dos letras para el sistema operativo que está ejecutando el juego, tal y como establece el estándar ISO639. Si la información no está disponible, simplemente contendrá un string vacío "", o "en" para el idioma "inglés". Ten en cuenta que algunos idiomas también tienen un Código Regional relevante, así que para distinguir entre diferentes regiones del mismo país utiliza la función os_get_region().

La siguiente tabla muestra ejemplos de algunos de los principales códigos de dos letras de las lenguas definidos por la norma ISO 639:

NOTA: No se devuelve el código de país de la ubicación, sino el código de idioma del sistema operativo.

Sintaxis:

os_get_language()

Devuelve:

Ejemplo:

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

El código anterior comprueba el idioma del sistema operativo y abre un archivo *.ini diferente en función del valor devuelto.
