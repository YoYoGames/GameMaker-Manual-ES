os_is_network_connected

Con esta función puedes comprobar si tu dispositivo tiene actualmente una conexión a Internet y devolverá true si la tiene, o false si no la tiene, y dependiendo del SO, puede intentar hacer una conexión antes de devolver un valor.

La función tiene un argumento opcional para activar/desactivar el intento de establecer una conexión cuando se llama, y esta opción es actualmente sólo para el objetivo Nintendo Switch. Cuando se establece en true la función intentará hacer una conexión, y cuando se establece en false simplemente se limitará a comprobar la conexión. Tenga en cuenta que el intento de establecer una conexión puede hacer que el sistema operativo Switch se cuelgue durante unos segundos.

NOTA Esta función comprueba la API interna del dispositivo que controla las conexiones, por lo que puede devolver true si hay una conexión bluetooth, una conexión Wi-Fi, o incluso sólo una conexión de red normal que permite el acceso a Internet.

Sintaxis:

os_is_network_connected([attempt_connection]);

Argumento Tipo Descripción
attempt_connection Booleano OPTIONAL Sólo para el objetivo de Nintendo Switch. Establecido en true para intentar una conexión a nivel de SO cuando se llame.

Devuelve:

Boolean

Ejemplo:

if os_is_network_connected()
{
    global.connected = true;
}

El código anterior comprueba si el dispositivo tiene conexión a Internet y, si es así, establece una variable global.
