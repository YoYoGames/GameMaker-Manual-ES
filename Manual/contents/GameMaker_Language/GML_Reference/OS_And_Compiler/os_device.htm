os_device

Esta variable de sólo lectura contiene uno de los varios valores constantes listados a continuación para decirte en qué dispositivo estás ejecutando el juego actualmente. Tenga en cuenta que esta variable está obsoleta en favor de la función os_get_info() que devuelve información más precisa sobre el dispositivo que ejecuta el juego.

Sintaxis:

os_device

Devuelve:

Device Type Constant

Tipo de dispositivo Constante
Constante Descripción
device_ios_ipad iPad
device_ios_ipad_retina Nuevo iPad con pantalla Retina de 2048 x 1536
device_ios_iphone6 iPhone6 con pantalla de 1334 x 750
device_ios_iphone6plus Un iPhone 6 más grande con pantalla de 1920 x 1080
device_ios_iphone5 iPhone5 con tamaño de pantalla 640 x 1136)
device_ios_iphone Un iPhone/iPod Touch antiguo (pantalla de 480 x 320) o un teléfono Android
device_ios_iphone_retina Nuevo iPhone/iPod Touch con pantalla Retina de 960 x 640
device_emulator El dispositivo es en realidad un emulador (Windows Phone o Android)
device_tablet Tableta Android
device_ios_unknown Desconocido o no iOS

Ejemplo:

if (os_browser = browser_not_a_browser)
{
    switch (os_device)
    {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
    }
}
else
{
    global.Config = 1;
}

El código anterior comprueba si el juego se está ejecutando en un navegador o no y establece una variable global con un valor que depende del resultado de la comprobación.
