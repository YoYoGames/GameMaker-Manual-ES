os_version

Esta variable le indicará el número de versión del sistema operativo que está ejecutando su juego. Por ejemplo, si lo estás ejecutando en Windows 10, os_version será igual a 655360.

NOTA: Si necesita más información sobre el sistema operativo Windows puede utilizar el environment_get_variable().

Sintaxis:

os_version

Devuelve:

Ejemplo:

if (os_type == os_android) && (os_version > 10)
{
    global.GFX = 1;
}

El código anterior comprueba el tipo de sistema operativo y el número de versión y ambos son correctos, entonces la variable global se establece en 1.
