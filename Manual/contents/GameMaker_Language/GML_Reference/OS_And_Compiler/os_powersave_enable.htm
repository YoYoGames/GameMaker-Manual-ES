os_powersave_enable

Con esta función puedes activar o desactivar las funciones de ahorro de energía del dispositivo. Esto es importante ya que ciertos juegos (por ejemplo los que utilizan las funciones de inclinación) pueden no generar eventos que el SO pueda interpretar como entrada del usuario y así apagar la pantalla o salir del juego. Configurando esta función en false puedes desactivar las funciones de ahorro de energía y asegurarte de que la pantalla (y el juego) siempre funcionan.

NOTA: Esto se limita a los objetivos iOS y Android.

Sintaxis:

os_powersave_enable(flag)

ArgumentoTipo Descripción
flag Set to true o false para activar o desactivar las funciones de ahorro de energía.

Devuelve:

Ejemplo:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

El código anterior comprueba el tipo de sistema operativo y si es Android o iOS entonces se desactivan las funciones de ahorro de energía.
