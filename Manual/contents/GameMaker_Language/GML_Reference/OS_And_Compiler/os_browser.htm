os_browser

Esta variable de sólo lectura contiene una de varias constantes (listadas abajo) que GameMaker tiene para decirle en qué navegador está ejecutando el juego actualmente (si es que hay alguno).

Sintaxis:

os_browser

Devuelve:

Browser Type Constant

Tipo de navegador Constante
Constante Descripción
browser_not_a_browser El juego no se reproduce en un navegador
browser_unknown Navegador desconocido
browser_ie Internet Explorer
browser_ie_mobile Internet Explorer en un dispositivo móvil
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Safari
browser_safari_mobile Safari en un dispositivo móvil
browser_opera Ópera
browser_tizen Navegador de dispositivos móviles Tizen
browser_windows_store Aplicación Windows

Ejemplo:

if (os_browser == browser_not_a_browser)
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

El código anterior comprueba si el juego se está ejecutando en un navegador o no y establece una variable global con un valor que depende del resultado de la comprobación.
