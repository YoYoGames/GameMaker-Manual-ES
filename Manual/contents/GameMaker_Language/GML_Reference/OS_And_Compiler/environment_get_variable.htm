environment_get_variable

Esta función devuelve el valor (un string) de la variable de entorno con el nombre dado (también un string). Puede obtener las variables de entorno disponibles en macOS y Ubuntu (Linux) escribiendo "env" en la aplicación terminal, y para información sobre las variables de entorno de Windows, si está utilizando el símbolo del sistema escriba "echo %PATH%", y utilizando PowerShell es "ls env". Tenga en cuenta que tanto en macOS como en Ubuntu (Linux) la variable de entorno "HOME" devolverá la variable de entorno "~/" path que se asigna a "/Users/<username>" en macOS y "/home/<username>" en Ubuntu (Linux).

NOTA: Esta función es sólo para Windows, macOS y Ubuntu (Linux).

Sintaxis:

environment_get_variable(name);

ArgumentoTipo Descripción
name The name (a string) of the environment variable to check.

Devuelve:

Ejemplo:

e_str = environment_get_variable("APPDATA");

El código anterior devolverá el path completo para la variable de entorno "%appdata%", que normalmente es "C:\Users\{username}\AppData\Roaming".
