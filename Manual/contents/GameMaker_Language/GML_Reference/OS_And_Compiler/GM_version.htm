Versión GM

Esta constante mantiene el número de versión definido en las Opciones de Juego para cada plataforma de destino. El valor se almacena como string.

Sintaxis:

GM_version;

Devuelve:

Ejemplo:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

El código anterior toma las constantes de fecha y versión de GM y las dibuja en la pantalla.
