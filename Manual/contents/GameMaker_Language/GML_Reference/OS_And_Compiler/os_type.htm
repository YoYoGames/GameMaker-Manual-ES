tipo_os

Esta variable de sólo lectura contiene una de las varias constantes que tiene GameMaker para decirle para qué sistema operativo ha sido creado el juego. Tenga en cuenta que esto no es necesariamente lo mismo que el sistema operativo del dispositivo que lo ejecuta, ya que - por ejemplo - su juego podría estar funcionando en un sistema operativo Amazon Fire, pero habrá sido construido para la plataforma Android (en cuyo caso os_type será os_android).

Se pueden devolver las siguientes constantes:

Tipo de SO Constante
Constante Descripción
os_windows Sistema operativo Windows
os_uwp Plataforma Windows 10 Universal Windows
os_operagx Opera GX
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_ps5 Sony PlayStation 5
os_xboxone Microsoft Xbox One
os_xboxseriesxs Microsoft Xbox Serie X/S
os_switch Nintendo Switch
os_unknown Sistema operativo desconocido

Sintaxis:

os_type;

Devuelve:

OS Type Constant

Ejemplo:

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

El código anterior comprueba el sistema operativo que ejecuta el juego y establece una variable global en consecuencia.
