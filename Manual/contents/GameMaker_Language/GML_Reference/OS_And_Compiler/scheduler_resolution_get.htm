scheduler_resolution_get

Esta función se utiliza para recuperar la resolución del planificador de hilos de Windows en milisegundos. Si la resolución del programador está establecida en el valor por defecto (como se establece en Windows), la función devolverá -1.

Para obtener información sobre el programador de hilos, consulte la página de scheduler_resolution_set().

NOTA: Esta función es sólo para Windows .

Sintaxis:

scheduler_resolution_get();

Devuelve:

(or -1 for default)

Ejemplo:

scheduler_resolution = scheduler_resolution_get();

Este ejemplo recupera la resolución del programador de hilos Windows, y la almacena en la variable "scheduler_resolution".
