external_call

Si ha creado una llamada a una función externa a dll o dylib utilizando external_define()puede utilizar esta función para llamarla. Se proporciona el nombre de la función previamente definida, así como cada uno de los argumentos que requiere (cada argumento debe ser del tipo correcto, ya sea real o string) y la función devuelve el resultado de la llamada externa.

Sintaxis:

external_call(id, args[0...15]);

Argumento Tipo Descripción
id Función externa The name of the function that you want to call
args[0...10] Real o Cadena The different arguments that you want to pass through to the external function

Devuelve:

Variable (the type of value returned will depend on the defined function)

Ejemplo:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

El código de ejemplo anterior llama a una función externa previamente definida y almacena el valor devuelto en una variable local.
