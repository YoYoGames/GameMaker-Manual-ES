GM_build_type

Esta constante de string contiene el tipo de compilación: "exe" para un ejecutable creado o "run" para una ejecución de prueba iniciada desde IDE.

Sintaxis:

GM_build_type;

Devuelve:

String

Ejemplo:

draw_text(32, 32, GM_build_type);

El código anterior dibujará el tipo de compilación del ejecutable que se está ejecutando actualmente.
