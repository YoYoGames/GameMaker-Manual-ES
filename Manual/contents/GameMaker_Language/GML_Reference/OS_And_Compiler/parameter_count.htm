número_de_parámetros

Los parámetros de la línea de comandos son aquellos comandos extra que puedes añadir a un exe para cambiar la forma en que se ejecuta el programa. Puedes encontrar el número de parámetros para el juego actual usando esta función, donde el primer parámetro tiene el índice 1 y el último tiene el índice devuelto por la función (un valor de 0 es especial ya que es el nombre de archivo del ejecutable del juego, incluyendo el path). Cabe destacar que esta función funcionará para en la plataforma HTML5, recuperando los parámetros de la url.

Sintaxis:

parameter_count();

Devuelve:

Ejemplo:

p_num = parameter_count();

El código anterior almacenará el número de parámetros de la línea de comandos que se han utilizado para el juego en la variable "p_num".
