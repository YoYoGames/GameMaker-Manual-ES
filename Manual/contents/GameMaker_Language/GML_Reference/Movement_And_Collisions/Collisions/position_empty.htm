posición_vacía

Esta función comprobará si una posición dada entra en colisión con cualquier instancia con una máscara de colisión válida en la posición dada.

Sintaxis:

position_empty(x, y);

ArgumentoTipo Descripción
x The x position to check.
y The y position to check.

Devuelve:

Ejemplo:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if (position_empty(xx, yy))
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Esto comprobará una posición aleatoria en algún lugar de la room para una colisión y si no hay ninguna creará una instancia de obj_Bomb.
