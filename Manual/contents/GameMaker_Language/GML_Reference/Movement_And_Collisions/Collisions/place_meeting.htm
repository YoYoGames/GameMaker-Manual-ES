lugar_de_reunión

Con esta función puede comprobar una posición para una colisión con otra instancia o todas las instancias de un object utilizando la máscara de colisión de la instancia que ejecuta el código para la comprobación. Cuando usted utiliza esto usted está pidiendo efectivamente a GameMaker que mueva la instancia a la nueva posición, compruebe una colisión, retroceda y le diga si una colisión fue encontrada o no. Esto funcionará para colisiones precisas, pero sólo si tanto la instancia como el object que se está comprobando tienen máscaras de colisión precisas seleccionadas, de lo contrario, sólo se aplican las colisiones del cuadro delimitador. Esta función también acepta las palabras clave especiales all y other. Tenga en cuenta que si necesita obtener el identificador único de la instancia de object con la que se está colisionando, debe utilizar instance_place().

Tenga en cuenta que las coordenadas x/y dadas se rebajarán al número entero más cercano antes de realizar la comprobación.

Sintaxis:

place_meeting(x, y, obj);

Argumento Tipo Descripción
x Real The x position to check.
y Real The y position to check.
obj Activo del objeto o ID de la instancia The instance or object to check for.

Devuelve:

Boolean

Ejemplo:

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

El código anterior comprueba si no hay una colisión a la izquierda de la instancia y mueve la instancia si no la hay.
