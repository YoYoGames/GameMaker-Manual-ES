punto_en_círculo

Cuando se utiliza esta función, se define un área circular y GameMaker calculará si el punto dado cae dentro de sus límites o no. Si el punto cae dentro del círculo definido la función devolverá true de lo contrario la función devolverá false.

Sintaxis:

point_in_circle(px, py, x1, y1, rad);

ArgumentoTipo Descripción
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the circle centre.
y1 The y coordinate of the circle centre.
rad The radius of the circle.

Devuelve:

Ejemplo:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

El código anterior utiliza la función point_in_circle para comprobar si la posición del ratón cae dentro del área circular definida, estableciendo la variable "over" en true si lo hace, o false en caso contrario.
