posición_reunión

Con esta función se puede comprobar una posición para una colisión con otra instancia o con todas las instancias de un object. Cuando se utiliza esto se está comprobando un único punto en el room para una instancia o un object. Esta comprobación se hará contra el cuadro delimitador de la instancia o contra la máscara de la instancia si esa instancia tiene colisiones precisas comprobadas. Si necesita obtener la instancia única id del object con el que se está colisionando debe utilizar instance_position(). Esta función también acepta las palabras clave especiales all y other.

Sintaxis:

position_meeting(x, y, obj);

Argumento Tipo Descripción
x Real The x position to check.
y Real The y position to check.
obj Activo del objeto o ID de la instancia The object (or instance id, or the keywords "all" or "other") to check for a collision with.

Devuelve:

Boolean

Ejemplo:

if mouse_check_button(mb_left)
{
    if !position_meeting(mouse_x, mouse_y, all)
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

El código anterior comprueba el botón izquierdo del ratón, y si está pulsado comprueba la posición x/y del ratón para ver si hay una colisión con alguna instancia. Si no hay ninguna, se crea una instancia de "obj_Wall".
