posición_destrucción

Esta función simplemente destruye todas las instancias que se encuentran en colisión con la posición especificada. Las colisiones se basan en la máscara de las instancias, y si alguna parte de la máscara se solapa con el punto de destino, la función destruirá esa instancia. Las instancias destruidas de este modo activarán sus eventos de Destrucción y Limpieza.

Sintaxis:

position_destroy(x, y);

ArgumentoTipo Descripción
x The x coordinate of where to destroy colliding instances.
y The y coordinate of where to destroy colliding instances.

Devuelve:

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
    position_destroy(mouse_x, mouse_y);
}

Esto destruirá todas las instancias que colisionen con la posición del cursor del ratón cuando se pulse el botón izquierdo del mismo.
