colisión_elipse

Collision_ellipse utiliza los cuatro primeros argumentos (x1,y1,x2,y2) para definir la anchura y la altura de una elipse dentro de la actual room y luego comprueba si algún object que esté definido por el argumento "obj" está en colisión con esa área. Esta colisión puede ser comprobada como precisa o no, y también puede elegir comprobar la instancia que ejecuta el propio código o no. Considere esta imagen:

Aquí, la instancia del centro está usando una elipse de colisión para comprobar la bola objects. Ahora bien, las azules no tienen un cuadro delimitador preciso y, como puedes ver, aunque la sprite no esté tocando realmente la elipse, la colisión puede producirse (incluso si estableces la opción precisa en la función como verdadera) ya que el cuadro delimitador de esa sprite sobrepasa el área elíptica definida por collision_circle. Por otro lado, las bolas verdes sólo se considerarán en colisión si el sprite real sobrepasa la elipse definida. Recuerde que para que se consideren las colisiones precisas, tanto el object sprite como la función de colisión deben tener marcada la precisión como activada. También hay que tener en cuenta que el valor de retorno de la función puede ser el id de cualquiera de las instancias consideradas en colisión.

Tenga en cuenta que en lugar de un índice object puede proporcionar una instancia id para buscar una instancia específica, o las palabras clave de instancia all, o other (dependiendo del evento y del ámbito del código actual).

Sintaxis:

collision_ellipse(x1, y1, x2, y2, obj, prec, notme);

Argumento Tipo Descripción
x1 Real The x coordinate of the left side of the ellipse to check.
y1 Real The y coordinate of the top side of the ellipse to check.
x2 Real The x coordinate of the right side of the ellipse to check.
y2 Real The y coordinate of the bottom side of the ellipse to check.
obj Activo del objeto o ID de la instancia The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (true, que es más lento) o su caja delimitadora en general (false, más rápido).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).

Devuelve:

Instance ID or noone

Ejemplo:

if collision_ellipse(50, 50, 200, 100, obj_Player, false, true)
{
    instance_create_layer(obj_Player.x, obj_Player.y, "Effects", obj_Splash);
}

Esto comprobará una zona elíptica dentro de los límites de 50x, 50y y 200x, 100y para el object "obj_Player". Si hay una colisión con ese object, entonces creará una instancia de "obj_Splash" en las coordenadas x/y de obj_Player.
