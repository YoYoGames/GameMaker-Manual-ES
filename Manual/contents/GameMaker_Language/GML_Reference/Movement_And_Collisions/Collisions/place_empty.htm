lugar_vacío

Puedes utilizar esta función para comprobar si la instancia que llama colisiona con cualquier otra instancia de un object o con todas las instancias de tu juego. Para que esta colisión se resuelva correctamente, la instancia que ejecuta el código debe tener una máscara de colisión válida (ya sea para el propio sprite, o a través del mask_index) y sólo registrará colisiones con aquellas instancias que también tengan una máscara válida.

La función comprueba si no hay colisiones en caso de que la instancia que llama se sitúe en una posición específica, y se puede suministrar un argumento opcional para refinar la comprobación para que sólo se compruebe si una posición está libre de colisiones con instancias del tipo dado. Tenga en cuenta que si no se suministra un ID opcional de object, la comprobación se realizará con todas las instancias dentro de room. La comprobación de colisiones será precisa o se basará en el cuadro delimitador de la instancia, dependiendo del tipo de máscara de colisión que se haya seleccionado, pero para que las colisiones precisas funcionen correctamente, todas las instancias de la comprobación deben tener máscaras de colisión precisas.

Tenga en cuenta que las coordenadas x/y dadas se rebajarán al número entero más cercano antes de realizar la comprobación.

Sintaxis:

place_empty(x, y, [object_id]);

Argumento Tipo Descripción
x Real The x position to check.
y Real The y position to check.
[object_id] Activo de objeto OPTIONAL El objeto a comprobar.

Devuelve:

Boolean

Ejemplo:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
    x = mouse_x;
    y = mouse_y;
}

El código anterior comprobará si hay una colisión con cualquier otra instancia de la object "obj_Enemigo", como si la instancia que llama se colocara en la misma posición que el ratón. Si no se detecta ninguna colisión, entonces la instancia tiene sus coordenadas x/y fijadas a las del ratón.
