collision_circle

Collision_circle crea una zona circular del radio dado por el usuario alrededor de los puntos x1,y1 y, a continuación, comprueba si esa zona colisiona con cualquier instancia de object especificada por el argumento "obj". Esta comprobación puede ser precisa o no, pero para que las colisiones precisas estén habilitadas, el object o la instancia que se está comprobando debe tener también habilitadas las colisiones precisas para su sprite. Si no es así, la comprobación por defecto se basa en cajas delimitadoras. La siguiente imagen ilustra cómo funciona esto:

Aquí, la instancia del centro está usando un círculo de colisión para comprobar la bola objects. Ahora bien, las azules no tienen un cuadro delimitador preciso y, como puedes ver, aunque la sprite no esté tocando realmente el círculo, la colisión puede producirse (incluso si estableces la opción precisa en la función como verdadera) ya que el cuadro delimitador de esa sprite sobrepasa el área circular definida por collision_circle. Por otro lado, las bolas verdes sólo se considerarán en colisión si el sprite real sobrepasa el círculo definido. Recuerde que para que se consideren las colisiones precisas, tanto el object sprite como la función de colisión deben tener marcada la precisión como activada. También hay que tener en cuenta que el valor de retorno de la función puede ser el id de cualquiera de las instancias consideradas en colisión.

Tenga en cuenta que en lugar de un índice object puede proporcionar una instancia id para buscar una instancia específica, o las palabras clave de instancia all, o other (dependiendo del evento y del ámbito del código actual).

Sintaxis:

collision_circle(x1, y1, rad, obj, prec, notme);

Argumento Tipo Descripción
x1 Real The x coordinate of the center of the circle to check.
y1 Real The y coordinate of the center of the circle to check.
rad Real The radius (distance in pixels from its center to its edge).
obj Activo del objeto o ID de la instancia The object to check for instance collisions.
prec Booleano Whether the check is based on pixel-perfect collisions (true = slow) or its bounding box in general (false = fast).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).

Devuelve:

Instance ID or noone

Ejemplo:

if collision_circle(x, y, 20, obj_Cursor, false, true)
{
    image_index = 1;
}
else image_index = 0;

El código de arriba comprobará un área circular con un radio de 20pixels para una colisión con "obj_Cursor" y si hay una establecerá el image_index del object ejecutando el código a 1, pero si no hay establecerá el image_index del object a 0.
