mover_hacia_el_punto

Esta función simplemente le dice a la instancia que ejecuta el código que se mueva hacia un punto establecido a una velocidad determinada. Tenga en cuenta que es usted quien debe decirle qué hacer una vez que llegue a ese punto, ya que no se detendrá por sí mismo. Esto significa que puedes obtener una instancia "vibrante" mientras se mueve (por ejemplo) 3 píxeles hacia adelante, sobrepasando la posición de destino x/y en 2, luego se mueve 3 píxeles hacia atrás de nuevo, sobrepasando el punto de destino en uno y así sucesivamente... Hay varias maneras de evitar esto, por ejemplo usando distance_to_point() para ver a qué distancia de la ubicación especificada se encuentra la instancia antes de moverse.

NOTA: esta función establece la velocidad de la instancia, por lo que incluso si se detiene usando este código, la instancia seguirá moviéndose en la dirección anterior, por lo que es necesario establecer la instancia speed a 0 si desea que se detenga. Vea el ejemplo de código más abajo.

Sintaxis:

move_towards_point( x, y, sp );

Argumento Tipo Descripción
x Real The x position of the point to move towards.
y Real The y position of the point to move towards.
sp Real The speed to move at in pixels per step.

Devuelve:

N/A

Ejemplo:

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

El código anterior moverá la instancia hacia la posición de la instancia indexada en la variable "target" a una velocidad de 5 píxeles por paso. Una vez que alcanza esa posición la velocidad se pone a 0.
