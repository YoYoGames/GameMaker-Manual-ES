move_bounce_all

Con esta función puedes ordenar a una instancia que rebote contra todas las instancias dentro de room, con la única excepción de aquellas que no tienen sprite o índice de máscara asignado. También puedes decirle que utilice la comprobación precisa de colisiones cuando esté activada, pero ten en cuenta que esto requiere que todas las instancias tengan activadas las máscaras precisas y ralentizará mucho tu juego cuando haya muchas instancias implicadas debido a la cantidad de procesamiento que hay que hacer. Esto debería ir normalmente en el evento de paso de una instancia, pero puede ser usado selectivamente en un evento de colisión también, como se ilustra en el ejemplo de código de abajo.

Sintaxis:

move_bounce_all( adv );

ArgumentoTipo Descripción
adv Whether to enable advanced bouncing (true) or not (false).

Devuelve:

Ejemplo:

if other.visible
{
    move_bounce_all(false);
}

El código anterior se colocaría en un evento de colisión con otro object. Primero comprueba si el object es visible y si lo es entonces realiza la acción move_bounce_all(). Ten en cuenta que en este caso el rebote es selectivo y sólo se calculará para esta colisión, en lugar de para cada instancia, cada paso.
