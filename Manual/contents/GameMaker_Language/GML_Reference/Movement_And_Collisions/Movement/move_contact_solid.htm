move_contact_solid

Esta función moverá la instancia que ejecuta el código un número determinado de píxeles en la dirección especificada hasta que se encuentre con una instancia con sólido marcada como verdadera. Puede utilizar -1 o 0 para la distancia máxima siendo esta por defecto de 1000px, es decir: GameMaker moverá la instancia continuamente hacia arriba 1000 píxeles hasta que esté fuera de colisión.

Sintaxis:

move_contact_solid( dir, maxdist );

ArgumentoTipo Descripción
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Devuelve:

Ejemplo:

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

El código anterior comprobará debajo de la instancia si hay una colisión con "obj_Floor", y si no la hay, entonces la moverá hacia abajo hasta que haya una colisión, o se hayan cubierto 1000 píxeles.
