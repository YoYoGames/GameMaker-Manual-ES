move_outside_solid

Con esta función puede decirle a una instancia que se mueva fuera de una colisión con cualquier instancia marcada como sólido en cualquier dirección y cualquier número de píxeles en cada paso, con un valor de -1 o 0 para la distancia máxima siendo un valor por defecto de 1000px, es decir: GameMaker moverá la instancia continuamente hacia arriba 1000 píxeles hasta que esté fuera de la colisión.

Sintaxis:

move_outside_solid(dir, maxdist);

ArgumentoTipo Descripción
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Devuelve:

Ejemplo:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

El código anterior entraría en un evento de colisión y comprobaría si la "otra" instancia en la colisión está marcada como "sólida". Si lo es, moverá la instancia fuera de la colisión.
