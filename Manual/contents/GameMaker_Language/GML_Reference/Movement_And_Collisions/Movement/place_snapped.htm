place_snapped

Con esta función puedes comprobar si el origen de una instancia (su posición x e y) está alineado con una rejilla con los valores hsnap y vsnap especificados por ti.

Sintaxis:

place_snapped(hsnap, vsnap);

ArgumentoTipo Descripción
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

Devuelve:

Ejemplo:

with (obj_Pieces)
{
    if (!place_snapped(32, 32))
    {
        move_snap(32, 32);
    }
}

El código anterior comprueba todas las instancias de "obj_Pieces" para ver si están ajustadas a una cuadrícula de 32x32 píxeles, y si no lo están las ajusta a la posición más cercana de esa cuadrícula.
