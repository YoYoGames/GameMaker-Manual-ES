move_outside_all

Con esta función puedes decirle a una instancia que se mueva fuera de una colisión en cualquier dirección y cualquier número de píxeles en cada paso, con un valor de -1 o 0 para el maxdist siendo un valor por defecto de 1000px, es decir: GameMaker moverá la instancia continuamente hacia arriba 1000 píxeles hasta que esté fuera de la colisión.

Sintaxis:

move_outside_all(dir, maxdist);

ArgumentoTipo Descripción
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Devuelve:

Ejemplo:

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

El código anterior moverá la instancia hacia arriba 1 píxel cada paso que esté en colisión con cualquier otra instancia.
