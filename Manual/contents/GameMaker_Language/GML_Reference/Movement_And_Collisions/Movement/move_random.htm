move_random

Con esta función puedes establecer la instancia en una posición en cualquier lugar de room, pero alineada a una cuadrícula "invisible". Por lo tanto, un valor de 32 para el hsnap y el vsnap establecerá la instancia en una posición aleatoria que está alineada a una cuadrícula de 32x32 cuadrados. puedes establecer estos valores a 1 para obtener una posición en cualquier lugar en el room.

NOTA: Esta función se ajustará a las mismas posiciones cada vez que el juego se ejecute de nuevo debido a que GameMaker genera la misma semilla aleatoria inicial cada vez para hacer que el código de debugging sea una tarea mucho más fácil. Para evitar este comportamiento utilice randomise() al inicio de su juego. Esto sólo es cierto cuando se prueba y debugging el juego, ya que el paquete ejecutable final no mostrará este comportamiento y será aleatorio cada vez que se juegue.

Sintaxis:

move_random(hsnap, vsnap);

ArgumentoTipo Descripción
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Devuelve:

Ejemplo:

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

Esto moverá la instancia a una posición aleatoria en cualquier lugar de la room cuando se pulse la tecla espacio.
