mp_potencial_paso_objeto

Esta función permite a la instancia dar un paso hacia una posición determinada definida por xgoal/ygoal, intentando al mismo tiempo evitar los obstáculos. Cuando la instancia se encuentre con una instancia de la object especificada por el argumento "obj", cambiará la dirección del movimiento para intentar evitar esa instancia y moverse alrededor de ella. No está garantizado que este enfoque funcione, pero en la mayoría de los casos fáciles hará que la instancia se mueva hacia la meta. La función devuelve si el objetivo fue alcanzado o no.

Sintaxis:

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

Argumento Tipo Descripción
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Activo del objeto o ID de la instancia The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

Devuelve:

Boolean

Ejemplo:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

El código anterior primero comprueba si hay alguna instancia de "obj_Enemigo" en el room. Si la hay, entonces encuentra la más cercana y almacena su id en una variable. Esta variable se utiliza entonces para decirle a mp_potential_step_object que mueva la instancia con el código hacia la posición x/y del object que se encontró a una velocidad de 5 píxeles por paso mientras se evitan sólo las instancias del object "obj_Wall".
