mp_linear_step

Con esta función le dices a una instancia que dé un "paso" hacia un punto específico, especificado por los valores xgoal y ygoal. El tamaño del paso (es decir, cuántos píxeles debe mover la instancia en cada paso) se indica mediante el tamaño del paso, y si la instancia ya está en la posición se detendrá y no se moverá más, al contrario que otras funciones más sencillas como move_towards_point(). El tamaño de los pasos es también la distancia "por delante" a la que object comprobará cada paso en busca de una colisión, y se puede elegir si la instancia se detiene en una colisión con cualquier instancia o sólo con aquellas que estén marcadas como sólidas. La función devolverá si ha alcanzado la posición de meta (true) o si ha fallado (false).

NOTA Esta función no intenta realizar desvíos si encuentra un obstáculo, simplemente falla y deja de moverse.

Sintaxis:

mp_linear_step(xgoal, ygoal, stepsize, checkall);

ArgumentoTipo Descripción
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Devuelve:

Ejemplo:

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

El código anterior hará que el object se mueva hacia el ratón a una velocidad de 5 píxeles por paso. Una vez que alcance la posición del ratón creará un object "obj_Explosion" y se destruirá a sí mismo.
