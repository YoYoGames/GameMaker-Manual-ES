mp_grid_to_ds_grid

Con esta función puedes copiar la rejilla MP dada en una rejilla DS. La rejilla DS debe ser del mismo tamaño que la rejilla MP, aunque no tiene por qué serlo (se perderán datos si es más pequeña, y si es más grande todas las celdas extra de la rejilla serán 0). Una celda de la cuadrícula DS contendrá el valor -1 si fue marcada como ocupada en la cuadrícula MP, o 0 si no lo fue.

Sintaxis:

mp_grid_to_ds_grid(source, destination);

Argumento Tipo Descripción
source MP Grid ID Index of the mp_grid that is to be used
destination DS Grid ID Index of the ds_grid that is to be used to copy the grid data to.

Devuelve:

Boolean

Ejemplo:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

El código anterior creará una nueva cuadrícula DS y luego copiará los datos de la cuadrícula MP contenidos en la variable "mp_grid" en la nueva cuadrícula DS.
