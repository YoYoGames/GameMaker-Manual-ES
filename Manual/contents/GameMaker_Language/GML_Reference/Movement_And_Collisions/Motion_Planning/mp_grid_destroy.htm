mp_grid_destroy

Puede utilizar esta función para destruir el mp_grid indicado (como se define en la función mp_grid_create()) y liberar la memoria asignada. Es esencial que llames a esta función cada vez que la cuadrícula de MP haya terminado o podrías potencialmente obtener una fuga de memoria, lo que significa que tu juego se ralentizará con el tiempo y eventualmente se bloqueará.

NOTA: El uso de las funciones de mp_grid_* en una rejilla después de haberla destruido dará lugar a un error.

Sintaxis:

mp_grid_destroy(id);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be destroyed

Devuelve:

N/A

Ejemplo:

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

El código anterior destruirá el mp_grid indexado en la variable "grid" si la variable "timer" es igual a 0, y luego pasará a otro room.
