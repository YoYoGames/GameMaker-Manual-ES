mp_potencial_paso

Esta función permite a la instancia dar un paso hacia una posición particular definida por xgoal/ygoal, mientras intenta evitar los obstáculos. Cuando la instancia se encuentra con una instancia sólida (o cualquier instancia cuando checkall es verdadero) cambiará la dirección del movimiento para intentar evitar esa instancia y moverse alrededor de ella. No se garantiza que este enfoque funcione, pero en la mayoría de los casos fáciles hará que la instancia se mueva efectivamente hacia la meta. La función devuelve si el objetivo fue alcanzado o no.

Sintaxis:

mp_potential_step(xgoal, ygoal, stepsize, checkall)

ArgumentoTipo Descripción
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Devuelve:

Ejemplo:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step(inst.x, inst.y, 5, false);
}

El código anterior primero comprueba si hay alguna instancia de "obj_Enemigo" en el room. Si la hay, entonces encuentra la más cercana y almacena su id en una variable. Esta variable se utiliza entonces para decirle a mp_potential_step que mueva la instancia con el código hacia la posición x/y del object que se encontró a una velocidad de 5 píxeles por paso, evitando sólo las instancias marcadas como sólidas.
