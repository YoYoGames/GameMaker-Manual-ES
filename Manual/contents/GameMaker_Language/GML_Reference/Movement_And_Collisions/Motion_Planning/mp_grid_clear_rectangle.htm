mp_grid_clear_rectangle

Con esta función se puede definir un área en coordenadas room que borrará las celdas correspondientes en la cuadrícula MP especificada. Incluso si una celda se encuentra parcialmente dentro de la región rectangular definida, se borrará.

Sintaxis:

mp_grid_clear_rectangle(id, x1, y1, x2, y2);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be used
x1 Real The x coordinate of the left side of the rectangle to check.
y1 Real The y coordinate of the top side of the rectangle to check.
x2 Real The x coordinate of the right side of the rectangle to check.
y2 Real The x coordinate of the bottom side of the rectangle to check.

Devuelve:

Boolean

Ejemplo:

mp_grid_clear_rectangle(grid, 0, 0, 100, 200);

El código anterior marcará como libres todas las celdas del mp_grid indexadas en la variable "grid" que estén dentro del área 0,0 a 100,200.
