mp_grid_get_cell

Con esta función se puede comprobar cualquier celda del mp_grid para ver si ha sido marcada como ocupada o no, dando el índice del mp_grid (previamente creado) y las coordenadas x e y de la celda a comprobar.

Si ha sido ocupada o la posición que se comprueba está fuera de los límites de la cuadrícula, la función devolverá -1, en caso contrario devolverá 0.

Sintaxis:

mp_grid_get_cell(id, x , y);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be used
x1 Real The x coordinate of the grid to check.
y1 Real The y coordinate of the grid to check.

Devuelve:

Real

Ejemplo:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

El código anterior comprobará la celda mp_grid que corresponde a la posición del ratón y si está ocupada pone la variable image_blend en rojo, y si no está ocupada la pone en verde.
