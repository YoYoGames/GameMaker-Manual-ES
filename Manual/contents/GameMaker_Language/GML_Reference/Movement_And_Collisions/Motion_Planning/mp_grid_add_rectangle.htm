mp_grid_add_rectangle

Esta función le pide que defina un rectángulo dentro de room, y luego marca todas las celdas de la cuadrícula MP que "tocan" ese rectángulo como prohibidas, lo que significa que las funciones de búsqueda de rutas no pueden cruzarlas. La imagen siguiente ilustra cómo funciona esto:

Como puede ver, el rectángulo definido por (50, 90) a (200, 180) marca como prohibidas todas las celdas de la cuadrícula MP equivalente que toca.

Sintaxis:

mp_grid_add_rectangle(id, x1, y1, x2, y2);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be used
x1 Real The x coordinate of the left side of the rectangle to check.
y1 Real The y coordinate of the top side of the rectangle to check.
x2 Real The x coordinate of the right side of the rectangle to check.
y2 Real The x coordinate of the bottom side of the rectangle to check.

Devuelve:

N/A

Ejemplo:

mp_grid_add_rectangle(grid, 0, 0, 100, 200);

El código anterior marcará como prohibidas todas las celdas del mp_grid indexadas en la variable "grid" que estén dentro del área 0,0 a 100,200.
