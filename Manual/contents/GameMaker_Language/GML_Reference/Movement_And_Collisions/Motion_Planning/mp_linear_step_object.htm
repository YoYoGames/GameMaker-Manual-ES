mp_linear_step_object

Con esta función le dices a una instancia que dé un "paso" hacia un punto específico, especificado por los valores xgoal y ygoal. El tamaño del paso (es decir, cuántos píxeles debe mover la instancia en cada paso) se indica mediante el tamaño del paso, y si la instancia ya está en la posición se detendrá y no se moverá más, al contrario que otras funciones más sencillas como move_towards_point(). El tamaño del paso es también la distancia "por delante" que el object comprobará en cada paso en busca de una colisión, y usted puede elegir si la instancia se detiene en una colisión con un object o instancia de su elección.

Sintaxis:

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

Argumento Tipo Descripción
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Activo del objeto o ID de la instancia The object that is to block the path. Can be an object index, an instance id or the special keyword, all

Devuelve:

Boolean

Ejemplo:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

El código anterior hará que el object se mueva hacia el ratón a una velocidad de 5 píxeles por paso, comprobando únicamente las colisiones con el object "obj_Wall". Una vez que alcance la posición del ratón, creará una object "obj_Explosion" y se destruirá a sí misma.
