mp_grid_draw

Esta función dibujará la cuadrícula MP especificada (definida por mp_grid_create()), marcando las celdas libres en verde y las prohibidas en rojo. Esta función es esencial como herramienta de depuración, pero debe tenerse en cuenta que es muy lenta y sólo funciona cuando se utiliza en el evento de dibujo de la instancia, y que se puede establecer el alfa de dibujo para cambiar la opacidad de la rejilla, lo que le permite dibujarla como una superposición y ver lo que está realmente en el room al mismo tiempo.

Sintaxis:

mp_grid_draw(id);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be drawn

Devuelve:

Boolean

Ejemplo:

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

El código anterior dibujará la cuadrícula mp_grid indexada en la variable "grid" como una superposición semitransparente (pero sólo si la instancia que ejecuta el código tiene una profundidad menor que el resto).
