mp_grid_clear_all

Con esta función puede limpiar una cuadrícula MP de todas las celdas "prohibidas".

Sintaxis:

mp_grid_clear_all(id);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be used

Devuelve:

Boolean

Ejemplo:

if !instance_exists(obj_Player)
{
    mp_grid_clear_all(grid);
}

El código anterior borrará el mp_grid indexado en la variable "grid", marcando todas las celdas como libres, si ya no existe una instancia del object "obj_Player" en el room.
