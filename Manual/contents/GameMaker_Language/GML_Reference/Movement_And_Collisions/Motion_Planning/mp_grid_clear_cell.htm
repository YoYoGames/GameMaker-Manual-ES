mp_grid_clear_cell

Esta función puede utilizarse para borrar una "celda" específica de una cuadrícula MP. Las celdas no se calculan como coordenadas de room, sino como coordenadas de cuadrícula, donde (0,0) es la esquina superior izquierda de la cuadrícula. esto significa que para borrar una celda en una posición específica en el room, debemos cambiar las coordenadas x e y en coordenadas de celda dividiéndolas por la resolución de la cuadrícula MP. El ejemplo de código que sigue muestra cómo funciona esto.

Sintaxis:

mp_grid_clear_cell(id, h, v);

Argumento Tipo Descripción
id MP Grid ID Index of the mp_grid that is to be used
h Real Horizontal position of the cell to clear
v Real vertical position of the cell to clear

Devuelve:

Boolean

Ejemplo:

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

El código anterior hará que todos los "obj_Box" se destruyan a sí mismos y que marquen como libres las celdas que ocupaban en el mp_grid indexado en la variable "grid". En este ejemplo, encontramos la celda apropiada tomando la coordenada x/y del object y dividiéndolas por la resolución de la rejilla (usando floor para mantener los valores como enteros).
