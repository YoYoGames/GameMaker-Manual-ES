mp_linear_path

Esta función calcula una línea recta path desde la posición actual de la instancia hasta el punto especificado por los valores "xgoal" y "ygoal". Utiliza el tamaño de paso indicado, el mismo que en la función mp_linear_step(). El path indicado debe existir ya y será sobrescrito por el nuevo path y la función devolverá si se encontró un path completo (true) o no (false). Si se devuelve false, se sigue generando un path, pero sólo se ejecutará hasta la posición en la que se bloqueó el path.

NOTA: Esta función no mueve la instancia. Establece un path solamente, y usted debe usar las funciones Path para el movimiento.

Sintaxis:

mp_linear_path(path, xgoal, ygoal, stepsize, checkall);

ArgumentoTipo Descripción
path The index of the path to be used.
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check for collisions with all instances (true) or just solid ones (false).

Devuelve:

Ejemplo:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
    path_start(path, 2, 0, 0);
}

Esto hace que el object compruebe si hay un path lineal desde su posición actual hasta la posición de "obj_Player". Si la hay, se inicia el path.
