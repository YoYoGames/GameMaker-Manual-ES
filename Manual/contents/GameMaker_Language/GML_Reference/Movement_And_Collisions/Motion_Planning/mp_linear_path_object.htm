mp_linear_path_object

Esta función calcula una línea recta path desde la posición actual de la instancia hasta el punto especificado por los valores "xgoal" y "ygoal". Utiliza el tamaño de paso indicado, el mismo que en la función mp_linear_step(). El path indicado debe existir ya y será sobrescrito por el nuevo path y la función devolverá si se ha encontrado un path completo (true) o no (false). Sólo se encuentra un path completo si no hubo colisión con el object o la instancia indicada y si se devuelve false se sigue generando un path, pero sólo se ejecutará hasta la posición en la que se bloqueó el path.

NOTA: Esta función no mueve la instancia. Establece un path solamente, y usted debe usar las funciones Path para el movimiento.

Sintaxis:

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

Argumento Tipo Descripción
path Activo de la ruta The index of the path to be used
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Activo del objeto o ID de la instancia The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

Devuelve:

Boolean

Ejemplo:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

El código anterior comprueba si hay una colisión con "obj_Wall" a lo largo de path entre el object que ejecuta el código y la posición x/y del ratón. Si no hay colisión y se genera el path completo, entonces iniciará el object a lo largo del path generado.
