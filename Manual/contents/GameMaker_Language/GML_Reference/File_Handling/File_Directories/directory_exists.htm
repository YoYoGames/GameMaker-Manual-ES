directorio_existe

Esta función devolverá true si el directorio indicado existe o false si no existe. El nombre especificado debe incluir el path completo, no un path relativo y, por defecto, no se puede acceder a ningún directorio desde fuera del paquete de juegos, ya que todos los juegos están protegidos por una caja de arena (véase la sección sobre el sistema de archivos para más información).

Sintaxis:

directory_exists(dname)

ArgumentoTipo Descripción
dname The name of the directory to look for.

Devuelve:

Ejemplo:

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Esto comprobará si el directorio especificado existe y, si lo hace, irá allí y devolverá el primer archivo de "sólo lectura" doc encontrado.
