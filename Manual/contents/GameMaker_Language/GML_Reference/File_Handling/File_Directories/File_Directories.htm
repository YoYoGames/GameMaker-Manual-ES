Directorios de archivos

Cuando se trabaja con archivos, habrá momentos en los que se necesite hacer referencia al directorio en el que están almacenados los archivos y GameMaker tiene algunas funciones y variables especiales relacionadas con los diferentes directorios posibles que se pueden necesitar:

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory
  • game_save_id (de Control General del Juego)

NOTA No se pueden crear ni destruir directorios en HTML5.
