temp_directory

Esto se puede utilizar para devolver el directorio temporal creado para su juego cada vez que se ejecuta (la raíz no contiene el "\" final). Este directorio contendrá archivos y se puede acceder a él mientras el juego se está ejecutando, pero se eliminará (junto con todos los archivos que contiene) cuando el juego se cierre. con todos los archivos que contiene) cuando el juego se cierra.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

temp_directory

Devuelve:

Ejemplo:

ini_open(temp_directory + "\temp_ini.ini");

Esto abrirá un archivo ini en el directorio temporal del juego (creándolo si no existe ya).
