directorio_de_programas

Esto devolverá el directorio donde se almacena el ejecutable del juego. Sin embargo, esto puede no ser siempre útil, particularmente porque algunos dispositivos ejecutan el exe desde un archivo *.zip, por lo que esto devolvería lo mismo sin importar desde dónde se esté ejecutando el juego.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

NOTA: Si desactiva el sandbox, no podrá escribir en este directorio para evitar cualquier posible problema de borrado de los archivos necesarios del juego.

Sintaxis:

program_directory

Devuelve:

Ejemplo:

dir = program_directory;

Esto almacenará en una variable el directorio donde se encuentra el ejecutable.
