directory_destroy

Esta función eliminará un directorio con el nombre dado en el área de guardado.

NOTA: Esta función no funcionará en HTML5 ya que no se pueden crear o eliminar directorios en el almacenamiento local del navegador.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

directory_destroy(dname)

ArgumentoTipo Descripción
dname The name of the directory to remove.

Devuelve:

Ejemplo:

if directory_exists("DLC")
{
    directory_destroy("DLC");
}

Esto comprobará si el directorio especificado existe en la carpeta de datos local y, si lo hace, se elimina.
