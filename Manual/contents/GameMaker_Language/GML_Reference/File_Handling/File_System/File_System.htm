Sistema de archivos

En GameMaker existen una serie de funciones que permiten manipular archivos de todo tipo de diferentes maneras:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

Las siguientes funciones se pueden utilizar para cambiar los nombres de los archivos:

NOTA ¡Estas funciones no trabajan con los archivos reales! Sólo se ocupan de los strings relacionados con los diferentes elementos de path y deben utilizarse junto con las demás funciones de archivo.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Por último, hay algunas funciones especiales para obtener nombres de archivos que pueden utilizarse para cargar y guardar archivos con filtros de tipo de archivo. Estas funciones abrirán una ventana del explorador y, por tanto, harán que el juego pierda el foco. Esto significa que cualquier entrada de la que dependa el juego puede no ser registrada correctamente, y como tal recomendamos que (si se usan estas funciones en un evento de teclado) se use el evento de tecla arriba y no los eventos de pulsación o abajo.

NOTA Estas funciones son sólo para Windows y macOS.

ADVERTENCIA ¡Estas funciones pueden no funcionar como usted espera debido a que GameMaker está en modo sandbox! Por favor, vea la sección sobre el Sistema de Archivos para más información.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
