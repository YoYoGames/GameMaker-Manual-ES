cambio_de_archivo_ext

Esta función devuelve el nombre de archivo indicado con el extension (incluido el punto) cambiado por el nuevo extension. Utilizando un string vacío como nuevo extension se puede eliminar la parte extension por completo.

Sintaxis:

filename_change_ext(fname, newext);

ArgumentoTipo Descripción
fname The file to use.
newext The new extension to use.

Devuelve:

Ejemplo:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

El código anterior obtiene el nombre del archivo (como string) con la parte extension eliminada.
