file_rename

Esta función renombrará el archivo especificado con el nombre especificado. La función devolverá true si el archivo ha sido renombrado con éxito, o false en cualquier otra circunstancia.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

file_rename(oldname, newname);

ArgumentoTipo Descripción
oldname The name of the file to change.
newname The new name to give the file.

Devuelve:

Ejemplo:

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

Esto comprobaría la existencia de un archivo y si existe se le cambia el nombre.
