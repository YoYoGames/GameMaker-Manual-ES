copia_de_archivo

Esta función copiará el archivo especificado, dándole el nuevo nombre que usted elija.

Si estás copiando un archivo en Android desde tu área de Archivos Incluidos (que estará dentro de tu APK) se copiará primero en la memoria, así que, ten en cuenta que copiar archivos muy grandes desde el APK puede hacer que los dispositivos se queden sin memoria. En el objetivo de HTML5, utilizamos la API de almacenamiento local de HTML5 para guardar en el lado del cliente, y las funciones no admiten operaciones con archivos binarios, por lo que se asume que se está copiando un archivo de texto.

La función fallará si el archivo newname ya existe.

¡ATENCIÓN! Esta función puede no funcionar como esperas debido a que GameMaker está en un sandbox. Por favor, consulte la sección sobre los límites del sistema de archivos para obtener más información.

Sintaxis:

file_copy(fname, newname);

ArgumentoTipo Descripción
fname The name of the file to copy.
newname The name of the new, copied file.

Devuelve:

N/A

Ejemplo:

if file_exists("level1.txt")
{
    file_copy("level1.txt", "level2.txt");
}

Esto comprobaría un archivo y si existe se copia.
