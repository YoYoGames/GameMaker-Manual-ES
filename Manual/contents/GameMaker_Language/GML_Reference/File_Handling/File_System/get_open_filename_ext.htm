get_open_filename_ext

Esta función abre un diálogo y pide al jugador un nombre de archivo para abrirlo con el filtro dado. El filtro tiene la forma "nombre1|máscara1|nombre2|máscara2|...", donde la máscara puede contener las diferentes opciones con un punto y coma entre ellas y también se puede utilizar un "*" para significar cualquier string. Por ejemplo: "bitmaps|*.bmp;*.wmf", pediría al usuario que seleccionara el nombre de un archivo de mapa de bits del sistema, pero sólo aquellos que tengan los dos extensions especificados (bmp y png). ¡esta función no abre el archivo en sí! Sólo devuelve un string con el nombre completo y path del archivo. Si el usuario pulsa "Cancelar" se devuelve un string vacío. También puede añadir un directorio desde el que iniciar la búsqueda, y una leyenda que se mostrará en la parte superior de la ventana en lugar de la predeterminada.

Una cosa importante a tener en cuenta cuando se utiliza esta función es que le otorga ciertos permisos para ese archivo, durante la duración de su juego. Por lo tanto, una vez que obtengas el archivo path del usuario, podrás acceder a él una y otra vez sin tener que pedirlo. Sin embargo, como esta función es para leer un archivo, los permisos concedidos son sólo para leer y guardar no serán permitidos por el SO. Si necesita leer y escribir en el archivo, puede obtener el permiso utilizando la función get_save_filename().

NOTA: Esta función sólo es válida en los objetivos Windows y macOS.

Sintaxis:

get_open_filename_ext(filter, fname, directory, caption);

ArgumentoTipo Descripción
filter The file type filter and/or name.
fname The suggested filename to use.
directory The directory to start the search from.
caption The caption for the window.

Devuelve:

Ejemplo:

var file;
file = get_open_filename_ext("text file|*.txt", "", working_directory, "Open a saved level");
if file != ""
{
    file_text_open_read(file);
}

El código anterior pedirá al usuario que seleccione un archivo de texto para abrirlo desde working_directory y luego comprobará el nombre del archivo devuelto y path, y si existe abrirá el archivo para su lectura.
