archivo_existe

Esta función devolverá true si el archivo especificado existe y false si no existe. Tenga en cuenta que la función sólo puede utilizarse para comprobar los archivos locales, pero no cualquier archivo almacenado en un server remoto.

Sintaxis:

file_exists(fname);

ArgumentoTipo Descripción
fname The name of the file to check for.

Devuelve:

Ejemplo:

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

Esto comprobaría la existencia de un archivo y, si existe, se abre para su lectura.
