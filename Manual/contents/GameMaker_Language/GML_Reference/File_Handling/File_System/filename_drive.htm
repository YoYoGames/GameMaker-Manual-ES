filename_drive

Esta función devuelve la información de la unidad de disco del nombre del archivo.

Sintaxis:

filename_drive(fname);

ArgumentoTipo Descripción
fname The file to use.

Devuelve:

Ejemplo:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

El código anterior obtiene la información de la unidad (como string) del archivo especificado.
