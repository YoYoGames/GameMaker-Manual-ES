archivo_eliminar

Esta función borrará el archivo especificado del sistema. Debe tenerse en cuenta que esta función sólo borrará aquellos archivos que GameMaker es capaz de crear y analizar: archivos ini, archivos de texto y archivos binarios, o aquellos archivos hechos para almacenar recursos creados por el juego como sprites o superficies. Sin embargo, no eliminará ningún otro archivo. La función también devolverá true si el archivo ha sido eliminado con éxito, o false en cualquier otra circunstancia.

¡ATENCIÓN! Esta función puede no funcionar como esperas debido a que GameMaker está en un sandbox. Por favor, consulte la sección sobre los límites del sistema de archivos para obtener más información.

Sintaxis:

file_delete(fname);

ArgumentoTipo Descripción
fname The name of the file to delete.

Devuelve:

Ejemplo:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Esto comprobaría la existencia de un archivo y si existe se elimina.
