file_text_write_real

Con esta función puede escribir un número en el archivo de texto previamente abierto. Tenga en cuenta que como el valor a escribir puede ser un número real, todos los decimales se escribirán con un punto "." como separador. Si el fichero ya contiene información, ésta se borrará y el string se escribirá al principio del fichero, a menos que haya abierto el fichero con la función file_text_open_append(). También puede evitarlo utilizando la función file_text_readln() junto con la función file_text_eof() para loop a través del contenido del archivo hasta llegar al final y luego comenzar a escribir.

Es importante tener en cuenta que cuando se escriben números muy grandes en un archivo de texto utilizando esta función, puede ser traducido a un formato estándar simplificado, como "6.6624e+003", que no puede ser leído de nuevo en GameMaker correctamente. Para evitar este tipo de problemas, debería convertir el valor en un string y utilizar la función file_text_write_string() en su lugar.

Sintaxis:

file_text_write_real(fileid, val);

ArgumentoTipo Descripción
fileid The id of the file to edit.
val The real value to write to the file.

Devuelve:

Ejemplo:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

El código anterior abre un archivo para escribir y luego loops a través de dos matrices, escribiendo cada valor de la matriz en una nueva línea del archivo. El archivo se cierra cuando el loop ha terminado.
