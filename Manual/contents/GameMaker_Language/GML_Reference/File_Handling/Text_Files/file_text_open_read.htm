file_text_open_read

Esta función abre el archivo de texto con el nombre de archivo indicado sólo para su lectura, devolviendo el id único del archivo que debe ser almacenado en una variable ya que será utilizado para todas las acciones posteriores que tengan que ver con ese archivo. Si el archivo no existe, la función devolverá el valor -1.

NOTA: Sólo puede tener un máximo de 32 archivos abiertos al mismo tiempo. También debe cerrar siempre los archivos cuando haya terminado, ya que esto escribe la información y libera la memoria asociada al archivo.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

file_text_open_read(fname);

ArgumentoTipo Descripción
fname The name of the file to read from.

Devuelve:

Text File ID or -1

Ejemplo:

file = file_text_open_read(working_directory + "level.txt");

Esto abriría "level.txt" del mismo directorio que el juego y almacenaría el id del archivo en la variable "file".
