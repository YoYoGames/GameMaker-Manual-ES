file_text_open_from_string

Esta función creará un archivo de texto a partir de un string y lo abrirá para su lectura, devolviendo el "handle" del archivo que deberá ser utilizado en todas las llamadas posteriores a la función de archivo para leer de este archivo. Tenga en cuenta que este archivo es temporal y de sólo lectura, y como tal será eliminado de la memoria en el momento en que se cierre.

NOTA: Sólo puede tener un máximo de 32 archivos abiertos al mismo tiempo. También debe cerrar siempre los archivos cuando haya terminado, ya que esto escribe la información y libera la memoria asociada al archivo.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

file_text_open_from_string(string);

ArgumentoTipo Descripción
string The string to create the file from.

Devuelve:

File ID or -1

Ejemplo:

file = file_text_open_from_string(reset_str);

El código anterior toma el string almacenado en la variable "reset_str" y lo utiliza para crear un archivo de texto de sólo lectura. El "handle" de este archivo se almacena entonces en la variable "file" para que todas las funciones de archivo posteriores lo utilicen.
