file_text_open_append

Esta función abre el archivo de texto con el nombre de archivo indicado para su escritura (si el archivo no existe, se crea), devolviendo el id único del archivo que debe ser almacenado en una variable, ya que se utilizará para todas las acciones posteriores que tengan que ver con ese archivo. La posición dentro del fichero para escribir se establece en la última línea de texto que contenga el fichero, por lo que no se sobrescribe ningún dato existente.

Tenga en cuenta que si el archivo no puede ser creado (debido a un nombre de archivo ilegal, por ejemplo) la función devolverá -1.

NOTA: Sólo puede tener un máximo de 32 archivos abiertos al mismo tiempo. También debe cerrar siempre los archivos cuando haya terminado, ya que esto escribe la información y libera la memoria asociada al archivo.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

file_text_open_append(fname);

Argumento Tipo Descripción
fname Cadena The name of the file to append to.

Devuelve:

Text File ID or -1

Ejemplo:

file = file_text_open_append(working_directory + "save.txt");

Esto abriría "save.txt" desde el mismo directorio del juego y almacenaría el id del archivo en la variable "file".
