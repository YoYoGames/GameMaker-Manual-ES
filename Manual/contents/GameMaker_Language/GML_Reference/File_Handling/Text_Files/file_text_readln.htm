file_text_readln

Con esta función se puede saltar el resto de la línea actual de un archivo de texto abierto y pasar al comienzo de la siguiente. La función también devolverá la línea completa como string, lo que la convierte en una forma fácil de leer "trozos" completos de datos para analizarlos posteriormente.

Sintaxis:

file_text_readln(fileid);

ArgumentoTipo Descripción
fileid The id of the file to read from.

Devuelve:

Ejemplo:

var file = file_text_open_read(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i)
{
    scr[i] = file_text_read_real(file);
    file_text_readln(file);
    scr_name[i] = file_text_read_string(file);
    file_text_readln(file);
}
file_text_close(file);

El código anterior abre un archivo para su lectura y luego loops a través de las líneas del archivo leyendo alternativamente un valor numérico real y un string en dos matrices diferentes para su uso futuro. El archivo se cierra cuando el loop ha terminado.
