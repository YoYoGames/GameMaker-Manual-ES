file_text_open_write

Esta función abre el archivo de texto con el nombre de archivo indicado sólo para escribir (si el archivo no existe, se crea), devolviendo el id único del archivo que debe ser almacenado en una variable ya que se utilizará para todas las acciones posteriores que tengan que ver con ese archivo.

Tenga en cuenta que si el archivo no puede ser creado (debido a un nombre de archivo ilegal, por ejemplo) la función devolverá -1.

Si el archivo ya existe, el uso de esta función hará que se sobrescriba cuando se guarden los datos. Si desea añadir datos a un archivo manteniendo intacto su contenido, utilice file_text_open_append().

NOTA: Sólo puede tener un máximo de 32 archivos abiertos al mismo tiempo. También debe cerrar siempre los archivos cuando haya terminado, ya que esto escribe la información y libera la memoria asociada al archivo.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

file_text_open_write(fname);

Argumento Tipo Descripción
fname Cadena The name of the file to write to.

Devuelve:

Text File ID or -1

Ejemplo:

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

El código anterior abrirá el archivo "level.txt" para escribir y luego escribirá el string almacenado en la variable "level_data" antes de cerrar finalmente el archivo de nuevo.
