file_text_write_string

Con esta función puede escribir un string en un archivo de texto previamente abierto. Si el archivo ya contiene información, esta información se borrará y el string se escribirá al principio del archivo, a menos que haya abierto el archivo con la función file_text_open_append(). También puede evitarlo utilizando la función file_text_readln() junto con la función file_text_eof() para loop a través del contenido del archivo hasta llegar al final y luego comenzar a escribir.

Sintaxis:

file_text_write_string(fileid, str);

ArgumentoTipo Descripción
fileid The id of the file to edit.
str The string to write to the file.

Devuelve:

Ejemplo:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

El código anterior abre un archivo para escribir y luego loops a través de dos matrices, escribiendo cada valor de la matriz en una nueva línea del archivo. El archivo se cierra cuando el loop ha terminado.
