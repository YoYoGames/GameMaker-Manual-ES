file_bin_read_byte

Esta función devolverá un byte de datos desde la posición actual dentro del archivo con el ID de archivo dado. Usted proporciona el valor del ID del archivo, tal y como lo devuelve la función file_bin_open().

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_read_byte(binfile);

ArgumentoTipo Descripción
binfile The ID of the file to get the data from.

Devuelve:

Ejemplo:

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Esto abriría un archivo del mismo directorio que el juego, y obtendría un byte de datos de él antes de cerrarlo de nuevo.
