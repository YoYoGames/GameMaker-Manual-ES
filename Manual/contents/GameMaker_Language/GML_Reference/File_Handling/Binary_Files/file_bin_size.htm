file_bin_size

Esta función devolverá el tamaño (en bytes) de un archivo que ha sido abierto para lectura y/o escritura. El ID del archivo es el valor devuelto por la función file_bin_open().

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_size(binfile);

ArgumentoTipo Descripción
binfile The ID of the file to get the size of.

Devuelve:

Ejemplo:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file);

Esto abriría un archivo del directorio local del juego, y asignaría su índice a la variable "archivo". Luego obtendría el tamaño de ese archivo y lo cerraría de nuevo.
