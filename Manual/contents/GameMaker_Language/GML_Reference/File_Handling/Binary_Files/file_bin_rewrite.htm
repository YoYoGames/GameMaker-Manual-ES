file_bin_rewrite

Esta función toma el nombre del archivo devuelto por la función file_bin_open() y luego reescribe el archivo, borrando todos los datos anteriores para empezar a escribir desde el principio del archivo.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_rewrite(binfile);

ArgumentoTipo Descripción
binfile The ID of the file to rewrite.

Devuelve:

Ejemplo:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

Esto abriría un archivo del mismo directorio que el juego, y asignaría su índice a la variable "archivo". A continuación, se reescribiría el archivo (limpiándolo de datos).
