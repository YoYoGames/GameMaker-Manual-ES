file_bin_position

Esta función devuelve la posición actual en bytes, donde 0 es la primera posición, del archivo con el id de archivo dado. Usted proporciona el valor del ID del archivo, tal y como lo devuelve la función file_bin_open().

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_position(binfile);

ArgumentoTipo Descripción
binfile The ID of the file to get the position in.

Devuelve:

Ejemplo:

pos = file_bin_position(file);

Esto almacenaría la posición actual en la variable "pos".
