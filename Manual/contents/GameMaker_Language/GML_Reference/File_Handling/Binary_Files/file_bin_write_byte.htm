file_bin_write_byte

Esta función escribirá un byte de datos en el archivo identificado por el ID del archivo en la posición de escritura actual. Usted suministra el valor del ID del archivo, devuelto por la función file_bin_open() y el byte de datos a escribir.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_write_byte(binfile, byte);

ArgumentoTipo Descripción
binfile The ID of the file to write to.
byte The data to write.

Devuelve:

Ejemplo:

file_bin_write_byte(file, data);

Esto escribiría un byte en el archivo seleccionado.
