file_bin_close

Cierra un archivo binario previamente abierto. Se suministra el valor del ID del archivo, tal y como lo devuelve la función file_bin_open().

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_close(binfile);

ArgumentoTipo Descripción
binfile The ID of the file to close.

Devuelve:

Ejemplo:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Esto abriría un archivo del mismo directorio que el juego, y asignaría su índice a la variable "archivo". A continuación, reescribiría el archivo (borrando los datos), y lo cerraría de nuevo.
