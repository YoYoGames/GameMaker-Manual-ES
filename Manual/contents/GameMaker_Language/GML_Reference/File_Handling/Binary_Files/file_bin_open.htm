file_bin_open

Esta función abrirá el archivo binario con el nombre indicado. El modo indica lo que se puede hacer con el archivo:

  • 0 = lectura
  • 1 = escritura
  • 2 = tanto la lectura como la escritura

Cuando el archivo no existe se crea, y la función devuelve el id del archivo que debe ser utilizado en las otras funciones. Puede abrir varios archivos al mismo tiempo (32 como máximo), pero no olvide cerrarlos una vez que haya terminado con ellos.

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.

Sintaxis:

file_bin_open(fname, mode);

Argumento Tipo Descripción
fname Cadena The name of the file to read from.
mode Real The indicator of what can be done with the file.

Devuelve:

Binary File ID

Ejemplo:

file = file_bin_open("myfile.bin", 2);

Esto abriría un archivo del mismo directorio que el juego, y asignaría su índice a la variable "archivo".
