cargar_csv

Esta función cargará un archivo en formato CSV y lo convertirá en una cuadrícula DS, devolviendo el valor de identificación único de la cuadrícula creada.

Su archivo CSV debe seguir la especificación CSV. Si su sintaxis CSV no es correcta, obtendrá resultados inesperados al cargar el archivo.

Sintaxis:

load_csv(filename)

Argumento Tipo Descripción
filename Cadena The name of the file to open (as a string)

Devuelve:

DS Grid ID

Ejemplo:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
    for (var j = 0; j < hh; j++;)
    {
        draw_text(xx, yy, file_grid[# i, j]);
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

El código anterior abrirá el archivo CSV dado y almacenará la cuadrícula DS devuelta en la variable "file_grid". Esta rejilla es entonces analizada en un par de for lops y el contenido es dibujado en la pantalla.
