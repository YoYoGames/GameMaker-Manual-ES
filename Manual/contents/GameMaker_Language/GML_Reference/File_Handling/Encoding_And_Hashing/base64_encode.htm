base64_encode

Esta función convertirá un string en un formato base64 codificado string. Este es un esquema de codificación comúnmente utilizado para cualquier medio que necesite ser almacenado o transferido a través de Internet como texto, y hace que la salida sea ilegible para el ojo humano. el ojo humano.

Sintaxis:

base64_encode(string)

ArgumentoTipo Descripción
string The string to encode.

Devuelve:

Ejemplo:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

El código anterior convertirá el string almacenado en "game_data" en un string codificado en base64 que se almacenará en un archivo de texto externo.
