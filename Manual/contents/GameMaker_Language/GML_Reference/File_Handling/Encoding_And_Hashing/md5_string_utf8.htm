md5_string_utf8

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se utiliza habitualmente para comprobar la integridad de los archivos y strings. Esta función tomará una entrada UTF-8 string (que tiene un número variable de bytes por carácter) y devolverá el hash MD5 de 32 caracteres hexadecimales que es único para ese string. De este modo, se puede generar una clave segura que se puede almacenar y utilizar para comprobar la integridad de la información que se envía a (o se recibe de) un server externo (por ejemplo).

NOTA: Hay dos formatos para la codificación MD5, UTF-8 y unicode. Ambos se proporcionan para facilitar la comunicación con diferentes configuraciones de server, pero el más común de usar es unicode.

NOTA: MD5 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

md5_string_utf8(string)

ArgumentoTipo Descripción
string The string to hash.

Devuelve:

Ejemplo:

var hash, str;
str = json_encode(hiscore_map);
hash = md5_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)

El código anterior codificará un mapa DS en un JSON string. A continuación, se genera un hash MD5 que se almacena en un archivo ini para que pueda utilizarse posteriormente para comprobar la integridad del JSON en caso de que se reciba la misma información más adelante desde server. El JSON se envía entonces.
