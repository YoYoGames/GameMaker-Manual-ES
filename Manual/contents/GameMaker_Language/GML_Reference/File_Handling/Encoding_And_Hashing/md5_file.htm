md5_file

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se utiliza habitualmente para comprobar la integridad de los archivos y strings. Esta función tomará el archivo dado y generará un MD5 único para ese archivo que puede ser almacenado para su uso posterior.

NOTA: MD5 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

md5_file(filename)

ArgumentoTipo Descripción
filename The file to generate the MD5 hash for.

Devuelve:

Ejemplo:

hash = md5_file(working_directory + "game_data.ini")

El código anterior generará un hash MD5 para el archivo especificado y almacenará el valor devuelto en la variable "hash".
