zip_unzip

Esta función abrirá un archivo zip almacenado y extraerá su contenido en el directorio dado. Tenga en cuenta que si no proporciona un path completo al directorio ZIP, se utilizará la raíz de la unidad actual, y si quiere colocarlo en un path relativo al directorio de trabajo del paquete del juego, entonces debe utilizar la variable working_directory como parte del path (el paths relativo usando "." o ".." no funcionará como se espera, por lo que debe evitarse). Tenga en cuenta también que el zip debe ser parte del paquete del juego (es decir, un archivo incluido) o haber sido descargado en el área de almacenamiento utilizando http_get_file().

La función devolverá un valor que indica el número de archivos extraídos, o devolverá 0 o menos si la extracción ha fallado.

Sintaxis:

zip_unzip(zip_file, target_directory)

ArgumentoTipo Descripción
zip_file The zip file to open
target_directory The target directory to extract the files to

Devuelve:

Ejemplo:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

El código anterior abrirá el archivo zip almacenado en el directorio "descargas" y extraerá su contenido al directorio "extraído" (creando ese directorio si no existe ya) y luego comprobará que la extracción ha sido correcta, mostrando un mensaje de depuración si falla.
