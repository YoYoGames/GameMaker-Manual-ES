base64_decode

Esta función convertirá un string codificado previamente usando el formato base64, en texto estándar. Base64 es un esquema de codificación comúnmente utilizado para cualquier medio que necesite ser almacenado o transferido a través de Internet como texto, y hace que la salida ilegible para el ojo humano.

Sintaxis:

base64_decode(string)

ArgumentoTipo Descripción
string The string to decode.

Devuelve:

Ejemplo:

var str, file;
str = base64_encode(game_data);
file = file_text_open_read("save.txt");
str = file_text_read_string(file); level_data = base64_decode(str);
file_text_close(file);

El código anterior abrirá un archivo de texto y leerá un string del mismo en la variable local "str". Este string se decodifica y el resultado se almacena en la variable de instancia "level_data".
