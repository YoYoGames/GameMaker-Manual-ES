archivo_shav1

En criptografía, SHA-1 es una función hash criptográfica diseñada por la Agencia de Seguridad Nacional de los Estados Unidos y se emplea en varias aplicaciones populares Git, donde se utiliza para comprobar los cambios en los archivos, y los protocolos TLS y SSL, PGP, SSH, S/MIME e IPsec. Esta función tomará un archivo de entrada y devolverá un compendio de mensajes de 160 bits en formato ASCII único para ese archivo que se utilizará para la verificación de la integridad en cualquier momento posterior.

NOTA: SHA-1 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

sha1_file(filename)

Argumento Tipo Descripción
filename Cadena The file to generate the sha1 hash for.

Devuelve:

String

Ejemplo:

hash = sha1_file(working_directory + "game_data.ini")

El código anterior generará un hash sha1 para el archivo especificado y almacenará el valor devuelto en la variable "hash".
