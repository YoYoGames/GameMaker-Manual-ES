ini_read_real

Puede utilizar esta función para leer un número de un archivo de datos ini. Los archivos ini se dividen en secciones y luego cada sección se divide en pares clave - valor. Por lo tanto, un archivo ini típico sería algo así:

Sintaxis:

ini_read_real(section, key, default);

ArgumentoTipo Descripción
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

Devuelve:

Ejemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Esto abrirá "savedata.ini" y establecerá la puntuación al valor bajo "save1" > "score" en él, luego cerrará el .ini de nuevo. Si no hay ningún valor en "save1" > "score", o no hay ningún archivo "savedata.ini" presente, la puntuación se establecerá en 0.
