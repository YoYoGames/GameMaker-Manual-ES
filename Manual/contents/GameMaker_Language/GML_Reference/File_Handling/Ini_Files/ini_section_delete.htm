ini_section_delete

Con esta función puede eliminar toda una sección de un archivo ini, lo que también eliminará todos los pares clave-valor que estén asociados a ella.

Sintaxis:

ini_section_delete(section);

ArgumentoTipo Descripción
section The section to delete.

Devuelve:

Ejemplo:

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

Este ejemplo abrirá "savedata.ini" y escribirá un valor en "save1" > "Score". A continuación, borrará la sección "save1" y cerrará el archivo .ini.
