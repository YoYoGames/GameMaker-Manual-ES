ini_close

Esta función debe ser llamada en el momento en que se termina de leer o escribir en cualquier archivo ini abierto. Si no utiliza la función después de haber utilizado cualquiera de las funciones de escritura ini, no se escribirá nada en el disco, ya que la información del archivo se mantiene en memoria hasta que se llama a esta función, que fuerza la escritura. Si intentas abrir un ini sin haber cerrado previamente otro (o el mismo) también obtendrás un error.

La función también devolverá un string con el archivo ini codificado en él. Este string puede guardarse en un server y/o utilizarse de nuevo junto con la función ini_open_from_string() para volver a crear el ini.

Sintaxis:

ini_close();

Devuelve:

String

Ejemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Esto abrirá "savedata.ini" y leerá un valor de la sección "save1" con la clave "score", luego cerrará el archivo ini. Si no hay un valor en "save1" llamado "score" o no hay un archivo "savedata.ini" presente, se devolverá 0 y se creará un nuevo archivo ini.
