ini_read_string

Puede utilizar esta función para leer un string (texto) de un archivo de datos ini. Los archivos ini se dividen en secciones y luego cada sección se divide en pares clave - valor. Así que un archivo ini típico tendría un aspecto similar al siguiente:

Sintaxis:

ini_read_string(section, key, default);

Argumento Tipo Descripción
section Cadena The section of the .ini to read from.
key Cadena The key within the relevant section of the .ini to read from.
default Cadena The string to return if a string is not found in the defined place (or the .ini file does not exist). Must be a string.

Devuelve:

String

Ejemplo:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

Esto abrirá "savedata.ini" y establecerá global.name en el string bajo "save1" > "Name" en él, y luego cerrará el .ini de nuevo. Si no hay string en "save1" > "Name", o no hay un archivo "savedata.ini" presente, global.name se establecerá en "Player1".
