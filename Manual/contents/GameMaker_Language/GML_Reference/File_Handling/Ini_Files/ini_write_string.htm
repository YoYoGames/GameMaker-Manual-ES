ini_write_string

Puede utilizar esta función para escribir un string (texto) en un archivo de datos ini. Los archivos ini se dividen en secciones y luego cada sección se divide en pares clave - valor. Por lo tanto, un archivo ini típico sería algo así:

Vale la pena señalar que el uso de comillas y caracteres de escape al escribir ini strings puede causar problemas al leer posteriormente los datos, ya que pueden ser truncados prematuramente. Por ejemplo, escribir esto será un problema:

ini_write_string("test2", "section", "hello \"Fritz\"");

Sintaxis:

ini_write_string(section, key, value);

Argumento Tipo Descripción
section Cadena The section of the .ini to write to.
key Cadena The key within the relevant section of the .ini to write to.
value Cadena The string to write to the relevant destination.

Devuelve:

N/A

Ejemplo:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

El código anterior abre un archivo ini para leer y escribir, luego escribe el string almacenado en la variable global "Nombre" en la sección "Guardar" con la clave "Reproductor" antes de cerrar el archivo de nuevo.
