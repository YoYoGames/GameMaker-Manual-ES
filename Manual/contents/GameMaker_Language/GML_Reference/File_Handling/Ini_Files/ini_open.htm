ini_open

Esto abre un archivo ini para leer y/escribir. Si el archivo ini_file no existe en la ubicación que está comprobando, GameMaker puede crear uno, pero sólo si escribe datos en él. Si sólo ha leído información del archivo ini, entonces los valores por defecto de la función de lectura serán devueltos, pero el archivo ini no se creará realmente.

Tenga en cuenta que sólo puede tener un archivo ini abierto al mismo tiempo y recuerde utilizar ini_close() una vez que haya terminado de leer/escribir desde el archivo .ini, ya que la información no se almacena realmente en el disco hasta entonces (también se almacena en la memoria hasta que se cierra el archivo). disco hasta entonces (también se almacena en la memoria hasta que se cierra el archivo).

¡ATENCIÓN! Esta función puede no funcionar como usted espera debido a que GameMaker está en un entorno aislado. Por favor, vea la sección sobre el Sistema de Archivos para más información.

Sintaxis:

ini_open(name);

ArgumentoTipo Descripción
name The filename for the .ini file.

Devuelve:

Ejemplo:

ini_open("Settings/savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Esto abrirá 'savedata.ini' y leerá el valor de la puntuación bajo la sección "save1" con la clave "score" en ella, luego cerrará el .ini de nuevo. Si no hay ningún valor en "save1", "score" o no hay ningún archivo "savedata.ini" presente, la puntuación se establecerá en 0 (el valor por defecto). Tenga en cuenta que el archivo ini se ha colocado en el subdirectorio "Settings", que es la carpeta que contiene el archivo ini en los archivos incluidos del navegador Asset.
