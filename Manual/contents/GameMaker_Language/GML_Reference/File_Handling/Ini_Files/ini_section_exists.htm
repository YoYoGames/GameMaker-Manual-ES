ini_section_exists

Esta función comprueba si una sección existe en el ini actualmente abierto y devolverá true si es así o false en caso contrario. Esta no es una comprobación necesaria para evitar errores, ya que, cuando una sección no existe, la lectura de una sección inexistente simplemente devolverá un valor por defecto, sin embargo puede ser útil para ver si un archivo ini ha guardado datos específicos.

Sintaxis:

ini_section_exists(section);

ArgumentoTipo Descripción
section The section in the open .ini file to check for.

Devuelve:

Ejemplo:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

Esto pondrá la variable global.savegame en "1" si NO se encuentra "save1". Con un for loop, esto podría utilizarse como una forma básica de calcular cuántas partidas guardadas hay para un juego (es decir, crear un sistema para múltiples partidas guardadas simultáneas para un juego).
