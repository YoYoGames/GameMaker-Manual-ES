ini_open_from_string

Esta función creará un archivo ini temporal a partir de un string y lo abrirá para leer/escribir. El string debe estar correctamente formateado como un archivo ini (es decir, con secciones, claves de saltos de línea y valores), de lo contrario el archivo ini no se creará correctamente. Tenga en cuenta que este archivo ini es temporal y se eliminará de la memoria en el momento en que se cierre, perdiendo cualquier información que se haya almacenado en él, sin embargo la función ini_close() devuelve un string del archivo ini completo que puede guardarse en un server o en el disco.

Sintaxis:

ini_open_from_string(string);

ArgumentoTipo Descripción
string The string containing all the ini information.

Devuelve:

N/A

Ejemplo:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

El código anterior establece un ini temporal para mantener la información de la string "str", luego lee del ini antes de cerrarlo de nuevo.
