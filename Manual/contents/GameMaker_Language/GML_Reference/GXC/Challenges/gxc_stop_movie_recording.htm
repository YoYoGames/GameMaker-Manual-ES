gxc_stop_movie_recording

Esta función se utiliza para detener una grabación GXC iniciada previamente con gxc_start_movie_recording(). También envía la grabación a la URL especificada en los argumentos, aunque el comportamiento es diferente según el tipo de string especificado:

  • Si especifica un nombre de archivo, la grabación se guardará con ese nombre en su carpeta de descargas
  • Si especifica una URL, la grabación se cargará en esa URL

El método de la URL debe utilizarse cuando la grabación debe cargarse en GXC, y el archivo debe guardarse localmente sólo para comprobar si la grabación se ha realizado correctamente.

Sintaxis:

gxc_stop_movie_recording(filename);

Argumento Tipo Descripción
filename Cadena The file name or URL where the recording is to be saved or uploaded

Devuelve:

N/A

Ejemplo:

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

El código anterior utiliza la biblioteca de la API de GXC para cargar una puntuación en el reto actualmente activo, y cuando se recibe una respuesta en su método de devolución de llamada, recupera la URL de carga del vídeo y carga la grabación en ella.
