gxc_resumir_grabación_de_película

Esta función se utiliza para reanudar una grabación GXC que fue previamente pausada con gxc_pause_movie_recording().

Sintaxis:

gxc_resume_movie_recording();

Devuelve:

Ejemplo:

if (keyboard_check_pressed(vk_enter))
{
    gxc_resume_movie_recording();
}

Esto reanudará la grabación de la película GXC actualmente en pausa si se pulsa la tecla enter.
