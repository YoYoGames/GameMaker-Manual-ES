Opera GX y GXC

Esta sección describe las funciones proporcionadas para la plataforma de destino Opera GX. Las funciones enumeradas en esta página le permiten grabar su pantalla y subirla a GXC junto con su puntuación del desafío, sin embargo, tenga en cuenta que se requiere el uso de la Biblioteca API GXC para lograr esto. Por favor, consulte la sección de Opera GX en el YoYo Games Helpdesk para obtener instrucciones sobre la implementación de la biblioteca GXC API y el trabajo con los desafíos.

Las siguientes funciones se ofrecen para cargar grabaciones de vídeo con desafíos:

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resumir_grabación_de_película
