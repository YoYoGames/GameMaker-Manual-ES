physics_particle_set_group_flags

Con esta función puede cambiar el grupo flags para un grupo de partículas. El valor de group es el que se devolvió al crear el grupo de partículas mediante la función physics_particle_group_end(), mientras que el flags es el valor de retorno de una combinación de las siguientes constantes:

Constante Descripción
phy_particle_group_flag_solid Un grupo de partículas sólidas impide que otros accesorios se alojen en su interior. En caso de que algo lo penetre, el grupo de partículas sólidas empuja el accesorio ofensivo hacia su superficie, lo que hace que un grupo de partículas sólidas posea una fuerza de repulsión especialmente fuerte.
phy_particle_group_flag_rigid Los grupos de partículas rígidas son aquellos cuya forma no cambia, incluso cuando colisionan con otros artefactos.

Estos grupos flags utilizan el enmascaramiento de bits para crear un valor de salida final que luego se comprueba para establecer las diferentes propiedades básicas del grupo, aparte de las establecidas por la partícula flags. Puedes usar ninguna, una o ambas de estas constantes usando el bitwise o "|" para enmascarar los bits apropiados (como se muestra en el ejemplo de abajo).

Sintaxis:

physics_particle_set_group_flags(group, flags)

Argumento Tipo Descripción
group Identificación del grupo de partículas físicas The particle group to set.
flags Física Grupo de partículas Flag Constante(s) The flags to set on the particle group.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

El código anterior creará una variable para almacenar el valor flags y luego lo utilizará para establecer el flags de todas las partículas creadas en el grupo indexado por la variable "grupo".
