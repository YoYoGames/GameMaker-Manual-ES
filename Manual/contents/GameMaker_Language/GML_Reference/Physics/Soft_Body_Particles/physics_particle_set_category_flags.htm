physics_particle_set_category_flags

Con esta función puede cambiar la partícula flags para una categoría de partículas. El valor de category es el que ha definido en la función physics_particle_create() o (si ha creado un grupo) physics_particle_group_begin(), mientras que el flags es el valor de retorno de una combinación de las siguientes constantes:

Constante Descripción
phy_particle_flag_water Las propiedades por defecto de una partícula de cuerpo blando.
phy_particle_flag_zombie Una partícula zombi es aquella que será destruida en un solo paso con todas las demás marcadas de esta manera.
phy_particle_flag_wall Esto define la partícula como estática, esencialmente creándola como un inamovible object en la simulación física, ya que permanecerán en una posición fija sin importar lo que colisione con ellas. Deberías usar este flag en lugar de establecer la densidad a 0.
phy_particle_flag_spring Las partículas del muelle producen el efecto de estar unidas entre sí, como si se tratara de un muelle. Las partículas creadas con este flag están "conectadas" por parejas, y cada partícula está conectada a la que estaba más cerca de ella en el momento de la creación. Una vez emparejadas, las partículas no cambian de "pareja" , y cuanto más lejos las separe una fuerza externa, mayor será la potencia con la que colisionarán cuando se retire esa fuerza externa. Obsérvese que, por mucho que las partículas emparejadas se alejen una de otra, la conexión entre ellas no se romperá.
phy_particle_flag_elastic Las partículas elásticas se deforman y también pueden rebotar cuando chocan con otros cuerpos rígidos en la simulación física.
phy_particle_flag_viscous Una partícula viscosa es aquella que se muestra "pegajosa", como el aceite. Las partículas viscosas se agrupan y se pegan más.
phy_particle_flag_powder Las partículas de polvo producen un efecto de dispersión como el que se puede ver con la arena o el polvo.
phy_particle_flag_tensile Las partículas de tracción se utilizan para producir el efecto de la tensión superficial, o la curvatura tensa en la superficie de un cuerpo de líquido. Pueden utilizarse, por ejemplo, para crear la tensión superficial que se observa en una gota de agua. Una vez que se rompe la tensión, las partículas rebotan como si fueran elásticas, pero también siguen atrayéndose entre sí. Como resultado, las partículas tienden a formar grupos al rebotar.
phy_particle_flag_colourmixing Las partículas que mezclan colores adoptan parte del color de otras partículas con las que colisionan. Ten en cuenta que si sólo una de las dos partículas que colisionan es de mezcla de colores, la otra partícula conserva su color anterior a la colisión.

Estos flags utilizan el enmascaramiento de bits para crear un valor de salida final que luego se comprueba para establecer las diferentes propiedades básicas de la categoría de partículas (siendo la propiedad base siempre la de phy_particle_flag_water). Por ejemplo, si se quiere simular un líquido viscoso con tensión superficial se utilizaría el enmascaramiento de bits o "|" para enmascarar los bits apropiados como se muestra en el ejemplo siguiente. De esta manera, puede establecer diferentes propiedades (aparte de las propiedades globales) para cada categoría de partícula creada.

Sintaxis:

physics_particle_set_category_flags(category, flags)

Argumento Tipo Descripción
category Real The particle category to set.
flags Física Partícula Flag Constante(s) The flags to set on the particle category.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_set_category_flags(1, flags);

El código anterior creará una variable para almacenar el valor de flags y luego lo utilizará para establecer el flags de todas las partículas creadas con el valor de categoría de 1.
