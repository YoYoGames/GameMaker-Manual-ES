physics_particle_draw_ext

Esta función dibujará un sprite en la posición de todas las partículas que compartan el mismo valor de categoría definido por el usuario, y que tengan flags que coincidan con los establecidos para la máscara. El valor de la máscara se define de la misma manera que se define el flags al crear las partículas (ver las funciones physics_particle_create() para ver un ejemplo), es decir, usando el bitwise o "|" para enmascarar los bits apropiados para cada flag.

NOTA: Esta función dibujará todas las partículas de la simulación física con flags que coincidan con las de la máscara, ya sean partículas individuales o grupos, si la categoría del usuario es la misma.

La partícula dibujada ignorará los valores de color y alfa establecidos al crearla, y en su lugar se dibujará utilizando los parámetros de escala, ángulo, color y alfa dados en esta función. Tenga en cuenta que esto es puramente para dibujar un sprite en la posición de la partícula y de ninguna manera afecta el comportamiento de la partícula.

Sintaxis:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Argumento Tipo Descripción
typemask Física Partícula Flag Constante(s) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Activo Sprite The sprite to use for each particle.
subimg Real The image_index del sprite a dibujar.
xscale Real The scale factor to apply to the sprite along the x axis.
yscale Real The scale factor to apply to the sprite along the y axis.
ang Real The angle (in degrees) to draw the sprite at.
col Color The colour to blend with the sprite (default is c_white).
alpha Real The alpha value to use when drawing (0 - 1).

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

El código anterior dibujará un sprite para todas las partículas de la categoría definida por el usuario "2" que tengan el mismo flags que la máscara dada, escalado por 2 en ambos ejes y con un alfa de 0,5.
