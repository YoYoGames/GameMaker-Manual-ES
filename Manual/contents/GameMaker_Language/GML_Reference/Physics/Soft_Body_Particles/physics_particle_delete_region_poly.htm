physics_particle_delete_region_poly

Con esta función puede eliminar (quitar) todas las partículas que caen dentro de los límites del área poligonal definida de la simulación física en el actual room. La función toma un archivo (previamente creado) ds_list() que contiene la posición x/y de cada punto del polígono, siendo las posiciones pares de la lista las coordenadas x, y las impares las coordenadas y, es decir: para un triángulo, su lista tendría seis entradas, siendo las entradas 0, 2 y 4 las coordenadas x y 1, 3 y 5 las coordenadas y. El polígono se utilizará entonces para eliminar todas las partículas que caen dentro del área definida.

Tenga en cuenta que el polígono definido debe tener al menos tres puntos, y como máximo 8.

Sintaxis:

physics_particle_delete_region_poly(pointList)

ArgumentoTipo Descripción
pointList A ds_list of points to use to create the polygon.

Devuelve:

Ejemplo:

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

El código anterior eliminará todas las partículas que se encuentren en el área poligonal definida por los puntos añadidos a la DS List.
