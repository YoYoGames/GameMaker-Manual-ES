physics_particle_group_box

Esta función establecerá la forma del grupo de partículas que se está creando. Primero debe haber comenzado la definición del grupo usando la función physics_particle_group_begin()y luego usar esta función para establecer la mitad aproximada de la anchura y la mitad de la altura del grupo en píxeles - aproximada porque la anchura y la altura exactas dependerán del tamaño de las partículas base, como se define en la función physics_particle_set_radius() ya que la simulación física intenta "encajar" tantas partículas como sea posible en la forma definida. Por último, hay que llamar a physics_particle_group_end() para crear el grupo de partículas en el room.

Sintaxis:

physics_particle_group_box(halfWidth, halfHeight)

ArgumentoTipo Descripción
halfWidth The half width of the box.
halfHeight The half height of the box.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_box(100, 100);
mLastGroup = physics_particle_group_end();

El código anterior almacena el flags para el tipo de partícula y las propiedades del grupo de partículas en variables y luego las utiliza para crear un grupo de partículas rectangular con lados de 200px en la posición del ratón.
