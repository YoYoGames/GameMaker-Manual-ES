physics_particle_group_get_data

Esta función devuelve varios datos sobre un grupo de partículas en la simulación física utilizando el flags dado como comprobado. El índice del grupo (su ID) es el devuelto por la función physics_particle_group_end()y el buffer utilizado debe haber sido creado previamente con la función buffer_create(). Debe ser del tipo "grow", con un tamaño aproximado al de los datos de retorno esperados. Los propios flags se establecen utilizando las constantes que se indican a continuación, y se utilizaría el bitwise o "|" para crear un único valor flag para obtener la información deseada.

Constante Descripción Tipo de datos
phy_particle_data_flag_typeflags El valor de flags para la partícula. buffer_u32
phy_particle_data_flag_position La posición x e y de la partícula. 2 x buffer_f32
phy_particle_data_flag_velocity La velocidad horizontal y vertical. 2 x buffer_f32
phy_particle_data_flag_colour El color y el valor alfa (hexadecimal). buffer_f32
phy_particle_data_flag_category La categoría de la partícula (como se definió al crear el grupo al que pertenece). buffer_u32

Sintaxis:

physics_particle_group_get_data(group, buffer, flags)

Argumento Tipo Descripción
group Identificación del grupo de partículas físicas The group index (ID) of the particle group to get the data from.
buffer ID del búfer The (previously created) buffer to use to store the data.
flags Datos de las partículas físicas Flag Constante(s) The flags to use to extract data about specific particle types.

Devuelve:

N/A

Ejemplo:

var count = physics_particle_group_count(gp);
var flags = phy_particle_data_flag_position | phy_particle_data_flag_colour;
if (count > 0)
{
    var buffer = buffer_create(count * 12, buffer_grow, 4);
    physics_particle_group_get_data(gp, buffer, flags);
    for (var n = 0; n < count; n++;)
    {
        var xx = buffer_read(buffer, buffer_f32);
        var yy = buffer_read(buffer, buffer_f32);
        var argb = buffer_read(buffer, buffer_u32);
        var alpha = (argb >> 24) & 255;
        draw_sprite_ext(sprBlob, 0, xx, yy, 1, 1, 0, c_green, alpha);
    }
    buffer_delete(buffer);
}

El código anterior obtiene la posición y la velocidad de cada partícula del grupo indexada por la variable "gp", almacena los datos de buffer en una serie de variables, y luego los utiliza para dibujar un sprite en la posición de cada partícula del grupo.
