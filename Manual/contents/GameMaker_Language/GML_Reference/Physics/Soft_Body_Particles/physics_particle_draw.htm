physics_particle_draw

Esta función dibujará un sprite en la posición de todas las partículas que compartan el mismo valor de categoría definido por el usuario, y que tengan flags que coincidan con los establecidos para la máscara. El valor de la máscara se define de la misma manera que se define el flags al crear la partícula (véase la función physics_particle_create() para ver un ejemplo), es decir, utilizando la función "| " para enmascarar los bits apropiados para cada flag.

NOTA: Esta función dibujará todas las partículas de la simulación física con flags que coincidan con las de la máscara, ya sean partículas individuales o grupos, si la categoría del usuario es la misma.

La partícula dibujada utilizará los valores de color y alfa establecidos al crearla, y girará con el momento angular de la partícula. Sin embargo, si desea escalar la sprite o cambiar cualquier otra de sus propiedades dibujadas, deberá utilizar la función physics_particle_draw_ext().

Sintaxis:

physics_particle_draw(typemask, category, sprite, subimg)

Argumento Tipo Descripción
typemask Física Partícula Flag Constante(s) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Activo Sprite The sprite to use for each particle.
subimg Real The image_index del sprite a dibujar.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

El código anterior dibujará un sprite para todas las partículas de la categoría "2" definida por el usuario que tengan el mismo flags que la máscara dada.
