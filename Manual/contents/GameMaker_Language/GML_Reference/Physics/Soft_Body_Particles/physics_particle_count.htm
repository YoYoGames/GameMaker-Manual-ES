physics_particle_count

Esta función devolverá el número de partículas que están activas en una física habilitada room.

Sintaxis:

physics_particle_count()

Devuelve:

Ejemplo:

if (physics_particle_count() < physics_particle_get_max_count())
{
    physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

El código anterior comprobará si hay menos del número máximo de partículas permitidas en el room, y si es así creará una más.
