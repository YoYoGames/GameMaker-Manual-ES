physics_particle_group_add_point

Esta función establece un punto en el room para definir la forma de un polígono que se utilizará para crear un grupo de partículas de cuerpo blando. Usted debe haber señalado previamente a GameMaker que va a definir una forma de polígono utilizando la función physics_particle_group_polygon() y luego utilizar esta función para definir los puntos individuales del polígono. Debe dar al menos tres puntos cuando defina la forma del polígono, pero puede dar hasta ocho, y la función permitirá la definición de polígonos cóncavos. Sin embargo, si genera un polígono con alguna cavidad, se ignorarán los puntos de su interior y se creará una forma convexa para el grupo de partículas.

Sintaxis:

physics_particle_group_add_point(x, y)

ArgumentoTipo Descripción
x The x position in the room for the point.
y The y position in the room for the point.

Devuelve:

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

El código anterior almacena el flags para el tipo de partícula y las propiedades del grupo de partículas en variables y luego las utiliza para crear un grupo de partículas poligonales de tres lados en la posición del ratón.
