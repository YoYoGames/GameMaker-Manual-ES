physics_particle_group_begin

Con esta función puede crear un grupo de partículas en un room. Un grupo de partículas es aquel en el que las partículas necesarias para crear una forma específica se crean todas a la vez como un grupo, lo que le permite crear cuerpos blandos simulados con varias propiedades establecidas por el flags utilizado. Las "banderas" son el valor de retorno de una combinación de las constantes de las partículas (véase la función physics_particle_create() para más detalles), mientras que los "groupFlags" son una combinación de las siguientes constantes de grupo (o 0 para no flags):

Física Grupo de partículas Flag Constante
Constante Descripción
phy_particle_group_flag_solid Un grupo de partículas sólidas impide que otros accesorios se alojen en su interior. En caso de que algo lo penetre, el grupo de partículas sólidas empuja el accesorio ofensivo hacia su superficie, lo que hace que un grupo de partículas sólidas posea una fuerza de repulsión especialmente fuerte.
phy_particle_group_flag_rigid Los grupos de partículas rígidas son aquellos cuya forma no cambia, incluso cuando colisionan con otros artefactos.

Estos grupos flags utilizan el enmascaramiento de bits para crear un valor de salida final que luego se comprueba para establecer las diferentes propiedades básicas del grupo, aparte de las establecidas por la partícula flags. Puedes usar ninguna, una o ambas de estas constantes usando el bitwise o "|" para enmascarar los bits apropiados (como se muestra en el ejemplo de abajo).

Aparte de flags, también se puede establecer la posición en la room para crear el grupo de partículas (la posición dada será el centro de masa del grupo terminado), la velocidad horizontal y vertical inicial, así como la velocidad angular del grupo, lo que permite crear grupos en movimiento y en rotación.

También puede establecer el color inicial y el alfa de todas las partículas del grupo, y los valores se utilizarán al dibujar las partículas mediante la función physics_particle_draw(). A continuación, se establece la fuerza de cohesión de las partículas del grupo, que es otra forma de decir la fuerza con la que se "pegan" unas a otras. La fuerza puede ser un valor entre 0 y 1, donde 0 es ninguna cohesión, y 1 es la máxima cohesión.

Finalmente tienes que dar la categoría para tu grupo de partículas. Este valor es un valor entero arbitrario que le das a cada grupo de partículas que quieres que tenga propiedades similares, y se puede utilizar para establecer flags más tarde para esa categoría específica, así como una serie de otras cosas.

NOTA: Puede utilizar cualquier valor entero para la categoría, excepto 0, que está reservado por la simulación y se utiliza para seleccionar todas las categorías en otras funciones (como para el dibujo).

Es importante notar que llamar a esta función no crea el grupo en la física room. Esta función simplemente comienza el proceso y establece las propiedades de las partículas base para el grupo. Para crear realmente el grupo de partículas hay que darle una forma (ver el ejemplo de abajo) y luego llamar a la función physics_particle_group_end()en cuyo momento se creará.

NOTA: El tipo de partícula flags que establezcas también influirá en la cohesión del grupo de partículas, de tal manera que si flag las partículas son de tipo agua (por ejemplo), cuando el grupo colisione con otro accesorio se romperá, mientras que si flag la partícula es de tipo resorte, entonces mantendrá su cohesión en la colisión.

Sintaxis:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Argumento Tipo Descripción
flags Física Partícula Flag Constante(s) The flags to set on the particle.
groupflags Física Grupo de partículas Flag Constante(s) The group flags to set on the particle group (see table at the top).
x Real The x position to create the particle group at.
y Real The y position to create the particle group at.
ang Real The angle (in degrees) to rotate the group to on create.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
ang_velocity Real The initial angular velocity.
col Color The base colour to use for the particle group.
alpha Real The base alpha to use for the particle group.
strength Real Set the cohesion strength between particles in the group.
category Real The user defined category to which the group belongs.

Devuelve:

N/A

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

El código anterior almacena el flags para el tipo de partícula y las propiedades del grupo de partículas en variables y luego las utiliza para crear un grupo de partículas circular con un radio de 100px en la posición del ratón.
