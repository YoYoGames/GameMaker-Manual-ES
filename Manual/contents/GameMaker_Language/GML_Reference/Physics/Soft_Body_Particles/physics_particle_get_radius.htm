physics_particle_get_radius

Con esta función puedes averiguar cuál es el radio actual (en píxeles) de las partículas en la simulación física (puedes establecer este valor usando physics_particle_set_radius()).

Sintaxis:

physics_particle_get_radius()

Devuelve:

Ejemplo:

if (physics_particle_get_radius() < 32)
{
    physics_particle_set_radius(physics_particle_get_radius() + 1);
}

El código anterior comprobará el radio actual de las partículas en la simulación y si es menor de 32, entonces aumentará su tamaño en 1.
