physics_particle_get_group_flags

Con esta función puedes recuperar el grupo flags para un grupo de partículas. El valor de group es el que se devolvió cuando se creó el grupo de partículas con la función physics_particle_group_end()y la función devolverá un valor que es el valor combinado del grupo flags.

Sintaxis:

physics_particle_get_group_flags(group)

Argumento Tipo Descripción
group Identificación del grupo de partículas físicas The particle group to get.

Devuelve:

Real

Ejemplo:

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

El código anterior creará una variable para almacenar el valor de flags y luego lo utilizará para comprobar el flags del grupo indexado en la variable "grupo". Si no son iguales, el grupo se establece con estos flags.
