physics_particle_group_end

Esta función se utiliza para finalizar la definición de una forma de grupo de partículas. La llamada a esta función creará las partículas dentro de los parámetros de la forma dada, y también devolverá un valor de ID que puede ser almacenado y utilizado en otras funciones para las interacciones con el grupo de partículas.

Sintaxis:

physics_particle_group_end()

Devuelve:

Physics Particle Group ID

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

El código anterior almacena el flags para el tipo de partícula y las propiedades del grupo de partículas en variables y luego las utiliza para crear un grupo de partículas circular con un radio de 100px en la posición del ratón. El ID del grupo creado se almacena en la variable "mLastGroup".
