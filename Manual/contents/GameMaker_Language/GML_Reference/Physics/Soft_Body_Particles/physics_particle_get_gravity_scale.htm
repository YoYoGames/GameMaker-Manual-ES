physics_particle_get_gravity_scale

Con esta función puedes averiguar cuál es el factor de escala de gravedad actual para las partículas en la simulación física (puedes establecer este valor usando physics_particle_set_gravity_scale()).

Sintaxis:

physics_particle_get_gravity_scale()

Devuelve:

Ejemplo:

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

El código anterior establecerá la escala de gravedad.
