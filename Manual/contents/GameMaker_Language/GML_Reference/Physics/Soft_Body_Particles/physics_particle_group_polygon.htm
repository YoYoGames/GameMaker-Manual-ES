physics_particle_group_polygon

Esta función establecerá la forma del grupo de partículas que se está creando. Primero debe haber comenzado la definición del grupo utilizando la función physics_particle_group_begin()y luego utilizar esta función para decirle a GameMaker que va a añadir en los puntos aproximados que se utilizarán para definir la forma del polígono - aproximado porque la forma exacta dependerá del tamaño de las partículas de base, como se define por la función physics_particle_set_radius() ya que la simulación física trata de "encajar" el mayor número de partículas posible en la forma definida.

A continuación se definirían los puntos del polígono llamando a la función physics_particle_group_add_point()y se establece la posición x/y en room para cada uno (mínimo de 3, máximo de 8, y pueden pueden ser definidos como cóncavos pero los puntos dentro de la forma serán ignorados). Finalmente hay que llamar a physics_particle_group_end() para crear el grupo de partículas en el room.

Sintaxis:

physics_particle_group_polygon()

Devuelve:

Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

El código anterior almacena el flags para el tipo de partícula y las propiedades del grupo de partículas en variables y luego las utiliza para crear un grupo de partículas poligonales de tres lados en la posición del ratón.
