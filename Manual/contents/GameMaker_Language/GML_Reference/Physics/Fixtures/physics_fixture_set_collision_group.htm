physics_fixture_set_collision_group

Una vez que hayas creado tu fixture, puedes definirlo como parte de un grupo de colisión. Esto significa que puedes decirle a las instancias con ese fixture que siempre colisionen o que nunca colisionen con otras instancias que tengan fixtures vinculados a ellas con el mismo grupo de colisión. Los valores que se pueden asignar se basan en números positivos o negativos, siendo 0 el valor por defecto (lo que significa que la instancia con esta configuración de fixture se ignora a menos que se cree un evento de colisión para ella). Aquí hay una imagen para ayudar a explicar:

Como puede ver, establecer grupos de colisión puede ser muy útil para crear cuerpos físicos complejos que interactúan con el mundo físico que los rodea, pero no entre sí. Sin embargo, hay que tener en cuenta que debido a la complejidad de calcular las colisiones basadas en grupos como este, es recomendable que intentes mantener el número de grupos en uso en cualquier room a un mínimo absoluto.

Los valores del grupo de colisión pueden oscilar entre -32.768 y 32.767 (ambos inclusive).

Sintaxis:

physics_fixture_set_collision_group(fixture, group);

Argumento Tipo Descripción
fixture ID de la instalación física The fixture that is to be bound
group Real The group that this fixture is assigned to (either a positive value, 0, or a negative value)

Devuelve:

N/A

Ejemplo:

physics_fixture_set_collision_group(fix_Wheel, -1);

El código anterior establecerá la fijación indexada en la variable "fix_Wheel" para ser parte del grupo "-1".
