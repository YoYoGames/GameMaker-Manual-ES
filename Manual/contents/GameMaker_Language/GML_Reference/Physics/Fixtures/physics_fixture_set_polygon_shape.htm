physics_fixture_set_polygon_shape

Esta función establece la forma de un polígono para la fijación, pero tendrá que utilizar physics_fixture_add_point() para definir la forma de este polígono en relación con el origen del dispositivo. El polígono se cierra cuando el fixture está ligado a una instancia. También hay que tener en cuenta que esta función debe llamarse antes de definir cualquier punto, y también debe tener al menos tres puntos definidos para su polígono antes de vincularlo a una instancia o se producirá un error.

Sintaxis:

physics_fixture_set_polygon_shape(fixture)

ArgumentoTipo Descripción
fixture the index of the fixture

Devuelve:

Ejemplo:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

El código anterior aplicará una forma poligonal a la fijación indexada en la variable "fix_Ship" y luego define tres puntos para crear una forma triangular.
