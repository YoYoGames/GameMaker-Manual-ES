physics_set_restitution

Cuando se vincula un fixture a una instancia utilizando physics_fixture_bind() esta función devuelve un "id" para el fixture vinculado. Puedes utilizar este id para establecer el valor de restitución del fixture vinculado, no del fixture "base", en cualquier momento utilizando esta función. La restitución se establece normalmente como un valor entre 0 y 1, pero se pueden utilizar valores más altos si es necesario, aunque los resultados pueden ser impredecibles.

Sintaxis:

physics_set_restitution(fixture, restitution)

ArgumentoTipo Descripción
fixture the id of the bound fixture
restitution the new restitution value to apply

Devuelve:

Ejemplo:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

El código anterior obtiene el valor de restitución actual para las propiedades físicas vinculadas de la instancia y luego las establece en un valor diferente.
