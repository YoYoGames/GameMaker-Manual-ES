physics_get_restitution

Cuando se vincula un fixture a una instancia utilizando physics_fixture_bind() esto devuelve un "id" para el fixture vinculado. Puedes usar este id para obtener el valor de restitución (la propiedad "bounciness") del accesorio vinculado, no del accesorio "base", en cualquier momento utilizando esta función.

Sintaxis:

physics_get_restitution(fixture)

ArgumentoTipo Descripción
fixture the id of the bound fixture

Devuelve:

Ejemplo:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

El código anterior obtiene el valor de restitución actual para las propiedades físicas vinculadas de la instancia y luego las establece en un valor diferente.
