physics_fixture_set_edge_shape

Esta función define una forma de fijación de "borde". Una forma de borde es simplemente una línea que generará una colisión cuando otras fijaciones la sobrepasen, y puede ser muy útil para generar (por ejemplo) terreno, o para crear bordes alrededor de un room. La posición del borde se define usando el espacio local, es decir: la posición x/y de la instancia se considera (0,0), por lo que esto debe tenerse en cuenta al crearlas (en el ejemplo de código de abajo, la instancia se habría colocado en (0,0) en el room para evitar complicaciones).

Sintaxis:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

ArgumentoTipo Descripción
fixture the index of the fixture
local_x1 start x position for the edge
local_y1 start y position for the edge
local_x2 end x position for the edge
local_y2 end y position for the edge

Devuelve:

Ejemplo:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
    var _fix = physics_fixture_create();
    physics_fixture_set_edge_shape(_fix, _xx, _y1, _xx + 50, _y2);
    physics_fixture_bind(_fix, id);
    physics_fixture_delete(_fix);
    _xx += 50;
    _y1 = y2;
    _y2 = room_height - 50 - irandom(100);
}

El código anterior creará una línea de luminarias de "borde" con una variedad de alturas a lo largo de la room.
