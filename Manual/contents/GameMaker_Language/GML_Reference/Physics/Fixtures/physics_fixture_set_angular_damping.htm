physics_fixture_set_angular_damping

Si piensas en cualquier object en rotación en el "mundo real", a menos que tenga un motor o esté en el espacio, se ralentiza con el tiempo debido a la influencia de fuerzas externas (como la fricción con el aire que le rodea). Podemos utilizar la función physics_fixture_set_angular_damping() en GameMaker para simular este efecto y reducir la velocidad de rotación de las instancias en el mundo de la física, ya que, sin ella, cualquier instancia en rotación seguiría girando infinitamente. Los parámetros de amortiguación deben estar entre 0 e infinito, 0 significa que no hay amortiguación, e infinito significa amortiguación total. Normalmente se utiliza un valor de amortiguación entre 0 y 1, pero se puede utilizar cualquier valor no negativo si es necesario.

Sintaxis:

physics_fixture_set_angular_damping(fixture, damping)

ArgumentoTipo Descripción
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

Devuelve:

Ejemplo:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

El código anterior establecerá la amortiguación angular de la fijación indexada en "fix_ball" a 0,1.
