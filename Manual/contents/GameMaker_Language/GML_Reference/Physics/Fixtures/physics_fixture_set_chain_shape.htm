physics_fixture_set_chain_shape

Esta función define una forma de fijación "en cadena". Una fijación en cadena se compone de un número de puntos, que se conectan entre sí utilizando formas de borde para unirlos. La función en sí toma el índice (ID) de la fijación y puede decirle a la fijación que loop o no. Si se establece en true se conectará el último punto con el primer punto con una arista, mientras que si se establece en false no lo hará. Tenga en cuenta que esta función prepara el dispositivo para aceptar los puntos necesarios para hacer la cadena, y estos deben ser añadidos después de llamar a esta función utilizando physics_fixture_add_point()de la misma manera que cuando se construye un polígono.

Al crear una fijación de cadena, debes darle al menos dos puntos, pero no estás limitado en el número de puntos subsiguientes que puedes añadir para conformar la fijación terminada.

Sintaxis:

physics_fixture_set_chain_shape(fixture, loop)

ArgumentoTipo Descripción
fixture The index of the fixture
loop Whether to close the chain (true) o no (false).

Devuelve:

Ejemplo:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(_fix, _xx, _yy);
    _xx += 50 + random(150);
    _yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

El código anterior creará una línea de accesorios de "borde" con una variedad de alturas a lo largo de la room.
