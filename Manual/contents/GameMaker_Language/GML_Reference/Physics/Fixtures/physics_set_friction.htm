physics_set_friction

Cuando se vincula un fixture a una instancia utilizando physics_fixture_bind() esto devuelve un "id" para el fixture vinculado. Puedes usar este id para establecer el valor de fricción del fixture vinculado, no del fixture "base", en cualquier momento usando esta función. Tenga en cuenta que la fricción se establece normalmente en un valor entre 0 y 1, pero puede utilizar cualquier valor no negativo si es necesario.

Sintaxis:

physics_set_friction(fixture, friction)

ArgumentoTipo Descripción
fixture the id of the bound fixture
friction the new friction value to apply

Devuelve:

Ejemplo:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

El código anterior obtiene el valor de fricción actual para las propiedades físicas vinculadas de la instancia y luego las establece a un valor diferente.
