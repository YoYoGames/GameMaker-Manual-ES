physics_remove_fixture

Esta función elimina (o "desvincula") un accesorio de una instancia o instancias. Requiere el "id" único del fixture vinculado (devuelto por la función physics_fixture_bind() y eliminará todas las propiedades físicas definidas actualmente para la instancia, permitiéndole redefinir un nuevo fixture y vincularlo a la instancia. De este modo, puede cambiar las propiedades físicas de la instancia sin tener que destruirla y volver a crearla.

Sintaxis:

physics_remove_fixture(id, fixture)

Argumento Tipo Descripción
id ID de la instancia The ID of the instance to remove the fixture from
fixture ID de la instalación física The ID of the fixture that is to be removed from the instance

Devuelve:

N/A

Ejemplo:

physics_remove_fixture(id, my_fix);

El código anterior eliminará el accesorio con el "id" almacenado en la variable "mi_fijo" de la instancia.
