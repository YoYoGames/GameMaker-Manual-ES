physics_fixture_bind

Una vez que hayamos definido nuestro fixture, éste debe estar vinculado a una instancia. Esto significa que sus propiedades se transfieren a la instancia seleccionada, no al propio fixture, de modo que un fixture puede estar ligado a múltiples instancias si todas deben tener las mismas propiedades. Puede especificar un índice object para el objetivo y todas las instancias presentes en el room en ese momento recibirán las propiedades de ese fixture (pero no las nuevas instancias del object creadas posteriormente), o puede utilizar las palabras clave especiales other y all. Incluso se puede especificar un padre object y todas las instancias hijas con ese padre también recibirán el fixture. Una vez que el fixture ha sido vinculado a todas las instancias que necesitas, puede ser eliminado si ya no es necesario y las instancias con las propiedades de ese fixture no se verán afectadas y mantendrán esas propiedades.

La fijación se vinculará a la instancia con el centro de masa situado en el origen de la instancia, y las fijaciones poligonales se vinculan en función de la posición de los puntos en relación con el origen. Si necesitas que tu fixture se vincule a un punto distinto del origen, deberás utilizar physics_fixture_bind_ext().

La función también devolverá un valor "id" único para el fixture vinculado(no el fixture en sí) que puede utilizarse para eliminar ("desvincular") las propiedades físicas de la instancia utilizando la función physics_remove_fixture(). Esto permite añadir y eliminar propiedades físicas de una instancia sin destruir y volver a crear objects.

NOTA: Los accesorios deben ser eliminados cuando ya no se necesiten, ya que si no se hace así se puede producir una fuga de memoria que ralentizará y acabará por bloquear el juego.

Sintaxis:

physics_fixture_bind(fixture, target)

Argumento Tipo Descripción
fixture ID de la instalación física The fixture that is to be bound
target ID de la instancia o activo del objeto The target instance that is to receive the fixture (can be an instance id, an object id, other, or all)

Devuelve:

Real

Ejemplo:

var fix, inst;
fix = physics_fixture_create();
physics_fixture_set_circle_shape(fix, 16);
physics_fixture_set_density(fix, 1.0);
inst = instance_create_layer(x, y, "Instances", genericBodyObject);
my_fix = physics_fixture_bind(fix, inst);
physics_fixture_delete(fix);

El código anterior creará un fixture y asignará su índice a la variable "fix". A continuación, define la forma y la densidad del fixture antes de vincularlo a la instancia que se creó con el índice del fixture vinculado almacenado en la variable "mi_fijo". Por último, se elimina el fixture para evitar fugas de memoria, ya que no es necesario.
