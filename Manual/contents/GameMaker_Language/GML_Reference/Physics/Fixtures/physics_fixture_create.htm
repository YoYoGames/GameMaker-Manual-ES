physics_fixture_create

El primer paso para configurar un fixture es crearlo con esta función. El índice que se devuelve debe almacenarse en una variable que se utilizará en todas las funciones posteriores que se utilicen para definir y utilizar esta fijación.

Sintaxis:

physics_fixture_create()

Devuelve:

Physics Fixture ID

Ejemplo:

fix_Ball = physics_fixture_create();

El código anterior creará un accesorio y almacenará su índice en la variable "fix_Ball".
