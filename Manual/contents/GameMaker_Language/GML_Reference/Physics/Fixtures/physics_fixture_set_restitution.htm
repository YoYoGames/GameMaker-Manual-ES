physics_fixture_set_restitution

En física, la restitución se define como "el retorno de un object o sistema a su estado original después de una deformación elástica", pero como los fixtures en el GameMaker son realmente cuerpos rígidos y no pueden ser deformados, la restitución es restitución es una forma de decir cuán "rebotante" es el accesorio. Este ajuste afectará a cuánto "rebota" un object cuando colisiona con otros objects y es co-dependiente de otras fuerzas que actúan sobre la instancia como la gravedad y la fricción, y suele ser un valor entre 0 y 1 (se pueden utilizar valores más altos pero pueden dar resultados imprevisibles). Aquí hay una ilustración de cómo funciona:

Sintaxis:

physics_fixture_set_restitution(fixture, restitution)

ArgumentoTipo Descripción
fixture The index of the fixture
restitution The restitution of the fixture (usually between 0 and 1)

Devuelve:

Ejemplo:

physics_fixture_set_restitution(fix_Ball, 0.9);

El código anterior establecerá la restitución del accesorio indexado en "fix_ball" a 0,9.
