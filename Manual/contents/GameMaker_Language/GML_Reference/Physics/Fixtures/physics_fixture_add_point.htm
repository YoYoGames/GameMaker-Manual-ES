physics_fixture_add_point

Esta función define los puntos de un polígono que ha sido previamente fijado a la fijación utilizando physics_fixture_set_polygon_shape() o puede utilizarse para añadir puntos a una cadena de bordes utilizando la función physics_fixture_set_chain_shape(). Para una forma poligonal, los puntos son relativos al origen del fixture, siendo la posición 0,0 del eje x/y el centro, y las coordenadas subsiguientes se calculan como es habitual en GameMaker, siendo abajo/derecha +x,+y y arriba/izquierda -x,-y. Aquí hay una ilustración para ayudarle a visualizar este proceso:

Como puede ver, cualquier polígono convexo simple puede ser modelado de esta manera, pero los polígonos cóncavos no están soportados en este momento. Por último, para cerrar el polígono, debe vincular la fijación a una instancia. También debe tener en cuenta que debe tener al menos tres puntos definidos para su polígono antes de vincularlo a una instancia o recibirá un error y que añadir un punto a una fijación que no tiene una forma de polígono establecida previamente también causará un error. También hay que tener en cuenta que el número máximo de puntos permitido para una fijación es de ocho.

Para las formas de cadena (que son simplemente una serie de formas de borde conectadas) esta función establecerá un punto como "conexión" para encadenar los bordes. Por lo tanto, es necesario haber definido al menos dos puntos antes de vincular la fijación a la instancia de lo contrario, se producirá un error. Sin embargo, a diferencia de lo que ocurre con los polígonos, no está limitado a un máximo de ocho puntos y puede añadir tantos como necesite para crear la cadena, y los puntos se definen en coordenadas reales de room, en lugar de relativas a la instancia.

Sintaxis:

physics_fixture_add_point(fixture, xpos, ypos)

ArgumentoTipo Descripción
fixture the index of the fixture
xpos the x position of the point (relative to the fixture origin for polygon shapes)
ypos the y position of the point (relative to the fixture origin for polygon shapes)

Devuelve:

Ejemplo:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

El código anterior aplicará una forma poligonal a la fijación indexada en la variable "fix_Ship" y luego define tres puntos para crear una forma triangular.
