physics_test_overlap

Esta función puede utilizarse para comprobar si un cuerpo físico (es decir, la fijación de una instancia) se solapa, o se solapará, cuando se rote y se coloque en una posición determinada en la página room. El argumento "angle" es el ángulo de rotación que tiene (o tendrá) la instancia que se llama en la posición que se va a comprobar, y el argumento "obj" puede ser un único id de instancia, y el índice object o el palabras clave todo u otro.

Sintaxis:

physics_test_overlap(xpos, ypos, angle, obj);

Argumento Tipo Descripción
xpos Real The x position in the room to check
ypos Real The y position in the room to check
angle Real The angle to check (of the calling instance)
obj Activo del objeto o ID de la instancia The object to check for

Devuelve:

Boolean

Ejemplo:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

El código anterior comprobará una posición para un solapamiento de la fijación de la física, y si hay uno, establece una variable y una alarma.
