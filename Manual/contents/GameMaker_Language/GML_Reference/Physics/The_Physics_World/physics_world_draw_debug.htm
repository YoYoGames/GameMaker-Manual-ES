physics_world_draw_debug

Esta función dibujará una representación del sistema de física del mundo de room para ayudar con debugging cuando tengas problemas para que la escena de física funcione correctamente. A veces el problema puede estar en la parte de renderizado de tu juego, por ejemplo un sprite puede ser dibujado en la posición o rotación equivocada, dando la apariencia de una física incorrecta, o puede ser que tus cálculos no sean correctos. En cualquier caso, esta función te ayudará a encontrar y resolver el problema dibujando lo que ocurre en tu sistema de física. Para hacer esto, necesitas decirle qué parte del sistema dibujar, y esto se hace pasando un valor creado a partir de los bits almacenados en varias constantes "flag":

Bandera de depuración de la física
Bandera Descripción
phy_debug_render_aabb Esto muestra el cuadro delimitador absoluto de cada accesorio en relación con el eje room
phy_debug_render_collision_pairs Esto mostrará cualquier accesorio que esté actualmente en colisión
phy_debug_render_coms Esto marca el centro de masa de cada accesorio en la habitación
phy_debug_render_core_shapes Muestra las formas básicas que componen los accesorios de la habitación
phy_debug_render_joints Esto dibujará cada una de las uniones de todos los accesorios de la habitación
phy_debug_render_obb Esto muestra el cuadro delimitador relativo para los accesorios en la sala
phy_debug_render_shapes Esto muestra las formas reales que componen los accesorios dentro de la habitación

Por lo tanto, para dibujar dos de esas opciones tendríamos que hacer una "o" a nivel de bits (en GameMaker esto se representa con | ) en las dos y almacenar la variable resultante para que la función la utilice. Aquí hay un ejemplo de cómo se establecería una variable para utilizar con la función que dibujará sólo los centros de masa y las articulaciones de todos los accesorios en una habitación:

flag = phy_debug_render_coms | phy_debug_render_shapes;

Lo que te daría algo como esto:

Como puede ver con el código anterior, sólo se dibujan en la pantalla las formas y los centros de masa (así como los colores que representan sus estados - para más información consulte la documentación de Box 2D en línea). De la misma manera puedes mostrar más, o incluso todas, las constantes que desees depurar... ¡Simplemente "o" todas juntas! Hay que tener en cuenta una cosa más - esta función debe ser llamada sólo en el evento de dibujar de una instancia que tenga una profundidad inferior a las que desea depurar o de lo contrario no podrá ver la información de depuración.

Sintaxis:

physics_world_draw_debug(flag)

Argumento Tipo Descripción
flag Bandera de depuración de la física(s) The flag value for showing different debug features

Devuelve:

N/A

Ejemplo:

flag = phy_debug_render_aabb | phy_debug_render_collision_pairs | phy_debug_render_obb;
physics_world_draw_debug(flag);

El código anterior dibujará todas las cajas delimitadoras absolutas, las cajas delimitadoras relativas, así como los accesorios en colisión dentro del sistema de física actual.
