physics_draw_debug

Al crear un sistema de física en GameMaker, a menudo es importante tener una representación visual de lo que realmente está sucediendo con una instancia. Esto se puede lograr mediante el uso de physics_draw_debug(), que dibuja un esquema de las propiedades físicas asociadas a la instancia que ejecuta el código. Aquí hay una imagen de cómo se ve:

Hay que tener en cuenta que para que esto funcione debe estar en el evento de dibujo de la y se dibujará utilizando el color de dibujo definido actualmente.

Sintaxis:

physics_draw_debug()

Devuelve:

Ejemplo:

draw_set_colour(c_red);
physics_draw_debug();

El código anterior dibujará una representación de las propiedades físicas asociadas a esa instancia utilizando el color rojo.
