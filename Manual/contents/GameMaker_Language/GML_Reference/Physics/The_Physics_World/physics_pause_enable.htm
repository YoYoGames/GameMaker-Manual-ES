physics_pause_enable

Normalmente, para que una simulación de física funcione, debe ser continua y no puede ser detenida e iniciada, o tener instancias movidas repentinamente de un lugar a otro en el room. Sin embargo, hay momentos en los que necesitas pausar la simulación ya que (por ejemplo) el dispositivo os ha pausado, y entonces usarías esta función. Esta pausa la simulación si el flag se establece en true y no se harán más cálculos de física hasta que el flag se establezca en false de nuevo.

NOTA: Esto es especialmente útil si desea desactivar todas las instancias en un room, ya que incluso cuando se desactiva un cuerpo físico seguirá siendo calculado y simulado en el mundo de la física.

Sintaxis:

physics_pause_enable(flag)

ArgumentoTipo Descripción
flag This can be set to true para pausar la simulación, o false para iniciarla de nuevo.

Devuelve:

Ejemplo:

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

El código anterior comprueba si el sistema operativo ha sido pausado y si lo ha sido entonces pausa el mundo de la física, desactiva todo excepto a sí mismo, y luego crea una instancia del menú de pausa.
