phy_speed_x

Esta variable puede utilizarse para obtener o cambiar el componente x del vector de velocidad lineal de la instancia y se define en píxeles por paso (para píxeles por segundo, véase phy_linear_velocity_x). Si se modifica esta variable para una instancia estática (es decir, una instancia con densidad 0) se convertirá en una instancia cinemática.

Sintaxis:

phy_speed_x

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

if phy_speed_x != 0
{
    phy_speed_x = 0;
}

El código anterior comprobará el componente x del vector de velocidad lineal y si es distinto de 0 lo pondrá a 0.
