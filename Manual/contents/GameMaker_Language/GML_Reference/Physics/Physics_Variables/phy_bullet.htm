phy_bullet

Esta variable define si la instancia se mueve extremadamente rápido (por ejemplo una bala). El valor por defecto es false pero si se establece en true esto le dice a GameMaker que la instancia se moverá a una velocidad tan alta que requerirá una detección de colisiones más costosa para asegurar que no pase a través de otras instancias sin ser detectada

Sintaxis:

phy_bullet;

Devuelve:

(or undefined if the instance is not physics enabled)

Ejemplo:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

El código anterior crea una nueva instancia y luego la define como una "bala" en el mundo de la física antes de darle un impulso masivo a lo largo del eje y.
