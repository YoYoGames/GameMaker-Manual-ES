phy_linear_damping

Esta variable se puede utilizar para establecer la amortiguación lineal de la instancia, o se puede utilizar para obtener la amortiguación lineal actual. La amortiguación es la cantidad de "resistencia" al movimiento hacia adelante que tiene la instancia habilitada por la física, con un valor menor que permite que la instancia se mueva y acelere más rápido y un valor mayor que requiere un empuje más fuerte

Sintaxis:

phy_linear_damping;

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_linear_damping = 10;
}
else
{
    phy_linear_damping = 3;
}

El código anterior comprobará si hay una colisión entre la instancia que llama y las instancias de "obj_Water" y cambiará la amortiguación lineal en consecuencia.
