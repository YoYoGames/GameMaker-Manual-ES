phy_active

Esta variable controla si la instancia está actualmente "activa" o no. Si se establece en false, la instancia no participará en el mundo de la física, y si se establece en true, volverá a participar. Por favor, ten en cuenta que esto no es lo mismo que desactivar la instancia, ya que la instancia sigue siendo visible en la pantalla y todavía se puede cambiar a través del código, sino que esta función sólo impide que participe en la simulación de la física

Sintaxis:

phy_active

Devuelve:

(or undefined if the instance is not physics enabled)

Ejemplo:

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause;
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

El código anterior detectará una pulsación de la letra "P" y entonces cambiará la variable global "Pause" de true a false y viceversa. Esta variable se utiliza entonces para establecer si la física está activa o no en las instancias hijas del object indexadas en la variable "obj_Parent".
