phy_linear_velocity_x

Esta variable puede utilizarse para obtener o cambiar la componente x del vector de velocidad lineal de la instancia y se define en píxeles por segundo (para píxeles por paso, véase phy_speed_x). La modificación de esta variable para una instancia estática (es decir, una instancia con densidad 0) la convertirá en una instancia cinemática

Sintaxis:

phy_linear_velocity_x

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

if (phy_linear_velocity_x != 0)
{
    phy_linear_velocity_x = 0;
}

El código anterior comprobará la componente x del vector de velocidad lineal y si es distinta de 0 la pondrá a 0.
