phy_kinematic

Esta variable de sólo lectura devolverá true si la instancia está clasificada como cinemática object, o false si no lo está. Una instancia cinemática es aquella que tiene una masa infinita (una densidad de 0) pero que puede moverse. Por lo tanto, para hacer que una instancia sea cinemática, primero debes crear una instancia estática y luego establecer una o más de las variables de instancia relacionadas con el movimiento (por ejemplo phy_speed_x, phy_speed_yo phy_angular_velocity)

Sintaxis:

phy_kinematic;

Devuelve:

(or undefined if the instance is not physics enabled)

Ejemplo:

if !phy_kinematic
{
    phy_speed_x = 5;
}

El código anterior comprueba si la instancia es cinemática y si no lo es, establece la velocidad horizontal en 5.
