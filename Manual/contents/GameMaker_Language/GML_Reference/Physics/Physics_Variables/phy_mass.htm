phy_mass

Esta variable de sólo lectura devuelve la masa de la instancia en kilogramos. Este valor se calcula automáticamente en base a la superficie de los artefactos asignados y sus valores de densidad, pero puede modificarse mediante la función physics_mass_properties().

Sintaxis:

phy_mass

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

El código anterior es del evento de colisión de la instancia con otra y compara la masa de cada instancia y destruye la que tiene menos masa.
