phy_fixed_rotation

Esta variable se puede utilizar para establecer si la instancia puede ser afectada por fuerzas de rotación o no (por defecto es false). Si se establece en true, ninguna fuerza externa (ya sea de impulsos o fuerzas codificadas, o de colisiones) afectará el valor de rotación de la instancia y esto tendría que ser establecido manualmente usando la phy_rotation variable

Sintaxis:

phy_fixed_rotation;

Devuelve:

(or undefined if the instance is not physics enabled)

Ejemplo:

phy_rotation = 0;
phy_fixed_rotation = true;

El código anterior switch la instancia para tener una rotación fija, a continuación, establecer el ángulo de rotación.
