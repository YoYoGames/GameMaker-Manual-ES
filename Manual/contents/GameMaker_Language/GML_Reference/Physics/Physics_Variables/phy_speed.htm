velocidad_física

Esta variable de sólo lectura devuelve la velocidad actual de la instancia de física habilitada, definida en píxeles por paso. Si necesitas cambiar este valor, debes hacerlo cambiando los vectores x e y usando las variables phy_speed_x y phy_speed_y.

Sintaxis:

phy_speed;

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

El código anterior comprueba la velocidad de la instancia habilitada por la física y luego cambia la amortiguación lineal basada en el valor devuelto.
