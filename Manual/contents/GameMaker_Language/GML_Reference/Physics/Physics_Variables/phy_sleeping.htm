phy_sleeping

Esta variable de sólo lectura devuelve si la instancia está actualmente "durmiendo" (true) o no (false), Una instancia "durmiente" es aquella que no está participando activamente en ninguna simulación física. GameMaker pondrá objects a dormir para ahorrar ciclos de simulación cuando una instancia está en reposo y no está en colisión con otra instancia

Sintaxis:

phy_sleeping;

Devuelve:

(or undefined if the instance is not physics enabled)

Ejemplo:

if phy_sleeping
{
    instance_destroy();
}

El código anterior comprueba si el object está siendo simulado activamente o no y si no lo está se destruye.
