phy_col_normal_x

Esta variable de sólo lectura devuelve la componente x de la normal de colisión correspondiente al phy_collision_x valor de la matriz. Para cada punto de contacto hay una normal de contacto asociada (que suele ser la misma normal para todos los puntos de contacto de la colisión). Esta normal de contacto es un vector unitario que apunta de una instancia de la colisión a otra, y puede utilizarse, por ejemplo, para calcular la dirección correcta de "empuje" para resolver las colisiones

NOTA: Esta variable sólo está disponible en el evento de colisión de una instancia con física habilitada.

Sintaxis:

phy_col_normal_x;

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

El código anterior utiliza la normal de contacto para establecer la dirección de movimiento de una instancia creada en el evento de colisión entre dos instancias habilitadas por la física.
