phy_collision_points

Esta variable de sólo lectura devuelve el número de puntos de colisión detectados entre los dos objects en la colisión

NOTA: Esta variable sólo está disponible en el evento de colisión de una instancia con física habilitada.

Sintaxis:

phy_collision_points;

Devuelve:

(integer value, or undefined if the instance is not physics enabled)

Ejemplo:

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
}

El código anterior crea partículas en todos los puntos definidos de una colisión entre dos instancias habilitadas para la física.
