phy_angular_velocidad

Esta variable se puede utilizar para establecer la velocidad angular de la instancia, o se puede utilizar para obtener la velocidad angular actual, en grados por segundo y el valor utilizado puede ser positivo (para la rotación en sentido de las agujas del reloj) o negativo (para la rotación en sentido contrario). Si se establece esto en una instancia que previamente era estática (es decir, tiene una densidad de 0) se convertirá en una cinemática object y comenzará a girar

Sintaxis:

phy_angular_velocity;

Devuelve:

(single precision floating point value, or undefined if the instance is not physics enabled)

Ejemplo:

if abs(phy_angular_velocity) > 0
{
    phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
}
else
{
    phy_angular_velocity = 0;
}

El código anterior comprobará la velocidad angular de la instancia y si no es 0 entonces añadirá (o restará) una pequeña cantidad en cada paso hasta que el valor sea 0.
