physics_apply_force

Cuando se trabaja con la física es necesario aplicar no sólo la gravedad a objects, sino también las fuerzas. Una fuerza es cualquier influencia que hace que un object sufra un cambio de velocidad, un cambio de dirección o un cambio de forma, y se define en el mundo de la física de GameMaker igual que definimos la gravedad, obteniendo el vector de las dos componentes xforce/yforce. Hay que tener en cuenta que las fuerzas no se aplican en el centro de masa de object, sino en un punto de room y que cambiarán la dirección y la velocidad, así como el par de torsión (giro), dependiendo de cualquier otra fuerza que esté actuando en object (como la gravedad). Aquí hay una imagen de ejemplo para ayudar a ilustrar:

En el ejemplo anterior el muelle object en la parte inferior ha creado una fuerza sobre la bola object haciendo que se mueva en una dirección diferente. La fuerza exacta está definida por el vector que obtenemos de los componentes xforce/yforce en relación a las coordenadas xpos/ypos - lo que significa simplemente que la fuerza se calcula como la distancia desde xpos/ypos a xforce/yforce en Newtons (un Newton se define como la cantidad de fuerza neta requerida para acelerar una masa de un kilogramo a un ritmo de un metro por segundo al cuadrado) y la dirección es el ángulo que obtenemos desde xpos/ypos a xforce/yforce. El código para una colisión como la anterior iría en el evento de colisión del muelle object y se vería algo así como lo proporcionado en el ejemplo de abajo.

NOTA: El mundo de la física utiliza metros para todos sus cálculos, y la proporción de píxeles por metro se define en el editor room o utilizando physics_world_create().

Sintaxis:

physics_apply_force(xpos, ypos, xforce, yforce)

Argumento Tipo Descripción
xpos Real The x coordinate in the room where the force will be applied
ypos Real The y coordinate in the room where the force will be applied
xforce Real The x component of the force vector
yforce Real The y component of the force vector

Devuelve:

N/A

Ejemplo:

with (other)
{
    physics_apply_force(x, y, 0, -30);
}

El código anterior aplicará el código al otro object en la colisión con una fuerza de 30 Newtons y una dirección.
