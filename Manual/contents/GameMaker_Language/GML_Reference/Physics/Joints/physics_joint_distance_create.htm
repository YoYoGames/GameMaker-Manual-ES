physics_joint_distance_create

Una de las uniones más sencillas es la de distancia, que establece que la distancia entre dos puntos de dos instancias debe ser constante. Cuando se especifica una articulación de distancia, las dos instancias deben estar ya creadas y tener una fijación asignada, entonces se definen los dos puntos de anclaje en coordenadas room. El primer punto de anclaje se conecta a la instancia 1, el segundo punto de anclaje se conecta a la instancia 2 y la distancia entre estos puntos implica la longitud de la restricción de distancia. La imagen siguiente muestra cómo funciona esto:

Como puede ver, los puntos de anclaje se especifican como coordenadas de room, por lo que debe tener cuidado al definirlos, especialmente si las instancias se están creando al mismo tiempo que las articulaciones en lugar de colocarse en el room a través del editor room. También debe tener en cuenta que las articulaciones se crean independientemente del tamaño de sprite de las instancias o de las fijaciones que tienen adjuntas. Por lo tanto, si crea una junta de distancia en algún lugar distinto al origen de la instancia, seguirá siendo válida y limitará las dos instancias en relación con la posición en la que fue creada. Si estableces el valor de "col" en true entonces las dos instancias pueden interactuar y colisionar entre sí pero sólo si tienen eventos de colisión, sin embargo si se establece en false, no colisionarán pase lo que pase.

Sintaxis:

physics_joint_distance_create(inst1, inst2, w_anchor1_x, w_anchor1_y, w_anchor2_x, w_anchor2_y, col)

Argumento Tipo Descripción
inst1 ID de la instancia The first instance to connect with the joint
inst2 ID de la instancia The second instance to connect with the joint
w_anchor1_x Real The first x coordinate for the joint, within the game world
w_anchor1_y Real The first y coordinate for the joint, within the game world
w_anchor2_x Real The second x coordinate for the joint, within the game world
w_anchor2_y Real the second y coordinate for the joint, within the game world
col Booleano Whether the two instances can collide (true) or not (false)

Devuelve:

Physics Joint ID

Ejemplo:

var mainFixture, o_id;
mainFixture = physics_fixture_create();
physics_fixture_set_circle_shape(mainFixture, sprite_get_width(sprite_index) / 2);
o_id=instance_create_layer(x+300, y, "Instances", obj_Rudder);
physics_fixture_bind(mainFixture, id);
physics_fixture_bind(mainFixture, o_id);
physics_joint_distance_create(id, o_id, x + 50, y, o_id.x - 50, o_id.y, 0);
physics_fixture_delete(mainFixture);

El código anterior crea y define un nuevo fixture y luego crea una instancia de "obj_Rudder". A continuación, se asigna el fixture a la instancia que está ejecutando el código, así como a la recién creada, y se crea una unión entre ellas. Por último, se elimina el fixture porque ya no es necesario.
