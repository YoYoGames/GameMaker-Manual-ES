physics_joint_get_value

Utilizando una serie de constantes predefinidas, puedes pedirle a GameMaker que te diga una serie de cosas sobre el estado de cualquier articulación. Esto es muy útil, ya que le da la capacidad de eliminar las articulaciones o cambiar el comportamiento de una instancia en función de lo que sus necesidades son en el momento. Hay una serie de constantes que se pueden utilizar en esta función y se pueden encontrar aquí: Physics Joint Constants, pero tenga en cuenta que se realizan cálculos complejos cuando los llama, por lo que deben ser utilizados con cuidado y sólo cuando sea necesario y tenga en cuenta que muchos son únicos para un tipo específico de articulación.

Si la propiedad no existe (si, por ejemplo, se comprueba la traslación prismática de una articulación de polea), el valor de retorno será 0.

Sintaxis:

physics_joint_get_value(joint, value)

Argumento Tipo Descripción
joint Física ID conjunta The index of the joint that you wish to test
value Constante de unión física The constant for the joint property that you wish to test

Devuelve:

Real

Ejemplo:

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

El código anterior comprueba si la variable "shipJoint" contiene un índice de articulación y, si es así, calcula la fuerza que se aplica a esa articulación utilizando las dos constantes. Finalmente, si la fuerza total es mayor que 2, se elimina la articulación.
