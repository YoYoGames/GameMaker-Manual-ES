physics_joint_delete

Una vez que dos instancias con representaciones físicas han sido unidas por una junta, ésta puede ser borrada de nuevo en cualquier momento. Normalmente esto ocurrirá automáticamente cuando una de las dos instancias sea destruida, o cuando el room termine, pero hay veces en las que usted puede desear hacer esto manualmente. En esos casos se utilizaría esta función.

También hay que tener en cuenta que siempre que se destruya una instancia que forme parte de una unión de engranajes, la unión de engranajes debe ser eliminada utilizando esta función antes que cualquiera de las instancias implicadas en la formación del engranaje (pero el resto de uniones se eliminarán automáticamente), por ejemplo en el evento de destrucción de la instancia.

Sintaxis:

physics_joint_delete(joint)

Argumento Tipo Descripción
joint Física ID conjunta The index of the joint that you wish to delete

Devuelve:

N/A

Ejemplo:

var reactionForceX, reactionForceY, reactionForce;
if (shipJoint)
{
    reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

El código anterior comprueba si la variable "shipJoint" contiene un índice de articulación y, si es así, calcula la fuerza que se aplica a esa articulación utilizando las dos constantes. Finalmente, si la fuerza total es mayor que 2, se elimina la articulación.
