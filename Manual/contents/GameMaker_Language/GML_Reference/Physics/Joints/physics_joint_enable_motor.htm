physics_joint_enable_motor

Cuando tienes una articulación con un motor(prismático o revolucionado), puedes querer ser capaz de switch encender o apagar el motor dependiendo de variables y condiciones dentro del juego. Para esto, necesitas haber almacenado el índice de la articulación previamente en una variable y luego puedes switch encender o apagar el motor usando esta función y estableciendo el argumento "motor" a true o false.

Sintaxis:

physics_joint_enable_motor(joint, motor)

Argumento Tipo Descripción
joint Física ID conjunta The joint that you wish to enable or disable the motor on
motor Booleano Whether you wish to turn the motor on (true) or off (false)

Devuelve:

N/A

Ejemplo:

var t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, sprite_get_width(sprite_index) / 2);

var o_id = instance_create_layer(x+300, y, "Instances", obj_Door);
physics_fixture_bind(t_fix, id);
physics_fixture_bind(t_fix, o_id);

perma_joint = physics_joint_revolute_create(id, o_id, x+25, y, -90, 90, 1, 10, 2, 0, 0);
physics_joint_enable(perma_joint, 1);
physics_fixture_delete(t_fix);

El código anterior crea y define un nuevo fixture y luego crea una instancia de "obj_Puerta", vinculando el fixture creado a las dos instancias. A continuación, se unen mediante una junta revolutiva sin motor y con los ángulos limitados a una oscilación de +/- 90 grados, y almacenamos el índice de la junta en la variable "perma_joint". A continuación, switch enciende el motor utilizando esta variable, antes de borrar finalmente la articulación de la memoria, ya que no es necesaria.
