xboxlive_achievement_load_friends

Esta función enviará una solicitud a server para obtener información sobre todos los amigos de los usuarios conectados y activará una llamada de retorno Social Asynchronous Event que contiene el async_load con los pares clave/valor relevantes. La clave id de este DS Map se utiliza para identificar la devolución de llamada correcta (puede haber más de una función de activación para cualquier evento asíncrono dado), y se emparejará con la constante xboxlive_achievement_friends_info así como con una serie de otros pares clave/valor para cada amigo. El contenido exacto del mapa se muestra a continuación:

  • "id" - Para esta función debe ser xboxlive_achievement_friends_info
  • "FriendN" - El nombre del amigo, donde "N" es un valor entero correspondiente a su posición dentro de la lista de amigos.
  • "FriendidN" - El id de usuario único del amigo, "N".

Sintaxis:

xboxlive_achievement_load_friends();

Devuelve:

N/A

Ejemplo ampliado:

El siguiente código se llamaría probablemente después de que el jugador haya entrado en su cuenta de juego para obtener una lista de todos los amigos de ese usuario:

xboxlive_achievement_load_friends();

Esto enviará una solicitud de información sobre los amigos de los usuarios y generará una devolución de llamada asíncrona con el mapa especial async_load DS que contiene los siguientes datos:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_friends_info
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

El código anterior comprueba el DS Map devuelto en el Evento Social Asíncrono y si su "id" coincide con la constante que se está comprobando, entonces loops a través del mapa almacenando todos los diferentes valores en una serie de arrays.
