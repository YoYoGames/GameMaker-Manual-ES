xboxlive_fire_event

Esta función se puede utilizar para disparar un evento de estadísticas. El argumento "event_name" es el nombre del evento a disparar tal y como está definido en la consola XDP de su juego, y los siguientes parámetros adicionales también dependerán de lo que tenga configurado para la stat. La función devolverá 0 si el evento fue enviado (y debe ser recibido/procesado por el server) o -1 si hubo un error (es decir, su evento no fue configurado como el archivo de manifiesto de eventos incluido en el proyecto dice otro número).

Sintaxis:

xboxlive_fire_event(event_name, [stat_parameters...]);

Argumento Tipo Descripción
event_name Cadena The name of the event to fire.
stat_parameters Variable OPTIONAL Parámetros adicionales para añadir al evento.

Devuelve:

Real

Ejemplo:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

El código anterior obtiene cierta información sobre el usuario y luego lanza un evento de estadísticas junto con estos datos (y otros adicionales).
