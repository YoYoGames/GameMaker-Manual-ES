xboxlive_achievement_load_leaderboard

Esta función enviará una petición a server para obtener información sobre la tabla de clasificación dada. La tabla de clasificación debe haber sido creada previamente en el panel de control del desarrollador para su juego y el ID dado es el que debe suministrar como primer argumento de la función. Se activará una llamada de retorno Social Asynchronous Event que contiene el async_load mapa rellenado con los pares clave/valor relevantes. La clave de id de este DS Map se utiliza para identificar la devolución de llamada correcta (puede haber más de una función de activación para cualquier evento asíncrono dado), y se emparejará con la constante xboxlive_achievement_leaderboard_info así como con una serie de otros pares clave/valor para cada jugador. El contenido exacto del mapa se muestra a continuación:

  • "id" - Para esta función debe ser xboxlive_achievement_leaderboard_info
  • "leaderboardid" - El ID único de la tabla de clasificación tal y como se define en el panel del proveedor.
  • "numentries" - El número de entradas en la tabla de clasificación que has recibido.
  • "JugadorN" - El nombre del jugador, donde "N" es un valor entero correspondiente a su posición dentro de la lista de entradas de la tabla de clasificación.
  • "PlayeridN" - El id de usuario único del jugador, "N".
  • "RangoN" - El rango del jugador "N" dentro de la tabla de posiciones.
  • "PuntuaciónN" - La puntuación del jugador "N".

Puedes especificar la parte exacta de la tabla de clasificación que se mostrará dando un valor de índice mínimo y máximo, así como solicitar que los resultados se filtren para recuperar sólo las puntuaciones de tus amigos o las publicadas por todos los jugadores utilizando las siguientes constantes de filtro:

Filtro de logros de Xbox Live constante
Constante Descripción
xboxlive_achievement_filter_friends_only  Obtén sólo las puntuaciones de los jugadores amigos.
xboxlive_achievement_filter_all_players  Obtén todas las puntuaciones de todos los jugadores.

Sintaxis:

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Argumento Tipo Descripción
ident Cadena The unique ID of the leaderboard as shown on the developer dashboard.
minindex Real The starting index value to get the leaderboard data from.
maxindex Real The maximum index value to get the leaderboard data from.
filter Filtro de logros de Xbox Live constante Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Devuelve:

N/A

Ejemplo ampliado:

El siguiente código se llamaría probablemente después de que el jugador haya entrado en su cuenta de juego para obtener una lista de todas las posiciones de los amigos en la tabla de clasificación del juego:

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Esto enviará una solicitud de información sobre la tabla de clasificación dada y generará una devolución de llamada asíncrona con el mapa especial async_load DS que contiene los siguientes datos:

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_leaderboard_info)
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

El código anterior comprueba el DS Map devuelto en el Evento Social Asíncrono y si su "id" coincide con la constante que se está comprobando, entonces comprueba que la tabla de clasificación que se está enviando coincide con la llamada a la función original antes de hacer un bucle a través del mapa y almacenar todos los diferentes valores en una serie de arrays.
