xboxlive_stats_flush_user

Esta función se puede utilizar para vaciar los datos de las estadísticas de un usuario determinado desde el gestor de estadísticas, a la página web server, asegurando que server está al día con los valores actuales. De acuerdo con la documentación de XBox, los desarrolladores deben tener cuidado de no llamar a esta función con demasiada frecuencia, ya que XBox limitará la velocidad de las solicitudes, y el sistema operativo de XBox también vaciará automáticamente las estadísticas aproximadamente cada 5 minutos.

NOTA: La eliminación del usuario puede devolver un error si las estadísticas que se han establecido en el usuario no son válidas (como los nombres de las estadísticas que contienen caracteres numéricos no alfabéticos).

Para utilizar la función, se proporciona el ID de usuario devuelto por (por ejemplo) la función xboxlive_get_user()y un valor de prioridad (que será 1 para prioridad alta y 0 para prioridad baja). La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito. La función también generará una llamada de retorno que activará un Evento Asíncrono del Sistema. Este evento tendrá el mapa especial DS async_load que deberá ser analizado para las siguientes claves:

  • "id" - Mantendrá la constante achievement_stat_event
  • "eventname" - Mantendrá el string "StatisticUpdateComplete"
  • "userid" - El ID de usuario asociado a la solicitud
  • "error" - 0 si tiene éxito, algún otro valor si ha habido un error
  • "errormessage" - Un string con un mensaje de error, si hay alguno disponible

Tenga en cuenta que las estadísticas se vaciarán automáticamente cuando elimine un usuario del gestor de estadísticas. Consulte xboxlive_stats_remove_user() para más información.

Sintaxis:

xboxlive_stats_flush_user(user_id, priority);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The user ID (a pointer) to use
priority Real The prority value (0 or 1)

Devuelve:

Real

Ejemplo:

for(var i = 0; i < array_length(user_id); ++i;)
{
    xboxlive_stats_flush_user(user_id[i], 0);
}

El código anterior loops a través de los IDs de cuentas de usuario almacenados y luego vacía los datos del gestor de estadísticas para cada usuario.
