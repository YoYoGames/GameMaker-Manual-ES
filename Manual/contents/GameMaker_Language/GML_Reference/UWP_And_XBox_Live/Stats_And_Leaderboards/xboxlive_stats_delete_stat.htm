xboxlive_stats_delete_stat

Esta función se puede utilizar para eliminar una estadística del gestor de estadísticas para el ID de usuario dado. Se suministra el ID de usuario devuelto por (por ejemplo) la función xboxlive_get_user()y la estadística string que se desea eliminar. Esto borra el valor de la estadística y lo elimina del gestor de estadísticas, lo que significa que ya no será devuelto por las funciones xboxlive_stats_get_stat_names() o xboxlive_stats_get_stat().

La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito

Sintaxis:

xboxlive_stats_delete_stat(user_id, stat);

ArgumentoTipo Descripción
user_id The user ID pointer to delete the stat of
stat The statistic to delete (a string)

Devuelve:

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

El código anterior loops a través de los usuarios conectados y borra la estadística especificada del gestor de estadísticas para cada uno.
