xboxlive_achievements_set_progress

Esta función puede utilizarse para actualizar el progreso de un logro. Se suministra el ID de usuario devuelto, por ejemplo, por la función xboxlive_get_user()y luego la cadena del logro (este es el ID del logro tal y como se asignó en el XDP/Windows Dev Center cuando se creó). Por último, se establece el valor de progreso a un valor de 0 a 100. Tenga en cuenta que el sistema de logros rechazará las actualizaciones que sean inferiores al valor de progreso actual actual.

Sintaxis:

xboxlive_achievements_set_progress(user_id, achievement, progress);

ArgumentoTipo Descripción
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

Devuelve:

Ejemplo ampliado:

El siguiente es un ejemplo ampliado de cómo se puede utilizar esta función. Para empezar, la llamarías en algún evento como Inicio de Sala o Creación:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

El código anterior se llamaría para obtener una lista de todas las posiciones de la tabla de clasificación global para el juego, y generará una llamada de vuelta al evento social asíncrono que trataríamos de la siguiente manera:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

El código anterior crea un valor porcentual basado en algunas variables globales y luego lo utiliza para establecer el progreso del logro "Game_Completed".
