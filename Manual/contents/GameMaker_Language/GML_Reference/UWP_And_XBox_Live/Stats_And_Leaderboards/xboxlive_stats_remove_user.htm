xboxlive_stats_remove_user

Esta función se puede utilizar para eliminar (desregistrar) a un usuario determinado del gestor de estadísticas, realizando un vaciado de los datos de las estadísticas al live server. De acuerdo con la documentación de XBox, el juego no tiene que eliminar al usuario del gestor de estadísticas, ya que el sistema operativo de XBox vaciará periódicamente las estadísticas de todos modos. Si proporcionas el ID del usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y la función devolverá -1 si hubo un error o si el ID de usuario es inválido, o cualquier otro valor si la función fue llamada con éxito.

NOTA: La eliminación del usuario puede devolver un error si las estadísticas que se han establecido en el usuario no son válidas (como los nombres de las estadísticas que contienen caracteres numéricos no alfabéticos).

La función generará una llamada de retorno que activará un Evento Asíncrono del Sistema. Este evento tendrá el mapa especial DS async_load que deberá ser analizado en busca de las siguientes claves:

  • "id" - Mantendrá la constante achievement_stat_event
  • "eventname" - Mantendrá el string "LocalUserRemoved"
  • "userid" - El ID de usuario asociado a la solicitud
  • "error" - 0 si tiene éxito, algún otro valor si ha habido un error
  • "errormessage" - Un string con un mensaje de error, si hay alguno disponible

Tenga en cuenta que si quiere vaciar los datos de las estadísticas en la página web server en cualquier momento sin eliminar al usuario, puede utilizar la función xboxlive_stats_flush_user().

Sintaxis:

xboxlive_stats_remove_user(user_id);

ArgumentoTipo Descripción
user_id The user ID (a pointer) to remove

Devuelve:

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

El código anterior loop a través de los IDs de cuentas de usuario almacenados y eliminar (desregistrar) cada usuario del gestor de estadísticas.
