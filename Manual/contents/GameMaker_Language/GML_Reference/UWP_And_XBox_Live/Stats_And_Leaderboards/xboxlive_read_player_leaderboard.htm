xboxlive_read_player_leaderboard

La función devolverá la información de la tabla de clasificación para el nombre de usuario dado. La tabla de posiciones debe haber sido creada previamente en el tablero XDP para su juego, y la llamada a la función activará un evento social asíncrono que contiene el mapa async_load mapa rellenado con los pares clave/valor relevantes. La clave "id" de este mapa DS se utiliza para identificar la devolución de llamada correcta (puede haber más de una función de activación para cualquier evento asíncrono dado), y se emparejará con la constante achievement_leaderboard_info así como una serie de otros pares clave/valor. El contenido exacto del mapa se muestra a continuación:

  • "id" - Para esta función debe ser achievement_leaderboard_info
  • "leaderboardid" - El ID único de la tabla de clasificación tal y como se define en el panel del proveedor.
  • "numentries" - El número de entradas en la tabla de clasificación que has recibido.
  • "PlayerN" - El nombre del jugador, donde "N" es un valor entero correspondiente a su posición dentro de la lista de entradas de la tabla de clasificación.
  • "PlayeridN" - El id de usuario único del jugador, "N".
  • "RangoN" - El rango del jugador "N" dentro de la tabla de clasificación.
  • "PuntuaciónN" - La puntuación del jugador "N".


La función requiere que se dé una de las siguientes constantes para establecer las propiedades del filtro:

Constante Descripción
xboxlive_achievement_filter_all_players Consigue todas las puntuaciones.
xboxlive_achievement_filter_friends_only Obtenga sólo las puntuaciones de los amigos, en orden ascendente.
xboxlive_achievement_filter_favorites_only Obtener sólo las puntuaciones de los favoritos, en orden ascendente
xboxlive_achievement_filter_friends_alt Obtener sólo las puntuaciones de los amigos en orden descendente.
xboxlive_achievement_filter_favorites_alt Obtener sólo las puntuaciones de los favoritos en orden descendente.

Sintaxis:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Argumento Tipo Descripción
leaderboard_name Cadena The name of the leaderboard to read, as set up on XDP.
user_name ID de usuario de Xbox The ID of the user to read from.
num_items Real The number of items to retrieve.
friend_filter Filtro de logros de Xbox Live constante One of the filter constants (see the description, above).

Devuelve:

Real

Ejemplo:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

El código anterior obtiene el ID de usuario y luego lo utiliza para solicitar todos los datos de los jugadores de los 10 primeros lugares de la tabla de posiciones dada.
