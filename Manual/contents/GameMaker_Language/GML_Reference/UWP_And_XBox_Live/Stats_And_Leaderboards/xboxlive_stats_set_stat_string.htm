xboxlive_stats_set_stat_string

Esta función puede utilizarse para establecer una estadística en string para el ID de usuario dado. Se suministra el ID de usuario devuelto por (por ejemplo) la función xboxlive_get_user(), luego la estadística string a establecer(si la estadística string no existe ya, entonces se creará una nueva estadística y se establecerá con el valor dado) y luego la string a la que se establecerá la estadística. Tenga en cuenta que el nombre de la estadística debe ser sólo alfanumérico, sin símbolos ni espacios, pero el valor string que está estableciendo puede tener espacios y caracteres especiales.

Al establecer la estadística string, cualquier string anterior será anulada, por lo tanto, es usted quien debe determinar si la estadística string debe ser actualizada o no, comparando la estadística actual string con la nueva antes de establecerla.

La función devolverá -1 si hubo un error o el ID de usuario no es válido, o cualquier otro valor si la función fue llamada con éxito.

Sintaxis:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The user ID pointer to set the stat for
stat Cadena The statistic to set (a string)
value Cadena The value to set the stat to (a string)

Devuelve:

Real

Ejemplo:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

El código anterior establece la estadística "LevelName" al string almacenado en la variable global dada.
