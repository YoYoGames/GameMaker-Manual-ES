Estadísticas y tablas de clasificación

Las estadísticas de Xbox One (logros y tablas de clasificación) son funcionales cuando se utiliza UWP y tienen algunas funciones especiales para tratarlas, aunque también utilizan algunas de las funciones estándar incorporadas para las tablas de clasificación. El sistema de estadísticas de Xbox Live de 2017 simplifica todo el procedimiento de creación de logros y tablas de clasificación, permitiendo crear y utilizar los diferentes sistemas desde GameMaker. La principal diferencia entre el sistema de 2017 (Title-Managed) y los sistemas anteriores es que ya no tienes que definir todas las estadísticas que utilizas en el juego en el Centro de Socios, y tampoco necesitas vincular el desbloqueo de logros a un evento/regla de estadísticas (ya que en su lugar se desbloquean mediante una llamada de función directa).

NOTA Aunque no se requiere un archivo de manifiesto con este sistema, puedes configurar algunas estadísticas en XDP/Centro de Socios. Las que se configuren allí se considerarán estadísticas destacadas, lo que significa que puedes tener tablas de clasificación globales y las estadísticas aparecerán en la página del tablero de los juegos. Para más información, consulta este artículo de Microsoft.

Antes de llamar a cualquiera de las funciones de estadísticas listadas a continuación, debes asegurarte de que el usuario está conectado (ver las funciones de Usuario listadas arriba) y luego configurar el sistema de estadísticas y añadir el usuario a él usando estas funciones:

  • xboxlive_stats_setup (Sólo para estadísticas basadas en eventos (2013))
  • xboxlive_stats_add_user

Las siguientes funciones se utilizan con el sistema Stats 2017 (gestionado por título):

  • xboxlive_stats_remove_user
  • xboxlive_stats_flush_user
  • xboxlive_stats_set_stat_real
  • xboxlive_stats_set_stat_int
  • xboxlive_stats_set_stat_string
  • xboxlive_stats_delete_stat
  • xboxlive_stats_get_stat_names
  • xboxlive_stats_get_stat
  • xboxlive_stats_get_leaderboard
  • xboxlive_stats_get_social_leaderboard
  • xboxlive_achievement_show_achievements
  • xboxlive_achievement_load_friends
  • xboxlive_achievement_load_leaderboard
  • xboxlive_achievements_set_progress

Las siguientes funciones se utilizan con el sistema de estadísticas 2013 (basado en eventos):

  • xboxlive_get_stats_for_user
  • xboxlive_read_player_leaderboard
  • xboxlive_fire_event
