xboxlive_stats_get_stat

Esta función puede utilizarse para recuperar un único valor de estadísticas del gestor de estadísticas para un usuario determinado. Se suministra el ID del usuario tal y como lo devuelve (por ejemplo) la función xboxlive_get_user()y la estadística string tal y como se definió al crearla mediante una de las funciones de xboxlive_stats_set_stat_*. El valor de retorno puede ser un string o un real (dependiendo de la estadística que se esté comprobando) o la constante GML undefined si la estadística no existe o ha habido un error.

Sintaxis:

xboxlive_stats_get_stat(user_id, stat);

ArgumentoTipo Descripción
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

Devuelve:

Real, String or undefined

Ejemplo:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

El código anterior comprueba una variable y si es true, entonces comprobará el valor de la estadística "PercentDone". Si este valor es inferior a 100, se genera un valor y la estadística se establece en ese valor.
