xboxlive_achievement_show_achievements

Esta función abrirá la página de logros de la plataforma elegida. Ten en cuenta que se trata de una función asíncrona, es decir, tu juego seguirá ejecutándose en segundo plano mientras se muestra la página de logros. Por lo tanto, debes tener cuidado cuando uses esta función y asegurarte de pausar el juego o permitir que se muestre sólo en áreas de tu juego donde no interfiera con la jugabilidad.

Sintaxis:

xboxlive_achievement_show_achievements()

Devuelve:

Ejemplo:

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

El código anterior comprueba una variable personalizada para ver si los logros están habilitados, y si es así establece una variable global y crea una instancia de "pausa" del controlador antes de llamar a la función para mostrar los logros en la pantalla.
