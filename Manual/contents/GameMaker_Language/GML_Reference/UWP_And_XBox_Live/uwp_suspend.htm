uwp_suspender

Esta función indica al sistema que la aplicación ha terminado de guardar los datos tras la detección de una solicitud de suspensión. Llamar a esta función es un requisito para la presentación de Xbox One, y siempre debe utilizarse junto con la función uwp_is_suspending()como se muestra en el siguiente ejemplo.

Sintaxis:

uwp_suspend();

Devuelve:

Ejemplo:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

El código anterior comprueba si la aplicación va a entrar en suspensión y si lo hace llama a un script para guardar los datos del juego antes de suspenderlo.
