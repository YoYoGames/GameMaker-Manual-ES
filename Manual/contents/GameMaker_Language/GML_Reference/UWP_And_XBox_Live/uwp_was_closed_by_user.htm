uwp_fue_cerrado_por_usuario

Con esta función se puede averiguar si el juego fue cerrado previamente por el usuario mientras jugaba (es decir, fue terminado correctamente y no por el sistema). Si lo ha sido entonces la función devolverá true, de lo contrario devolverá false.

Sintaxis:

uwp_was_closed_by_user();

Devuelve:

Ejemplo:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

El código anterior comprueba si la aplicación ha sido terminada previamente por el usuario, y si es así, establece una variable global.
