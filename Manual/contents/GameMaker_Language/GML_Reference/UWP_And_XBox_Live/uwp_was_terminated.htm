uwp_fue_terminado

Con esta función se puede averiguar si el juego fue cerrado previamente por el sistema (es decir, no es la primera ejecución del juego). Si lo ha sido entonces la función devolverá true, de lo contrario devolverá false. Un ejemplo de uso sería comprobarlo y ver si el juego fue previamente terminado por el sistema, y entonces (si devuelve true) cargar la etapa anterior en la que el jugador estaba en el juego. Si se apunta a la XBox One entonces el comportamiento requerido de los juegos si esto se detecta será diferente entre las diferentes versiones de los documentos de certificación, por lo que vale la pena comprobar la última versión para ver lo que el juego debe hacer.

Sintaxis:

uwp_was_terminated();

Devuelve:

Ejemplo:

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

El código anterior comprueba si la aplicación ha sido terminada previamente por el sistema, y si es así, carga un buffer y establece una variable global.
