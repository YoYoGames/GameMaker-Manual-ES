uwp_license_trial_time_remaining

Esta función devolverá el número aproximado de segundos hasta que expire una prueba basada en el tiempo. El valor devuelto por esta función no tiene sentido si la prueba no está basada en el tiempo o si el juego no se está ejecutando en modo de prueba.

Sintaxis:

uwp_license_trial_time_remaining();

Devuelve:

Ejemplo:

if uwp_license_trial_version()
{
    var secs = uwp_license_trial_time_remaining() mod 60;
    var mins = uwp_license_trial_time_remaining() div 60;
    var hours = mins div 60;
    if secs < 10 secs = "0" + string(secs) else secs = string(secs);
    if mins > 60 mins -= (hours * 60);
    if mins < 10 mins = "0" + string(mins) else mins = string(mins);
    if hours > 9 hours = "9" else hours = string(hours);
    time_string = hours + ":" + mins + ":" + secs;
}

El código anterior comprueba si el juego se está ejecutando con una licencia de prueba y, si es así, crea un string con el tiempo que falta para que expire la licencia.
