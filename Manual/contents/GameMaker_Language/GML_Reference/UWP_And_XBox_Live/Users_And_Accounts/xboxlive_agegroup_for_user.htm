xboxlive_agegroup_for_user

Esta función devolverá una de las cuatro constantes (mostradas a continuación) para indicar el rango de edad asignado al puntero de ID de usuario especificado.

Sintaxis:

xboxlive_agegroup_for_user(user_id);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The user ID (a pointer) to check

Devuelve:

Xbox Age Group Constant

Grupo de edad de Xbox Constante
Constante Descripción
xboxlive_agegroup_unknown El grupo de edad del usuario es desconocido
xboxlive_agegroup_child El grupo de edad de los usuarios está entre 8 y 12 años (inclusive)
xboxlive_agegroup_teen El grupo de edad de los usuarios está entre los 13 y los 17 años (inclusive, y entre los 13 y los 19 en Corea del Sur)
xboxlive_agegroup_adult El grupo de edad de los usuarios es de 18 años o más (o 20 y más en Corea del Sur

Ejemplo:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
{
    global.ContentControl = true;
}
else global.ContentControl = false;

El código anterior comprueba el grupo de edad del usuario activador y establece una variable global a true o false dependiendo de la constante devuelta.
