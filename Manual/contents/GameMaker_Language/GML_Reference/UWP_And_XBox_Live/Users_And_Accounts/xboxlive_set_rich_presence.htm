xboxlive_set_rich_presence

Esta función establecerá la presencia enriquecida string para el usuario dado. Una presencia enriquecida string muestra la actividad de un usuario en el juego después del nombre del juego que el usuario está jugando, separado por un guión. Este string se muestra bajo el Gamertag de un jugador en la lista de "Amigos y Clubes" así como en el perfil de usuario de Xbox Live del jugador.

Cuando se utiliza esta función es necesario suministrar el puntero de ID de usuario para el usuario, y luego se puede flag el usuario como actualmente activo en el juego o no (utilizando true/false). Si se establece en false, entonces la presencia rica string se anexará con "/afk" o algo apropiado. El siguiente argumento es el ID de presencia enriquecida string que se mostrará, y finalmente puedes (opcionalmente) suministrar un service_config_id string. Este argumento es opcional, y cuando no se suministra, utiliza el SCID especificado en las Opciones de Juego de Xbox.

Sintaxis:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The ID pointer of the user to check.
is_user_active Booleano Flag the user as active or not.
rich_presence_string Cadena The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id Cadena OPTIONAL Este es el config_id string para el juego.

Devuelve:

N/A

Ejemplo:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

El código anterior obtiene el puntero de ID de usuario para el usuario asignado a la ranura [0] del gamepad, y luego establece la presencia rica string para ese usuario.
