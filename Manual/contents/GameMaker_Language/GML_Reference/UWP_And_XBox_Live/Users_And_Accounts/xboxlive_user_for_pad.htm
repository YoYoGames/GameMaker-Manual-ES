xboxlive_user_for_pad

Esta función devolverá el puntero del ID de usuario asociado al valor del índice del gamepad dado, o pointer_null si no hay ningún usuario asociado. Ten en cuenta que esta función sólo debe usarse con índices de gamepad, y no con xboxlive_get_user_count().

Sintaxis:

xboxlive_user_for_pad(pad_index);

ArgumentoTipo Descripción
pad_index The index (an integer) of the gamepad slot to check

Devuelve:

Xbox User ID or pointer_null

Ejemplo:

for(var i = 0; i < 11; ++i;)
{
    var u_id = xboxlive_user_for_pad(i);
    if (u_id != pointer_null)
    {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
    }
    else
    {
        pad_name[i] = "Press Play";
    }
}

El código anterior loops a través de todas las ranuras disponibles de la almohadilla y comprueba si hay un puntero de identificación de usuario asociado con él. Si lo hay, se establece un array con el nombre del usuario, de lo contrario el array se establece con algún texto por defecto.
