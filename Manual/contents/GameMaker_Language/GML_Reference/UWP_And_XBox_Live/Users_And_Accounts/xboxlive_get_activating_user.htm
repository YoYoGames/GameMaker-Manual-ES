xboxlive_get_activating_user

Con esta función puedes recuperar el puntero de identificación del usuario que lanzó el juego desde la consola.

NOTA: El uso de este puntero dependerá de lo que quieras hacer en el juego, pero no debes confiar en que sea el ID de usuario que está jugando actualmente, ya que el usuario que lo activa puede cerrar la sesión mientras el juego está en marcha.

Sintaxis:

xboxlive_get_activating_user();

Devuelve:

Xbox User ID

Ejemplo:

global.activUser = xboxlive_get_activating_user();

Lo anterior almacena el puntero de identificación del usuario activador en una variable global
