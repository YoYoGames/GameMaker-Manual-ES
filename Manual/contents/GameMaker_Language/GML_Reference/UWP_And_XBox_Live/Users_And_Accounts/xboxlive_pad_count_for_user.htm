xboxlive_pad_count_for_user

Con esta función puede averiguar cuántas "ranuras" de pads están activas para el usuario actual (véase xboxlive_pad_for_user() para más detalles).

Sintaxis:

xboxlive_pad_count_for_user(user_id, slot);

ArgumentoTipo Descripción
user_id The user ID (a pointer) to check

Devuelve:

Ejemplo:

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

El código anterior almacena el número de almohadillas asociadas con el puntero de ID de usuario dado.
