xboxlive_gamertag_for_user

Esta función obtendrá la etiqueta de jugador del usuario que ha iniciado la sesión. Tenga en cuenta que esta función sólo es válida si el usuario ha iniciado sesión y, por lo tanto, debe comprobarlo con la función xboxlive_user_is_signed_in() antes de solicitar la etiqueta de jugador. Si llamas a esta función cuando ningún usuario ha iniciado sesión, obtendrás una cadena vacía"".

Sintaxis:

xboxlive_gamertag_for_user();

Devuelve:

Ejemplo:

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

El código anterior comprueba si un usuario ha iniciado sesión en XBox Live y, si lo ha hecho, el código obtendrá el gamertag del usuario y lo almacenará en una variable global para su uso futuro. Si no está registrado, el código comprueba si está en proceso de registro, en cuyo caso no ocurrirá nada más, y si no lo está, abrirá el selector de cuentas, pidiéndole que se registre.
