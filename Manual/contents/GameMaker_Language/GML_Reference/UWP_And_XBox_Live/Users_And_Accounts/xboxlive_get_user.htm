xboxlive_get_user

Con esta función se puede recuperar el puntero de identificación del usuario indexado. Si el usuario no existe, la función devolverá la constante pointer_null en su lugar. Puede encontrar el número de usuarios actualmente conectados con la función xboxlive_get_user_count().

IMPORTANTE: Esta función sólo debe usarse con xboxlive_get_user_count() - no la uses con gamepads. Utilice xboxlive_user_for_pad() en su lugar.

Sintaxis:

xboxlive_get_user(index);

Argumento Tipo Descripción
index Real The index (an integer) to get the User ID from.

Devuelve:

Xbox User ID or pointer_null

Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Lo anterior loops a través de todos los usuarios registrados y almacena su puntero de identificación única en una matriz.
