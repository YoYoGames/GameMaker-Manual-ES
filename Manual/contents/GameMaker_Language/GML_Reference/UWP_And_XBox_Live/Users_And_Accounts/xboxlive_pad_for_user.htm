xboxlive_pad_for_user

Los usuarios de Xbox pueden tener varios pads asignados a su ID de usuario, y cada uno de ellos llena una "ranura". Estas ranuras están numeradas desde el 0, y pueden contener cualquier índice de pad, por lo que la ranura 0 podría ser asignada al pad 3 y la ranura 1 podría ser asignada al pad 2. Esta función se puede utilizar para obtener el pad que se asigna a cualquier ranura, y es particularmente útil si un usuario deja un gamepad sin entrada durante un cierto período de tiempo, ya que se switch apagado, por lo que puede comprobar cualquier ID de usuario dado el uso de esta función para asegurarse de que tienen un gamepad activo, y si no lo hacen entonces reaccionar en consecuencia por (por ejemplo) la pausa de su juego. Si el pad está activo, su valor de índice será devuelto, pero si no, la función devolverá -1.

Tenga en cuenta que si un pad en cualquier ranura se vuelve inactivo, las ranuras posteriores se "deslizarán" a la posición vacante, por lo que si tiene tres pads activos asignados a las ranuras 0, 1 y 2, y el pad en la ranura 0 está inactivo, las ranuras 1 y a se convertirán ahora en las ranuras 0 y 1, por lo que la ranura 0 siempre debe contener un pad activo a menos que todos los pads estén apagados (en cuyo caso devolverá -1).

Sintaxis:

xboxlive_pad_for_user(user_id, slot);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The User ID (a pointer) to check
slot Real The index (an integer) of the gamepad slot to check

Devuelve:

Xbox Pad ID

Ejemplo:

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
    for(var i = 0; i < gamepad_get_device_count(); i++;)
    {
        var u_id = xboxlive_user_for_pad(i);
        if (u_id == -1)
        {
            async_ID = xboxlive_pad_for_user(i, u_id);
        }
    }
}

El código anterior comprueba la ranura 0 para el puntero de ID de usuario dado para ver si se ha asignado algún gamepad, y si no, se asigna uno.
