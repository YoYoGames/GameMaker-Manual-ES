xboxlive_get_file_error

Esta función devolverá una constante (que se muestra a continuación) para ayudar a depurar por qué falló una función de archivo para que puedas tomar las medidas adecuadas en tu código de juego. Cuando una de las funciones de archivo falla, puedes llamar a la función para saber exactamente cuál fue el error.

NOTA: Cuando se utilizan las funciones asíncronas de guardar/cargar, el mapa async_load tendrá una clave extra, "error" que también contendrá una de las constantes (mostradas abajo).

Sintaxis:

xboxlive_get_file_error();

Devuelve:

Xbox File Error Constant

Constante de error del archivo Xbox
Constante Descripción
xboxlive_fileerror_outoflocalstorage Si esto ocurre mientras se guarda, significa que todo el espacio disponible para guardar la partida se ha agotado, y el usuario ha rechazado la oferta (presentada por el SO) de liberar espacio. El juego debe confirmar con el usuario que tiene la intención de seguir jugando sin guardar. Si todavía quieren poder guardar, el juego debería reintentar la operación de guardado.
xboxlive_fileerror_quotaexceeded Si esto ocurre, el juego debería notificar al usuario que no puede guardar más datos hasta que libere algo de espacio, y luego presentarle una forma de hacerlo.
xboxlive_fileerror_noerror La función de guardar o cargar archivos se realizó sin problemas.

Ejemplo:

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

El código anterior comprueba si hubo un error al guardar y, si es así, el código de error se imprime en la consola.
