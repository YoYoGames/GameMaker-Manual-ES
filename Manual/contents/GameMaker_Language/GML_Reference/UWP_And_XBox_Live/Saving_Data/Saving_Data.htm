Guardar datos

Las siguientes funciones tienen que ver específicamente con el guardado de datos usando la exportación UWP para Xbox One y usando las características de XBox Live.

Mientras que los datos guardados relacionados con el progreso del jugador deben estar siempre asociados a un usuario en particular, puedes guardar datos genéricos en el área de "almacenamiento de la máquina" a proporcionando el valor -1 como argumento de usuario. Esta área es específica del título, pero no está asociada a un usuario concreto.

NOTA: ¡Ningún archivo guardado puede ser mayor de 16 megas!

Las siguientes funciones son para tratar los datos de guardado de Xbox Live:

  • xboxlive_set_savedata_user
  • xboxlive_get_savedata_user
  • xboxlive_get_file_error
