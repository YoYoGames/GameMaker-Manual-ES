xboxlive_set_savedata_user

Esta función especifica que las futuras operaciones de archivo que operan en el área de guardado del juego (es decir, todas las escrituras de archivos, y las lecturas de archivos que no están en el área del paquete) se asociarán con el puntero de ID de usuario especificado. Se puede llamar a esta función tantas veces como sea necesario para redirigir los datos de guardado al usuario apropiado, o se puede utilizar la constante pointer_null para guardar en el área de almacenamiento genérico de la máquina.

Sintaxis:

xboxlive_set_savedata_user(user_id);

Argumento Tipo Descripción
user_id ID de usuario de Xbox o pointer_null The user ID (a pointer) to set for saving, or pointer_null

Devuelve:

N/A

Ejemplo:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

El código anterior comprueba si un usuario está actualmente asignado como objetivo de guardado, y si no lo está entonces se le asigna.
