uwp_is_constrained

Esta función devolverá true si el juego se está ejecutando en un modo de recursos limitados o false si se ejecuta normalmente. El modo de recursos limitados puede activarse cuando el juego no tiene foco, y en este modo el juego tiene recursos reducidos de la GPU. La respuesta a esto debería ser normalmente pausar el juego (ya que el usuario ya no tiene el control). Además, si se observa una ralentización en este modo, puede ser una buena idea desactivar cualquier efecto especialmente exigente. Tenga en cuenta que, a diferencia de la función uwp_is_suspending() esta función devolverá true continuamente mientras el juego esté en modo restringido (y no sólo una vez cuando ocurre por primera vez).

Sintaxis:

uwp_is_constrained();

Devuelve:

Ejemplo:

if xboxliveis_constrained()
{
    if !instance_exists(obj_PauseMenu)
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

El código anterior comprueba si la aplicación está restringida y, si lo está, crea una instancia de un object (utilizado para pausar el juego).
