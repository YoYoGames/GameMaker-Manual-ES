xboxlive_matchmaking_session_leave

Esta función hace que el usuario que creó o encontró la sesión especificada la abandone. Puede obtener el puntero del ID de la sesión del async_load mapa DS que se devuelve en el Evento Social Asíncrono al crear o encontrar una sesión (ver xboxlive_matchmaking_create() para más detalles).

Sintaxis:

xboxlive_matchmaking_session_leave(session_id);

ArgumentoTipo Descripción
session_id The Session ID pointer of the session to leave.

Devuelve:

Ejemplo:

if gamepad_button_check_pressed(0, gp_start)
{
    xboxlive_matchmaking_session_leave(global.SessionID);
}

El código anterior comprueba si se ha pulsado un botón del gamepad y si se detecta uno, el usuario saldrá de la sesión de matchmaking dada.
