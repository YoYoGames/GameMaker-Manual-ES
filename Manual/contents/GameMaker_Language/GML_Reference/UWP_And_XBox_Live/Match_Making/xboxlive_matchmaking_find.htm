xboxlive_matchmaking_find

Esta función buscará una sesión multijugador utilizando la plantilla de sesión, la tolva de emparejamiento y la plantilla de asociación de dispositivos seguros, que debería haber creado de antemano en el panel de control de XDP. El ID de usuario que puede recuperar para un pad de juego en particular llamando a la función xboxlive_user_for_pad() función. La función devolverá un valor único de ID de solicitud, que puede utilizarse para identificar el evento social asíncrono correcto para esta función. Este evento se activará cuando se haya creado la sesión y contendrá un Mapa DS en la variable async_load con los siguientes pares clave/valor:

  • "requestid" - contiene el ID de la solicitud devuelto por la función de llamada
  • "status" - contiene el string "session_created" para informar del tipo de evento que se ha disparado.
  • "num_results" - contiene 1 o más si se ha encontrado una sesión, en caso contrario contiene 0.
  • "sessionid<index> " - contiene el ID de la sesión, o -1 en caso de fallo(<index/gt; es un valor de 0 a "num_results" - 1).
  • "sessionOwner<index> " - contiene el ID del anfitrión de la sesión si se ha encontrado una sesión(<index> es un valor de 0 a "num_results" - 1).
  • "error" - contiene 0 en caso de éxito, o -1 en caso de fallo

Sintaxis:

xboxlive_matchmaking_find(user_id, template, hopper, sdatemplate, [matchattributes]);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The user ID pointer to use
template Cadena The name of the session template
hopper Cadena The name of the matchmaking hopper
sdatemplate Cadena The name of the secure device association template
matchattributes Real OPTIONAL Los atributos del partido

Devuelve:

Async Request ID

Ejemplo:

var userID = xboxone_user_for_pad(global.PadIndex);
result = xboxone_matchmaking_find(userID, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Lo anterior recuperará el ID del usuario en el gamepad dado y luego tratará de encontrar una sesión de matchmaking para ellos.
