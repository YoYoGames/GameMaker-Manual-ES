xboxlive_matchmaking_create

Esta función creará una sesión multijugador utilizando la plantilla de sesión, la tolva de emparejamiento y la plantilla de asociación de dispositivos seguros, que debería haber creado de antemano en el panel de control de XDP. El ID de usuario que puede recuperar para un pad de juego concreto llamando a la función xboxlive_user_for_pad() y la visibilidad será una de las siguientes constantes:

Constante de visibilidad de Xbox Match
Constante Descripción
xboxlive_match_visibility_open especifica que la sesión puede ser unida por los demás.
xboxlive_match_visibility_private especifica que la sesión no es visible para los usuarios que no son miembros de la sesión.
xboxlive_match_visibility_usetemplate especifica que se debe usar el valor utilizado en la plantilla de la sesión (normalmente se debe usar esto ya que no se puede anular el valor de una plantilla si se ha establecido una en XDP).

La función devuelve un valor único de ID de solicitud, que puede utilizarse para identificar el evento social asíncrono correcto para esta función. Este evento se activará cuando se haya creado la sesión y contendrá un Mapa DS en la variable async_load con los siguientes pares clave/valor:

  • "requestid" - contiene el ID de la solicitud devuelto por la función de llamada
  • "status" - contiene el string "session_created" para informar del tipo de evento que se ha disparado.
  • "sessionid" - contiene el ID de la sesión, o -1 en caso de fallo
  • "error" - contiene 0 en caso de éxito, o -1 en caso de fallo
  • "correlationid" - el manejador de sesión para la correlación con la función xboxlive_matchmaking_join_session()

Sintaxis:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Argumento Tipo Descripción
user_id ID de usuario de Xbox The user ID pointer to use
visibility Constante de visibilidad de Xbox Match One of the constants listed above
template Cadena The name of the session template
hopper Cadena The name of the matchmaking hopper
sdatemplate Cadena The name of the secure device association template
matchattributes Real OPTIONAL Los atributos del partido

Devuelve:

Async Request ID

Ejemplo:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Lo anterior recuperará el ID del usuario en el gamepad dado y luego iniciará una sesión de matchmaking para ellos.
