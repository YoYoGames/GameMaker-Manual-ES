xboxlive_matchmaking_join_invite

Esta función puede ser llamada para procesar una invitación de unión una vez que se ha llamado con éxito a xboxlive_matchmaking_start(). Se suministra el UserID local y los valores de la invitación (ID e ID de host) que fueron devueltos cuando se recibió la invitación, junto con el nombre de la plantilla de sesión que creó (un string). Entonces se unirá al usuario a la sesión a la que ha sido invitado, y recibirá los Eventos Sociales Asíncronos como si hubiera llamado a xboxlive_matchmaking_find().

Sintaxis:

xboxlive_matchmaking_join_invite(user_who_received_invite, invitation_id, invitation_host, sda_template_name);

ArgumentoTipo Descripción
user_who_received_invite The local user ID pointer.
invitation_id The invitation ID.
invitation_host The invitation host ID.
sda_template_name The session template name.

Devuelve:

Ejemplo:

var userID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_join_invite(userID, global.InviteID, global.InviteHost, global.SessionTemplate);

Lo anterior obtiene el ID de usuario para el gamepad dado y luego intenta unirse a la sesión a la que han sido invitados.
