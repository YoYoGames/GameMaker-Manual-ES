xboxlive_matchmaking_set_joinable_session

Esta función establece una sesión (previamente creada) para que esté disponible para que otros usuarios se unan a través de la interfaz del sistema. Un usuario sólo tiene una sesión a la que puede unirse a la vez, y cuando abandona la sesión (o la sesión termina) esto se borrará, sin embargo, puede borrarlo manualmente pasando -1 en el argumento de la sesión.

Sintaxis:

xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable);

ArgumentoTipo Descripción
local_user The local user ID pointer.
session_that_is_joinable The session ID to make joinable, or -1.

Devuelve:

Ejemplo:

if (global.session_ID != -1)
{
    xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID);
}

El código anterior comprueba si hay un valor de ID de sesión válido, y si se detecta uno, establece que la sesión es unificable.
