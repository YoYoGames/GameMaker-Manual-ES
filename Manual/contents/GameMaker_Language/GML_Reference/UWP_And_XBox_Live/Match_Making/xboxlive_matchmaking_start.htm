xboxlive_matchmaking_start

Esta función debe ser llamada antes de poder iniciar cualquier sesión multijugador para el usuario. Toma el puntero del ID de usuario, que puedes recuperar para un pad de juego en particular llamando a la función xboxlive_user_for_pad() e inicializa la API multijugador.

Sintaxis:

xboxlive_matchmaking_start(user_id);

ArgumentoTipo Descripción
user_id The user ID pointer to use

Devuelve:

Ejemplo:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

El código anterior obtiene el ID de usuario para el índice del pad dado y luego inicializa las capacidades multijugador.
