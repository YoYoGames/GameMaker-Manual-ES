UWP y Xbox Live

En esta sección se describen las funciones que se utilizan específicamente cuando se apunta a la plataforma de destino UWP. El conjunto inicial de funciones sirve para controlar el comportamiento general del juego al crear una aplicación UWP:

  • uwp_suspend
  • uwp_is_suspending
  • uwp_is_constrained
  • uwp_was_terminated
  • uwp_was_closed_by_user
  • uwp_show_help
  • uwp_license_trial_version
  • uwp_license_trial_user
  • uwp_license_trial_time_remaining
  • uwp_check_privilege

UWP también permite crear juegos para la plataforma XBox One utilizando la API de XBox Live mediante las siguientes funciones. Estas funciones requieren que hayas marcado la opción Habilitar XBox Live en las Opciones de Juego de UWP. Algunas de estas funciones requieren que el usuario haya iniciado sesión en XBox Live, lo que GameMaker intentará hacer automáticamente cuando ejecute el proyecto. Este inicio de sesión "silencioso" generará un Evento Asíncrono del Sistema que puede ser comprobado para ver si fue exitoso o no antes de continuar. Tenga en cuenta que, a diferencia de la exportación de la consola Xbox One, la exportación UWP para XBox Live sólo permite que un usuario inicie sesión a la vez.

¡IMPORTANTE! El objetivo UWP sólo tiene un mínimo de soporte para XBox Live cuando se configura como parte del Programa de Creadores, que se limita a algunas funciones básicas de usuario y cuenta (ver aquí para más información sobre la configuración de esto). Para poder utilizar toda la funcionalidad que se menciona aquí necesitarás tener una cuenta ID@XBox.
