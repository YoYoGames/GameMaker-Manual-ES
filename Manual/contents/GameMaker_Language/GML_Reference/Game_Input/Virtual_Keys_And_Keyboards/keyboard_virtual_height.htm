keyboard_virtual_height

Esta función devolverá la altura actual en píxeles del teclado virtual, basándose en el tamaño de la pantalla. Si el teclado no es visible, se devolverá 0.

Sintaxis

keyboard_virtual_height();

Devuelve:

Ejemplo:

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

El código anterior comprobará el estado del teclado virtual del sistema operativo, y si es visible establecerá una variable con la altura del teclado.
