virtual_key_show

Una vez que haya creado su llave virtual para sus dispositivos, es posible que necesite depurar y probar que está correctamente posicionada y funciona adecuadamente. Para ello llamarías a esta función, usando el índice de la tecla virtual que quieres ver (previamente creada y almacenada usando virtual_key_add()) que dibujará un rectángulo de color para representar la tecla en la pantalla. Una vez que esté satisfecho con las cosas, se recomienda que cree y coloque su propio gráfico en su área.

NOTA El rectángulo que se dibuja se verá afectado por las opciones de mezcla, color y alfa establecidas por las funciones de dibujo correspondientes.

Sintaxis

virtual_key_show(index);

Argumento Tipo Descripción
index ID de la llave virtual The index of the virtual key to show.

Devuelve:

N/A

Ejemplo:

if global.Debug
{
    virtual_key_show(global.left);
}
else
{
    virtual_key_hide(global.left);
}

El código anterior comprueba la variable global "global.Debug" y si es verdadero entonces la tecla virtual indexada en la variable "global.Left" se dibujará en la pantalla, y si es falso entonces la tecla se ocultará.
