virtual_key_delete

Si tu juego tiene diferentes rooms o instancias para controlar los menús y el juego y otras cosas, entonces es probable que necesites cambiar la posición y los mapas de las teclas virtuales en algún momento. Para ello puedes utilizar esta función para borrar las antiguas antes de crear las nuevas (si es necesario). La función requiere que proporcione el ID de la llave virtual a borrar - como lo devuelve virtual_key_add().

Tenga en cuenta que las llaves virtuales se eliminarán automáticamente de la room cuando se cambie la room, por lo que esta función sólo es necesaria cuando se desea eliminar manualmente las llaves antes de que la propia room haya terminado.

Sintaxis

virtual_key_delete(index);

Argumento Tipo Descripción
index ID de la llave virtual The index of the virtual key to delete.

Devuelve:

N/A

Ejemplo:

virtual_key_delete(global.Left);

El código anterior borra la clave virtual indexada en la variable global "global.Left".
