gesture_drag_time

Esta función se utiliza para establecer el tiempo que tarda un evento de Gesto de Ar rastre en ser activado por un toque o un clic. Este tiempo también afectará a la forma en que se desencadena el Evento de Arrastre, ya que si se toca o se hace clic y se suelta antes de que transcurra este tiempo, se considerará un Arrastre. El tiempo se mide en segundos y tiene un valor por defecto de 0,16.

Sintaxis:

gesture_drag_time(time);

ArgumentoTipo Descripción
time The time (in seconds) to set for drag gesture event detection.

Devuelve:

Ejemplo:

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

El código anterior comprueba si el tiempo de arrastre de los gestos está establecido en 0,16 segundos y, si no lo está, lo establece en ese valor.
