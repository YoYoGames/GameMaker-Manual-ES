gesto_obtener_tiempo_de_rotación

Esta función se utiliza para obtener el ángulo que debe superar un par de toques para desencadenar un Gesto de Inicio de Rotación. El ángulo se mide en grados y tiene un valor por defecto de 5°.

Sintaxis:

gesture_get_rotate_angle();

Devuelve:

(inches)

Ejemplo:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

El código anterior comprueba si el ángulo de rotación de los gestos está establecido en 5° y, si no lo está, lo establece en ese valor.
