Entrada gestual

Las funciones de entrada de dispositivo están muy bien para tratar con simples toques en la pantalla o conseguir la orientación del dispositivo, pero a veces querrás que tus proyectos detecten cosas más complejas, como ser capaz de arrastrar o "mover" una instancia a través de un room. Estas cosas se pueden hacer con las funciones de dispositivo, pero para hacer las cosas más fáciles GameMaker tiene una gama completa de eventos de gestos para detectar dobles toques, arrastrar, mover, pellizcar y rotar, tanto a nivel de instancia como global, así como devolver información sobre el evento. Las funciones enumeradas aquí están diseñadas para darle un mayor control sobre cómo y cuándo se activarán estos eventos.

IMPORTANTE Estas funciones no funcionarán en la plataforma de destino HTML5. No darán un error si se utilizan, sino que simplemente se ignorarán.

Están disponibles las siguientes funciones gestuales diferentes:

  • gesture_get_drag_time
  • gesture_get_drag_distance
  • gesture_get_flick_speed
  • gesture_get_double_tap_time
  • gesture_get_double_tap_distance
  • gesture_get_tap_count
  • gesture_get_pinch_distance
  • gesture_get_pinch_angle_towards
  • gesture_get_pinch_angle_away
  • gesture_get_rotate_time
  • gesture_get_rotate_angle
  • gesture_drag_time
  • gesture_drag_distance
  • gesture_flick_speed
  • gesture_double_tap_time
  • gesture_double_tap_distance
  • gesture_tap_count
  • gesture_pinch_distance
  • gesture_pinch_angle_towards
  • gesture_pinch_angle_away
  • gesture_rotate_time
