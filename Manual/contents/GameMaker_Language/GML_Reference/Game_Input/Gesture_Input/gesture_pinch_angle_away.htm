gesture_pinch_angle_away

Esta función se utiliza para establecer el ángulo dentro del cual un toque debe alejarse de otro toque antes de que pueda iniciarse potencialmente un gesto de pellizco. El ángulo se mide en grados y tiene un valor por defecto de 45°.

Sintaxis:

gesture_pinch_angle_away();

Devuelve:

(inches)

Ejemplo:

if gesture_get_pinch_angle_away() != 45
{
    gesture_pinch_angle_away(45);
}

El código anterior comprueba si el ángulo de extracción de los gestos está establecido en 45° y, si no lo está, lo establece en ese valor
