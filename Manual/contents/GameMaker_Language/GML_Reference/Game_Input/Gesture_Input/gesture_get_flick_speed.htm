gesture_get_flick_speed

Esta función se utiliza para obtener la velocidad necesaria para que se active un evento de Gesto Flick cuando se suelta un toque o un clic. La velocidad se mide en pulgadas por segundo y tiene un valor por defecto de 2,0.

Sintaxis:

gesture_get_flick_speed();

Devuelve:

(inches per second)

Ejemplo:

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

El código anterior comprueba si la velocidad de deslizamiento de los gestos está establecida en 2 pulgadas por segundo y, si no lo está, la establece en ese valor.
