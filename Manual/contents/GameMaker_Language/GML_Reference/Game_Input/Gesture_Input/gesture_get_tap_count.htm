gesture_get_tap_count

Esta función se utiliza para comprobar si el conteo de toques está activado o desactivado. La función devolverá true si está activada, y false en caso contrario, y puede activar o desactivar el conteo de toques utilizando la función gesture_tap_count(). Cuando está habilitado, cada evento de tap tendrá una entrada adicional en el mapa DS "tapcount" que habrá registrado el número de tap, y cuando está habilitado significa que todos los eventos de tap serán disparados, es decir: dos tap dispararán tanto el evento de tap simple como el evento de tap doble, siendo el tap simple 1 y el tap doble 2. El valor de la cuenta de toques se restablecerá a 0 una vez que haya transcurrido el tiempo establecido para la detección de un toque doble (véase la función gesture_double_tap_time()). Si el recuento de toques está desactivado, el toque inicial no se registrará hasta que haya pasado el tiempo de doble toque y no se haya detectado un segundo toque. Tenga en cuenta que esto está activado por defecto.

Sintaxis:

gesture_get_tap_count();

Devuelve:

Ejemplo:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

El código anterior comprueba si el recuento de grifos está activado y, si no lo está, se activa.
