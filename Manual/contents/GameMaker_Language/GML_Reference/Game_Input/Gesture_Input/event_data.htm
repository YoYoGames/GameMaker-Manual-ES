datos_del_evento

Esta variable es de alcance global y se utiliza para mantener un Mapa DS cuando se utiliza en el evento apropiado (por ejemplo, Eventos de Gesto y Mensajes de Difusión), y -1 en todas las demás ocasiones. El contenido real del mapa DS dependerá del tipo de evento que lo haya activado, así que consulta las secciones individuales para esos eventos.

Sintaxis:

event_data;

Devuelve:

DS Map ID

Ejemplo:

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

El código anterior se toma del evento del gesto Drag End y comprueba si la ventilación es un evento "flick", y si lo es, extrae los datos necesarios del mapa event_data DS y los utiliza para establecer algunas variables. Si no se detecta un evento "flick", entonces las mismas variables se ponen a 0.
