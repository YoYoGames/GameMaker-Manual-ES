gesto_pinch_distancia

Esta función se utiliza para establecer la distancia dentro de la cual hay que tocar/hacer clic en la pantalla y moverse con dos dedos antes de activar un gesto de pellizco. La distancia se mide en pulgadas y tiene un valor por defecto de 0,1.

Sintaxis:

gesture_pinch_distance();

Devuelve:

(inches)

Ejemplo:

if gesture_get_pinch_distance() != 0.1
{
    gesture_pinch_distance(0.1);
}

El código anterior comprueba si la distancia de pellizco para los gestos está establecida en 0,1 pulgadas y, si no lo está, la establece en ese valor.
