gesto_doble_tap_tiempo

Esta función se utiliza para establecer el tiempo que se necesita entre dos toques/clics para desencadenar un evento de Gesto de Doble Toque. El tiempo se mide en segundos y tiene un valor por defecto de 0,16.

Sintaxis:

gesture_double_tap_time(time);

Argumento Tipo Descripción
time Real The time (in seconds) to set for double tap gesture event detection.

Devuelve:

N/A

Ejemplo:

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

El código anterior comprueba si el tiempo de doble toque para los gestos está establecido en 0,16 segundos y si no lo está lo establece en ese valor.
