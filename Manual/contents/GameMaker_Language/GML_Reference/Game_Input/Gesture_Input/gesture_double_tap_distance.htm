gesture_double_tap_distance

Esta función se utiliza para establecer la distancia dentro de la cual hay que volver a tocar/hacer clic en la pantalla después de un solo toque para activar un gesto de doble toque. La distancia se mide en pulgadas y tiene un valor por defecto de 0,1.

Sintaxis:

gesture_double_tap_distance(distance);

ArgumentoTipo Descripción
distance The distance (in inches) to set for double tap gesture event detection.

Devuelve:

Ejemplo:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_get_double_tap_distance(0.1);
}

El código anterior comprueba si la distancia de doble toque para los gestos está establecida en 0,1 pulgadas y, si no lo está, la establece en ese valor.
