gesture_get_drag_time

Esta función se utiliza para obtener el tiempo que tarda un evento de Gesto de Arrastre e Inicio en ser activado por un toque o clic. Este tiempo también afectará a la forma en que se desencadena el Evento de Arrastre, ya que un toque/clic y la liberación antes de que se cumpla este tiempo se considerará un Arrastre. El tiempo se mide en segundos y tiene un valor por defecto de 0,16.

Sintaxis:

gesture_get_drag_time();

Devuelve:

(time in seconds)

Ejemplo:

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

El código anterior comprueba si el tiempo de arrastre de los gestos está establecido en 0,16 segundos y, si no lo está, lo establece en ese valor.
