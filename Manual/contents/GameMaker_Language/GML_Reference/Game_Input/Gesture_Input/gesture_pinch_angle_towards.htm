gesto_pellizco_ángulo_hacia_el_futuro

Esta función se utiliza para establecer el ángulo dentro del cual un toque debe estar moviéndose hacia otro toque antes de que se pueda iniciar potencialmente un gesto de pellizco. El ángulo se mide en grados y tiene un valor por defecto de 45°.

Sintaxis:

gesture_pinch_angle_towards();

Devuelve:

(inches)

Ejemplo:

if gesture_get_pinch_angle_towards() != 45
{
    gesture_pinch_angle_towards(45);
}

El código anterior comprueba si el ángulo de pellizco para los gestos está establecido en 45° y, si no lo está, lo establece en ese valor.
