gesture_get_double_tap_distance

Esta función se utiliza para obtener la distancia dentro de la cual tiene que tocar/clic en la pantalla de nuevo después de un solo toque con el fin de desencadenar un gesto de doble toque. La distancia se mide en pulgadas y tiene un valor por defecto de 0,1.

Sintaxis:

gesture_get_double_tap_distance();

Devuelve:

(inches)

Ejemplo:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_double_tap_distance(0.1);
}

El código anterior comprueba si la distancia de doble toque para los gestos está establecida en 0,1 pulgadas y, si no lo está, la establece en ese valor.
