tiempo_de_rotación_del_gesto

Esta función se utiliza para establecer el tiempo dentro del cual un par de toques debe estar rotando en una dirección consistente para que se active un Gesto de Inicio de Rotación. El tiempo se mide en segundos y tiene un valor por defecto de 0,16s.

¡IMPORTANTE! Actualmente, para un juego de 60fps sólo se puede establecer un máximo de un segundo, de lo contrario no se detectarán las rotaciones. Esto aumentará para una tasa de fotogramas más baja, por ejemplo un juego de 30fps puede tener un tiempo máximo de 2 segundos.

Sintaxis:

gesture_rotate_time();

Devuelve:

(inches)

Ejemplo:

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

El código anterior comprueba si el tiempo de rotación de los gestos está establecido en 0,1 segundos y, si no lo está, lo establece en ese valor.
