gamepad_get_option

Esta función se puede utilizar para recuperar el valor de la opción dada. Se suministra el "slot" del gamepad a comprobar, así como la opción string y la función devolverá el valor asociado. La opción disponible string dependerá de la plataforma en la que se ejecute el proyecto, como se indica a continuación:

Opción Plataforma Descripción Devuelve
"allow_rotation" tvOS Devuelve si el Siri Remote estará bloqueado por rotación (es decir, sólo puede ser utilizado verticalmente) o no. Cuando está desbloqueado, la rotación del mando cambiará la entrada, de forma que una rotación de 90º a la izquierda hará que el lado "derecho" del panel táctil vuelva a estar "arriba", etc... de forma que toda la entrada será relativa a la orientación del dispositivo Booleano
"dpad_absolute" tvOS Devuelve si el panel táctil del Siri Remote acepta entradas absolutas o relativas. En el modo absoluto, el centro del panel táctil se considera la posición (0,0) - es decir, donde el stick analógico está en reposo o donde estaría el centro del dpad - y el movimiento alrededor de esta posición generará la entrada. Cuando se establece en modo relativo, la posición (0,0) se considerará donde se realice el toque inicial en el pad y toda la entrada se generará desde esa posición, no desde el centro del pad. Booleano

Sintaxis:

gamepad_get_option(device, option_name);

ArgumentoTipo Descripción
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

Devuelve:

Boolean, Real, String, or N/A (depends on platform and option)

Ejemplo:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

El código anterior loops a través de todas las ranuras del gamepad y comprueba el "Siri Remote" en la plataforma tvOS. Si se detecta uno, entonces configura el mando para permitir la rotación de la entrada.
