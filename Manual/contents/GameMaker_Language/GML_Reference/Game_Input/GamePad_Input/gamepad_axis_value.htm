gamepad_axis_value

Puedes utilizar esta función para obtener el valor de los diferentes ejes de una ranura de gamepad determinada. Se suministra el índice de la ranura del gamepad a comprobar, junto con una constante de índice de eje (como se indica aquí), o un valor entero entre 0 y gamepad_axis_count()-1. El valor devuelto estará entre -1 y 1 para cada uno de los ejes horizontales y verticales disponibles.

Tenga en cuenta que el valor devuelto se normaliza para tener en cuenta el valor establecido para la zona muerta del eje (utilizando la función gamepad_set_axis_deadzone()), y por lo tanto siempre estará entre -1 y 0, y 0 y 1. Por ejemplo, el establecimiento de la zona muerta a 0.2 significará que empujando el stick hacia la derecha sólo empezará a devolver un valor de 0 - 1 cuando el valor bruto del eje sea superior a 0,2 - así, por ejemplo, cuando el valor bruto sea 0,2, el valor de retorno será 0, cuando el valor bruto esté en 0,5, el valor de retorno será 0,375, o si el valor bruto es 0,9, entonces el valor de retorno será 0,875.

Sintaxis:

gamepad_axis_value(device, axisIndex);

Argumento Tipo Descripción
device Real Which gamepad device "slot" to check.
axisIndex Constante del eje del Gamepad The axis index to check (see the constants list).

Devuelve:

Real

Ejemplo:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

El código anterior establece la velocidad y la dirección de la instancia basándose en los valores devueltos del controlador analógico izquierdo del gamepad conectado a la "ranura" 0 del dispositivo.
