gamepad_hat_count

Esta función devolverá el número total de sombreros disponibles para el gamepad conectado a la "ranura" del dispositivo dado. Los sombreros generalmente se refieren a los botones arriba/abajo/izquierda/derecha, y tenga en cuenta que en el objetivo Windows, los sombreros sólo están disponibles en los controladores DirectInput (es decir, a partir de la ranura 4).

Sintaxis:

gamepad_hat_count(device);

ArgumentoTipo Descripción
device Which gamepad device "slot" to check.

Devuelve:

Ejemplo:

h_num = gamepad_hat_count(4);

El código anterior devolverá el número de sombreros disponibles en el gamepad conectado a la "ranura" 4 del dispositivo y almacenará el valor en la variable "h_num".
