gamepad_get_guid

Con esta función puedes recuperar el GUID del gamepad conectado al índice de la ranura dada. Se suministra la ranura del gamepad a comprobar (de 0 a 11), y la función devolverá una de las siguientes cadenas:

  • "ninguno" si no hay ningún GUID disponible o no hay ningún gamepad conectado
  • "índice del dispositivo fuera de rango" si el índice de la ranura del gamepad está fuera de rango
  • El GUID string, algo así como "050000005e040000fd020000ffff3f00", si la ranura tiene un gamepad asignado y puede obtener el GUID para él

Esta función suele utilizarse junto con gamepad_get_description() para generar parte de la SDL string necesaria para la reasignación de controladores en Windows, Android y macOS.

Notas específicas de la plataforma
Plataforma Nota
Opera GX En la plataforma de destino Opera GX, esta función siempre devolverá "none" para los dispositivos no DirectInput

Sintaxis:

gamepad_get_guid(index);

ArgumentoTipo Descripción
index Which gamepad "slot" index to check

Devuelve:

Ejemplo:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

El código anterior obtiene el GUID y la descripción del nombre strings, luego los concatena y almacena el string final en una variable global para futuras referencias.
