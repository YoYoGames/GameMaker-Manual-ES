gamepad_get_description

Esta función devolverá un string con la descripción del gamepad dado, por ejemplo, un mando de PS3 puede devolver un string similar a "PLAYSTATION(R)3 Controller", o un mando de XBox360 podría devolver "Xbox 360 Controller (XInput STANDARD GAMEPAD)". Este string depende del hardware y el valor devuelto dependerá del gamepad conectado a la "ranura" del dispositivo que se está comprobando.

Sintaxis:

gamepad_get_description(device);

ArgumentoTipo Descripción
device Which gamepad "slot" to get the name of.

Devuelve:

Ejemplo:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    var _gamepad = gamepads[i];
    if (gamepad_is_connected(_gamepad))
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(_gamepad));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "Gamepad not found");
    }
}

El código anterior loop a través de todas las ranuras de gamepad para comprobar los dispositivos conectados y luego dibujar algún texto en la pantalla en función de si un gamepad está conectado a la ranura o no.
