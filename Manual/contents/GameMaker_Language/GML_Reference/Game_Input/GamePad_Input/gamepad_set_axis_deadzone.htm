gamepad_set_axis_deadzone

Esta función puede utilizarse para establecer la "zona muerta" del eje del joystick. Se especifica la ranura del dispositivo a ajustar, y luego se establece un valor de 0 a 1 y si la cantidad de entrada es menor que el valor dado, se considera que el eje del joystick está en 0. Tenga en cuenta que este es un ajuste global que afectará a todos los ejes de todos los joysticks conectados a la ranura del dispositivo especificado.

El uso de esta función no significa que el valor del eje parta del valor de la zona muerta, ya que la función gamepad_axis_value() siempre devolverá un valor normalizado entre -1 y 0 o 0 y 1. Por ejemplo, establecer la zona muerta en 0,2 significará que al empujar el stick hacia la derecha sólo empezará a devolver un valor de 0 - 1 cuando el valor bruto del eje sea superior a 0,2 - por lo que cuando el valor bruto sea 0,2, el valor de retorno será 0, cuando el valor bruto esté en 0,5, el valor de retorno será 0,375, o si el valor bruto es 0,9, entonces el valor de retorno sería 0,875.

Sintaxis:

gamepad_set_axis_deadzone(device, deadzone);

ArgumentoTipo Descripción
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

Devuelve:

Ejemplo:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

El código anterior establecerá la zona muerta del eje del joystick en cualquiera de los 4 dispositivos conectados a 0,05.
