gamepad_get_device_count

Esta función te da el número de pads de juego conectados, o el número de "ranuras" disponibles para que los pads de juego se conecten.

El valor real de retorno dependerá de la plataforma y de la configuración interna de la misma, por lo que esta función debe utilizarse junto con la función gamepad_is_connected() para asegurarse del número exacto de pads conectados en cada momento.

Sintaxis:

gamepad_get_device_count();

Devuelve:

Real

Ejemplo:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

El código anterior loops a través de los pads de juego disponibles (o ranuras de gamepad) y luego comprueba cada uno para un gamepad conectado. el valor devuelto se utiliza entonces para establecer una matriz global a true o false para su uso en futuras comprobaciones.
