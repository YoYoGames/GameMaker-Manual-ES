gamepad_get_axis_deadzone

Esta función puede utilizarse para obtener el valor de la "zona muerta" del eje del joystick. Se especifica la ranura del dispositivo a obtener, y la función devolverá un valor entre 0 y 1, donde el valor refleja el umbral por debajo del cual se considera que el eje del joystick está en 0.

Sintaxis:

gamepad_get_axis_deadzone(device);

ArgumentoTipo Descripción
device Which gamepad device "slot" to check.

Devuelve:

Ejemplo:

if gamepad_get_axis_deadzone(global.PadId) != 0.5
{
    gamepad_set_axis_deadzone(global.PadId, 0.5);
}

El código anterior comprueba el umbral del eje analógico del gamepad conectado al dispositivo "slot" almacenado en la variable global y si no es el valor por defecto de 0,5, se establece en ese valor.
