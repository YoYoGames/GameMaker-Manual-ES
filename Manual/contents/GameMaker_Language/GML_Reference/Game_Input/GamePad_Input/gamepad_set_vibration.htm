gamepad_set_vibration

Con esta función puedes establecer la vibración de los motores del gamepad, con cualquiera de los motores utilizando un valor de 0 (sin vibración) a 1 (vibración completa). Ten en cuenta que no hay límite de tiempo en esta función, por lo que tendrás que utilizar una variable o una alarma para switch desactivar la vibración (poner los motores a 0) después de que haya pasado un tiempo determinado, de lo contrario el gamepad seguirá vibrando indefinidamente.

NOTA: Esta función sólo está disponible actualmente para los módulos de destino estándar de Windows, PS4 y Xbox One.

Sintaxis:

gamepad_set_vibration(device, left_motor, right_motor);

ArgumentoTipo Descripción
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Devuelve:

Ejemplo:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

El código anterior se utilizaría (por ejemplo) en un evento de colisión para hacer que el gamepad conectado a la "ranura" 0 vibre durante medio segundo, y la alarma que se establezca se utilizará para switch para apagarlo de nuevo una vez que haya pasado ese tiempo.
