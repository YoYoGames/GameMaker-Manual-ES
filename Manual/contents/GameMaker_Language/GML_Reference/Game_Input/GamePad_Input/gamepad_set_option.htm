gamepad_set_option

Esta función puede utilizarse para establecer cualquiera de las opciones disponibles del gamepad. Debes suministrar la "ranura" del gamepad para establecer la opción, así como la opción string a establecer y el valor a utilizar. La opción disponible string dependerá de la plataforma en la que se ejecute el proyecto, como se indica a continuación:

Opción Plataforma Descripción Valor
"allow_rotation" tvOS Establezca si el Siri Remote estará bloqueado en cuanto a la rotación (es decir, sólo puede ser utilizado verticalmente) o no. Cuando está desbloqueado (un valor de true) la rotación del mando cambiará la entrada, de forma que una rotación de 90º a la izquierda hará que el lado "derecho" del panel táctil vuelva a estar "arriba", etc... de forma que toda la entrada será relativa a la orientación del dispositivo. Cuando se bloquea (un valor de false) entonces la entrada no se cambiará en relación con la posición del controlador y siempre se tratará como si el controlador estuviera en posición vertical. Booleano
"dpad_absolute" tvOS Establece si el panel táctil del Siri Remote acepta entradas absolutas o relativas. En modo absoluto (un valor de true) el centro del panel táctil se considera la posición (0,0) - es decir, donde el stick analógico está en reposo o donde estaría el centro del dpad - y el movimiento alrededor de esta posición generará la entrada. Cuando se establece como relativo (un valor de false ) entonces la posición (0,0) se considerará como donde se hace el toque inicial en el pad y toda la entrada se generará desde esa posición, no el centro del pad. Booleano

Sintaxis:

gamepad_set_option(device, option_name, value);

Argumento Tipo Descripción
device Real Which gamepad device "slot" to set.
option_name Cadena The name of the option to set (a string, see the table above)
value Real, booleano o cadena The value to set the option to (can be boolean, real or string - see the table above)

Devuelve:

N/A

Ejemplo:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

El código anterior loops a través de todas las ranuras del gamepad y comprueba el "Siri Remote" en la plataforma tvOS. Si se detecta uno, entonces configura el mando para permitir la rotación de la entrada.
