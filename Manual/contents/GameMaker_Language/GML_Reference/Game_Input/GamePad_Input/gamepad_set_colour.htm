gamepad_set_colour

Esta función se puede utilizar para establecer el color de los LEDs dentro de un controlador de PlayStation. Se especifica la ranura del dispositivo que se va a configurar, y luego se da un color, que puede ser cualquiera de las constantes de color o un valor de color creado utilizando las funciones de color específicas o un valor HEX (como $FFFFFFF).

Sintaxis:

gamepad_set_colour(device, colour);

ArgumentoTipo Descripción
device Which gamepad device "slot" to set.
colour The colour to use.

Devuelve:

Ejemplo:

if health < 10
{
    gamepad_set_colour(0, c_red);
}

El código anterior pondrá los LEDs del controlador PlayStation en rojo si la variable health cae por debajo de 10.
