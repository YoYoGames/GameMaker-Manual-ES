gamepad_is_connected

Esta función devolverá si un gamepad está conectado a la "ranura" dada (devuelve true) o no (devuelve false). Normalmente se utiliza esta función junto con la función gamepad_get_device_count() para obtener el número correcto de gamepads y/o "ranuras" de gamepad disponibles. Tenga en cuenta que puede haber un ligero retraso entre el usuario que conecta el gamepad y GameMaker que lo detecta como conectado (esto es especialmente el caso cuando se trata de controladores conectados por bluetooth).

NOTA: En algunas plataformas -especialmente en las consolas- esta función puede devolver un falso inmediatamente después de que se haya conectado/seleccionado un gamepad y puede ser necesario comprobar en una alarma unos cuantos fotogramas (pasos) después para que detecte correctamente el gamepad.

Sintaxis:

gamepad_is_connected(numb);

ArgumentoTipo Descripción
device Which gamepad "slot" to check.

Devuelve:

Ejemplo:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++)
{
    if gamepad_is_connected(i)
        global.gp[i] = true
    else
        global.gp[i] = false;
}

El código anterior loops a través de los pads de juego disponibles (o ranuras de gamepad) y luego comprueba cada uno para un gamepad conectado. el valor devuelto se utiliza entonces para establecer una matriz global a true o false para su uso en futuras comprobaciones.
