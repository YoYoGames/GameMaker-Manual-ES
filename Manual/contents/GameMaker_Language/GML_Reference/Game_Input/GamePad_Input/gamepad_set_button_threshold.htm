gamepad_set_button_threshold

Esta función permite establecer el ajuste del umbral actual de los botones analógicos para un dispositivo determinado. El umbral por defecto para todos los botones analógicos es 0,5, con un rango de 0 a 1.

El umbral define en qué momento se considera que el botón está "pulsado" para los juegos que requieren que actúen como botón digital.

Esta función afectará a los estados de comprobación, pulsación y liberación de los botones analógicos, pero no afectará al valor devuelto por la función gamepad_button_value()que siempre devolverá el valor bruto del botón.

Sintaxis:

gamepad_set_button_threshold(device, threshold);

Argumento Tipo Descripción
device Real Which gamepad device "slot" to check.
threshold Real The new threshold value (from 0 - 1, default 0.5).

Devuelve:

N/A

Ejemplo:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

El código anterior comprueba el umbral del botón analógico del gamepad conectado a la "ranura" 0 del dispositivo y si no es el valor por defecto de 0,5, se establece en ese valor.
