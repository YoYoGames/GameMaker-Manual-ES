gamepad_button_value

Con esta función puedes obtener el valor actual de un botón analógico, de 0 a 1, donde 0 es sin presión y 1 es presión total. Se suministra el índice de la ranura del gamepad que se va a comprobar, junto con una constante de botón (como se indica aquí), o un valor entero entre 0 y gamepad_button_count() - 1.

Tenga en cuenta que esto devolverá el valor bruto del botón, y no tiene en cuenta la configuración del umbral (ver aquí para más información sobre los umbrales).

Sintaxis:

gamepad_button_value(device, button);

Argumento Tipo Descripción
device Real Which gamepad device "slot" to check.
button Botón constante del Gamepad Which gamepad button constant to check for.

Devuelve:

Real

Ejemplo:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

El código anterior utiliza el valor de disparo analógico del gamepad conectado a la "ranura" 0 del dispositivo para establecer la velocidad de la instancia.
