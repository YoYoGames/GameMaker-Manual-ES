gamepad_hat_value

Con esta función puedes obtener el valor actual de un "sombrero" del gamepad. Cada valor del sombrero es una máscara de bits para las diferentes direcciones donde:

  • Arriba = 1
  • Derecho = 2
  • Abajo = 4
  • Izquierda = 8

Ten en cuenta que se pueden combinar (por ejemplo, suministrando un índice de sombrero de 3 estaría comprobando hacia arriba y hacia la derecha, o puedes usar el operador | para combinar los distintos valores) pero sólo se pueden pulsar como máximo 2 bits a la vez. La función devolverá un valor real entre 0 y 1, donde 0 es no pulsado y 1 es totalmente pulsado (y puede haber valores intermedios dependiendo de si el gamepad soporta entrada analógica para sombreros).

Sintaxis:

gamepad_hat_value(device, hatindex);

Argumento Tipo Descripción
device Real Which gamepad device "slot" to check.
hatindex Real Which gamepad hat (or hats) to check for.

Devuelve:

Real

Ejemplo:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
}

El código anterior almacena el estado del sombrero "0" en una variable local, luego comprueba cuál es el valor de retorno y actúa en consecuencia.
