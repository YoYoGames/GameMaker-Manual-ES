gamepad_get_button_threshold

Esta función permite detectar el umbral actual de los botones analógicos de un dispositivo determinado. El umbral por defecto para todos los botones analógicos es 0,5, con un rango de 0 a 1. El umbral define en qué momento se considera que el botón está "pulsado" para los juegos que requieren que actúen como un botón digital.

Sintaxis:

gamepad_get_button_threshold(device);

ArgumentoTipo Descripción
device Which gamepad device "slot" to check.

Devuelve:

Ejemplo:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

El código anterior comprueba el umbral del botón analógico del gamepad conectado a la "ranura" 0 del dispositivo y si no es el valor por defecto de 0,5, se establece en ese valor.
