gamepad_is_supported

Con esta función puedes averiguar si la plataforma de destino soporta game pads (devuelve true) o no (devuelve false).

Sintaxis:

gamepad_is_supported();

Devuelve:

Ejemplo:

global.GP = gamepad_is_supported();

El código anterior comprueba si un gamepad es compatible y almacena el valor de retorno en una variable global para futuras comprobaciones.
