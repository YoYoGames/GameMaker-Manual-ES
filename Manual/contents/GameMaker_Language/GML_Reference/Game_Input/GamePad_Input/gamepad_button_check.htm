gamepad_button_check

Esta función devolverá true o false dependiendo de si se detecta que el botón del gamepad está pulsado o no. Si está comprobando un botón analógico, la comprobación tendrá en cuenta el ajuste del umbral y sólo devolverá true mientras el valor bruto de entrada esté por encima del umbral dado (puede obtener este valor bruto utilizando la función gamepad_button_value()).

Sintaxis:

gamepad_button_check(device, button);

Argumento Tipo Descripción
device Real Which gamepad device "slot" to check.
button Botón constante del Gamepad Which gamepad button constant to check for.

Devuelve:

Boolean

Ejemplo:

if gamepad_button_check(0, gp_face1)
{
    if (canshoot = true)
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

El código anterior detectará si el botón inferior de la cara superior (la "X" en un mando de ps3) se mantiene pulsado y si es así, disparará una instancia "bala" y activará una alarma.
