gamepad_get_mapping

Esta función se puede utilizar para recuperar el mapeo string para el gamepad. Este string será cualquiera:

  • El mapa actual string para el gamepad, por ejemplo:

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • Un string con "sin mapeo" si no hay mapeo establecido ( no es lo mismo que cuando no hay mapeo disponible)
  • Un string con "índice de dispositivo fuera de rango" si el índice de la ranura no es un índice de gamepad válido
  • Un string vacío "" si no hay ninguna asignación disponible para el gamepad conectado

Para más información sobre el formato del mapa devuelto string, consulte la función gamepad_test_mapping().

Sintaxis:

gamepad_get_mapping(index);

ArgumentoTipo Descripción
index Which gamepad index "slot" to get the mapping from.

Devuelve:

Ejemplo:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

El código anterior obtiene el mapa string para la ranura del gamepad dada, y luego lo envía a la consola para debugging.
