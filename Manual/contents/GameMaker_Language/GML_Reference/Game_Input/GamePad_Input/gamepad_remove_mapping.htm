gamepad_remove_mapping

Esta función se puede utilizar para eliminar el mapeo del dispositivo actual del índice de ranura del gamepad dado. Una vez llamada, el índice de la ranura tendrá que ser reasignado usando la función gamepad_test_mapping() si quieres ser capaz de usar las constantes del gamepad para detectar la entrada correctamente (la entrada directa siempre puede ser recuperada usando las funciones gamepad_axis/button/hat_count() y gamepad_axis/button/hat_value() juntas).

Sintaxis:

gamepad_remove_mapping(index);

ArgumentoTipo Descripción
index Which gamepad index "slot" to remove the mapping from.

Devuelve:

Ejemplo:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

El código anterior eliminará el mapeo de la ranura de índice del gamepad dado en base al valor de una variable.
