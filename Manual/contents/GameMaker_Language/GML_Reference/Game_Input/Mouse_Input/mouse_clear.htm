mouse_clear

Esta función borrará el estado actual del botón del ratón dado. Esto significa que la comprobación de que se mantiene pulsado no devolverá true hasta que el jugador suelte el botón y lo pulse de nuevo (pero el estado de liberación seguirá siendo detectado si el borrado se realiza mientras se mantiene pulsado el botón del ratón).

Puede suministrar una de las constantes del botón del ratón que aparecen en esta página.

Sintaxis:

mouse_clear(button);

Argumento Tipo Descripción
button Constante del botón del ratón Which mouse button constant to check for.

Devuelve:

Boolean

Ejemplo:

mouse_clear(mb_any);

El código anterior borrará el estado de inactividad de todos los botones del ratón.
