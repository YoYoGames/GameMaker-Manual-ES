mouse_wheel_up

Esta función devuelve true si la rueda del ratón está girando hacia arriba y false en caso contrario.

Sintaxis:

mouse_wheel_up();

Devuelve:

Ejemplo:

if mouse_wheel_up()
{
    y -= 10;
}

Esto mueve la instancia actual hacia arriba en la pantalla si la rueda del ratón se gira hacia arriba.
