botón_del_ratón

Esta variable devuelve el último botón del ratón que se ha pulsado y puede devolver cualquiera de las constantes especiales del ratón excepto mb_any (también se puede establecer esta variable a una de las constantes).

Sintaxis:

mouse_lastbutton;

Devuelve:

Mouse Button Constant (except mb_any)

Ejemplo:

if mouse_lastbutton = mb_left
{
    x -= 1;
}

Esto mueve la instancia actual a la izquierda si el último botón pulsado fue el botón izquierdo del ratón.
