ratón_y

Esta variable de sólo lectura devuelve la posición actual del eje y del ratón dentro de room.

Sintaxis:

mouse_y;

Devuelve:

Ejemplo:

y = median(64, mouse_y, room_height - 64);

El código anterior mantendrá la instancia en la posición y del ratón siempre que esté dentro de los límites de 64 píxeles desde la parte superior e inferior de room.
