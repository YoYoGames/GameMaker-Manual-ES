mouse_check_button_released

Esta función devolverá true si el botón del ratón que se está comprobando ha sido liberado o false si no lo ha sido. Esta función sólo se activará una vez para cualquier botón del ratón cuando se suelte y para activarla de nuevo el botón tendrá que haber sido pulsado y soltado de nuevo. El botón del ratón a comprobar se suministra a partir de una de las siguientes constantes:

Sintaxis:

mouse_check_button_released(numb);

Argumento Tipo Descripción
numb Constante del botón del ratón Which mouse button constant to check for.

Devuelve:

Boolean

Ejemplo:

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

El código anterior comprobará si se ha soltado el botón derecho del ratón y, en caso afirmativo, establecerá la velocidad de la instancia a una décima parte de la distancia entre la posición actual x/y y la posición x/y del ratón.
