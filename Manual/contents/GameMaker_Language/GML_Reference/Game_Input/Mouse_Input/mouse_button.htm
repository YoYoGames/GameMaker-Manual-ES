botón_del_ratón

Esta variable de sólo lectura devuelve el botón del ratón que se está pulsando actualmente (actualmente, como en este paso) y puede devolver cualquiera de las constantes especiales del ratón excepto mb_any.

Sintaxis:

mouse_button;

Devuelve:

Mouse Button Constant (except mb_any)

Ejemplo:

if mouse_button = mb_left
{
    x -= 1;
}

Esto mueve la instancia actual a la izquierda si el botón izquierdo del ratón está presionado.
