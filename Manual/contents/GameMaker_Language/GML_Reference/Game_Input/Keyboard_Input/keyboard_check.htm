teclado_comprobación

Con esta función puedes comprobar si una tecla está pulsada o no. A diferencia de la función keyboard_check_pressed() o keyboard_check_released() que sólo se activan una vez al pulsar o soltar la tecla, esta función se activa cada vez que se mantiene la tecla pulsada. La función tomará un valor de código de tecla devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes de vk_* listadas en la página principal de Entrada del Teclado.

Sintaxis:

keyboard_check(key);

Argumento Tipo Descripción
key Constante de la llave virtual (vk_*) The key to check the down state of.

Devuelve:

Boolean

Ejemplo:

if keyboard_check(vk_left)
{
    x -= 5;
}

El código anterior comprobará si se está pulsando la tecla de flecha y moverá la instancia 5 píxeles a la izquierda cada vez que devuelva true.
