keyboard_key_press

Con esta función se puede simular la pulsación de cualquier tecla del teclado. Cuando se utilice, la tecla se marcará como pulsada hasta que se llame a la función de liberación correspondiente (véase keyboard_key_release()). La función tomará un valor de código de tecla devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes de vk_* listadas en la página principal de Entrada de Teclado.

Tenga en cuenta que después de llamar a esta función, si la tecla se pulsa físicamente en el teclado, este evento de pulsación no se registrará de nuevo hasta que la tecla se haya liberado físicamente (activando el evento de liberación y deteniendo esta función), o se haya llamado a la función de liberación de la tecla correspondiente y se haya vuelto a pulsar la tecla.

Sintaxis:

keyboard_key_press(key);

ArgumentoTipo Descripción
key The key to simulate a press of.

Devuelve:

Ejemplo:

keyboard_key_press(vk_space);

Esto simulará una pulsación de la barra espaciadora.
