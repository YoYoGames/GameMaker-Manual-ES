keyboard_check_direct

Esta función devolverá true si la tecla con el código de tecla concreto está pulsada, o false si no lo está, comprobando el hardware directamente. Permite algunas comprobaciones más, en particular puede utilizar los códigos de teclas vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol y vk_ralt para comprobar si la tecla shift, control o alt izquierda o derecha está pulsada. La función tomará un valor de código de teclas devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes de vk_* listadas en la página principal de Entrada del Teclado.

NOTA Esta función sólo está disponible para el objetivo estándar Windows y el resultado es independiente de la aplicación que tenga el foco.

Sintaxis:

keyboard_check_direct(key);

ArgumentoTipo Descripción
key The key to check the down state of.

Devuelve:

Ejemplo:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

El código anterior comprobará si se han pulsado las teclas alt izquierda o derecha, y si lo han hecho establece la variable "crouch" a true.
