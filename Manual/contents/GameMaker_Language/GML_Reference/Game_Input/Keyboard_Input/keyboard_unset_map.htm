keyboard_unset_map

Con esta función puedes borrar todas las teclas re-mapeadas para que vuelvan a su estado por defecto, es decir, que todas las teclas se mapeen a sí mismas.

Sintaxis:

keyboard_get_map();

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

El código de ejemplo anterior restablecerá todas las teclas asignadas a su configuración por defecto si el usuario pulsa la tecla "escape".
