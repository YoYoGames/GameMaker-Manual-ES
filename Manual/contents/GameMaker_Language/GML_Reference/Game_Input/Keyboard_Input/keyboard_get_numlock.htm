keyboard_get_numlock

Puede utilizar esta función para encontrar el estado del bloqueo de números del teclado, devolviendo true para on, y false para off.

NOTA Esta funcionalidad sólo está disponible en las compilaciones de Windows exe y no funcionará en ningún otro dispositivo.

Sintaxis:

keyboard_get_numlock();

Devuelve:

Ejemplo:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

El código de ejemplo anterior obtendrá el estado de la tecla numberlock y si está activada (true) la pondrá en off (false) y viceversa.
