keyboard_check_released

Con esta función puede comprobar si una llave ha sido liberada o no. A diferencia de la función keyboard_check() esta función sólo se ejecutará una vez por cada vez que se levante la tecla, por lo que para que se active de nuevo, la tecla debe ser pulsada primero y luego liberada de nuevo. La función tomará un valor de código de tecla como el devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes de vk_* listadas en la página principal de Entrada del Teclado.

Sintaxis:

keyboard_check_released(key);

ArgumentoTipo Descripción
key The key to check the released state of.

Devuelve:

Ejemplo:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

El código anterior comprobará si la tecla "P" ha sido liberada y si es así, creará una instancia de "obj_Pause".
