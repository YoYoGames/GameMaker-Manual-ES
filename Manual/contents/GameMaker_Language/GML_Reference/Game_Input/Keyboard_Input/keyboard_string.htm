cadena_de_teclado

Esta variable contiene un string que contiene los últimos (como máximo) 1024 caracteres escritos en el teclado. Este string sólo contendrá los caracteres imprimibles escritos, pero responderá correctamente a la pulsación de la tecla de retroceso borrando el último carácter. Esta variable no es de sólo lectura y puedes cambiarla, por ejemplo para ponerla a "" (un string vacío) si ya la has manejado, y puedes usar las Funciones de Cadena para manipularla. Tenga en cuenta que cuando utilice el Teclado Virtual en pantalla, sólo se actualizará esta variable con la entrada del teclado.

Sintaxis:

keyboard_string;

Devuelve:

Ejemplo:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

El código anterior limitará la longitud del teclado string a 15 caracteres, eliminando los que superen ese límite copiando los primeros quince caracteres en la variable.
