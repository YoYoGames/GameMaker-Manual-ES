keyboard_lastchar

Esta variable almacena un string de la última tecla pulsada. Esta variable no es de sólo lectura y puedes cambiarla, por ejemplo para ponerla a "" (un string vacío) si ya la has manejado.

Sintaxis:

keyboard_lastchar;

Devuelve:

Ejemplo:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

El código anterior comprueba si la variable lastkey es mayor que -1, y si lo es añade lo que haya sido la última tecla como string a la variable "str".
