keyboard_set_numlock

Puede utilizar esta función para switch activar o desactivar el bloqueo numérico del teclado (si está activado es verdadero y si está desactiv ado es falso ).

NOTA Esta funcionalidad sólo está disponible en las construcciones de destino Windows y no funcionará en ningún otro dispositivo.

Sintaxis:

keyboard_set_numlock(value);

Argumento Tipo Descripción
value Booleano Set this to true for "on" and false for "off"

Devuelve:

N/A

Ejemplo:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

El código de ejemplo anterior obtendrá el estado de la tecla numberlock y si está activada (true) la pondrá en off (false) y viceversa.
