keyboard_set_map

A veces, cuando se hace un juego, se desea que una tecla haga lo mismo que otra. Por ejemplo, mucha gente utiliza las teclas WASD para moverse, pero también hay mucha gente que utiliza las teclas de dirección. Entonces, ¿qué hacer? Bueno, podrías codificar el sistema de movimiento dos veces, pero eso es un poco complicado y afortunadamente redundante, ya que esta función te permite "mapear" una tecla a otra y así cualquier entrada de cualquier tecla será interpretada como la misma. Para ello, elija una tecla que desee asignar (tecla2 - esta será la tecla para la que escribas el código) y una tecla a la que quieres que se mapee (tecla1). Después de esto, las pulsaciones de cualquiera de las dos teclas serán interpretadas por GameMaker como procedentes de la tecla2. También puede utilizar esta función para diseñar un sistema en el que el usuario pueda definir sus propias teclas para jugar, simplemente asignando la tecla de entrada del usuario a la tecla que usted ha codificado en el juego.

La función toma un valor de código de teclas devuelto por la función ord() (sólo letras mayúsculas de la A a la Z o números del 0 al 9), o cualquiera de las constantes de vk_* listadas en la página principal de Entrada de Teclado.

¡IMPORTANTE! La tecla a la que está asignando (tecla1) dejará de ser utilizable como su símbolo de tecla real una vez que utilice esta función. Por ejemplo, si asigna la flecha hacia arriba a "W", ya no podrá detectar la pulsación de la tecla "W" como una "W", siempre será considerada como la flecha hacia arriba. Para deshacer esto, o bien asigna la tecla a sí misma - por lo que tanto la tecla1 como la tecla2 serían ord("W") - o utiliza la función keyboard_unset_map().

Sintaxis:

keyboard_set_map(key1, key2);

ArgumentoTipo Descripción
key1 This is the key that key1 is to be mapped to
key2 This is the key that is to be mapped

Devuelve:

Ejemplo:

keyboard_set_map(ord("A"), vk_left);

El código de ejemplo anterior asignará la tecla "A" a la tecla de la flecha izquierda. Esto significa que el jugador puede usar la tecla "A" o la flecha izquierda, y que todo el código escrito para la flecha izquierda también responderá a la tecla "A" en su lugar.
