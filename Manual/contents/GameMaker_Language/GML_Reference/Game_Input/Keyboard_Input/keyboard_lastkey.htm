teclado_última tecla

Esta variable se refiere al valor que keyboard_key estaba en el fotograma anterior, devolviendo el keycode de esa tecla (se devuelven todas las constantes de keycode estándar). Esta variable no es de sólo lectura y puedes cambiarla, por ejemplo para ponerla a -1 si ya la manejaste.

Sintaxis:

keyboard_lastkey;

Devuelve:

Virtual Key Constant

Ejemplo:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

El código anterior comprueba si la variable lastkey no es igual a -1, y si lo es añade lo que haya sido la última tecla como string a la variable "str", y luego reinicia la variable keyboard_lastkey para aceptar más entradas.
