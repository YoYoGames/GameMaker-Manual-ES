device_mouse_x

Esta función devuelve la posición x de un toque en el dispositivo. Si está ejecutando esto en los módulos HTML5 o PC y Mac, entonces este valor se actualiza constantemente, siempre y cuando el dispositivo (por lo general un ratón) esté conectado, sin embargo, para los dispositivos móviles, esto sólo se actualizará mientras la pantalla está siendo tocada y tenga en cuenta que el número máximo de toques que se pueden detectar dependerá en gran medida del dispositivo que se utiliza y el sistema operativo que ejecuta.

Sintaxis:

device_mouse_x(device);

ArgumentoTipo Descripción
device The device (from 0 - n) that is being checked.

Devuelve:

Ejemplo:

if device_mouse_check_button(0, mb_left)
{
    if (device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32)
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

El código anterior comprueba si el dispositivo está siendo presionado y, si es así, sondea la posición x del dispositivo para ver si está dentro de los parámetros. Si lo está, establece la variable "pressed" a true, de lo contrario la establece a false.
