device_mouse_check_button

Esta función devuelve true o false dependiendo de si el botón del ratón dado se mantiene pulsado en el dispositivo dado. Para el botón del ratón se puede suministrar una de las constantes enumeradas en esta página.

Esta función se puede utilizar para pantallas táctiles, y el argumento del dispositivo puede ser de 0 a n, donde n es el número máximo de "dedos" que pueden estar tocando la pantalla a la vez (el número máximo de toques que pueden dependerá en gran medida del dispositivo que se utilice y del sistema operativo que ejecute, pero la mayoría de los dispositivos deberían detectar al menos hasta 4). Tenga en cuenta que mb_right sólo se detectará si se detecta un doble toque (este comportamiento se puede desactivar utilizando la función device_mouse_dbclick_enable()).

Sintaxis:

device_mouse_check_button(device, button);

ArgumentoTipo Descripción
device The device (from 0 - n) that is being checked
button The button of the device that is being checked.

Devuelve:

Ejemplo:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

El código anterior comprueba si los dos toques se mantienen al mismo tiempo y si lo hacen entonces pasa a otro room.
