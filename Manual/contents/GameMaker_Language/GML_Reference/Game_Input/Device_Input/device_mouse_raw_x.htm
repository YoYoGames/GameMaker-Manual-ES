device_mouse_raw_x

Esta función devuelve la posición x cruda de un toque en el dispositivo. Esto significa que devuelve la definición real del dispositivo de la posición x que se está tocando, no la de GameMaker, y como tal ignorará cosas como la posición de la vista y la escala. Tenga en cuenta que el número máximo de toques que se puede detectar dependerá en gran medida del dispositivo que se utiliza y el sistema operativo que se ejecuta

NOTA: Esta función depende en gran medida del dispositivo y debe experimentar primero con el módulo y el dispositivo de destino deseados para ver qué se devuelve exactamente.

Sintaxis:

device_mouse_raw_x(device);

ArgumentoTipo Descripción
device The device (from 0 - n) that is being checked.

Devuelve:

Ejemplo:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));
    y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1));
}

El código anterior comprueba si el dispositivo1 y el dispositivo2 están siendo pulsados, y si lo están calcula la posición media de las coordenadas x/y entre cada punto de pulsación.
