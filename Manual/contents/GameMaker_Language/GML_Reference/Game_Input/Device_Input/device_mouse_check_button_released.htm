device_mouse_check_button_released

Esta función devuelve true o false dependiendo de si el toque del dispositivo que se especifica ha sido liberado o no. El argumento dispositivo se refiere al número de toques del dispositivo, que puede ser de 0 a n, y tenga en cuenta que el número máximo de toques que se pueden detectar dependerá en gran medida del dispositivo que se utilice y del sistema operativo que ejecute (la mayoría de los dispositivos soportarán al menos 4 toques simultáneos). Esta función sólo se dispara una vez por la acción de soltar, y las constantes listadas en esta página pueden usarse para comprobar los botones del ratón. Tenga en cuenta que mb_right sólo se habrá detectado si se detecta un toque doble y se mantiene en el segundo toque antes de soltarlo (este comportamiento se puede desactivar utilizando la función device_mouse_dbclick_enable()).

Sintaxis:

device_mouse_check_button_released(device, button);

ArgumentoTipo Descripción
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Devuelve:

Ejemplo:

if device_mouse_check_button_released(0, mb_left)
{
    press = false;
}

El código anterior comprueba si el dispositivo ha sido pulsado y establece una variable a false si es así.
