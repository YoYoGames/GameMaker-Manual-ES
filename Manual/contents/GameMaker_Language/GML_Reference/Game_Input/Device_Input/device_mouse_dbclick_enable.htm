device_mouse_dbclick_enable

Esta función permite configurar el dispositivo para que detecte un doble toque del mb_left (botón izquierdo del ratón) como una entrada de mb_right (botón derecho del ratón).

Por defecto está configurado en true, lo que significa que cada vez que el usuario toca la pantalla del dispositivo dos veces de forma rápida y consecutiva, el valor de retorno es el mismo que si se hubiera pulsado el botón derecho del ratón. Cuando esto está activado, el primer toque será detectado como mb_left, y el segundo como mb_right, así que asegúrate de que cualquier código que utilices tenga esto en cuenta.

Sintaxis:

device_mouse_dbclick_enable(bool);

Argumento Tipo Descripción
bool Booleano Set double-click detection on (true) o apagado (false).

Devuelve:

Boolean

Ejemplo:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

El código anterior comprueba si el dispositivo que ejecuta el juego es un PC Windows o un Mac y, si es cualquiera de ellos, desactiva la función de doble toque.
