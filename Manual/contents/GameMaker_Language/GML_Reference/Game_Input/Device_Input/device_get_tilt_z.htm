device_get_tilt_z

Esta función devuelve un valor entre -1 y 1 dependiendo del ángulo de "inclinación" del dispositivo. La correlación real entre los grados de inclinación y el valor devuelto depende del dispositivo y del sistema operativo que utilice, pero generalmente un valor de 1 o -1 es lo mismo que +/-90°. Ejemplo:

y += sign(device_get_tilt_z());

El código anterior utiliza el valor de la inclinación para mover al jugador a lo largo del eje y.
