shader_get_name

Con esta función se puede recuperar el nombre de un recurso shader. Usted proporciona el valor de identificación único para el shader para obtener el nombre de y la función devolverá el nombre del recurso como un string.

Sintaxis:

shader_get_name(shader);

ArgumentoTipo Descripción
shader The index of the shader to get the name of.

Devuelve:

Ejemplo:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

El código anterior obtendrá el nombre del shader dado y lo dibujará en la pantalla.
