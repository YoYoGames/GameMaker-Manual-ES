shader_set_uniform_matrix_array

Con esta función se puede establecer una constante shader para mantener un array de valores de la matriz. Deberás haber obtenido previamente el "handle" de la constante utilizando la función shader_get_uniform(), y tendrás que haber inicializado previamente la matriz como una matriz de valores en coma flotante, donde cada grupo sucesivo de 16 flotantes es una matriz de 4x4.

NOTA: Todos los uniformes deben establecerse después de llamar a la función shader_set()y antes de llamar a la función shader_reset().

Sintaxis:

shader_set_uniform_matrix_array(handle, array);

Argumento Tipo Descripción
handle Manipulación de sombreado uniforme The handle of the shader constant to set.
array Matriz A previously initialised array of floating point values.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

El código anterior obtendrá el manejador de la constante shader "cMatrixArray" y luego establecerá esa constante al array dado.
