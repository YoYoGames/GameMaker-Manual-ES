shader_actualidad

Esta función devolverá el valor del ID del índice del shader que se está utilizando actualmente para el renderizado, o devolverá -1 si no se está utilizando ningún shader.

Sintaxis:

shader_current();

Devuelve:

Shader Asset or -1 (if no shader is assigned)

Ejemplo:

if (shader_current() == -1)
{
    shader_set(sh_warp);
}

El código anterior comprobará cuál es el shader actual y si devuelve -1 (no se está utilizando shader ) entonces se establece un shader.
