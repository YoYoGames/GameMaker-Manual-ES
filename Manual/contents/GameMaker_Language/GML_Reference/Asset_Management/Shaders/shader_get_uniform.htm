shader_get_uniform

Dado que no se puede cambiar el valor de una constante shader dentro de la propia shader, hay que fijarla antes de llamar a la shader utilizando una de las funciones de fijación uniforme disponibles. Sin embargo, para poder hacerlo, primero debe llamar a esta función para obtener el "handle" de la constante shader que querrá cambiar.

NOTA: Aunque un shader se compone de dos programas discretos (vertex y fragmento) esta función no diferenciará entre los dos y devolverá el handle de la constante shader de cualquiera de ellos.

Sintaxis:

shader_get_uniform(shader, uniform);

Argumento Tipo Descripción
shader Activo de sombreado The index of the shader to use.
uniform Cadena The shader constant to get the handle of (a string).

Devuelve:

Shader Uniform Handle

Ejemplo:

shader_params = shader_get_uniform(shd_glass, "u_vRefractColour");

El código anterior obtendrá el manejador de la constante shader "u_vRefractColour".
