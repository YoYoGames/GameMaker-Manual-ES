shader_is_compiled

Esta función comprobará un shader en tiempo de ejecución para asegurarse de que ha sido compilado con éxito. Si lo ha hecho, la función devuelve true, de lo contrario devuelve false. Esta función debe ser utilizada al inicio del juego para asegurarse de que la plataforma que ejecuta su juego ha compilado correctamente cualquier shaders utilizado (particularmente en Windows donde algunos ordenadores pueden estar utilizando DX9 con Shader nivel 2.0 y no una versión posterior utilizando shader nivel 3.0).

Si tu shader NO ha sido compilado y llamas a shader_set() el juego se bloqueará, por lo que merece la pena tener algún tipo de comprobación siempre que se utilice algo que no sea el simple GLSL ES shaders.

Sintaxis:

shader_is_compiled(shader);

ArgumentoTipo Descripción
shader The shader to check.

Devuelve:

Boolean

Ejemplo:

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

El código anterior establecerá una variable global a false, y luego si los dos shaders que se están comprobando han compilado correctamente, se establecerá a true.
