shader_reset

Esta función restablece el shader utilizado para dibujar y debe llamarse cuando ya no se desea utilizar el shader actual (establecido con shader_set()).

Sintaxis:

shader_reset();

Devuelve:

Ejemplo:

shader_set(shader_Glass);
draw_self();
shader_reset();

El código anterior establecerá un shader para ser usado para dibujar, luego dibujará el sprite actual usado para la instancia que lo usa. Por último, restablecerá el shader para volver al shader predeterminado de GameMaker.
