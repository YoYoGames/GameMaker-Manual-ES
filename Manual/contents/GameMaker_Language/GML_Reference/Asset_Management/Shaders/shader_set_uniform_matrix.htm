shader_set_uniform_matrix

Con esta función puede establecer el valor (o los valores) de una constante de shader a la matriz de transformación actual (tal y como se establece utilizando las funciones de matriz). Para ello debe haber obtenido previamente el "handle" de la constante utilizando la función shader_get_uniform().

NOTA: Todos los uniformes deben establecerse después de llamar a la función shader_set()y antes de llamar a la función shader_reset().

Sintaxis:

shader_set_uniform_matrix(handle);

Argumento Tipo Descripción
handle Manipulación de sombreado uniforme The handle of the shader constant to set.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

El código anterior obtendrá el manejador de la constante "u_vMatrix" de shader y luego establecerá esa constante a la matriz de transformación actual.
