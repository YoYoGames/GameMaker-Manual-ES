shader_set_uniform_f

Con esta función puede establecer el valor (o los valores) de una constante de shader. Para ello debe haber obtenido previamente el "handle" de la constante mediante la función shader_get_uniform()y tendrás que saber qué tipo de constante es para pasarle el número correcto de valores en coma flotante, es decir, si tienes una vec2 tendrás que pasar dos valores a la función.

Una excepción a la regla anterior es cuando se trata de valores de color de 32 bits. Puede introducir en shader hasta ocho valores de color de 32 bits y se convertirán automáticamente en valores vec4 de 0 - 1. Sin embargo, para ello debe utilizar el nombre de la constante shader in_Colour , para un solo color, o in_Colour0, in_Colour1, etc... hasta in_Colour7 para múltiples valores de entrada. Los siguientes ejemplos muestran cómo se vería esto en el propio shader:

    //Single colour
    attribute vec2 in_Position;
    attribute vec4 in_Colour;
    attribute vec2 in_TextureCoord;

    //Multiple colours
    attribute vec2 in_Position;
    attribute vec4 in_Colour0;
    attribute vec4 in_Colour1;
    attribute vec2 in_TextureCoord;

Ten en cuenta también que si quieres hacer normales comprimidas, etc., utilizando un formato de color de 4 bytes, entonces también tendrás que utilizar el mismo formato "in_Colour" que se muestra arriba.

NOTA: Todos los uniformes deben establecerse después de llamar a la función shader_set()y antes de llamar a la función shader_reset().

Sintaxis:

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);

Argumento Tipo Descripción
handle Manipulación de sombreado uniforme The handle of the shader constant to set.
value1 ... value4 Real The floating point value (or values) to set the shader constant to.

Devuelve:

N/A

Ejemplo:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

El código anterior obtendrá el manejador de la constante "u_vParams" de shader ( a vec3), y luego establecerá esa constante a los valores dados.
