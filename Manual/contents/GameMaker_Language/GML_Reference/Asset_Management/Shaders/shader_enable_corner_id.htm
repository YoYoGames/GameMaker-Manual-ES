shader_enable_corner_id

Con esta función se puede establecer un estado global para todos los shaders que se utilicen en el que, cuando se activa, el shader "roba" 2 bits de los valores de color de entrada. El primero es del bit inferior del valor de color rojo, y el segundo es del bit inferior del valor del color azul. Estos valores se pueden recuperar en shader para saber con qué vertex se está tratando (es decir, con qué esquina).

Sintaxis:

shader_enable_corner_id(enable);

ArgumentoTipo Descripción
enable Enable (true) o desactivar (false) esta función.

Devuelve:

Ejemplo:

shader_enable_corner_id(true);

El código anterior habilitará el uso de los bits de color para el identificador de esquina para todos los shaders.
