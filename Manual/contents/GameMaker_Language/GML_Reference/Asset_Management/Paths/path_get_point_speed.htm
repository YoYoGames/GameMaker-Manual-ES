ruta_obtener_velocidad_de_punto

Con esta función puede obtener la velocidad del punto (tal y como se define en el Editor de Trayectorias o cuando se añade dinámicamente un punto de path utilizando path_add_point()) expresado como un porcentaje. Así, si tienes un punto de path establecido en 50 en el editor de path, esta función devolvería 50 cuando se utilice.

Sintaxis:

path_get_point_speed(index, n);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to check.
n Real The point number to check.

Devuelve:

Real

Ejemplo:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

El código anterior comprueba el punto path "0" del path indexado por "pth_Patrol" para ver si la velocidad no es igual a 100. Si no lo es, entonces establece la velocidad de ese punto a 100.
