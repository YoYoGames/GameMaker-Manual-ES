path_get_kind

Las trayectorias pueden ser suaves o rectas (una path suave calcula una path curvada alrededor de los puntos definidos, mientras que una path recta sólo va de un punto a otro). Esta función función puede utilizarse para averiguar si el path dado es suave (true) o no (false).

Sintaxis:

path_get_kind(index);

ArgumentoTipo Descripción
index The index of the path to check.

Devuelve:

Ejemplo:

path_kind = path_get_kind(pth_Patrol);

Esto establecerá la variable "path_kind" a true o false dependiendo del tipo de path que sea el índice path dado.
