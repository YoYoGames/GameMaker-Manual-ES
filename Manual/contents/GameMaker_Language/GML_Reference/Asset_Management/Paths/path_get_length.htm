path_get_length

Puede utilizar esta función para obtener la longitud exacta de un path en píxeles. no se trata de una longitud aproximada de punto a punto, sino de una longitud exacta a lo largo de la forma del path, incluso cuando el path es suave con una alta precisión de curvatura.

Sintaxis:

path_get_length(index);

ArgumentoTipo Descripción
index The index of the path to measure.

Devuelve:

Ejemplo:

path_len = path_get_length(pth_AI);

Esto establecerá "path_len" a la longitud total de la path indexada en "pth_AI" en píxeles.
