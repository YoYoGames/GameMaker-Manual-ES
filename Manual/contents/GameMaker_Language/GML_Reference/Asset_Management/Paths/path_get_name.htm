path_get_name

Esta función devolverá el nombre del path al que se hace referencia como string. El nombre es el que se haya utilizado para definir el path en el editor o (si el path se ha creado a través de una función de código) devolverá un string con el formato " _newpathXX" donde "XX" es el número del path generado, comenzando en 0 e incrementando en uno cada vez que se crea un nuevo path. Tenga en cuenta que esto es sólo un string y no puede utilizarse para referenciar el path directamente - para eso necesitaría el ID del índice de la ruta. Sin embargo, puede utilizar este string para obtener el ID del índice de la ruta utilizando el string devuelto junto con con la función asset_get_index().

Sintaxis:

path_get_name(index);

ArgumentoTipo Descripción
index The index of the path to check.

Devuelve:

Ejemplo:

path_name = path_get_name(path_array[0]);

Esto establecerá "nombre_de_la_ruta" al nombre de la path indexada en el array dado en la posición 0.
