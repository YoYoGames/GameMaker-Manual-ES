path_set_precision

Con esta función puede cambiar la "suavidad" de un path. Este valor debe estar entre 1 y 8, con un valor bajo creando bordes más rectos con curvas más pronunciadas entre los puntos, mientras que un valor más alto redondeará los puntos y hará que el path sea mucho más "curvo". Tenga en cuenta que esta función no tendrá ningún efecto visible si el path no ha sido configurado como suave en el editor path o utilizando la función path_set_kind(). Esta función cambia el activo real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_set_precision(index, prec);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to change.
prec Real The precision of the path. Must be an integer between 1 and 8.

Devuelve:

N/A

Ejemplo:

path_set_precision(pth_Patrol, 2);

Esto establecerá la precisión de la ruta dada a 2.
