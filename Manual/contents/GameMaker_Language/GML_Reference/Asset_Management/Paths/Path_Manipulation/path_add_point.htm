ruta_añadir_punto

Con esta función puede añadir un punto a la path especificada y establecer su factor de velocidad. Este punto se añade al final de path, independientemente de la posición del punto, y el factor es igual al porcentaje de path_speed que la siguiente instancia realmente va cuando alcanza este punto en el path. Si desea colocar un punto de path en alguna otra posición que no sea el final, debe utilizar path_insert_point().

Si se utiliza en un path del Navegador Asset, tenga en cuenta que la función cambiará el activo actual, y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_add_point(index, x, y, speed);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to add the point to.
x Real The x coordinate of the new point.
y Real The y coordinate of the new point.
speed Real The speed factor of the point (default value is 100).

Devuelve:

N/A

Ejemplo:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

El código anterior creará un path circular alrededor de la posición x/y de la instancia que ejecuta el código.
