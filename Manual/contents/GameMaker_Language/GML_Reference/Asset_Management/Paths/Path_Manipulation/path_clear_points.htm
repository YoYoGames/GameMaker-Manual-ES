ruta_de_punto_de_claro

Con esta función puede eliminar todas las definiciones de puntos de un path, haciendo efectivamente un path"vacío". Esto no elimina el path, para eso debe utilizar path_delete()Sin embargo, debe tenerse en cuenta que esta función cambia el activo path , y por lo tanto afectará permanentemente a la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_clear_points(index);

ArgumentoTipo Descripción
index The index of the path to clear.

Devuelve:

Ejemplo:

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

El código anterior comprueba si hay algún punto en el path indexado en la variable "mypath" y si lo hay, borra el path.
