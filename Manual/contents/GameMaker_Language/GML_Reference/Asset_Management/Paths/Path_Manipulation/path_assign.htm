ruta_de_asignación

Con esta función puede copiar los datos de path de un path a otro. El path al que se está copiando se borrará primero (si tiene algún punto de path ) y se sobrescribirá completamente con el path del que se está copiando. copiado. Ningún path se borra en el proceso y el resultado son dos paths, con dos índices diferentes, pero con la misma forma y puntos. En general, se recomienda utilizar esta función en un path creado con path_add(), ya que si la usas en un path asset, afectará permanentemente al path para todas las instancias del juego desde el momento en que se usa la función hasta el final del juego.

Sintaxis:

path_assign(index, path);

ArgumentoTipo Descripción
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

Devuelve:

Ejemplo:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

El código anterior creará un nuevo path vacío indexado en la variable "mypath" y luego copiará los datos de path de uno de los tres recursos disponibles path.
