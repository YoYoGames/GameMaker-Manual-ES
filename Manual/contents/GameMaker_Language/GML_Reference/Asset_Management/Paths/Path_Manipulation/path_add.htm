ruta_add

Con esta función puede crear un path en GameMaker sin necesidad de utilizar el editor path. Esta función devolverá el índice del path que deberá ser almacenado en una variable y utilizado como referencia para ese path a partir de entonces. Tenga en cuenta que el path creado está vacío, es decir, no tiene puntos definidos, por lo que tendrá que utilizar las otras funciones disponibles para añadir puntos al path o utilizar las rejillas de MP para generar el path. Una vez que haya terminado de utilizar el path, o desee crear uno nuevo y almacenar su índice en la misma variable, deberá eliminar primero el antiguo path con path_delete para evitar fugas de memoria que pueden llegar a colapsar tu juego.

Sintaxis:

path_add();

Devuelve:

Path Asset

Ejemplo:

global.newpath = path_add();

Esto creará un nuevo path y asignará su índice a global.newpath.
