path_rotate

Puede utilizar esta función para rotar un determinado path alrededor de su centro. Recuerde que en GameMaker (a menos que esté utilizando la física) los ángulos se calculan en sentido contrario a las agujas del reloj, por lo que girar el path 90 grados giraría el path hacia la izquierda. Esta función cambia el recurso real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

NOTA: Una vez que se ha asignado un path a una instancia mediante la función path_start() se puede utilizar la variable path_orientation para cambiar también la rotación de path.

Sintaxis:

path_rotate(index, angle);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to flip.
angle Real The angle of rotation degrees.

Devuelve:

N/A

Ejemplo:

path_rotate(pth_Patrol, 90);

Esto haría girar lo dado 90 grados en sentido contrario a las agujas del reloj.
