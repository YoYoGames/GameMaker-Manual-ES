ruta_recalada

Esta función puede ser utilizada para re-escalar el path dado a lo largo de ambos (o cualquiera) el eje vertical y horizontal, básicamente moviendo cada uno de los puntos de path a una nueva posición correspondiente a esta escala alrededor del centro del path. Esta función cambia el activo actual path, y por lo tanto afectará permanentemente a la forma en que el path es utilizado por todas las instancias del juego desde el momento en que se utiliza la función hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_rescale(index, xscale, yscale);

ArgumentoTipo Descripción
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

Devuelve:

Ejemplo:

path_rescale(pth_AI, 3, 3);

De este modo, se reescalaría el path dado al triple de su tamaño original.
