ruta_espejo

Esta función toma todos los puntos de path y los refleja a lo largo del eje vertical. Esta función cambia el activo real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el duplicado asset (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_mirror(index);

ArgumentoTipo Descripción
index The index of the path to mirror.

Devuelve:

Ejemplo:

path_mirror(mypath);

Esto reflejaría el path indexado en la variable "mypath" a lo largo del eje vertical.
