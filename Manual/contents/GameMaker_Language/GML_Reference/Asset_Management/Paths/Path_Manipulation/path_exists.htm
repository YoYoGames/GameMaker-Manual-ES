ruta_existe

Esta función devuelve si existe o no un path con el índice dado. Tenga en cuenta que si comprueba la existencia de un path a través de una variable que aún no ha sido declarada, esto arrojará un error.

Sintaxis:

path_exists(index);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to check for.

Devuelve:

Boolean

Ejemplo:

if (path_exists(my_path))
{
    path_delete(my_path);
}

Este código comprueba si la variable dada almacena un índice path y si lo hace, entonces el path se borra de la memoria.
