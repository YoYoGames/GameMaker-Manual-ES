path_set_closed

Esta función puede utilizarse para flag un determinado path como abierto (falso) o cerrado (verdadero). Un path cerrado tiene su punto de inicio conectado a su punto final, formando un loop, y un path abierto tiene un inicio y un final definitivos e inconexos. Esta función cambia el activo real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego.Si esto no es lo que usted requiere, entonces usted debe utilizar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no se olvide de llamar a path_delete() en el asset cuando ya no es necesario).

Sintaxis:

path_set_closed(index, closed);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to change.
closed Booleano Whether the path is closed (true) or not (false).

Devuelve:

N/A

Ejemplo:

path_set_closed(pth_Patrol, true);

Esto hará que el path dado sea un path cerrado.
