desplazamiento_de_trayectoria

Con esta función puedes desplazar un path a lo largo del eje horizontal y vertical. Tenga en cuenta que esto no tendrá ningún efecto visible en el juego si la instancia que sigue a la path que se está desplazando la sigue relativamente (en lugar de absolutamente) ya que la posición de la ruta no se tiene en cuenta la posición de la ruta. Esta función cambia el activo real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego. Si esto no es no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el duplicado asset (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_shift(index, xshift, yshift);

ArgumentoTipo Descripción
index The index of the path to change.
xshift The number of horizontal pixels to shift the path. Negative=left, positive=right.
yshift The number of vertical pixels to shift the path. Negative=up, positive=down.

Devuelve:

Ejemplo:

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

El código anterior duplica un path asset y almacena su valor de identificación en una variable. Esta variable se utiliza entonces para desplazar la posición del path duplicado y luego la instancia inicia el path.
