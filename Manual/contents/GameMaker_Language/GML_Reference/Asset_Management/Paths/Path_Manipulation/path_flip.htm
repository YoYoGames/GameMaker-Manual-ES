camino_inclinado

Esta función toma todos los puntos de path y los voltea a lo largo del eje horizontal. Esta función cambia el activo real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función se utiliza hasta el final del juego.Si esto no es lo que usted requiere, entonces usted debe utilizar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no se olvide de llamar a path_delete() en el asset cuando ya no es necesario).

Sintaxis:

path_flip(index);

ArgumentoTipo Descripción
index The index of the path to flip.

Devuelve:

Ejemplo:

path_flip(mypath);

Esto voltearía el path indexado en la variable "mypath" a lo largo del eje horizontal.
