path_reverse

Con esta función se puede invertir el orden de numeración de cada uno de los puntos de path, de modo que, por ejemplo, si el path tiene 5 puntos, el punto 0 pasaría a ser el punto 4, el punto 1 sería el punto 3 y el punto 2 no se modificaría. La posición real de los puntos sigue siendo la misma, sólo se cambia el orden en que se procesan. Esta función cambia el activo real path , y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que la función función se utiliza hasta el final del juego. Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_reverse(index);

ArgumentoTipo Descripción
index The index of the path to change.

Devuelve:

Ejemplo:

path_reverse(pth_AI);

Esto invertiría el orden en que se procesan todos los puntos de la ruta dada.
