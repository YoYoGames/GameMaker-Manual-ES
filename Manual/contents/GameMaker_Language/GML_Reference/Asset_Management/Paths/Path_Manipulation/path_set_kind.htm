path_set_kind

Esta función permite establecer el tipo de path que se desea que sea el recurso especificado. Puede ser una línea recta path (ajustada a 0) o una línea suavizada path (ajustada a 1), en cuyo caso también hay que tener en cuenta la precisión de path (la precisión también puede establecerse mediante path_set_precision()). Esta función cambia el recurso real path, y por lo tanto afectará permanentemente la forma en que el path es utilizado por todas las instancias en el juego desde el momento en que Si esto no es lo que necesitas, entonces deberías usar una función como path_duplicate() para crear una copia del path primero, y luego llamar a esta función en el asset duplicado (no olvides llamar a path_delete() en el asset cuando ya no sea necesario).

Sintaxis:

path_set_kind(index, val);

ArgumentoTipo Descripción
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

Devuelve:

Ejemplo:

path_set_kind(pth_Patrol, true);

Esto hará que el path dado sea un path"suave".
