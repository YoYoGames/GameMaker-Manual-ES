path_append

Con este código puedes anexar un path a otro, uniéndolos efectivamente, y para que esto funcione, ambos paths deben existir previamente. En general, querrás usar esta función en paths creados usando path_add(), ya que si la usas en un path asset, afectará permanentemente a los path para las instancias en el juego desde el momento en que se usa la función hasta el final del juego.

NOTA Esto no eliminará el path que se está añadiendo al juego. Sigue ahí, sólo que ahora no tiene puntos y si ya no desea utilizarlo debe eliminarlo con la función path_delete().

Sintaxis:

path_append(index, path);

ArgumentoTipo Descripción
index The new index of the path.
path The index of the path that will be appended to 'index'.

Devuelve:

N/A

Ejemplo:

path_append(mypath, mppath);
path_delete(mppath);

El código anterior añade los datos de path indexados en la variable "mppath" a los de path indexados en la variable "mypath" y luego borra "mppath" de la memoria.
