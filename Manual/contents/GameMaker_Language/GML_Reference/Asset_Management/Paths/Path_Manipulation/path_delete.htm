ruta_suprimir

Puede utilizar este código para eliminar un path de la memoria. Si este path ha sido creado dinámicamente usando path_add()la variable que contiene el índice de path ya no será válida para acceder a path porque ya no existe, y si el path fue creado usando el Path Editor ese path ya no puede ser accedido en todo el juego porque lo estás borrando permanentemente.

Sintaxis:

path_delete(index);

ArgumentoTipo Descripción
index The index of the path to delete.

Devuelve:

Ejemplo:

var t_path = path_add();
if (mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1))
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

El código anterior creará un path y almacenará su índice en una variable local. Este path se utiliza entonces para almacenar un mp_grid_path() path generado que, si logra encontrar su camino hacia el objetivo, se asigna al path indexado en "mypath". Por último, se elimina "t_path".
