ruta_duplicada

Esta función toma un path y lo copia en un nuevo path. El nuevo path se crea en el proceso, y su índice se devuelve para ser utilizado en todas las llamadas posteriores para utilizar este nuevo path.

Sintaxis:

path_duplicate(index);

ArgumentoTipo Descripción
index The index of the existing path to duplicate.

Devuelve:

Path Asset

Ejemplo:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

El código anterior elige uno de los cuatro recursos de path y lo duplica, almacenando el índice del nuevo path en la variable "mypath".
