Caminos

GameMaker tiene una amplia colección de funciones y variables incorporadas para utilizar paths dentro de su juego. Esta sección trata de las básicas que se usan cuando se trata de paths que han sido creadas previamente en el Path Editor, pero existen más funciones que le permiten crear su propio paths usando código, y manipularlo de varias maneras.

Existen las siguientes funciones básicas que se ocupan de las rutas:

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

Para ayudar con debugging paths, tiene una función para dibujarlo en la pantalla también:

  • draw_path

Cada instancia de un object también tiene una serie de variables incorporadas diseñadas para facilitar el uso de paths, y estas variables se enumeran en la siguiente sección:

  • Variables de trayectoria

Por último, es posible manipular el contenido de un path así como crear un nuevo paths cuando el juego está en marcha. Las funciones que permiten estas cosas y más están disponibles en la siguiente sección:

  • Manipulación de la trayectoria
