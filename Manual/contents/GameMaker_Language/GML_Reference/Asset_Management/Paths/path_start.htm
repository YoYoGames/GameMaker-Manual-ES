inicio_de_la_caminata

Un path se crea a partir de una serie de puntos de definición que se enlazan entre sí y que luego se utilizan para planificar los movimientos de una instancia. Se pueden crear con código, o en el Editor Path y se asignan a una instancia para que los utilice en el juego. Esta función sirve para indicar a la instancia qué path debe seguir, a qué velocidad debe seguir el path (medida en píxeles por paso), cómo debe comportarse cuando llegue al final del path, y si debe seguir la posición absoluta o relativa del path. Esta última parte significa que, o bien comienza y sigue el path exactamente como usted lo diseñó y colocó en el Path Editor (absoluto), o bien comienza y sigue el path desde la posición en la que se creó la instancia (relativo).

Como puede ver en la imagen anterior, las dos instancias están siguiendo el mismo path iniciado con path_start(), pero una de las instancias está siguiendo el path absoluto mientras que la otra está siguiendo el path relativo. Hay que tener en cuenta que cuando relative está configurado como falso (absoluto) la instancia "saltará" al inicio de path sin importar en qué parte del room se encuentre.

Las constantes utilizadas para definir el comportamiento de la instancia cuando llega al final de la path se dan en esta página.

Sintaxis:

path_start(path, speed, endaction, absolute);

Argumento Tipo Descripción
path Activo de la ruta The path index to start.
speed Real The speed of which to follow the path in pixels per step, negative meaning going backwards.
endaction Constante de acción de fin de trayecto What to do when the end of the path is reached (see the constants listed above).
absolute Booleano Whether the calling instance should follow the absolute path as it is in the editor (true) or a relative path to its current position (false).

Devuelve:

N/A

Ejemplo:

path_start(path, 4, path_action_reverse, 0);

Esto hará que la instancia que llama siga el path indexado en la variable "path" a una velocidad de 4 píxeles por paso. Seguirá el path en relación con su propia posición, y cuando llegue al final invertirá la dirección y volverá por donde ha venido.
