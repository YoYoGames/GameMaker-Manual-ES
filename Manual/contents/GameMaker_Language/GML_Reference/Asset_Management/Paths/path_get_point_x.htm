ruta_obtener_punto_x

Esta función devolverá la posición x (en coordenadas room ) del punto que introduzca para el path que indexe. Si el punto está fuera del rango de path (es decir, un path tiene 8 puntos y usted pide la posición x del punto 10) entonces se devolverá un valor de 0.

Sintaxis:

path_get_point_x(index, n);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to check.
n Real The point number to check.

Devuelve:

Real

Ejemplo:

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

El código anterior comprobará si una instancia está al final de un path. Si lo está, elegirá un punto al azar en el path y moverá la instancia a ese punto.
