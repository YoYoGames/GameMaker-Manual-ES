path_get_closed

Esta función puede utilizarse para devolver si el path está marcado como cerrado (true) o abierto (false), es decir, si el path loops o si tiene un principio y un final definitivos.

Sintaxis:

path_get_closed(index);

Argumento Tipo Descripción
index Activo de la ruta The index of the path to check.

Devuelve:

Boolean

Ejemplo:

state = path_get_closed(pth_Patrol);

Esto pondrá "state" en true o false dependiendo del estado cerrado de path indexado en "pth_Patrol".
