camino_final

Al llamar a esta función, se terminará el path actual que la instancia está siguiendo, tal y como se estableció cuando se llamó a la función path_start() fue llamada..

Sintaxis:

path_end();

Devuelve:

Ejemplo:

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

El código anterior terminará el path actual si la instancia detecta cualquier colisión con cualquier instancia del objeto dado.
