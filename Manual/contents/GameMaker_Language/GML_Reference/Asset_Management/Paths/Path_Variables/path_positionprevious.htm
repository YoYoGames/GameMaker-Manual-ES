posición_de_camino_anterior

Esta variable puede utilizarse para obtener o establecer la posición de una instancia a lo largo de su actual path en el paso anterior, y es un valor normalizado entre 0 y 1, es decir: 0 es la posición inicial de la path y 1 sería la posición final. Es similar a los parámetros xprevious y yprevious en su funcionamiento, sólo que es específica para paths. Puede ser útil para cosas como detener temporalmente un seguidor de path si hay algo en el camino (ver el código de ejemplo más abajo).

Sintaxis:

path_positionprevious;

Devuelve:

Ejemplo:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(xx, yy, 16, obj_Player, false,true))
{
    path_position = path_positionprevious;
}

El código anterior comprueba un área delante de la instancia para ver si hay una colisión con el object "obj_Player" y, si la hay, devuelve la instancia a la posición anterior que ocupaba en el path actual, asignada por la función path_start().
