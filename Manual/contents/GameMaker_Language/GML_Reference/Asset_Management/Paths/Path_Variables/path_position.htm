posición_de_camino

Esta función puede utilizarse para obtener o establecer la posición de una instancia a lo largo de un path. El valor está normalizado de 0 a 1, por lo que si se establece, por ejemplo, en 0,5, la instancia se moverá exactamente a la mitad de la path.

Sintaxis:

path_position;

Devuelve:

Ejemplo:

path_position = random(1);

El código anterior establece la instancia en una posición aleatoria en cualquier lugar de la actual path.
