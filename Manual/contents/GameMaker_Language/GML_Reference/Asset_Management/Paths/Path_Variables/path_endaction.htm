ruta_de_actuación

Esta variable se puede utilizar para obtener o cambiar la reacción de una instancia cuando llega al final de la actual path. Normalmente, esto se establece cuando se inicia el path usando path_start() pero es posible que desees cambiar este comportamiento dependiendo de cualquier número de eventos en tu juego. Los valores disponibles se expresan mediante las siguientes constantes:

Constante de acción de fin de trayecto
Constante Descripción
path_action_stop Detener el camino
path_action_restart Continúa desde la posición inicial, saltando al inicio si el path no está cerrado
path_action_continue Inicie el path de nuevo desde la posición actual
path_action_reverse Invertir la velocidad de la path (hacer funcionar la path en reversa)

Sintaxis:

path_endaction;

Devuelve:

Path End Action Constant

Ejemplo:

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

El código anterior comprobará la acción final de path y si está establecida como parada, entonces la acción final se cambiará a inversa.
