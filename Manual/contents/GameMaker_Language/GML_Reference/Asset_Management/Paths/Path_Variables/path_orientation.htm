orientación_de_la_vía

Esta variable contiene la orientación actual de la path que ha sido asignada a la instancia cuando se llamó a la función path_start(). Cuando se crea un path, su orientación es la predeterminada 0 grados, pero usted puede establecer este valor a cualquier cosa que desee utilizando esto. Recuerde que en GameMaker (a menos que esté utilizando la física) los ángulos se calculan en sentido contrario a las agujas del reloj, por lo que establecer la orientación de path a 90° rotaría el path hacia la izquierda.

Sintaxis:

path_orientation;

Devuelve:

Ejemplo:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

El código anterior duplica un path asset aleatorio, ya hecho, en la variable "mypath". Este nuevo path se inicia y gira 90°.
