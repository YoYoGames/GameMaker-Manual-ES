velocidad_de_camino

Puede utilizar esta función para obtener o establecer la velocidad de un path después de que se haya iniciado utilizando la función path_start(). Puede utilizar valores negativos para indicar que la instancia debe seguir el path en sentido inverso.

Sintaxis:

path_speed;

Devuelve:

Ejemplo:

path_speed = -1 + random(2);

El código anterior hará que la instancia recorra la path a una velocidad aleatoria entre -1 y 1.
