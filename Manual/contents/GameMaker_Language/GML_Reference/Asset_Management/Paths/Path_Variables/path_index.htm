path_index

Todos los recursos en GameMaker tienen un número de identificación único, y el path_index es una variable de sólo lectura que mantiene el valor del índice para un determinado path asset que ha sido asignado a una instancia utilizando la función path_start(). Si la instancia no tiene asignado ningún path, la variable se pondrá a -1.

Sintaxis:

path_index

Devuelve:

Path Asset

Ejemplo:

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

El código anterior comprueba si se ha asignado un path a la instancia, y si no es así, inicia un nuevo path (asignándolo al mismo tiempo a path_index ).
