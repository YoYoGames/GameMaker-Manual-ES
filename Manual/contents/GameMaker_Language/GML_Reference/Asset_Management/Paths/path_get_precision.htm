ruta_obtener_precisión

Esta función devuelve con qué precisión se ha "suavizado" el path dado, y será un valor entero de 1 a 8. Aunque se puede obtener (y establecer) este valor para una línea recta path no tendrá ninguna influencia sobre cómo una instancia utiliza el path ya que sólo es relevante cuando el tipo de path se establece como "suave".

Sintaxis:

path_get_precision(index);

ArgumentoTipo Descripción
index The index of the path to check.

Devuelve:

Ejemplo:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

El código anterior comprueba el path indexado en "pth_Patrol" para ver qué tipo es y si es suave, entonces comprueba la precisión del suavizado. Si es inferior a 8, se establece un valor de 8.
