script_execute

Esta función se diseñó originalmente para llamar a los activos de script de forma dinámica, pero con la introducción de las funciones definidas por el usuario ahora puede utilizarse de dos maneras:

  • La primera forma es soportar el legado scripts, donde cada script era una sola función y el título del script era también el nombre de la función. Usado de esta manera, funcionará como antes y ejecutará un script asset creado por el usuario y le pasará los argumentos dados. Tenga en cuenta que en realidad no está llamando al script en este caso sino, a la función del script del mismo nombre dentro de script.
  • La segunda forma es llamar a una función desde un script asset que contenga varias funciones. En este caso, deberá proporcionar el nombre de la función y los argumentos que requiere en lugar del nombre de script asset. Tenga en cuenta que esto sólo funcionará para las funciones de script y no para los métodos, ya que sólo las funciones de script tienen un ID de script asignado.

Aunque esta función está diseñada principalmente para el soporte de legado, no está obsoleta, ya que todavía tiene su lugar en situaciones de red y de llamadas a procedimientos remotos. También hay que tener en cuenta que esta función es lenta, y usarla mucho puede afectar negativamente al rendimiento.

NOTA Esta función no se puede utilizar con las funciones incorporadas.

Sintaxis:

script_execute(scr, arg0, arg1, arg2..., etc...);

Argumento Tipo Descripción
scr Activo de script o Función de script The function/script that you want to call
arg0, arg1, arg2..., etc... Variable The different arguments that you want to pass through to the function/script

Devuelve:

Variable (Will depend on the return value from the script/function being called)

Ejemplo:

script_execute(choose(move_up, move_down, move_left, move_right), irandom(5));

El código de ejemplo anterior utilizará script_execute para elegir una de las cuatro funciones definidas por el usuario (en este caso relacionadas con el movimiento) y le pasará un número entero aleatorio como primer argumento.
