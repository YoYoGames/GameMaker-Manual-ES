script_get_name

Esta función devolverá el nombre en forma de cadena del script especificado. Este nombre es el que se ha especificado para el script en el Navegador Asset de la ventana principal de GameMaker. Para más información sobre scripts, vea el Editor Script .

Sintaxis:

script_get_name(scr);

Argumento Tipo Descripción
scr Activo del guión The index of the script that you want to get the name of.

Devuelve:

String

Ejemplo:

scr_name = script_get_name(Help_File);

El código de ejemplo anterior almacenará el nombre del índice script indicado en la variable "scr_name".
