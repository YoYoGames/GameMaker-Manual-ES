script_execute_ext

Esta función funciona de forma similar a la función script_execute(), sólo que puede suministrar un array que contenga los argumentos necesarios para la función/script que se llama. También puede suministrar dos argumentos opcionales a la función para especificar un desplazamiento en la matriz para obtener los argumentos, así como el número de argumentos a utilizar de la matriz (esto debe ser un máximo de array_length - offset).

Sintaxis:

script_execute_ext(scr, array_args, [offset], [num_args]);

Argumento Tipo Descripción
scr Activo de script o función de script The function/script that you want to call
array_args Matriz The array containing the arguments for the function/script
[offset] Real OPTIONAL El desplazamiento en la matriz de argumentos
[num_args] Real OPTIONAL El número de argumentos a utilizar (a partir del desplazamiento)

Devuelve:

Variable (Will depend on the return value from the function/script being called)

Ejemplo:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

El código de ejemplo anterior utilizará script_execute_ext para llamar a la función dada, suministrando una matriz de cuatro argumentos, pero utilizando sólo uno de ellos al azar.

NOTA Esta función no se puede utilizar con las funciones incorporadas.
