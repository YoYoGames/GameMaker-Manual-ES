script_existe

Esta función devolverá true o false dependiendo de si existe la función script o script con el índice dado. Tenga en cuenta que no se trata de un string, sino del nombre asset que contiene el índice único para cada script (tal y como aparecería en el IDE) o la función script nombrada, tal y como se define en el script asset (tenga en cuenta que esto no funcionará para las variables de método). Para más información sobre scripts, véase El editor de Script .

Sintaxis:

script_exists(scr);

Argumento Tipo Descripción
scr Activo del guión The script index that you want to check.

Devuelve:

Boolean

Ejemplo:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

El ejemplo anterior añade dos funciones script y un valor en un array, luego procede a obtener un número aleatorio y lo utiliza para elegir una función script a ejecutar, a menos que se elija el -1 en cuyo caso no pasará nada.
