object_get_visible

Esta función le dirá si el object que está comprobando ha sido marcado como "visible" (ejecuta su evento de dibujo) o no (no ejecuta su evento de dibujo). Tenga en cuenta que esta no es una función de instancia. Por lo tanto, puede tener un object visible y una instancia invisible del mismo object y viceversa. Puedes establecer la visibilidad de una instancia individual usando la variable visible variable de instancia.

Sintaxis:

object_get_visible(obj);

ArgumentoTipo Descripción
obj The index of the object to check.

Devuelve:

Ejemplo:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

El código anterior comprobará la instancia que se está ejecutando para ver si es visible o no, así como el índice object de la instancia para ver si está marcado como visible o no. Si la instancia no es visible pero el índice object está marcado como activado, establecerá "visible" como verdadero para esa instancia.
