object_get_sprite

Esta función le dirá si el object que está comprobando tiene un sprite o no, y si lo tiene entonces devolverá el índice de ese sprite, o -1 si no lo tiene. Tenga en cuenta que ésta no es una función de instancia. Puede tener un object sin sprite mientras que una instancia de ese mismo object puede tener uno y viceversa, o incluso pueden tener diferentes sprites. Puedes establecer una instancia individual sprite usando la variable sprite_index variable de instancia.

Sintaxis:

object_get_sprite(obj);

ArgumentoTipo Descripción
obj The index of the object to check

Devuelve:

Sprite Asset

Ejemplo:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

El ejemplo anterior comprobará el sprite_index de la instancia con el sprite del object_index de la instancia. Si no son iguales, entonces asignará a la instancia el mismo sprite que el del índice object.
