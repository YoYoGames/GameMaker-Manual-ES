object_get_solid

Esta función le dirá si el object que está comprobando ha sido marcado como "sólido" o no. Un object sólido genera un evento de colisión especial cuando se utiliza el sistema de colisión tradicional (es decir, el mundo de la física está desactivado). Tenga en cuenta que esta no es una función de instancia. Por lo tanto, usted puede tener un sólido object y una instancia normal del mismo object y viceversa. Puede establecer una instancia individual sólida flag usando la variable solid variable de instancia.

Sintaxis:

object_get_solid(obj);

ArgumentoTipo Descripción
obj The index of the object to check.

Devuelve:

Ejemplo:

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

El código anterior comprobará la instancia que se está ejecutando para ver si es sólida o no, así como el índice object de la instancia para ver si está marcada como sólida o no. Si la instancia no es sólida pero el índice object está marcado como sólido, establecerá "sólido" como verdadero para esa instancia.
