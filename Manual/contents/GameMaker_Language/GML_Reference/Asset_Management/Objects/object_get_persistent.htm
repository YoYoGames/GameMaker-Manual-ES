object_get_persistent

Esta función le dirá si el object que está comprobando ha sido marcado como "persistente" o no. Un object persistente es uno que hará que cualquier instancia del mismo sea llevada de room a room a menos que sea explícitamente destruida. Tenga en cuenta que esta no es una función de instancia. Por lo tanto, puedes tener un object persistente y una instancia no persistente del mismo object y viceversa. Puede establecer una instancia individual persistente flag utilizando la variable persistent variable de instancia.

Sintaxis:

object_get_persistent(obj);

ArgumentoTipo Descripción
obj The index of the object to check.

Devuelve:

Ejemplo:

if (!persistent) && (object_get_persistent(object_index))
{
    persistent = true;
}

El código anterior comprobará la instancia que se está ejecutando para ver si es persistente o no, así como el índice object de la instancia para ver si está marcado como persistente o no. Si la instancia no es persistente pero el índice object está marcado como persistente, establecerá "persistente" como verdadero para esa instancia.
