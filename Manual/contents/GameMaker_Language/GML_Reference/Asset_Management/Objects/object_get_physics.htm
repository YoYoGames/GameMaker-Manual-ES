object_get_physics

Esta función le dirá si el object que está comprobando ha sido marcado como "physics enabled" - en cuyo caso devolverá true, - o no - en cuyo caso devolverá false.

Sintaxis:

object_get_physics(obj);

ArgumentoTipo Descripción
obj The index of the object to check.

Devuelve:

Ejemplo:

if object_get_physics(object_index)
{
    phy_active = true;
}

El código anterior comprobará la instancia que lo ejecuta para ver si el object desde el que se crea está habilitado para la física, y si lo está, activa la simulación de la física para la instancia.
