Objetos

En GameMaker tienes objects y tienes instancias. Objects son esencialmente la plantilla base para una instancia, y como tal nunca están presentes en un room directamente... sólo las instancias del object se colocan en el room. Esto significa que si quieres cambiar algo para todas las instancias que vas a crear, entonces puedes hacerlo cambiando el objeto.

¡IMPORTANTE! El cambio de cualquier cosa en un object no cambiará las instancias actualmente presentes en el room, sólo las que se creen después del cambio.

También puedes acceder a la información de un object base y utilizarla para dictar comportamientos o cambios en el juego. Por ejemplo, puedes hacer una comprobación de un ID de object padre, y en el siguiente código si se encuentra una instancia con ese padre puedes entonces comprobar su object_index para decidir qué debe hacerse.

GameMaker te da una serie de funciones que te permiten obtener varios detalles sobre un object. Ten en cuenta que un object no es una instancia (una entidad dentro del juego), es puramente un recurso del que se crean todas las instancias. Esto significa que que los valores de retorno de estas funciones pueden ser diferentes a los valores reales mostrados por las instancias en el room, ya que una vez creada una instancia puede ser modificada mediante código y acciones de GML Visual.

  • object_index
  • object_exists
  • object_get_name
  • object_get_mask
  • object_get_parent
  • object_get_persistent
  • object_get_solid
  • object_get_sprite
  • object_get_visible
  • object_get_physics
  • object_is_ancestor

También hay una serie de funciones que permiten establecer las propiedades de un object. Hay que tener en cuenta que las instancias de este object que ya existen en el room pueden no verse afectadas por estas funciones, pero todas las nuevas instancias de este objeto creadas en room, por lo que se recomienda no cambiar nunca las propiedades de un objects cuando haya instancias de ese object en el actual room.

  • object_set_mask
  • object_set_persistent
  • object_set_solid
  • object_set_sprite
  • object_set_visible
