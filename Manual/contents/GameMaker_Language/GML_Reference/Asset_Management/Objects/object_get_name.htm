object_get_name

Esta función devolverá el nombre en forma de cadena del object especificado. Este nombre es el que se ha especificado para el object en el navegador Asset de la ventana principal de GameMaker. Tenga en cuenta que esto es sólo una cadena y no se puede utilizar para referenciar el object directamente - para eso necesitaría el índice del objeto. Sin embargo, puede utilizar este string para obtener el índice del objeto utilizando el string devuelto junto con la función asset_get_index().

Sintaxis:

object_get_name(obj);

ArgumentoTipo Descripción
obj The index of the object to check.

Devuelve:

Ejemplo:

str = object_get_name(object_index);

El código anterior obtendrá el nombre del índice object para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "str".
