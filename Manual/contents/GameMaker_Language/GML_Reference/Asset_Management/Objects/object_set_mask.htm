object_set_mask

Con esta función se puede establecer el índice_de_máscara de un object específico (para más información sobre máscaras, véase El editor de Object ). Esto significa que todas las instancias de este object que se creen después de haber cambiado la más cara se crearán con esta nueva máscara, mientras que las instancias que ya están en el room no se verán afectadas. Se puede establecer este valor a -1 para eliminar una máscara sprite y así volver a la máscara definida para el sprite del object (o ninguna máscara si no se ha elegido un sprite ).

Tenga en cuenta que ésta no es una función de instancia. Puede establecer el índice de máscara de las instancias individuales utilizando la variable mask_index y así tener diez instancias con una máscara diferente a la de object desde la que se crean, e incluso si cambia el índice de máscara de object usando esta función, todas las instancias que están actualmente en room permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con la nueva máscara.

Sintaxis:

object_set_mask(index, spr);

Argumento Tipo Descripción
index Activo de objeto The index of the object to change.
spr Activo Sprite The new sprite to use as the object's mask.

Devuelve:

N/A

Ejemplo:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

El código anterior comprueba el valor de la variable global y luego cambia el índice de la máscara de object "obj_Platform" si es igual a diez.
