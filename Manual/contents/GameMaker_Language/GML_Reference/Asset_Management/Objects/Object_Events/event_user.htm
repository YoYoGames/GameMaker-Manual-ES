usuario_del_evento

Con esta función se indica a la instancia que ejecute las acciones o el código que se ha colocado dentro de uno de los 16 eventos definidos por el usuario. Estos eventos sólo pueden ser llamados de esta manera, o usando la función event_perform función.

Sintaxis:

event_user(numb);

Argumento Tipo Descripción
numb Real The number of User Event to call, between 0 and 15.

Devuelve:

N/A

Ejemplo:

event_user(4);

Esto llamaría al Evento Definido por el Usuario 4.
