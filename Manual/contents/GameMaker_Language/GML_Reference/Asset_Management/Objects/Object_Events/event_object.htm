event_object

Esta variable de sólo lectura devuelve object índice de la instancia que está ejecutando el evento que se está comprobando.

Sintaxis:

event_object

Devuelve:

Object Asset

Ejemplo:

global.obj = event_object;

El código anterior almacena el índice object de la instancia que realiza el evento en una variable global.
