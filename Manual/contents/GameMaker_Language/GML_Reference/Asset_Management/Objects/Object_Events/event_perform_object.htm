evento_realizar_objeto

Esta función funciona igual que event_perform() excepto que esta vez puedes especificar eventos de otro object. Hay muchas opciones aquí que permiten una simulación completa de todos los eventos posibles, pero ten en cuenta que esto literalmente sólo realiza todo el código del evento y el juego no modificará nada para que se dispare manualmente, por ejemplo si eliges realizar un evento de pulsación de teclado, el evento se disparará pero no se reconocerá que se ha pulsado la tecla correspondiente. O si realizas un evento de alarma, el contador de alarmas no se pondrá a -1 sino que continuará la cuenta atrás. Puede encontrar una lista completa de las constantes disponibles que requiere esta función en la página de la función event_perform().

NOTA Las acciones del evento llamado con esta función se aplican a la instancia actual, y no a las instancias del object dado.

Sintaxis:

event_perform_object(obj, type, numb);

Argumento Tipo Descripción
obj Activo de objeto The object that should have its event triggered.
type Tipo de evento Constante The type of event to perform.
numb Númeroreal o de evento Constante The specific event, if one is necessary (otherwise, just use 0).

Devuelve:

N/A

Ejemplo:

event_perform_object(obj_Player, ev_keypress, ord("W"));

Esto realizaría el evento asociado a Keyboard Check Pressed > W key de la object "obj_Player" en la instancia actual.
