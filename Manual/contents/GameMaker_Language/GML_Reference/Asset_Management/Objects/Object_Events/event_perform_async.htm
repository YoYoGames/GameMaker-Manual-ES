event_perform_async

Esta función se utiliza para realizar cualquiera de los Eventos Asíncronos proporcionados en GameMaker. Usted suministra la constante del evento Async (mostrada en la tabla de abajo) y un mapa DS que estará disponible en el evento Async llamado en la variable async_load.

Constante de tipo de evento asíncrono
Constante Descripción
ev_async_web_image_load Evento de imagen cargada
ev_async_web Evento HTTP
ev_async_dialog Evento de diálogo
ev_async_web_iap Evento de compra en la aplicación
ev_async_web_cloud Evento en la nube
ev_async_web_networking Evento para la creación de redes
ev_async_web_steam Evento de vapor
ev_async_social Evento social
ev_async_push_notification Evento de Notificación Push
ev_async_save_load Guardar/cargar evento
ev_async_audio_recording Evento de grabación de audio
ev_async_audio_playback Evento de reproducción de audio
ev_async_system_event Evento del sistema

Los eventos no asíncronos pueden ser llamados usando event_perform().

Tenga en cuenta que el mapa DS especificado en el segundo argumento no tiene que ser destruido manualmente, ya que será destruido automáticamente por la función después de que el evento Async se haya realizado.

Sintaxis:

event_perform_async(type, ds_map);

Argumento Tipo Descripción
type Constante de tipo de evento asíncrono The type of event to perform (see the table above).
ds_map ID del mapa DS The DS map to use as async_load en el evento llamado.

Devuelve:

N/A

Ejemplo:

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

El código anterior crea un mapa DS y lo rellena con entradas personalizadas que se leerán en el evento Async. Luego realiza el evento Async Social con el mapa recién creado pasado como async_load para el evento llamado.
