número_de_evento

Esta variable de sólo lectura devuelve el número del evento que se está llamando en ese momento, donde el número se refiere en realidad al "subevento" del evento, es decir, para el evento del paso el número del evento podría ser cualquiera de las constantes ev_step_normal, ev_step_begin, o ev_step_end. Para una lista completa de las constantes que están disponibles para los sub-acontecimientos específicos ver event_perform()y si necesita conocer el evento principal en sí, debe consultar el archivo event_type.

Sintaxis:

event_number;

Devuelve:

Event Number Constant

Ejemplo:

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

El código anterior podría ser llamado desde un script y utilizado para mostrar mensajes de depuración informando de qué evento se está activando en ese momento.
