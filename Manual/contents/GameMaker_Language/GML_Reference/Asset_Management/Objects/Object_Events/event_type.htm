event_type

Esta variable de sólo lectura devuelve el tipo de evento que se está ejecutando actualmente, que puede ser una de las siguientes constantes:

  • ev_crear
  • ev_destroy
  • ev_paso
  • ev_alarma
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ev_collision
  • ev_otro
  • ev_gestura
  • ev_draw


Para obtener una lista completa de las constantes relacionadas con los diferentes eventos, consulte event_perform()y si necesita el número de evento (es decir, el subevento), debe consultar el archivo event_number.

Sintaxis:

event_type

Devuelve:

Event Type Constant

Ejemplo:

show_debug_message("Current Event = " + string(event_type));

El código anterior mostrará un mensaje de depuración con el tipo de evento que se está realizando.
