object_set_solid

Con esta función puede flag un object específico como sólido o no (para más información sobre el "sólido" flag vea la sección sobre el Editor de Objetos). Esto significa que todas las instancias de este object que se creen después de haber cambiado el sólido se crearán con este nuevo estado, mientras que las instancias que ya están en el room pueden no verse afectadas.

Tenga en cuenta que ésta no es una función de instancia. Puede establecer el sólido flag de instancias individuales usando la variable solid y así tener diez instancias todas marcadas como sólidas aunque el object en cuestión tenga el flag establecido como falso en el IDE, e incluso si cambia el sólido flag del object a verdadero usando esta función, todas las instancias que actualmente en el room permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con el sólido marcado como verdadero.

Sintaxis:

object_set_solid( index, solid );

Argumento Tipo Descripción
index Activo de objeto The index of the object to change.
solid Booleano The new solidity of the object (true=sólido, false=no sólido).

Devuelve:

N/A

Ejemplo:

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

El código anterior comprueba el sólido flag del object "obj_Block" y si es falso entonces se cambia para ser marcado como verdadero.
