object_set_sprite

Con esta función se puede establecer el índice sprite de un object específico. Esto significa que todas las instancias de este object que se creen después de haber cambiado el sprite_index se crearán con este nuevo sprite_index, mientras que las instancias que ya están en el room pueden no verse afectadas.

Tenga en cuenta que ésta no es una función de instancia. Puede establecer el índice sprite de las instancias individuales utilizando la variable sprite_index y así tener diez instancias todas con un sprite diferente al object del que se crean, e incluso si cambia el índice sprite del object usando esta función, todas las instancias que están actualmente en el room permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con el nuevo sprite.

Sintaxis:

object_set_sprite( index, spr );

Argumento Tipo Descripción
index Activo de objeto The index of the object to change.
spr Activo Sprite The sprite to assign to the object.

Devuelve:

N/A

Ejemplo:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

El código anterior establecerá el índice object "obj_Player" sprite a diferentes valores dependiendo de la room en la que se encuentre la instancia que ejecuta el código. A continuación, crea una instancia de "obj_Player".
