objeto_existe

Esta función devuelve si el valor especificado es un object asset (true) o no (false). Tenga en cuenta que esto comprueba si un object está presente en el navegador Asset, y no realmente en el juego room. Para ello debe utilizar la función instance_exists().

Sintaxis:

object_exists(obj);

ArgumentoTipo Descripción
obj The index of the object to check.

Devuelve:

Ejemplo:

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

El código anterior lee los datos de un buffer que fue recibido a través de la red. Como contiene un ID de object (utilizado para generar instancias), primero necesitamos asegurarnos de que el valor no es inválido y que el object existe realmente. Para ello llamamos a object_exists() en una condición, y si es cierto, creamos una instancia de ese object en la posición recibida.
