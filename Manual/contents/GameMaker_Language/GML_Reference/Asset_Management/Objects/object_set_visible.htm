object_set_visible

Con esta función puede flag un object específico como visible o no (para más información sobre el "visible" flag vea la sección sobre el Editor de Objetos). Esto significa que todas las instancias de este object que se creen después de que se haya cambiado lo de visible se crearán con este nuevo estado, mientras que las instancias que ya están en el room pueden no verse afectadas.

Tenga en cuenta que ésta no es una función de instancia. Puede establecer el visible flag de instancias individuales utilizando la variable visible y así tener diez instancias marcadas como visibles aunque el object en cuestión tenga el flag establecido como falso en el IDE, e incluso si cambia el flag visible del object a verdadero usando esta función, todas las instancias que actualmente están en el room permanecerán como estaban, y sólo las instancias creadas después de llamar a la función comenzarán con el visible marcado como verdadero.

Sintaxis:

object_set_visible( index, vis );

Argumento Tipo Descripción
index Activo de objeto The index of the object to change.
vis Booleano The new visibility of the object (true=visible, false=invisible).

Devuelve:

N/A

Ejemplo:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

El código anterior comprueba el flag visible del object "obj_Block" y si es falso entonces se cambia para ser marcado como verdadero.
