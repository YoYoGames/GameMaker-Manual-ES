object_get_parent

Esta función obtendrá el índice object de cualquier padre que haya sido asignado al object especificado, o bien devolverá -100 para mostrar que el object no tiene ningún padre asignado, o -1 si el object que se está comprobando no existe. Para más información sobre los padres, consulte la sección sobre el Editor de Objetos.

Sintaxis:

object_get_parent(obj);

ArgumentoTipo Descripción
obj The index of the object to check

Devuelve:

Object Asset

Ejemplo:

par = object_get_parent(object_index);

El código anterior obtendrá el padre del índice object para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "par".
