object_get_mask

Esta función le dirá si el object que está comprobando tiene un índice de máscara o no, y si lo tiene entonces devolverá el índice de esa máscara (que es un sprite asset), o -1 si no lo tiene. Tenga en cuenta que ésta no es una función de instancia. Puede tener un object sin máscara mientras que una instancia de ese mismo object puede tener una y viceversa, o incluso pueden tener diferentes máscaras. Puede establecer el índice de máscara de una instancia individual utilizando la variable mask_index variable de instancia.

Sintaxis:

object_get_mask(obj);

ArgumentoTipo Descripción
obj The index of the object to check

Devuelve:

Sprite Asset

Ejemplo:

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

El ejemplo anterior comprobará el índice de máscara de la instancia con la máscara del índice_objeto de la instancia. Si no son iguales, entonces asignará a la instancia la misma máscara que tiene el índice object.
