object_index

Esta variable de sólo lectura devuelve el índice del object del que se ha creado la instancia. Esto no es lo mismo que el nombre de object, que es un string y puede ser encontrado usando object_get_name()ya que esta función devuelve el número de índice, que es un valor único que GameMaker asigna a cada object en el momento de su creación.

Sintaxis:

object_index

Devuelve:

Object Asset

Ejemplo:

obj_name = object_get_name(object_index);

El código anterior utilizará el object_index para encontrar el nombre del object del que se ha creado la instancia actual.
