timeline_add

Con esta función puedes añadir una nueva (vacía) timeline asset en tu juego. la función devuelve el índice de esta nueva línea de tiempo que debe ser almacenada en una variable para su uso en todas las llamadas de función posteriores que involucren esta nueva línea de tiempo. También debe asegurarse de utilizar la función timeline_delete() siempre que no desees utilizar más la línea de tiempo (como al final de room) para evitar posibles fugas de memoria que podrían ralentizar o bloquear tu juego. Para añadir momentos a un timeline creado de esta manera, vea la función timeline_moment_add_script().

Sintaxis:

timeline_add();

Devuelve:

Timeline Asset

Ejemplo:

global.tl = timeline_add();

El código anterior crea una nueva línea de tiempo y almacena su índice en la variable "global.tl".
