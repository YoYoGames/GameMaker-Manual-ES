borrar la línea de tiempo

Con esta función puede borrar una línea de tiempo específica de "momentos", eliminando todos los códigos y acciones para esa línea de tiempo y dejándola vacía.

Sintaxis:

timeline_clear(ind);

Argumento Tipo Descripción
ind Activo de la línea de tiempo The index of the timeline to clear.

Devuelve:

N/A

Ejemplo:

if timeline_position > 200
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

El código anterior borrará la línea de tiempo especificada e indexada por la variable "global.tl" de todos los momentos cuando se haya pasado un momento específico.
