existe la línea de tiempo

Con esta función puede comprobar y ver si existe una línea de tiempo (devuelve true) o no (devuelve false). Esto es particularmente útil cuando se crea Timelines dinámicamente utilizando la función timeline_add() pero hay que tener en cuenta que si se busca una variable no inicializada (que de otro modo se asignaría al índice de una línea de tiempo) se producirá un error.

Sintaxis:

timeline_exists(ind);

ArgumentoTipo Descripción
ind The index of the time line to check for.

Devuelve:

Ejemplo:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

El código anterior comprueba si existe una línea de tiempo y está indexada en la variable global "tl" y, si es así, borra esa línea de tiempo.
