timeline_delete

Con esta función puede eliminar cualquier línea de tiempo de su juego. Si esta línea de tiempo fue creada en el Navegador de GameMaker Asset, tenga en cuenta que se elimina completamente del juego y tratar de llamar a la línea de tiempo indexada de nuevo después de usar esta función causará un error. Si la línea de tiempo fue creada dinámicamente usando la función timeline_add() entonces esta función debe ser utilizada en el momento en que la línea de tiempo ya no es necesaria para evitar cualquier fuga de memoria que podría ralentizar (y eventualmente bloquear) su juego.

Sintaxis:

timeline_delete(ind);

ArgumentoTipo Descripción
ind The index of the time line to delete.

Devuelve:

Ejemplo:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

El código anterior comprueba si existe una línea de tiempo y está indexada en la variable global "tl" y, si es así, borra esa línea de tiempo.
