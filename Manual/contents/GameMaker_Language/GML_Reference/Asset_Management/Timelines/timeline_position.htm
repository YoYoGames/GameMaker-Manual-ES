timeline_position

Esta variable contiene la posición actual (momento) en que se encuentra una línea de tiempo. Puedes cambiar este valor para saltar partes de la línea de tiempo, o para repetir partes o para empezar la línea de tiempo de nuevo desde el principio.

Sintaxis:

timeline_position

Devuelve:

(single precision floating point value)

Ejemplo:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

El código anterior comprobará si la instancia está ejecutando una línea de tiempo, y si no lo está entonces restablece la línea de tiempo asignada para que comience en el primer momento y luego la inicia.
