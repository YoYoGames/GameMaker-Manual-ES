cronograma_de_ejecución

Esta variable mantiene el estado actual de la línea de tiempo asignada y devolverá true si está en marcha y false si no lo está. También puede establecer esta variable como true o false para iniciar y detener la línea de tiempo en cualquier momento. debe tenerse en cuenta que una línea de tiempo detenida no se reinicia, por lo que iniciarla de nuevo en un momento posterior la iniciará desde el momento exacto en que se detuvo.

Sintaxis:

timeline_running;

Devuelve:

Ejemplo:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

El código anterior comprobará si la instancia está ejecutando una línea de tiempo, y si no lo está entonces restablece la línea de tiempo asignada para que comience en el primer momento y luego la inicia.
