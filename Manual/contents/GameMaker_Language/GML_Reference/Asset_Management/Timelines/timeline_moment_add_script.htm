timeline_moment_add_script

Con esta función puedes añadir dinámicamente una función de script a Timelines en cualquier "momento" dentro de esa línea de tiempo, donde un "momento" es el equivalente a un tick (o paso) del juego. De esta manera puedes crear una nueva línea de tiempo usando la función timeline_add() y añadir diferentes comportamientos en cualquier punto, o simplemente modificar un recurso de línea de tiempo previamente creado con nuevos comportamientos. Ten en cuenta que la función no puede requerir ningún argumento adicional cuando se utiliza esta función, y si la utilizas para modificar un Timeline asset presente en el Navegador Asset, entonces todas las instancias que utilicen este timeline se verán afectadas, y el cambio durará hasta el final del juego (llamar a game_restart() tampoco restablecerá el cambio).

Sintaxis:

timeline_moment_add_script(ind, step, script);

Argumento Tipo Descripción
ind Activo de la línea de tiempo The index of the timeline to add a moment to.
step Real The moment (step) to add to.
script Función del guión The index of the script function to add into the moment.

Devuelve:

N/A

Ejemplo:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

El código anterior creará una nueva línea de tiempo y almacenará su índice en la variable "global.tl". A continuación, añadirá tres scripts a la línea de tiempo, elegidos al azar, a intervalos de un minuto.
