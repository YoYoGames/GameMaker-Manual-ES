Líneas de tiempo

Las líneas de tiempo son un poderoso mecanismo para controlar los eventos en tu juego, y se basan en "momentos", donde un "momento" es simplemente un solo paso del juego. Estos timelines se crean generalmente desde el Navegador de Activos utilizando el Editor de Líneas de Tiempo de forma similar a un object, es decir: creas un timeline, luego en cada momento que requieras que realice una acción añades algún código o un icono de GML Visual para que lo realice. Una vez creado, puedes detener, iniciar y manipular timelines mediante código. Incluso puedes modificar momentos individuales definiendo una función de script y añadiéndola en el timeline dinámicamente desde un controlador object (por ejemplo), lo que hace que sea una herramienta muy potente y versátil a la hora de crear tus juegos.

Las siguientes funciones sirven para añadir y manipular Timelines desde el juego:

  • timeline_exists
  • timeline_get_name
  • timeline_add
  • timeline_delete
  • timeline_moment_add_script
  • timeline_moment_clear
  • timeline_clear
  • timeline_size
  • timeline_max_moment

Las siguientes variables están "integradas" en todas las instancias para facilitar el trabajo con Timelines y darle control sobre ellas:

  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop

Tenga en cuenta que puede encontrar más información sobre las variables de instancia aquí: Variables de instancia.
