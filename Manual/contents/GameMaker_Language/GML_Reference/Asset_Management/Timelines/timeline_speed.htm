timeline_speed

Normalmente, en cada paso la posición en la línea de tiempo se incrementa en 1, sin embargo puede cambiar esta cantidad estableciendo esta variable a un valor diferente. Puede utilizar números reales (como 0,5, o 2,4 por ejemplo) y si el valor es mayor que uno, pueden ocurrir varios momentos dentro del mismo paso de tiempo (todos se realizarán en el mismo orden definido para la línea de tiempo, por lo que no se saltará ninguna acción).

Sintaxis:

timeline_speed

Devuelve:

(single precision floating point value)

Ejemplo:

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

El código anterior comprobará si la instancia está ejecutando su línea de tiempo asociada y, si no lo está, iniciará la línea de tiempo que se ejecuta a media velocidad.
