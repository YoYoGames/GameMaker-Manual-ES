bucle_de_tiempo

Esta variable devolverá si la línea de tiempo está en bucle (true) o no (false). Puedes cambiar esta variable para que switch active o desactive el bucle y funciona con una velocidad de línea de tiempo negativa (si la posición de la línea de tiempo baja de 0, comenzará de nuevo en el último momento definido).

Sintaxis:

timeline_loop;

Devuelve:

Ejemplo:

if !timeline_loop
{
    timeline_loop = true;
}

El código anterior comprueba si la línea de tiempo asignada actualmente a una instancia se loop o no y si no lo hace lo establece como verdadero.
