tamaño de la línea de tiempo

Con esta función puede obtener el número total de momentos activos para un timeline (un momento "activo" es aquel que tiene código o GML Visual añadido). Esto puede ser útil cuando se crea un timelines dinámico utilizando las funciones timeline_moment_add_script() y timeline_moment_clear() ya que puede comprobar si un timeline dado tiene el número correcto de momentos activos o ninguno.

Sintaxis:

timeline_size(ind);

ArgumentoTipo Descripción
ind The index of the timeline get the information from.

Devuelve:

Ejemplo:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

El código anterior comprueba el tamaño dado de timeline, y si devuelve 0 (es decir, el timeline no tiene momentos activos) añadirá una función script que se utilizará en un momento aleatorio dentro del timeline indexado en la variable global "tl".
