timeline_get_name

Esta función puede utilizarse para obtener el nombre de una línea de tiempo como string. si la línea de tiempo ha sido creada dinámicamente utilizando la función timeline_add() el nombre devuelto tendrá el formato formato "__newtimelineN" donde "N" es el número de la línea de tiempo (empezando por 0). Tenga en cuenta que esto es sólo un string y no se puede utilizar para hacer referencia a la timeline directamente - para para ello necesitaría el índice de la línea de tiempo. Sin embargo, puede utilizar este string para obtener el índice de la línea de tiempo utilizando el string devuelto junto con la función asset_get_index().

Sintaxis:

timeline_get_name(ind);

ArgumentoTipo Descripción
ind The index of the time line to check the name of.

Devuelve:

Ejemplo:

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

El código anterior obtiene el nombre del timeline actualmente asignado y lo muestra en la consola.
