índice_de_línea_de_tiempo

Esta variable contiene el índice de la timeline actualmente asociada a la instancia. Puede establecerla en una línea de tiempo concreta para utilizarla, o establecerla en -1 para dejar de utilizar una línea de tiempo para la instancia (si no se define ninguna línea de tiempo para la instancia, también se devuelve -1). Tenga en cuenta que esto no inicia la línea de tiempo - para ello utilice la variable timeline_running.

Sintaxis:

timeline_index;

Devuelve:

Timeline Asset

Ejemplo:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

El código anterior asignará a la instancia que ejecuta el código una línea de tiempo indexada en la variable "global.tl" si esa línea de tiempo existe.
