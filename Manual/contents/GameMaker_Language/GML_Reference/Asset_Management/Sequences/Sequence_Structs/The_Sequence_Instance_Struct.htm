La estructura de la instancia Sequence

Un sequence, cuando se coloca en un room, se coloca como un elemento en la capa, y este elemento contiene una instancia de secuencia. Esta instancia de sequence tendrá las siguientes propiedades en su estructura:

Estructura de la instancia de la secuencia
Variable Tipo Descripción
sequence Secuencia Object Struct Este es el ID del objeto de secuencia struct del que se ha creado la instancia sequence (ver aquí paramás detalles). Puedes obtener o establecer este struct, y así cambiar la base sequence a la que la instancia hace referencia.
headPosition Real Es la posición actual del cabezal de reproducción (en fotogramas) para la instancia sequence. Puede obtener o establecer este valor, pero tenga en cuenta que establecerlo a un valor mayor o menor que la región de reproducción definida actualmente tendrá diferentes efectos dependiendo del tipo de reproducción establecido para el sequence. Consulte layer_sequence_headpos() para más información.
headDirection Dirección de la secuencia Constante Esta es la dirección de reproducción actual para la instancia sequence. Puedes obtener o establecer este valor usando las constantes dadas en la siguiente tabla (Ten en cuenta que puedes usar las funciones de capa layer_sequence_get_headdir() y layer_sequence_headdir() para obtener o establecer este valor sin tener que acceder directamente a struct ).
speedScale Real Esta propiedad se puede utilizar para obtener o establecer la escala de velocidad de reproducción. La escala de velocidad es un multiplicador, donde 1 es la velocidad de reproducción por defecto y valores menores que 1 ralentizarán la reproducción y valores mayores que 1 la acelerarán, por ejemplo: un valor de 0.5 sería la mitad de la velocidad de reproducción, mientras que un valor de 2 sería el doble de la velocidad de reproducción. Tenga en cuenta que puede utilizar las funciones de capa layer_sequence_get_speedscale() y layer_sequence_speedscale() para obtener o establecer este valor sin tener que acceder directamente a struct.
paused Booleano Puede comprobar esta propiedad para ver si un sequence ha sido pausado o no, y será true si lo ha sido, o false en caso contrario. Esta es una propiedad de sólo lectura, pero puedes utilizar la función de capa layer_sequence_pause() para pausar la reproducción, y layer_sequence_play() reanudarla de nuevo si es necesario. También puede comprobar esta propiedad utilizando la función de capa layer_sequence_is_paused() en lugar de comprobar la propiedad en el struct directamente.
finished Booleano Puedes comprobar esta propiedad para ver si un sequence ha terminado de reproducirse o no, devolviendo true si ha terminado de reproducirse, y false en caso contrario. Esta es una propiedad de sólo lectura, y sólo devolverá true para las pistas que no estén configuradas como loop o ping-pong. También puedes comprobar esta propiedad utilizando la función de capa layer_sequence_is_finished() en lugar de comprobar la propiedad en el struct directamente.
elementID ID del elemento de secuencia Esta propiedad contiene el ID del elemento sequence. Este ID es un simple valor de identificación que se asocia a una capa en el editor room, y puede utilizarse con las funciones de capa apropiadas para encontrar la capa a la que se ha asignado el sequence o para cambiar ciertas propiedades de sequence sin tener que tratar con el struct.
activeTracks Matriz de Secuencia Pista Activa Estruct.s Esta propiedad contendrá una matriz de "evaluación" structs que contiene información sobre el estado actual de cada pista de activos en el sequence (gráficos, sequence, audio, etc...). Estas son las pistas de nivel superior, y el array devuelto es de sólo lectura. asset Para conocer el contenido de la pista struct que se devuelve en cada entrada del array, consulte la sección siguiente.

Dirección de la secuencia Constante
Constante Descripción Valor
seqdir_right El sequence reproducirá los fotogramas en un orden incremental de izquierda a derecha 1
seqdir_left El sequence reproducirá los fotogramas en orden decreciente de derecha a izquierda -1

Estructura activeTracks

Aunque esta propiedad ya existe en la instancia sequence struct en el evento Sequence Create, sólo se llena de pistas después del primer evento Sequence Begin Step. Así que la primera vez que encontrará valores en este struct es durante la primera iteración del evento Sequence Step.

La siguiente lista contiene las propiedades que pueden estar disponibles cuando se accede a una pista activa struct (tal y como se incluye en la matriz activeTracks de una Instancia Sequence struct , explicada anteriormente). Cada pista asset struct puede tener cualquiera de las siguientes propiedades, dependiendo del tipo de asset que utilice la pista (tenga en cuenta que puede modificar todas estas propiedades, excepto cuando se especifique como de sólo lectura), y todos los valores devueltos son para la posición actual del cabezal de reproducción:

Secuencia de la pista activa Struct
Variable Tipo Descripción
matrix Matriz Es la matriz de transformación utilizada para dibujar la pista (sprite, instancia y sequence pistas).
posx Real La posición del asset en el sequence a lo largo del eje X para la pista en la posición actual del cabezal de reproducción (todos assets).
posy Real La posición del asset en el sequence a lo largo del eje Y para la pista en la posición actual del cabezal de reproducción (todos assets).
scalex Real La escala del asset en el sequence a lo largo del eje X para la pista en la posición actual del cabezal de reproducción (sprite, instancia y sequence pistas).
scaley Real La escala del asset en el sequence a lo largo del eje Y para la pista en la posición actual del cabezal de reproducción (sprite, instancia y sequence pistas).
xorigin Real El origen X de la asset para la pista (todo assets).
yorigin Real El origen Y de la asset para la pista (todo assets).
width Real La anchura (en píxeles) del asset para la pista (sprite, instancia y sequence pistas).
height Real La altura (en píxeles) de la asset para la pista (sprite, instancia y sequence pistas).
imageindex Real El índice de imagen para el asset en la pista en el sequence en la posición actual del cabezal de reproducción (sprite y pistas de instancia).
imagespeed Real La velocidad de la imagen para el asset en la pista en el sequence en la posición actual del cabezal de reproducción (sprite y pistas de instancia).
colourMultiply Color El valor de multiplicación del color para el asset en la pista en el sequence en la posición actual del cabezal de reproducción (sprite, instancia y sequence pistas). Este valor será una matriz de cuatro valores ARGB con el formato [A, R, G, B]. Tenga en cuenta que los valores de cada componente se expresan entre 0 y 1, donde 0 corresponde al valor HEX #00 y 1 corresponde al valor HEX #FF (0 - 255 como se muestra en el selector de color para las pistas de multiples colores en el Editor Sequences ).
emitterIndex Identificación del emisor de audio El índice del emisor de audio perteneciente a esta pista (pistas de audio).
gain Real La ganancia (volumen) de la pista en la posición actual del cabezal de reproducción (pistas de audio).
pitch Real El tono de la pista en la posición actual del cabezal de reproducción (pistas de audio).
frameSizeX Real El tamaño horizontal del marco de texto (pistas de texto).
frameSizeY Real El tamaño vertical del marco de texto (pistas de texto).
characterSpacing Real El valor del espaciado entre caracteres (pistas de texto).
lineSpacing Real El valor del interlineado (pistas de texto).
paragraphSpacing Real El valor del espaciado entre párrafos (pistas de texto).
track Estructura de la pista de secuencia El Track Struct en el que se basa esta pista (todo assets, un valor de sólo lectura ).
parent Estructura de la instancia de la secuencia El ID de instancia de sequence para la pista (todos los assets, un valor de sólo lectura ).
activeTracks Matriz de Estructura de la pista de la secuencias Se trata de una matriz de evaluación structs para cada pista de parámetros que contiene la pista asset (todas assets, un valor de sólo lectura ). El contenido de cada struct en el array se encuentra en la sección Track Struct de parámetros.
