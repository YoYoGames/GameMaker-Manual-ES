La estructura de los fotogramas clave

Cada pista (como se define en la página que cubre los structs de pista) tendrá uno o más structs keyframe asignados (que puedes obtener de la propiedad keyframes de la pista struct), y cada uno tendrá las siguientes propiedades:

Secuencia Keyframe Struct
Variable Tipo Descripción
frame Real La posición (en fotogramas) a lo largo de timeline para el keyframe. El valor por defecto es 0.
length Real La longitud de keyframe. El valor por defecto es 1, y cuando se establece a valores mayores entonces la propiedad de la pista a la que se refiere keyframe se mantendrá en el valor inicial durante la duración dada. Tenga en cuenta que la propiedad stretch anulará esto si se establece en true.
stretch Booleano Si esta propiedad se establece en true entonces el keyframe se extiende hasta el siguiente keyframe para la pista o hasta el final de la pista si es el último keyframe. Puede obtener o establecer este valor, y el valor por defecto es false.
channels Matriz de Secuencia Keyframe Estructura de datoss Esta propiedad permite acceder a la lista de structs de datos de fotogramas clave para los canales de la pista. Cuando se obtiene esta propiedad se devuelve un array de datos keyframe structs , y cuando se establece esta propiedad se debe especificar un array de datos keyframe structs .
