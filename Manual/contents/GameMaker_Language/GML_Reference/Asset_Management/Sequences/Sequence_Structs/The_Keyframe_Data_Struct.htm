La estructura de datos de los fotogramas clave

Cuando se define una estructura de fotogramas clave para una pista, es necesario definir también los datos que se asociarán a ella. Esto se compone de diferentes canales, donde cada canal es simplemente un Keyframe Datos Struct. A un canal se le puede dar un tipo específico de datos, dependiendo del tipo de pista para la que esté configurando los datos keyframe.

Un Keyframe Datos Struct contiene las siguientes variables:

Secuencia Keyframe Data Struct
Variable Tipo Descripción
channel Real Este es el canal al que deben aplicarse los datos de keyframe. Es un valor entero positivo que comienza en 0, y vale la pena señalar que cuando se crean pistas de parámetros para la "posición" o la "escala" keyframes, es necesario utilizar valores de canal muy específicos. Estos son: el canal 0 es la posición X o la escala X, el canal 1 es la posición Y o la escala Y.
spriteIndex Activo Sprite El índice de la sprite asset que se utilizará para la pista. Esta propiedad sólo está disponible para las pistas del tipo seqtracktype_graphic y se puede obtener o establecer.
soundIndex Activo sonoro El índice del audio asset a utilizar para la pista. Esta propiedad sólo está disponible para las pistas del tipo seqtracktype_audio y se puede obtener o establecer.
playbackMode Secuencia de audio Clave constante El modo de reproducción del sonido. Esta propiedad sólo está disponible para las pistas del tipo seqtracktype_audio y se puede obtener o establecer. La propiedad debe ser una de las constantes dadas en la tabla debajo de esta.
curve Estructura de la curva de animación Esta propiedad requiere un animation curve struct (ver aquí para más información) y sólo está disponible para pistas del tipo seqtracktype_real. Si no se utiliza ninguna curva struct, el valor de esta propiedad será -1.
value Real Esta propiedad es simplemente un valor que se asocia con el canal de datos keyframe, y sólo está disponible para las pistas del tipo seqtracktype_real cuando no se suministra curve struct. Puede ser, por ejemplo, la posición X o Y de la pista si se coloca dentro de una pista de parámetro "posición ".
colour Matriz de Reals Esta propiedad devuelve (o requiere, si se establece) una matriz para el valor del color de la keyframe con el formato [A, R, G, B]. Sólo está disponible para las pistas del tipo seqtracktype_colour. Tenga en cuenta que los valores de cada componente deben expresarse entre 0 y 1, donde 0 corresponde al valor HEX #00 y 1 corresponde al valor HEX #FF (0 - 255 como se muestra en el selector de color para las pistas de color en el Editor Sequences ).
sequence Secuencia Object Struct Esta propiedad devolverá (o requerirá, si se establece) una secuencia object struct y sólo está disponible para las pistas del tipo seqtracktype_sequence.
objectIndex Activo de objeto Esta propiedad devolverá (o requerirá, si se establece) un índice object y sólo está disponible para las pistas del tipo seqtracktype_instance.
events Array de Cadenas Esta propiedad permite acceder a los eventos y mensajes de difusión asociados a los datos de keyframe struct . Se puede obtener o establecer esta propiedad, y al obtenerla se devuelve un array de strings, y al establecerla se debe especificar un array de strings. Para más información sobre los eventos, consulte la sección Eventos y momentos de la secuencia. Esta propiedad sólo está disponible para las pistas del tipo seqtracktype_message.
event Método Esta propiedad devolverá (o puede establecerse) el method asociado a los datos de keyframe struct . Si no se ha especificado ningún método o se desea eliminar el método, la propiedad debe ser -1. Esta propiedad sólo está disponible para las pistas del tipo seqtracktype_moment.
Cualquier variable adicional que figure en el cuadro o cuadros siguientes

La variable playbackMode puede ser una de las siguientes constantes:

Secuencia de audio Clave constante
Constante Descripción Valor
seqaudiokey_loop El sonido será loop cuando se reproduzca. 0
seqaudiokey_oneshot El sonido sólo se reproducirá una vez y luego se detendrá. 1

Datos de la pista de texto

El struct contendrá las siguientes variables adicionales si se asigna a una pista de texto (seqtracktype_text):

Secuencia Keyframe Data Struct
Variable Tipo Descripción
text Cadena Este es el texto string que se dibuja en la pista.
wrap Booleano Es un booleano que indica si el texto debe ser envuelto (true) o no (false).
alignmentH Constante de alineación del texto de la secuencia Esta es la alineación horizontal del texto, y será una de las Constante de alineación del texto de la secuenciaque se indican a continuación.
alignmentV Constante de alineación del texto de la secuencia Esta es la alineación vertical del texto, y será una de las Constante de alineación del texto de la secuenciaque se indican a continuación.
fontIndex Activo de fuentes Esta es la Activo de fuentes utilizado por la pista de texto.

alignmentH puede ser cualquiera de las siguientes constantes:

Constante de alineación del texto de la secuencia (alignmentH)
Constante Descripción
seqtextkey_left El texto se alineará a la izquierda del marco
seqtextkey_right El texto se alineará a la derecha del marco
seqtextkey_center El texto estará centrado horizontalmente dentro del marco
seqtextkey_justify El texto se justificará dentro del marco

alignmentV puede ser cualquiera de las siguientes constantes:

Constante de alineación del texto de la secuencia (alignmentV)
Constante Descripción
seqtextkey_top El texto se alineará en la parte superior del marco
seqtextkey_bottom El texto se alineará en la parte inferior del marco
seqtextkey_middle El texto se centrará verticalmente en el marco
