Secuencia Object Struct

A Secuencia Object Struct es el nombre que se le da al struct recuperado de un Sequence asset. Todos los datos de un Sequence se almacenan en este struct en runtime, y son referenciados por cualquier elemento del Sequence que se cree en un room. Esto significa que si cualquier variable en esta struct (o dentro de cualquiera de sus structs anidadas) se modifica, el Sequence original cambiará (hasta que el juego se cierre y se vuelva a abrir) y cualquiera de sus elementos Sequence existentes se verá afectado.

Esta estructura tendrá las siguientes propiedades que se pueden cambiar:

Secuencia Object Struct
Variable Tipo Descripción
name Cadena Este es el nombre del sequence como string y puede obtener o establecer este valor según sea necesario. Tenga en cuenta que los sequences creados con la función sequence_create() no tendrá un nombre y éste será simplemente un string vacío "".
loopmode Modo de reproducción de secuencias Constante Este es el modo de reproducción de la sequence object y puede ser obtenido o establecido. Puede ser cualquiera de las constantes que se muestran en la tabla inferior a esta.
playbackSpeed Real Especifica la velocidad de reproducción de sequence, que se interpreta como fotogramas por segundo o fotogramas por partida, dependiendo de playbackSpeedType (ver más abajo). Puedes obtener o establecer este valor.
playbackSpeedType Constante de velocidad del sprite Especifica cómo debe interpretarse el playbackSpeed y puede obtener o establecer este valor.
length Real La longitud de sequence en fotogramas. Puedes obtener o establecer este valor, pero ten en cuenta que hacer un sequence más corto puede causar problemas si una instancia de sequence que hace referencia a este sequence tiene su cabezal de reproducción establecido más allá de la nueva longitud.
volume Real Este es un valor escalar de 0 a 1 que se utiliza para escalar el volumen de todas las pistas de audio en el sequence. Puede obtener o establecer este valor y modificará la salida de audio global para todas las pistas - por ejemplo, si tiene una pista de audio con un volumen de 0,8 y luego establece la propiedad de volumen sequence a 0,5, la pista de audio tendrá un volumen final de 0,4.
xorigin Real Este es el origen de la sequence a lo largo del eje X.
yorigin Real Este es el origen de la sequence a lo largo del eje Y.
messageEventKeyframes Matriz de Secuencia Keyframe Structs Esto permite acceder al evento de mensaje keyframes para el sequence. Puedes obtener o establecer estos eventos de mensaje, y al obtener esta propiedad se devuelve un array de keyframe structs, y para establecer la propiedad debes suministrar un array de keyframe structs. Para más información, por favor consulta la página de Eventos de Secuencia y Momentos.
momentKeyframes Matriz de Secuencia Keyframe Structs Esto permite acceder al evento de momento keyframes para el sequence. Puedes obtener o establecer estos eventos de momento, y al obtener esta propiedad se devuelve un array de keyframe structs, y para establecer la propiedad debes suministrar un array de keyframe structs. Para más información, consulta la página de Eventos de Secuencia y Momentos.
tracks Matriz de Estructura de la pista de la secuencias Permite acceder a la lista de pistas de activos en el nivel superior de sequence. Puede obtener o establecer esta propiedad, y al obtenerla se devuelve un array de pistas structs, y para establecer la propiedad debe suministrar un array de pistas structs. Para más información, consulte la sección Estructuras de pistas.

loopMode puede ser cualquiera de las siguientes constantes:

Modo de reproducción de secuencias Constante
Constante Descripción Valor
seqplay_oneshot El sequence se reproducirá una vez y se detendrá al terminar. 0
seqplay_loop El sequence loop , con el cabezal de reproducción volviendo al principio cuando llega al final de la región de reproducción. 1
seqplay_pingpong El sequence loop , con el cabezal de reproducción invirtiendo la dirección cuando llega al final de la región de reproducción. 2


Observaciones

Tenga en cuenta que si quiere acceder a las propiedades de un sequence que ha sido creado en el navegador asset, debe llamar primero a la función sequence_get() en el índice asset para recuperar el sequence object struct . Ten en cuenta también que cualquier cambio realizado en este sequence struct significará que todas las instancias posteriores de este sequence asset también tendrán estos cambios, y se mantendrán mientras el juego esté en marcha, incluso si llamas a la función game_restart().
