crear_secuencia

Con esta función puedes crear un nuevo sequence object al que puedes añadir pistas. La función devuelve una estructura a la que puede acceder para configurar el nuevo sequence que ha creado. El contenido de este struct se detalla en esta página. El sequence object struct se puede utilizar para crear instancias del sequence en una capa room utilizando la función layer_sequence_create(). Tenga en cuenta que al crear sequence objects de esta manera debe eliminarlos de nuevo llamando a la función sequence_destroy() cuando ya no se necesiten, de lo contrario se producirá una fuga de memoria que puede ralentizar y, eventualmente, bloquear el juego.

Sintaxis:

sequence_create();

Devuelve:

Sequence Object Struct

Ejemplo:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

El código anterior crea un nuevo sequence object struct y establece su duración de reproducción y el modo loop.
