secuencia_in

Esta es una variable incorporada que forma parte de las variables de instancia creadas para cada instancia de object en su juego. Si la instancia está siendo controlada por un sequence, esta variable devolverá true, de lo contrario devolverá false. Esta es una variable de sólo lectura y no puede ser modificada.

Tenga en cuenta que esta variable se convertirá en false después de que el Sequence controlador haya terminado, incluso si su Sequence elemento todavía existe, y se convertirá en true si ese Sequence elemento se reproduce de nuevo.

Puedes usar esta variable en tu objects controlado por el jugador (o por la CPU) para asegurarte de que sólo se mueven por código cuando no están en un Sequence. Esto puede ser útil para las escenas, cuando necesitas usar las mismas instancias en un Sequence (usando la función sequence_instance_override_object() ) y necesitas asegurarte de que no pueden ser movidas por su código habitual mientras esa escena está activa.

Sintaxis:

in_sequence

Devuelve:

Ejemplo:

if (!in_sequence)
{
    x += move_x;
    y += move_y;
}

El código anterior comprueba la variable in_sequence, y si es false (lo que significa que la instancia no está siendo controlada por un sequence) entonces añade move_x y move_y a la posición de la instancia de la instancia, asegurándose de que sólo se mueve cuando no está en un Sequence.
