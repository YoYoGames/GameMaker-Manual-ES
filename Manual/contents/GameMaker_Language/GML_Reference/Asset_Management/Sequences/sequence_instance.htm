secuencia_instancia

Esta es una variable incorporada que forma parte de las variables de instancia creadas para cada instancia de object en tu juego. Si la instancia está siendo controlada por un sequence, esta variable contendrá la estructura de instancia de la secuencia para el Sequence que controla la instancia, de lo contrario será undefined. Esta es una variable de sólo lectura y no puede ser modificada.

Tenga en cuenta que esta variable se convertirá en undefined después de que el Sequence controlador haya terminado, incluso si su Sequence elemento todavía existe, y obtendrá el struct de nuevo si ese Sequence elemento se reproduce.

Sintaxis:

sequence_instance

Devuelve:

Sequence Instance Struct

Ejemplo:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

El código anterior comprueba la variable in_sequence y si es true (lo que significa que la instancia está siendo controlada por un sequence) entonces cambiará la escala de velocidad de ese Sequence a 2.
