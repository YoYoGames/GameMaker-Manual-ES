secuencia_destrucción

Con esta función puede destruir un sequence object que ha sido creado dinámicamente. Se suministra el sequence object struct (devuelto por la función sequence_create()) o el ID de sequence (devuelto por la función layer_sequence_get_sequence() o de la instancia sequence struct propiedad sequence). Esta función debe utilizarse siempre que un sequence creado dinámicamente ya no sea necesario para liberar la memoria asociada a él.

Sintaxis:

sequence_destroy(sequence_struct_or_id);

Argumento Tipo Descripción
sequence_struct_or_id Activo de la secuencia o secuencia Object Struct The sequence object struct or ID to destroy

Devuelve:

N/A

Ejemplo:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

El código anterior comprueba si el sequence object dado existe y si lo hace se destruye.
