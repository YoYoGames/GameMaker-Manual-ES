sequence_exists

Con esta función puede comprobar si un sequence object existe o no. Se suministra el sequence object struct (tal como lo devuelve la función sequence_create() o sequence_get()) o el ID de sequence (tal como lo devuelve la función layer_sequence_get_sequence() o de la instancia sequence struct propiedad sequence, o el índice del navegador asset ) y la función devolverá true si el sequence object existe o false si no existe.

Sintaxis:

sequence_exists(sequence_struct_or_id);

Argumento Tipo Descripción
sequence_struct_or_id Activo de la secuencia o secuencia Object Struct The sequence to check for, can be the asset reference itself or its Sequence object struct.

Devuelve:

Boolean

Ejemplo:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

El código anterior comprueba si el sequence object dado existe y si lo hace se destruye.
