secuencia_obtener

Con esta función se puede recuperar la Secuencia Object Struct a partir de un índice sequence asset. Se suministra el índice asset para el sequence como se define en el navegador asset, y la función devolverá el sequence object struct al que se puede acceder, o -1 si el sequence no existe o el índice asset dado no es un sequence.

Sintaxis:

sequence_get(sequence_index);

Argumento Tipo Descripción
sequence_index Activo de la secuencia The sequence asset as defined in the Asset Browser.

Devuelve:

Sequence Object Struct or -1

Ejemplo:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

El código anterior recupera el sequence object struct del sequence object "seq_Loop" y luego establece el modo loop para el sequence (todas las instancias sequence creadas posteriormente utilizarán ahora este modo loop ).
