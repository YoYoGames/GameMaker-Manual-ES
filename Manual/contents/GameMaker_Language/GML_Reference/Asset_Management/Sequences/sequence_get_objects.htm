sequence_get_objects

Con esta función se puede recuperar un array de todos los índices de object que tienen instancias creadas dentro del sequence dado. Puede suministrar el sequence object struct (tal y como lo devuelve la función sequence_create() o sequence_get()) o el ID de sequence (tal como lo devuelve la función layer_sequence_get_sequence() o de la instancia sequence struct propiedad sequence) y la función devolverá un array, donde cada elemento del array es un object_index para los diferentes objects que están siendo utilizados por el sequence para crear instancias.

Sintaxis:

sequence_get_objects(sequence_struct_or_id);

Argumento Tipo Descripción
sequence_struct_or_id Activo de la secuencia o secuencia Object Struct The sequence object struct or ID to get the objects from.

Devuelve:

Array

Ejemplo:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

El código anterior obtiene el struct para un sequence object y luego recupera el objects que utiliza y almacena la matriz en una variable.
