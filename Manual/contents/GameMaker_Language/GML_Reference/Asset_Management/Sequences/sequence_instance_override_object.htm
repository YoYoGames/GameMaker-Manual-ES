secuencia_instancia_sobrecarga_objeto

Con esta función puede anular (reemplazar) todas las instancias de un object utilizadas en un sequence con otro. Se proporciona el ID de la instancia sequence struct (tal y como se devolvió cuando se creó la instancia sequence en room o utilizando una de las funciones de la capa room - ver aquí), así como el índice object (tal y como se definió en el navegador asset ) para el object que se quiere anular. Por último, indique un índice object o un ID de instancia para utilizar como object que va a anular el sequence (si se proporciona un ID de instancia, simplemente se utilizará el object del que se creó la instancia como anulación). Tenga en cuenta que esto sólo puede hacerse en las instancias de sequence (no en sequence objects) y debe hacerse antes de que el sequence comience a reproducirse, de lo contrario no funcionará.

Sintaxis:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argumento Tipo Descripción
sequence_instance_struct Estructura de la instancia de la secuencia The sequence instance struct to modify.
object_id Activo de objeto The object index of the object within the sequence to override.
instance_or_object_id Activo del objeto o ID de la instancia The object index or instance ID to use to override the sequence objects.

Devuelve:

N/A

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

El código anterior crea una nueva instancia de sequence en la capa dada y luego la modifica para que todas las instancias de object "obj_Trees_Winter" sean reemplazadas por instancias de object "obj_Trees_Summer".
