audio_sound_get_listener_mask

Esta función devolverá los datos de la máscara de bits que definen de qué oyentes de audio debe reproducirse un sonido. Para más información, consulte la sección sobre oyentes de audio.

Sintaxis:

audio_sound_get_listener_mask(soundID);

Argumento Tipo Descripción
soundID ID de la instancia de sonido The unique ID of the sound to get the mask of

Devuelve:

Real

Ejemplo:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
    audio_sound_set_listener_mask(snd, global.PlayerMask);
}

El código anterior reproduce un sonido, luego comprueba los datos de la máscara del oyente para el sonido, y si no es el mismo que el que está almacenado en una variable global, establece el oyente(s) para que se reproduzca utilizando los datos de la máscara almacenados en la variable global.
