audio_master_gain

Con esta función puedes establecer el valor absoluto del volumen global de todos los sonidos y la música. Se basa en una escala lineal que va desde 0 (silencio) hasta cualquier valor superior a 0, aunque normalmente se consideraría que el volumen máximo es 1. Se puede utilizar cualquier valor superior a 1 pero, dependiendo del sonido utilizado y de la plataforma en la que se compile, se puede producir distorsión o recorte cuando se reproduzca el sonido. Esta función afectará al volumen relativo de todos los sonidos y la música que se reproduzcan en el juego.

Sintaxis:

audio_master_gain(gain);

Argumento Tipo Descripción
gain Real Value for the global volume (0 to 1).

Devuelve:

N/A

Ejemplo:

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if (keyboard_check(vk_down))
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

El código anterior comprueba las pulsaciones de las teclas de flecha arriba y abajo, que luego aumentan o disminuyen la variable "vol". Esta variable se utiliza entonces para establecer la ganancia global de todo el sonido y la música en el juego.
