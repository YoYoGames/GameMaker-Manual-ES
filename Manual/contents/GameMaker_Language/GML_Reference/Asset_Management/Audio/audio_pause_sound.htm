audio_pausa_sonido

Con esta función puedes pausar cualquier sonido que se esté reproduciendo en ese momento. El sonido puede ser una única instancia de un sonido (el índice de los sonidos individuales que se están reproduciendo puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un sonido asset, en cuyo caso se pondrán en pausa todas las instancias del sonido en cuestión.

Sintaxis:

audio_pause_sound(index);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to pause.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

El código anterior comprueba si se pulsa la tecla "P" del teclado y, si detecta una, establece la variable global "Pause" en true o en false y, a continuación, pausa el sonido indexado en la variable "snd_Waterfall" o reanuda el sonido desde su estado de pausa.
