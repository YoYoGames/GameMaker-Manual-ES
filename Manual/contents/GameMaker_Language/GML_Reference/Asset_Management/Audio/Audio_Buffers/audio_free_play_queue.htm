audio_free_play_queue

Esta función se utiliza para liberar la memoria asociada al audio dado queue. El índice queue es el valor devuelto cuando se creó el queue utilizando la función audio_create_play_queue()y esta función debe ser llamada cuando el queue ya no sea necesario para evitar fugas de memoria. Al liberar el queue se detendrá cualquier sonido que se esté reproduciendo, y no se puede eliminar el buffer desde el que se está transmitiendo un sonido hasta que se haya liberado el queue al que está asignado. Esta función activará un Evento Asíncrono de Reproducción de Audio, y en este evento se creará un mapa DS especial en la variable async_load con los siguientes pares clave/valor:

  • "queue_id" - el queue que ha terminado de reproducir datos
  • "buffer_id" - el buffer que ya no se reproduce desde
  • "queue_shutdown" - se pone a 0 durante la reproducción normal y a 1 cuando se recibe el evento porque se ha liberado la queue con esta función. Cuando se pone a 1, no se quiere queue más datos.

Sintaxis:

audio_free_play_queue(queueIndex);

Argumento Tipo Descripción
queueIndex Audio Queue ID The index of the queue to free.

Devuelve:

N/A

Ejemplo:

var q_id = async_load[? "queue_id"];
if (q_id == audio_queue)
{
    audio_free_play_queue(audio_queue);
    buffer_delete(audio_buffer);
}

El código anterior iría en el evento asíncrono de reproducción de audio y comprueba si el audio que ha terminado de reproducirse es de la "cola_de_audio" y si lo es libera el queue y el buffer asociado a él.
