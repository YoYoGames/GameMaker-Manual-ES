audio_get_recorder_count

Esta función devolverá el número de fuentes de grabación de audio (como micrófonos, etc...) actualmente disponibles para su juego. Así, si tiene un valor de retorno de, por ejemplo, cuatro, entonces tendrá entrada de audio en las fuentes 0, 1, 2 y 3, siendo uno de estos valores el que utilice para empezar a grabar desde la función audio_start_recording(). Este valor puede cambiar en cualquier momento a medida que se conectan/desconectan micrófonos u otras fuentes de entrada al dispositivo. Tenga en cuenta que puede utilizar la función audio_get_recorder_info() para obtener información sobre cada dispositivo conectado.

IMPORTANTE Esta función siempre devolverá 1 en el objetivo Opera GX.

Sintaxis:

audio_get_recorder_count();

Devuelve:

Real

Ejemplo:

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

El código anterior comprueba las grabadoras de audio disponibles y si hay 1 o más, comienza a grabar desde la primera indexada (fuente 0).
