audio_create_buffer_sound

Con esta función puedes crear un nuevo sonido a partir del contenido de un buffer. El buffer habrá sido creado previamente (ver las funciones del buffer para más detalles sobre cómo hacerlo), y se le habrán añadido o cargado datos. A continuación, se pasa a esta función con el formato de los datos (actualmente sólo se admite buffer_u8 o buffer_s16 ), la frecuencia de muestreo (que puede estar entre 1000hz y 48000hz), y un desplazamiento en el buffer para obtener los datos. También es necesario proporcionar el número de muestras en buffer y los canales que requiere el sonido. Estos canales están definidos por una de las siguientes constantes:

Tipo de canal de audio
Constante Descripción
audio_mono Audio mono (un solo canal).
audio_stereo Audio estéreo (doble canal).
audio_3D Audio 3D (5.1).

Tenga en cuenta que después de haber creado un sonido, debe liberar el índice de puntero asociado a él cuando ya no sea necesario utilizando la función audio_free_buffer_sound(). Si no hace esto y luego reasigna la variable o cambia rooms etc... el índice del sonido se perderá y tendrá una fuga de memoria. También tenga en cuenta que no puede borrar el buffer si se ha creado algún sonido a partir de él y el sonido no ha sido liberado primero. Por lo tanto, primero debe liberar el sonido (o los sonidos) y luego borrar el buffer. También vale la pena señalar que añadir cualquier cosa a buffer, o cambiar el tamaño de buffer, después de que se haya creado un sonido a partir de él dará resultados inesperados y no se recomienda - una vez que haya comenzado a crear sonidos a partir de cualquier buffer no debe manipularlo de ninguna otra manera después.

Sintaxis:

audio_create_buffer_sound(bufferId, bufferFormat, bufferRate, bufferOffset, bufferLength, bufferChannels);

Argumento Tipo Descripción
bufferId ID del búfer The ID of the buffer to use.
bufferFormat Tampón Data Type Constante The format of the data in the buffer (buffer_u8 o buffer_s16).
bufferRate Real The sample rate of the data in the buffer.
bufferOffset Real The offset into the buffer to read the sample data from (in bytes).
bufferLength Real The length of the buffer (the number of bytes of audio data, excluding the header).
bufferChannels Tipo de canal de audio The channels to use from one of the constants listed above.

Devuelve:

Sound Asset

Ejemplo:

var rate = 44100;
var hertz = irandom_range(220, 880);
var samples = 44100;

var bufferId = buffer_create(rate, buffer_fast, 1);
buffer_seek(bufferId, buffer_seek_start, 0);

var num_to_write = rate / hertz;
var length = buffer_get_size(bufferId);
var val_to_write = 1;

for (var i = 0; i < (samples / num_to_write) + 1; i++)
{
    for (var j = 0; j < num_to_write; j++)
    {
        buffer_write(bufferId, buffer_u8, val_to_write * 255);
    }
    val_to_write = (1 - val_to_write);
}

soundId = audio_create_buffer_sound(bufferId, buffer_u8, rate, 0, length, audio_stereo);

Lo anterior crea un buffer y luego lo llena procedimentalmente con datos. Estos datos se utilizan para crear un nuevo sonido, que se almacena en la variable "soundId".
