audio_queue_sound

Esta función agregará los datos de un buffer en el audio queue que usted creó previamente usando la función audio_create_play_queue(). Se especifica el índice de queue al que se va a añadir, y el ID de buffer del que se van a obtener los datos, así como la posición (offset) dentro de buffer y el número de bytes que se van a añadir. Una vez que haya añadido audio de un buffer a un queue, el buffer no podrá ser eliminado hasta que haya liberado primero el queue utilizando la función audio_free_play_queue()y las propiedades de buffer deben coincidir con las del queue al que está añadiendo el sonido.

Sintaxis:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Argumento Tipo Descripción
queueIndex Audio Queue ID The index of the queue to add to.
bufferId ID del búfer The buffer to add to the queue.
bufferOffset Real The offset within the source buffer to start from.
bufferLength Real The length of the buffer (the number of the bytes in the buffer).

Devuelve:

N/A

Ejemplo:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

El código anterior sería llamado en el evento asíncrono de Grabación de Audio y asigna algunos datos grabados a un buffer, que luego se agrega a un audio queue. Este es entonces reproducido.
