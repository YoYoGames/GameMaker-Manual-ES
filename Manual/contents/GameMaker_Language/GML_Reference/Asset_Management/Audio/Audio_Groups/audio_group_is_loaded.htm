audio_group_is_loaded

Esta función comprobará un grupo de audio específico para ver si se ha cargado en la memoria, listo para su uso.

Sintaxis:

audio_group_is_loaded(groupID);

Argumento Tipo Descripción
groupID ID del grupo de audio The index of the audio group to check (as defined in the Audio Groups Window)

Devuelve:

Boolean

Ejemplo:

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

El código anterior comprueba si se ha cargado un grupo de audio y si lo ha hecho lo descarga.
