audio_group_set_gain

Con esta función se puede hacer un fundido de entrada o salida de un grupo de sonidos durante un tiempo determinado, o se puede utilizar para ajustar la ganancia del grupo instantáneamente.

El tiempo se mide en milisegundos, y la función requiere que se introduzca un nivel final de ganancia para que el grupo haya alcanzado al final de ese tiempo. Esta ganancia puede estar entre 0 (silencio) y 1 (volumen total) y la escala es lineal, de forma que un valor de 0,5 sería la mitad del volumen. Para cambiar instantáneamente la ganancia, basta con poner el argumento del tiempo a 0. Ten en cuenta que en algunas plataformas puedes tener una ganancia mayor que 1, aunque un valor de 1 se considera "volumen completo" y cualquier cosa mayor puede introducir recortes de audio.

Sintaxis:

audio_group_set_gain(groupID, volume, time);

Argumento Tipo Descripción
groupID ID del grupo de audio The index of the audio group to stop (as defined in the Audio Groups Window)
volume Real The final value for the group volume.
time Real The length of the change in gain in milliseconds.

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

El código anterior comprueba la tecla "espacio" y luego desvanece todo el audio para "audiogroup1" a 0 durante 5 segundos.
