audio_group_load

Esta función cargará en memoria todos los sonidos marcados como pertenecientes al Grupo de Audio dado. La función devolverá true si se inicia la carga y false si el ID del grupo no es válido, o ya ha sido marcado para la carga. La función es asíncrona, por lo que tu juego continuará ejecutándose mientras se carga el audio en segundo plano. Esto significa que también lanzará un Evento de Carga/Guardado Asíncrono para informarle que todo el grupo ha sido cargado en la memoria y que los sonidos ya pueden ser utilizados.

Sintaxis:

audio_group_load(groupID);

Argumento Tipo Descripción
groupID ID del grupo de audio The index of the audio group to load (as defined in the Audio Groups Window)

Devuelve:

Boolean

Ejemplo:

if !audio_group_is_loaded(audiogroup_level1)
{
    audio_group_load(audiogroup_level1);
}

El código anterior comprueba si se ha cargado un grupo de audio y, si no, lo carga.
