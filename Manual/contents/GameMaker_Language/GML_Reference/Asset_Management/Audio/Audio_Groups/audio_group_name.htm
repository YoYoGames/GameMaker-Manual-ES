nombre_del_grupo_de_audio

Esta función devolverá un string que contiene el nombre del grupo de audio dado para su visualización o comprobación. Cuando se define un grupo de audio en las Opciones de Juego, se le da un "nombre" único que es realmente una constante para usar como valorde ID para el grupo. Todo lo que hace esta función es tomar el ID y devolver un string del nombre del ID que le diste.

Sintaxis:

audio_group_name(groupID);

Argumento Tipo Descripción
groupID ID del grupo de audio The index value constant of the audio group to check (as defined in the Audio Groups Window)

Devuelve:

String

Ejemplo:

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

El código anterior recupera el nombre de la constante del grupo de audio dado y lo muestra en la pantalla.
