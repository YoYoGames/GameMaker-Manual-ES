audio_sound_is_playable

Esta función puede utilizarse para comprobar si el índice de sonido dado puede reproducirse actualmente. Esto es necesario debido a las diferentes formas en que se maneja la reproducción de sonido en flujo y sin flujo en la plataforma de destino HTML5, y devolverá true si el sonido puede ser reproducido y false si no puede. Tenga en cuenta que en todas las demás plataformas que no sean HTML5, la función siempre devolverá true.

Sintaxis:

audio_sound_is_playable(index);

Argumento Tipo Descripción
index Activo sonoro The index of the sound to check.

Devuelve:

Boolean

Ejemplo:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

El código anterior reproducirá el sonido "snd_Shot" si es reproducible.
