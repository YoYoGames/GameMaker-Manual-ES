audio_is_paused

Esta función comprobará el sonido dado para ver si está actualmente en pausa. El sonido puede ser una sola instancia de un sonido (el índice de los sonidos individuales que se reproducen puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un sonido asset, en cuyo caso se comprobarán todas las instancias del sonido dado y si alguna de ellas está en pausa la función devolverá true, en caso contrario devolverá false.

Sintaxis:

audio_is_paused(index);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to check.

Devuelve:

Boolean

Ejemplo:

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

El código anterior comprueba si el sonido indexado en la variable "snd_Waterfall" está actualmente en pausa y si devuelve true se reanudará la reproducción del sonido.
