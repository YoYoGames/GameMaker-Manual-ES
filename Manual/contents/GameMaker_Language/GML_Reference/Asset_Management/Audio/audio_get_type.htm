audio_get_type

Cuando se añade audio a GameMaker, éste puede ser transmitido o en memoria. Si usted necesita saber si un índice de sonido dado es para el audio transmitido o no puede utilizar esta función que devolverá 1 para el streaming, 0 para el sonido en la memoria, y -1 si hay algún error o el índice no apunta a un recurso de sonido válido.

Sintaxis:

audio_get_type(index);

ArgumentoTipo Descripción
index The index of the sound to check.

Devuelve:

Ejemplo:

type = audio_get_type(snd_Music_1);

El código anterior comprueba el tipo de audio indexado en la variable "snd_Music_1" y almacena el valor devuelto en la variable "type".
