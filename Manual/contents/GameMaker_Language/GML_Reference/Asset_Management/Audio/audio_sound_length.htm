longitud_de_sonido_de_audio

Esta función devolverá la duración del sonido dado en segundos. El sonido puede ser una referencia del índice de un sonido individual que se está reproduciendo y que se ha almacenado en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() o un sonido real asset desde el navegador Asset.

Sintaxis:

audio_sound_length(index);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to check.

Devuelve:

Real

Ejemplo:

var len;
len = audio_sound_length(snd_Beam);
audio_play_sound(snd_Beam, 1, false);
alarm[0] = room_speed * len;

El código anterior obtiene la duración (en segundos) del sonido indexado en la variable "snd_Beam", luego reproduce el sonido y establece una alarma para que se active cuando el sonido haya terminado de reproducirse utilizando la duración del sonido para calcular el tiempo necesario para la alarma.
