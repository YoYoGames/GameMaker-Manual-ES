audio_get_listener_mask

Esta función devolverá los datos de la máscara de bits que define la máscara actual por defecto (global) para los oyentes de audio.

Sintaxis:

audio_get_listener_mask();

Devuelve:

Ejemplo:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

El código anterior obtiene los datos de la máscara de escucha actual y los compara con los datos almacenados en una variable global. Si no son iguales, el código comprueba el número de oyentes disponibles y luego loops a través de ellos y obtiene sus bits de máscara, que luego se combinan para crear una sola máscara que se aplica al sistema de audio para definir los oyentes globales.
