audio_get_listener_count

Algunas plataformas de destino permiten más de un oyente, por lo que es importante saber cuántos tiene el destino antes de cambiar o utilizar diferentes oyentes. Esta función devolverá el número de escuchas disponibles.

Sintaxis:

audio_get_listener_count();

Devuelve:

Ejemplo:

global.listener_num = audio_get_listener_count();

El código anterior obtiene el número de oyentes disponibles y almacena el valor de retorno en una variable global.
