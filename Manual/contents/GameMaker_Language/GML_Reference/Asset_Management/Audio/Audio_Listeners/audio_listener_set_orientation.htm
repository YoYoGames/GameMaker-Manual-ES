audio_listener_set_orientation

Con esta función puede cambiar la orientación del oyente dado dentro del espacio de audio 3D. El índice del oyente por defecto es 0, pero puede utilizar la función audio_get_listener_info() para obtener los diferentes índices disponibles para la plataforma de destino.

El vector de observación y el vector ascendente se basan en los valores que se resuelven a partir de las posiciones relativas x, y y z dadas, y por defecto son (0, 0, 1) para el vector de observación y (0, 1, 0) para el vector ascendente, como se muestra en la siguiente ilustración:

Cambiar la orientación del oyente con esta función cambiará la forma en que el sonido creado por los emisores de audio alrededor del juego room es percibido por el jugador de su juego. En el ejemplo siguiente, los sonidos creados por el emisor cuando el oyente está en la posición predeterminada parecerían venir de abajo y a la derecha del oyente, pero con la nueva posición y orientación del oyente ahora se percibirán como procedentes de arriba y a la derecha.

Sintaxis:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Argumento Tipo Descripción
index Real o ID de oyente de audio The listener to set the orientation of.
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Devuelve:

N/A

Ejemplo:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

El código anterior recupera la matriz de vista para la vista de cámara [0] y luego la utiliza para establecer la posición y la orientación del oyente de audio con el ID almacenado en la variable global "Player_Listener".
