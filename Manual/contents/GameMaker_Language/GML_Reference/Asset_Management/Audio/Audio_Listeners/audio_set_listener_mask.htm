audio_set_listener_mask

Cuando se utilizan varios oyentes en un sistema, se puede establecer la máscara de bits para un sonido y hacer que se escuche sólo desde el oyente marcado. Sin embargo, también puede establecer la máscara global utilizando esta función y todos los sonidos reproducidos normalmente se escucharán desde los oyentes marcados por esta máscara, sin necesidad de establecer la máscara para cada sonido individualmente.

Sintaxis:

audio_set_listener_mask(mask);

Argumento Tipo Descripción
mask Real The bit-mask data to set for the listeners

Devuelve:

N/A

Ejemplo:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

El código anterior comprueba el número de oyentes disponibles y luego loops a través de ellos obtiene sus bits de máscara, que luego se combinan para crear una sola máscara de bits que se aplica al oyente global.
