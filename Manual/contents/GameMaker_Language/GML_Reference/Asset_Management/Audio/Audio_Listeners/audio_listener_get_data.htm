audio_listener_get_data

Esta función creará un mapa DS y lo rellenará con los valores de posición, velocidad y orientación para el oyente dado. El índice del oyente por defecto es 0, pero se puede utilizar la función audio_get_listener_info() para obtener los diferentes índices disponibles para la plataforma de destino. Si proporcionas un índice de oyente incorrecto, la función devolverá -1.

NOTA Usted es responsable de la destrucción del mapa DS devuelto mediante la función correspondiente.

El mapa DS contendrá las siguientes claves:

  • "x" - La posición x del oyente
  • "y" - La posición y del oyente
  • "z" - La posición z del oyente
  • "vx" - La velocidad del eje x del oyente
  • "vy" - La velocidad del eje Y del oyente
  • "vz" - La velocidad del eje z del oyente
  • "lookat_x" - El componente x del vector look at del oyente
  • "lookat_y" - El componente y del vector look at del oyente
  • "lookat_z" - El componente z del vector look at del oyente
  • "up_x" - El componente x del vector ascendente del oyente
  • "up_y" - El componente y del vector ascendente del oyente
  • "up_z" - El componente z del vector ascendente del oyente

Sintaxis:

audio_listener_get_data(index);

Argumento Tipo Descripción
index ID de oyentereal o de audio The listener to get the data for (default is 0).

Devuelve:

DS Map ID

Ejemplo:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

El código anterior comprueba el número de oyentes disponibles, entonces loops a través de ellos y si su posición x no es 0, establece su posición a 0, 0, 0.
