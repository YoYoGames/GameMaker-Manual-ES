audio_listener_velocidad

Esta función se puede utilizar para dar al oyente efectos Doppler y simular el movimiento del audio basado en el vector que se resuelve a partir de las posiciones relativas x, y y z dadas. Si el oyente en sí no se va a mover nunca, o el movimiento no es un movimiento constante, normalmente no necesitarás establecer estos valores, pero, por ejemplo, si estás haciendo un juego de desplazamiento donde el jugador tiene un movimiento constante de abajo a arriba y los enemigos un movimiento constante de arriba a abajo, establecerás las velocidades del oyente y del emisor (para los emisores usarías audio_emitter_velocity()) a los vectores apropiados para simular el efecto Doppler correcto mientras se mueven más allá de la instancia del jugador.

NOTA si tiene varios oyentes debe utilizar la función audio_listener_set_velocity().

La imagen siguiente muestra cómo podría configurarse para el juego de ejemplo dado anteriormente:

Sintaxis:

audio_listener_velocity(vx, vy, vz);

Argumento Tipo Descripción
vx Real The x velocity component of the listener (default 0).
vy Real The y velocity component of the listener (default 0).
vz Real The z velocity component of the listener (default 0).

Devuelve:

N/A

Ejemplo:

if speed > 0
{
    audio_listener_velocity(abs(hspeed), abs(vspeed), 0);
}

El código anterior comprueba si la velocidad de la instancia del jugador es superior a 0 y, si lo es, utiliza los componentes absolutos apropiados hspeed y vspeed para establecer la velocidad del oyente.
