audio_get_listener_info

Esta función creará un mapa DS y lo rellenará con información para el oyente dado.

NOTA Usted es responsable de la destrucción de los DS Map devueltos utilizando la función correspondiente.

El mapa DS contendrá las siguientes claves:

  • "name" - El nombre de la escucha, como string, siendo "default" el nombre estándar de la escucha en la mayoría de las plataformas de destino
  • "máscara" - La máscara de bits para el oyente
  • "índice" - El valor del índice único del oyente

El valor de la máscara se puede utilizar para establecer que un sonido o emisor se reproduzca desde varios oyentes a la vez, simplemente utilizando el bitwise or "|" para generar una máscara para el sonido (ver el código de ejemplo más abajo), mientras que el índice se utiliza para establecer las propiedades como la posición o la velocidad para un oyente determinado utilizando funciones como audio_listener_set_position().

Sintaxis:

audio_get_listener_info(num);

Argumento Tipo Descripción
num ID de oyentereal o de audio The listener number to get the data for.

Devuelve:

DS Map ID

Ejemplo:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

El código anterior comprueba el número de oyentes disponibles y luego loops a través de ellos obtiene sus bits de máscara, que luego se combinan para crear una única máscara de bits que se aplica al oyente global.
