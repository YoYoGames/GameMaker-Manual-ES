audio_listener_set_position

Con esta función puedes cambiar la posición de un determinado oyente dentro del espacio de audio 3D. El índice del oyente por defecto es 0, pero puede utilizar la función audio_get_listener_info() para obtener los diferentes índices disponibles para la plataforma de destino. La imagen de ejemplo siguiente muestra la posición por defecto del oyente en el espacio de audio:

Como puedes ver, la posición por defecto es (0, 0, 0) pero normalmente usarás esta función para mover el oyente con el jugador object dentro de tu juego y así cambiar la forma en que el audio creado por los emisores es escuchado por el jugador, por ejemplo, en la imagen de abajo de un juego de arriba hacia abajo, la instancia del jugador establece el oyente que hará que el audio de los diversos emisores "cambie" a medida que el jugador se mueve alrededor del nivel:

Sintaxis:

audio_listener_set_position(index, x, y, z);

Argumento Tipo Descripción
index ID de oyentereal o de audio The listener to set the position of (default 0).
x Real The x position of the listener (default 0).
y Real The y position of the listener (default 0).
z Real The z position of the listener (default 0).

Devuelve:

N/A

Ejemplo:

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

El código anterior recupera la matriz de vista para la vista de cámara [0] y luego la utiliza para establecer la posición y la orientación del oyente de audio con el ID almacenado en la variable global "Player_Listener".
