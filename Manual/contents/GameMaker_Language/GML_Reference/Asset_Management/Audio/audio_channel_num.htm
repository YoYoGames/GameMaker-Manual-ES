audio_channel_num

Con esta función puedes establecer cuántos canales de audio están disponibles para reproducir audio en GameMaker. Lo que esto significa básicamente es que usted da el número de sonidos simultáneos que pueden ser reproducidos en cualquier momento, y si el número excede la cantidad, aquellos sonidos con una prioridad más baja son detenidos para liberar un canal para los sonidos con una prioridad más alta. Puedes utilizar esta función para optimizar tu juego para los dispositivos, ya que cuanto menor sea el número de canales para el audio, mejor será el rendimiento, pero ten en cuenta que esto también puede causar un corte notable de ciertos sonidos si se reproducen muchos a la vez.

Sintaxis:

audio_channel_num(num);

ArgumentoTipo Descripción
num Number of available audio channels (default is 128).

Devuelve:

Ejemplo:

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

El código anterior comprueba la plataforma en la que se ejecuta el juego y cambia el número de canales de sonido disponibles para aumentar el rendimiento.
