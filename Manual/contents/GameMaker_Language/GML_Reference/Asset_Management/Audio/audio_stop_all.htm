audio_stop_all

Esta función detendrá todos los sonidos que se estén reproduciendo en ese momento.

Sintaxis:

audio_stop_all();

Devuelve:

Ejemplo:

if !global.SFX
{
    audio_stop_all();
}

El código anterior comprueba la variable global "SFX" y si devuelve false, detendrá todos los sonidos que se estén reproduciendo en ese momento.
