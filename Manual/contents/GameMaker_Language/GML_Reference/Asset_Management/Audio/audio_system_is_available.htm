sistema_de_audio_está_disponible

Esta función se puede utilizar para comprobar y ver si el sistema de audio se ha inicializado, o si el contexto de audio se está ejecutando. En todas las plataformas, esta función devolverá true inmediatamente después de Game Start cuando el motor de audio se inicialice, excepto en el objetivo HTML 5. En HTML5, el estado del contexto de audio puede cambiar en cualquier momento dependiendo de la entrada del usuario, del navegador que se esté utilizando y de su versión, por lo que esta función se puede utilizar para comprobar si el audio se puede reproducir o no. Si la función devuelve false (es decir, el estado del contexto de audio no se está ejecutando), entonces sólo podrán reproducirse los sonidos no transmitidos (pero no está garantizado, y deberías asumir que no se puede reproducir ningún audio), mientras que si devuelve true entonces se reproducirá todo el audio.

Sintaxis:

audio_system_is_available();

Devuelve:

Ejemplo:

if audio_system_is_available()
{
    if audio_is_paused(global.Music)
    {
        audio_resume_sound(global.Music)
    }
    else
    {
        if !audio_is_playing(global.Music)
        {
            global.Music = audio_play_sound(snd_Music, 0, true);
        }
    }
}
else
{
    if audio_is_playing(global.Music)
    {
        audio_pause_sound(global.Music);
    }
}

El código anterior pondrá en pausa/despacio una pista de audio dependiendo de si el sistema de audio está inicializado y disponible o no.
