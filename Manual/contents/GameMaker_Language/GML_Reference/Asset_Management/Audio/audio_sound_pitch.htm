audio_sound_pitch

Esta función puede utilizarse para cambiar el tono de un sonido determinado. El sonido puede ser uno referenciado a partir de un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() o un sonido real asset desde el navegador Asset. Si se trata de un índice de un sonido que se está reproduciendo, entonces sólo se cambiará esa instancia, sin embargo cuando se utiliza un sonido asset del Navegador Asset, se cambiarán todas las instancias de ese sonido asset que se está reproduciendo.

El argumento de tono es un multiplicador de tono, en el sentido de que el valor de entrada multiplica el tono actual por esa cantidad, por lo que el valor por defecto de 1 no supone ningún cambio de tono, mientras que un valor menor que 1 bajará el tono y mayor que 1 lo subirá. Es mejor utilizar incrementos pequeños para esta función, ya que cualquier valor inferior a 0 o superior a 5 puede no ser audible de todos modos. Hay que tener en cuenta que el cambio total de tono permitido está fijado en (1/256) - 256 octavas, por lo que cualquier valor por encima o por debajo de éste no se registrará.

El valor indicado arriba es el que GameMaker intenta fijar en el rango, pero este valor no está garantizado en todas las plataformas. iOS Por ejemplo, GameMaker se ajusta a (1/256) - 8, por lo que es posible que tenga que experimentar en cada plataforma de destino y tener diferentes versiones de un recurso de sonido, cada uno de ellos pre-desplazado, en caso de que necesite valores de octava más altos o más bajos.

Sintaxis:

audio_sound_pitch(index, pitch);

Argumento Tipo Descripción
index Activo de sonido o ID de la instancia de sonido The index of the sound to change.
pitch Real The pitch multiplier (default 1).

Devuelve:

N/A

Ejemplo:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
    case 1: audio_sound_pitch(s_engine, 0.8); break;
    case 2: audio_sound_pitch(s_engine, 0.9); break;
    case 3: audio_sound_pitch(s_engine, 0.95); break;
    case 4: audio_sound_pitch(s_engine, 1); break;
    case 5: audio_sound_pitch(s_engine, 1.2); break;
}

El código anterior cambiará el tono del audio reproducido del sonido indexado en la variable "s_engine" en función del valor de la variable "gear".
