audio_pause_all

Con esta función puedes pausar todos los sonidos que se están reproduciendo en ese momento.

Sintaxis:

audio_pause_all();

Devuelve:

Ejemplo:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

El código anterior comprueba si se pulsa la tecla "P" del teclado y, si detecta una, establece la variable global "Pause" en true o en false y, a continuación, pausa todos los sonidos o reinicia todos los sonidos previamente pausados.
