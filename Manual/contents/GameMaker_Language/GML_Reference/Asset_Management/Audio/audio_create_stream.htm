audio_create_stream

Con esta función se puede crear un nuevo índice de sonido que luego se puede utilizar en las funciones de audio regulares para transmitir audio directamente desde una fuente de archivo OGG externa. La función requiere el nombre del archivo (que puede ser un archivo incluido, por ejemplo) y devolverá el nuevo índice de sonido para su uso. Tenga en cuenta que cuando ya no necesite el sonido deberá llamar a la función audio_destroy_stream() con el índice de sonido para eliminarlo de la memoria, de lo contrario puedes obtener una fuga de memoria que ralentizará y eventualmente bloqueará tu juego.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_create_stream(filename);

Argumento Tipo Descripción
filename Cadena Path to the file (OGG only) to stream the audio from.

Devuelve:

Sound Asset

Ejemplo:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

El código anterior crea un nuevo índice de sonido en la variable "snd" a partir del archivo dado, y luego reproduce este sonido.
