audio_sound_set_track_position

Esta función establece la posición (en segundos) para el ID de sonido dado o asset.

El "índice" suministrado puede ser una instancia única de un sonido (como la devuelta por audio_play_sound()) o un sonido asset (por ejemplo, uno añadido a través del navegador Asset ). El comportamiento de esta función será diferente según el tipo de índice que se haya especificado:

  • Si se trata de un ID de sonido único que se utiliza, que ya se está reproduciendo, entonces su posición cambiará inmediatamente a la hora dada.
  • Si se trata de un sonido asset, todas las reproducciones posteriores del sonido dado asset comenzarán a la nueva hora.

Esta función sólo cambiará la posición de la pista para el sonido que se está reproduciendo, o su siguiente reproducción (en caso de que se suministre un sonido asset ). Si el sonido se reproduce con el bucle activado, las siguientes reproducciones siempre empezarán desde el principio (0,0 segundos), no desde la posición de la pista definida con esta función. Por ejemplo, si se inicia un sonido loop a los 5,7 segundos, se reproducirá el primer sonido desde ese punto, pero después se seguirá repitiendo toda la pista, desde el principio hasta el final.

Sintaxis:

audio_sound_set_track_position(index, time);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to change.
time Real The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

Devuelve:

N/A

Ejemplo:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

El código anterior reproduce un sonido y luego utiliza el valor de ID devuelto para establecer la posición de inicio del sonido a 32 segundos.
