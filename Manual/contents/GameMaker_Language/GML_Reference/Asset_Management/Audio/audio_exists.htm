audio_existe

Esta función devuelve si un sonido existe (true) o no (false). Tenga en cuenta que si el índice que busca no ha sido inicializado previamente, esta función causará un error ya que está buscando índices inexistentes asset. El sonido a comprobar puede ser una única instancia de un sonido (el índice de los sonidos individuales que se reproducen puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un sonido asset.

Sintaxis:

audio_exists(index);

Argumento Tipo Descripción
index Activo The index of the sound to check the existence of.

Devuelve:

Boolean

Ejemplo:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

El código anterior comprueba si existe un sonido. Si lo hace, se configura para que se reproduzca en un loop.
