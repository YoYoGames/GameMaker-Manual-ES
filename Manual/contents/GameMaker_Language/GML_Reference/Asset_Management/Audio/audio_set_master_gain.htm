audio_set_master_gain

Con esta función puedes establecer el valor absoluto del volumen global de todos los sonidos y la música para un oyente específico. El índice del oyente por defecto es 0, pero puedes utilizar la función audio_get_listener_info() para obtener los diferentes índices disponibles para la plataforma de destino. El valor de ganancia se basa en una escala lineal de 0 (silencio) a 1 (volumen máximo) y afectará al volumen relativo de todos los sonidos y la música que se reproduzcan desde el juego a través de ese oyente.

Sintaxis:

audio_set_master_gain(listenerIndex, gain);

Argumento Tipo Descripción
listenerIndex ID de oyentereal o de audio The index of the listener to set the gain on.
gain Real Value for the global volume (0 to 1).

Devuelve:

N/A

Ejemplo:

var num = audio_get_listener_count();
for (var i = 0; i < num; i++)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

El código anterior loops a través de los oyentes disponibles y luego establece su ganancia maestra a 0,75.
