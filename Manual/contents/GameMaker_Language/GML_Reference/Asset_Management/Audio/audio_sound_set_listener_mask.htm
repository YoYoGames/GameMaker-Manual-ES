audio_sound_set_listener_mask

Esta función puede utilizarse para establecer la máscara de bits de un sonido, de modo que sólo se reproduzca en los oyentes especificados. Puede crear una máscara de bits utilizando la función audio_get_listener_info() y luego usando el bitwise or ("|") para crear una máscara personalizada con aquellos oyentes desde los que se requiere que el sonido se reproduzca, y luego aplicar esta máscara personalizada al sonido después de haberlo reproducido. Esta máscara anulará la máscara por defecto o la que haya establecido mediante la función audio_set_listener_mask().

Esta función sólo se puede utilizar en un sonido después de que se haya configurado para reproducirlo y requiere el ID del sonido devuelto por la función utilizada para reproducirlo.

Sintaxis:

audio_sound_set_listener_mask(soundID, mask);

Argumento Tipo Descripción
soundID ID de la instancia de sonido The unique ID of the sound to set the mask of
mask Real The bitmask data to set for the sound

Devuelve:

N/A

Ejemplo:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

El código anterior reproduce un sonido y luego establece la(s) escucha(s) que debe(n) reproducir utilizando los datos de la máscara almacenada en una variable global.
