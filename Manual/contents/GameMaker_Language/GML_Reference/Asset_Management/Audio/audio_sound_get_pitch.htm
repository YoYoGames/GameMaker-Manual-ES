audio_sound_get_pitch

Esta función puede utilizarse para obtener el tono de un sonido dado. El sonido puede ser uno referenciado a partir de un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() o un sonido real asset desde el navegador Asset.

Sintaxis:

audio_sound_get_pitch(index);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to get the pitch of.

Devuelve:

Real

Ejemplo:

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

El código anterior cambiará el tono del audio reproducido del sonido indexado como "snd_Explode" si su tono no es igual a 1.
