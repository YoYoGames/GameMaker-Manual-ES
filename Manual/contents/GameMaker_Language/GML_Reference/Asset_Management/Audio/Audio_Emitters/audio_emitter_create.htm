audio_emitter_create

Esta función crea un nuevo emisor de audio y devuelve su índice. Este índice debe ser almacenado en una variable para todas las funciones posteriores que se relacionen con este emisor, y luego, cuando ya no sea necesario, debe ser eliminado de la memoria utilizando la función audio_emitter_free() para evitar fugas de memoria que pueden llegar a bloquear el juego.

El emisor de audio se crea con la siguiente configuración por defecto:

  • Una ganancia y un paso de 1
  • Una máscara de oyente configurada sólo para el oyente por defecto (1)
  • Una posición de (0, 0, 0)
  • Una velocidad de 0 (vx = 0, vy = 0, vz = 0)

Sintaxis:

audio_emitter_create();

Devuelve:

Audio Emitter ID

Ejemplo:

s_emit = audio_emitter_create();

El código anterior crea un nuevo emisor de audio y asigna su índice a la variable "s_emit".
