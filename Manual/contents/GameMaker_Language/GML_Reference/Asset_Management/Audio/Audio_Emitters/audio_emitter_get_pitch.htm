audio_emitter_get_pitch

Esta función devuelve el valor de tono actual establecido para el emisor de audio dado.

Sintaxis:

audio_emitter_get_pitch(emitter);

Argumento Tipo Descripción
emitter Identificación del emisor de audio The index of the emitter to use.

Devuelve:

Real

Ejemplo:

if audio_emitter_get_pitch(emitter_player) != 1
{
    audio_emitter_pitch(emitter_player, 1);
}

El código anterior comprueba el paso actual de un determinado emisor y si no es igual a 1 se pone a 1.
