audio_emitter_get_z

Esta función devuelve la posición z actual del emisor de audio dado.

Sintaxis:

audio_emitter_get_z(emitter);

Argumento Tipo Descripción
emitter Identificación del emisor de audio The index of the emitter to use.

Devuelve:

Real

Ejemplo:

if audio_emitter_get_z(emitter_player) != 0
{
    var ex = audio_emitter_get_x(emitter_player);
    var ey = audio_emitter_get_y(emitter_player);
    audio_emitter_position(emitter_player, ex, ey, 0);
}

El código anterior comprueba la posición z actual de un emisor determinado y si no es igual a 0, se pone a 0.
