ganancia_emisora_de_audio

Esta función establece la ganancia máxima (volumen) del sonido. El volumen percibido para un sonido puede cambiar dependiendo del valor de caída y de la posición que tenga con respecto al oyente, pero al establecer la ganancia con esta función, el volumen total nunca superará el valor de ganancia especificado. La imagen siguiente ilustra cómo la ganancia afecta al volumen del emisor cuando el fall-off es mayor que 0:

Esta función cambiará el volumen del sonido mientras se está reproduciendo, así como todos los sonidos posteriores que se reproduzcan a través del emisor dado. Ten en cuenta que en algunas plataformas puedes tener una ganancia superior a 1, aunque un valor de 1 se considera "volumen máximo" y cualquier valor superior puede introducir recortes de audio o distorsión.

NOTA el volumen final también se verá influenciado por la ganancia de audio global que se haya establecido mediante la función audio_master_gain().

Sintaxis:

audio_emitter_gain(emitter, gain);

Argumento Tipo Descripción
emitter Identificación del emisor de audio The index of the emitter to change.
gain Real The maximum gain (default 1).
time Real  The length of the change in gain in milliseconds. Defaults to 0 (instantaneous change) if not passed.

Devuelve:

N/A

Ejemplo:

if up
{
    gain += 0.05;
    if (gain >= 1) up = false;
}
else
{
    gain -= 0.05;
    if (gain <= 0) up = true;
}

audio_emitter_gain(s_emit, gain);

El código anterior establece la variable "gain" a diferentes valores y luego utiliza esa misma variable para establecer la ganancia del emisor indexado en la variable "s_emit".
