audio_emitter_set_listener_mask

Esta función puede utilizarse para establecer la máscara de bits de un emisor, de modo que todos los sonidos reproducidos a través del emisor se reproduzcan sólo desde los oyentes especificados. Se puede crear una máscara de bits utilizando la función audio_get_listener_info() y luego usando el bitwise or ("|") para crear una máscara personalizada con aquellos oyentes de los que se requiere que el sonido se reproduzca, y luego aplicar esta máscara personalizada al emisor. Esta máscara anulará la máscara por defecto o la que hayas establecido con la función audio_set_listener_mask().

Sintaxis:

audio_emitter_set_listener_mask(emitterID, mask);

Argumento Tipo Descripción
emitterID Identificación del emisor de audio The unique ID of the emitter to set the mask of
mask Real The bitmask data to set for the sound

Devuelve:

N/A

Ejemplo:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

El código anterior crea un emisor de audio y luego establece la(s) escucha(s) para reproducir usando los datos de la máscara almacenados en una variable global.
