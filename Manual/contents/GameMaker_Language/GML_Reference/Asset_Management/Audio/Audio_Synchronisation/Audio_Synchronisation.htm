Sincronización de audio

Las funciones de grupo de sincronización de audio te permiten reproducir varios sonidos que comienzan exactamente al mismo tiempo y se mantienen sincronizados (a nivel de muestra). Esto te permite (por ejemplo) añadir varias pistas para una misma canción y luego fundir secciones sin tener que preocuparte de si habrá algún retraso entre ellas. Ten en cuenta que estas funciones actualmente sólo funcionan con audio comprimido, es decir, con archivos de formato *.ogg y *.mp3.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Para la sincronización de las pistas de audio existen las siguientes funciones:

  • audio_create_sync_group
  • audio_play_in_sync_group
  • audio_start_sync_group
  • audio_stop_sync_group
  • audio_pause_sync_group
  • audio_resume_sync_group
  • audio_sync_group_get_track_pos
  • audio_destroy_sync_group
  • audio_sync_group_debug
  • audio_sync_group_is_playing
