audio_sync_group_debug

sta función puede utilizarse para mostrar información de depuración sobre un grupo de sincronización determinado (el índice del grupo es el valor devuelto cuando se creó el grupo mediante la función audio_create_sync_group()). Puede llamar a la función con un valor de -1 para que switch desactive la superposición.

Cuando esté debugging los grupos de sincronización de audio, se mostrará la superposición:

  • Qué grupo de sincronización está depurando
  • Cuántos canales tiene
  • Cuántas veces ha fallado el grupo de sincronización (esto significa que ha intentado hacer demasiado trabajo)
  • Una lista de los canales
    • El color rojo significa que este canal no está haciendo ningún trabajo
    • Amarillo significa que el canal está en estado inicial (aún no está decodificando o reproduciendo)
    • Verde significa que el canal está buscando/decodificando actualmente
    • El color púrpura significa que el canal se ha quedado sin datos
  • Una barra de perfil que muestra qué trabajo está haciendo el hilo de decodificación
    • azul significa que se está leyendo un archivo por primera vez
    • el rojo significa que el hilo está leyendo un archivo para saber de dónde leer
    • el verde significa que el hilo está decodificando datos
    • el color púrpura significa que el hilo está durmiendo hasta que tenga que trabajar

Tenga en cuenta que para el general debugging de audio, también tiene la función audio_debug().

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_sync_group_debug(group_index);

Argumento Tipo Descripción
group_index ID de grupo de sincronización de audio The group index to debug, or -1 to switch off the debugging.

Devuelve:

N/A

Ejemplo:

if (debug_mode)
{
    audio_sync_group_debug(sg);
}
else
{
    audio_sync_group_debug(-1);
}

El código anterior switch activará o desactivará la superposición de la sincronización del grupo de audio mostrando el grupo de sincronización indexado en la variable "sg" dependiendo de si el juego se ejecuta en modo de depuración o no.
