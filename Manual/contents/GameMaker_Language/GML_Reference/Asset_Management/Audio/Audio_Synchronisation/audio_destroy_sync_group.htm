audio_destroy_sync_group

Los grupos de sincronización de audio deben ser destruidos cuando no estén en uso para liberar la memoria y los recursos de sonido asociados a ellos utilizando esta función. Toma el índice del grupo tal y como se devolvió cuando se creó el grupo mediante la función audio_create_sync_group()y libera todos los recursos utilizados por el grupo.

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_destroy_sync_group(group_index);

Argumento Tipo Descripción
group_index ID de grupo de sincronización de audio The group index to be destroyed.

Devuelve:

N/A

Ejemplo:

audio_destroy_sync_group(sg);

El código anterior destruye el grupo de sincronización indexado en la variable "sg", y probablemente se utilizaría en los eventos destroy o Room End.
