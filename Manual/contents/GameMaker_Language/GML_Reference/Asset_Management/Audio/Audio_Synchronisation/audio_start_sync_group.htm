audio_start_sync_group

Con esta función puedes empezar a reproducir un grupo de sincronización previamente creado. Es necesario suministrar el índice del grupo, que es el valor devuelto al crear el grupo mediante la función audio_create_sync_group().

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.

Sintaxis:

audio_start_sync_group(group_index);

Argumento Tipo Descripción
group_index ID de grupo de sincronización de audio The group index to play.

Devuelve:

N/A

Ejemplo:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Lo anterior crea un nuevo grupo de sincronización y asigna el índice del grupo a la variable "sg". A continuación, se añaden cuatro sonidos al grupo, con la ganancia de tres de ellos ajustada a 0. Finalmente se reproduce el grupo de sincronización.
