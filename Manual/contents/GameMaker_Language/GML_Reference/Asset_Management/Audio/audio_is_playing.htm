audio_is_playing

Esta función comprobará el sonido dado para ver si se está reproduciendo actualmente. El sonido puede ser una instancia única de un sonido (el índice de los sonidos individuales que se están reproduciendo puede almacenarse en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() ) o un sonido asset, en cuyo caso se comprobarán todas las instancias del sonido dado y si alguna de ellas se está reproduciendo la función devolverá true, en caso contrario devolverá false. Tenga en cuenta que esta función seguirá devolviendo true si el sonido que se está comprobando ha sido pausado previamente utilizando la función audio_pause_sound() función.

Sintaxis:

audio_is_playing(index);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to check.

Devuelve:

Boolean

Ejemplo:

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

El código anterior comprueba si el sonido indexado en la variable "snd_Waterfall" se está reproduciendo actualmente y si devuelve false entonces el sonido se reproducirá en bucle en su posición room, con una distancia de caída de 300 y una prioridad baja.
