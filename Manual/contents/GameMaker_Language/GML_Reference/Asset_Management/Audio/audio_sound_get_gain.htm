audio_sound_get_gain

Esta función devolverá el valor de ganancia actual para el sonido dado. El sonido puede ser uno referenciado a partir de un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se utiliza la función audio_play_sound() o audio_play_sound_at() o un sonido real asset desde el navegador Asset. La ganancia se calcula normalmente como un valor de 0 a 1, pero en algunas plataformas se puede tener una ganancia superior a 1, aunque un valor de 1 se considera "volumen completo" y cualquier valor superior puede introducir recortes de audio.

Sintaxis:

audio_sound_get_gain(index);

Argumento Tipo Descripción
index Activo de sonido o ID de instancia de sonido The index of the sound to get the gain of.

Devuelve:

Real

Ejemplo:

if audio_sound_get_gain(snd_Music) != 1
{
    audio_sound_gain(snd_Music, 1, 0);
}

El código anterior cambiará la ganancia del audio reproducido del sonido indexado como "snd_Music" si su ganancia no es igual a 1.
