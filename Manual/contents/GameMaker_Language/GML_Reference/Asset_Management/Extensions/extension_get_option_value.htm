extension_get_option_value

Esta función devuelve el valor de la opción dada en el extension con el nombre dado.

La función devolverá undefined si el nombre de extension proporcionado no es válido.

Sintaxis:

extension_get_option_value(extension_name, option_name);

Argumento Tipo Descripción
extension_name Cadena The name of your extension asset as a string
option_name Cadena The name of the option to read

Devuelve:

Any

Ejemplo:

var _enabled = extension_get_option_value("MyExtension", "enabled");

Este código obtiene el valor de la opción habilitada en la página web MyExtension extension.
