extension_get_option_names

Esta función devuelve un array que contiene los nombres de todas las opciones que existen en la extension con el nombre dado.

La función devolverá undefined si el nombre de extension proporcionado no es válido.

Sintaxis:

extension_get_option_names(extension_name);

Argumento Tipo Descripción
extension_name Cadena The name of your extension asset as a string

Devuelve:

Array of Strings

Ejemplo:

var _names = extension_get_option_names("MyExtension");
var _count = extension_get_option_count("MyExtension");

for (var i = 0; i < _count; i ++)
{
    var _option_name = _names[i];

    show_debug_message(_option_name + ": ");
    show_debug_message(extension_get_option_value("MyExtension", _option_name));
}

Este código obtiene los nombres de todas las opciones, y el número de opciones en extension MyExtension.

loops a través de todos los nombres de las opciones, e imprime el nombre y el valor de cada opción en el registro de salida.
