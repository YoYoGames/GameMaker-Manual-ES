extension_get_options

Esta función devuelve un struct que contiene todas las opciones, y sus valores, para el extension con el nombre dado.

La función devolverá undefined si el nombre de extension proporcionado no es válido.

Sintaxis:

extension_get_options(extension_name);

Argumento Tipo Descripción
extension_name Cadena The name of your extension asset as a string

Devuelve:

Struct

Ejemplo:

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

Este código obtiene las opciones struct para un extension y lee un valor de él.
