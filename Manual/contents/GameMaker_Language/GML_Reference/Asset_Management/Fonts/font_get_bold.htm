font_get_bold

Con esta función puede comprobar cualquier font asset para ver si tiene la negrita flag o no. Si lo tiene la función devolverá true, en caso contrario devolverá false.

Sintaxis:

font_get_bold(ind);

ArgumentoTipo Descripción
ind Index of the font to check.

Devuelve:

Ejemplo:

if font_get_bold(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Esto establecerá el dibujo activo font en fnt_Main si se establece como negrita en sus propiedades font.
