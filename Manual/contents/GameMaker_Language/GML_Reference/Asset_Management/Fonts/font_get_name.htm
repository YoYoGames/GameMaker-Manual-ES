font_get_name

Esta función se puede utilizar para obtener el nombre (como string) que se le dio a la font cuando se añadió como asset al Navegador de GameMaker Asset. Tenga en cuenta que esto es sólo un string y no puede ser utilizado para referenciar el font directamente - para eso necesitaría el índice de la fuente. Sin embargo, puede utilizar este string para obtener el índice de la fuente utilizando el string devuelto junto con la función función asset_get_index().

Sintaxis:

font_get_name(ind);

ArgumentoTipo Descripción
ind Index of the font to check.

Devuelve:

Ejemplo:

fnt_Name = font_get_name(font0);

El código anterior obtendrá el nombre de un recurso font tal y como aparece en el navegador Asset y lo almacenará como string en la variable "fnt_Name".
