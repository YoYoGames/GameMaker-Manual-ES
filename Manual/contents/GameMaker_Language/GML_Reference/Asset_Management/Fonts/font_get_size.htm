font_get_size

Con esta función puede obtener el tamaño de cualquier recurso de font, que es el valor en puntos que muestra el diálogo de recursos de font.

Sintaxis:

font_get_size(ind);

ArgumentoTipo Descripción
ind Index of the font to get the size of.

Devuelve:

Ejemplo:

sz = font_get_size(font0);

Esto obtendrá el tamaño del font indexado por la variable "font0" y lo almacenará en la variable "sz".
