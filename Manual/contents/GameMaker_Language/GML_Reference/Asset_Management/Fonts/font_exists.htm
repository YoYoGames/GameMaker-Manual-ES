fuente_existe

Esta función devuelve si existe o no un font con el índice especificado. Puede comprobar los índices de font tal y como se definen en el navegador Asset, o fonts que se han añadido utilizando funciones como font_add().

Sintaxis:

font_exists(ind);

ArgumentoTipo Descripción
ind Index of the font to check.

Devuelve:

Ejemplo:

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Esto establecerá el dibujo activo font a fnt_Main si existe.
