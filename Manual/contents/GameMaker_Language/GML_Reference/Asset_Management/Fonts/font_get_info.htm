font_get_info

Esta función se utiliza para recuperar la información del font dado. Se suministra un índice font (que puede ser un asset añadido a través del navegador de activos o un font añadido en runtime) y la función devuelve una estructura con las siguientes variables:

Estructura de información de fuentes
Nombre de la variable Tipo de datos Descripción
ascenderOffset Real Desplazamiento máximo desde la línea de base hasta la parte superior de font (en píxeles)
size Real El tamaño aproximado de font (en píxeles)
spriteIndex Activo Sprite El índice de sprite para el font si fue creado a partir de un sprite, de lo contrario -1
texture Puntero -1 si el font fue creado a partir de un sprite, de lo contrario el ID de la textura de la fuente
name Cadena El nombre de la fuente
bold Booleano true si el font está en negrita, en caso contrario false (véase la NOTA más abajo)
italic Booleano true si el font está en cursiva, en caso contrario false (véase la NOTA más abajo)
glyphs Estructura del glifo de la fuente Un struct que contiene información para cada glifo en el font (más información a continuación)

NOTA Las variables bold y italic sólo reflejan la configuración del usuario para font, por lo que pueden no ser precisas para fonts que aparecen en negrita o cursiva por defecto.

Si el font suministrado no existe, la función devolverá undefined.

Tenga en cuenta también que el cambio de los valores de cualquiera de estas variables, o de cualquier variable contenida en glyphs struct, no cambiará la forma en que se representa font, por lo que esta información debe considerarse de sólo lectura.

Estructura de los glifos

La variable glyphs en el struct devuelto será un struct por sí mismo, que contendrá información para cada glifo incluido en el font. Cada variable en esta estructura será el nombre del carácter de un glifo, que contendrá un struct con información para ese glifo. Puede imaginar estos structs anidados en el siguiente formato (empezando por el principal struct):

{
    glyphs:
    {
        A: {},
        B: {},
        C: {},
        // ...other glyphs
    },
    // ...other font info
}

Puede utilizar el nombre del carácter de un glifo para recuperar su información de struct, como en el siguiente código:

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ is a struct accessor

show_debug_message(_info_A);

El código anterior obtiene el glifo struct para el carácter "A" y lo imprime en el registro de salida. También puede sustituir el string ("A") por una variable que contenga un string, lo que le permitirá obtener información para cualquier glifo a través de esa variable en runtime.

El struct devuelto para un glifo contendrá las siguientes variables:

NOTA En HTML5, los glifos struct estarán vacíos para cualquier archivo cargado fonts (como *.ttf o *.otf) ya que el runtime no tendrá información sobre esos glifos.

Estructura del glifo de la fuente
Nombre de la variable Tipo de datos Descripción
char real Si el font fue creado a partir de un sprite, será el índice de imagen del glifo de ese sprite, de lo contrario será su número de carácter Unicode

Nota: Todas las variables por debajo de ésta no estarán presentes en el struct si el font fue creado a partir de un sprite
x real La posición X del glifo en la página de la textura (en texels)
y real La posición Y del glifo en la página de la textura (en texels)
w real La anchura del glifo en la página de la textura (en texels)
h real La altura del glifo en la página de la textura (en texels)
shift real El número de píxeles a desplazar a la derecha cuando se avanza al siguiente carácter (puede ser negativo para desplazar a la izquierda)
offset real El número de píxeles para desplazar horizontalmente la representación de este glifo (puede ser positivo o negativo)
kerning matriz Una matriz de enteros que contiene información sobre el interletraje en pares (o grupos de 2). El primer entero de un par es el valor Unicode de un carácter, y el segundo entero es la cantidad que hay que añadir al valor de desplazamiento de ese carácter (puede ser positivo o negativo) si está precedido por el carácter de este glifo.

NOTA Para el archivo fonts cargado en runtime, los valores de x y y para un glifo pueden ser -1 si no ha sido renderizado aún, ya que no tendrá posición en la página de textura utilizada para la caché.

Sintaxis:

font_get_info(font);

Argumento Tipo Descripción
font Activo de fuentes The font for which information should be retrieved.

Devuelve:

Font Info Struct (or undefined)

Ejemplo:

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
    image_xscale *= 2;
    image_yscale *= 2;
}

Este código recupera la información de font almacenada en la variable my_font. Si el valor devuelto no esundefined y el font dado está en negrita, la escala de la instancia se duplica.
