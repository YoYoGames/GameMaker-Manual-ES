font_replace

Esta función se puede utilizar para reemplazar un font en su juego de aquellos fonts que están instalados en el sistema en el que se está ejecutando. Tenga en cuenta que el font que se sustituye no puede ser un font del navegador Asset, y debe ser un font que se añadió previamente utilizando la función font_add().

Al reemplazar un font de esta manera, puede definir el tamaño del font (en puntos), así como si el font debe estar en negrita o en cursiva, y también puede definir el rango de caracteres a incluir. Si la función falla, lo hará silenciosamente y el font dado no será reemplazado.

Para el módulo de destino de HTML5, esta función sólo puede utilizarse para reemplazar las fuentes web, y sólo funcionará si WebGL no está habilitado - si WebGL está habilitado, debe utilizar un recurso general font (es decir, utilizando draw_set_font() junto con un font del navegador Asset ) o un mapa de bits fonts (véase font_add_sprite()).

IMPORTANTE Cuando se dirige a HTML5, el texto en cursiva y en negrita sólo se admite si el propio font asset o web font tiene incluidos estos caracteres. No se sintetizarán a partir de la base font.

En todos los demás objetivos (no web), puede utilizar esta función para reemplazar un font desde un archivo. El archivo debe incluirse en el paquete del juego utilizando la funcionalidad de Archivos Incluidos de GameMaker, y debe ser un archivo con formato *.ttf o *.otf font , útil para añadir fonts no estándar como el asiático o el árabe.

SUGERENCIA Si incluye un archivo de este tipo font con un juego, asegúrese de que tiene la licencia adecuada para su distribución con el juego.

Cuando se carga un font desde un archivo de esta manera, el tamaño del font está en píxeles y los valores primero y último se ignoran, lo que significa que se añadirán todos los glifos en el font. Los Fonts añadidos de esta manera se asignan a su propia página de textura, por lo que se debe tener cuidado al utilizar esta función, ya que aumentará el número de intercambios de textura al dibujar. También hay que tener en cuenta que fonts puede parecer un poco más grande cuando se dibuja, ya que los glifos pueden tener partes que se dibujan fuera de la caja delimitadora definida para el font. También hay que tener en cuenta que hay una limitación de alrededor de 200 glifos que se pueden renderizar en un solo cuadro para un font en particular.

NOTA Cuando cargues un font en GameMaker debes recordar eliminarlo de nuevo (con font_delete()) cuando ya no lo necesites, de lo contrario existe el riesgo de que se produzca una fuga de memoria que ralentizará y acabará por colapsar tu juego.

Sintaxis:

font_replace(ind, name, size, bold, italic, first, last);

Argumento Tipo Descripción
ind Activo de fuentes The index of the (previously added) font to be replaced.
name Cadena The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf o *.otf archivo.
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Booleano Whether the font is bold (true) or not (false).
italics Booleano Whether the font is italic (true) or not (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Devuelve:

N/A

Ejemplo:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

Esto sustituirá el font indexado en la variable "myfont" por un nuevo font que tiene un tamaño de 24pt, utiliza "Arial" y que es negrita y cursiva. El rango de font incluye letras mayúsculas y minúsculas, números y toda la puntuación común.
