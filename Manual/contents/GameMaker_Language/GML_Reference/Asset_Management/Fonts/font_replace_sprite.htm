font_replace_sprite

Con esta función puedes usar una "tira de sprites" para reemplazar un sprite font creado previamente asset, donde cada sub-imagen sería un símbolo o letra individual. El índice es el número de mapa UTF8 para el primer carácter a utilizar y los rangos a utilizar son los mismos que para un font normal, empezando por "!" pero recuerde que sólo debe incluir lo que necesita y que debe utilizar la función ord() para obtener el valor UTF8 correcto para la letra inicial.

También puede establecer que el espaciado de font sea proporcional (true) o no (false), donde un font proporcional se espacia en base a la anchura real de las letras (por lo que la letra "i" ocupa menos room que la letra "w", por ejemplo) mientras que un font no proporcional se espaciará en base a la anchura de la subimagen, como un font monoespaciado. Por último, se puede definir el espacio que se dejará entre cada letra al escribir, y éste puede ser cualquier valor entero, siendo 0 sin espacio (las letras se "tocarán" si son proporcionales).

Si la función falla, entonces fallará silenciosamente sin reemplazar el sprite font dado.

También hay que tener en cuenta que no puede haber espacios en el mapa de caracteres, de modo que si, por ejemplo, utiliza '.' y '0' pero no '/', deberá incluir '/' de todos modos o dejar una subimagen en blanco entre los dos caracteres utilizados. Sintaxis:

font_replace_sprite(ind, spr, first, prop, sep);

Argumento Tipo Descripción
ind Activo de fuentes The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Activo Sprite The sprite to add a font based on.
first Real The index of the first available character.
prop Booleano Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Real The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Devuelve:

N/A

Ejemplo:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

El código anterior sustituirá el font asset indexado en la variable global "Font" por un nuevo font asset de un archivo de imagen externo.
