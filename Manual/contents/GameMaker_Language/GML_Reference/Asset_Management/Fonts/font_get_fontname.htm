font_get_fontname

Con esta función se puede obtener el nombre real del sistema del font asset dado. Esta función devuelve una cadena y no un índice, y el nombre devuelto dependerá del font que se esté utilizando. Por ejemplo, puede tener un font asset llamado "fnt_Main" en el navegador Asset, y el propio font puede ser el sistema Windows font Arial. En este caso, la función devolverá "Arial ", ya que ese es el nombre del sistema de font. Si necesita el nombre que aparece en el navegador Asset, utilice font_get_name().

El comportamiento descrito anteriormente sólo se aplica a font assets añadidos a través de IDE. Cuando se utiliza en un font cargado en runtime (utilizando font_add()), la función devolverá el path completo a ese archivo font en lugar de su nombre font.

Sintaxis:

font_get_fontname(ind);

ArgumentoTipo Descripción
ind Index of the font to check.

Devuelve:

String

Ejemplo:

fnt_Name = font_get_fontname(font0);

El código anterior obtendrá el nombre del sistema de un recurso font y lo almacenará como string en la variable "fnt_Name".
