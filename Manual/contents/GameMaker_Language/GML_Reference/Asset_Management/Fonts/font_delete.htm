font_delete

Con esta función puedes eliminar un font asset del juego. Esta es una eliminación permanente, y cambiar rooms, o reiniciar el juego no traerá de vuelta el font eliminado. Para ello el jugador tendría que salir del juego y reiniciar de esa manera, así que ten cuidado cuando uses esta función. En general, sólo se necesita para liberar la memoria que ha sido utilizada por un font añadido al juego a través de las funciones font_add() o font_add_sprite().

Sintaxis:

font_delete(ind);

ArgumentoTipo Descripción
ind The index of the font to delete.

Devuelve:

Ejemplo:

font_delete(global.Font);

El código anterior borrará el font indexado en la variable global "Font".
