tamaño_de_página_de_textura_de_la_fuente

Esta variable incorporada se puede utilizar para obtener o establecer el tamaño de la página de la textura cuando se utiliza la función font_add(). Al añadir un font usando esa función, GameMaker creará una página de textura cache de los glifos requeridos hasta el tamaño definido por esta variable. El proceso es el siguiente:

  • Cada carácter que se utiliza se almacena en la memoria caché de font en una página de textura
  • Cuando la página de texturas está llena (porque usas personajes grandes o muchos personajes diferentes) GameMaker eliminará los personajes de la cache y los sustituirá por los nuevos que necesites
  • GameMaker trata de no hacer la textura demasiado grande porque no todos los dispositivos pueden soportar texturas grandes, y trata de no hacer la textura demasiado pequeña porque eliminar y volver a añadir caracteres a la textura lleva tiempo de la CPU y hace que su juego se ejecute lentamente

Si crees que necesitas un tamaño de página de textura mayor o menor que el predeterminado de 1024px que utiliza GameMaker, puedes establecer el valor utilizando esta variable incorporada. Si no está utilizando la función font_add(), entonces el ajuste de esto no tendrá ningún efecto en el rendimiento de sus juegos como fonts añadido en el IDE se almacenan en caché en las páginas de textura regular (que se puede controlar desde el Editor de Fuentes y las Opciones de Juego para la plataforma de destino).

Sintaxis:

font_texture_page_size;

Devuelve:

Real (maximum width/height in pixels)

Ejemplo:

font_texture_page_size = 512;
newfont = font_add("Arial", 24, true, true, 32, 128);

El código anterior establece la página de texturas font cache a un tamaño máximo de 512px y luego añade un nuevo font al juego.
