font_add_enable_aa

Esta función puede utilizarse para activar o desactivar el antialiasing (AA) para fonts añadido mediante font_add(). Esta función debe llamarse antes de añadir cualquier fonts y si se establece como true se activará el AA, y si se establece como false se desactivará. Por defecto, el AA está activado. Tenga en cuenta que esto no tendrá ningún efecto en fonts que se hayan añadido antes de llamar a la función, y la función sólo necesita ser llamada una vez cuando se añada el font, y no cada dibujo/paso que se esté utilizando el font.

Sintaxis:

font_add_enable_aa(enable);

ArgumentoTipo Descripción
enable Whether to enable (true) o desactivar (false) AA para añadir fonts.

Devuelve:

Ejemplo:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

El código anterior comprueba el estado del antialiasing para el añadido fonts y si no está habilitado, entonces lo habilitamos.
