font_get_texture

Esta función devuelve un puntero especial para la página de textura font. Este valor puede ser usado en otras funciones de dibujo, particularmente en el dibujo general cuando se usan primitivas así como en las funciones Shader. Puede obtener más información sobre la página de textura devuelta utilizando las diferentes funciones de texture_ que se encuentran aquí.

Sintaxis:

font_get_texture(font);

Argumento Tipo Descripción
font Activo de fuentes The index of the font to use.

Devuelve:

Texture

Ejemplo:

tex = font_get_texture(fnt_Main);

El código anterior obtendrá el puntero de textura para el font indexado como "fnt_Main".
