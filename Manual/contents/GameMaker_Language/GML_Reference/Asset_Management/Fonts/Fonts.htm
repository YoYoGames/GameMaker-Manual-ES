Fuentes

GameMaker dispone de una serie de funciones que pueden utilizarse para obtener información sobre el font assets incluido, así como otro conjunto de funciones que pueden utilizarse para cambiar, añadir o eliminar fonts de su juego en tiempo de ejecución. En particular, la función font_add() puede utilizarse para añadir fonts no estándar, como el glifo asiático fonts, desde un archivo *.ttf o *.otf.

Vale la pena notar que si agrega un font a su juego que no tiene los glifos requeridos por el texto que desea escribir, entonces el carácter Unicode 9647 (▯) es usado para sustituir esos glifos faltantes cuando se renderiza en el evento de dibujo. Así que si su font no tiene, por ejemplo, el símbolo º, entonces escribir 90º producirá realmente 90▯.

Las siguientes funciones pueden utilizarse para obtener información sobre los recursos de font:

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs
  • font_get_info

Las siguientes funciones se pueden utilizar para cambiar y manipular fonts durante una partida:

  • font_exists
  • font_set_cache_size
  • font_add_enable_aa
  • font_add_get_enable_aa
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
