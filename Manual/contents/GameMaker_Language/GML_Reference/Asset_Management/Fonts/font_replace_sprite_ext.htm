font_replace_sprite_ext

Esta función sustituirá a un sprite font creado previamente con la función font_add_sprite_ext(). Utiliza una "tira de sprites", donde cada sub-imagen es un símbolo o letra individual, y las imágenes son mapeadas a letras basadas en el argumento "string_map" de la función. Este argumento es un string que puedes usar para decirle a GameMaker en qué orden están las sub-imágenes del sprite font y las mapeará automáticamente al escribir el texto. Así, por ejemplo, si tienes un string-map de "AaBbCcDdEeFfGgHh", tu sprite font debe tener las sub-imágenes ordenadas de esta manera.

También puede establecer que el espaciado de font sea proporcional (true) o no (false), donde un font proporcional se espacia en base a la anchura real de las letras (por lo que la letra "i" ocupa menos room que la letra "w", por ejemplo) mientras que un font no proporcional se espaciará en base a la anchura de la subimagen, como un font monoespaciado. Por último, se puede definir el espacio que se dejará entre cada letra al escribir, y éste puede ser cualquier valor entero, siendo 0 sin espacio (las letras se "tocarán" si son proporcionales). La función devuelve un valor de índice que debe ser almacenado en una variable, ya que será necesario en todos los códigos posteriores que hagan referencia a este font.

Si la función falla, entonces fallará silenciosamente sin reemplazar el sprite font dado.

NOTA Esta función requiere que hayas añadido previamente un sprite font usando font_add_sprite()ya que lo sustituirá y no se puede utilizar en fonts que se incluye como parte de los recursos del juego.

Sintaxis:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argumento Tipo Descripción
font Activo de fuentes The font to replace.
spr Activo Sprite The sprite to add a font based on.
string_map Cadena String from which sprite sub-image order is taken.
prop Booleano Set as proportional font or not.
sep Real The space to leave between each letter.

Devuelve:

N/A

Ejemplo:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

El código anterior reemplazará un sprite font asset que está indexado en la variable "global.Font" con imágenes sprite indexadas en la variable "spr_CalcFont".
