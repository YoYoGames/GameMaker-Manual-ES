font_add_get_enable_aa

Esta función puede utilizarse para comprobar si el antialiasing (AA) está activado para fonts añadido mediante font_add(). La función devolverá true si el AA está habilitado, y false si no lo está. Tenga en cuenta que el AA está activado por defecto, pero puede cambiar el estado del AA para los fonts añadidos utilizando la función font_add_enable_aa()siempre y cuando se llame antes de añadir el font.

Sintaxis:

font_add_get_enable_aa();

Devuelve:

Ejemplo:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

El código anterior comprueba el estado del antialiasing para el añadido fonts y si no está habilitado, entonces lo habilitamos.
