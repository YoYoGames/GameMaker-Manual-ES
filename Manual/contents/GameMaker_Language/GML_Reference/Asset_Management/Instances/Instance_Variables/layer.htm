capa

Esta variable incorporada se crea para cada instancia en un room y contiene el valor del ID de la capa a la que está asignada la instancia. Este valor puede utilizarse en otras funciones como layer_get_depth() o se puede cambiar para mover la instancia a otra capa, pero ten en cuenta que si la capa que se está asignando no existe en el actual room, entonces obtendrás un error que obligará a cerrar tu juego. Cuando se asigna una capa, se debe suministrar el ID único de la capa devuelto por la función layer_get_id() (cuando se utilizan capas con nombre room ), o tal como lo devuelve la función layer_create() (cuando creas tus propias capas en tiempo de ejecución).

IMPORTANTE Si ha creado la instancia con la función instance_create_depth() o ha cambiado manualmente la variable depth la capa asignada a la instancia se convierte en una capa "gestionada", es decir, una capa que GameMaker controla y gestiona automáticamente. En estos casos la variable layer devolverá -1.

Sintaxis:

layer;

Devuelve:

Layer ID

Ejemplo:

layer = layer_create(-1000);

El código anterior creará una nueva capa con una profundidad de -1000 y luego establecerá la variable de instancia layer al ID de capa devuelto, moviendo la instancia de la capa en la que se encuentra actualmente a la nueva capa que se está creando.
