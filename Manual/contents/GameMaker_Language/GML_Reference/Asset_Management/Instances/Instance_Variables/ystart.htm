ystart

Esta variable almacena la posición inicial y de la instancia cuando se crea por primera vez en el room. No es una variable de sólo lectura y puede ser establecida así como leída.

Sintaxis:

ystart;

Devuelve:

(single precision floating point value)

Ejemplo:

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

El código anterior comprobará si hay una colisión con una instancia de "obj_spike" y, si la hay, descuenta 100 de la puntuación y devuelve la instancia a su posición inicial.
