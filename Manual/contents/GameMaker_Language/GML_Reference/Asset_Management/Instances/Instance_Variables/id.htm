id

Esta variable de sólo lectura contiene el número de identificación único de la instancia. Cada instancia que se crea - ya sea a través de código o añadiéndolas a un room en el Editor Room - recibe un número que se utiliza internamente para identificar esta instancia y la variable id es la que se puede utilizar para referenciarla. El id también es devuelto (y puede ser almacenado en una variable) cuando se crea una instancia utilizando instance_create_layer() o instance_create_depth(), así como otras funciones de instancia.

Esta variable es ahora una referencia a una instancia, en lugar de ser simplemente un número real. Puede obtener el número de identificación de la instancia llamando a real() o int64() sobre esta variable.

NOTA El valor de esta variable no es el mismo que el identificador que se le da a la instancia en el editor room y también es diferente al instance_id que contiene todos los ids de todas las instancias actualmente activas.

Sintaxis:

id

Devuelve:

Instance ID

Ejemplo:

for (var i = 0; i < instance_count; i++;)
{
    if (instance_id[i] != id)
    {
        instance_id[i].scr += 5;
    }
}

El código anterior añade 5 a la variable "scr" para cada instancia activa en room excepto la que ejecuta el código. Lo hace recorriendo TODAS las instancias activas (utilizando el array "instance_id" para devolver el ID de cada instancia activa) y comparándolas con la variable "id" incorporada, que es el ID de la instancia original que ejecuta el código.
