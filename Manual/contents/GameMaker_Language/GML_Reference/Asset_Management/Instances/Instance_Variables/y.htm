y

El valor y de una instancia es la posición vertical en el room actual, medida en píxeles. Este valor puede ser 0, positivo o negativo, donde 0 es la parte superior de room y si se desplaza hacia abajo aumenta y, si se desplaza hacia arriba disminuye y (un valor negativo de y significa que la instancia ha salido de la parte superior de room). Incluso se puede dar a y un valor real como 12,345, pero como no se puede obtener 0,345 de un píxel, la instancia puede parecer que no se ha movido, aunque estos valores serán mantenidos y utilizados internamente por GameMaker y son perfectamente válidos.

Ajustando los valores y (y x) puedes hacer que el object salte alrededor del room a la posición que elijas, o puedes sumar y restarles cantidades menores para dar la ilusión de movimiento sin usar realmente los speed y direction funciones. Esto debe tenerse en cuenta al hacer tus juegos, ya que a menudo un tipo de movimiento se presta a un estilo particular de juego más que otro.

Sintaxis:

y;

Devuelve:

(single precision floating point value)

Ejemplo:

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

El código anterior utilizará las teclas de flecha para sumar o restar a la posición y y así crear la ilusión de movimiento.
