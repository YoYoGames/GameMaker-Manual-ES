hspeed

hspeed es una de las propiedades "incorporadas" que tienen todas las instancias y define la velocidad de movimiento horizontal (a lo largo del eje x) de la instancia en píxeles por paso. Así, un hspeed de 3 significa 3 píxeles de movimiento a la derecha (+x) a cada paso, y un hspeed de -3 significaría 3 píxeles de movimiento a la izquierda (-x) a cada paso.

Tenga en cuenta que si ajusta las variables speed y/o direction, el valor de hspeed se actualizará automáticamente para reflejar estos cambios, y del mismo modo, el cambio del valor de hspeed también afectará a los valores de speed y direction. Tenga en cuenta también que gravity, gravity_direction y friction pueden modificar el valor de esta variable cuando se utilizan en sus juegos.

Sintaxis:

hspeed;

Devuelve:

(single precision floating point value)

Ejemplo:

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

El código anterior cambiará la velocidad horizontal dependiendo de las teclas que se pulsen.
