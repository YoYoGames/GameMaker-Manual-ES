vspeed

vspeed es una de las propiedades "incorporadas" que tienen todas las instancias y define la velocidad de movimiento vertical (a lo largo del eje y) de la instancia en píxeles por paso. Así, un vspeed de 3 significa 3 píxeles de movimiento hacia abajo (+y) a cada paso, y un vspeed de -3 significaría 3 píxeles de movimiento hacia arriba (-y) a cada paso.

Tenga en cuenta que si ajusta las variables speed y/o direction, el valor de vspeed se actualizará automáticamente para reflejar estos cambios, y del mismo modo, el cambio del valor de vspeed también afectará a los valores de speed y direction. Tenga en cuenta también que gravity, gravity_direction y friction pueden modificar el valor de esta variable cuando se utilizan en sus juegos.

Sintaxis:

vspeed;

Devuelve:

(single precision floating point value)

Ejemplo:

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

El código anterior cambiará la velocidad vertical dependiendo de las teclas que se pulsen.
