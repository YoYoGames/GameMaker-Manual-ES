solid

Una instancia puede ser marcada como sólida a través de las propiedades de object en el Editor de Objetos, o cambiando el valor de esta variable incorporada. Si solid se establece en true entonces, cuando se detecta una colisión, la instancia que colisiona se devuelve automáticamente a la posición en la que se encontraba en el paso anterior a la colisión (y entonces se ejecuta cualquier código o acción en el evento de colisión). Si se establece en false, todo el posicionamiento debe ser tratado a través del evento de colisión.

Sintaxis:

solid

Devuelve:

Ejemplo:

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

El código anterior comprobará el sólido flag de la "otra" instancia en una colisión y si se establece en true moverá esa instancia a su posición anterior.
