gravedad

gravity es una de las variables "incorporadas" que tienen todas las instancias y, cuando se fija, aplicará una fuerza constante en el gravity_direction de la instancia, influyendo tanto en la instancia speed como en direction. Ten en cuenta que la gravedad es una fuerza acumulativa y acelerará el object si decides no limitar la velocidad final, y es habitual que fijes esta variable en valores decimales pequeños como 0,01. Si estableces el gravity a 0, entonces no se aplicará gravedad a la instancia (este es el valor por defecto).

Sintaxis:

gravity;

Devuelve:

(single precision floating point value)

Ejemplo:

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

El código anterior sólo aplicará la gravedad si la instancia no encuentra ninguna instancia de "obj_Ground" debajo de ella.
