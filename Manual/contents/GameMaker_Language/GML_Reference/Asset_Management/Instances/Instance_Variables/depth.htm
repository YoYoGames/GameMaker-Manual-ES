profundidad

Cuando creas un object puedes asignarle una profundidad inicial que define cómo se dibujarán las instancias de ese object en el room cuando se esté jugando y esta variable se puede utilizar para obtener y cambiar ese valor de profundidad mientras el juego se está ejecutando. Normalmente no necesitarás usar esta variable, ya que deberías configurar las instancias para que se añadan a capas discretas, que a su vez están configuradas a una profundidad específica, pero puede ser que quieras cambiar la profundidad de una instancia usando este valor, en cuyo caso se creará una "capa temporal" específicamente para la instancia a la profundidad dada. Tenga en cuenta que cuando no haya instancias en la misma profundidad, esta capa temporal se eliminará de la memoria (a diferencia de las capas normales, que permanecerán aunque no tengan nada en ellas).

¡IMPORTANTE! Cuando se modifica la variable depth y GameMaker gestiona las capas, la variable incorporada layer devolverá -1 y no el ID de la capa, ya que las capas gestionadas no pueden ser manipuladas mediante código.

En GameMaker, cuanto menor sea el valor de profundidad de una instancia, más "cerca de la cámara" se dibujará esa instancia, mientras que un valor de profundidad mayor significa que la instancia se dibujará "más lejos de la cámara", es decir: -1000 se dibuja encima de -100, que se dibuja encima de 0, y así sucesivamente.

Tenga en cuenta que las instancias de un object que tienen la misma profundidad pueden dibujarse por encima o por debajo de las demás, incluso cuando parecen estar colocadas una encima de la otra en el editor room. Si quieres garantizar que algo se dibuje por encima o por debajo de todo lo demás, siempre debes establecer la profundidad (o capa) explícitamente. Ten en cuenta también que hay un valor de profundidad mínimo (-16000) y máximo (+16000) fuera del cual las instancias no se dibujarán, aunque seguirán existiendo y procesando eventos.

NOTA No se puede establecer la profundidad de una instancia en su evento Draw (todos los demás eventos están bien).

Tenga en cuenta que la modificación de la profundidad de una instancia puede cambiar los Filtros y Efectos que se aplican en ella, ya que si se cambia la profundidad para que sea inferior a la profundidad de una capa FX, ya no se aplicará su efecto en la instancia.

Sintaxis:

depth;

Devuelve:

Real (single precision floating point value)

Ejemplo:

if y != yprevious
{
    depth = -y;
}

El código anterior comprobará si la posición y ha cambiado y, si lo ha hecho, la profundidad también se ajustará para que se corresponda con ella.
