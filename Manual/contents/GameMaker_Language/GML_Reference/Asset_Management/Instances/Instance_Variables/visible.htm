visible

Una instancia puede ser marcada como visible o no estableciendo esta variable como true (visible) o false (invisible). Esto funciona diciéndole a GameMaker que omita el evento de dibujo para esta instancia, por lo que se debe tener cuidado al usar esto ya que significa que ningún código colocado en el evento de dibujo se ejecutará mientras la instancia no sea visible. También tenga en cuenta que si el layer a la que está asignada la instancia está marcada como invisible, establecer esta variable a true no tendrá efecto hasta que la propia capa esté marcada como visible también.

Sintaxis:

visible

Devuelve:

Ejemplo:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

El código anterior comprobará el flag visible de la "otra" instancia en un evento de colisión y si se establece en true moverá esa instancia a su posición anterior.
