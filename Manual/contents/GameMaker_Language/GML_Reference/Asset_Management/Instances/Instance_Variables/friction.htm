fricción

Friction es una de las propiedades "incorporadas" que pueden tener las instancias y puede utilizarse para ralentizar la instancia en el tiempo cuando el valor de speed es distinto de cero. Funciona simplemente restando una cantidad de la velocidad a cada paso hasta que el object tiene una velocidad de 0, así que si la fricción se establece en, por ejemplo, 0,1 y la velocidad de la instancia es 1 (1 píxel por paso), se ralentizará y se detendrá después de que hayan pasado 10 pasos. Ten en cuenta también que la fricción se aplica a las velocidades positivas y negativas por igual, con el resultado neto de que el object tiene una velocidad de 0 después de un tiempo determinado.

Sintaxis:

friction;

Devuelve:

(single precision floating point value)

Ejemplo:

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

El código anterior sólo aplicará la fricción si la velocidad absoluta de la instancia es superior a 0.
