velocidad

Todas las instancias en GameMaker tienen ciertas propiedades "incorporadas" que puedes usar y establecer para gobernar su aspecto y comportamiento. speed es una de esas propiedades y define cuántos píxeles se moverá la instancia en cada paso. A diferencia de hspeed y vspeed speed no tiene una dirección asociada, ya que se rige por el valor direction de la instancia, pero puede tener un valor puede tener un valor negativo, en cuyo caso la instancia se desplazará en la dirección opuesta a la establecida por la función direction (es decir, direction - 180°). Tenga en cuenta que el ajuste de speed y/o direction, también modificará los valores de las variables hspeed y vspeed, y que gravity, gravity_direction y friction pueden modificar el valor de esta variable cuando se utilizan en sus juegos.

Sintaxis:

speed;

Devuelve:

(single precision floating point value)

Ejemplo:

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

El código anterior utilizará las teclas de flecha para establecer la velocidad y la dirección de la instancia.
