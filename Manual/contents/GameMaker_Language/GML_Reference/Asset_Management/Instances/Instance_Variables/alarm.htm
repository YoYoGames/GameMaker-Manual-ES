alarma

Esta matriz de 1 dimensión se utiliza para obtener el valor actual de las alarmas que pueda tener la instancia, o puede utilizarse para establecer dichas alarmas. Hay doce alarmas incorporadas en cada instancia de un object, y cada una tiene su propia event que se ejecutará cuando esta variable llegue a 0.

Las alarmas son contadas al inicio de cada paso, después del evento Begin Step pero antes del evento principal Step, y antes de que se renderice nada para el frame. Esto significa que si estableces una alarma a 1 en el evento Begin Step, su evento se ejecutará en ese mismo fotograma (ya que la alarma será contada a 0 después de que Begin Step haya terminado), pero si estableces una alarma a 1 en el evento Step, se ejecutará en el siguiente fotograma.

Hay que tener en cuenta que la alarma no termina cuando llega a 0 (aunque el evento se haya disparado) ya que en el siguiente paso bajará a -1, por lo que si necesitas parar una alarma por cualquier motivo deberás poner su valor en el array a -1 no a 0. Los tiempos de las alarmas se calculan en pasos de juego, siendo un valor de 30 pasos, 60 pasos, etc.

NOTA Una alarma sin acciones o código en ella no se ejecutará. Sin embargo, incluso con sólo un comentario y sin código, la alarma tendrá una cuenta atrás.

Puedes establecer la matriz de alarmas directamente (y recuperar el valor de la alarma actual directamente usando la matriz también), pero en algunas circunstancias este no es el método más apropiado, así que también puedes usar la función alarm_set() para establecer una alarma, y la función alarm_get() para obtener el valor de una alarma.

Sintaxis:

alarm[val];

Devuelve:

(-1 if the alarm isn't running)

Ejemplo:

if canshoot
{
    if keyboard_check_pressed(vk_space)
    {
        canshoot = false;
        alarm[0] = room_speed;
        instance_create_layer(x, y, "Bullets", obj_Bullet);
    }
}

El código anterior comprueba una variable y, si es verdadera, comprueba si se ha pulsado una tecla y, si es verdadera, finalmente establece la variable como falsa, establece una alarma (donde se establecerá como verdadera de nuevo) y crea una instancia.
