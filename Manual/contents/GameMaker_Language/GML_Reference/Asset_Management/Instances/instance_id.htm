instance_id

Este array de sólo lectura contiene todos los id s de cada instancia activa dentro de room. Esto significa que si ha utilizado alguna de las funciones de Desactivación de Instancias, las instancias que han sido desactivadas no estarán incluidas en este array (si ha utilizado un valor de este array previamente, ahora devolverá la palabra clave noone).

Sintaxis:

instance_id[num];

Devuelve:

Instance ID

Ejemplo:

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

El código anterior loop a través de todas las instancias dentro de la room y añadir 0,1 a su velocidad.
