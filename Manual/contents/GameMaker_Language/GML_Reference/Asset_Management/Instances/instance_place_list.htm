lista_de_lugares_de_instancia

Con esta función puede comprobar una posición para una colisión con todas las instancias de un object utilizando la máscara de colisión de la instancia que ejecuta el código para la comprobación. Cuando usted utiliza esto usted está pidiendo efectivamente a GameMaker que mueva la instancia a la nueva posición, compruebe si hay una colisión, entonces se mueve hacia atrás y le dice si se encontró una colisión o no. Esto funcionará para colisiones precisas, pero sólo si tanto la instancia como las instancias de la object que se están comprobando tienen máscaras de colisión precisas seleccionadas, de lo contrario, sólo se aplican las colisiones de caja delimitadora.

También se suministra una lista DS, por lo que los id de cualquier instancia que colisione con la instancia que llama se añadirá al final de la lista dada. Puedes borrar la lista antes de llamar a esta función para que sólo contenga los resultados de esta llamada a la función. También tiene la opción de ordenar la lista en función de la distancia desde el origen de la instancia que realiza la comprobación hasta el origen de las instancias que se encuentran en colisión.

Tenga en cuenta que la función también acepta la palabra clave especial allen cuyo caso se listarán todas las instancias que se encuentren en colisión. La función devuelve el número de instancias encontradas, o 0 si no se encuentra ninguna.

Sintaxis:

instance_place_list(x, y, obj, list, ordered);

Argumento Tipo Descripción
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Activo del objeto o ID de la instancia The object to check for instances of.
list ID de la lista DS The DS list to use to store the IDs of colliding instances.
ordered Booleano Whether the list should be ordered by distance (true) o no (false).

Devuelve:

Real (The number of instances found to be in collision)

Ejemplo:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);

if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}

ds_list_destroy(_list);

El código anterior buscará una colisión con todas las instancias encontradas en la posición de la instancia que llama. Éstas se añadirán a una lista DS, que luego se recorrerá para destruir cada una de las instancias en colisión.
