conteo_de_instancias

Con esta variable de sólo lectura se puede obtener un recuento de todas las instancias activas que están en el room. Esto incluirá la instancia que ejecuta el código, pero no incluye aquellas instancias que han sido desactivadas usando las funciones de desactivación de instancias. Tenga en cuenta que esta variable sólo le dará el número de instancias al inicio del paso, por lo que cualquier cambio en las instancias en el room realizado después de que el paso haya comenzado no se tendrá en cuenta.

Sintaxis:

instance_count;

Devuelve:

Ejemplo:

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

El código anterior creará múltiples instancias de object "obj_Star" hasta que el número total de instancias llegue a 100.
