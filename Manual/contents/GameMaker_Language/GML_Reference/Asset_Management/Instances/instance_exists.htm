instancia_existe

Esta función puede utilizarse de dos maneras, dependiendo de lo que se desee comprobar. Puede darle un object_index para comprobar, en cuyo caso esta función devolverá true si existe alguna instancia activa del object especificado en el room actual, o también puede suministrarle un id de instancia, en cuyo caso esta función devolverá true si esa instancia específica existe y está activa en el room actual.

Tenga en cuenta que esta función no tiene en cuenta las instancias que han sido desactivadas mediante las funciones de desactivación de instancias.

Sintaxis:

instance_exists(obj);

Argumento Tipo Descripción
obj ID de la instancia o activo del objeto The object or instance to check for the existence of.

Devuelve:

Boolean

Ejemplo:

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

El código anterior comprueba si existe alguna instancia de la object "obj_Enemy" y si no es así la añade a la variable "score" y cambia room.
