copia_de_instancia

Con esta función se puede "clonar" una instancia, ya que esto creará una nueva versión de la instancia ejecutando el código en su misma posición. El argumento "perf" se utiliza para indicar a esta nueva instancia que realice o no el evento de creación. Esta función devuelve el id de la nueva instancia que puede ser almacenada en una variable o utilizada para acceder a dicha instancia.

NOTA: Si decide no realizar el evento de creación, puede encontrar errores si la instancia depende de alguna variable inicializada en este evento.

Sintaxis:

instance_copy(perf);

Argumento Tipo Descripción
perf Booleano Whether to perform the new instance's Create event (true) or not (false).

Devuelve:

Instance ID

Ejemplo:

var inst = instance_number(object_index);
if (inst < 10)
{
    instance_copy(true);
}

El código anterior crea una variable local y la utiliza para almacenar el número de instancias del object que ejecuta el código en el room. Si el número es menor de 10, la instancia entonces hace una copia de sí misma.
