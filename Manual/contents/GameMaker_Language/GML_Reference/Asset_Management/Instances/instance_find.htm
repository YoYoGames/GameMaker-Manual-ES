buscar_instancias

Todas las instancias tienen un identificador único (id) que se puede utilizar para modificar y manipular mientras un juego se está ejecutando, pero no siempre se puede saber cuál es el id de una instancia específica y por lo que esta función puede ayudar como se puede utilizar para iterar a través de todos ellos para encontrar lo que usted necesita. Especifica la object de la que quieres encontrar la instancia y un número, y si hay una instancia en esa posición en la lista de instancias entonces la función devuelve el id de esa instancia, y si no devuelve la palabra clave especial noone. También puede utilizar la palabra clave all para iterar a través de todas las instancias en un room, así como un padre object para iterar a través de todas las instancias que forman parte de esa jerarquía padre / hijo, e incluso puede especificar una instancia (si tiene su id) como una comprobación para ver si realmente existe en el actual room. Tenga en cuenta que como las instancias se ordenan de forma arbitraria, no hay un orden específico para la comprobación de las instancias por parte de esta función, y cualquier instancia puede estar en cualquier posición.

El valor máximo de "n" en esta función sería

  • Para la palabra clave all: instance_count - 1
  • Para un índice object: instance_number(OBJ) - 1

Sintaxis:

instance_find(obj, n);

Argumento Tipo Descripción
obj Activo de objeto The object to find the nth instance of
n Real The number of the instance to find.

Devuelve:

Instance ID or noone

Ejemplo:

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

El código anterior utilizará un for loop para iterar a través de todas las instancias de "obj_Enemy" y almacenar su id en el array "enemy[]".
