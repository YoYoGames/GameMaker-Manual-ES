instancia_más_cercana

Esta función comprobará todas las instancias de la object dada para ver cuál está más cerca del punto x/y dado. Todas las comprobaciones se harán desde la posición x/y dada hasta el origen (la posición x/y) de las instancias del object especificado. Si no existen instancias del object, la función devolverá la palabra clave noonepero si hay instancias entonces devolverá el id de la instancia encontrada. Tenga en cuenta que si la instancia que ejecuta el código tiene el mismo índice object que el object que se está comprobando, entonces se incluirá en la comprobación (esto incluye la comprobación del padre objects si la instancia que llama es también un hijo del padre).

Sintaxis:

instance_nearest(x, y, obj);

ArgumentoTipo Descripción
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

Devuelve:

Instance ID or noone

Ejemplo:

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

El código anterior mueve la instancia actual 10000 píxeles y luego comprueba su posición anterior para encontrar la instancia más cercana del mismo tipo object. Si esa instancia es la misma, no hará nada más que volver a su posición original, pero si la instancia encontrada es diferente, dibujará una línea entre las dos.
