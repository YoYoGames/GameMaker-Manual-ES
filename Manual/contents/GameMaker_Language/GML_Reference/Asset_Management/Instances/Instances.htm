Instancias

Las inst ancias se crean a partir de Objetos y, una vez creadas, pueden ser modificadas y manipuladas a través del código GML y de las acciones de GML Visual. Las principales funciones relacionadas con las instancias que están disponibles y son las más utilizadas son:

  • instance_create_layer
  • instance_create_depth
  • instance_destroy
  • instance_exists
  • instance_change
  • instance_copy
  • instance_find
  • instance_furthest
  • instance_nearest
  • instance_number
  • instance_place
  • instance_place_list
  • instance_position
  • instance_position_list

Tenga en cuenta que hay dos variables adicionales incorporadas (son variables de ámbito global ) y una función relacionada con el uso de instancias que puede ser útil para recuperar ciertos detalles:

  • instance_id_get
  • instance_id
  • instance_count

Hay una serie de variables incorporadas disponibles dentro de cada instancia de un object que se pueden utilizar para controlar diferentes aspectos de cómo se muestra la instancia, cómo utiliza otros assets, y cómo reacciona a las colisiones, etc... que se enumeran aquí:

  • Variables de instancia

Las instancias tienen una variable de matriz que se puede utilizar para obtener o establecer los valores del evento de alarma, pero también hay dos funciones que se pueden utilizar para hacer lo mismo:

  • alarm_get
  • alarm_set

Por último, también tienes la posibilidad de desactivar instancias, lo que significa que no procesarán ninguno de sus eventos, aunque seguirán existiendo dentro del juego. Las funciones relacionadas con esto están disponibles en la siguiente página:

  • Desactivación de instancias
