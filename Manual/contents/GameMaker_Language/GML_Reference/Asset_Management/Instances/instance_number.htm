instance_number

Con esta función puede averiguar cuántas instancias activas del object especificado existen en el room. Cuando se comprueba utilizando esta función, si el object es un padre, entonces todos los hijos objects también se incluirán en el valor de retorno, y también tenga en cuenta que aquellas instancias que han sido desactivadas con las funciones de desactivación de instancias no se incluirán en esta comprobación.

Sintaxis:

instance_number(obj);

ArgumentoTipo Descripción
obj The object to total the number of instances of.

Devuelve:

Ejemplo:

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

El código anterior comprobará el número de instancias que se crean de la misma object que la instancia actual y, si hay menos de 50, creará otra en una posición aleatoria dentro de la room.
