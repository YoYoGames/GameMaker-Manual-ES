lugar_de_instancia

Con esta función puede comprobar una posición para una colisión con otra instancia o todas las instancias de un object utilizando la máscara de colisión de la instancia que ejecuta el código para la comprobación. Cuando usas esto estás pidiendo efectivamente a GameMaker que mueva la instancia a la nueva posición, compruebe si hay colisión, vuelva a moverse y te diga si se encontró una colisión o no.

Esto funcionará para las colisiones precisas, pero sólo si tanto la instancia como la object que se está comprobando tienen seleccionadas máscaras de colisión precisas; de lo contrario, sólo se aplicarán las colisiones del cuadro delimitador.

Esta función devolverá la única instancia id del object que se está colisionando, pero si no se necesita es ligeramente más rápido utilizar la función place_meeting(). Esta función también acepta las palabras clave especiales all y other y devolverá la palabra clave noone si no se produce ninguna colisión, o el valor del ID de la instancia única encontrada si se produce una colisión.

Sintaxis:

instance_place(x, y, obj);

Argumento Tipo Descripción
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Activo del objeto o ID de la instancia The object to check for instances of.

Devuelve:

Instance ID or noone

Ejemplo:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

El código anterior comprobará si hay una colisión con las instancias de "obj_Enemigo" y si la hay, reducirá la variable "hp" en la cantidad almacenada en la variable "dmg" de la instancia que colisiona y luego destruirá la instancia que colisiona.
