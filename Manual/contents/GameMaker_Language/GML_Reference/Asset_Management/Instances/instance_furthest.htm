instance_furthest

Esta función comprobará todas las instancias del object dado para ver cuál está más lejos del punto x/y dado. Todas las comprobaciones se harán desde la posición x/y dada hasta el origen (la posición x/y) de las instancias del object especificado. Si no existen instancias de object, la función devolverá la palabra clave noonepero si hay instancias entonces devolverá el id de la instancia encontrada. Tenga en cuenta que si la instancia que ejecuta el código tiene el mismo índice object que el object que se está comprobando, entonces se incluirá en la comprobación (esto incluye la comprobación del padre objects si la instancia que llama es también un hijo del padre).

Sintaxis:

instance_furthest(x, y, obj);

ArgumentoTipo Descripción
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

Devuelve:

Instance ID or noone

Ejemplo:

var inst;
inst = instance_furthest(x, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}

El código anterior encontrará la instancia más lejana que comparta el mismo índice object que la instancia que ejecuta el código y almacenará el id en una variable. A continuación, se comprueba si esta variable es la misma que el id de la instancia que llama y, si no lo es, se traza una línea entre las dos instancias.
