instancia_desactivar_objeto

Con esta función puede desactivar una sola instancia o todas las instancias de un object específico de todas las que se han activado previamente. Tenga en cuenta que si ha desactivado una instancia o object que ha sido marcada como Persistente, entonces deberá volver a activarlo con la función instance_activate_object() antes de cambiar room, de lo contrario no se transferirá y se descartará. Tenga en cuenta también que la desactivación no es instantánea, y una instancia que ha sido desactivada de esta manera no se considerará inactiva hasta el final del evento en el que se llamó a la función.

NOTA: Si desactiva una instancia al iniciar room (es decir, desde el código de creación de room, o desde un evento de creación de una instancia dentro de room) todas las instancias que se coloquen dentro de room desde el editor de room seguirán ejecutando su evento de creación antes de ser desactivadas.

ADVERTENCIA: Desactivar las instancias que tienen la física activada NO impedirá que sus accesorios interactúen dentro de la simulación de la física. Para ello, debe establecer su phy_active variable a true o false al activar/desactivar las instancias.

Sintaxis:

instance_deactivate_object(obj);

ArgumentoTipo Descripción
obj The object or instance to deactivate (the keyword all can also be used).

Devuelve:

Ejemplo:

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

El código anterior desactiva todas las instancias del object "obj_Enemy" y luego activa una región dentro del room.
