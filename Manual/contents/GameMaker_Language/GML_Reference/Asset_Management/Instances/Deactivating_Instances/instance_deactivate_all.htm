instance_deactivate_all

Esta función desactivará todas las instancias que estén en el room en el momento en que se ejecute el código. Esto puede incluir la instancia que ejecuta el código si el "notme" flag se establece en false, pero normalmente se querrá que esa instancia esté activa, en cuyo caso el "notme" flag debe establecerse en true. Tenga en cuenta que la desactivación no es instantánea, y una instancia que ha sido desactivada de esta manera no se considerará inactiva hasta el final del evento en el que se llamó a la función.

NOTA: Si desactiva una instancia al iniciar room (es decir, desde el código de creación de room, o desde un evento de creación de una instancia dentro de room) todas las instancias que se coloquen dentro de room desde el editor de room seguirán ejecutando su evento de creación antes de ser desactivadas.

ADVERTENCIA: Desactivar las instancias que tienen la física activada NO impedirá que sus accesorios interactúen dentro de la simulación de la física. Para ello, debe establecer su variable phy_active a true o false al activar/desactivar las instancias.

Sintaxis:

instance_deactivate_all(notme);

Argumento Tipo Descripción
notme Booleano Whether to keep the calling instance activated (true) or not (false).
collision_space Collision Space  The collision space where instances will be deactivated

Devuelve:

N/A

Ejemplo:

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

El código anterior desactiva todas las instancias excepto la que está ejecutando el código y luego activa una región dentro de la room.
