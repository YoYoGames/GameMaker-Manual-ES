instancia_activar_capa

Con esta función puede activar una capa que ha sido desactivada previamente. Debe proporcionar el ID de la capa, que puede ser el nombre de la capa tal y como está escrito en el editor de código (como string) o el valor real del ID de la capa devuelto por la función layer_create() y todas las instancias desactivadas en esa capa se activarán de nuevo. Tenga en cuenta que si ha desactivado una capa que tiene instancias persistentes, tendrá que reactivar la capa de nuevo con esta función antes de cambiar room, de lo contrario las instancias persistentes no se trasladarán y serán descartadas. Ten en cuenta también que la activación no es instantánea, y una instancia que haya sido activada de este modo no se considerará activa hasta el final del evento en el que se llamó a la función.

Sintaxis:

instance_activate_layer(layer_id);

Argumento Tipo Descripción
layer_id ID de capa o cadena The layer name string (or ID value) to be used

Devuelve:

N/A

Ejemplo:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

El código anterior activa todas las instancias dentro de la room y luego desactiva las que están fuera de los límites de la vista de la cámara actual, excepto las instancias de la capa "Player_Layer" que se vuelven a activar al final.
