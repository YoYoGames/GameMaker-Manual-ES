instancia_desactivar_capa

Con esta función puede desactivar todas las instancias asignadas a una capa específica. Debe proporcionar el ID de la capa, que puede ser el nombre de la capa tal y como está escrito en el editor de código (como string) o el valor real del ID de la capa devuelto por la función layer_create()y tenga en cuenta que sólo puede desactivar las capas de instancia con esta función. Tenga en cuenta que si ha desactivado una capa que tiene instancias de objects marcadas como Persistentes, entonces tendrá que reactivar la capa de nuevo con la función instance_activate_layer() antes de cambiar room, de lo contrario las instancias persistentes de la capa no serán transferidas y serán descartadas. Ten en cuenta también que la desactivación no es instantánea, y una instancia que ha sido desactivada de esta manera no se considerará inactiva hasta el final del evento en el que se llamó a la función.

NOTA: Si desactiva una instancia al iniciar room (es decir, desde el código de creación de room, o desde un evento de creación de una instancia dentro de room) todas las instancias que se coloquen dentro de room desde el editor de room seguirán ejecutando su evento de creación antes de ser desactivadas.

ADVERTENCIA: Desactivar las instancias que tienen la física activada NO impedirá que sus accesorios interactúen dentro de la simulación de la física. Para ello, debe establecer su variable phy_active a true o false al activar/desactivar las instancias.

Sintaxis:

instance_deactivate_layer(obj);

Argumento Tipo Descripción
layer ID de capa o cadena The layer name string (or ID value) to be used

Devuelve:

N/A

Ejemplo:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

El código anterior desactiva todas las instancias asignadas a la capa "Enemy_Layer" y luego activa una región dentro de la room.
