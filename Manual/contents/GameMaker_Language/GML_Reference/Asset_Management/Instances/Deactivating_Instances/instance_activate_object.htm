instance_activate_object

Con esta función puede activar una sola instancia o todas las instancias de un object específico de entre todas las que han sido desactivadas previamente. Tenga en cuenta que si ha desactivado una instancia o object que ha sido marcada como Persistente, entonces tendrá que reactivarla de nuevo con esta función antes de cambiar room, de lo contrario no se trasladará y se descartará en su lugar. Tenga en cuenta también que la activación no es instantánea, y una instancia que haya sido activada de esta manera se considerará activa hasta el final del evento en el que se llamó a la función.

Sintaxis:

instance_activate_object(obj);

ArgumentoTipo Descripción
obj The object or instance to activate (the keyword all can also be used).

Devuelve:

Ejemplo:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

El código anterior activa todas las instancias dentro de la room y luego desactiva las que están fuera de los límites de la vista de la cámara actual, excepto las object "obj_Lights" que se vuelven a activar al final.
