alarm_get

Esta función se puede utilizar para obtener el valor actual de la alarma dada. Se suministra el número de alarma de 0 a 11 y esta función devolverá el valor en el que se encuentra la alarma actualmente. Este es un método alternativo a la obtención del valor de la matriz de alarmas directamente.

Sintaxis:

alarm_get(index);

Argumento Tipo Descripción
index Real The alarm index to get, from 0 to 11.

Devuelve:

Real (integer)

Ejemplo:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

El código anterior comprueba todas las alarmas en la instancia de llamada y si son mayores que 0 las pone a -1, impidiendo que sigan contando.
