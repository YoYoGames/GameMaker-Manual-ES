alarm_set

Esta función permite ajustar una alarma. Se suministra el número de alarma de 0 a 11, y luego el valor para establecer la alarma. El valor debe ser un valor entero, y puede establecerlo en -1 para detener la alarma (los valores no enteros serán redondeados al entero más cercano). Este es un método alternativo a la configuración de la matriz de alarmas directamente.

Sintaxis:

alarm_set(index, value);

Argumento Tipo Descripción
index Real The alarm index to set, from 0 to 11.
value Real The value (an integer) to set the alarm to.

Devuelve:

N/A

Ejemplo:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

El código anterior comprueba todas las alarmas en la instancia de llamada y si son mayores que 0 las pone a -1, impidiendo que sigan contando.
