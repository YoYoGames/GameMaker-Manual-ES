asset_remove_tags

Con esta función puede eliminar una o más etiquetas strings a cualquier asset del navegador asset. Se suministra el nombre asset (como string) o su índice asset, así como una sola etiqueta string o un array donde cada elemento es una sola etiqueta string. Si suministra un valor de índice asset, tendrá que suministrar el argumento opcional de tipo asset (una constante), ya que assets de diferentes tipos puede tener el mismo índice, aunque no puedan tener el mismo nombre. Los tipos disponibles de asset se enumeran en la siguiente tabla:

Constante Descripción
asset_object El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
asset_room El nombre se refiere a un room.
asset_tiles El nombre dado se refiere a un conjunto de baldosas.
asset_path El nombre dado se refiere a una ruta.
asset_script El nombre se refiere a un script.
asset_font El nombre dado se refiere a una fuente.
asset_timeline El nombre de pila hace referencia a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
asset_sequence El nombre dado se refiere a una Secuencia.

Si la función consigue eliminar la(s) etiqueta(s) devolverá true de lo contrario devolverá false.

Sintaxis:

asset_remove_tags(name_or_index, tags, [asset_type]);

Argumento Tipo Descripción
name_or_index Cadena o Activo The name of the asset (a string) or its index value (an integer).
tags Cadena o matriz de cadenas A single asset tag string or an array with various asset tags.
[asset_type] Tipo de activo Constante OPTIONAL! The type of asset to remove the tags from, only used when supplying an index for the first argument.

Devuelve:

Boolean

Ejemplo:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

El código anterior creará una matriz de etiquetas y luego las eliminará del object dado.
