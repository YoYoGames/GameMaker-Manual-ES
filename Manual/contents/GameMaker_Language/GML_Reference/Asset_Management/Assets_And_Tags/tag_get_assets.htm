tag_get_assets

Con esta función puede recuperar los nombres de todos los assets a los que se ha asignado la etiqueta o etiquetas dadas. Puede proporcionar una sola etiqueta string o una matriz, donde cada elemento de la matriz es una etiqueta string. La función devolverá una matriz en la que cada entrada es el nombre (como string) del asset con la etiqueta dada. Si necesita el índice único de asset, puede utilizar la función asset_get_index() junto con el nombre devuelto. Si no hay assets con la(s) etiqueta(s) dada(s), o si hay un error (por ejemplo: las etiquetas dadas no existen), entonces se devolverá un array vacío.

Sintaxis:

tag_get_assets(tags);

Argumento Tipo Descripción
tags Cadena o matriz de cadenas A single asset tag string or an array with various asset tags.

Devuelve:

Array

Ejemplo:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

El código anterior crea una lista y luego recupera los nombres de todos los assets con la etiqueta "background". Se loops a través de la matriz de nombres devueltos, comprobando si alguno de ellos es sprite assets, y si lo es entonces el valor del índice único para el asset se añade a la lista para su uso futuro.
