asset_get_tags

Con esta función puede recuperar todas las etiquetas asignadas a un asset desde el navegador Asset. Se proporciona el nombre asset (como string) o su índice asset, y la función devolverá una matriz de etiquetas para ese asset. Si no se encuentra ninguna etiqueta o hay un error (es decir, el nombre string dado no existe), la matriz devuelta estará vacía. Si proporciona un valor de índice asset, entonces tendrá que proporcionar el argumento opcional de tipo asset (una constante), ya que assets de diferentes tipos puede tener el mismo índice, aunque no pueden tener el mismo nombre. Los tipos disponibles de asset se enumeran en la siguiente tabla:

Constante Descripción
asset_object El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
asset_room El nombre se refiere a un room.
asset_tiles El nombre dado se refiere a un conjunto de baldosas.
asset_path El nombre dado se refiere a una ruta.
asset_script El nombre se refiere a un script.
asset_font El nombre dado se refiere a una fuente.
asset_timeline El nombre de pila hace referencia a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
asset_sequence El nombre dado se refiere a una Secuencia.

Sintaxis:

asset_get_tags(name_or_index, [asset_type]);

Argumento Tipo Descripción
name_or_index Cadena o Activo The name of the asset (a string) or its index.
[asset_type] Tipo de activo Constante OPTIONAL! The type of asset to get the tags for, only used when supplying an index for the first argument.

Devuelve:

Array

Ejemplo:

my_tags = asset_get_tags(object_get_name(object_index));

El código anterior recuperará todas las etiquetas asignadas a la object desde la que se ha creado la instancia que ejecuta el código.
