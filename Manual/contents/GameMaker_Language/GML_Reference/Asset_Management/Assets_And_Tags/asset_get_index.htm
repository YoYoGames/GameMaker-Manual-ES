asset_get_index

Puede utilizar esta función para obtener el índice de identificación único de un juego asset a partir de su nombre. Si no se encuentra el asset, la función devolverá un valor de -1, de lo contrario devolverá el id de índice único para el asset que se está comprobando. Este identificador se puede utilizar en otras funciones como cualquier otro valor de índice, como el sprite_index o el path_indexpor ejemplo. Por favor, tenga en cuenta que aunque esta función puede usarse para referenciar assets desde strings (ver ejemplo más abajo) siempre debe asegurarse de que el asset existe antes de usarlo, de lo contrario puede obtener errores que bloquearán su juego.

Sintaxis:

asset_get_index(name);

ArgumentoTipo Descripción
name The name of the game asset to get the index of (a string).

Devuelve:

Asset (any asset type)

Ejemplo:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

El código anterior obtendrá un índice object de un string, y si ese índice existe, creará una instancia del object en el juego.
