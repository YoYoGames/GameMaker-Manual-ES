tag_get_asset_ids

Con esta función puede obtener todos los assets de un tipo determinado que tengan asignadas las etiquetas dadas. Se suministra una sola etiqueta (como string) o una matriz, donde cada elemento de la matriz es una etiqueta (como string), así como el tipo de asset a comprobar. el tipo de asset debe ser una de las siguientes constantes:

Constante Descripción
asset_object El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
asset_room El nombre se refiere a un room.
asset_tiles El nombre dado se refiere a un conjunto de baldosas.
asset_path El nombre dado se refiere a una ruta.
asset_script El nombre se refiere a un script.
asset_font El nombre dado se refiere a una fuente.
asset_timeline El nombre de pila hace referencia a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
asset_sequence El nombre dado se refiere a una Secuencia.

La función devolverá una matriz, donde cada elemento de la matriz será un único valor de índice de asset. Si no hay ningún assets del tipo que tenga la(s) etiqueta(s) dada(s), se devolverá un array vacío.

Sintaxis:

tag_get_asset_ids(tags, asset_type);

Argumento Tipo Descripción
tags Cadena o matriz de cadenas A single asset tag string or an array with various asset tags.
asset_type Tipo de activo Constante An asset type constant (listed above)

Devuelve:

Array

Ejemplo:

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

El código anterior utiliza la etiqueta "enemigo" para encontrar todos los path assets con esa etiqueta, antes de elegir uno al azar y asignarlo a la instancia que ejecuta el código.
