asset_get_type

Con esta función se puede obtener el tipo de asset al que se hace referencia a partir de su nombre (un string). Se devolverá una de las constantes que se indican a continuación.

Sintaxis:

asset_get_type(name);

Argumento Tipo Descripción
name Cadena The name of the game asset to get the type of (a string).

Devuelve:

Asset Type Constant

Constante Descripción
asset_object El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
asset_room El nombre se refiere a un room.
asset_tiles El nombre dado se refiere a un conjunto de baldosas.
asset_path El nombre dado se refiere a una ruta.
asset_script El nombre se refiere a un script.
asset_font El nombre dado se refiere a una fuente.
asset_timeline El nombre de pila hace referencia a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
asset_sequence El nombre dado se refiere a una Secuencia.
asset_unknown El nombre se refiere a un asset que o bien no existe, o bien no es uno de los enumerados anteriormente.

Ejemplo:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

El código anterior comprueba un nombre asset creado dinámicamente para ver si el asset es del tipo correcto. Si no lo es, se mostrará un mensaje de depuración, de lo contrario el nombre asset se utiliza para asignar el asset a la instancia.
