asset_clear_tags

Con esta función puede borrar todas las etiquetas presentes en cualquier asset desde el navegador asset. Se suministra el nombre de asset (como string) o su índice asset, y si se suministra un valor de índice asset, será necesario suministrar el argumento opcional de tipo asset (una constante), ya que assets de diferentes tipos pueden tener el mismo índice, aunque no puedan tener el mismo nombre. Los tipos disponibles de asset se enumeran en la siguiente tabla:

Constante Descripción
asset_object El nombre dado se refiere a un objeto.
asset_sprite El nombre dado se refiere a un sprite.
asset_sound El nombre de pila hace referencia a un sonido.
asset_room El nombre se refiere a un room.
asset_tiles El nombre dado se refiere a un conjunto de baldosas.
asset_path El nombre dado se refiere a una ruta.
asset_script El nombre se refiere a un script.
asset_font El nombre dado se refiere a una fuente.
asset_timeline El nombre de pila hace referencia a una línea de tiempo.
asset_shader El nombre dado se refiere a un sombreador.
asset_animationcurve El nombre dado se refiere a una Curva de Animación.
asset_sequence El nombre dado se refiere a una Secuencia.

Si la función consigue borrar las etiquetas devolverá true de lo contrario devolverá false.

Sintaxis:

asset_clear_tags(name,[asset_type]);

Argumento Tipo Descripción
name_or_index Cadena o Activo The name of the asset (a string) or its index value (an integer).
[asset_type] Tipo de activo Constante OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

Devuelve:

Boolean

Ejemplo:

asset_clear_tags(obj_Enemy_Parent, asset_object);

El código anterior borrará todas las etiquetas de la dirección object asset.
