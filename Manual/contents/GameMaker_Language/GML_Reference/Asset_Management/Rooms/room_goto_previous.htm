room_goto_previous

Con esta función puedes hacer que tu juego vaya a la anterior, tal y como aparece en el Gestor de Salas en el momento en que se compiló el juego. Si este room no existe, se lanzará un error y el juego será forzado a cerrarse. Tenga en cuenta que el room no cambiará hasta el final del evento en el que la función fue llamada, por lo que cualquier código después de que esto ha sido llamado todavía se ejecutará si en el mismo evento. Esta función también activará el evento Room End.

Sintaxis:

room_goto_previous();

Devuelve:

Ejemplo:

if room_exists(room_previous(room))
{
    room_goto_previous();
}

El código anterior comprobará si hay otro room antes del actual y si es así irá a ese room.
