layer_get_visible

Con esta función puede comprobar si una capa es visible o no. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devolverá true si es visible, y false en caso contrario.

Sintaxis:

layer_get_visible(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)

Devuelve:

Boolean

Ejemplo:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

El código anterior obtiene el valor del ID de la capa llamada "Instances" en el editor room, y luego utiliza el ID para comprobar si la capa es visible o no, alternando la visibilidad de la capa en función del valor devuelto.
