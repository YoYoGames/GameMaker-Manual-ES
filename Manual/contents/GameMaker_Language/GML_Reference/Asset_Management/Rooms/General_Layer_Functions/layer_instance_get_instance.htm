layer_instance_get_instance

Esta función puede utilizarse para obtener el ID de instancia único del elemento de instancia dado. Se da el ID del elemento de instancia (ver el ejemplo de código más abajo para saber cómo obtenerlo), y la función devolverá un valor real que representa el ID de instancia único para el elemento. Si el elemento no es una instancia, la función devolverá -1.

Sintaxis:

layer_instance_get_instance(element_id)

Argumento Tipo Descripción
element_id ID del elemento de la instancia The unique ID value of the instance element to get the ID of

Devuelve:

Instance ID

Ejemplo:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
     if (layer_get_element_type(elements[i]) == layerelementtype_instance)
     {
         var layerelement = elements[i];
         var inst = layer_instance_get_instance(layerelement);
         inst.x = inst.x + 10;
     }
}

El código anterior comprobará todos los elementos de instancia en una capa, luego obtendrá su valor de ID único y lo utilizará para moverlos 10px a la derecha.
