layer_add_instance

Esta función se puede utilizar para mover una instancia dada de la capa en la que se encuentra actualmente a otra capa. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y el ID de la instancia a mover entre capas. Por ejemplo, digamos que tu jugador está en una capa que está a menor profundidad que otra capa y quieres que aparezca detrás de las capas entre las dos. Puede utilizar esta función para switch el reproductor de la capa de primer plano a la capa de fondo y hacer que aparezca detrás de las otras capas que se están dibujando.

NOTA Esta función no puede utilizarse para añadir una nueva instancia a una capa. Debe haber creado la instancia previamente y haber almacenado su ID en una variable.

NOTA: Esta función sólo es válida dentro del ámbito de la actual room y no puede utilizarse cuando el ámbito ha sido modificado mediante la función layer_set_target_room().

Sintaxis:

layer_add_instance(layer_id, instance_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the instance layer to target (or the layer name as a string)
instance_id ID de la instancia The unique instance ID value of the instance to move over to the target layer

Devuelve:

N/A

Ejemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

El código anterior obtendrá primero el índice de la instancia más cercana a la posición x/y dada y lo almacenará en una variable local. A continuación, obtiene el ID de la capa de instancia única para la capa denominada "Frente de instancias", y mueve la instancia encontrada a esa capa.
