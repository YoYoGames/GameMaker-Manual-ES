layer_set_target_room

Cuando usted llama a esta función le está diciendo a GameMaker que todas las funciones de capas posteriores deben ser aplicadas a la habitación dada. De este modo, puede cambiar o generar procedimentalmente capas y contenidos de capas en un room que no sea el actual room. Cuando haya terminado de añadir capas o elementos de capa a un room, llame a la función layer_reset_target_room() para restablecer el objetivo room (o llame a esta función de nuevo con un argumento room de -1). Tenga en cuenta que esta función sólo puede utilizarse en salas distintas a la actual, y está diseñada para que pueda añadir/eliminar capas y elementos de capa a rooms que no sea el room que se está ejecutando actualmente.

ADVERTENCIA Mientras se dirige a otro room, puede utilizar todas las funciones regulares de la capa, excepto que no puede crear instancias utilizando instance_create_layer() o instance_create_depth()ni la función de capa layer_add_instance() estará disponible.

Sintaxis:

layer_set_target_room(room)

ArgumentoTipo Descripción
room The room to target for all further layer functions

Devuelve:

Ejemplo:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    var _x = irandom(1000);
    var _y = irandom(1000);
    layer_sprite_create(l, _x, _y, spr_Trees);
}
layer_reset_target_room();

El código anterior establece el objetivo room al room "rm_Game" y luego obtiene el ID de la capa llamada "SpriteAssets" en ese room. Este ID de capa se utiliza entonces para añadir 50 sprite assets al azar a la capa, antes de que el objetivo de la capa se restablezca al actual room.
