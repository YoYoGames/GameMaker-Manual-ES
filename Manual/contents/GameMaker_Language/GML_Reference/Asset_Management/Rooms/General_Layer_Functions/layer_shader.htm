sombreador_de_capas

Con esta función se puede asignar un recurso shader a cualquier capa y la capa se renderizará utilizando ese shader. Se suministra el valor de ID único de la capa o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) , junto con el ID del shader a utilizar. El shader debe haber sido creado previamente en el navegador Asset y el índice shader (el nombre del recurso shader ) se pasa a esta función. Si la capa asignada tiene instancias añadidas, entonces el shader se aplicará a todos los eventos de dibujo que la instancia utilice - por ejemplo si la instancia tiene un evento Draw GUI Begin, entonces el shader se aplicará automáticamente a él. El shader también afectará a cualquier otro elemento gráfico dibujado en esa capa, como los mapas sprite assets o tile.

Tenga en cuenta que la función no está pensada para ser llamada en cualquier evento de dibujo o eventos de paso, sino que sólo necesita ser llamada al inicio de la room en el código de creación de la sala o en el evento de creación / evento de inicio de la sala de una instancia.

Sintaxis:

layer_shader(layer_id, shader)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
shader Activo de sombreado The shader index to assign to the layer

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

El código anterior asignará el recurso shader "shd_Sepia" a la capa dada para todo el dibujo.
