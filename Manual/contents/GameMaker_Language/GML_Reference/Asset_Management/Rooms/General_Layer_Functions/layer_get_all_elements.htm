layer_get_all_elements

Puede utilizar esta función para obtener los ID de los elementos de la capa dada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devolverá una matriz de IDs, donde cada entrada en la matriz es un ID único para un elemento en esa capa. Por ejemplo, si la capa es una capa Asset, la matriz se llenará con los valores de ID de cada sprite asset que se asigne a la capa. Tenga en cuenta que el uso de código para trabajar con capas significa que puede asignar diferentes tipos de elementos a la misma capa - por lo que puede tener sprite assets junto con instancias, por ejemplo - en cuyo caso puede utilizar la función layer_get_element_type() para obtener el tipo de elemento al que se refiere el ID.

Sintaxis:

layer_get_all_elements(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de la capa The unique ID value of the layer to get the elements from (or the layer name as a string)

Devuelve:

Array of Layer Element IDs

Ejemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++)
{
    if (layer_get_element_type(a[i]) == layerelementtype_sprite)
    {
        layer_sprite_destroy(a[i]);
    }
}

El código anterior obtiene los IDs de todos los elementos de instancia asignados a la capa de la instancia que ejecuta el código. A continuación, el código comprueba si alguno de los elementos devueltos es sprite assets y, si lo es, se destruye.
