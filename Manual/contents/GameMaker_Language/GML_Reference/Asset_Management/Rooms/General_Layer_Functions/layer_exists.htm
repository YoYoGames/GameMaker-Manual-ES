capa_existe

Esta función puede utilizarse para comprobar si la capa dada existe. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devolverá un valor booleano de true si existe o false si no existe.

NOTA: Esta función trabaja dentro del ámbito del objetivo actual room - por defecto el room en el que se llama a la función - que puede establecerse utilizando la función layer_set_target_room().

Sintaxis:

layer_exists(layer_name)

Argumento Tipo Descripción
layer_name Cadena o ID de capa The name of the layer (a string or ID value)

Devuelve:

Boolean

Ejemplo:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

El código anterior comprobará si la capa almacenada en la variable global existe realmente, y si no existe entonces se crea.
