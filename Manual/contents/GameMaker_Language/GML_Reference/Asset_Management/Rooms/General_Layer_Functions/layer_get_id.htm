layer_get_id

Esta función se puede utilizar para obtener el valor de ID único de una capa determinada. En el IDE, todas las capas tienen un nombre y un tipo, y para poder editarlas o cambiarlas a través del código hay que dar el valor del ID de la capa. Esta función se utiliza para recuperar este ID utilizando el nombre (a string) de la capa (como está escrito en el IDE). Si crea una nueva capa a través de código utilizando la función layer_create() entonces esa función devolverá el valor del ID único en su lugar (las capas creadas dinámicamente no obtienen nombres). Tenga en cuenta que si da el nombre de una capa que no existe en el actual room, entonces obtendrá un error y el proyecto se bloqueará.

Sintaxis:

layer_get_id(layer_name)

ArgumentoTipo Descripción
layer_name The name of the layer (a string)

Devuelve:

Layer ID or -1 (if the layer specified doesn't exist)

Ejemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

El código anterior obtendrá primero el índice de la instancia más cercana a la posición x/y dada y lo almacenará en una variable local. A continuación, obtiene el ID de la capa de instancia única para la capa llamada "Frente de Instancias", y mueve la instancia encontrada a esa capa.
