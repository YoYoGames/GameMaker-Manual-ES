layer_get_vspeed

Puede utilizar esta función para recuperar la velocidad vertical (en píxeles por fotograma del juego) de la capa dentro del ámbito actual room. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devuelve un número real para la velocidad vertical, donde un valor positivo es hacia abajo y un valor negativo hacia arriba. El valor por defecto es 0 (a menos que se establezca en el editor room ).

Sintaxis:

layer_get_vspeed(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to get the vertical speed from

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

El código anterior comprueba las velocidades horizontales y verticales de la capa dada y si no están ambas a 0 entonces las pone a 0.
