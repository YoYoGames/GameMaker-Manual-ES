layer_destroy_instances

Esta función puede utilizarse para destruir todas las instancias asignadas a la capa dada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento), y entonces todas las instancias que están en la capa serán eliminadas del juego, activando sus eventos Destroy y Clean Up.

Sintaxis:

layer_destroy_instances(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer (or the layer name as a string)

Devuelve:

N/A

Ejemplo:

if global.game_over
{
    layer_destroy_instances(layer);
}

El código anterior comprobará una variable global y si es true entonces todas las instancias que están en la capa de la instancia que llama serán destruidas (incluyendo la instancia que llama).
