layer_get_shader

Esta función puede utilizarse para comprobar si la capa dada tiene un shader asignado. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento), y la función devolverá el índice shader del shader asignado, o -1 si no hay ningún shader asignado.

Sintaxis:

layer_get_shader(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)

Devuelve:

Shader Asset or -1 (if no shader is assigned)

Ejemplo:

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

El código anterior comprobará si la capa que la instancia que ejecuta el código tiene un shader asignado y si no lo tiene se le asigna uno.
