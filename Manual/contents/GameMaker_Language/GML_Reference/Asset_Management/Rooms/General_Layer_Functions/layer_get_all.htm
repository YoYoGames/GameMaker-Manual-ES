layer_get_all

Esta función devolverá una matriz poblada con los valores de identificación únicos de cada capa en el room.

Sintaxis:

layer_get_all()

Devuelve:

Array (populated with Layer IDs)

Ejemplo:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

El código anterior recupera todas las capas en un room y añade sus valores de ID a un array. Esta matriz se analiza a continuación para destruir o las capas de room.
