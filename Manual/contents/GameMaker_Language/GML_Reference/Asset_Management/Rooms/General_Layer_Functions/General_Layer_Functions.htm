Funciones generales de la capa

El editor de salas de GameMaker le permite añadir capas de diferentes assets a su room. Estas pueden ser mapas tile (usando tile sets), fondos, sprites o sequences, e instancias, pero tenga en cuenta que, a diferencia del editor room, no está limitado a un solo tipo de asset por capa, y puede añadir múltiples tipos de asset a la misma capa - por ejemplo, puede tener un fondo y un mapa tile todo en la misma capa. Tenga en cuenta que cada asset que se añade a una capa se considera un elemento de esa capa.

En esta sección se describen las funciones generales utilizadas al trabajar con las capas de room, así como algunas funciones específicas para trabajar con capa de instancia elementos:

También tenemos un par de funciones adicionales relacionadas con la orientación de las capas en un room diferente. Puede ser que quieras cambiar algo en un room que no sea el actual room, o tal vez quieras generar rooms procedimentalmente, o cualquier número de cosas. Para permitir esto utilizamos las siguientes funciones:

  • layer_set_target_room
  • layer_get_target_room
  • layer_reset_target_room

Lo que se hace aquí es establecer el objetivo room usando la función apropiada y luego llamar a las funciones de capa son normales. Estas funciones de capa se aplicarán ahora sólo al objetivo room y no al actual. Cuando hayas terminado, llamarás a la función reset para devolver el ámbito del código al actual room de nuevo.

ADVERTENCIA Mientras se dirige a otro room, puede utilizar todas las funciones regulares de la capa, excepto que no puede crear instancias utilizando instance_create_layer() o instance_create_depth()ni la función de capa layer_add_instance() estará disponible.

Por último, y esto es exclusivamente para su uso con instancias en capas (aunque la capa en sí puede tener otros elementos, no se verán afectados por las siguientes funciones), tienes un par de funciones que se pueden utilizar para desactivar o reactivar todas las instancias:

  • instance_deactivate_layer
  • instance_activate_layer
