destruir_capas

Esta función destruirá la capa dada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y esto la eliminará del room actual. Si la capa se ha diseñado en el editor room, la próxima vez que salga de room y vuelva, la capa se volverá a crear con el contenido original; sin embargo, si room es persistente, la capa se eliminará a menos que se vuelva a desactivar la persistencia de room. Cuando se destruye una capa de esta manera, todo su contenido será eliminado también, por lo que cualquier ID de referencia para fondos o mapas tile, etc... dejará de ser válido y cualquier instancia asignada a la capa será destruida (realizando su Evento de Destrucción al mismo tiempo, si tienen uno, así como el Evento de Limpieza).

Sintaxis:

layer_destroy(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to destroy (or the layer name as a string)

Devuelve:

N/A

Ejemplo:

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

El código anterior comprobará si existe alguna instancia de object "obj_Bullet_Parent", y si no existe destruirá la capa con el ID almacenado en la variable global.
