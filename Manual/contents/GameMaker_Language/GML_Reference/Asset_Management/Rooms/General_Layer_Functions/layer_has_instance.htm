layer_has_instance

Esta función se puede utilizar para comprobar si una instancia dada está actualmente asignada a la capa dada. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y el ID de la instancia a comprobar. También puede dar un object_index (es decir, el nombre del object en el navegador Asset ) y la función comprobará si hay alguna instancia de ese object en la capa dada. La función devolverá true si la instancia está en la capa y false si no lo está.

Sintaxis:

layer_has_instance(layer_id, instance_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
instance_id ID de la instancia The unique instance ID or the object index of the instance to check for

Devuelve:

Boolean

Ejemplo:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

El código anterior comprobará si la capa dada contiene alguna instancia de object "obj_Bullet_Parent" y si no es así destruirá la capa.
