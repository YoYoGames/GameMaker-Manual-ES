layer_get_name

Puede utilizar esta función para obtener el nombre de la capa dada. Se suministra el valor único del ID de la capa y si la capa es una de las capas con nombre creadas en el editor room, la función devolverá un string con el nombre de la capa. Si la capa no es una de las creadas en el editor room (es decir, fue creada con layer_create()) entonces se devolverá un string vacío.

Sintaxis:

layer_get_name(layer_id)

Argumento Tipo Descripción
layer_id ID de la capa The unique ID value of the layer to get the name of

Devuelve:

String

Ejemplo:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if (layer_get_name(_arr_layers[i]) != "")
    {
        array_push(_arr_layers_named, _arr_layers[i]);
    }
}

El código anterior obtiene los IDs de todas las capas en room y luego loops a través de ellos comprobando si alguno es una capa con nombre. Si lo son, su ID se añade a una lista.
