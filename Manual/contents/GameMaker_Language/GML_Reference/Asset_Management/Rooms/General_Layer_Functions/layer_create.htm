crear_capa

Esta función se utiliza para crear una nueva capa dentro de la actual room. Usted proporciona la profundidad que debe ocupar la nueva capa y la función devuelve el valor único del ID de la capa que puede utilizarse para acceder a la capa en otras funciones. También puede suministrar un argumento opcional "nombre", que le permitirá dar a la capa un nombre específico (como string), y en lugar de utilizar el ID de la capa para acceder a ella, puede utilizar este nombre string en su lugar, aunque debe tenerse en cuenta que utilizar el nombre strings tendrá un mayor impacto en el rendimiento que utilizar sólo el valor del ID. Si no se especifica el nombre de la capa, se asignará a la capa creada un nombre con el formato "_layer_XXX", donde "XXX" es un valor hexadecimal utilizado para dar a la capa un nombre único.

Cuando se da un valor de profundidad, éste será mayor cuanto más "lejos" de la cámara esté la capa y menor cuanto más "cerca" de la cámara esté la capa, así que si tienes tres capas con profundidades -100, 0, 100, las capas se dibujarán en el orden 100, 0, -100, de modo que la capa "superior" (es decir, la más cercana a la vista de la cámara y por tanto dibujada sobre todo lo demás) será la capa con la profundidad -100.

Vale la pena señalar que, si bien se puede asignar más de una capa a una sola profundidad, no hay manera de garantizar que las capas en la misma profundidad se dibujen en un orden determinado y, por lo tanto, es mejor asignar sólo una capa a cualquier profundidad. Tenga en cuenta también que las profundidades mínima y máxima de las capas son de -16000 a 16000, y todo lo que esté fuera de esas profundidades no se renderizará. Si necesita una profundidad fuera de ese rango, tendrá que utilizar la función layer_force_draw_depth().

Sintaxis:

layer_create(depth, [name])

Argumento Tipo Descripción
depth Real The depth that the layer has to be created at.
name Cadena OPTIONAL El nombre que se le dará a la capa (un string).

Devuelve:

Layer ID

Ejemplo:

global.Bullet_Layer = layer_create(-100);

El código anterior creará una nueva capa con un valor de profundidad de -100 y asignará su valor de ID a una variable global.
