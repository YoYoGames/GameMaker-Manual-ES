capa_obtener_y

Puede utilizar esta función para recuperar la posición y de la capa dentro del ámbito actual room. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devuelve un número real para la posición x de la capa, relativa a la posición (0,0) de room. El valor predeterminado es 0.

Sintaxis:

layer_get_y(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to get the y position of

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

El código anterior comprueba la posición de la capa dada y si no está establecida en (0, 0) entonces se establece en esa posición.
