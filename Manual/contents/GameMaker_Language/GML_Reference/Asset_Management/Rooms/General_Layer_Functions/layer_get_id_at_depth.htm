layer_get_id_at_depth

Puede utilizar esta función para obtener los IDs de todas las capas asignadas a una profundidad específica. Se da la profundidad a comprobar y la función devolverá un array con 1 o más entradas dependiendo de si hay o no capas en la profundidad dada. Si no hay capas en la profundidad dada, el array tendrá una única entrada en la posición [0] con un valor de -1, pero, si hay capas en la profundidad, entonces se hará una entrada en el array para cada capa encontrada - el valor de la entrada será el valor de ID único para una capa.

Sintaxis:

layer_get_id_at_depth(depth)

Argumento Tipo Descripción
depth Real The depth to check and retrieve the layer IDs from.

Devuelve:

Array

Ejemplo:

var a = layer_get_id_at_depth(0);
if (a[0] != -1)
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

El código anterior recupera datos sobre las capas con una profundidad de 0. Se comprueba si existe alguna capa a esa profundidad, y si la hay se analiza el array devuelto y se destruye cada una de las capas encontradas.
