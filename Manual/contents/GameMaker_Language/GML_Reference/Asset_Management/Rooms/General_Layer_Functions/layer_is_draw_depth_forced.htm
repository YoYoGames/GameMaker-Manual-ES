layer_is_draw_depth_forced

Esta función se puede utilizar para comprobar y ver si la profundidad Z forzada está activada para el renderizado de las capas en el proyecto. Consulte layer_force_draw_depth() para más información.

Sintaxis:

layer_is_draw_depth_forced()

Devuelve:

Ejemplo:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

El código anterior comprueba si la profundidad Z de la capa está forzada o no y si no lo está, establece la profundidad Z a 0 y la habilita.
