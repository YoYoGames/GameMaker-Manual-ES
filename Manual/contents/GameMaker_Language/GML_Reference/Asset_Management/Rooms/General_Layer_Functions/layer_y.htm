capa_y

Puede utilizar esta función para establecer la posición y de la capa dentro del ámbito actual room. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función moverá la capa el número dado de píxeles a lo largo del eje vertical de room.

Sintaxis:

layer_y(layer_id, y)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to set the y position of
y Real The y position in the room to set the layer to

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

El código anterior comprueba la posición de la capa dada y si no está establecida en (0, 0) entonces se establece en esa posición.
