layer_set_visible

Con esta función puede alternar la visibilidad de una capa. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) así como el valor de conmutación para la capa donde visible es true e invisible es false.

Cuando una capa es invisible, no se dibujará nada de lo que esté asignado a la capa, y si hay alguna instancia asignada a la capa, ni siquiera se ejecutará su evento de dibujo (lo mismo que si se establece la variable de instancia visible a false). Ten en cuenta que las instancias que ya están marcadas como invisibles no serán marcadas como visibles si la capa en la que están se establece como visible, ya que son dos configuraciones independientes, aunque su comportamiento es el mismo.

Sintaxis:

layer_set_visible(layer_id, visible)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
visible Booleano Whether the layer should be visible (true) o no (false)

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
layer_set_visible(lay_id, false);
}
else
{
layer_set_visible(lay_id, true);
}

El código anterior obtiene el valor del ID de la capa llamada "Instances" en el editor room, y luego utiliza el ID para comprobar si la capa es visible o no, alternando la visibilidad de la capa en función del valor devuelto.
