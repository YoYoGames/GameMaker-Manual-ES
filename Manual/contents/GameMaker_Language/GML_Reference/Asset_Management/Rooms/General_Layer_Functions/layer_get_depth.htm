capa_obtener_profundidad

Puede utilizar esta función para obtener el valor de profundidad asociado a una capa determinada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devolverá la profundidad de esa capa como un número real. Tenga en cuenta que la profundidad se define como más alta cuanto más "lejos" de la cámara y más baja cuanto más "cerca" de la cámara, por lo que si tiene tres capas con profundidades -100, 0, 100, las capas se dibujarán en el orden 100, 0, -100, de modo que la capa "superior" (es decir, la más cercana a la vista de la cámara y por lo tanto dibujada sobre todo lo demás) será la capa con la profundidad -100.

Tenga en cuenta que si suministra un valor de ID de capa no válido obtendrá un error.

Sintaxis:

layer_get_depth(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to get the depth of (or the layer name as a string)

Devuelve:

Real

Ejemplo:

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

El código anterior comprueba la profundidad de un ID de capa almacenado en una variable global y si no es -100 destruye la capa y la vuelve a crear a la profundidad de -100.
