layer_get_element_type

Puede utilizar esta función para obtener el tipo de elemento para el elemento dado. Se suministra el valor único del ID del elemento (por ejemplo, el devuelto por la función que creó el elemento o por el editor room ) y la función devolverá una de las siguientes constantes (o -1 si el elemento no existe o el valor del ID es erróneo):

Tipo de elemento de capa Constante
Constante Descripción
layerelementtype_background El elemento es un fondo.
layerelementtype_instance El elemento es una instancia.
layerelementtype_sprite El elemento es un sprite asset.
layerelementtype_tilemap El elemento es un tilemap.
layerelementtype_particlesystem El elemento es un sistema de partículas.
layerelementtype_tile El elemento es un fondo heredado tile (esto sólo es válido para los proyectos que se han importado de versiones anteriores de GameMaker).
layerelementtype_sequence El elemento es un sequence asset.

Tenga en cuenta que esta función es más útil cuando tiene varios tipos de elementos diferentes asignados a la misma capa, y que puede obtener una lista de todos los elementos de cualquier capa utilizando la función layer_get_all_elements().

Sintaxis:

layer_get_element_type(element_id)

Argumento Tipo Descripción
layer ID del elemento de la capa The unique ID value of the element to get the type of

Devuelve:

Layer Element Type Constant or -1 (if element does not exist or is invalid)

Ejemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i])
    }
}

El código anterior obtiene los IDs de todos los elementos de instancia asignados a la capa de la instancia que ejecuta el código. A continuación, el código comprueba si alguno de los elementos devueltos es sprite assets y, si lo es, se destruye.
