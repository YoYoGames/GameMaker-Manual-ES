layer_get_script_end

Esta función devuelve el índice de la función de script de la función asignada para ejecutarse al final del renderizado de la capa dada, o devolverá -1 si no hay ninguna función asignada. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento). Puede asignar funciones script a una capa con layer_script_begin() y layer_script_end().

Sintaxis:

layer_get_script_end(layer_id);

Argumento Tipo Descripción
layer_id Cadena o ID de la capa The unique ID value of the layer to target (or the layer name as a string)

Devuelve:

Script Function or -1 (if no function is assigned)

Ejemplo:

if layer_get_script_end(layer) == -1
{
    layer_script_end(layer, scr_ResetShaderValues);
}

El código anterior comprobará si la capa que la instancia que ejecuta el código tiene una función script asignada y si no la tiene se le asigna una.
