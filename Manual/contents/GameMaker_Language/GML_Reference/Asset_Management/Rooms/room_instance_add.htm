room_instance_add

Con esta función puede añadir una instancia en cualquier room que no sea la actual y en cualquier posición dentro de ese room. La función devuelve el nombre único id de la instancia, que puede utilizarse en otras llamadas a la función para establecer propiedades, etc., de esa instancia, pero sólo una vez que el juego haya entrado en la sala especificada. Si se desea crear una instancia en la actual room se debe utilizar la función instance_create_layer(). Tenga en cuenta que llamar a esta función en un room asset creado en el navegador Asset añadirá permanentemente la instancia a la sala, e incluso llamando a game_restart() no devolverá el room a su estado original (sólo terminando el juego y abriéndolo de nuevo comenzará con el room en su estado original de nuevo).

Sintaxis:

room_instance_add(index, x, y, obj);

ArgumentoTipo Descripción
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

Devuelve:

Instance ID

Ejemplo:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

El código anterior añadirá un nuevo room al juego y luego copiará el contenido del room indexado como "rm_Base" en él. Luego añadirá una instancia de object "obj_player" en la posición 100,100 de este nuevo room.
