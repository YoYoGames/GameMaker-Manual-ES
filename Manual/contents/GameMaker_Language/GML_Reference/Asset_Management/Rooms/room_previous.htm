room_previous

Con esta función se puede recuperar el índice de la room antes de la entrada de room en la función. Por ejemplo, puede utilizar la variable room para obtener el índice del room actual y luego utilizar esta función para encontrar el room que le precede, tal y como aparece en el gestor de salas. Si no hay ningún room antes del que ha introducido, se devuelve -1. Tenga en cuenta que esta función no reconocerá ni tendrá en cuenta los rooms que se hayan añadido dinámicamente utilizando room_add() o room_duplicate().

Sintaxis:

room_previous(numb);

ArgumentoTipo Descripción
numb The index of the room to get the one before from.

Devuelve:

Room Asset

Ejemplo:

if room_previous(room) != -1
{
    room_goto_previous();
}

El código anterior comprobará si el anterior room existe y si es así, irá a él.
