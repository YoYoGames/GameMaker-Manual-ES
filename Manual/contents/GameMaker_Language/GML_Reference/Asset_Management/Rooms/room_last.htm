habitación_última

Esta variable de sólo lectura devuelve el índice del último room en el juego (esto se define por el orden en que aparecen los rooms en el Gestor de Salas y no por el orden en que fueron creados). Tenga en cuenta que esta variable no reconocerá ni tendrá en cuenta rooms que se hayan añadido dinámicamente utilizando room_add() o room_duplicate().

Sintaxis:

room_last;

Devuelve:

Room Asset

Ejemplo:

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

El código anterior comprueba si se ha pulsado una tecla y, si es así, pasa a la última room del juego.
