habitación_existe

Con esta función se puede comprobar si el room que se especifica existe o no. Esta función toma el índice room (un número real) y no el nombre room (un string). Esta función es más útil cuando se crea rooms dinámicamente utilizando la función room_add()pero también puede utilizar las variables de sólo lectura room_first y room_last o las funciones room_next() y room_previous() para obtener un índice específico de room, o proporcionar una variable que haya almacenado el índice de cualquier otro room en tu juego.

Sintaxis:

room_exists(index);

ArgumentoTipo Descripción
index The index of the room to check.

Devuelve:

Ejemplo:

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

El código anterior comprueba si el room indexado en el array "global.rm[]" existe y si lo hace entonces va a ese room.
