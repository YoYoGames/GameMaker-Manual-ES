layer_background_stretch

Con esta función puede cambiar un elemento de fondo sprite para que se estire y se adapte a room o para que permanezca a 1:1 con la resolución. Se suministra el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y luego establece el argumento de estiramiento a true o false. Cuando se establece a true el elemento sprite se estirará para ajustarse a room (ya sea escalado o reducido dependiendo de los tamaños de sprite y room), y cuando se establece a false, el elemento sprite se mostrará en su resolución nativa.

Sintaxis:

layer_background_stretch(background_element_id, stretch)

Argumento Tipo Descripción
background_element_id ID del elemento de fondo The unique ID value of the background element to change
stretch Booleano The stretch toggle, which can be true o false

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_stretch(back)
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

El código anterior comprueba si se ha pulsado el botón del ratón y, si se detecta alguno, alternará el estiramiento del elemento de fondo sprite asignado a la capa en la que se encuentra la instancia que ejecuta el código.
