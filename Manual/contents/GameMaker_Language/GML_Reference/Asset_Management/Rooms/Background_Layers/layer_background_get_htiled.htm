layer_background_get_htiled

Esta función se puede utilizar para averiguar si el elemento de fondo está en mosaico horizontal o no. Se da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo con layer_background_create() o cuando se utiliza la función layer_background_get_id()), y la función devolverá true si el elemento está actualmente en mosaico, o false si no lo está.

Sintaxis:

layer_background_get_htiled(background_element_id);

ArgumentoTipo Descripción
background_element_id The unique ID value of the background element to get the information from

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_htiled(back_id)
{
    layer_background_htiled(back_id, true);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar si el elemento está en mosaico a través del eje horizontal, y si no lo está entonces se establece para hacerlo.
