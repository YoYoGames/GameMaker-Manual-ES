layer_background_get_alpha

Esta función se puede utilizar para obtener el valor alfa del elemento de fondo. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y la función devolverá un valor entre 0 (totalmente transparente) y 1 (totalmente opaco).

Sintaxis:

layer_background_get_alpha(background_element_id);

ArgumentoTipo Descripción
background_element_id The unique ID value of the background element to get the information from

Devuelve:

(from 0 to 1)

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_alpha(back_id) < 0.1)
{
    layer_background_destroy(back_id);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar el alfa del elemento y si es menor de 0,1, entonces el elemento de la capa se destruye.
