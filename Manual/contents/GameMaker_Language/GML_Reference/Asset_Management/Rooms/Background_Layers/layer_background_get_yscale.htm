layer_background_get_yscale

Esta función se puede utilizar para obtener el valor actual del multiplicador de escala del elemento de fondo. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y la función devolverá un valor real que representa el multiplicador de escala que se está utilizando para dibujar el sprite. El valor por defecto es 1.

Sintaxis:

layer_background_get_yscale(background_element_id);

ArgumentoTipo Descripción
background_element_id The unique ID value of the background element to get the information from

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_yscale(back_id) != -1
{
    layer_background_yscale(back_id, -1);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar la escala y del elemento y si no es -1, se establece en -1.
