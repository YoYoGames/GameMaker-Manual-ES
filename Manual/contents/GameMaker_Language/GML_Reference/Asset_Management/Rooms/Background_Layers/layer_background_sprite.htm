layer_background_sprite

Con esta función puede establecer el índice sprite del elemento de fondo. Se suministra el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y luego da un índice sprite para ser utilizado. La imagen del elemento de fondo será reemplazada por el nuevo sprite. Si da un valor de -1, el elemento no tendrá asignado sprite (pero seguirá existiendo y se le podrá asignar un sprite más adelante).

Esta función también tiene el alias layer_background_change().

Sintaxis:

layer_background_sprite(background_element_id, sprite_index)

Argumento Tipo Descripción
background_element_id ID del elemento de fondo The unique ID value of the background element to change
sprite_index Activo Sprite The sprite index of the sprite to use for the background element

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

El código anterior obtendrá el ID de la capa llamada "Background_sky" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para comprobar el sprite asignado al elemento, estableciéndolo en el sprite "spr_Clouds" si no lo está ya.
