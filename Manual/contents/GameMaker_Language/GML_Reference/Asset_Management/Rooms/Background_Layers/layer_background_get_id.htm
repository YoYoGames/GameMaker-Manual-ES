layer_background_get_id

Esta función se puede utilizar para recuperar el valor de ID único del elemento de fondo en una capa. Se proporciona el ID de la capa (que se obtiene cuando se utiliza el nombre de la capa junto con layer_get_id()) y la función devolverá el valor de ID asociado al elemento de fondo de la capa. Tenga en cuenta que esta función está diseñada específicamente para su uso con fondos que han sido añadidos en IDE, ya que si añade un fondo a una capa a través de código utilizando la función layer_background_create()devolverá el ID único del elemento de fondo añadido.

Sintaxis:

layer_background_get_id(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target

Devuelve:

Background Element ID

Ejemplo:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

El código anterior obtendrá el ID de la capa llamada "Background_trees" y lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar el elemento sprite.
