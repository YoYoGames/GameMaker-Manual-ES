layer_background_vtiled

Con esta función puede cambiar si el elemento de fondo dado en una capa debe ser embaldosado verticalmente o no. Se da el ID del elemento de fondo (que se obtiene al crear un elemento de fondo con layer_background_create() o al utilizar la función layer_background_get_id()), y luego se establece el valor del mosaico. Si se establece en true, el elemento se mostrará en forma de mosaico verticalmente en room, y si se establece en false, el elemento no será mosaico. Esta función es sólo para proyectos 2D, y no funcionará correctamente cuando se utilice una proyección de cámara 3D.

Sintaxis:

layer_background_vtiled(background_element_id, tiled);

Argumento Tipo Descripción
background_element_id ID del elemento de fondo The unique ID value of the background element to change
tiled Booleano The vertical tiling toggle, which can be true o false

Devuelve:

N/A

Ejemplo:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

El código anterior comprobará el elemento de fondo asignado a la capa en la que se encuentra la instancia que ejecuta el código y, si no está embaldosado en ninguna dirección, lo pondrá en tile vertical y horizontalmente.
