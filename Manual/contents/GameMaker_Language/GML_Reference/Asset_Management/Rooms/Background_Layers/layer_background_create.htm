layer_background_create

Con esta función puede asignar un recurso sprite a una capa para utilizarla como fondo en su proyecto. Se suministra el ID de la capa (que se obtiene cuando se crea la capa usando layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y un índice sprite (que sería el nombre del sprite como se muestra en el navegador Asset ), y se añadirá a la capa. La función devuelve el valor de identificación único del elemento, que puede utilizarse en otras funciones de la capa para los fondos.

Sintaxis:

layer_background_create(layer_id, sprite)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
sprite Activo Sprite The sprite index to be used

Devuelve:

Background Element ID

Ejemplo:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

El código anterior crea una nueva capa y luego añade un nuevo elemento de fondo a la misma, estableciendo un sprite para ser la imagen de fondo utilizada.
