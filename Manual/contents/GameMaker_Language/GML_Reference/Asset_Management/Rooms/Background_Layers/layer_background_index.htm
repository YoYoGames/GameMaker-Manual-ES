layer_background_index

Esta función se puede utilizar para establecer el índice de la imagen del fondo sprite que tiene múltiples sub-imágenes. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o al utilizar la función layer_background_get_id()), y luego se establece el índice de la imagen a utilizar. Si establece un valor fuera del rango de sub-imágenes, entonces el índice de la imagen será loop alrededor.

 

Sintaxis:

layer_background_index(background_element_id, image_index);

Argumento Tipo Descripción
background_element_id Background Element ID The unique ID value of the background element to change
index Real The image index to use for the background

 

Devuelve:

N/A

 

Ejemplo:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

El código anterior obtendrá el ID de la capa llamada "Árboles_de_fondo" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar el índice de la imagen del elemento.
