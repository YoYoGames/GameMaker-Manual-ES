layer_background_speed

Esta función se puede utilizar para establecer la velocidad de una animación sprite que se ha asignado a un elemento de fondo. Se le da el ID del elemento de fondo (que se obtiene cuando se crea un elemento de fondo usando layer_background_create() o al utilizar la función layer_background_get_id()) y, a continuación, se establece la velocidad de la animación. Si el IDE ha establecido que la velocidad de animación para el sprite utilizado se calcule como Fotogramas por Fotograma de Juego, entonces el valor de la velocidad dado es un multiplicador, siendo 1 el valor por defecto, y estableciéndolo a 0,5 se reducirá a la mitad la velocidad de animación, mientras que estableciéndolo a 2 se duplicará. Sin embargo, si el IDE ha establecido que la velocidad de la animación para el sprite utilizado se calcule como Fotogramas por Segundo, entonces el valor de la velocidad dado es un valor literal de fotogramas por segundo, por lo que establecer esto a 5, por ejemplo, establecerá la animación a loop 5 fotogramas cada segundo que el juego se ejecute. Si el elemento de fondo no tiene sprite adjunto o el sprite utilizado no tiene sub-imágenes, esta función no tendrá efecto.

Sintaxis:

layer_background_speed(background_element_id, speed);

Argumento Tipo Descripción
background_element_id ID del elemento de fondo The unique ID value of the background element to change
speed Real The speed multiplier/value to use for the element

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

El código anterior obtendrá el ID de la capa llamada "Agua_de_fondo" y luego lo utilizará para obtener el ID del elemento de fondo en esa capa. Este ID se utiliza entonces para cambiar la velocidad de la imagen del elemento.
