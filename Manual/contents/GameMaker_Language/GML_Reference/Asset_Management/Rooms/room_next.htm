room_next

Con esta función puede recuperar el índice de la room después de la entrada de room en la función. Por ejemplo, puede utilizar la variable room para obtener el índice del room actual y luego utilizar esta función para encontrar el room que le sigue, tal y como aparece en el gestor de salas. Si no hay ningún room después del introducido, se devuelve -1. Tenga en cuenta que esta función no reconocerá ni tendrá en cuenta los rooms que se hayan añadido dinámicamente utilizando room_add() o room_duplicate().

Sintaxis:

room_next(numb);

ArgumentoTipo Descripción
numb The index of the room to get the next one after.

Devuelve:

Room Asset

Ejemplo:

if room_next(room) != -1
{
    room_goto_next();
}

El código anterior comprobará si el siguiente room existe y, si es así, irá a él.
