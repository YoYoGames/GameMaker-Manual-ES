room

Esta variable contiene el índice room para el actual room que su juego está ejecutando. Esta no es una variable de sólo lectura, sin embargo, cambiarla no cambiará el valor del índice para el room actual, sino que cambiará el room para que coincida con el índice al que has establecido la variable. Hay que tener cuidado al hacer esto, ya que si el índice al que cambias la variable no es válido, el juego lanzará un error y se cerrará. En general, es mucho mejor usar room_goto() para cambiar rooms.

Nota: Los IDs de Room no se basan en su orden en el Navegador Asset o en el Gestor Room, por lo que debe evitar suministrar un valor numérico directamente. En su lugar, utilice la constante de sala para el asset al que desea hacer referencia (que es simplemente su nombre) o recupérelo a través de una función.

Sintaxis:

room

Devuelve:

Room Asset

Ejemplo:

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

El código anterior comprobará si el room actual es el primer nivel, y en ese caso reproducirá la música de ese nivel.
