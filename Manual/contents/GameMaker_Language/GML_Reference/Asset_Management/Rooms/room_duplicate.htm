room_duplicate

Esto duplicará un room dado y devolverá el índice de duplicados que se utilizará en todas las llamadas posteriores para hacer referencia al nuevo room.

NOTA: Los nuevos rooms no forman parte de la ordenación habitual de room, por lo que no tienen un "anterior" o "siguiente" room (lo que significa que las funciones room_next() y room_previous() no funcionarán). Para saltar de un room añadido a otro, debe utilizar el índice del propio room.

Sintaxis:

room_duplicate(index);

ArgumentoTipo Descripción
index The index of the original room to be duplicated.

Devuelve:

Room Asset

Ejemplo:

global.myroom = room_duplicate(rm_level);

Esto duplicará el room indexado por "rm_level" y asignará el índice room de este nuevo room a la variable "global.myroom".
