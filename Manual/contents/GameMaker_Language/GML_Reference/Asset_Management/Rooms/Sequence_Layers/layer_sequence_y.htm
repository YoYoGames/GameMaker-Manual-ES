layer_sequence_y

Con esta función puede establecer la posición a lo largo del eje Y (vertical) de la room para el elemento sequence dado. Se proporciona el ID del elemento sequence devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa junto con la posición Y a establecer y el sequence se moverá a la nueva posición.

Sintaxis:

layer_sequence_y(sequence_element_id, pos_y)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target
pos_y The Y position to move the sequence element to

Devuelve:

Ejemplo:

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

El código anterior comprueba si el elemento sequence al que se hace referencia en la variable "my_seq" existe, y si lo hace establece la posición x/y a la de la instancia que ejecuta el código.
