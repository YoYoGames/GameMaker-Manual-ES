layer_sequence_is_paused

Con esta función se suministra el ID del elemento sequence -como el devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa - y comprobará si el sequence está actualmente en pausa o no, devolviendo true si está en pausa, y false si no lo está.

Sintaxis:

layer_sequence_is_paused(sequence_element_id)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target

Devuelve:

Ejemplo:

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

El código anterior comprueba si el ID del elemento sequence almacenado en la variable "title_sequence" está en pausa, y si lo está comienza a reproducirlo.
