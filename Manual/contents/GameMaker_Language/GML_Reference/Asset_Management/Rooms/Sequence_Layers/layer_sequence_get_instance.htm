layer_sequence_get_instance

Con esta función se suministra el ID del elemento sequence - como el devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa - y devolverá la estructura de instancia sequence. Puede encontrar más información sobre el formato de la estructura de instancia sequence en esta página.

Sintaxis:

layer_sequence_get_instance(sequence_element_id)

Argumento Tipo Descripción
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target

Devuelve:

Sequence Instance Struct

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

El código anterior crea un nuevo elemento sequence y luego recupera su ID de instancia sequence. Esto se utiliza entonces para cambiar la propiedad de escala de velocidad de reproducción de la instancia sequence.
