secuencia_de_capa_escala_x

Con esta función puede establecer la escala X para el elemento sequence dado. Se suministra el ID del elemento sequence devuelto por layer_sequence_create() o por una de las funciones de elementos de capa junto con la nueva escala a establecer en el eje X y el sequence será escalado por esta cantidad. Una escala de 1 indica que no hay escala (1:1), los valores más pequeños reducirán la escala (0,5, por ejemplo, reducirá a la mitad la anchura de sequence), los valores más grandes aumentarán la escala y los valores negativos reflejarán el sequence sobre su origen y lo escalarán, a menos que el valor utilizado sea exactamente -1 (en cuyo caso el sequence se reflejará sobre su origen sin escala).

Es muy importante tener en cuenta que la aplicación de un escalado desigual (por ejemplo, escalar el eje X en 3 y el eje Y en 2) a los elementos de sequence que contienen cualquier instancia que utiliza la rotación, puede causar problemas con el dibujo de la instancia, las colisiones, el culling, y muchas otras cosas. Básicamente, si su sequence depende de alguna propiedad de instancia, no recomendamos que combine el escalado desigual con la rotación de instancia.

También tenga en cuenta que no debe cambiar las variables image_xscale \ image_yscale \ image_angle \ x \ y para cualquier instancia en un sequence que utiliza esta función, ya que se sobrescribirá tan pronto como el sequence se actualiza después de llamar a la función.

Sintaxis:

layer_sequence_xscale(sequence_element_id, xscale)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target
xscale The new X axis scale value to apply to the sequence element

Devuelve:

Ejemplo:

if seq_scale < 2
{
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
}

El código anterior comprueba el valor mantenido en la variable seq_scale, y si es menor que 2 entonces lo suma y luego utiliza el valor para establecer la escala X e Y del elemento sequence referenciado en la variable my_seq.
