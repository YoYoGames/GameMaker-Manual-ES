layer_sequence_angle

Con esta función puede establecer el ángulo de rotación para el elemento sequence dado. Se proporciona el ID del elemento sequence devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa junto con el nuevo ángulo a establecer y el sequence girará sobre su origen hasta la nueva posición. Los ángulos en GameMaker se calculan con 0º a la derecha, y van en sentido contrario a las agujas del reloj - así que 90º es arriba, 180º es izquierda y 270º es abajo - y el ángulo por defecto para un sequence sería 0º.

Tenga en cuenta que no debe cambiar las variables image_xscale \ image_yscale \ image_angle \ x \ y para cualquier instancia en un sequence que utilice esta función, ya que se sobrescribirán tan pronto como el sequence se actualice después de llamar a la función.

Sintaxis:

layer_sequence_angle(sequence_element_id, angle)

Argumento Tipo Descripción
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target
angle Real The new angle to rotate the sequence element to

Devuelve:

N/A

Ejemplo:

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

El código anterior comprueba el valor mantenido en la variable current_angle, y si es menor de 90 entonces lo suma y luego utiliza el valor para establecer el ángulo del elemento sequence referenciado en la variable my_seq.
