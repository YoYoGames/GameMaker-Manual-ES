layer_sequence_pause

Con esta función se puede poner en pausa la reproducción del sequence dado. Se suministra el ID del elemento sequence devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa y la función pondrá en pausa el sequence hasta que vuelva a iniciar la reproducción utilizando la función layer_sequence_play().

Sintaxis:

layer_sequence_pause(sequence_element_id)

Argumento Tipo Descripción
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target

Devuelve:

N/A

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if (global.Pause)
{
    layer_sequence_pause(_seq);
}

El código anterior crea un nuevo sequence y almacena su ID en una variable local. A continuación, comprueba si el juego está en pausa, y si lo está, pausa la reproducción del sequence.
