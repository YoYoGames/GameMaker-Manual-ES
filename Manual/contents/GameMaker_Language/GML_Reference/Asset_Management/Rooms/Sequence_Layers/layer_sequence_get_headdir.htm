layer_sequence_get_headdir

Con esta función se suministra el ID del elemento sequence - como el devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa - y devolverá la dirección actual del cabezal de reproducción, que será una de las constantes enumeradas a continuación.

Sintaxis:

layer_sequence_get_headdir(sequence_element_id)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target

Devuelve:

Constante Descripción Valor
seq_dir_right El sequence reproduce los fotogramas en un orden incremental de izquierda a derecha 1
seq_dir_left El sequence reproduce los fotogramas en orden decreciente de derecha a izquierda -1

Ejemplo:

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

El código anterior comprueba la dirección actual del cabezal de reproducción del elemento sequence ID almacenado en la variable "title_sequence", y si no está establecido en seq_dir_left, se establece en este valor.
