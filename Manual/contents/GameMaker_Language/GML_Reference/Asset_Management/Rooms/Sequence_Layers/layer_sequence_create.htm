crear_secuencia_de_capa

Con esta función puede crear una instancia de un sequence asset en la capa dada. Se suministra el ID de la capa, que puede ser un string del nombre de la capa -como se define en el editor room - o el ID único de la capa -como devuelve la función layer_get_id()así como la posición X e Y en el room para crear el sequence, y finalmente el ID del sequence a crear. El ID de sequence es la constante del nombre que se ha definido en el navegador Asset para el elemento sequence. La función devolverá el ID único del elemento sequence, que puede utilizarse en todas las funciones de capa posteriores para sequences, o puede utilizarse para recuperar la instancia sequence struct utilizando la función layer_sequence_get_instance().

Cabe destacar que si el sequence contiene alguna pista object, sus instancias se crearán en cuanto se cree el propio elemento sequence, independientemente de la posición de sus teclas asset en la Hoja de Claves. El controlador Sequence simplemente cambia la visibilidad de la instancia para ocultarla y mostrarla dependiendo de la posición y duración de la tecla asset de la pista y no la crea y destruye repetidamente. Como resultado, las instancias ejecutarán sus eventos Create cuando se cree el elemento sequence y no cuando comiencen sus claves de activos.

Sintaxis:

layer_sequence_create(layer_id, x, y, sequence_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the instance layer to target
x Real The x position in the room to create the sequence at
y Real The y position in the room to create the sequence at
sequence_id Activo de la secuencia The sequence asset to use, as defined in the Asset Browser

Devuelve:

Sequence Element ID

Ejemplo:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

El código anterior crea un nuevo sequence en la capa "Fondo" y luego lo pausa.
