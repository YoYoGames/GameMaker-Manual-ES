layer_sequence_headdir

Con esta función puede establecer la dirección del cabezal de reproducción de sequence. Se proporciona el ID del elemento sequence devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa, y luego das la dirección del cabezal de reproducción que debe ser una de las siguientes constantes:

Constante Descripción Valor
seq_dir_right El sequence reproducirá los fotogramas en un orden incremental de izquierda a derecha 1
seq_dir_left El sequence reproducirá los fotogramas en orden decreciente de derecha a izquierda -1

Sintaxis:

layer_sequence_headdir(sequence_element_id, direction)

Argumento Tipo Descripción
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target
direction Dirección de la secuencia Constante The playhead direction, a constant, listed above

Devuelve:

N/A

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

El código anterior crea un nuevo sequence y almacena su ID en una variable local. Este ID se utiliza entonces para establecer la dirección del cabezal de reproducción para disminuir los fotogramas (reproducción de derecha a izquierda).
