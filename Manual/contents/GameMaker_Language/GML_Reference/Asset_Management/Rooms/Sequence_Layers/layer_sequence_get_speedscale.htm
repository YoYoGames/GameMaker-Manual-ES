layer_sequence_get_speedscale

Con esta función se suministra el ID del elemento sequence -tal como lo devuelve layer_sequence_create() o por una de las funciones de los elementos de la capa - y devolverá la escala de velocidad de reproducción actual. Este es el valor del multiplicador utilizado para reducir o acelerar la velocidad de reproducción. Un valor de 1 es el valor por defecto, y los valores inferiores a 1 significan que la reproducción se ralentiza y los valores superiores a 1 significan que la reproducción se acelera.

Sintaxis:

layer_sequence_get_speedscale(sequence_element_id)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target

Devuelve:

Ejemplo:

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

El código anterior comprueba la escala de velocidad del cabezal de reproducción actual del ID del elemento sequence almacenado en la variable "title_sequence", y si no está establecido en 1 se establece en este valor.
