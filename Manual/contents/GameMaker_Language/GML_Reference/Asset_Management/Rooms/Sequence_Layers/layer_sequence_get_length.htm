secuencia_de_capa_obtener_longitud

Con esta función se suministra el ID del elemento sequence -tal como lo devuelve layer_sequence_create() o por una de las funciones de los elementos de la capa - y devolverá la longitud de sequence. Este es el número de fotogramas durante los que se ejecutará sequence.

Sintaxis:

layer_sequence_get_length(sequence_element_id)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target

Devuelve:

Ejemplo:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

El código anterior recupera el número de fotogramas que ejecutará un sequence y luego utiliza este valor para establecer una alarma.
