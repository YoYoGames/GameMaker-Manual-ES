secuencia_de_capa_de_reproducción

Con esta función puede iniciar la reproducción del sequence dado. Se suministra el ID del elemento sequence devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa y la función reproducirá el sequence, que luego podrá pausar si lo desea utilizando la función layer_sequence_pause().

¡IMPORTANTE! Si su sequence tiene algunas instancias en él, estas instancias no deben cambiar sus variables image_xscale / image_yscale / image_angle / x / y ya que se sobrescribirán cuando el sequence actualice cada paso después de comenzar a ser reproducido.

Sintaxis:

layer_sequence_play(sequence_element_id)

Argumento Tipo Descripción
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if layer_get_element_type(a[i]) == layerelementtype_sequence
        {
            if global.Pause
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

El código anterior comprueba si el juego ha sido pausado o no cuando se pulsa una tecla. Si el juego está pausado, entonces loops a través de todos los elementos de sequence en la capa actual (la capa de la instancia que llama) y pausa su reproducción, y si el juego no está pausado, entonces el loop comenzará su reproducción de nuevo.
