secuencia_de_capas_obtener_x

Con esta función se suministra el ID del elemento sequence - como el devuelto por layer_sequence_create() o por una de las funciones del elemento de la capa - y devolverá la posición X actual en el juego room para el sequence.

Sintaxis:

layer_sequence_get_x(sequence_element_id)

Argumento Tipo Descripción
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target

Devuelve:

Real

Ejemplo:

if layer_sequence_get_x(title_sequence) != room_width / 2
{
    layer_sequence_x(title_sequence, room_width / 2);
}

El código anterior comprueba la posición X del ID del elemento sequence almacenado en la variable "title_sequence", y si no está ajustado a la mitad del ancho de room, se ajusta a este valor.
