secuencia_de_capa_obtener_secuencia

Con esta función se suministra el ID del elemento sequence - como el devuelto por layer_sequence_create() o por una de las funciones de los elementos de la capa - y devolverá la estructura del objeto sequence. Esta función evita la necesidad de obtener primero la instancia sequence struct y le permite acceder directamente a los datos de sequence. Puede encontrar más información sobre el formato de sequence object struct en esta página.

Sintaxis:

layer_sequence_get_sequence(sequence_element_id)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target

Devuelve:

Sequence Object Struct

Ejemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

El código anterior crea un nuevo elemento sequence y luego recupera sus datos sequence struct . A continuación, se utiliza para cambiar la velocidad de reproducción y el tipo de sequence.
