layer_sequence_exists

Con esta función puede comprobar si existe un elemento sequence en la capa dada. Se suministra el ID de la capa, que puede ser un string del nombre de la capa -como se define en el editor de room - o el ID único de la capa -como lo devuelve la función layer_get_id(), así como el ID del elemento sequence - devuelto por layer_sequence_create() o por una de las funciones de elementos de capa - y devolverá true si el elemento dado existe o false en caso contrario.

Sintaxis:

layer_sequence_exists(layer_id, sequence_element_id)

Argumento Tipo Descripción
layer_ID Cadena o ID de capa The unique ID or name of the layer to check
sequence_element_id ID del elemento de secuencia The unique ID value of the sequence element to target

Devuelve:

Boolean

Ejemplo:

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

El código anterior comprueba si el elemento sequence dado existe, y si no existe entonces crea un nuevo sequence en la capa dada y luego lo pausa.
