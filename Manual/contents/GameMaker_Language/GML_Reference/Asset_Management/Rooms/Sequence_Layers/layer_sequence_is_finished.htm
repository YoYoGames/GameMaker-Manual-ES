secuencia_de_capas_está_terminada

Con esta función se suministra el ID del elemento sequence -como el devuelto por layer_sequence_create() o por una de las funciones de elemento de capa- y comprobará si el sequence ha terminado de reproducirse o no, devolviendo true si lo hace, y false si no lo hace. Tenga en cuenta que esto sólo es aplicable cuando el sequence no está configurado como loop o ping-pong en el modo de reproducción.

Sintaxis:

layer_sequence_is_finished(sequence_element_id)

ArgumentoTipo Descripción
sequence_element_id The unique ID value of the sequence element to target

Devuelve:

Ejemplo:

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

El código anterior comprueba si el ID del elemento sequence almacenado en la variable "title_sequence" ha finalizado, y si lo hace comienza a reproducirse de nuevo y establece una alarma.
