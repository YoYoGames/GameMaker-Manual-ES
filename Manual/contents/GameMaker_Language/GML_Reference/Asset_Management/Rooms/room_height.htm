room_height

Esta variable contiene la altura del room actual en píxeles. Puedes modificar esta variable para cambiar la altura de room en cualquier momento, y los cambios se aplicarán a la parte inferior de room, ya que se considera que el origen es la esquina superior izquierda. Así, por ejemplo, si el room tiene una altura de 480px y lo ajustas a 640px, el room se expandirá hacia abajo con 180px extra añadidos en la parte inferior.

Sintaxis:

room_height

Devuelve:

Ejemplo:

if bbox_bottom > room_height
{
    y += room_height - bbox_bottom;
}

El código anterior comprueba si el cuadro delimitador de la instancia actual sprite es mayor que la altura de room, y si lo es, mueve la instancia hacia arriba.
