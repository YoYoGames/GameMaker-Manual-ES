room_get_viewport

Con esta función puede recuperar los detalles de un puerto de vista en un room distinto al actual. Se da el ID de room y el índice del puerto de vista a recuperar (de 0 a 7) y la función devolverá un array de 5 índices, donde:

  • [0] = visible (Booleano: true / false)
  • [1] = posición x (real)
  • [2] = posición y (real)
  • [3] = ancho (real)
  • [4] = altura (real)

Sintaxis:

room_get_viewport(rm, vind);

Argumento Tipo Descripción
rm Activo de la habitación The index of the room to get viewport data from
vind Real The index of the view port to get

Devuelve:

Array (5 elements: visible, x, y, width, height)

Ejemplo:

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

El código anterior recupera los datos del puerto de vista para el room dado y luego comprueba si el puerto está marcado como visible. Si no lo está, los datos del puerto de visualización se establecen para hacerlo visible.
