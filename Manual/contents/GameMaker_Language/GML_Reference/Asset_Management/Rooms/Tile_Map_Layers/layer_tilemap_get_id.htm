layer_tilemap_get_id

Esta función puede utilizarse para recuperar el valor de identificación único del elemento de mapa tile en una capa. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y la función devolverá el valor de ID asociado con el elemento de mapa tile en la capa. Tenga en cuenta que esta función está diseñada específicamente para su uso con los mapas tile que se han añadido en el IDE, ya que si se añade un mapa tile a una capa a través de código utilizando la función layer_tilemap_create(), entonces devolverá el ID único del elemento de mapa tile añadido. Si el ID del tilemap dado es incorrecto o el tilemap no existe, la función devolverá -1.

Sintaxis:

layer_tilemap_get_id(layer_id)

Argumento Tipo Descripción
layer_id Cadena o ID de la capa The unique ID value of the layer to target (or the layer name as a string)

Devuelve:

Tile Map Element ID or -1

Ejemplo:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

El código anterior obtendrá el ID de la capa llamada "tilemap_trees" y luego lo utilizará para obtener el ID del elemento de mapa tile en esa capa. Este ID se utiliza entonces para eliminar el mapa tile de la capa.
