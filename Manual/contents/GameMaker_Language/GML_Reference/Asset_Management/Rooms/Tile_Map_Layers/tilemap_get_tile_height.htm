tilemap_get_tile_height

Con esta función puede obtener la altura (en píxeles) de cada celda de tile del elemento de mapa tile. Se le da el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y la función devolverá la altura de la celda de tile set.

Sintaxis:

tilemap_get_tile_height(tilemap_element_id);

ArgumentoTipo Descripción
tilemap_element_id The unique ID value of the tile map element to get the tile cell height of

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

El código anterior utiliza el ID del mapa tile recuperado para obtener el ancho y el alto de la celda tile y utilizarlos para establecer dos variables globales.
