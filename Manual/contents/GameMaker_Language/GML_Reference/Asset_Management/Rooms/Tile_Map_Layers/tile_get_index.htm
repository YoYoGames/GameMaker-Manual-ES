tile_get_index

Esta función permite obtener el índice tile (la posición de tile dentro de la imagen tile set ) a partir de un conjunto de datos de mosaico. Se especifican los datos del mosaico, que pueden recuperarse con la función tilemap_get()y la función devolverá un valor entero para el índice.

Al utilizar esta función, asegúrese de que los datos del mosaico dado son válidos, de lo contrario el valor del índice devuelto tampoco será válido, lo que puede dar lugar a un comportamiento inesperado.

Sintaxis:

tile_get_index(tiledata)

Argumento Tipo Descripción
tiledata Datos de las baldosas the tile-data to check

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa tile de la capa dada y luego lo utiliza para obtener los datos de la celda bajo la posición del ratón. Estos datos se utilizan para establecer el índice de tile en un número aleatorio.
