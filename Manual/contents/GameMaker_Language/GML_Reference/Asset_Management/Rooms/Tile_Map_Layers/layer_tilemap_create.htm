layer_tilemap_create

Con esta función puede asignar un recurso de conjunto de azulejos a una capa que se utilizará como mapa tile en su proyecto. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y luego una posición inicial (x, y) para añadir el mapa tile a la room, el recurso tile set a utilizar, y luego la anchura y la altura del mapa tile en celdas (es decir: una anchura de 20 y una altura de 10 creará un mapa tile con 200 celdas que tiene 20 celdas de ancho y 10 celdas de alto), con el tamaño de las propias celdas definido por el tile set elegido. Cabe destacar que no se puede colocar tiles en posiciones negativas dentro del mapa tile, por lo que todos los tiles deben colocarse dentro de los espacios de celdas de 0 a ancho - 1, 0 a alto - 1.

Sintaxis:

layer_tilemap_create(layer_id, x, y, tileset, width, height)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
x Real The x position of the tile map in the room
y Real The y position of the tile map in the room
tileset Activo del conjunto de baldosas The Tile Set asset to be used
width Real The width tile map (in cells)
height Real The height of the tile map (in cells)

Devuelve:

Tile Map Element ID

Ejemplo:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

El código anterior crea una nueva capa y luego le añade un nuevo elemento de mapa tile, estableciendo la posición del mapa tile en (0,0), así como el tile set que se utilizará y la anchura y altura del mapa tile.
