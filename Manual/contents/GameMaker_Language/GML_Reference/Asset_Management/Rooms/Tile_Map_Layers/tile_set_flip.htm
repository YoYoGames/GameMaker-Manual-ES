tile_set_flip

Esta función se puede utilizar para establecer un conjunto dado de datos de baldosas para voltear el tile o no. Se dan los datos de las baldosas, que se pueden recuperar con la función tilemap_get()y, a continuación, se establece el argumento "flip" (voltear) en true si se desea voltear tile, o en false si se desea que tile esté en su estado predeterminado, sin voltear. La función devolverá un conjunto de datos de baldosas modificado que puede aplicarse con la función tilemap_set() función.

Sintaxis:

tile_set_flip(tiledata, flip)

ArgumentoTipo Descripción
tiledata the tile-data to set
flip Set to true para darle la vuelta y false para dejarlo como está

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_flip(data);
data = tile_set_flip(data, bool);
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa tile de la capa dada y luego obtiene la posición de la celda x e y para el tile bajo el ratón. Esta posición se utiliza entonces para obtener los datos del mosaico que se voltea y luego se utiliza para establecer el tile de nuevo.
