layer_tilemap_destroy

Esta función destruirá el elemento de mapa tile dado. Se proporciona el ID del mapa tile (que se obtiene al crear el mapa tile con layer_tilemap_create() o cuando se utiliza el ID de la capa junto con layer_get_tilemap_id()) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, sólo el mapa tile de la misma, y si el mapa tile es uno que ha sido añadido en el editor room, entonces la próxima vez que salga de room y luego regrese, el mapa tile será recreado de nuevo. Sin embargo, si el room es persistente, el mapa tile será eliminado a menos que se desactive de nuevo la persistencia de room.

Sintaxis:

layer_tilemap_destroy(tilemap_element_id)

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map to be destroyed

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

El código anterior comprueba la capa "Tiles_trees" para ver si el elemento de mapa tile dado existe y si lo hace, entonces se destruye (pero no la capa).
