tilemap_set_height

Esta función puede utilizarse para cambiar el tamaño de un elemento de mapa tile. Se da el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y la nueva altura del mapa tile en celdas tile.

Sintaxis:

tilemap_set_height(tilemap_element_id, height)

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map element to set the height of
height Real The height value (in "cells")

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_height(map_id) != room_height div 16)
{
    tilemap_set_height(map_id, room_height div 16);
}

El código anterior comprueba la anchura de un mapa específico de tile y, si no tiene el tamaño correcto, se fija la anchura.
