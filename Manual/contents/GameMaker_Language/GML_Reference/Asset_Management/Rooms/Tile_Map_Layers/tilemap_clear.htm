tilemap_clear

Con esta función puede borrar/ajustar todos los tiles de un determinado mapa de baldosas. Se le da el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()) y, a continuación, los datos de tile con los que desea borrar la capa. Un valor predeterminado de 0 borrará todos los tiles de la capa (esencialmente haciendo que todos los tiles estén "vacíos"), mientras que puede utilizar las funciones dedicadas a tile_* para crear sus propios datos tile con los que borrar el mapa tile.

Sintaxis:

tilemap_clear(tilemap_element_id, tiledata)

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map element to change
tiledata Datos de las baldosas (reales) The tile data to use to clear the layer

Devuelve:

N/A

Ejemplo:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

El código anterior obtiene el valor del ID de un mapa tile creado en el editor room, y luego lo borra utilizando "empty" tiles.
