tile_set_rotate

Esta función se puede utilizar para establecer un conjunto dado de datos de baldosas para girar el tile 90 grados o no. Se dan los datos del mosaico, que pueden ser recuperados usando la función tilemap_get()y, a continuación, se establece el argumento de rotación en true si se desea rotar el tile, o en false si se desea que el tile se encuentre en su estado predeterminado, sin rotar. La función devolverá un conjunto de datos de baldosas modificado que puede aplicarse con la función tilemap_set() función.

Sintaxis:

tile_set_rotate(tiledata, rotate)

Argumento Tipo Descripción
tiledata Datos de las baldosas The tile-data to set
rotate Booleano Set to true para girar y false para dejarlo como está

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa tile de la capa dada y luego obtiene la posición de la celda x e y para el tile bajo el ratón. Esta posición se utiliza entonces para obtener los datos del mosaico que se gira y luego se utiliza para establecer el tile de nuevo.
