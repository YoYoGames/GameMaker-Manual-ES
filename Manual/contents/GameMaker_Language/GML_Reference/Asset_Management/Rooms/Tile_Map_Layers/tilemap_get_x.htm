tilemap_get_x

Con esta función puede recuperar la posición x (dentro de room) del elemento de mapa tile. Se le da el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y la función devolverá la posición del eje x.

Sintaxis:

tilemap_get_x(tilemap_element_id);

ArgumentoTipo Descripción
tilemap_element_id The unique ID value of the tile map element to get the x position of

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10);

El código anterior utiliza el ID del mapa tile recuperado para obtener la posición x e y de tile del mapa tile y luego utiliza esos valores para moverlo.
