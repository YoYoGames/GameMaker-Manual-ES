tile_set_index

Esta función permite establecer el índice tile (la posición de tile dentro de la imagen tile set ) para un conjunto de datos de mosaico. Se dan los datos de los mosaicos, que pueden recuperarse con la función tilemap_get()y, a continuación, el valor entero que representa la posición de tile (su índice) dentro de tile set. La función devolverá un conjunto de datos de mosaico modificado que puede aplicarse mediante la función tilemap_set() función.

Sintaxis:

tile_set_index(tiledata, index)

Argumento Tipo Descripción
tiledata Datos de las baldosas the tile-data to set
index Real The integer index value of the tile to use

Devuelve:

Tile Data

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID del mapa tile de la capa dada y luego lo utiliza para obtener los datos de la celda bajo la posición del ratón. Estos datos se utilizan para establecer el índice de tile en un número aleatorio.
