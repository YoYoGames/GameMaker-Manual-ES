tilemap_x

Esta función controla la posición a lo largo del eje x del room del elemento de mapa asset tile en la capa. Se da el ID del elemento de mapa tile (que se obtiene al crear un elemento de mapa tile con layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego se establece el valor x a utilizar (basado en las coordenadas de room ).

Sintaxis:

tilemap_x(tilemap_element_id, x);

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map element to change
x Real The x position for the tile map

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

El código anterior obtiene el valor de ID del mapa tile asset asignado a la capa "Asset_sky" y luego establece su posición x a un valor aleatorio entre 0 y el ancho del room.
