layer_tilemap_exists

Puede utilizar esta función para comprobar si existe un elemento del mapa tile en una capa determinada. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o cuando se utiliza la función layer_tilemap_get_id()) y la función devolverá true si el elemento existe, o false si no existe.

NOTA: Esta función trabaja dentro del ámbito del objetivo actual room - por defecto el room en el que se llama a la función - que puede establecerse utilizando la función layer_set_target_room().

Sintaxis:

layer_tilemap_exists(layer_id, tilemap_element_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map element to check

Devuelve:

Boolean

Ejemplo:

var lay_id = layer_get_id("tilemap_trees");
if (layer_tilemap_exists(lay_id, global.Treestilemap))
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

El código anterior comprueba la capa "tilemap_trees" para ver si el elemento de mapa tile dado existe y si lo hace, entonces se destruye (pero no la capa).
