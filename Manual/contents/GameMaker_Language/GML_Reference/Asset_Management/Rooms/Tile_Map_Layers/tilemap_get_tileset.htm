tilemap_get_tileset

Con esta función puede recuperar el valor del índice de Tile Set asset asignado a un elemento de mapa tile dado en una capa. Se da el ID del elemento de mapa Tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y la función devolverá el índice de Tile Set asset.

Sintaxis:

tilemap_get_tileset(tilemap_element_id);

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the Tile Map element to get the tile set from

Devuelve:

Tile Set Asset

Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(map_id) != ts_Nighttime)
{
    tilemap_tileset(map_id, ts_Nighttime);
}

El código anterior comprueba el tile set actual asignado a la capa "Tiles_trees" y si no es "ts_Nighttime" entonces ese tile set se asigna al mapa tile.
