tilemap_get

Con esta función puede recuperar los datos de tile de una celda del elemento de mapa tile. Se le da el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), así como la posición x e y de la celda, y la función devolverá el "blob" de datos de tile. Estos datos son esencialmente una máscara de bits que contiene el índice tile, los booleanos de volteo/rotación/espejo y cualquier dato de máscara que se haya aplicado (ver tilemap_set_mask() para más detalles), y el valor de los datos resultantes puede utilizarse en las funciones de tile_* para cambiar las propiedades de tiles. Tenga en cuenta que estamos utilizando las posiciones de las celdas x/y basadas en el tamaño de la rejilla del mapa tile y no como una posición en el room, por lo que "cell_x" es un valor de 0 a la anchura del mapa de azulejos, y "cell_y" es un valor de 0 a la altura del mapa de azulejos. Si necesita obtener los datos de un tile en una posición específica de room, deberá utilizar la función tilemap_get_at_pixel().

¡IMPORTANTE! Si los tiles en el mapa tile no han sido modificados (es decir, no han sido rotados o volteados, etc...), entonces el valor de retorno de los datos de tile set será exactamente igual al índice del tile en el tile set. Por lo tanto, puede crear "mapas de colisión" de tiles utilizando un tile en el índice 1 en el tile set - por ejemplo - y luego utilizar esta función para comprobar 1 o 0 (un tile vacío) para calcular las colisiones.

Sintaxis:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map element to get the tile-data of
x_cell Real The cell along the x-axis to get the tile data from
y_cell Real The cell along the y-axis to get the tile data from

Devuelve:

Tile Data (-1 if there is an error)

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

El código anterior obtiene el ID del mapa tile en la capa "Tiles_Sky" y luego lo utiliza para obtener los datos de tile en la celda (0, 0). Estos datos de tile se voltean antes de ser utilizados para establecer el tile en el mapa tile de nuevo.
