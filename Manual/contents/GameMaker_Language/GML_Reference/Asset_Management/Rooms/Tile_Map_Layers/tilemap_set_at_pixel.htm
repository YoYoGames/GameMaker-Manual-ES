tilemap_set_at_pixel

Esta función se puede utilizar para establecer una celda dentro del elemento de mapa tile en la capa a un nuevo tile utilizando la posición real del tile para cambiar dentro del room. Usted da el ID del elemento de mapa tile (que obtiene cuando crea un elemento de mapa tile utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego los datos de tile que hay que establecer, así como la posición dentro de room. Puede recuperar los datos de tile utilizando la función tilemap_get_at_pixel() y luego utilizar las funciones de tile_ para modificarlos antes de volver a fijarlos mediante esta función. La función devolverá true si el tile se estableció con éxito y false si hubo un problema y no se estableció.

Sintaxis:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tile map element to change
tiledata Datos de las baldosas (reales) The tile set data to set
xcell Real The x position (in the room)
ycell Real The y position (in the room)

Devuelve:

Boolean

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

El código anterior obtiene el ID del mapa tile en la capa "Tiles_Sky" y luego lo utiliza para obtener los datos de tile en la posición (4, 4). Estos datos de tile se voltean antes de ser utilizados para establecer el tile en el mapa tile de nuevo.
