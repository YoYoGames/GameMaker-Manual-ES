tilemap_get_cell_at_pixel

Con esta función puede recuperar la posición del eje y de una celda individual del mapa tile dando la posición relativa del eje y dentro de room. Se da el ID del elemento del mapa tile (que se obtiene cuando se crea un elemento del mapa tile usando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), así como la posición x e y dentro de la sala y la función devolverá la posición y de la celda dentro del mapa tile para ese punto. Tenga en cuenta que si el valor está fuera del área del mapa tile, y no hay ninguna celda disponible, devolverá -1.

Sintaxis:

tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y);

ArgumentoTipo Descripción
tilemap_element_id The unique ID value of the tile map element to get the cell y position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

Devuelve:

(y-axis cell position or -1 if there is an error)

Ejemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

El código anterior utiliza el ID del mapa de tile recuperado para obtener la posición de la celda a partir de un conjunto de coordenadas de room y, a continuación, almacena los datos de cualquier tile que se encuentre allí en una variable de instancia.
