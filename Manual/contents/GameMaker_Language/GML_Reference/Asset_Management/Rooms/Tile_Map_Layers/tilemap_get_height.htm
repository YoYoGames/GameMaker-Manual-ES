tilemap_get_height

Con esta función puede obtener la altura (en celdas) del elemento de mapa tile. Se le da el ID del elemento de mapa tile (que se obtiene cuando se crea un elemento de mapa tile utilizando layer_tilemap_create() o cuando utilice la función layer_tilemap_get_id()), y la función devolverá la altura (número de celdas de ancho).

Sintaxis:

tilemap_get_height(tilemap_element_id);

ArgumentoTipo Descripción
tilemap_element_id The unique ID value of the tile map element to get the height of

Devuelve:

Ejemplo:

var lay_id = layer_get_id("espadrilles");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

El código anterior utiliza el ID del mapa de tile recuperado para obtener la anchura y la altura de tile del mapa de tile y luego utiliza esos valores para establecer una variable de instancia.
