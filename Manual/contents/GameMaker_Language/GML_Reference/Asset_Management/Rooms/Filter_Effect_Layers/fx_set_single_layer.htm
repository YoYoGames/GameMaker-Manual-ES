fx_set_single_layer

Esta función se utiliza para activar o desactivar el modo "Capa única" para un filtro/efecto. Por defecto, un filtro/efecto se aplica a la capa a la que está asignado y a todas las capas por debajo de esa capa, sin embargo, puede activar el modo de capa única en un FX struct para asegurarse de que sólo se aplica a la capa a la que está asignado.

Se especifica un FX struct (como el devuelto por fx_create() o layer_get_fx()) y un valor booleano para activar (true) o desactivar (false) el modo de capa única.

La siguiente imagen muestra un filtro utilizado con el modo de capa única desactivado (que es el comportamiento por defecto para todas las capas FX) y el mismo filtro con el modo de capa única activado:

NOTA Cuando habilite el modo de capa única para un FX, asegúrese de aplicarlo a una capa que no sea una capa FX, porque una capa FX por sí misma no contiene nada y por tanto ningún filtro/efecto será visible.

Sintaxis:

fx_set_single_layer(filter_or_effect, enable);

ArgumentoTipo Descripción
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) o desactivar (false) el modo de capa única

Devuelve:

Ejemplo:

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

El código anterior crea un nuevo screenshake FX, activa el modo de capa única en él y luego lo aplica a una capa de room. Esto significa que el filtro screenshake sólo se aplicará a la capa "ShakeyThings ".
