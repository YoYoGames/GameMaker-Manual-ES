fx_get_name

Esta función se utiliza para recuperar el nombre de un filtro/efecto de su struct. Se especifica el FX struct que se va a leer (tal y como se devuelve desde fx_create() o layer_get_fx()) y la función devuelve su nombre como string. Este nombre se puede pasar a fx_create() para crear un nuevo FX struct utilizando el mismo tipo de filtro/efecto.

Sintaxis:

fx_get_name(filter_or_effect);

ArgumentoTipo Descripción
filter_or_effect The FX struct to get the name of

Devuelve:

Ejemplo:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

El código anterior ejecuta un bucle for bucle a través de todas las capas presentes en el room, y comprueba en cada capa si hay un FX struct. Si una capa tiene un FX struct, comprueba el nombre de ese filtro/efecto llamando a fx_get_name() sobre él; si es igual a "_filter_tintfilter", lo que significa que es un filtro "Tinte de color", cambia su color de tinte a azul.
