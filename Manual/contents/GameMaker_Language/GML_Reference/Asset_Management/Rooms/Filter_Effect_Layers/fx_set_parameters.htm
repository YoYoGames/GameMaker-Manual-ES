fx_set_parameters

Esta función se utiliza para cambiar los parámetros de un filtro/efecto. Se especifica un FX struct (como el devuelto por fx_create() o layer_get_fx()) y un struct que contiene sus parámetros (como el devuelto por fx_get_parameters()). Esto hará que sus cambios sean visibles en la capa a la que está asignado el FX struct.

No es necesario que su parámetro struct contenga todos los parámetros aplicables al filtro/efecto, y sólo se actualizarán para el filtro/efecto las variables que estén presentes en el struct dado.

Sintaxis:

fx_set_parameters(filter_or_effect, parameter_struct);

ArgumentoTipo Descripción
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

Devuelve:

Ejemplo:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

El código anterior obtiene el FX struct para una capa que se supone que tiene aplicado el filtro "Color Tint", y recupera su parámetro struct llamando a fx_get_parameters(). Después crea un valor oscilante usando sin() y current_time, que luego se usa para los valores RGBA para el efecto de tinte para que siga animando. La matriz RGBA se asigna a la variable g_TintCol en el parámetro struct, y el struct se aplica de nuevo al FX struct llamando a fx_set_parameters().
