fx_get_parameter

Esta función se utiliza para recuperar el valor de un parámetro de un FX Struct. Se especifica el FX struct que se va a leer (como el devuelto por fx_create() o layer_get_fx()) y el nombre del parámetro como string, y la función devuelve su valor actual. Este puede ser un valor real, o un array si el parámetro es un vector (que es un grupo de valores, como un vec4 que almacena un color RGBA).

Encuentra los nombres de los parámetros de un filtro/efecto en esta página: Tipos y parámetros de FX.

Sintaxis:

fx_get_parameter(filter_or_effect, parameter_name);

Argumento Tipo Descripción
filter_or_effect Estructura FX The FX struct to get the parameter from
parameter_name Cadena The name of the parameter to get the value of (as a string)

Devuelve:

Real or Array

Ejemplo:

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

El código anterior recupera el FX struct de una capa que se supone que tiene aplicado el filtro "Color Tint", y obtiene el valor de su parámetro "g_TintCol", que devuelve una matriz que contiene los 4 valores de su color de tinte (rojo, verde, azul y alfa). A continuación, imprime ese valor en el registro de salida.
