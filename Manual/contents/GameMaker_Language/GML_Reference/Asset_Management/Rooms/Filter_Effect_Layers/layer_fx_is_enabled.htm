layer_fx_is_enabled

Esta función devuelve si el filtro/efecto de una capa está activado.

Se especifica el ID o el nombre de la capa a la que se quiere apuntar, y la función devuelve un valor booleano que indica si el FX de la capa está activado(true) o desactivado(false).

Sintaxis:

layer_fx_is_enabled(layer_name_or_id);

Argumento Tipo Descripción
layer_name_or_id Cadena o ID de la capa The name or ID of the layer to check

Devuelve:

Boolean

Ejemplo:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

El código anterior "reinicia" la instancia, moviéndola a su posición original y rellenando su HP. A continuación, comprueba si la capa Desaturar FX está activada, y si lo está, la desactiva.
