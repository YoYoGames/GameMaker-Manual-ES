layer_enable_fx

Esta función activa/desactiva el filtro/efecto asignado a una capa de Room. Se especifica el ID o el nombre de la capa que se desea modificar y, a continuación, un valor booleano que indica si el FX debe estar activado(true) o desactivado(false).

Pasar false no eliminará el FX de la capa, sino que simplemente lo hará invisible. Utilice layer_clear_fx() para eliminar un FX de una capa.

Del mismo modo, pasar true no hará nada si no hay ningún FX aplicado a la capa. Utilice layer_set_fx() para aplicar un FX a una capa.

Sintaxis:

layer_enable_fx(layer_name_or_id, enable);

Argumento Tipo Descripción
layer_name_or_id Cadena o ID de la capa The name or ID of the layer to modify
enable Booleano Whether to enable or disable the FX

Devuelve:

N/A

Ejemplo:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

El código anterior activa la capa Desaturate FX si el valor de HP de la instancia es menor o igual a 3, de lo contrario la desactiva. Esto indica al jugador que su HP es bajo, desaturando la pantalla.
