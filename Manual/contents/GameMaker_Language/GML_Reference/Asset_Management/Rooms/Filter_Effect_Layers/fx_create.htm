fx_create

Esta función se utiliza para crear un nuevo FX Struct para un tipo de efecto determinado, que luego se puede aplicar a una capa.

Los nombres de los Filtros/Efectos que puede pasar a esta función están listados en esta página: Todos los tipos de filtros/efectos.

Uso

Puedes leer los parámetros del efecto creado llamando a fx_get_parameter/s en el struct devuelto, y asignarles nuevos valores llamando a fx_set_parameter/s.

Tenga en cuenta que el struct devuelto por esta función no contendrá miembros visibles y sólo podrá ser modificado a través de una de las Funciones de Filtro/Efecto.

Fallos de funcionamiento

La función devolverá -1 si el struct falló al crearse por cualquier razón; esto puede ocurrir si el nombre de FX especificado era incorrecto o el FX no fue cargado en el paquete del juego.

Para asegurarse de que esta función funciona correctamente, asegúrese de que el FX que necesita está cargado en su paquete de juego, como se explica en la página de Funciones de Filtros y Efectos:

Si no se cumplen estas condiciones, es decir, si no se pasa un literal string y el filtro/efecto no está presente en ningún rooms, la función no podrá cargar ese filtro/efecto y devolverá -1.

Sintaxis:

fx_create(filter_or_effect_name);

Argumento Tipo Descripción
filter_or_effect_name Cadena The name of the filter/effect to create (get them from All Filter/Effect Types).

Devuelve:

FX Struct or -1 (on failure)

Ejemplo:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

El código anterior crea un nuevo FX struct a partir del efecto "_filter_tintfilter ", que es el filtro "Colour Tint" que se encuentra en el editor Room. Asigna un valor a su parámetro "g_TintCol " que es el color del tinte, y como es un vec4 internamente, toma un array que contiene cuatro valores (correspondientes a sus valores rojo, verde, azul y alfa). El FX struct para el tinte se aplica entonces a una capa FX.
