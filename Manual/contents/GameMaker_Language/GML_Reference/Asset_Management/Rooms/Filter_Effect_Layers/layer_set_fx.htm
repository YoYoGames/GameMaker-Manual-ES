capa_set_fx

Esta función se utiliza para asignar un FX struct a una capa Room. Se especifica el ID o el nombre de la capa que se desea modificar y, a continuación, un FX struct (como el devuelto por fx_create() o layer_get_fx()) que se aplicará a la capa especificada, haciendo que el efecto sea visible en la room.

Si especifica una capa que no es un Filtro/Capa de Efecto sino que es de otro tipo (como una Capa de Instancia, Asset Layer, Tile Layer, etc.) entonces el efecto se aplicará al contenido de esa capa así como a cualquier capa por debajo de ella.

Si la capa especificada ya tiene un FX struct asignado, será sustituido por el nuevo.

NOTA Llamar a esta función en el nivel raíz de un script no hará nada, ya que scripts se ejecuta antes de que se cargue el primer room. Puede establecer una sala de destino primero y luego llamar a esta función.

Sintaxis:

layer_set_fx(layer_name_or_id, filter_or_effect);

Argumento Tipo Descripción
layer_name_or_id Cadena o ID de la capa The name or ID of the layer to modify
filter_or_effect Estructura FX The FX struct to apply to the layer

Devuelve:

N/A

Ejemplo:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

El código anterior crea un nuevo FX struct a partir del efecto "_filter_tintfilter ", que es el filtro "Colour Tint" que se encuentra en el editor Room. Asigna un valor a su parámetro "g_TintCol " que es el color del tinte, y como es un vec4 internamente, toma un array que contiene cuatro valores (correspondientes a sus valores rojo, verde, azul y alfa). El FX struct para el tinte se aplica entonces a una capa FX.
