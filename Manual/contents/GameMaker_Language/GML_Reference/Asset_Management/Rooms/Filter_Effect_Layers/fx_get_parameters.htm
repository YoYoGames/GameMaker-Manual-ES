fx_get_parameters

Esta función se utiliza para recuperar una estructura que contiene todos los parámetros para el FX struct dado. Se especifica un FX struct (como el devuelto por fx_create() o layer_get_fx()) y esta función devuelve un struct que contiene variables para todos los parámetros aplicables a ese filtro/efecto. Los parámetros que utilizan vectores (es decir, grupos de valores) estarán presentes como matrices.

Puede obtener los nombres de todos los parámetros en un array separado llamando a fx_get_parameter_names(), o encontrarlos en esta página: Tipos y Parámetros FX.

El cambio de valores en este struct no aplicará esos cambios al filtro/efecto; tendrá que llamar a fx_set_parameters() para aplicar el struct modificado de nuevo al filtro/efecto para que los cambios surtan efecto.

Sintaxis:

fx_get_parameters(filter_or_effect);

Argumento Tipo Descripción
filter_or_effect Estructura FX The FX struct for which the parameters will be returned

Devuelve:

Struct (or -1 on failure)

Ejemplo:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

El código anterior obtiene el FX struct para una capa que se supone que tiene aplicado el filtro "Color Tint", y recupera su parámetro struct llamando a fx_get_parameters(). Después crea un valor oscilante utilizando sin() y current_time, que luego se utiliza para los valores RGBA para el efecto de tinte para que siga animando. La matriz RGBA se asigna a la variable g_TintCol en el parámetro struct, y el struct se aplica de nuevo al FX struct llamando a fx_set_parameters().
