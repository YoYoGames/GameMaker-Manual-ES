layer_clear_fx

Esta función se utiliza para borrar el FX struct que se aplica a la capa especificada. Especifique el ID o el nombre de la capa que desea modificar y esta función eliminará el FX struct que se aplica a esa capa, desactivando sus efectos por completo.

Si utiliza esta función para desactivar temporalmente un efecto y necesita restaurarlo en la capa, primero debe guardar el FX struct para la capa llamando a layer_get_fx() y luego llamar a esta función para borrarlo en la capa. Más tarde puedes volver a aplicarlo a la capa llamando a layer_set_fx(), que volverá a activar el efecto.

Sintaxis:

layer_clear_fx(layer_name_or_id);

Argumento Tipo Descripción
layer_name_or_id Cadena o ID de la capa The name or ID of the layer to clear the FX for

Devuelve:

N/A

Ejemplo:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

El código anterior comprueba si se pulsa la tecla "E", y luego comprueba si la capa "EffectLayer" tiene un filtro/efecto aplicado; en ese caso almacena su FX struct en una variable de instancia temporal y limpia la capa de cualquier filtro/efecto. Cuando se pulsa la tecla "F", comprueba si la variable temporal contiene algo distinto de -1, y en ese caso aplica ese FX de nuevo a la capa y restablece la variable temporal a -1. Todo esto te permite esencialmente activar y desactivar un efecto cuando lo necesites sin perder sus parámetros.
