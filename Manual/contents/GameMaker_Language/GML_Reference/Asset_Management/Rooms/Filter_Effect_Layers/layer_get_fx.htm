layer_get_fx

Esta función se utiliza para recuperar el FX struct de una capa. Se especifica el ID o el nombre de la capa a la que se quiere apuntar y la función devolverá un struct que contiene información sobre su efecto aplicado. Este struct será similar al struct que se obtiene de fx_create(), y las funciones fx_get_parameter/s y fx_set_parameter/s pueden utilizarse en él para leer y modificar sus parámetros.

Si la capa especificada no tiene filtros/efectos aplicados, la función devolverá -1.

Sintaxis:

layer_get_fx(layer_name_or_id);

Argumento Tipo Descripción
layer_name_or_id Cadena o ID de capa The name or ID of the layer to read

Devuelve:

FX Struct (or -1 if not found)

Ejemplo:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

El código anterior ejecuta un bucle for bucle a través de todas las capas presentes en el room, y comprueba en cada capa si hay un FX struct. Si una capa tiene un FX struct, comprueba el nombre de ese filtro/efecto llamando fx_get_name() en él; si es igual a "_filter_tintfilter", lo que significa que es un filtro "Color Tint", cambia su color de tinte a azul.
