fx_get_single_layer

Esta función devuelve si el filtro/efecto suministrado utiliza el modo de capa única (true) o no (false). Consulte fx_set_single_layer() para obtener información sobre el modo de capa única.

Se especifica un FX struct (como el devuelto por fx_create() o layer_get_fx()) y la función devuelve un booleano que indica si el modo de capa única está activado o no para el FX.

Sintaxis:

fx_get_single_layer(filter_or_effect);

ArgumentoTipo Descripción
filter_or_effect The FX struct to read

Devuelve:

Boolean

Ejemplo:

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

El código anterior obtiene el FX struct de una capa llamada "Distorsión" y comprueba si el modo de capa única está activado en ella. Si no lo está, activa el modo de capa única para ese FX.
