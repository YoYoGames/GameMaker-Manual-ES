fx_set_parameter

Esta función se utiliza para cambiar el valor de un parámetro en el FX struct dado. Se especifica un FX struct (como el devuelto por fx_create() o layer_get_fx()), un nombre de parámetro como string, y su nuevo valor. Esto hará que sus cambios sean visibles en la capa a la que está asignado el FX struct.

Puedes pasar múltiples argumentos para el parámetro en caso de que sea un vector, sin embargo también puedes simplemente especificar un array como único argumento que contendría tus múltiples valores.

Encuentra los nombres de los parámetros de un filtro/efecto en esta página: Tipos y parámetros de FX.

Sintaxis:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

Argumento Tipo Descripción
filter_or_effect Estructura FX The FX struct to modify
parameter_name Cadena The name of the parameter to change the value of (as a string)
val... Real o Array The new value(s) for that parameter as a real or array

Devuelve:

N/A

Ejemplo:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

El código anterior crea un nuevo FX struct a partir del efecto "_filter_tintfilter ", que es el filtro "Colour Tint" que se encuentra en el editor Room. Asigna un valor a su parámetro "g_TintCol " que es el color del tinte, y como es un vec4 internamente, toma un array que contiene cuatro valores (correspondientes a sus valores rojo, verde, azul y alfa). El FX struct para el tinte se aplica entonces a una capa FX.

Tenga en cuenta que también puede pasar valores al vector como argumentos separados en lugar de un array:

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
