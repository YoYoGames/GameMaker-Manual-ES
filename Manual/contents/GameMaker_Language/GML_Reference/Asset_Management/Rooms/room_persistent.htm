habitación_persistente

Esta variable se puede utilizar para obtener y establecer el flag persistente para el room actual. Si se establece como true el room se considera persistente, en cuyo caso cada vez que se abandona el room y se vuelve de nuevo se habrá mantenido el estado de las instancias dentro de ese room. Sin embargo, si se marca como false, cada vez que se vuelva a room se restablecerá su estado inicial. Debes tener en cuenta que un room persistente utiliza considerablemente más memoria que un room normal y no es recomendable tener demasiados en tu juego.

Sintaxis:

room_persistent;

Devuelve:

Ejemplo:

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

El código anterior comprueba la variable "vidas" y si es menor que 1, establecerá la persistencia de room en false y luego cambiará rooms.
