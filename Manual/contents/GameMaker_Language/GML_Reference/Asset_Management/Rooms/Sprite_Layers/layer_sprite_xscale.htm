layer_sprite_xscale

Usando esta función puede cambiar si el elemento sprite dado en una capa debe ser escalado a lo largo del eje x o no. Se da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego establece el valor de la escala. Una escala de 1 indica que no hay escala (1:1), los valores más pequeños reducirán la escala (0,5, por ejemplo, reducirá a la mitad la anchura del sprite utilizado), los valores más grandes aumentarán la escala, y los valores negativos voltearán el sprite y lo escalarán, a menos que el valor utilizado sea exactamente -1 (en cuyo caso el sprite utilizado simplemente se volteará de derecha a izquierda sobre su posición de origen sin escalar).

Sintaxis:

layer_sprite_xscale(sprite_element_id, xscale);

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite element to change
xscale The xscale value (default is 1)

Devuelve:

Ejemplo:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

El código anterior comprobará el elemento sprite asignado a la capa en la que se encuentra la instancia que ejecuta el código y, si está escalado en cualquier dirección, establecerá tanto la escala del eje x como la del eje y en 1.
