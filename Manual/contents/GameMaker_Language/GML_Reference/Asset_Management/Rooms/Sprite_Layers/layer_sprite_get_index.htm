layer_sprite_get_index

Esta función se puede utilizar para obtener el valor del índice de la imagen actual del elemento sprite. Se da el ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y la función devolverá un valor real que representa el índice de imagen que se muestra para el sprite. La función devolverá -1 si el elemento sprite no existe o el elemento no tiene un sprite válido asignado.

Sintaxis:

layer_sprite_get_index(sprite_element_id);

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite element to get the information from

Devuelve:

Real or -1

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_index(spr_id) < 4)
{
    layer_sprite_index(spr_id, 4);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza entonces para comprobar si el índice de la imagen del elemento es menor que 4, y si es así se establece en 4.
