layer_sprite_create

Con esta función puede asignar un recurso sprite a una capa que se utilizará en su proyecto. Se suministra el ID de la capa (que se obtiene al crear la capa con layer_create() o cuando se utiliza el nombre de la capa junto con layer_get_id()), una posición dentro de room, y un índice sprite (que sería el nombre de sprite como se muestra en el navegador Asset ), y se añadirá a la capa. La función devuelve el valor de identificación único del elemento, que puede utilizarse en otras funciones de capa para sprites.

Sintaxis:

layer_sprite_create(layer_id, x, y, sprite)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target
x Real The x position to use
y Real The y position to use
sprite Activo Sprite The sprite index to be used

Devuelve:

Sprite Element ID

Ejemplo:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

El código anterior crea una nueva capa y luego añade 10 nuevos elementos de sprite a la misma, almacenando el ID de cada elemento en un array.
