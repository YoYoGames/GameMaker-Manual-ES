layer_sprite_get_sprite

Esta función puede utilizarse para obtener el índice actual de sprite del elemento sprite. Se da el ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y la función devolverá un valor real que representa el índice sprite que se muestra. Si el elemento no tiene asignado sprite, la función devolverá -1.

Sintaxis:

layer_sprite_get_sprite(sprite_element_id);

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite element to get the information from

Devuelve:

Sprite Asset or -1

Ejemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
    layer_sprite_change(spr_id, spr_Clouds);
}

El código anterior obtendrá el ID de la capa llamada "sprite_sky" y luego lo utilizará para obtener el ID del elemento sprite en esa capa. Este ID se utiliza entonces para comprobar el sprite asignado al elemento, estableciéndolo en el sprite "spr_Clouds" si no lo está ya.
