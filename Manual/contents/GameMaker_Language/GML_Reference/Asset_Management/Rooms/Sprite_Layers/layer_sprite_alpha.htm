capa_sprite_alpha

Esta función controla el alfa (transparencia) del sprite en la capa asset. Se le da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y luego establece el valor alfa a utilizar. El valor alfa puede estar entre 0 (totalmente transparente) y 1 (totalmente opaco), siendo el valor alfa por defecto para el elemento sprite 1. Tenga en cuenta que si la capa a la que se ha asignado el elemento sprite no es visible -o el propio elemento se ha hecho invisible- no verá ninguna diferencia con esta función hasta que la capa o el elemento se hayan hecho visibles de nuevo.

Sintaxis:

layer_sprite_alpha(sprite_element_id, alpha);

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

El código anterior obtiene el valor de ID de la sprite asset llamada "Clouds" asignada a la capa "Asset_sky" y luego establece su alfa a un valor aleatorio entre 0 y 1.
