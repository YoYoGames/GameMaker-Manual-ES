capa_espiga_y

Esta función controla la posición a lo largo del eje y del room del elemento asset sprite en la capa. Se da el ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego se establece el valor y a utilizar (basado en las coordenadas de room ).

Sintaxis:

layer_sprite_y(sprite_element_id, y);

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite element to change
y The y position for the sprite

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

El código anterior obtiene el valor de ID de la sprite asset "Clouds" asignada a la capa "Asset_sky" y luego establece su posición y a un valor aleatorio entre 50 y la altura de la room menos 200.
