layer_sprite_destroy

Esta función destruirá el elemento sprite dado. Se suministra el ID de sprite (que se obtiene cuando se crea el sprite usando layer_sprite_create() o cuando se utiliza el ID de la capa junto con layer_get_sprite_id()) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, sólo el sprite de la misma, y si el sprite es uno que ha sido añadido en el editor room, entonces la próxima vez que salga del room y luego regrese, el sprite será recreado de nuevo. Sin embargo, si el room es persistente, el sprite será eliminado a menos que se desactive de nuevo la persistencia del room.

Sintaxis:

layer_sprite_destroy(sprite_element_id)

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite to be destroyed

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Asset_Trees");
var spr_id = layer_sprite_get_id(lay_id, "graphic_254367CB");

if (layer_sprite_exists(lay_id, spr_id))
{
    layer_sprite_destroy(lay_id);
}

El código anterior comprueba la capa "Asset_Trees" para ver si el elemento sprite dado existe y si lo hace, entonces se destruye (pero no la capa).
