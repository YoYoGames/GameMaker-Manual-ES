layer_sprite_blend

Esta función controla la mezcla (o "tinte") de la sprite sprite y el valor por defecto es -1 (que representa la constante c_white, que también puede utilizarse). Cualquier otro valor (incluyendo las constantes de color internas como c_red, o c_aqua) mezclará el color especificado con el original sprite. El ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o cuando se utiliza la función layer_sprite_get_id()), y luego establece el color de mezcla a utilizar. A continuación puede ver un ejemplo de un sprite que ha sido mezclado con diferentes colores:

Tenga en cuenta que debe intentar limitar las mezclas en la plataforma HTML5 (a menos que utilice WebGL), ya que cada sprite mezclado tiene que ser almacenado en caché por separado y, por lo tanto, tener muchos sprites mezclados puede afectar negativamente al rendimiento (también puede establecer el tamaño de cache utilizando la función sprite_set_cache_size()).

Sintaxis:

layer_sprite_blend(sprite_element_id, blend);

ArgumentoTipo Descripción
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

Devuelve:

Ejemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

El código anterior obtiene el valor de ID del sprite llamado "Clouds" asignado a la capa "Asset_sky" y luego lo tiñe de un color.
