layer_sprite_exists

Puede utilizar esta función para comprobar si existe un elemento sprite en una capa determinada. Se proporciona el ID de la capa (que se obtiene al crear la capa con layer_create()) o el nombre de la capa (como string - esto tendrá un impacto en el rendimiento) y el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()) y la función devolverá true si el elemento existe, o false si no existe.

NOTA: Esta función trabaja dentro del ámbito del objetivo actual room - por defecto el room en el que se llama a la función - que puede establecerse utilizando la función layer_set_target_room().

Sintaxis:

layer_sprite_exists(layer_id, sprite_element_id)

Argumento Tipo Descripción
layer_id Cadena o ID de capa The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id ID del elemento Sprite The unique ID value of the sprite element to check

Devuelve:

Boolean

Ejemplo:

var lay_id = layer_get_id("Asset_Trees");
if (layer_sprite_exists(lay_id, global.TreesSprites))
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

El código anterior comprueba la capa "Asset_trees" para ver si el elemento sprite dado existe y si lo hace, entonces se destruye (pero no la capa).
