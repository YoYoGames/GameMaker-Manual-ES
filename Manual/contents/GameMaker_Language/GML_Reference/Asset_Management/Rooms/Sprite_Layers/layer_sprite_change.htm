cambio_de_capa_de_espectro

Con esta función se puede cambiar el recurso sprite asignado a un determinado elemento sprite en una capa. Se da el ID del elemento sprite (que se obtiene al crear un elemento sprite con layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego suministrar un elemento de índice sprite asset se cambiará. Tenga en cuenta que si el sprite tiene sub-imágenes entonces también se animará (esto puede ser controlado usando los botones layer_sprite_index() y layer_sprite_speed() ). Tenga en cuenta que puede asignar un valor de -1 como el nuevo índice sprite y no se mostrará ningún sprite, aunque el elemento seguirá existiendo y se podrá volver a cambiar más tarde.

Sintaxis:

layer_sprite_change(sprite_element_id, sprite_index)

Argumento Tipo Descripción
sprite_element_id ID del elemento Sprite The unique ID value of the sprite element to change
sprite_index Activo Sprite The new sprite index to use

Devuelve:

N/A

Ejemplo:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

El código anterior comprueba el índice sprite del elemento con el ID almacenado en la variable global global.Asset_sprite y si no es "spr_Nighttime" entonces se asigna ese sprite al elemento.
