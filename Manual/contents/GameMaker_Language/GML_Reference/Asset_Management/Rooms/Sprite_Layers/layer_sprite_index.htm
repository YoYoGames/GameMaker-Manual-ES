layer_sprite_index

Esta función se puede utilizar para establecer el índice de la imagen de un sprite asset que tiene múltiples sub-imágenes en una capa. Se da el ID del elemento sprite (que se obtiene cuando se crea un elemento sprite usando layer_sprite_create() o al utilizar la función layer_sprite_get_id()), y luego establece el índice de la imagen a utilizar. Si establece un valor fuera del rango de sub-imágenes, entonces el índice de la imagen se loop alrededor.

Sintaxis:

layer_sprite_index(sprite_element_id, image_index);

Argumento Tipo Descripción
sprite_element_id Activo Sprite The unique ID value of the sprite element to set
index Real The image index to use for the sprite

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

El código anterior obtendrá el ID de la capa llamada "sprite_trees" y luego lo utilizará para obtener el ID del elemento sprite dado en esa capa. Este ID se utiliza entonces para cambiar el índice de la imagen del elemento.
