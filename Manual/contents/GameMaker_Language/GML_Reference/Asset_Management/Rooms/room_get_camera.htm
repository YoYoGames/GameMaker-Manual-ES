room_get_camera

Con esta función puedes obtener el índice único de la cámara asignada a una vista específica en un room distinto al actual. Se da el room a utilizar, el puerto de vista a utilizar (de 0 a 7) y la función devolverá un índice de cámara.

Sintaxis:

room_get_camera(rm, vind);

Argumento Tipo Descripción
rm Activo de la habitación The index of the room to get the view camera of
vind Real The index of the view port to get the camera of

Devuelve:

Camera ID

Ejemplo:

var cam = room_get_camera(rm_Game, 0);
if (cam != global.MainCam)
{
    room_set_camera(rm_Game, 0, global.MainCam);
}

El código anterior asigna una cámara en un room recién creado al puerto de visualización [0].
