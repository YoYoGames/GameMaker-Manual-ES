room_get_name

Esta función se puede utilizar para devolver el nombre del room especificado como un string. Tenga en cuenta que esto es sólo un string y no se puede utilizar para hacer referencia al room directamente - para eso necesitaría el índice de la habitación. Sin embargo, puede utilizar este string para obtener el índice de la sala utilizando el string devuelto junto con la función asset_get_index().

Sintaxis:

room_get_name(index);

ArgumentoTipo Descripción
index The index of the room to check the name of.

Devuelve:

Ejemplo:

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

El código anterior obtendrá el nombre del room actual y lo dibujará en la pantalla.
