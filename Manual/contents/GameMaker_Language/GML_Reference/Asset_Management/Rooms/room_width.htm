ancho_de_habitación

Esta variable contiene la anchura de la actual room en píxeles. Puede modificar esta variable para cambiar la anchura de room en cualquier momento.

Sintaxis:

room_width;

Devuelve:

Ejemplo:

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

El código anterior comprueba si el cuadro delimitador de la instancia actual sprite es mayor que la anchura de room, y si lo es, mueve la instancia hacia arriba.
