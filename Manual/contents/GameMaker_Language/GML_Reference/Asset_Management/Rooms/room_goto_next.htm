room_goto_next

Con esta función puedes hacer que tu juego vaya a la siguiente que aparece en el Gestor de Salas en el momento en que se compiló el juego. Si este room no existe, se lanzará un error y el juego será forzado a cerrarse. Tenga en cuenta que el room no cambiará hasta el final del evento en el que la función fue llamada, por lo que cualquier código después de que esto haya sido llamado todavía se ejecutará si en el mismo evento.

Sintaxis:

room_goto_next();

Devuelve:

Ejemplo:

if room_exists(room_next(room))
{
    room_goto_next();
}

El código anterior comprobará si hay otro room después del actual y si es así irá a ese room.
