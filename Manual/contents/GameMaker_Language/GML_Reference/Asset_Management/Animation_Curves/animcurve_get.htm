animcurve_get

Esta función devolverá una estructura que contiene todos los datos del animation curve dado. Usted proporciona el ID de animation curve asset (como se define en el navegador Asset ), y la función devolverá un struct con las siguientes variables:

Estructura de la curva de animación
Nombre de la variable Tipo de datos Descripción
name Cadena Este es el nombre de la página web animation curve.
channels Array de Canal de la curva de animación Structs Se trata de una matriz, donde cada elemento de la matriz es un canal struct.


La variable channels en el struct es un array donde cada elemento del array es un Canal de la curva de animación Struct con datos relativos a un canal dentro de la curva. El canal struct se explica en esta página.

Al igual que en el caso de los canales, los puntos de un mismo canal se almacenan como structs en una matriz, donde cada elemento de la matriz es un único punto struct. El punto struct se explica en esta página.

Tenga en cuenta que si la función falla (por ejemplo, la Curva de Animación dada asset no existe) entonces la función devolverá -1.

Sintaxis:

animcurve_get(curve_id);

Argumento Tipo Descripción
curve_id Activo de la curva de animación The asset browser ID (index) of the animation curve to get.

Devuelve:

Animation Curve Struct or -1

Ejemplo:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
{
    _channel.type = animcurvetype_linear;
}

El código anterior recupera la animation curve struct para la curva asset "ac_ButtonTween", luego comprueba el tipo de curva para el primer canal, y si no es lineal lo establece como lineal.
