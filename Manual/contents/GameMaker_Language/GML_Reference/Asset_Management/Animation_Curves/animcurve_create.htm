animcurve_create

Esta función creará una estructura animation curve vacía, lista para que la rellenes con los datos de los canales. La función devolverá un puntero a la estructura struct y ésta se utilizará para añadir canales y otros datos a la estructura animation curve. También puede suministrarla a funciones como animcurve_get() para obtener posteriormente los datos de la curva.

Cuando se utiliza esta función, el struct creado no tendrá datos en él, y para utilizarlo se debe añadir al menos un canal y el canal debe tener puntos añadidos. Para añadir un canal, vea la función animcurve_channel_new()y para añadir puntos al canal, vea la función animcurve_point_new(). Además, puede dar un nombre a animation curve struct estableciendo la variable "name" (como se muestra en el código de ejemplo, abajo). Los canales añadidos a un nuevo animation curve deben añadirse como una matriz, donde cada elemento de la matriz hace referencia a un canal struct, y los puntos de un canal también deben añadirse como una matriz, donde cada elemento de la matriz hace referencia a un punto struct.

Ten en cuenta que los animation curves creados de esta manera deben ser eliminados cuando ya no se necesiten, ya que de lo contrario ocuparán espacio en la memoria, lo que puede provocar una fuga de memoria, ralentizando y eventualmente colapsando tu juego. Puedes eliminar un animation curve cuando ya no lo necesites utilizando la función animcurve_destroy(). No es necesario limpiar los datos de los canales o de los puntos dentro de la curva, ya que esto lo hará el recolector de basura automáticamente cuando se destruya la curva.

Sintaxis:

animcurve_create();

Devuelve:

Animation Curve Struct

Ejemplo:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

El código anterior crea un nuevo animation curve struct y lo almacena en la variable "mi_curva". Este struct se rellena entonces con un nombre y una matriz de canales. La matriz de canales contiene un único canal con tres puntos.
