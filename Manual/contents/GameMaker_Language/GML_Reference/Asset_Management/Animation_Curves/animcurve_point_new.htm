animcurve_point_new

Esta función puede utilizarse para crear una nueva estructura de puntos que se añadirá a un canal de animation curve. Cuando se crea una estructura de puntos struct está vacía y es necesario establecer las siguientes variables para generar los datos de los puntos:

Animación Curve Point Struct
Nombre de la variable Tipo de datos Descripción
posx real La posición en el tiempo (normalizada de 0 a 1) del punto.
value real El valor del punto.

Cada punto struct que cree debe añadirse a un array y éste es el array que se añade a la variable "channels" del canal animation curve struct (como se muestra en el ejemplo siguiente).

Sintaxis:

animcurve_point_new();

Devuelve:

Animation Curve Point Struct

Ejemplo:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

El código anterior crea un nuevo animation curve struct y lo almacena en la variable "mi_curva". Este struct se rellena entonces con un nombre y una matriz de canales. La matriz de canales contiene un único canal con tres puntos.
