animcurve_get_channel_index

Esta función devolverá el valor del índice para cualquier canal animation curve dado. Usted suministra el ID de animation curve o struct, donde el ID de la curva es el nombre del animation curve tal y como fue definido en el navegador Asset, o el puntero struct devuelto por la función animcurve_create(). A continuación, se proporciona el nombre del canal, como string, y la función devolverá el valor del índice asociado a ese canal. Tenga en cuenta que si la curva o el canal no existen, obtendrá un error.

Sintaxis:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Argumento Tipo Descripción
curve_struct_or_id Estructura de la curva de animación The ID or struct pointer of the animation curve to target
channel_name Cadena The channel name (a string).

Devuelve:

Real

Ejemplo:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i)
{
    _points[i].value += 1;
}

El código anterior recupera el canal struct para el canal llamado "x_pos" en la curva asset "ac_ButtonTween", luego loops a través de los puntos de la curva del canal y añade uno a su valor.
