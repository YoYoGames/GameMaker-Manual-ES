Curvas de animación

Las curvas de animación pueden ser configuradas en el Navegador de Activos como assets usando el Editor de Curvas de Animación y luego pueden ser usadas en el Editor de Secuencias para controlar las pistas en un sequence. Sin embargo, también pueden ser usadas en tu código para cambiar los valores en el tiempo en tus juegos para cualquier cosa, desde efectos de interpolación en los botones, hasta el movimiento, el cambio de color y más. También puedes crear tu propio animation curves en runtime utilizando las funciones que se enumeran aquí, lo que las convierte en una herramienta muy flexible para tus juegos.

Las funciones disponibles para las Curvas de Animación son las siguientes:

  • animcurve_get
  • animcurve_get_channel_index
  • animcurve_get_channel
  • animcurve_channel_evaluate
  • animcurve_create
  • animcurve_exists
  • animcurve_channel_new
  • animcurve_point_new
  • animcurve_destroy
