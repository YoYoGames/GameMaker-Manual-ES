animcurve_get_channel

Esta función devolverá la estructura que contiene los datos del canal especificado en un animation curve asset. Usted suministra el ID de animation curve o struct así como el nombre o índice del canal, y la función devolverá un struct con el siguiente formato:

Nombre de la variable Tipo de datos Descripción
name cadena Este es el nombre del canal.
type Animación Curva Interpolación Tipo Constante Será una de las constantes animcurvetype_linear para la interpolación lineal entre puntos, animcurvetype_catmullrom para la interpolación "suave" entre los puntos utilizando la interpolación catmull-rom, o animcurvetype_bezier para la interpolación Bezier.
iterations entero Si el canal está utilizando interpolación catmull-rom ("suave"), este valor contiene cuántos puntos se han generado para cada segmento de la curva (tenga en cuenta que estos puntos adicionales son internos a la función y sólo se utilizan para los cálculos de runtime ). Si el canal utiliza la interpolación lineal, este valor seguirá existiendo pero puede ser ignorado ya que no tiene relación con la forma en que se interpola la curva.
points puntero de matriz Se trata de un array, donde cada elemento del array es un punto struct.


El ID de animation curve es el nombre del animation curve tal y como se definió en el navegador Asset, o el puntero struct devuelto por la función animcurve_create(). El nombre del canal es un string que se refiere al canal tal y como fue definido en la Curva de Animación asset, o puede suministrar un valor de índice, que va de 0 a n, donde n es el último canal de la curva asset (por ejemplo: si un animation curve tiene 4 canales, serán indexados de 0 a 3). Tenga en cuenta que pasar un valor de índice requerirá menos procesamiento que pasar un nombre de canal. Si la función falla (es decir, no existe ningún canal con el nombre o el índice dado), obtendrá un error.

Los puntos de un mismo canal se almacenan como structs en una matriz, donde cada elemento de la matriz es un único punto struct. El punto struct tiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
posx real La posición en el tiempo (normalizada de 0 a 1) del punto.
value real El valor del punto.

Sintaxis:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Argumento Tipo Descripción
curve_struct_or_id Estructura de la curva de animación The ID or struct pointer of the animation curve to target
channel_name_or_index Cadena o Real The channel name (a string) or the channel index (an integer).

Devuelve:

Animation Curve Channel Struct

Ejemplo:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

El código anterior recupera el canal struct para el canal 0 en la curva asset "ac_ButtonTween", luego loops a través de los puntos de la curva del canal y añade uno a su valor.
