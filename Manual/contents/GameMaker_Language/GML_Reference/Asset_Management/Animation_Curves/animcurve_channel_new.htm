animcurve_channel_new

Esta función permite crear una nueva estructura de canal animation curve. Un canal struct tiene las siguientes variables:

Canal de la curva de animación Struct
Nombre de la variable Tipo de datos Descripción
nombre cadena Este es el nombre del canal.
tipo Animación Curva Interpolación Tipo Constante Será una de las constantes indicadas en la tabla siguiente.
iteraciones entero Si el canal está utilizando interpolación catmull-rom ("suave"), este valor contiene cuántos puntos se han generado para cada segmento de la curva (tenga en cuenta que estos puntos adicionales son internos a la función y sólo se utilizan para los cálculos de runtime ). Si el canal utiliza la interpolación lineal, este valor seguirá existiendo pero puede ser ignorado ya que no tiene relación con la forma en que se interpola la curva.
puntos matriz Se trata de un array, donde cada elemento del array es un Estructura de puntos de la curva de animación.

Animación Curva Interpolación Tipo Constante
Constante Descripción
animcurvetype_linear Se utiliza para la interpolación lineal entre puntos.
animcurvetype_catmullrom Se utiliza para la interpolación suave entre los puntos utilizando la interpolación de catmull-rom.
animcurvetype_bezier Se utiliza para la interpolación Bezier entre puntos.


Por defecto, cuando se crea un nuevo canal struct, la variable "nombre" será un string vacío, el "tipo" será animcurvetype_linear y las "iteraciones" serán 16. Todas estas variables pueden ser ajustadas a los valores que usted desee (tenga en cuenta que el valor de "iteraciones" no tiene efecto en los tipos de curva lineal). Una vez creada la curva, hay que añadir puntos a la misma, lo que se hace añadiendo diferentes puntos structs a la matriz de"puntos". Estos puntos structs pueden crearse utilizando la función animcurve_point_new().

Sintaxis:

animcurve_channel_new();

Devuelve:

Animation Curve Channel Struct

Ejemplo:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

El código anterior crea un nuevo animation curve struct y lo almacena en la variable "mi_curva". Este struct se rellena con un nombre y una matriz de canales. La matriz de canales contiene un único canal con tres puntos.
