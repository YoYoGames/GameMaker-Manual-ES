animcurve_exists

Esta función devolverá true si el Activo de la curva de animación o Estructura de la curva de animación existe y es una Curva de Animación válida, o false si no lo es.

Sintaxis:

animcurve_exists(curve_struct_or_id);

Argumento Tipo Descripción
curve_struct_or_id Estructura de la curva de animación o Activo de la curva de animación The Animation Curve that will be checked.

Devuelve:

Boolean

Ejemplo:

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

El código anterior comprueba si la Curva de Animación almacenada en la variable personalizada spring_curve existe. Si lo hace, ejecuta un método personalizado para utilizar esa Curva de Animación.
