animcurve_channel_evaluate

Esta función se puede utilizar para obtener el valor en un punto específico en el tiempo de una estructura de canal. Se suministra el puntero struct para el canal (tal como lo devuelve la función animcurve_get_channel()o como se devuelve en animation curve struct desde la función animcurve_get()) y la posición "x" (tiempo) a lo largo del canal de la curva a evaluar. Esta posición debe estar entre 0 y 1, y la función devolverá el valor de la curva en esa posición, o devolverá 0 si el canal struct suministrado no es válido.

Sintaxis:

animcurve_channel_evaluate(channel_struct, posx);

Argumento Tipo Descripción
channel_struct Canal de la curva de animación Struct The struct pointer for the channel to evaluate.
posx Real The position in time to check (from 0 to 1).

Devuelve:

Real

Ejemplo:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

El código anterior obtiene el canal struct para el animation curve asset "ac_AlphaCurve". A continuación, utiliza el valor de evaluación devuelto para establecer el alfa de la imagen para la instancia.
