sprite_flush_multi

Con esta función puede eliminar de la memoria de textura cualquier número de páginas de textura para el sprites dado. Se suministran los índices de sprite como una matriz y las páginas de textura a las que están asignadas se eliminarán de la memoria de textura. Tenga en cuenta que si una de las páginas de textura se utiliza en otra parte de room (por otra instancia sprite o un fondo, etc...) se puede obtener un pequeño impacto en el rendimiento ya que la página se vuelve a cargar inmediatamente en la memoria, por lo que se debe tener cuidado al utilizar esta función. Tenga en cuenta que la función devolverá -1 si la descarga no es compatible con los recursos elegidos, o devolverá 0 si todo ha funcionado correctamente.

Sintaxis:

sprite_flush_multi(array);

ArgumentoTipo Descripción
array Array with the sprite indices to flush

Devuelve:

(-1 or 0)

Ejemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

El código anterior crea una matriz en la que cada elemento contiene un índice sprite. Esta matriz se utiliza entonces para borrar esas texturas de sprite de la memoria.
