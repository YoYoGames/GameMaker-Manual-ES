sprite_delete

Esta función borrará un sprite del juego, liberando la memoria que estaba reservada para él. Esta es una función muy importante para aquellos momentos en los que se necesita crear y cambiar sprites desde fuentes externas (como cargar un sprite desde un archivo con sprite_add() o duplicar un sprite usando sprite_duplicate()) y siempre debe usarse para eliminar aquellos assets que ya no son que ya no necesita un juego, o para borrar un asset indexado de una variable antes de reasignar otro asset a esa variable. Esto no se puede utilizar para eliminar sprites que se incluyen en el juego como parte de la assets en el navegador Asset .

Sintaxis:

sprite_delete(index);

ArgumentoTipo Descripción
index The index of the sprite to be deleted.

Devuelve

Ejemplo:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

El código anterior crea una variable local y luego almacena el índice del sprite creado a partir de la superficie de la aplicación. Este sprite se fusiona entonces con el asset indexado en la variable "spr_Player" antes de ser eliminado de nuevo de la memoria.
