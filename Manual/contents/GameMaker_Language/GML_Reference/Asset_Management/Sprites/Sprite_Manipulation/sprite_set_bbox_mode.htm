sprite_set_bbox_mode

Esta función puede utilizarse para establecer el modo del cuadro delimitador para un sprite. Usted proporciona el índice sprite y el modo a utilizar, que debe ser una de las siguientes constantes:

Constante del modo de caja delimitadora
Constante Descripción
bboxmode_automatic Automático - El cuadro delimitador se calculará automáticamente, basándose en la configuración de la tolerancia del sprite
bboxmode_fullimage Imagen completa: el cuadro delimitador se establecerá para utilizar toda la anchura y la altura de la página sprite, independientemente de la tolerancia y de los píxeles "vacíos".
bboxmode_manual Manual - El cuadro delimitador se ajusta manualmente a los valores definidos por el usuario, lo que requiere el uso de la función sprite_set_bbox()

NOTA Esta función afecta al activo de sprite para que todas las instancias posteriores con este sprite tengan el mismo modo de caja delimitadora.

Sintaxis:

sprite_set_bbox_mode(ind, mode);

Argumento Tipo Descripción
ind Activo Sprite The index of the sprite to change the mode of.
mode Constante del modo de caja delimitadora The mode to set (a constant).

Devuelve

N/A

Ejemplo:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

El código anterior comprueba el modo bbox para el actual sprite y si no es automatic, entonces se establece en ese valor.
