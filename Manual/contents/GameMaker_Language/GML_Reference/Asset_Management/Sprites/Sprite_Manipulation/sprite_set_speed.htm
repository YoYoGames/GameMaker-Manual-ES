sprite_set_speed

Esta función se puede utilizar para establecer la velocidad y el tipo de animación base como para el recurso sprite. Esto se establece normalmente en el Editor de Sprite, pero puede haber momentos en los que se requiera establecer esto manualmente para un recurso sprite dado. Necesitas suministrar el índice de sprite a establecer, la velocidad de la animación y el tipo de velocidad de la animación a aplicar. El tipo de velocidad de animación puede ser cualquiera de las siguientes constantes:

Constante Descripción
spritespeed_framespersecond El sprite se definió con la animación en cuadros por segundo.
spritespeed_framespergameframe El sprite se definió con en la animación en fotogramas por cuadro de juego.


La selección del tipo de velocidad que se va a utilizar para calcular las animaciones es muy importante, ya que hay una gran diferencia entre las dos opciones. Los dos ejemplos siguientes ilustran esta diferencia:

  • Si tienes un sprite que dibuja 1 fotograma por segundo y estableces la velocidad de la imagen en 0,5, dibujará a 0,5 fotogramas por segundo. Si la velocidad de fotogramas de tu juego es de 60 fotogramas por segundo, entonces el sprite dibujará 1 fotograma por cada 120 fotogramas del juego.
  • Si tienes un sprite que dibuja 1 fotograma por fotograma del juego y estableces la velocidad de la imagen en 0,5, dibujará 0,5 fotogramas por fotograma del juego. Si la velocidad de fotogramas de tu juego es de 60 fotogramas por segundo, el sprite dibujará 30 fotogramas por cada 60 fotogramas del juego.

Puedes averiguar el valor de la velocidad utilizada para la animación sprite utilizando la función sprite_get_speed()y puedes obtener el tipo de velocidad de la animación usando sprite_get_speed_type().

¡IMPORTANTE! Esta función afectará al recurso sprite, por lo que cualquier cambio que realice con esta función afectará a todas las instancias que estén utilizando el sprite y también a todas las instancias futuras.

Sintaxis:

sprite_set_speed(index, speed, type)

Argumento Tipo Descripción
index Activo Sprite The index of the sprite to set the animation speed of
speed Real The speed value to set
type Constante de velocidad del sprite The speed type to use

Devuelve:

Constant (see above)

Ejemplo:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

El código anterior establece la velocidad de animación base para el sprite asignado a la instancia a 10 cuadros por segundo.
