sprite_flush

Con esta función puede eliminar de la memoria de textura la página de textura dada para el sprite dado. Se suministra el índice sprite (tal y como se definió al crear el recurso sprite ) y la página de textura a la que está asignada se eliminará de la memoria de textura. Ten en cuenta que si la página de textura se utiliza en otra parte de room (por otra instancia sprite o un fondo, etc...) puedes obtener un pequeño impacto en el rendimiento ya que la página se recarga inmediatamente, por lo que hay que tener cuidado al utilizar esta función. Tenga en cuenta que la función devolverá -1 si la descarga no es compatible con el recurso elegido, o devolverá 0 si todo ha funcionado correctamente.

Sintaxis:

sprite_flush(ind)

ArgumentoTipo Descripción
ind The index (resource name) of the sprite asset to flush

Devuelve:

(-1 or 0)

Ejemplo:

sprite_flush(spr_Player_Aura);

El código anterior borra el sprite "spr_Player_Aura" de la memoria.
