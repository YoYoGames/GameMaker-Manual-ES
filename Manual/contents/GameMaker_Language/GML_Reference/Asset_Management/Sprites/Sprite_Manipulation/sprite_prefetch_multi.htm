sprite_prefetch_multi

Esta función se puede utilizar para precargar (colocar en la memoria de la textura) un número de páginas de la textura que contienen el sprites dado. Se suministra un array con los índices de sprite (definidos al crear sprite asset) y las páginas de textura en las que se encuentran se cargarán en la memoria. Tenga en cuenta que la función devolverá -1 si la precarga no es compatible con el recurso elegido o la plataforma de destino es HTML5, o devolverá 0 si todo ha funcionado correctamente.

NOTA: Hay un impacto en el rendimiento ya que la textura se carga en la memoria de la textura en la mayoría de los dispositivos, por lo que se recomienda que usted llame a sprite_prefetch_multi() al inicio de un nivel para evitar que se detenga.

Sintaxis:

sprite_prefetch_multi

Argumento Tipo Descripción
array Conjunto de activos de sprites Array with the sprite indices to fetch

Devuelve:

Real (-1 or 0)

Ejemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

El código anterior crea una matriz en la que cada elemento contiene un índice sprite. Esta matriz se utiliza entonces para colocar esas texturas de sprite en la memoria.
