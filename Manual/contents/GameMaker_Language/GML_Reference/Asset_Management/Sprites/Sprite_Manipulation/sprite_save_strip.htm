sprite_save_strip

Esta función creará una tira de imágenes a partir de todas las subimágenes de un sprite, guardándola en el disco con el nombre de archivo especificado. El sprite debe haber sido añadido en runtime (no se puede guardar sprites añadido a través del IDE) y el archivo debe ser guardado con un *.png extension.

NOTA: Dependiendo de la plataforma de destino que se elija, usted está limitado en cuanto a dónde puede guardar y cargar archivos. Consulte la sección Manejo de archivos para obtener más información.

Sintaxis:

sprite_save_strip(ind, filename);

Argumento Tipo Descripción
ind Activo Sprite The index of the sprite to save.
filename Cadena The filename for the saved sprite strip.

Devuelve

N/A

Ejemplo:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

El código anterior crea un sprite a partir de la superficie indexada en la variable "surf", asignando su índice a la variable local "spr_Custom", y luego utiliza un for loop para desplazarse por la superficie y capturar varias secciones que se añaden en el sprite como subimágenes. este nuevo sprite se guarda entonces como una tira de png antes de ser eliminado de la memoria.
