sprite_existe

Esta función devuelve si un sprite con el índice especificado existe o no en el proyecto que se está ejecutando.

Sintaxis:

sprite_exists(index);

ArgumentoTipo Descripción
index The index of the sprite to check.

Devuelve

Ejemplo:

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

El código anterior comprueba un array para ver si contiene un índice sprite válido y si es así asigna ese sprite a la instancia, pero si no, asigna un sprite de los recursos incluidos.
