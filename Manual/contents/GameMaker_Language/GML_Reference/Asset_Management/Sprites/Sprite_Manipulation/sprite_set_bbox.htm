sprite_set_bbox

Esta función puede utilizarse para establecer los valores del cuadro delimitador de un sprite. Se suministra el índice sprite que se va a utilizar y, a continuación, los valores izquierdo, superior, derecho e inferior para las posiciones del cuadro delimitador. Las posiciones son valores absolutos, en los que la posición (0, 0) corresponde a la esquina superior izquierda de sprite, independientemente del desplazamiento de sprite, de los píxeles "vacíos" que pueda tener sprite o del lugar en el que se esté dibujando en room.

NOTA Esta función afecta al activo sprite para que todas las instancias posteriores con este sprite tengan el mismo cuadro delimitador.

Sintaxis:

sprite_set_bbox(ind, left, top, right, bottom);

ArgumentoTipo Descripción
ind The index of the sprite to set the bounding box on.
left The left side of the bounding box
top The top of the bounding box.
right The right side of the bounding box
bottom The bottom of the bounding box.

Devuelve

Ejemplo:

if sprite_get_bbox_mode(sprite_index) == 2
{
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
}

El código anterior comprobará el modo de caja delimitadora de la sprite asignada a la sprite_index, y si se establece en manual entonces tendrá su caja delimitadora cambiada.
