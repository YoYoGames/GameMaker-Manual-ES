sprite_duplicate

Esta función devolverá el índice de un sprite recién creado que es un duplicado (copia) del introducido como argumento "índice". Si la función falla, se devuelve -1. Esta función debe utilizarse para copiar cualquier sprites del assets original antes de poder realizar cualquier transformación en él. Por ejemplo, si desea cambiar el cuadro delimitador de un sprite, o establecer su alfa desde otro sprite, primero debe duplicarlo, luego realizar la operación en el sprite duplicado y utilizarlo. Un sprite duplicado será colocado en su propia y única página de textura cuando sea creado, lo que significa que duplicar múltiples sprites creará múltiples páginas de textura y tendrá un impacto en el rendimiento, así que utilice esta función sólo cuando sea necesario.

NOTA: Cuando duplica un sprite en GameMaker debe recordar eliminarlo de nuevo (con sprite_delete()) cuando ya no lo necesites, de lo contrario existe el riesgo de que se produzca una fuga de memoria que ralentizará y eventualmente bloqueará tu juego.

Sintaxis:

sprite_duplicate(index);

ArgumentoTipo Descripción
index The index of the sprite to duplicate.

Devuelve

Sprite Asset or -1

Ejemplo:

new_spr = sprite_duplicate(sprite_index)

El código anterior duplica el sprite que se utiliza actualmente como índice de la instancia sprite y almacena el índice de este nuevo sprite en la variable "new_spr".
