sprite_assign

Esta función toma dos índices sprite previamente creados (o incluidos), y copia la imagen de uno a otro. De este modo, puede copiar (o "clonar") un sprite en otro índice. Ten en cuenta que no puedes copiar en un recurso del juego. Tienes que haber creado previamente el sprite al que se va a copiar utilizando el botón sprite_add() o sprite_duplicate() para copiarlo.

Sintaxis:

sprite_assign(index, sprite);

ArgumentoTipo Descripción
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

Devuelve

Ejemplo:

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

El código anterior comprueba si la variable global "Player_Sprite" contiene un sprite y si lo contiene utiliza sprite_assign para cambiarlo por uno que haya sido cargado desde un archivo externo. Si no contiene un sprite se carga uno y su índice se almacena en esa variable.
