sprite_set_nineslice

Esta función se utiliza para aplicar una estructura N ine Slice a un sprite. Se suministra el índice sprite y la Nine Slice struct que se va a aplicar, que se puede recuperar con sprite_nineslice_create() o sprite_get_nineslice().

NOTA Esta función afecta al activo de sprite, por lo que cualquier cambio que se haga con esta función afectará a todas las instancias que estén usando este sprite y también a todas las instancias futuras.

Sintaxis:

sprite_set_nineslice(ind, nineslice);

Argumento Tipo Descripción
ind Activo Sprite The index of the sprite to modify
nineslice Estructura de nueve rodajas The Nine Slice struct to apply

Devuelve:

N/A

Ejemplo:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

El código anterior crea una nueva Nine Slice struct, la habilita y establece los desplazamientos de las guías. El struct se aplica entonces a un sprite, cambiando sus propiedades de Nine Slice.
