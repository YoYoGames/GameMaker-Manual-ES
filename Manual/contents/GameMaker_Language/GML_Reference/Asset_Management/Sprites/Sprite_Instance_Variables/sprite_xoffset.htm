sprite_xoffset

Esta variable de sólo lectura devuelve el xoffset local (el componente x del origen definido en el editor sprite ) de la sprite que se ha asignado a la instancia. Este xoffset se devuelve en píxeles y dependerá del image_xscalede modo que (por ejemplo) si un cuadrado de 32x32 sprite se define con el origen x en 16, cuando ese sprite se escala en 2 esta variable devolverá 32. Si necesita el origen x sin escalar, deberá utilizar sprite_get_xoffset().

Sintaxis:

sprite_xoffset;

Devuelve:

Real

Ejemplo:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

El código anterior comprueba si el borde izquierdo de la sprite de la instancia está fuera de la room y, si lo está, mueve la instancia para que todo el sprite sea visible.
