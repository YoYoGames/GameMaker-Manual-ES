sprite_yoffset

Esta variable de sólo lectura devuelve el yffset local (el componente y del origen definido en el editor sprite ) de la sprite que se ha asignado a la instancia. Este yffset se devuelve en píxeles y dependerá del image_yscalede modo que (por ejemplo) si un cuadrado de 32x32 sprite se define con el origen y en 16, cuando ese sprite se escala en 2 esta variable devolverá 32. Si necesita el origen y sin escalar, deberá utilizar sprite_get_yoffset().

Sintaxis:

sprite_yoffset;

Devuelve:

Real

Ejemplo:

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

El código anterior comprueba si el borde superior de la sprite de la instancia está fuera de la room y, si lo está, mueve la instancia para que todo el sprite sea visible.
