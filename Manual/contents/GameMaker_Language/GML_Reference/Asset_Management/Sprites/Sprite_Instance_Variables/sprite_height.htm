sprite_height

Esta variable de sólo lectura devuelve la altura de la sprite que se ha asignado a la instancia. Esta altura se devuelve en píxeles y dependerá de la variable image_yscale. Si necesita la altura sin escalar debe utilizar sprite_get_height().

Sintaxis:

sprite_height

Devuelve:

Real

Ejemplo:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

El código anterior comprueba la altura de la sprite asignada a la instancia que ejecuta el código con la altura del recurso sprite y si no es la misma, restablece la image_yscale a 1,
