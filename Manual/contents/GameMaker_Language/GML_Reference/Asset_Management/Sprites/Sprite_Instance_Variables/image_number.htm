número_de_imagen

Esta variable de sólo lectura se puede utilizar para obtener el número de subimágenes en un sprite que ha sido asignado a una instancia (si necesita el número de subimágenes para un sprite distinto del asignado a la instancia debe utilizar sprite_get_number()). Tenga en cuenta que cuando hay (por ejemplo) 1 sub-imagen esta variable devolverá "1" pero el image_index de esa sub-imagen es 0.

Sintaxis:

image_number;

Devuelve:

(single precision floating point value)

Ejemplo:

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

El código anterior comprueba si el sprite está animado, y si lo está, comprueba si el image_index actual es mayor que el número de subimágenes y si lo es, destruye la instancia.
