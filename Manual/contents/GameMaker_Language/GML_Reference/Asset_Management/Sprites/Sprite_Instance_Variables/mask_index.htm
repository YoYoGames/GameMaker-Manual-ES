índice_de_máscara

Cuando se define un object en GameMaker se le puede asignar una máscara que se utilizará para las colisiones en lugar de la que corresponde al sprite definido. Esta variable se puede utilizar para encontrar el sprite_index de esa máscara (o devolverá -1 si no se ha asignado ninguna sprite ) o para establecer la máscara de una instancia a la sprite elegida. Establecer el índice de la máscara significa que se puede tener, por ejemplo, un sprite para la instancia con una forma irregular, pero darle una máscara de colisión circular que se obtiene de un sprite diferente.

Sintaxis:

mask_index;

Devuelve:

Sprite Asset

Ejemplo:

mask_index = spr_Round;

El código anterior establece la máscara de la instancia a la de sprite "spr_Round".
