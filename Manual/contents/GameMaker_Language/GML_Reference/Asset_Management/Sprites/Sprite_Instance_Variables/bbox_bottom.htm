bbox_bottom

Esta variable de sólo lectura devuelve la posición y (dentro de room) de la parte inferior del cuadro delimitador para la instancia, donde el cuadro delimitador está definido por la anchura y la altura máximas de la máscara para la instancia (según lo establecido por sprite_index o por mask_index). Incluso cuando un sprite tiene una máscara de colisión precisa, el cuadro delimitador existe y se utiliza para ciertas cosas, por lo que se puede utilizar esta variable para encontrarlo. Tenga en cuenta que cuando la instancia no tiene asignado sprite el valor devuelto será el mismo que la posición Y de la instancia.

Sintaxis:

bbox_bottom;

Devuelve:

(integer)

Ejemplo:

if bbox_bottom > room_height
{
    y = room_height - sprite_yoffset;
}

El código anterior comprueba si el cuadro delimitador de la instancia está fuera de room y, si lo está, coloca la instancia en una nueva posición.
