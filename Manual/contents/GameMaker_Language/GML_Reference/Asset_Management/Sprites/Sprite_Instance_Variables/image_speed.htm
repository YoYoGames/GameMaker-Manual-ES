velocidad_de_imagen

Esta variable determina la velocidad en la que GameMaker hará un ciclo a través de las sub-imágenes para la instancia actual sprite. El valor de la velocidad dada es un multiplicador, siendo 1 el valor por defecto, y estableciéndolo a 0.5 se reducirá a la mitad la velocidad de la animación - como se establece en el Editor de Sprite o el Editor de Imágenes - mientras que estableciéndolo a 2 se duplicará. Si el sprite utilizado no tiene subimágenes, esta variable no tendrá ningún efecto.

Sintaxis:

image_speed;

Devuelve:

(single precision floating point value)

Ejemplo:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

El código anterior crea una instancia de object "obj_Explosion" y establece su image_speed en 0,5 (reduciendo efectivamente la velocidad de ejecución a la mitad).
