image_angle

Este valor establece el ángulo (rotación) de la sprite y se mide en grados, siendo la derecha 0º, arriba 90º, izquierda 180º y abajo 270º. Establezca esta variable a 0 para que el sprite se dibuje como se definió en el editor de sprite. Por favor, tenga en cuenta que para que los cambios en esta variable sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker dibujará por defecto el sprite) o se dibujará usando una de las funciones de dibujo extendidas como draw_self() o draw_sprite_ext().

Sintaxis:

image_angle

Devuelve:

Real

Ejemplo:

image_angle = point_direction(x, y, mouse_x, mouse_y);

El código anterior girará la sprite de la instancia para que apunte siempre a la posición del ratón.
