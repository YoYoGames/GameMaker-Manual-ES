sprite_index

Esta variable devuelve el índice del sprite actual para la instancia, o -1 si la instancia no tiene ningún sprite asociado. Puedes cambiarla para dar a la instancia un sprite diferente dándole el nombre de un sprite del árbol de recursos o utilizando una variable que tenga indexado un sprite cargado externamente. Cambiar el sprite no cambia el índice del fotograma actualmente visible, así que si cambias el sprite en el fotograma número 3, el nuevo sprite se dibujará con ese fotograma visible (asumiendo que tiene el mismo número de fotogramas).

Sintaxis:

sprite_index

Devuelve:

Sprite Asset

Ejemplo:

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

El código anterior loop a través de todas las instancias de "obj_Check" comprobando si hay una línea de colisión entre ellas y la instancia que ejecuta el código. El sprite de esas instancias se cambiará dependiendo del valor de retorno (verdadero o falso) para la línea de colisión.
