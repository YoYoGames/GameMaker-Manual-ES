ancho_de_sprites

Esta variable de sólo lectura devuelve el ancho de la sprite que se ha asignado a la instancia. Esta anchura se devuelve en píxeles y dependerá de la variable image_xscale. Si necesita la anchura sin escalar debe utilizar sprite_get_width().

Sintaxis:

sprite_width;

Devuelve:

Real

Ejemplo:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

El código anterior comprueba la anchura de la sprite asignada a la instancia que ejecuta el código frente a la anchura del recurso sprite y, si no es la misma, restablece la image_xscale a 1,
