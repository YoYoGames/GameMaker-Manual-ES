imagen_alfa

Esta variable se utiliza para obtener o establecer el valor alfa para el sprite. El alfa siempre se calcula como un valor entre 0 y 1, donde 0 es completamente transparente y 1 es completamente opaco. Por favor, tenga en cuenta que para que los cambios en esta variable sean visibles, la instancia debe no tener ningún evento de dibujo (y así GameMaker dibujará por defecto el sprite) o ser dibujado usando una de las funciones de dibujo extendidas como draw_self() o draw_sprite_ext().

Sintaxis:

image_alpha;

Devuelve:

Real

Ejemplo:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

El código anterior reducirá lentamente el image_alpha hasta que llegue a 0.
