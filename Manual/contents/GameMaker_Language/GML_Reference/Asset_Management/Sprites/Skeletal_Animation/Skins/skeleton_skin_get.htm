skeleton_skin_get

Con la animación esquelética sprites, puede asignar texturas separadas (llamadas "pieles") a la animación, utilizando así una animación para múltiples cosas diferentes. Esta función devolverá el nombre de la piel (como string) que está actualmente asignada a la animación esquelética sprite que su instancia está utilizando. El nombre devuelto es el que usted estableció cuando creó el sprite en su programa de animación.

Sintaxis:

skeleton_skin_get();

Devuelve:

Ejemplo:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

El código anterior comprobaría la piel actualmente asignada para la animación y si es "skin_Enemy1", elige y establece una nueva piel de una de las tres opciones.
