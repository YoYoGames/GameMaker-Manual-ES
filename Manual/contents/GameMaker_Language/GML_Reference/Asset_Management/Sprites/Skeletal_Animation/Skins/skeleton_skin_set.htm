skeleton_skin_set

Una animación esquelética sprite puede ser dibujada con una variedad de texturas (llamadas "skins"), permitiéndole usar una sola animación para múltiples propósitos diferentes. Por ejemplo, puedes tener una animación genérica de caminar y ésta puede ser utilizada por tus personajes NPC, enemigos, etc... en un juego simplemente cambiando el skin con el que se dibuja. Estas pieles necesitan haber sido creadas previamente y nombradas en su programa de animación, y este nombre se pasa a la función (como string) para establecer que sprite use esa piel cuando se dibuje.

Sintaxis:

skeleton_skin_set(skinname);

ArgumentoTipo Descripción
skinname The name (a string) of the skin to use.

Devuelve:

Ejemplo:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

El código anterior comprobaría el skin asignado actualmente para la animación y si es "skin_Enemigo1", elige y establece un nuevo skin de una de las tres opciones.
