Pieles

Cada animación esquelética puede tener una o más pieles. Esto significa que puedes usar la misma animación base con diferentes gráficos para crear 2 o más personajes. Por ejemplo, digamos que tienes un enemigo que camina, corre y lucha y quieres tener una forma masculina y femenina para él. En lugar de crear dos sprites, crearías uno y luego tendrías dos pieles y simplemente switch entre las pieles al crear el sprite para elegir la forma masculina o femenina.

Las siguientes funciones sirven para tratar las pieles de animación del esqueleto:

  • skeleton_skin_get
  • skeleton_skin_set
  • skeleton_skin_list
