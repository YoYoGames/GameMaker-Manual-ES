skeleton_animation_get_event_frames

Esta función puede utilizarse para recuperar todos los fotogramas del evento dado, en la animación dada. Se suministra el nombre de la animación (como string, como se define en el programa utilizado para hacer la animación, o como se devuelve utilizando la función skeleton_animation_get()) y un nombre de evento. La función devuelve un array con los fotogramas de ese evento.

Si el nombre del evento especificado no existe, la función devolverá un array con -1 como primer (y único) elemento.

Sintaxis:

skeleton_animation_get_event_frames(anim_name, event_name);

Argumento Tipo Descripción
anim_name Cadena The animation name to get the frames of.
event_name Cadena The event name to get the frames for.

Devuelve:

Array of Reals

Ejemplo:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

El código anterior obtiene los fotogramas del evento "Footstep ". Si devuelve un array con los fotogramas válidos, lo recorre loops y comprueba si el fotograma actual es igual a alguno de los fotogramas del array. En ese caso, reproduce el efecto de sonido de la pisada y rompe (detiene) el loop.
