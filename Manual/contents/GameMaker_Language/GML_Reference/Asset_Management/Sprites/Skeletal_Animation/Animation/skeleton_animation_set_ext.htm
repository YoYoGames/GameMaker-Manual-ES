skeleton_animation_set_ext

Cuando estableces un set de animación para tu animación esquelética sprite, se le asigna la pista 0 internamente. Sin embargo, puedes asignar más sets a otras pistas (1 y superiores) y de esta manera hacer que tu sprite reproduzca dos (o más) animaciones al mismo tiempo. Tenga en cuenta que el sprite debe haber sido diseñado en su programa de animación con esto en mente, de tal manera que la animación "base" en la pista 0 sólo tiene keyframes para ciertos huesos, mientras que las siguientes pistas tienen keyframes para los demás. Por ejemplo, si tienes una animación de "caminar" y su keyframes sólo afecta a las piernas del sprite, puedes tener un conjunto de animaciones de la cabeza y de los brazos que puedes establecer en las pistas 1 y 2 respectivamente, permitiéndote tener un movimiento base y adaptarte a él en tiempo real mientras se desarrolla el juego.
La animación se reproduce en bucle, que es el valor por defecto.

Sintaxis:

skeleton_animation_set_ext(animname, track, [loop]);

Argumento Tipo Descripción
animname Cadena The name (a string) of the animation set to use.
track Real The track for the animation set to use.
loop Booleano Si la animación debe loop o no (por defecto es true)

Devuelve:

N/A

Ejemplo:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

El código anterior cambiará el conjunto de animación que está siendo utilizado por la pista 1 al conjunto "bodyfight" o al conjunto "bodywalk" cuando se presione el botón izquierdo del ratón.
