skeleton_animation_get_position

Esta función devolverá la posición de la animación en la pista de animación especificada como un valor que va de 0 a 1. De esta manera se puede obtener la posición de la pista como un porcentaje donde el valor 0 corresponde al 0% y el valor 1 al 100%.

Devolverá -1 si no hay ninguna animación asignada a la pista específica dada o si la instancia no tiene configurado sprite.

Sintaxis:

skeleton_animation_get_position(track);

Argumento Tipo Descripción
track Real The animation track to get the position of.

Devuelve:

Real

Ejemplo:

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

El código anterior comprueba la posición actual de la animación del esqueleto asignada a la pista 0 y la almacena en una variable temporal _position. Luego convierte ese valor en un porcentaje multiplicándolo por 100 y almacena el resultado en otra variable temporal _position_percent.
Finalmente muestra un mensaje de depuración que muestra el valor del porcentaje.
