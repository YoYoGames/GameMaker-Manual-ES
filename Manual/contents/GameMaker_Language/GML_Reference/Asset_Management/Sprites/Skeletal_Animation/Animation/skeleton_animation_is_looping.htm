skeleton_animation_is_looping

Con esta función puedes comprobar si la animación actual en la pista dada está configurada en loop.

Sintaxis:

skeleton_animation_is_looping(track);

Argumento Tipo Descripción
pista Real El índice de la pista de animación

Devuelve:

Boolean

Ejemplo:

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

El código anterior comprobará si la animación de la pista 0 está configurada en loop. Si ese es el caso, muestra un mensaje de depuración.
