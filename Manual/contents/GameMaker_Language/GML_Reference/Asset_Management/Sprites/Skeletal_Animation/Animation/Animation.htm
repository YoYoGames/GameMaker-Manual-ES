Animación

Cuando se trabaja con una animación esquelética sprites creada con Spine, se pueden tener múltiples animaciones por sprites, y como tal se necesita ser capaz de switch entre ellos en los momentos apropiados de su juego. Para ayudarle a hacer eso, GameMaker proporciona las siguientes funciones:

  • skeleton_animation_get
  • skeleton_animation_set
  • skeleton_animation_get_ext
  • skeleton_animation_set_ext
  • skeleton_animation_get_duration
  • skeleton_animation_mix
  • skeleton_animation_list
  • skeleton_animation_clear
  • skeleton_animation_get_frames
  • skeleton_animation_get_frame
  • skeleton_animation_set_frame
  • skeleton_animation_get_event_frames
  • esqueleto_animación_es_suficiente
  • esqueleto_animación_está_acabado
