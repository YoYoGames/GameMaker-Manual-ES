skeleton_animation_get_duration

Esta función devolverá el tiempo necesario para que el conjunto de animación dado se ejecute antes de volver al principio. El valor de retorno está en segundos.

Sintaxis:

skeleton_animation_get_duration(animname);

ArgumentoTipo Descripción
animname The name (a string) of the animation set to use.

Devuelve:

Ejemplo1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

El código anterior establecerá el image_index al valor correcto para la animación esquelética asignada actualmente sprite.

Ejemplo2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará la animación esquelética dada sprite usando delta-tiempo para establecer el cuadro que se dibuja.
