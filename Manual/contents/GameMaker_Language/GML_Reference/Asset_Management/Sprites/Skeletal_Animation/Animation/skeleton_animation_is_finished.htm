esqueleto_animación_está_acabado

Con esta función puedes comprobar si la animación esquelética actual que no es de bucle ha terminado de reproducirse en la pista dada.

Sintaxis:

skeleton_animation_is_finished(track);

Argumento Tipo de datos Descripción
pista Real El índice de la pista de animación

Devuelve:

Boolean

Ejemplo:

var _finished = skeleton_animation_is_finished(1);
if (_finished)
{
    show_debug_message("The animation has finished playing");
}

El código anterior comprueba primero si la animación de la pista 1 ha terminado de reproducirse y almacena el resultado en una variable local _finished. Luego muestra un mensaje de depuración en caso de que la animación haya terminado de reproducirse.
