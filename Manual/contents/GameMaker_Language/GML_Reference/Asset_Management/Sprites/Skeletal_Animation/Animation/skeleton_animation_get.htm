skeleton_animation_get

Con esta función puedes obtener el conjunto de animación actual que está siendo utilizado por tu animación esquelética sprite. El valor de retorno es un string, que será el nombre del conjunto tal y como lo has definido en tu programa de animación esquelética.

Sintaxis:

skeleton_animation_get();

Devuelve:

Ejemplo:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

El código anterior cambiará el set de animación que se está utilizando al set de "salto" cuando se pulse la tecla espacio, pero sólo si el set actual que se está utilizando no es ya "salto".
