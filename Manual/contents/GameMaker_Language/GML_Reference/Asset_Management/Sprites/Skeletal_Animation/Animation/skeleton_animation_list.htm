skeleton_animation_list

Con esta función se puede rellenar una lista DS (precreada) con todos los nombres de las animaciones incluidas como parte de la animación esquelética sprite. Los nombres serán strings y luego pueden ser utilizados en las otras funciones de animación para estos tipos de sprite.

Sintaxis:

skeleton_animation_list(sprite, list);

Argumento Tipo Descripción
sprite Activo Sprite The sprite index of the Spine skeletal animation to get the list from.
list ID de la lista DS The ID of the DS list to populate with the animation names.

Devuelve:

N/A

Ejemplo:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

El código anterior crea una lista DS con todos los nombres de las animaciones de sprite que están siendo utilizadas por la instancia. A continuación, loops a través de estos y establece el valor de la mezcla para todos ellos con la primera animación a 0,5.
