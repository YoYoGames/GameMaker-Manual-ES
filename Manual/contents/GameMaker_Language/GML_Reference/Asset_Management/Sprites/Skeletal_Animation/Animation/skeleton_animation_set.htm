skeleton_animation_set

Una vez que hayas asignado una animación esquelética sprite a tu instancia, puedes definir qué conjunto de animación debe utilizar en cada momento con esta función. Cuando creaste tu sprite (en Spine) habrás definido múltiples sets de animación para diferentes estados y habrás nombrado cada uno de ellos apropiadamente. Estos nombres son los que usarás para decirle a GameMaker que conjunto de animación debe usar en cualquier momento en tu juego. Por ejemplo, si tu sprite es para un juego de plataformas, entonces puedes tener un set de "salto", un set de "carrera" y un set de "inactividad", todos contenidos en el mismo sprite y puedes switch entre ellos en tu juego usando esta función.
La animación se reproduce en bucle, que es el valor por defecto.

Tenga en cuenta que al establecer una nueva animación de esta manera se restablecerá el image_index a 0, comenzando el nuevo conjunto de animación desde el principio de su animación loop. Si este no es el comportamiento deseado, entonces puede establecer un valor de "mezcla" usando skeleton_animation_mix() y cuando establezcas una nueva animación se interpolará suavemente con el conjunto anterior. También debes asegurarte de tener un keyframe al principio de cada animación para evitar que los datos se retengan de animaciones anteriores, a menos que planees usar pistas de animación separadas.

Sintaxis:

skeleton_animation_set(animname, [loop]);

Argumento Tipo Descripción
animname Cadena The name (a string) of the animation set to use.
loop Booleano Si la animación debe loop o no (por defecto es true)

Devuelve:

N/A

Ejemplo:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

El código anterior cambiará el set de animación que se está utilizando al set de "salto" cuando se pulse la tecla espacio, pero sólo si el set actual que se está utilizando no es ya "salto".
