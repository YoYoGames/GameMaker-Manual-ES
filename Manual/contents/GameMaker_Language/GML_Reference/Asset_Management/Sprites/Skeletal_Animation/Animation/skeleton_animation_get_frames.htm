skeleton_animation_get_frames

Esta función puede utilizarse para obtener el número de fotogramas que tiene una animación de esqueleto determinada. Se suministra el nombre de la animación del esqueleto (como string, como se define en el programa utilizado para hacer la animación, o como se devuelve utilizando la función skeleton_animation_get()) y la función devuelve los fotogramas que tiene como un valor entero. La función devolverá 0 si la animación especificada no existe.

Sintaxis:

skeleton_animation_get_frames(anim_name);

Argumento Tipo Descripción
anim_name Cadena The animation name to get the frames of.

Devuelve:

Real

Ejemplo:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

El código anterior obtendrá el número de fotogramas de la animación y luego establecerá el sprite en el último fotograma y dejará de animar.
