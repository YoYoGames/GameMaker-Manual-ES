skeleton_slot_alpha_get

Con esta función se puede obtener el valor alfa de una ranura de fijación. Usted proporciona el nombre de la ranura (un string) y la función devolverá un valor real para el alfa entre 0 y 1.

Sintaxis:

skeleton_slot_alpha_get(slot);

Argumento Tipo Descripción
slot Cadena The slot name to check, a string

Devuelve:

Real

Ejemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

El código anterior crea una lista DS y luego la rellena con los nombres de las ranuras encontradas en la posición del ratón en el room. Luego loops a través de la lista de ranuras y restablece el color de las ranuras encontradas a blanco con un alfa de 1 si el alfa no es ya 1.
