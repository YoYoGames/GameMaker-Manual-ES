skeleton_slot_colour_set

Con esta función puede establecer el color y el alfa de una ranura de adjunto para que todos los sprites adjuntos a ella se dibujen con estos valores de mezcla. Tenga en cuenta que la instancia sprite puede tener un color de mezcla y un ajuste alfa (image_blend y image_alpha), al igual que el archivo adjunto (véase la función skeleton_attachment_create_colour()), por lo que el color y el alfa finales que tendrá el adjunto asignado sprite para la ranura serán un compuesto de todos estos valores.

Sintaxis:

skeleton_slot_colour_set(slot, colour, alpha);

Argumento Tipo Descripción
slot Cadena The slot name to set, a string
colour Color The colour to set, either an integer, a constant, or a hex value.
alpha Real The alpha to set from 0 to 1.

Devuelve:

N/A

Ejemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

El código anterior crea una lista DS y luego la rellena con los nombres de las ranuras encontradas en la posición del ratón en el room. Luego loops a través de la lista de ranuras y restablece el color de las ranuras encontradas a blanco con un alfa de 1 si el color no es ya blanco.
