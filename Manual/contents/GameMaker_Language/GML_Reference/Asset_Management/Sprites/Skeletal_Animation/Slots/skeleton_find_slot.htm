skeleton_find_slot

Con esta función se puede encontrar qué ranuras están en una posición de espacio especificada en la Spine sprite asociada a la instancia actual. Se crea una lista DS y se suministra su ID junto con una posición x/y para comprobar y la lista se rellenará con el nombre string para cada una de las ranuras de fijación disponibles para el sprite (incluyendo cualquier modificación de fijación). Tenga en cuenta que la lista siempre se ordena en orden descendente empezando por la ranura más alta.

Sintaxis:

skeleton_find_slot(x, y, list);

Argumento Tipo Descripción
x Real The x position in the room to check.
y Real The y position in the room to check.
list ID de la lista DS The ID of the DS list to populate with the DS maps.

Devuelve:

N/A

Ejemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

El código anterior crea una lista DS y luego la rellena con los nombres de las ranuras que se encuentran en la posición del ratón en la página room.
