skeleton_attachment_get

Una animación esquelética sprite puede tener otros sprites añadidos como adjuntos, siendo estos sprites añadidos a una ranura con nombre (el nombre se da cuando se crea la ranura del adjunto en su programa de animación) y serán dibujados junto con la animación del actual sprite. Con esta función puede obtener el nombre (como string) del adjunto para la ranura dada del sprite asignado actualmente. Tenga en cuenta que los archivos adjuntos sprites se referencian a través de su cadena de nombre asignada en Spine, o cuando llamó a skeleton_attachment_create().

Sintaxis:

skeleton_attachment_get(slot);

Argumento Tipo Descripción
slot Cadena The slot name (a string) to get the attachment of.

Devuelve:

String

Ejemplo:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

El código anterior comprobaría el nombre de archivo adjunto actualmente asignado para la ranura llamada "slot_leftHand" y si se devuelve un string vacío, se adjunta un nuevo sprite.
