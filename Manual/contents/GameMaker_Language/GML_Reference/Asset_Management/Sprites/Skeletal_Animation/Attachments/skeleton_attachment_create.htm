skeleton_attachment_create

Cuando creas tu animación de esqueleto sprite, puedes asignar ranuras de adición y aditamentos para ir en ellas. Estos son simplemente imágenes que están separadas de la animación pero que cuando se adjuntan se mueven junto con el hueso al que están adjuntos. Normalmente asignarías estos adjuntos en tu programa de animación (Spine), pero con esta función puedes crear los tuyos propios en tiempo de ejecución usando un sprite asset de tu juego. la función requiere que le des un nombre al adjunto (como string) y luego establezcas el sprite_index y el image_index a usar, así como el origen x e y (que puede ser diferente al definido por el sprite en las propiedades de sprite ), y también puedes establecer cualquier transformación que desees que se aplique a la imagen cuando se adjunte. Si el archivo adjunto fue creado con éxito, la función devolverá 1 y si no lo fue (usted suministró un índice sprite inválido, o la base sprite no es un Spine sprite) entonces devolverá -1.

Sintaxis:

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

Argumento Tipo Descripción
name Cadena The name (as a string) of the attachment to create.
sprite Activo Sprite The sprite_index to get the attachment image from.
ind Real The image_index to get the attachment image from.
xorigin Real The x origin for the image being used.
yorigin Real The y origin for the image being used.
xscale Real The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the image. 0=normal, 90=turned 90 degrees counter-clockwise etc.

Devuelve:

Real (1 if successful, -1 if not)

Ejemplo:

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

El código anterior comprobaría el nombre de archivo adjunto actualmente asignado para la ranura llamada "slot_leftHand" y si se devuelve un string vacío, se adjunta un nuevo sprite.
