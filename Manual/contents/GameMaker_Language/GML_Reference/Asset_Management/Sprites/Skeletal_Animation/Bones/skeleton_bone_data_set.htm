skeleton_bone_data_set

Tu animación de esqueleto se compone de un número de "huesos", a los que habrás definido y dado nombres en tu programa de animación, y esta función se puede utilizar para establecer ciertos datos para el hueso nombrado en cualquier momento. Ten en cuenta que estos datos se refieren a la pose por defecto del esqueleto, y no a la pose actual que se está dibujando (para ello utiliza la función skeleton_bone_state_set()), y debe establecerse a partir de un mapa DS previamente creado, que debe tener las siguientes claves y sus valores equivalentes:

  • "x": La posición local x del hueso en relación con el hueso padre.
  • "y": La posición local y del hueso en relación con el hueso padre.
  • "ángulo": La rotación local del hueso con respecto al hueso padre.
  • "xscale": La escala horizontal local del hueso.
  • "yscale": La escala vertical local del hueso.
  • "padre": El nombre (un string) del hueso padre.

Sintaxis:

skeleton_bone_data_set(bone, map);

Argumento Tipo Descripción
bone Cadena The name (as a string) of the bone.
map ID del mapa DS The (previously created) DS map that stores the bone data.

Devuelve:

N/A

Ejemplo:

var bone_map = ds_map_create();
skeleton_bone_data_get("head", bone_map);
ds_map_replace(bone_map, "parent", "body");
skeleton_bone_data_set("head", bone_map);
ds_map_destroy(bone_map);

El código anterior crea un mapa DS y luego lo rellena con los datos del hueso llamado "cabeza". Luego reemplaza la clave del hueso "padre" en el mapa con un nuevo valor y establece el hueso "cabeza" de nuevo con el nuevo conjunto de datos.
