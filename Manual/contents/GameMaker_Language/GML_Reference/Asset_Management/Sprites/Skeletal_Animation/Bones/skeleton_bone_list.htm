lista_esqueleto_de_bonos

Con esta función se puede rellenar una lista DS (pre-creada) con todos los nombres de los huesos utilizados como parte de la animación del esqueleto sprite. Los nombres serán strings y luego pueden ser utilizados en las otras funciones de huesos de animación del esqueleto para estos tipos de sprite.

Sintaxis:

skeleton_bone_list(sprite, list);

Argumento Tipo Descripción
sprite Activo Sprite The sprite index of the Spine skeletal animation to get the list from.
list ID de la lista DS The ID of the DS list to populate with the bone names.

Devuelve:

N/A

Ejemplo:

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

El código anterior crea una lista DS y la rellena con los nombres de los huesos para su uso posterior.
