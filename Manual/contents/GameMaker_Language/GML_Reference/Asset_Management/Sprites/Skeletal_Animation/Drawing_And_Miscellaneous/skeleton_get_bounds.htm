skeleton_get_bounds

Esta función devolverá una matriz de valores asociados con cualquier cuadro delimitador para la animación de esqueleto asignada actualmente sprite. Se proporciona el número de índice del cuadro delimitador para obtener los detalles (se puede recuperar el número total de cuadros delimitadores para el sprite utilizando la función skeleton_get_num_bounds()) y la función devolverá un array con los siguientes elementos:

  • [0] - el número de puntos de la caja delimitadora (un real)
  • [1] - el nombre del cuadro delimitador (un string)
  • [2] - la posición x del primer punto
  • [3] - la posición y del primer punto
  • [etc...] - más datos de posición x/y hasta el número de puntos del cuadro delimitador

Sintaxis:

skeleton_get_bounds(index);

Argumento Tipo Descripción
index Real The bounding box index to get the values of.

Devuelve:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

Ejemplo:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

El código anterior loop a través de cada uno de los cuadros delimitadores asociados con el sprite actualmente asignado y luego dibujar esos datos como un string a la pantalla.
