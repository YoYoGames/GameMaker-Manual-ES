Dibujo y otros

Las siguientes funciones son herramientas adicionales para obtener diferentes conjuntos de datos sobre un sprite de animación de esqueleto:

  • skeleton_get_minmax
  • skeleton_get_num_bounds
  • skeleton_get_bounds

También puede utilizar varias funciones de dibujo diseñadas específicamente para estos tipos de sprite (aparte de las funciones normales draw_sprite() ). Estas funciones le dan más control sobre lo que se está dibujando y le permiten cambiar las características específicas de la animación, como el tiempo y las pieles:

  • skeleton_collision_draw_set
  • draw_skeleton
  • draw_skeleton_instance
  • draw_skeleton_collision
  • draw_skeleton_time
  • draw_enable_skeleton_blendmodes
  • draw_get_enable_skeleton_blendmodes
