draw_skeleton_collision

Esta función dibujará las máscaras de colisión asociadas con la animación esquelética dada. Usted suministra la base sprite, el conjunto de animación a utilizar y el marco para obtener la información, y también puede establecer las propiedades de transformación para adaptarse.

Sintaxis:

draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, rot, colour)

Argumento Tipo Descripción
sprite Activo Sprite The index of the sprite to draw.
animname Cadena The name of the animation to get the frame from (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Color The colour with which to blend the sprite.

Devuelve:

N/A

Ejemplo:

draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará los datos de la máscara de colisión para el actual sprite, utilizando las transformaciones actuales, para el conjunto de animación "salto".
