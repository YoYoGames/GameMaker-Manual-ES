draw_skeleton

Esta función es sólo para usar con sprites que han sido creados usando un programa de animación esquelética como Spine. Aunque puede dibujar estos sprites utilizando las funciones de dibujo normales (como draw_self()), hay ocasiones en las que puede querer dibujar un solo fotograma o mostrar un cambio de textura de la piel sin cambiar realmente las propiedades de sprite. En estos casos puedes dibujar el sprite con esta función, donde das el índice sprite y luego el nombre de la animación para obtener el cuadro (un string, como se establece dentro del programa utilizado para hacer el sprite). A continuación, se da el nombre de la piel a utilizar (de nuevo, como se establece cuando el sprite se hizo), y un valor de marco. El valor de fotograma es para decirle a GameMaker qué parte de la animación debe mostrar y es un valor entre 0 y el image_number. El resto de los argumentos son los mismos que se utilizan para el dibujo normal de sprite y cambiarán la escala, el ángulo, el color de mezcla y el alfa del cuadro de animación final que se dibuja.

¡ATENCIÓN! Debido a la forma en que las animaciones de esqueleto se interpolan entre varios fotogramas "clave", hay una sobrecarga adicional asociada a esta llamada y, por lo tanto, se recomienda evitar dibujar esqueletos basados en sprites utilizando este método a menos que sea absolutamente necesario.

Sintaxis:

draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Argumento Tipo Descripción
sprite Activo Sprite The index of the sprite to draw.
animname Cadena The name of the animation to get the frame from (a string).
skinname Cadena The name of the skin to use (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Color The colour with which to blend the sprite.
alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

N/A

Ejemplo:

draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0.5);

El código anterior dibujará el primer fotograma de la animación esquelética dada sprite, utilizando el conjunto de animación "salto", la piel definida como "skin1" y en la misma escala y posición que definen las propiedades de la instancia, pero con un alfa de 0,5.
