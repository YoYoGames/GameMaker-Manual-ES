dibujar_esqueleto_instancia

Esta función sólo se puede utilizar con instancias que tengan asignado sprites creado con el programa de animación de esqueletos Spine. Aunque puede dibujar estos sprites utilizando las funciones normales de dibujo (como draw_self()), hay ocasiones en las que puede querer dibujar un solo fotograma o mostrar un cambio en la textura de la piel sin cambiar realmente las propiedades de sprite para una instancia. En estos casos puedes dibujar el sprite que ha sido asignado a la instancia con esta función, donde das el ID único de la instancia y luego el nombre de la animación para obtener el marco (un string, como se establece dentro del programa utilizado para hacer el sprite). A continuación, se da el nombre de la piel a utilizar (de nuevo, como se establece cuando el sprite se hizo), y un valor de marco. El valor del marco es para decirle a GameMaker qué parte de la animación mostrar y es un valor entre 0 y el image_number. El resto de los argumentos son los mismos que se utilizan para el dibujo normal de sprite y cambiarán la escala, el ángulo, el color de mezcla y el alfa del cuadro de animación final que se dibuja.

¡ATENCIÓN! Debido a la forma en que las animaciones de esqueleto se interpolan entre varios fotogramas "clave", hay una sobrecarga adicional asociada a esta llamada y, por lo tanto, se recomienda evitar dibujar esqueletos basados en sprites utilizando este método a menos que sea absolutamente necesario.

Sintaxis:

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Argumento Tipo Descripción
instance ID de la instancia The index of the instance to draw the sprite from.
animname Cadena The name of the animation to get the frame from (a string).
skinname Cadena The name of the skin to use (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Color The colour with which to blend the sprite.
alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

N/A

Ejemplo:

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

El código anterior dibujará el primer fotograma de la animación esquelética sprite asignada a la instancia dada, utilizando el conjunto de animación "salto" y el skin definido como "skin1".
