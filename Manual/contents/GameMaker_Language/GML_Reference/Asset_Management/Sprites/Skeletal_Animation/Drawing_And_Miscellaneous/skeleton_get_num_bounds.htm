skeleton_get_num_bounds

Esta función devolverá el número de cuadros delimitadores asociados a la animación del esqueleto sprite asignada a la instancia que ejecuta el código. Esto puede ser utilizado junto con la función skeleton_get_bounds() para recuperar datos sobre cada uno de los cuadros delimitadores.

Sintaxis:

skeleton_get_num_bounds();

Devuelve:

Real

Ejemplo:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

El código anterior loop a través de cada uno de los cuadros delimitadores asociados con el sprite actualmente asignado y luego dibujar esos datos como un string a la pantalla.
