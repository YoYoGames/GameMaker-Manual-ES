skeleton_collision_draw_set

Con esta función, puede activar (true) o desactivar (false) el dibujo de los datos de colisión para la animación esquelética actual sprite que está utilizando la instancia. Si se activa, el cuadro delimitador y la máscara de colisión precisa se dibujarán como contornos alrededor de sprite.

Esta función sólo funciona cuando el dibujo del esqueleto sprite está siendo manejado por el object. Para dibujar la caja delimitadora de colisión cuando se dibuja un esqueleto sprite manualmente, utilice draw_skeleton_collision().

Sintaxis:

skeleton_collision_draw_set(flag);

Argumento Tipo Descripción
flag Booleano Set to true para activar el dibujo, y false para desactivarlo.

Devuelve:

N/A

Ejemplo:

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

El código anterior comprueba si el juego se está ejecutando en modo de depuración y, si es así, cambia los datos de colisión del esqueleto de la instancia a mostrar.
