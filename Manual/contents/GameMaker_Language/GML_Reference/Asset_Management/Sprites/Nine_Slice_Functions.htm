Nueve funciones de corte

Además de la sección Nine Slice en el Editor de Sprite, GameMaker proporciona funciones para establecer las propiedades de Nine Slice para sprites en runtime. Antes de leer acerca de estas funciones, se recomienda leer esta página para obtener una comprensión de la técnica Nine Slice.

Las siguientes funciones se pueden utilizar para recuperar y cambiar las propiedades de Nine Slice para un sprite asset en tiempo de ejecución:

  • sprite_nineslice_create
  • sprite_get_nineslice
  • sprite_set_nineslice

Estas funciones hacen uso de una rebanada de nueve struct, y usted puede leer sobre el contenido de ese struct en esta página.

Dibujo de nueve sprites

No hay funciones separadas para dibujar Nine Slice sprites ya que trabajan con las funciones de dibujo de sprites existentes (excepto las funciones que dibujan sólo una parte de sprite o lo distorsionan, como draw_sprite_part() o draw_sprite_pos()).

Se recomienda utilizar la función draw_sprite_stretched() para dibujar una Nine Slice sprite, ya que le permite especificar las dimensiones absolutas de la sprite en píxeles y será fácil de usar junto con Nine Slice. También puede utilizar draw_sprite_ext() que en su lugar toma valores de escala.
