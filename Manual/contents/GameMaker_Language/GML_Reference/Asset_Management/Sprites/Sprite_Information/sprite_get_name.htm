sprite_get_name

Esta función devolverá el nombre en forma de cadena del sprite especificado. Este nombre es el que se ha especificado para el sprite en el árbol de recursos de la ventana principal de GameMaker. Por favor, tenga en cuenta que esto es sólo un string y no puede ser usado para referenciar el sprite directamente - para eso necesitaría el índice del sprite. Sin embargo, puede utilizar este string para obtener el índice del sprite utilizando el string devuelto junto con la función asset_get_index().

Sintaxis:

sprite_get_name(index);

Argumento Tipo Descripción
index Activo The index of the sprite to get the name of.

Devuelve

String

Ejemplo:

str = sprite_get_name(sprite_index);

El código anterior obtendrá el nombre del índice sprite para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "str".
