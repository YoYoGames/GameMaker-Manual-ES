sprite_get_info

Esta función se utiliza para recuperar la información del sprite dado. Se suministra un índice sprite (que puede ser un asset añadido a través del Navegador de Activos o un sprite añadido en runtime) y la función devuelve una estructura con las siguientes variables:

Estructura de información de sprites
Variable Tipo Descripción
anchura real La anchura del sprite (en píxeles)
altura real La altura del sprite (en píxeles)
xoffset real El desplazamiento/origen X del sprite (en píxeles)
yffset real El desplazamiento/origen Y del sprite (en píxeles)
transparente booleano true si el sprite está marcado como transparente, en caso contrario false
(Esto sólo puede especificarse a través de sprite_add() o funciones similares, y será false para sprites creado en el IDE)
suave booleano true si el sprite está marcado como suave, en caso contrario false
(Esto sólo puede especificarse a través de sprite_add() o funciones similares, y será false para sprites creado en el IDE)
tipo real El tipo de sprite:
0 - Bitmap (Regular sprites)
1 - SWF
2 - Lomo
bbox_left real Posición del borde izquierdo del cuadro delimitador (en píxeles)
bbox_top real Posición del borde superior del cuadro delimitador (en píxeles)
bbox_right real Posición del borde derecho del cuadro delimitador (en píxeles)
bbox_bottom real Posición del borde inferior del cuadro delimitador (en píxeles)
nombre cadena El nombre del sprite
número_subimágenes real El número de subimágenes (o cuadros) del sprite
usar_máscara booleano true si este sprite utiliza una máscara de colisión, en caso contrario false
número_de_máscaras real El número de máscaras en este sprite (será mayor que 1 si el sprite utiliza máscaras por fotograma)
rotated_bounds boolean Whether the mask is "Rectangle with rotation" or not
nineslice struct La estructura de nueve cortes para este sprite, o undefined si no tiene datos de nueve cortes
mensajes matriz Matriz de mensajes de difusión para este sprite, donde cada mensaje de difusión es un struct que contiene información sobre el mensaje (más información en "Datos generales Sprite ")
frame_info matriz Matriz de fotogramas para este sprite, donde cada fotograma es un struct que contiene información sobre su temporización (más información en "Datos generales Sprite ")

Esta variable adicional sólo está disponible para los sprites Bitmap (regulares):

Estructura de información de sprites
Variable Tipo Descripción
marcos matriz Matriz de fotogramas para este sprite, donde cada fotograma es un struct que contiene información sobre su textura (más información en "Bitmap Sprite Data")

Estas variables adicionales sólo están disponibles para los sprites de Spine:

Estructura de información de sprites
Variable Tipo Descripción
num_atlas real El número de texturas del atlas utilizadas
atlas_textura matriz Matriz de identificadores de textura utilizados para el atlas
premultiplicado booleano true si este sprite está marcado como premultiplicado, en caso contrario false
nombres_de_animación matriz Array que contiene los nombres de cada animación en este sprite
nombres_de_piel matriz Matriz que contiene los nombres de cada piel de este sprite
huesos matriz Matriz que contiene structs para cada hueso de este sprite (más información en "Datos de la columna vertebral Sprite ")
ranuras matriz Matriz que contiene structs para cada ranura de este sprite (más información en "Datos de la columna vertebral Sprite ")

La función devolverá undefined si el sprite dado no existe. Tenga en cuenta también que la información devuelta en este struct debe considerarse de sólo lectura, ya que la modificación de cualquiera de estas variables no afectará al sprite.

Las secciones siguientes contienen información sobre las matrices y structs incluidas en el struct devuelto en función del tipo sprite:

General Sprite Datos

Esta sección contiene información sobre las variables incluidas en struct para todo tipo de sprites.

La variable messages es una matriz que contiene información sobre los mensajes de difusión que existen en el sprite dado. Cada mensaje de difusión en este array es un struct que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
marco real El tiempo de este mensaje de difusión desde el inicio de la animación (en fotogramas)
mensaje cadena La cadena de mensajes de difusión

La variable frame_info es un array que contiene información sobre los tiempos de los fotogramas del sprite. Cada fotograma de este array es un struct que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
marco real El tiempo para el inicio de este cuadro (en cuadros)
duración real La duración de este fotograma (en fotogramas)
índice_de_imagen real El índice de la imagen de este fotograma

Mapa de bits Sprite Datos

Esta sección contiene información sobre las variables incluidas en el struct para Bitmap sprites (sprites que no son Spine o SWF tipo sprites).

La variable frames es un array que contiene información sobre las texturas de los fotogramas del sprite. Cada cuadro de este array es un struct que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
x real La posición X de este marco en su página de textura (en píxeles)
y real La posición Y de este marco en su página de textura (en píxeles)
w real La anchura lógica de la trama (en píxeles) utilizada internamente
h real La altura lógica del cuadro (en píxeles) utilizada internamente
textura real El ID de la página de la textura para este marco
ancho_original real La anchura original del cuadro (en píxeles)
altura_original real La altura original del cuadro (en píxeles)
ancho_de_recorte real La anchura real del marco en la página de la textura después de recortar y escalar (ya que GameMaker recorta automáticamente el espacio vacío alrededor de una imagen, y también la reduce si no cabe)
altura_de_cultivo real La altura real del marco en la página de la textura después de recortar y escalar
x_offset real El desplazamiento X desde el borde izquierdo de la imagen original hasta el borde izquierdo de la sección recortada
y_offset real El desplazamiento Y desde el borde superior de la imagen original hasta el borde superior de la sección recortada

Columna vertebral Sprite Datos

Esta sección contiene información sobre las variables incluidas en struct para Spine sprites.

La variable bones es un array que contiene información sobre todos los huesos del Spine sprite dado. Cada hueso de este array es un struct que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
padre cadena El nombre del hueso padre, o undefined si este hueso no tiene padre
nombre cadena El nombre de este hueso
índice real El índice de este hueso
longitud real La longitud de este hueso
x real La posición X de este hueso
y real La posición Y de este hueso
rotación real La rotación de este hueso
escala_x real (Interno a Spine) Valor de la escala en X
escala_y real (Interno a Spine) Valor de la escala en Y
shear_x real (Interna a Spine) Valor de cizallamiento en X
shear_y real (Interna a Spine) Valor de cizallamiento en Y
modo_de_transformación real (Interno a Spine) El modo de transformación

NOTA: Consulte la documentación de Spine para obtener más información sobre las variables internas de Spine.

La variable slots es una matriz que contiene información sobre todas las ranuras en el Spine sprite dado. Cada ranura de este array es un struct que contiene las siguientes variables:

Nombre de la variable Tipo de datos Descripción
nombre cadena El nombre de la ranura
índice real El índice de la ranura
hueso cadena El nombre del hueso de la ranura, o "(none)" si no hay hueso para esta ranura
archivo adjunto cadena Nombre del archivo adjunto
rojo real Componente rojo del color de la ranura (0-1)
verde real Componente verde del color de la ranura (0-1)
azul real Componente azul del color de la ranura (0-1)
alfa real Componente alfa del color de la ranura (0-1)
blendMode real (Interno a Spine) Modo de mezcla para la ranura
dark_red* real Componente rojo del color oscuro de la ranura (0-1)
dark_green* real Componente verde del color oscuro de la ranura (0-1)
dark_blue* real Componente azul del color oscuro de la ranura (0-1)
dark_alpha* cadena Componente alfa del color oscuro de la ranura (0-1)

* NOTA: Las variables de color oscuro sólo están disponibles si la ranura tiene activado el Tinte Negro.

Sintaxis:

sprite_get_info(index);

Argumento Tipo Descripción
index Activo Sprite The index of the sprite to get the information for.

Devuelve

Sprite Info Struct (or undefined)

Ejemplo:

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

El código anterior obtiene la información struct para la instancia sprite, y luego comprueba si no es undefined y si su tipo es 0 (lo que significa que es un mapa de bits sprite). En ese caso, obtiene la matriz de mensajes de difusión para ese sprite y luego ejecuta un for loop para imprimir cada mensaje de difusión (junto con su trama) en el registro de salida.
