sprite_get_number

Un sprite tiene que tener al menos una sub-imagen y esta función devolverá cuántas tiene. Recuerde que una sub-imagen de sprite comienza a numerarse a partir de 0, por lo que si esta función devuelve 3 (el sprite que se está comprobando tiene 3 sub-imágenes) se numerarán 0, 1 y 2.

Sintaxis:

sprite_get_number(index);

ArgumentoTipo Descripción
index The index of the sprite.

Devuelve

Ejemplo:

image_index = floor(random(sprite_get_number(sprite_index)));

El código anterior establecerá el índice de la imagen en un marco aleatorio basado en el número de sub-imágenes que tiene el sprite.
