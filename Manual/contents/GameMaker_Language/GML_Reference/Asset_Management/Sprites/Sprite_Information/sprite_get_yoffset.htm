sprite_get_yoffset

Cuando se define un sprite en el editor sprite, es necesario establecer el origen para ese sprite. Este es el punto en el que el sprite se "adjuntará" o "dibujará" cuando se utilice con una instancia. Esta función devuelve el desplazamiento relativo para el eje y del origen basado en que la esquina superior izquierda es la posición 0,0, con un +y siendo hacia abajo y un -y siendo hacia arriba desde esto. La siguiente imagen lo ilustra:

Sintaxis:

sprite_get_yoffset(index);

ArgumentoTipo Descripción
index The index of the sprite to find the yoffset of.

Devuelve

Ejemplo:

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = sprite_get_yoffset(sprite_index);
}

El código anterior se asegurará de que se mantenga una instancia dentro de la room basada en la sprite (de modo que la sprite sea siempre visible).
