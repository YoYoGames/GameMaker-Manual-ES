sprite_get_bbox_mode

Esta función devolverá el "modo" actual para los cálculos de la caja delimitadora. Usted suministra el índice sprite del sprite a comprobar, y la función devolverá una de las constantes que se muestran a continuación.

Sintaxis:

sprite_get_bbox_mode(ind);

ArgumentoTipo Descripción
ind The index of the sprite to check.

Devuelve

Bounding Box Mode Constant

Constante Descripción
bboxmode_automatic Automático - El cuadro delimitador se calculará automáticamente, basándose en la configuración de la tolerancia del sprite
bboxmode_fullimage Imagen completa: el cuadro delimitador se establecerá para utilizar toda la anchura y la altura de la página sprite, independientemente de la tolerancia y de los píxeles "vacíos".
bboxmode_manual Manual - El cuadro delimitador se ha ajustado manualmente a los valores definidos por el usuario (ya sea en el editor sprite, o utilizando la función sprite_set_bbox())

Ejemplo:

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Lo anterior comprueba el modo de bbox para el actual sprite y si no es automatic, entonces se establece a ese valor.
