sprite_get_xoffset

Cuando se define un sprite en el editor sprite, es necesario establecer el origen para ese sprite. Este es el punto en el que el sprite se "adjuntará" o "dibujará" cuando se utilice con una instancia. Esta función devuelve el desplazamiento relativo para el eje x del origen basado en que la esquina superior izquierda es la posición 0,0, siendo +x la derecha y -x la izquierda de ésta. La siguiente imagen lo ilustra:

Sintaxis:

sprite_get_xoffset(index);

ArgumentoTipo Descripción
index The index of the sprite to find the xoffset of.

Devuelve

Ejemplo:

if x - sprite_get_xoffset(sprite_index) < 0
{
    x = _xoffset;
}

El código anterior se asegurará de que se mantenga una instancia dentro de la room basada en la sprite (de modo que la sprite sea siempre visible).
