Información sobre Sprite

Las siguientes funciones pueden utilizarse para obtener información sobre un sprite asset. Tenga en cuenta que estas funciones obtienen la información de la base sprite (es decir, el sprite asset, tal y como se creó en el Navegador de Activos) y no del sprite tal y como se está utilizando en una instancia. Por ejemplo, puede utilizar la variable sprite variable de instancia sprite_width para obtener la anchura del sprite que está utilizando una instancia y esta anchura tendrá en cuenta cualquier escalado que se haya aplicado. Sin embargo, si necesita conocer la anchura de un sprite tal y como sería normalmente, sin transformaciones aplicadas, tendría que utilizar la función sprite_get_width() que aparece a continuación.

Para obtener más información sobre las variables de instancia de sprite, consulte la sección sobre variables de instancia.

  • sprite_get_number
  • sprite_get_speed
  • sprite_get_speed_type
  • sprite_get_width
  • sprite_get_height
  • sprite_get_xoffset
  • sprite_get_yoffset
  • sprite_get_bbox_bottom
  • sprite_get_bbox_left
  • sprite_get_bbox_right
  • sprite_get_bbox_top
  • sprite_get_bbox_mode
  • sprite_get_nineslice
  • sprite_get_tpe
  • sprite_get_texture
  • sprite_get_uvs
  • sprite_get_info
