sprite_get_nineslice

Esta función devuelve la estructura Nine Slice para un sprite, que contiene todas sus propiedades Nine Slice tal y como se han establecido en el Editor de Sprite o utilizando sprite_set_nineslice(). El contenido de este struct se detalla en esta página.

Si el sprite suministrado no tiene asignado un Nine Slice struct, se creará un nuevo struct con propiedades Nine Slice por defecto para el sprite y se devolverá.

El cambio de cualquier valor en este struct modificará las propiedades de Nine Slice del original sprite, afectando a cualquier futura llamada de dibujo realizada con ese sprite.

Sintaxis:

sprite_get_nineslice(ind);

ArgumentoTipo Descripción
ind The index of the sprite from which the Nine Slice struct will be retrieved

Devuelve:

Nine Slice Struct (or -1 if the sprite doesn't exist)

Ejemplo:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

El código anterior recupera el Nine Slice struct de un sprite, habilita el Nine Slicing para él y establece los offsets de la guía.
