sprite_get_height

Con esta función puede encontrar la altura de la base sprite asset, sin transformaciones, en píxeles.

Sintaxis:

sprite_get_height(index);

ArgumentoTipo Descripción
index The index of the sprite to find the height of.

Devuelve

Ejemplo:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

El código anterior comprueba la altura de la sprite tal y como está en la instancia actual y si hay una diferencia entre ésta y la base original sprite, restablece la escala del eje y.
