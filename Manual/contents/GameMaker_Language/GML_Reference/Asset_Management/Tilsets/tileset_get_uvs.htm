tileset_get_uvs

Esta función devuelve un array con las coordenadas UV y otros datos de la textura del tile set dado en la página de la textura. La función devuelve un array con los siguientes 8 elementos:

  • [0] = izquierda
  • [1] = arriba
  • [2] = derecho
  • [3] = fondo
  • [4] = cantidad de píxeles que el compilador de asset ha recortado del lado izquierdo de tile set
  • [5] = cantidad de píxeles que el compilador de asset ha recortado de la parte superior de tile set
  • [6] = porcentaje normalizado de datos de píxeles del ancho original de tile set que se ha guardado en la página de la textura
  • [7] = porcentaje normalizado de datos de píxeles de la altura original de tile set que se ha guardado en la página de la textura

Esta matriz puede ser utilizada en otras funciones de dibujo, particularmente cuando se trabaja en 3D o se utilizan las funciones primitivas 2D, así como cuando se trabaja con las funciones Shader.

Sintaxis:

tileset_get_uvs(tileset);

ArgumentoTipo Descripción
tileset The index of the tile set to use.

Devuelve

(1D, 8 elements)

Ejemplo:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

El código anterior almacenará las coordenadas UV para el tile set dado en un array local y luego asignará los valores a las variables de instancia.
