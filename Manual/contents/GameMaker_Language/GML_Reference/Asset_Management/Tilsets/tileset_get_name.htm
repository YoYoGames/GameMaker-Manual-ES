tileset_get_name

Esta función devolverá el nombre en forma de cadena del tile set asset especificado. Este nombre es el que se ha especificado para el tile set en el navegador Asset de la ventana principal de GameMaker. Por favor, tenga en cuenta que esto es sólo un string y no puede ser usado para referenciar el tile set directamente - para eso necesitaría el índice del conjunto de baldosas. Sin embargo, puede utilizar este string para obtener el índice del conjunto de baldosas utilizando el string devuelto junto con la función asset_get_index().

Sintaxis:

tileset_get_name(index);

Argumento Tipo Descripción
index Activo del conjunto de baldosas The index of the tile set to get the name of.

Devuelve

String

Ejemplo:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

El código anterior obtendrá el nombre del índice tile set para la capa dada, almacenando el retorno string en la variable tileset_name.
