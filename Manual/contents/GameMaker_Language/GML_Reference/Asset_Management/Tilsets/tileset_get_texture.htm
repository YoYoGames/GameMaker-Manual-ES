tileset_get_texture

Esta función devuelve un puntero especial para la página de textura tile set. Este valor puede ser usado en otras funciones de dibujo, particularmente en las funciones primitivas 2D, así como en las funciones de Shader. Puede obtener más información sobre la página de textura devuelta utilizando las diferentes funciones de texture_ que se encuentran aquí.

Sintaxis:

tileset_get_texture(tileset);

Argumento Tipo Descripción
tileset Activo del conjunto de baldosas The index of the tile set to use.

Devuelve

Texture

Ejemplo:

var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vertex texturizados con la textura mantenida en la variable "tex".
