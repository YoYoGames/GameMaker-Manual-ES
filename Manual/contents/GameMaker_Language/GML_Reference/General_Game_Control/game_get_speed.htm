game_get_speed

Esta función se puede utilizar para obtener la velocidad del juego, ya sea como el número de fotogramas de juego para ejecutar por segundo o como el número de micro segundos por fotograma de juego. Tenga en cuenta que este no es el valor real de la velocidad de ejecución FPS (para eso use la variable fps_real ), sino el número de cuadros de juego (FPS) que el juego intentará mantener cada segundo, o la duración de cada cuadro de juego en microsegundos que el juego intentará mantener (MPF). Cuando se utiliza esta función es necesario dar una de las siguientes constantes que determinarán el tipo del valor de retorno:

Velocidad de juego constante
Constante Descripción
gamespeed_fps Obtiene la velocidad del juego utilizando fotogramas por segundo.
gamespeed_microseconds Obtiene la velocidad del juego usando microsegundos por cuadro.

Así, por ejemplo, si la velocidad del juego está fijada en 30 en las Opciones de Juego y utilizas el tipo FPS, entonces la función devolverá 30, pero si utilizas el MPF entonces la función devolverá 33333.

Sintaxis:

game_get_speed(type);

Argumento Tipo Descripción
type Velocidad de juego constante The type of method used to get the game speed (see the constants above).

Devuelve:

Real

Ejemplo:

if game_get_speed(gamespeed_fps) != 60
{
    game_set_speed(60, gamespeed_fps);
}

El código anterior comprueba si el juego se está ejecutando con una velocidad de juego de 60 FPS y, si no es así, se ajusta a 60 FPS.
