valor_de_puntuación_alta

Con esta función se puede recuperar el valor de la puntuación que se ha almacenado en la lista de puntuaciones altas en la posición dada. Si no se ha introducido ninguna puntuación, la función devolverá 0.

Sintaxis:

highscore_value(place);

ArgumentoTipo Descripción
place The place on the table (1-10).

Devuelve:

Ejemplo:

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

El código anterior se loop a través de la lista de alta puntuación y almacenar todas las puntuaciones en una matriz.
