valor_de_puntuación_alta

Con esta función puedes borrar la lista global interna de puntuaciones, restableciendo todos los nombres al valor por defecto de "Desconocido" y todas las puntuaciones al valor por defecto de 0.

Sintaxis:

highscore_clear();

Devuelve:

Ejemplo:

highscore_clear();

De esta forma, se borraría la lista de puntuaciones más altas.
