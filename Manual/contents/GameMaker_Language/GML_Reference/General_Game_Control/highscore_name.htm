nombre_de_la_punta_alta

Con esta función se puede recuperar el nombre string que se ha almacenado en la lista de puntuaciones altas en la posición dada. Si no se ha introducido ningún nombre, se devolverá el string "Desconocido".

Sintaxis:

highscore_name(place);

ArgumentoTipo Descripción
place The place on the table (1-10).

Devuelve:

Ejemplo:

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

El código anterior se loop a través de la lista de alta puntuación y almacenar todos los nombres en una matriz.
