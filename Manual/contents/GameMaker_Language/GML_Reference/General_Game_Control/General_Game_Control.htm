Control general del juego

Estas funciones y variables están relacionadas con funcionalidades específicas del juego, como cargar/guardar, reiniciar, etc.

Las siguientes variables integradas de alcance global son útiles para debugging su juego - pueden ser utilizadas como la salida a un archivo de registro en el caso de los problemas, por ejemplo - así como para mostrar el ID de construcción actual, etc... al usuario:

  • game_id
  • game_save_id
  • game_display_name
  • game_project_name

Las siguientes funciones se utilizan para el control general del juego:

  • game_end
  • game_restart
  • game_load
  • game_load_buffer
  • game_save
  • game_save_buffer
  • game_get_speed
  • game_set_speed

Para ayudarte a crear y mantener las puntuaciones máximas de tus juegos, GameMaker crea una matriz global de puntuaciones máximas de 10 lugares a la que puedes acceder, añadir y cambiar para crear tus propias listas de puntuaciones máximas personalizadas sin demasiadas complicaciones. Esta funcionalidad es local para el juego y funciona en todas las plataformas, por lo que puede almacenar y mantener fácilmente una tabla básica de puntuaciones máximas utilizando estas funciones junto con las funciones de manejo de archivos.

Existen las siguientes funciones que se ocupan de la lista interna de puntuaciones altas.

  • highscore_add
  • highscore_name
  • highscore_value
  • highscore_clear

GameMaker también incluye una función para dibujar las puntuaciones máximas en la pantalla como una lista, sin embargo, es muy básico y se recomienda que usted utiliza esto para debugging más que cualquier otra cosa como hacer su propia tabla de puntuación alta es fácil y le dará mucho más control sobre cómo se muestra:

  • draw_highscore

Por último, tenemos una función que permite asignar un sprite al cursor del ratón:

  • cursor_sprite
