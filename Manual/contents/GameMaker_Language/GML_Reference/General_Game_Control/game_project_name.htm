game_project_name

Esta variable de sólo lectura devuelve el nombre de visualización de tu juego para la plataforma de destino en un formato "apto para guardar" para la plataforma de destino. Si el nombre de la pantalla contiene caracteres no permitidos para un nombre de archivo, serán sustituidos automáticamente por "_". serán sustituidos automáticamente por "_". El nombre de visualización puede ser configurado en las Opciones de Juego. Tenga en cuenta que, como no hay restricciones en los nombres de archivo para HTML5, este string será probablemente el mismo que el devuelto por game_display_name.

Sintaxis:

game_project_name

Devuelve:

Ejemplo:

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

El código anterior obtiene el nombre del proyecto y lo utiliza para abrir (o crear) un archivo ini en el que se escribe.
