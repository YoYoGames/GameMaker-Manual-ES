game_load_buffer

Con esta función puedes cargar un estado de juego que ha sido guardado previamente. El juego se carga desde un "grow" creado previamente buffer (ver Buffers) y el buffer debe haber tenido un estado de juego guardado en él usando game_save_buffer() función.

Sintaxis:

game_load_buffer(buffer);

ArgumentoTipo Descripción
buffer The buffer id to load from.

Devuelve:

Ejemplo:

if keyboard_check_pressed(ord("L"))
{
    if global.Checkpoint game_load_buffer(save_buff);
}

El código anterior cargará un estado de juego previamente guardado desde el buffer indexado en la variable "save_buff", sólo si la variable global es true, cuando el jugador presione la tecla "L".
