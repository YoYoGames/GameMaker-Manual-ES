game_save_id

Esta variable de sólo lectura devolverá el ID completo de path del directorio que utiliza tu juego para guardar los archivos. Este directorio puede o no ser visible para otras aplicaciones, dependiendo de la plataforma.

En el objetivo HTML5 devolverá un string vacío.

NOTA Esta variable existe puramente para información y no debe ser usada para obtener un archivo path o algo similar, ya que no funcionará correctamente en todas las plataformas.

Sintaxis:

game_save_id

Devuelve:

String

Ejemplo:

save_dir = game_save_id;

Esto almacenará el directorio para guardar los archivos en una variable para su uso futuro.
