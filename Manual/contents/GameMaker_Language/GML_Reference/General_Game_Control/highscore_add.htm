highscore_add

Con esta función puedes añadir un nombre y una puntuación a la lista global interna de puntuaciones. No es necesario comprobar el valor para ver si es lo suficientemente alto como para introducirlo en la lista de puntuaciones, ya que GameMaker sólo almacenará aquellos valores que sean mayores que la décima posición almacenada.

Sintaxis:

highscore_add(str, numb);

Argumento Tipo Descripción
str Cadena The string (name) to attribute the score to.
numb Real The number (score) to add.

Devuelve:

N/A

Ejemplo:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

El código anterior comprobará las vidas y si están a 0, añadirá la puntuación actual y la string guardada en la variable global "Nombre" en la lista de puntuaciones altas, antes de poner finalmente la puntuación a 0 y enviar al jugador a la room indexada en "rm_MainMenu".
