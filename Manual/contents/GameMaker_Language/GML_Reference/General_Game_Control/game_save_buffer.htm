game_save_buffer

Se trata de una variante de la función game_save(), por lo que le recomendamos que lea primero su página, ya que contiene información importante relacionada con su uso y con el de esta función.

Con esta función puedes guardar el estado actual del juego en un "crecimiento" previamente creado buffer (ver Bufferes) que luego puede ser cargado de nuevo en cualquier momento usando game_load_buffer() función. Esta función está diseñada para su uso en un solo room a la vez (es decir, no se aconseja hacer un buffer guardar en un room, a continuación, tratar de cargar el buffer de otro) y debe ser utilizado idealmente sólo para los puntos de control o reinicios de nivel.

NOTA: Esta función es muy limitada y está diseñada para que el principiante pueda poner en marcha un sistema de puntos de control rápidamente, pero los usuarios más avanzados pueden preferir codificar su propio sistema utilizando las funciones de Archivo, debido a que el juego no guardará ninguno de los recursos dinámicos como data structures, superficies, añadidos sprites etc.

Sintaxis:

game_save_buffer(buffer);

ArgumentoTipo Descripción
buffer The buffer id to save to.

Devuelve:

Ejemplo:

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

El código anterior establecerá una variable global en true y luego guardará el estado actual del juego en el buffer indexado en la variable "save_buff" cuando se pulse la tecla "S".
