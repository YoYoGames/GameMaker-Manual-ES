nombre_del_juego

Esta variable de sólo lectura devuelve el nombre de pantalla de su juego para la plataforma de destino, tal como se establece en las Opciones de Juego.

Sintaxis:

game_display_name

Devuelve:

Ejemplo:

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

El código anterior obtiene el nombre de pantalla y el número de versión del juego y los dibuja.
