cursor_sprite

Al establecer esta variable se le indicará a GameMaker que use el sprite designado como cursor (básicamente lo pondrá en la posición actual del ratón x/y en cada paso). El valor por defecto es -1 que no es sprite para el cursor, pero puedes asignar cualquier índice sprite del juego assets o que haya sido importado de un recurso externo. Ten en cuenta que no hay forma de controlar la velocidad de la animación o el índice_de_imagen, por lo que si el sprite tiene subimágenes, éstas se desplazarán a la misma velocidad que el room. Para eliminar el cursor sprite, puede volver a establecer esta variable a -1.

También hay que tener en cuenta que esta variable no sustituye al cursor de la ventana del juego, y que se seguirá dibujando de forma normal. Para evitar esto puedes utilizar la función window_set_cursor() y establecerla en la constante cr_none, lo que hará que el cursor estándar sea invisible.

Sintaxis:

cursor_sprite

Devuelve:

Sprite Asset

Ejemplo:

cursor_sprite = spr_CustomCursor;

El código anterior establecerá el sprite indexado en la variable "spr_CustomCursor" para ser el cursor sprite para el juego.
