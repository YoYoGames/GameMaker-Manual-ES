game_id DEPRECATED

Esta variable de sólo lectura devuelve el identificador único del juego que has creado. Puedes usar esto si necesitas un nombre de archivo único, o cualquier otra cosa que necesite algo para identificar tu juego solamente. Esto se puede establecer en las opciones del juego.

Sintaxis:

game_id

Devuelve:

Real

Ejemplo:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

El código anterior realiza una codificación muy básica en la puntuación añadiendo el game_id a la misma antes de guardarla. Al leerlo de nuevo en el juego, se deduce el game_id para obtener de nuevo el valor correcto.
