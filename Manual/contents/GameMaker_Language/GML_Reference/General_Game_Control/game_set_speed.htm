game_set_speed

Esta función se puede utilizar para establecer la velocidad del juego. Se puede establecer de dos maneras - como cuadros de juego por segundo (FPS) o como microsegundos por cuadro de juego (MPF) - utilizando una de las dos constantes siguientes:

Constante Descripción
gamespeed_fps Establece la velocidad del juego utilizando fotogramas por segundo.
gamespeed_microseconds Establece la velocidad del juego usando microsegundos por cuadro.

Así, por ejemplo, una velocidad de juego de 30 fotogramas por segundo sería de 33333 microsegundos por fotograma de juego, lo que se expresaría mediante esta función como

game_set_speed(30, gamespeed_fps);

o:

game_set_speed(33333, gamespeed_microseconds);

Sintaxis:

game_set_speed(speed, type);

Argumento Tipo Descripción
speed Real The new game speed (as either FPS or MPF).
type Velocidad de juego constante The type of method used to set the game speed (see the constants above).

Devuelve:

N/A

Ejemplo:

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

El código anterior comprueba si el juego se está ejecutando en un navegador y establece la velocidad del juego en consecuencia como un valor de FPS.
