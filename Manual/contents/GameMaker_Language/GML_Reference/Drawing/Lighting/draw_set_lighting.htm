draw_set_lighting

Esta función se utiliza para habilitar todos los efectos de iluminación. Por defecto está desactivado (false).

Sintaxis:

draw_set_lighting(enable);

Argumento Tipo Descripción
enable Booleano Enable or disable all lighting (true o falso)

Devuelve:

N/A

Ejemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

El código anterior habilitará la iluminación para toda la escena, luego definirá una luz blanca direccional en el espacio room, y finalmente encenderá esa luz.
