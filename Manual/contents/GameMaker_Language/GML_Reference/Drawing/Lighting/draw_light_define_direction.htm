draw_light_define_direction

Esta función sirve para definir una luz direccional, en la que la dirección y la intensidad se establecen mediante los valores introducidos para el vector unitario x, y y z (lo que significa que deben estar entre -1 y 1, donde 1 o -1 es la intensidad total, y 0 es la ausencia de intensidad). Las luces de luces también pueden recibir un color, que también afectará a la intensidad percibida de la luz, ya que ciertos colores parecen más "oscuros" que otros. También hay que dar a la luz un número de índice que es lo que se utilizará en otras funciones para referirse a ella. a ella.

NOTA: Sólo hay 8 luces de hardware disponibles, por lo que sólo se pueden activar 8 luces definidas a la vez (aunque se pueden definir más).

Sintaxis:

draw_light_define_direction(ind, x, y, z, col);

ArgumentoTipo Descripción
ind The index number of the light (arbitrary)
x The x component of the light vector
y The y component of the light vector
z The z component of the light vector
col The colour to use for the light (either a constant, a real or a hex value).

Devuelve:

Ejemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

El código anterior habilitará la iluminación para toda la escena, luego definirá una luz blanca direccional en el espacio room, y finalmente encenderá esa luz.
