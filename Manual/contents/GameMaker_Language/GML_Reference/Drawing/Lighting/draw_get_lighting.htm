dibujar_obtener_iluminación

Esta función devolverá si la iluminación está activada (true) o no (false) para toda la escena.

Sintaxis:

draw_get_lighting()

Devuelve:

Ejemplo:

if !draw_get_lighting()
{
    draw_set_lighting(true);
}

El código anterior comprobará si la iluminación está habilitada para la escena, y si no la habilita.
