draw_enable_drawevent

Con esta función puedes elegir habilitar (true) o deshabilitar (false) el evento draw para todas las instancias del juego, dándote así el control sobre cómo y cuándo se dibujan las cosas, lo cual es útil si deseas implementar una técnica de "salto de cuadros". Ten en cuenta que esto no sólo impide que las instancias dibujen en la pantalla, sino que suprime el evento de dibujo por completo, lo que significa que hay que tener cuidado, ya que cualquier lógica de juego que esté presente en ese evento tampoco se ejecutará.

Una cosa importante que hay que entender sobre esta función es que si la llamas justo al comienzo del juego, antes de que se renderice el marco inicial (es decir: el evento Create del primer object en el primer room del juego), entonces la ventana del juego no se renderizará. Esto podría ser útil para proyectos multijugador que requieran un server dedicado que no necesite renderizar nada, sin embargo tenga en cuenta que esto no lo hace headless ya que GameMaker no lo soporta -- por lo que no podrá ejecutarlo en un server que no tenga una interfaz gráfica.

Sintaxis:

draw_enable_drawevent(enable);

ArgumentoTipo Descripción
Enable Set to true o false.

Devuelve:

Ejemplo:

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

El código anterior comprueba una variable y si devuelve true, habilita el evento de sorteo, de lo contrario el evento de sorteo se desactiva.
