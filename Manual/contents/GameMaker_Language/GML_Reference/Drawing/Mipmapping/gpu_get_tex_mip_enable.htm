gpu_get_tex_mip_enable

Con esta función puedes obtener si el mipmapping está desactivado, activado para todo o activado sólo para los grupos de texturas seleccionados en el Gestor de Grupos de Texturas. La función devolverá una de las constantes que se indican a continuación, siendo el valor por defecto mip_markedonly.

Sintaxis:

gpu_get_tex_mip_enable();

Devuelve:

Constant

Constante de Mipmapping
Constante Descripción
mip_off El mapeo está desactivado.
mip_on El mapeo de Mip para todas las texturas está activado.
mip_markedonly El mapeo se activa para las texturas que lo tienen activado en las opciones del Grupo de Texturas (por defecto).

Ejemplo:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

El código anterior comprobará si el mipmapping está habilitado y, si no lo está, lo habilitará.
