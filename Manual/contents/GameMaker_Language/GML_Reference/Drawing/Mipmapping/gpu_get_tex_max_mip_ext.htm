gpu_get_tex_max_mip_ext

Con esta función se puede obtener el nivel máximo de mipmap actualmente establecido que se debe utilizar para un muestreador shader determinado. Se suministra el valor del índice del muestreador shader (devuelto por la función shader_get_sampler_index()y la función devolverá el nivel máximo de mipmap actual, donde un valor de 0 es la resolución más alta, 1 es para usar el primer mipmap, 2 es el segundo, etc... Tenga en cuenta que esto puede ser muy útil para evitar los artefactos de sangrado al renderizar las texturas, por ejemplo, establecer el borde de la página de la textura a 8px y luego establecer el nivel máximo de mipmap a 3 se asegurará de que usted no tiene ningún problema de sangrado a mayores distancias de render.

Sintaxis:

gpu_get_tex_max_mip_ext(sampler_index);

ArgumentoTipo Descripción
sampler_index The index of the shader sampler to get

Devuelve:

(default: 16)

Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
{
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el nivel máximo de mipmap a 4 para el muestreador de textura shader dado si no ha sido ya establecido a 4.
