gpu_get_tex_mip_bias_ext

Con esta función puede recuperar el valor de sesgo del mipmap para un muestreador shader dado. Se suministra el valor del índice del muestreador shader (devuelto por la función shader_get_sampler_index()y la función devolverá un valor de 0,0 para que no haya sesgo, o un valor mayor en el que 1 equivale al primer mipmap, 2 equivale al segundo mipmap, etc. Esto controla la velocidad a la que el mapa mip se intercambia y generalmente hará que las texturas de shader sean más borrosas cuanto mayor sea el valor y mayor sea la "distancia" que se está viendo. Tenga en cuenta que esto también puede devolver valores negativos, en cuyo caso las texturas de shader serán más nítidas en una distancia mayor cuanto más bajo sea el valor.

Sintaxis:

gpu_get_tex_mip_bias_ext(sampler_index);

ArgumentoTipo Descripción
sampler_index The index of the shader sampler to get

Devuelve:

(default: 0)

Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el sesgo del filtro mip a 0 para el muestreador de textura shader dado si no ha sido ya establecido a 0.
