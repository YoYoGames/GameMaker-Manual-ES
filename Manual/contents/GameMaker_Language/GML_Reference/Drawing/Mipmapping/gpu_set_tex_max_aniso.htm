gpu_set_tex_max_aniso

Con esta función se puede establecer el nivel máximo de anisotropía actual cuando se utiliza el modo de filtro tf_anisotropic (véase gpu_get_tex_mip_filter() para más información). El valor de entrada debe oscilar entre 1 y 16.

Sintaxis:

gpu_set_tex_max_aniso(maxaniso);

ArgumentoTipo Descripción
maxaniso The maximum level for anisotropic filtering (default: 16)

Devuelve:

Ejemplo:

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

El código anterior comprobará el nivel máximo de filtrado anisotrópico actual y si no es 8 se pondrá a 8.
