gpu_set_tex_mip_filter_ext

Con esta función puede establecer el modo de filtro mip para un determinado muestreador de shader. Se suministra el valor del índice del muestreador shader (devuelto por la función shader_get_sampler_index()y, a continuación, una de las constantes de valor de modo que se enumeran a continuación.

Constante Descripción
tf_point Esto significa que la mezcla entre los niveles de mipmap está desactivada, lo que puede causar transiciones de textura visibles, pero da el mejor rendimiento.
tf_linear Esto significa que la mezcla entre los niveles del mipmap está activada (esto también se conoce como filtrado trilineal), lo que suaviza las transiciones de la textura, pero dará un golpe menor al rendimiento.
tf_anisotropic Esto significa que el filtrado anisotrópico está activado, lo que mejora enormemente la calidad de la transición de las texturas y puede reducir el desenfoque visible con otros modos de filtrado, pero tiene el mayor impacto en el rendimiento.

Sintaxis:

gpu_set_tex_mip_filter_ext(sampler_index, filter);

Argumento Tipo Descripción
sampler_index Mango de muestreo de sombreado The index of the shader sampler to get.
filter Constante del filtro Mipmapping The mip filter mode to use (a constant, default: tf_point).

Devuelve:

N/A

Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_filter_ext(_sampleIndex) != tf_point)
{
    gpu_set_tex_mip_filter_ext(_sampleIndex, tf_point);
}
texture_set_stage(_sampleIndex, _spriteTex);
shader_reset();

El código anterior establece el modo de filtro mip a tf_point (deshabilitando el mipmapping) para el muestreador de textura shader dado si no ha sido ya establecido.
