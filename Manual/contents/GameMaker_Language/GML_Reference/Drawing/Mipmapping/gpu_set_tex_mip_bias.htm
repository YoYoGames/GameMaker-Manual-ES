gpu_set_tex_mip_bias

Con esta función puedes establecer el valor de sesgo del mipmap, donde un valor de 0 equivale a ningún sesgo, 1 equivale al primer mipmap, 2 equivale al segundo mipmap, etc... Esto controla la velocidad a la que el mapa mip se intercambia y generalmente hará que las texturas renderizadas sean más borrosas cuanto más alto sea el valor y mayor sea la "distancia" que se vea. Tenga en cuenta que esta función también puede tomar valores negativos, en cuyo caso las texturas renderizadas serán más nítidas a mayor distancia cuanto menor sea el valor.

Sintaxis:

gpu_set_tex_mip_bias(bias);

ArgumentoTipo Descripción
bias The mipmap bias value to use (default: 0)

Devuelve:

Ejemplo:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

El código anterior comprobará el sesgo actual del mipmap y si no es 0 se pone a 0.
