gpu_get_tex_mip_enable_ext

Con esta función puedes obtener si el mipmapping está desactivado, activado para todo o activado sólo para los grupos de texturas seleccionados en el Gestor de Grupos de Texturas en un muestreador de shader. Se proporciona el valor del índice del muestreador shader (tal y como lo devuelve la función shader_get_sampler_index()y la función devolverá una de las constantes indicadas a continuación.

Sintaxis:

gpu_get_tex_mip_enable_ext(sampler_index);

Argumento Tipo Descripción
sampler_index Mango de muestreo de sombreado The index of the shader sampler to get

Devuelve:

Constant

Constante Descripción
mip_off El mapeo está desactivado.
mip_on Se ha activado el mapeado de todas las texturas.
mip_markedonly El mapeo se activa para las texturas que lo tienen activado en las opciones del Grupo de Texturas.

Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on)
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior habilita el mipmapping para el muestreador de textura shader dado si no ha sido ya habilitado.
