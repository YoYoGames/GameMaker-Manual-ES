Mipmapping

Esta sección contiene todas las funciones relacionadas con el uso de las funciones de mipmapping. Antes de utilizar estas funciones debe haber habilitado el mipmapping para las páginas de textura requeridas en el Editor de Grupos de Textura y/o haber habilitado el mipmapping para aquellos sprites que han sido configurados para utilizar sólo una página de textura desde las Opciones Generales del Juego. La imagen de abajo muestra la diferencia que el mipmapping puede hacer al renderizar su proyecto:

Una vez que hayas habilitado el mipmapping para el proyecto o una página de textura, puedes utilizar las siguientes funciones para obtener y establecer valores que cambiarán el aspecto de tu proyecto cuando se ejecute.

Las funciones listadas a continuación pueden ser utilizadas para obtener los diferentes valores asociados con el mipmapping en GameMaker:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

Las funciones que se enumeran a continuación se pueden utilizar para establecer los diferentes valores asociados con el mipmapping en GameMaker:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
