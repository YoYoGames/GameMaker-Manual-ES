gpu_set_tex_mip_bias_ext

Con esta función puede establecer el valor de sesgo del mipmap para un muestreador shader dado. Se proporciona el valor del índice del muestreador shader (devuelto por la función shader_get_sampler_index()y luego el valor del sesgo, donde 0 es para ningún sesgo, 1 es igual al primer mipmap, 2 es igual al segundo mipmap, etc... Esto controla la velocidad a la que el mapa mip se intercambia y generalmente hará que las texturas de shader sean más borrosas cuanto más alto sea el valor y mayor sea la "distancia" que se está viendo. Tenga en cuenta que esta función también puede tomar valores negativos, en cuyo caso las texturas de shader serán más nítidas a mayor distancia cuanto menor sea el valor.

Sintaxis:

gpu_set_tex_mip_bias_ext(sampler_index, bias);

ArgumentoTipo Descripción
sampler_index The index of the shader sampler to get
bias The mipmap bias value to use (default: 0)

Devuelve:

Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el sesgo del filtro mip a 0 para el muestreador de textura shader dado si no ha sido ya establecido a 0.
