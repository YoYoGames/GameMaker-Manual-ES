gpu_set_tex_min_mip

Con esta función puedes establecer el nivel mínimo de mipmap que se utiliza actualmente, donde un valor de 0 es la máxima resolución, 1 es utilizar el primer mipmap, 2 es el segundo, etc.

Sintaxis:

gpu_set_tex_min_mip(minmip);

ArgumentoTipo Descripción
minmip The minimum mipmap level to use

Devuelve:

Ejemplo:

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

El código anterior comprobará el nivel mínimo actual del mipmap y si no es 0 se pondrá a 0.
