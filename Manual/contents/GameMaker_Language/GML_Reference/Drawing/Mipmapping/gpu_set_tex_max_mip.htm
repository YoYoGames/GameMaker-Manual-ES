gpu_set_tex_max_mip

Con esta función puedes establecer el nivel máximo de mipmap que se va a utilizar, donde un valor de 0 es la máxima resolución, 1 es utilizar el primer mipmap, 2 es el segundo, etc... Tenga en cuenta que esto puede ser muy útil para evitar los artefactos de sangrado al renderizar las texturas, por ejemplo, estableciendo el borde de la página de la textura a 8px y luego estableciendo el nivel máximo de mipmap a 3 se asegurará de que usted no tiene ningún problema de sangrado a mayores distancias de renderizado.

Sintaxis:

gpu_set_tex_max_mip(maxmip);

ArgumentoTipo Descripción
maxmip The maximum mipmap level (default: 16)

Devuelve:

Ejemplo:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

El código anterior comprobará el nivel máximo actual del mipmap y si no es 4 se pondrá a 4.
