gpu_get_tex_max_aniso_ext

Con esta función se puede obtener el nivel máximo de anisotropía actual cuando se utiliza el modo de filtro tf_anisotropic (véase gpu_get_tex_mip_filter() para más información) en un muestreador shader. Se suministra el valor del índice del muestreador shader (devuelto por la función shader_get_sampler_index()y la función devolverá un valor entre 1 y 16.

Sintaxis:

gpu_get_tex_max_aniso_ext(sampler_index);

ArgumentoTipo Descripción
sampler_index The index of the shader sampler to get

Devuelve:

(default: 16)

Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_max_aniso_ext(_sampleIndex) != 8)
{
    gpu_set_tex_max_aniso_ext(_sampleIndex, 8);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el nivel máximo de anisotropía a 8 para el muestreador de textura shader dado, si no se ha establecido ya a 8.
