gpu_set_tex_mip_filter

Con esta función puede establecer el modo de filtrado mip actual a uno de los tres tipos admitidos. Se da la constante que hace referencia al modo de filtrado mip que se requiere, donde se puede elegir entre el filtrado puntual (configuración por defecto, que significa no filtrar), el filtrado lineal (esto también se conoce como filtrado trilineal) o el filtrado aniostópico (ver la tabla de constantes más abajo).

Constante Descripción
tf_point Esto deshabilitará el filtrado entre los niveles del mipmap, que puede causar transiciones de textura visibles, pero da el mejor rendimiento.
tf_linear Esto habilitará el filtrado lineal entre los niveles del mipmap (esto también se conoce como filtrado trilineal), que suaviza las transiciones de la textura, pero dará un golpe menor al rendimiento.
tf_anisotropic Esto habilitará el filtrado anisotrópico entre los niveles del mipmap, que mejora enormemente la calidad de la transición de la textura y puede reducir el desenfoque visible con otros modos de filtrado, pero tiene el mayor impacto en el rendimiento.

Sintaxis:

gpu_set_tex_mip_filter(filter);

Argumento Tipo Descripción
filter Constante del filtro Mipmapping The mip filter mode to use (a constant, default: tf_point).

Devuelve:

N/A

Ejemplo:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

El código anterior comprueba el teclado y, si se pulsa la tecla especificada, obtendrá el filtro mip actual y cambiará el valor al siguiente, recorriendo los diferentes modos.
