dibujar_espiga_ext

Esta función dibujará el sprite dado como en la función draw_sprite() pero con opciones adicionales para cambiar la escala, la mezcla, la rotación y el alfa del sprite que se está dibujando. El cambio de estos valores no modifica el recurso de ninguna manera (sólo la forma en que se dibuja), y puede utilizar cualquiera de las variables de sprite disponibles en lugar de valores directos para todos los argumentos de la función. La imagen siguiente ilustra cómo los diferentes valores afectan al dibujo del sprite:

NOTA: La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque todavía se puede establecer el color de mezcla si no está habilitado y se mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un duplicado de sprite que se almacena en cache y se utiliza cuando se necesita. Esto está lejos de ser óptimo y si utilizas múltiples cambios de color ralentizará el rendimiento de tus juegos a no ser que actives WebGL. si no deseas utilizar WebGL, puedes establecer el tamaño de font cache para intentar limitarlo si es necesario utilizando la función sprite_set_cache_size().

NOTA: Esta función puede no funcionar como se espera cuando se utiliza la animación del esqueleto sprites, y puede encontrar que la función sólo dibuja el primer cuadro de la pose por defecto. Deberías usar las funciones de draw_skeleton_* en su lugar.

Sintaxis:

draw_sprite_ext( sprite, subimg, x, y, xscale, yscale, rot, colour, alpha );

ArgumentoTipo Descripción
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.
xscale The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale The vertical scaling of the sprite as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot The rotation of the sprite. 0=right way up, 90=rotated 90 degrees counter-clockwise etc...
colour The colour with which to blend the sprite. c_white es mostrarlo normalmente.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

Ejemplo:

draw_sprite_ext(sprite_index, image_index, x, y, image_xscale, image_yscale, image_angle, image_blend, image_alpha);

Esto dibujará las instancias asignadas sprite con todos sus valores por defecto (esencialmente lo mismo que usar draw_self).
