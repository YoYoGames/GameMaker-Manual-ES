dibujar_mapa_de_tilema

Esta función se puede utilizar para dibujar un determinado tilemap en cualquier lugar de la página room. Se le da el ID del elemento tilemap (que se obtiene cuando se crea un elemento tilemap utilizando layer_tilemap_create() o al utilizar la función layer_tilemap_get_id()), y luego la posición x/y para el dibujo (en room). Tenga en cuenta que esto simplemente dibujará el tilemap en el punto especificado, utilizando la profundidad de la capa de la instancia que está llamando a la función. No mueve el tilemap - ni lo cambia de ninguna manera - y no importa si el tilemap está marcado como visible o no.

Sintaxis:

draw_tilemap(tilemap_element_id, x, y);

Argumento Tipo Descripción
tilemap_element_id ID del elemento del mapa de baldosas The unique ID value of the tilemap element to draw
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Devuelve:

N/A

Ejemplo:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

El código anterior obtiene el ID de la capa y luego lo utiliza para obtener un ID de mapa de azulejos específico que a su vez se utiliza para dibujar el mapa de azulejos en la posición del ratón.
