draw_tile

Con esta función se puede dibujar cualquier tile a partir de un tile set - compitiendo con rotaciones, volteos y espejos - estableciendo el marco (si es animado) y la posición dentro del room. El valor de la ID de tile set es el índice de tile set tal y como se establece en el IDE y los datos de tile son el "blob" de datos que contiene toda la información sobre el índice de tile y las transformaciones que se han aplicado. Puede utilizar la función tilemap_get() para obtener los datos de tile que luego pueden ser manipulados utilizando las Funciones de Mosaico, o puede construirlos usted mismo utilizando las siguientes constantes de tile junto con el valor del índice de tile a crear:

Constante Descripción
tile_rotate Se utiliza para establecer/obtener el bit de rotación de un blob de datos de tile.
tile_mirror Se utiliza para establecer/obtener el bit de espejo de un blob de datos de tile.
tile_flip Se utiliza para establecer/obtener el bit de volteo de un blob de datos de tile.
tile_index_mask Una constante especial que sirve para "y" con el blob de datos tile para extraer el índice tile.


Estas constantes se combinan con el valor del índice de tile para obtener los datos finales:

var tiledata = 8 | tile_rotate | tile_flip;

El código anterior simplemente crea una variable local para almacenar los datos de mosaico para el tile con el índice de 8 en el tile set, y se gira y voltea. Esto puede ser usado para dibujar el tile en ese índice desde el tile set que especifiques. esto no está vinculado a ninguna capa de tilemap y el tile será dibujado en la profundidad de la capa de la instancia que está haciendo el dibujo, en la posición x/y que especifiques en el room. Ten en cuenta que también puedes establecer el marco a dibujar. Si el tile indexado es parte de un conjunto de animación, entonces puedes establecer el número de fotograma aquí (desde 0 hasta el número de fotogramas - 1) y el tile indexado será simplemente el indicador de qué tile set animado estás usando mientras que será el fotograma dado el que se dibuje. Si el tile no es parte de un conjunto animado, entonces el marco debe ser establecido en 0.

Sintaxis:

draw_tile(tileset, tiledata, frame, x, y)

Argumento Tipo Descripción
tileset Activo del conjunto de baldosas The ID of the tile set to use
tiledata Datos de las baldosas The tile data to use
frame Real The frame number for the animated tile to use (default 0 for non-animated tiles)
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Devuelve:

Real

Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

El código anterior obtendrá los datos en un tile del tilemap dado, luego establecerá el "flip" flag y usará estos datos para dibujar el tile volteado en la posición de la instancia.
