dibujar_autoridad

Esta función dibuja el sprite asignado a la instancia exactamente como se dibujaría si el evento de dibujo no contuviera ningún código o acción, y reflejará y los cambios que se hayan realizado en las variables del sprite en otros eventos.

NOTA: Si la instancia no tiene asignado ningún sprite, esta función arrojará un error.

Sintaxis:

draw_self();

Devuelve:

Ejemplo:

draw_self();

Esto hace que la instancia se dibuje a sí misma con las propiedades definidas por las variables incorporadas en sprite.
