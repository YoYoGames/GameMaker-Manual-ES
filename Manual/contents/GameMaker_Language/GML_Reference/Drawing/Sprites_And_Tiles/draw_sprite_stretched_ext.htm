dibujar_espiga_extendida

Esta función hace exactamente lo mismo que la función draw_sprite_stretched() con la capacidad añadida de establecer la mezcla de colores y el valor alfa para el sprite cuando se dibuja (similar a la función draw_sprite_ext()).

NOTA Cuando se dibuja con esta función, el desplazamiento x y el desplazamiento y (u orígenes) de sprite se ignoran y el sprite se dibuja con la esquina superior izquierda en la posición x/y especificada en el room.

NOTA La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque todavía se puede establecer el color de mezcla si no está habilitado y se mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un duplicado de sprite que se almacena en cache y se utiliza cuando se necesita. Esto está lejos de ser óptimo y si usas múltiples cambios de color ralentizará el rendimiento de tus juegos a no ser que actives WebGL. si no quieres usar WebGL, entonces puedes establecer el tamaño de font cache para intentar limitarlo si es necesario usando la función sprite_set_cache_sizee().

Sintaxis:

draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, colour, alpha);

ArgumentoTipo Descripción
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.
w The width of the area the stretched sprite will occupy.
h The height of the area the stretched sprite will occupy.
colour The colour with which to blend the sprite. c_white is to display it normally.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

N/A

Ejemplo:

draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width, sprite_height / 2, c_white, 0.5);

Esto dibujará las instancias asignadas sprite y su subimagen con la esquina izquierda en la posición x /y de la instancia. Su anchura es la misma que la de sprite, y la altura es la de sprite dividida por dos. También se mezclará normalmente pero tendrá un valor alfa parcialmente transparente.
