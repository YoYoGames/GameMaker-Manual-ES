draw_enable_swf_aa

Con esta función puede activar o desactivar el antialiasing (AA) para el formato SWF vectorial sprites. El AA simplemente suaviza los bordes de las imágenes vectoriales para darles un aspecto más agradable. La cantidad de AA utilizada dependerá del valor establecido mediante la función draw_set_swf_aa_level(). Por defecto está desactivado.

Sintaxis:

draw_enable_swf_aa(enable);

ArgumentoTipo Descripción
enable Enable (true) o desactivar (false) AA para todos los SWF sprites.

Devuelve:

Ejemplo:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

El código anterior comprobará el valor AA para el formato SWF sprites, y si es 0 habilita el AA y establece el valor a 0,5.
