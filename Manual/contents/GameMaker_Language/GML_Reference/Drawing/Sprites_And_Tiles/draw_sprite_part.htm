dibujar_parte_de_espalda

Con esta función puede dibujar parte de cualquier sprite en una posición determinada dentro de room. Al igual que con draw_sprite() puede especificar un sprite y una sub-imagen para dibujar, entonces debe dar las coordenadas relativas dentro del sprite del área a seleccionar para dibujar. Esto significa que una posición izquierda de 0 y una posición superior de 0 sería la esquina superior izquierda del sprite y todas las coordenadas posteriores deberían tomarse desde esa posición. La imagen siguiente muestra un ejemplo de cómo funciona esto:

Debe tener en cuenta que si la página de textura permite el recorte automático, esta función puede no funcionar como se espera, ya que el espacio extra "vacío" alrededor de sprite se habrá eliminado para crear la página de textura. Para resolver este problema, tendrá que establecer la configuración de la página de textura (en el Editor de grupos de texturas) para desactivar la opción Recorte automático.

NOTA: Al dibujar con esta función, se ignoran el desplazamiento x y el desplazamiento y de sprite y la parte sprite se dibujará con la esquina superior izquierda en la posición x / y especificada en el room.

NOTA: Esta función sólo es útil para el mapa de bits sprites y no funcionará con SWF o JSON (Spine) sprites.

Sintaxis:

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

ArgumentoTipo Descripción
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
left The x position on the sprite of the top left corner of the area to draw.
top The y position on the sprite of the top left corner of the area to draw.
width The width of the area to draw.
height The height of the area to draw.
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.

Devuelve:

Ejemplo:

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

Esto dibujará las instancias asignadas sprite (sprite_index) y su cuadro actual de animación (image_index), sin embargo, recortará un margen de 4px de la anchura en ambos lados, y un margen de 8 píxeles de la altura desde la parte inferior del original de 24x24 pixel sprite.
