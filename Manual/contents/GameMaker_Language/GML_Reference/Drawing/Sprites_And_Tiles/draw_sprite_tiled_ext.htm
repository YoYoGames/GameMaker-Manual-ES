dibujar_espiga_etiqueta_acodada_ext

Esta función tomará un sprite y luego repetidamente tile a través de toda la vista (o room si no hay vista definida), comenzando desde las coordenadas que usted da en la función y con cada tile escalado, mezclado de color y con el alfa que usted defina (estas propiedades son las mismas que se utilizan en draw_sprite_ext()). Esta función es sólo para proyecciones 2D (ortográficas), y no funcionará correctamente cuando se utilice una proyección de cámara 3D.

NOTA: La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque todavía se puede establecer el color de mezcla si no está habilitado y se mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un duplicado sprite que se almacena en cache y se utiliza cuando es necesario. Esto está lejos de ser óptimo y si utiliza múltiples cambios de color ralentizará el rendimiento de sus juegos a menos que active WebGL. si no desea utilizar WebGL, entonces puede establecer el tamaño de font cache para intentar limitarlo en caso de que sea necesario utilizando la función sprite_set_cache_size().

Sintaxis:

draw_sprite_tiled_ext(sprite, subimg, x, y, xscale, yscale, colour, alpha);

ArgumentoTipo Descripción
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.
xscale The horizontal scaling of the sprite. A multiplier ' 1 = normal scaling, 0.5 is half etc.
yscale The vertical scaling of the sprite. A multiplier ' 1 = normal scaling, 0.5 is half etc.
colour The colour with which to blend the sprite. c_white is to display it normally.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

Ejemplo:

draw_sprite_tiled_ext(sprite_index, image_index, x, y, 2, 2, c_red, 0.5);

Esto dibujará la instancia asignada sprite (sprite_index) y su cuadro actual de animación (image_index) en la propia posición x e y de la instancia, pero escalada al doble del tamaño normal, mezclada en rojo y con la mitad del alfa normal. La página sprite se colocará en forma de mosaico horizontal y verticalmente a través de la vista.
