draw_sprite_pos

Con esta función se puede dibujar un sprite distorsionado sobre el área definida por las cuatro coordenadas de las esquinas. Los dos primeros argumentos son el sprite a dibujar y la subimagen del sprite (la misma que en la función draw_sprite()) pero los siguientes son los que definen la posición de cada una de las cuatro esquinas de la caja delimitadora del sprite dado. Estas coordenadas deben darse en el sentido de las agujas del reloj, por lo que la primera coordenada es la superior izquierda, luego la superior derecha, luego la inferior derecha y finalmente la inferior izquierda. También puede establecer un valor para el alfa del sprite para dibujarlo con transparencia. La imagen siguiente ilustra cómo funciona esta función:

¡ATENCIÓN! La imagen de arriba es sólo para fines ilustrativos, y si usted utiliza esta función en un sprite, obtendrá resultados diferentes y puede experimentar "cizallamiento" de la textura debido a la forma en que un sprite se construye a partir de un quad de primitivas.

NOTA: Esta función sólo es útil para el mapa de bits sprites y no funcionará con SWF o JSON (Spine) sprites.

NOTA: Al utilizar esta función, debe tener desactivado el ajuste Recorte automático para las páginas de textura.

Sintaxis:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

ArgumentoTipo Descripción
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

Ejemplo:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

El código anterior dibujará el sprite asociado a la instancia que ejecuta el código distorsionado alrededor de la posición x / y de la instancia y con un alfa totalmente opaco.
