draw_sprite_part_ext

Esta función dibujará una parte de la sprite elegida en la posición dada siguiendo las mismas reglas que por draw_sprite_part()sólo que ahora puedes escalar la parte, mezclar un color con ella, o cambiar su alfa cuando al dibujarla en la pantalla (igual que cuando se dibuja un sprite con draw_sprite_ext()). Debe tener en cuenta que si la página de la textura permite el recorte automático, esta función puede no funcionar como se espera, ya que el espacio extra "vacío" alrededor de sprite se habrá eliminado para crear la página de textura. Para resolver este problema, tendrá que establecer la configuración de la página de textura (en el Editor de grupos de texturas) para desactivar el Recorte automático.

NOTA Al dibujar con esta función, se ignoran el desplazamiento x y el desplazamiento y de sprite y la parte sprite se dibujará con la esquina superior izquierda en la posición x / y especificada en el room.

NOTA: Esta función sólo es útil para el mapa de bits sprites y no funcionará con SWF o JSON (Spine) sprites.

NOTA: La mezcla de colores sólo se recomienda para el objetivo HTML5 cuando WebGL está habilitado, aunque todavía se puede establecer el color de mezcla si no está habilitado y se mezclará el sprite de forma normal. Sin embargo, toda mezcla de este modo crea un duplicado sprite que se almacena en cache y se utiliza cuando es necesario. Esto está lejos de ser óptimo y si utiliza múltiples cambios de color ralentizará el rendimiento de sus juegos a menos que active WebGL. si no desea utilizar WebGL, entonces puede establecer el tamaño de font cache para intentar limitarlo en caso de que sea necesario utilizando la función sprite_set_cache_size().

Sintaxis:

draw_sprite_part_ext(sprite, subimg, left, top, width, height, x, y, xscale, yscale, colour, alpha);

ArgumentoTipo Descripción
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
left The x position on the sprite of the top left corner of the area to draw.
top The y position on the sprite of the top left corner of the area to draw.
width The width of the area to draw.
height The height of the area to draw.
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.
xscale The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
colour The colour with which to blend the sprite. c_white is to display it normally.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devuelve:

Ejemplo:

draw_sprite_part_ext(sprite_index, image_index, 8, 8, sprite_width-16, sprite_height-16, x, y, 2, 0.5, c_black, 1);

Esto dibujará las instancias asignadas sprite (sprite_index) y su cuadro actual de animación (image_index), sin embargo, afeitará un margen de 8px de los cuatro lados de la sprite. Luego se estirará al doble de su ancho habitual pero la mitad de su altura, y aunque el alfa sigue siendo 1, se mezclará con negro (convirtiéndolo en una silueta).
