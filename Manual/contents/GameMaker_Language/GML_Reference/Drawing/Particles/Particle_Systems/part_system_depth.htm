profundidad_sistema_parte

Con esta función puedes establecer la profundidad de dibujado para el sistema de partículas, de la misma manera que puedes establecer la profundidad de renderizado de las diferentes capas dentro de room, donde una profundidad de dibujado baja significa que aparecerá encima de todas las cosas dibujadas con una profundidad más alta, y una profundidad de dibujado alta colocándola debajo de todo con una profundidad de dibujado más baja.

Sintaxis:

part_system_depth( ind, depth );

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to change.
depth Real The depth at which to set the particle system.

Devuelve:

N/A

Ejemplo:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

El código anterior creará un sistema de partículas y almacenará su índice en la variable global "Sname". A este sistema se le da una profundidad baja de -1000, lo que significa que aparecerá por encima de todo lo que tenga una profundidad de dibujo mayor.
