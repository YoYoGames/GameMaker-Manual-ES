part_system_draw_order

Con esta función puedes establecer la forma en que se dibujan las partículas cuando se crean en la pantalla. El sistema por defecto utiliza un aspecto viejo > nuevo (la función está configurada a true), donde las partículas viejas se dibujan a una profundidad mayor que las nuevas y así aparecen "debajo" de ellas, pero configurando esta función a false puedes invertir este orden y hacer que las partículas nuevas se dibujen más arriba y así aparezcan "debajo" de las viejas. Las imágenes de abajo ilustran esto, con la imagen de la izquierda siendo el valor por defecto de true y la imagen de la derecha siendo false:

NOTA: Cuando las partículas se dibujan con un modo de mezcla aditivo, el efecto de esta función puede no ser siempre obvio.

Sintaxis:

part_system_draw_order(ind, oldtonew);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to change.
oldtonew Booleano Whether old particles should be drawn behind newer ones (true) or vice versa (false).

Devuelve:

N/A

Ejemplo:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Esto creará un nuevo sistema de partículas con el índice "mysystem", y luego establece las partículas para dibujar las partículas más nuevas sobre las más antiguas.
