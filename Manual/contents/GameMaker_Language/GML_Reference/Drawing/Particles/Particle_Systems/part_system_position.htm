posición_sistema_parte

Con esta función puede establecer la posición base para el sistema de partículas en relación con la posición (0,0) del room, lo que significa que todas las funciones de partículas posteriores relacionadas con este sistema se dibujarán ahora en relación con la nueva posición. Por defecto, esta posición es siempre (0,0), pero en algunas circunstancias muy especiales puede desear cambiarla por otra.

NOTA: Esta función cambiará todo dentro del sistema de partículas, así que si tienes un emisor en la posición (100,100) y luego estableces la posición del sistema de partículas en (0,100), el emisor se dibujará ahora en (100,200). Lo mismo ocurre si cambias el sistema y luego creas el emisor, ya que aunque lo crees en (100,100) se dibujará en (100,200).

Sintaxis:

part_system_position(ind, x, y);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to change.
x Real The new x coordinate of the particle system.
y Real The new y coordinate of the particle system.

Devuelve:

N/A

Ejemplo:

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

El código anterior comprobará si se pulsa el botón del ratón y, si lo detecta, el sistema de partículas indexado en la variable global "Sname" se desplaza a la posición x/y del ratón
