part_system_update

Esta función avanzará el sistema de partículas dado un paso en el tiempo del juego y puede ser usada para simular el comportamiento del sistema de partículas cuando la actualización automática está desactivada, o puede ser usada para avanzar un sistema de partículas hasta un punto específico en un solo paso. Por ejemplo, si tienes un sistema de partículas para la nieve, y quieres que parezca que la nieve está cayendo desde el momento en que el jugador entra en la room entonces puedes querer avanzar tus partículas de nieve durante unos segundos en el evento de creación de una instancia para que la primera vez que se dibujen, haya partículas por toda la pantalla (el ejemplo de código de abajo lo muestra). También puede utilizar esta función cuando la actualización automática del sistema está desactivada por la función part_system_automatic_update() para hacer avanzar el sistema.

Sintaxis:

part_system_update(ind);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to update.

Devuelve:

N/A

Ejemplo:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

El código anterior hará avanzar el sistema de partículas indexado en la variable global "RainSys" durante 3 segundos.
