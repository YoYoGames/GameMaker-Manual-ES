part_particles_count

Con esta función se puede comprobar si un sistema de partículas tiene actualmente alguna partícula creada en el room, y devolverá también el número de ellas.

Sintaxis:

part_particles_count(ind);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system.

Devuelve:

Real

Ejemplo:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

El código anterior comprobará el número de partículas en el sistema de partículas local indexado en la variable "Sname" y si no hay ninguna, destruirá el sistema y luego la instancia.
