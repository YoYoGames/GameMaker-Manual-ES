part_system_update

Con esta función puedes forzar a un sistema de partículas a dibujar todas las partículas actuales en la pantalla. Si part_system_automatic_draw() está desactivada, esta función mostrará las partículas cuando se utilice en el evento de dibujado de una instancia, o puede utilizarse cuando el objetivo de dibujado se establezca en una superficie (utilizando surface_set_target()) para dibujar las partículas del sistema en esa superficie.

Sintaxis:

part_system_drawit(ind);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to draw the particles of.

Devuelve:

N/A

Ejemplo:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

El código anterior comprueba si la superficie indexada en la variable "surf" existe, y si lo hace entonces establece el objetivo de dibujo a la superficie, dibuja el sistema de partículas con el ID almacenado en la variable global, y luego restablece el objetivo de dibujo para que todo el dibujo normal aparezca en la pantalla una vez más.
