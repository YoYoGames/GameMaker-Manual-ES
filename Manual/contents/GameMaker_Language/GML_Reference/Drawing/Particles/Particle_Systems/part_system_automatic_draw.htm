part_system_automatic_draw

Esta función se puede utilizar para switch desactivar el dibujo de un sistema de partículas de modo que cualquier actualización realizada al sistema (automática o de otro tipo) no se vea. Esta es una opción puramente visual y cuando se establece en false usted no será capaz de ver las partículas, ya que no se dibujan, pero todavía existe y están cambiando de posición, color, etc ... como lo harían normalmente. Cuando el dibujo automático está desactivado, puede ordenar explícitamente a GameMaker que dibuje el estado actual del sistema de partículas utilizando la función part_system_drawit(), y si estableces esta función en true de nuevo puedes switch volver a activar el dibujo automático.

Hay que tener en cuenta que si se utilizan los efectos simples creados por las funciones effect_create_above() o effect_create_below() entonces puedes usar los valores 0 (para los efectos de abajo) o 1 (para los efectos de arriba) como índice del sistema de partículas y así activar el dibujo automático para estos también (esto también funcionará para activar el dibujo para los efectos de partículas de GML Visual).

Sintaxis:

part_system_automatic_draw(ind, automatic);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to change.
automatic Booleano Whether automatic drawing is on (true) or not (false).

Devuelve:

N/A

Ejemplo:

part_system_automatic_draw(global.Sname, false);

El código anterior desactiva el dibujo automático para el sistema de partículas indexado en la variable global "Sname".
