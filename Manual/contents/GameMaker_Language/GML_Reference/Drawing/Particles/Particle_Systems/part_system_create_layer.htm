part_system_create_layer

Esta función creará un nuevo sistema de partículas en una capa determinada. Se le da el ID de capa único devuelto por la función layer_create() o el nombre de la capa a utilizar como string - por ejemplo "instance_layer" - y luego flag el sistema como persistente o no. Si el sistema no está marcado como persistente, se destruirá automáticamente al final de la room en la que fue creado (esto es lo mismo que si hubieras llamado a la función part_system_destroy() y también destruirá los emisores asociados al sistema). Sin embargo, cuando se marca como persistente, el sistema se llevará a la siguiente room cuando se cambie la room, y si la siguiente room no tiene una capa con el mismo nombre o profundidad que la asignada, entonces se creará una nueva capa para el sistema que persiste a través de la rooms, y tendrá el mismo nombre que la capa original. Al cambiar rooms, si hay otra capa en el siguiente rooms con el mismo nombre, entonces la instancia persistida se asignará a la capa con ese nombre independientemente de la profundidad de la capa. Por último, si un sistema persistente se mueve a un room con una capa con la misma profundidad que la capa en la que se creó el sistema, no se asignará a esta capa, sino que se creará una nueva capa con la misma profundidad (con el mismo nombre que la capa original).

La función devolverá un valor de ID único para el sistema de partículas que deberá utilizarse en todas las llamadas posteriores a la función en las que sea necesario dar un ID del sistema.

¡IMPORTANTE! Si usted flag el sistema de partículas como persistente, entonces él (y cualquier emisor asignado a él) tendrá que ser limpiado manualmente usando las funciones de destrucción apropiadas cuando no esté en uso, de lo contrario se corre el riesgo de una fuga de memoria que afectará negativamente a su juego final.

Sintaxis:

part_system_create_layer(layer_id, persistent);

Argumento Tipo Descripción
layer Cadena o ID de capa The layer ID (or name) to assign the particle system to (can be any layer type)
persistent Booleano Flag the particle system as persistent (set to true) o no (ajustado a false)

Devuelve:

Particle System ID

Ejemplo:

global.p_sys = part_system_create_layer("effects_layer", true);

El código anterior creará un nuevo sistema de partículas en la capa dada y flag como persistente en las siguientes rooms. El ID del sistema de partículas se almacena en una variable de ámbito global para futuras referencias.
