part_particles_create

Esta función es ideal para aquellos efectos que no requieren ninguna de las funcionalidades ofrecidas por los emisores de partículas (por ejemplo, para crear humo de un misil, o un simple efecto de explosión) ya que permite crear rápida y fácilmente partículas en cualquier posición del juego room. Tenga en cuenta que debe haber creado el sistema de partículas y el tipo de partículas previamente para que esta función pueda ser utilizada.

Sintaxis:

part_particles_create(ind, x, y, parttype, number);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Tipo de partícula ID The index (type) of the particles to be created.
number Real The number of particles to create.

Devuelve:

N/A

Ejemplo:

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

El código anterior comprueba si se ha pulsado el botón del ratón y si devuelve true genera 5 partículas en la posición del ratón.
