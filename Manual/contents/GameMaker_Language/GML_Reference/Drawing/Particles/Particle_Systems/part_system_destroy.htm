part_system_destroy

Con esta función puedes destruir un sistema de partículas dado y eliminarlo de la memoria. Esto siempre debe ser llamado cuando el sistema ya no es necesario, como al final de un room, o en el evento de destrucción de una instancia, de lo contrario podría terminar con las partículas que aparecen en rooms más tarde y no hay manera de eliminarlos, así como una fuga de memoria que eventualmente bloqueará su juego.

NOTA: Esta función también destruirá y eliminará cualquier emisor que haya sido creado y asociado con el sistema que se está destruyendo.

Sintaxis:

part_system_destroy(ind);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to destroy.

Devuelve:

N/A

Ejemplo:

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

El código anterior comprueba si el valor de una variable global es menor o igual a cero, y si lo es entonces destruye el sistema de partículas referenciado en la variable dada y luego pasa a la siguiente room.
