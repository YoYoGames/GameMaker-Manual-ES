capa_sistema_parte

Esta función se puede utilizar para switch un sistema de partículas de su capa actual a una nueva. Se suministra el valor del ID del sistema de partículas (devuelto por la función part_system_create_layer()) y luego el ID de la capa única (devuelto por la función layer_create() o el nombre de la capa a utilizar como string - por ejemplo "instance_layer" - como se define en el editor room ), y el sistema se moverá a la nueva capa.

Sintaxis:

part_system_layer(ps, layer);

Argumento Tipo Descripción
ps ID del sistema de partículas The particle system ID
layer Cadena o ID de capa The new layer ID

Devuelve:

N/A

Ejemplo:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

El código anterior comprobará un sistema de partículas para ver en qué capa está y si no está en la capa dada se cambiará.
