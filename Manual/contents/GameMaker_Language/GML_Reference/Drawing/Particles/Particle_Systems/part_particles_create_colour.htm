part_particles_create_colour

Esta función es ideal para aquellos efectos que no requieren ninguna de las funcionalidades ofrecidas por los emisores de partículas (por ejemplo, para crear humo de un misil, o un simple efecto de explosión) ya que permite crear rápida y fácilmente partículas en cualquier posición del juego room. También puedes colorear las partículas "sobre la marcha" a medida que se crean con esta función, y este color anulará el color predefinido de la partícula, pero no mezcla este color durante la vida de las partículas. Tenga en cuenta que debe haber creado el sistema de partículas y el tipo de partícula previamente para que esta función pueda ser utilizada.

Sintaxis:

part_particles_create_colour(ind, x, y, parttype, colour, number);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Tipo de partícula ID The index (type) of the particles to be created.
colour Color The colour blending for the particles.
number Real The number of particles to create.

Devuelve:

N/A

Ejemplo:

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

El código anterior generará una sola partícula cada paso que la instancia con el código tenga una velocidad mayor que uno. Estas partículas serán de un color aleatorio.
