part_system_clear

Con esta función puedes borrar el sistema indexado a su estado por defecto, eliminando todos los emisores y restableciendo la profundidad y la posición (si se habían cambiado). Tenga cuidado al usar esta función ya que si tiene algún ajuste de instancia o estallido o cualquier otra acción que use un emisor asociado a este sistema, obtendrá un error a menos que esté usando la función part_emitter_exists() para comprobarlo. Tampoco es necesario llamar a la función part_emitter_destroy() ya que esto también se hace automáticamente.

NOTA: esta función borrará las partículas visibles en el room, pero no borra las propiedades de las partículas, ni las elimina de la memoria. Para ello debe utilizar las funciones part_type_clear() y part_type_destroy().

Sintaxis:

part_system_clear(ind);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to clear.

Devuelve:

N/A

Ejemplo:

part_system_clear(global.Sname);

El código anterior borrará el sistema de partículas indexado en la variable global "Sname" a su estado por defecto.
