part_system_automatic_update

Esta función controla si GameMaker debe actualizar el sistema de partículas automáticamente o no, y normalmente usted no necesitaría usar esta función ya que el valor por defecto de true (la actualización automática está activada) es lo que usted desea que suceda. Sin embargo, para efectos especiales o para pausar el juego, puede establecer esto en false y evitará que se realice cualquier actualización en el sistema de partículas dado a menos que se ordene explícitamente mediante el uso de la función part_system_update() o que utilices esta función de nuevo para activar las actualizaciones automáticas. No actualizar significa que las partículas creadas en este sistema ya no cambiarán de posición, color o cualquier otro parámetro y que los emisores dejarán de funcionar también, "congelando" el sistema en el punto exacto en el que se estableció la actualización automática en false.

Hay que tener en cuenta que si se utilizan los efectos simples creados por las funciones effect_create_above() o effect_create_below() puedes utilizar los valores 0 (para los efectos inferiores) o 1 (para los efectos superiores) como índice del sistema de partículas y así activar la actualización automática para estos también (esto también funcionará para activar la actualización de los efectos de partículas de GML Visual).

Sintaxis:

part_system_automatic_update(ind, automatic);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system to change.
automatic Booleano Whether automatic updating is on (true) o no (falso).

Devuelve:

N/A

Ejemplo:

if global.Pause
{
    part_system_automatic_update(global.Sname, false);
}

El código anterior switch desactivará las actualizaciones de las partículas si la variable global "Paused" es verdadera.
