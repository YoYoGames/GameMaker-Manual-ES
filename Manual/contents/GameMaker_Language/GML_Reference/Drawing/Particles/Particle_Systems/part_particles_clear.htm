part_particles_clear

Con esta función puedes borrar todas las partículas creadas actualmente por el sistema desde la página room. No restablece o elimina los tipos de partículas en sí, sólo su representación visual, y si tienes alguna object transmitiendo partículas desde un emisor, estas partículas desaparecen pero comenzarán a aparecer de nuevo al siguiente paso después de llamar a este código.

Sintaxis:

part_particles_clear(ind);

Argumento Tipo Descripción
ind ID del sistema de partículas The index of the particle system.

Devuelve:

N/A

Ejemplo:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

El código anterior comprobará el valor de la variable "vidas" y si es igual a 0, borra todas las partículas del sistema y luego cambia room.
