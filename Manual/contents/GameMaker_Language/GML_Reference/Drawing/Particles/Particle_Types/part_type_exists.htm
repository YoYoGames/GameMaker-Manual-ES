tipo_parte_existe

Con esta función puedes comprobar si el tipo de partícula indexado existe en el juego o no. Tenga en cuenta que si la variable que se está comprobando es una variable no inicializada (a la que un tipo de partícula tendría asignado su índice) esto arrojará un error.

Sintaxis:

part_type_exists(ind);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to check for.

Devuelve:

Boolean

Ejemplo:

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

El código anterior comprueba si la variable global "p1" almacena el índice de un tipo de partícula y si no lo hace crea una.
