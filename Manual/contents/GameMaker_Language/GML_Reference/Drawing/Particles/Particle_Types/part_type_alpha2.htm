part_type_alpha2

Esta función puede utilizarse para establecer un gradiente de valor alfa (transparencia) para cada partícula creada del tipo dado. El primer alfa es con el que empezarán todas las partículas, y el segundo alfa es con el que terminará la partícula, y se producirá un suave cambio de gradiente en el alfa a lo largo de la vida de las partículas de uno a otro. Puede ser de 0 (transparente) a 1 (opaco).

Sintaxis:

part_type_alpha2(ind, alpha1, alpha2);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
alpha1 Real The start alpha of the particle.
alpha2 Real The end alpha of the particle.

Devuelve:

N/A

Ejemplo:

part_type_alpha2(global.Snow_Part, 0, 1);

El código anterior establecerá todas las partículas creadas del tipo de partícula indexado en la variable global "Snow_Part" para tener un valor alfa de 0 (transparente) y luego se desvanecerá para tener un alfa de 1 (opaco) al final de su vida.
