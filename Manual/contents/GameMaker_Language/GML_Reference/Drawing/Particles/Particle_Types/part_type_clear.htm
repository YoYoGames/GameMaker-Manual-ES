tipo_parte_claro

Con esta función puedes "resetear" una partícula, devolviendo todos los valores de cada una de las funciones relativas a la partícula (vida, color, alfa, orientación, etc...) a sus valores por defecto. Tenga en cuenta que esta función no elimina de la pantalla ninguna de las partículas visibles en ese momento en room, para ello debería utilizar part_particles_clear().

Sintaxis:

part_type_clear(ind);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to clear.

Devuelve:

N/A

Ejemplo:

part_type_clear(global.explode_part);

El código anterior borrará el tipo de partícula indexado en la variable global "explode_part" a sus valores por defecto.
