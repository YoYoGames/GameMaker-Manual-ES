tipo_parte_color3

Esta función puede utilizarse para establecer un gradiente de tres colores para cada partícula creada del tipo dado. El primer color es con el que empezarán todas las partículas, el segundo color es el que se mezclará a mitad de su vida y el tercer color es con el que terminará la partícula. A lo largo de la vida de las partículas se producirá un suave cambio de gradiente a través de los colores, pasando de un color a otro.

Sintaxis:

part_type_colour3( ind, colour1, colour2, colour3 );

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
colour1 Color The colour the particle will start at.
colour2 Color The colour the particle will be halfway through its lifespan.
colour3 Color The colour the particle will end at.

Devuelve:

N/A

Ejemplo:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

El código anterior hará que todas las partículas creadas del tipo indexado en la variable "part_Health" tengan una mezcla de colores de rojo a blanco y a granate durante su vida.
