part_type_blend

Con esta función puedes hacer que tus partículas se dibujen con un modo de mezcla aditivo (true) o no (false). La mezcla aditiva es un modo de mezcla especial que añade los valores de luminosidad de cada partícula a medida que se superponen, de modo que los colores claros se vuelven gradualmente más brillantes (hasta que parecen blancos) a medida que se superponen, y los colores oscuros se vuelven más y más transparentes, siendo el negro casi invisible. Esta función puede dar algunos efectos hermosos de partículas, pero no siempre es necesaria. Por ejemplo, una estela de humo tendría la mezcla aditiva desactivada para mantener el efecto gris, pero un efecto de llama probablemente la tendría activada para hacer las partículas más translúcidas y brillantes.

Sintaxis:

part_type_blend(ind, additive);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
additive Booleano Whether the particles should be blended additively (true) or normally (false).

Devuelve:

N/A

Ejemplo:

part_type_blend(part_Fire, true);

El código anterior establecerá todas las partículas creadas del tipo de partícula indexado en la variable "part_Fire" para tener un modo de mezcla aditivo.
