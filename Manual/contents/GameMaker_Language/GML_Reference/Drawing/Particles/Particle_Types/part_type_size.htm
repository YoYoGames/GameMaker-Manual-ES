part_type_size

Esta función se utiliza para determinar el tamaño de la partícula cuando se crea y también puede utilizarse para hacer que la partícula aumente o disminuya su tamaño a lo largo de su vida. Los valores de tamaño mínimo y máximo se fijan por defecto en 1 (lo que significa que el tamaño es la misma proporción que la partícula base después de part_type_scale() escala) y si se fijan en el mismo valor, el tamaño inicial de todas las partículas será el mismo, pero si se fijan en valores diferentes, cada partícula empezará con un tamaño aleatorio dentro del rango, por ejemplo si tienes un mínimo de 0,5 y un máximo de 2, todas las partículas se crearán con un tamaño entre la mitad y el doble de la escala base.

También puede establecer un valor de incremento que hará que la partícula sprite crezca (si es un número positivo) o se reduzca (si es un número negativo) durante el tiempo de vida de la partícula, añadiendo la cantidad dada al tamaño de la partícula en cada paso. Este valor puede ser un mínimo de (+/-)0,01.

Por último, puede establecer el factor de "meneo". Este es un valor que se sumará o restará aleatoriamente al tamaño en cada paso de la vida de las partículas. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede ser incluso negativo, estando el rango máximo entre -20 y 20.

Sintaxis:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
size_min Real The minimum size the particle can start at.
size_max Real The maximum size the particle can start at.
size_incr Real How much the particle should increase or decrease per step.
size_wiggle Real How much should randomly be added or subtracted from the particles size per step.

Devuelve:

N/A

Ejemplo:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

El código anterior establecerá varios valores de las partículas, incluyendo el tamaño, que se establece en el doble del tamaño cuando se crea cada partícula, y luego incrementa el tamaño en 0,01 en cada paso con un meneo de 0,1 también.
