parte_tipo_gravedad

Esta función establecerá la gravedad que debe afectar a cada partícula del tipo dado que se crea. El valor de la fuerza de gravedad se añade a la velocidad de la partícula a cada paso y es normalmente un valor pequeño como 0.5, mientras que la dirección es la dirección del "tirón" de la gravedad y sigue las direcciones estándar de GameMaker de 0° siendo derecha, 90° siendo arriba, 180° siendo izquierda y 270° siendo abajo.

Sintaxis:

part_type_gravity(ind, grav_amount, grav_direction);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
grav_amount Real Strength of the gravity.
grav_direction Real The direction of the gravity.

Devuelve:

N/A

Ejemplo:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

El código anterior establecerá varios valores de la partícula, incluyendo la gravedad, que añadirá 0,2 a la velocidad en cada paso con una dirección de 270º, por lo que tirará de la partícula "hacia abajo", hacia el fondo de la pantalla.
