part_type_destroy

Con esta función puedes eliminar del juego el tipo de partícula especificado. Cuando se utiliza esta función, todas las partículas del tipo dado desaparecerán de la room y la propia partícula se elimina de la memoria, por lo que esta función debe utilizarse sólo cuando ya no se necesita la partícula.

Sintaxis:

part_type_destroy(ind);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to destroy.

Devuelve:

N/A

Ejemplo:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

El código anterior comprueba un sistema de partículas para ver si hay alguna partícula visible en ese momento en la página web room y, si no es así, se destruyen las partículas, el sistema y la instancia.
