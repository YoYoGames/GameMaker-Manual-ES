part_type_sprite

Esta función se puede utilizar para establecer un tipo de partícula para utilizar un sprite personalizado del juego assets. Puedes seleccionar el sprite y luego decirle a GameMaker que use el sprite y lo anime o no, y si eliges animado, entonces también puedes elegir que las sub-imágenes cambien en un intervalo tal que la animación comience y termine al principio y al final del tiempo de vida de cada partícula creada (de modo que si tienes un tiempo de vida establecido en un mínimo de 15 pasos y un máximo de 30 pasos, no importa qué valor tenga el tiempo de vida la animación se hará para ajustarse, con ella corriendo más rápido para los tiempos de vida más cortos y más lento para los más largos). También puedes seleccionar elegir una sub-imagen aleatoria, de modo que si la sprite no está animada, se elegirá una imagen diferente al azar para crear la sprite, o si la partícula está animada, comenzará la animación en un punto aleatorio.

Sintaxis:

part_type_sprite(ind, sprite, animate, stretch, random);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
sprite Activo Sprite The index of the sprite to set the particle type to.
animate Booleano Whether to follow the sprite's animation normally (true) or not (false).
stretch Booleano Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random Booleano Whether to choose a random sub-image (true) or not (false).

Devuelve:

N/A

Ejemplo:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

El código anterior establece el tipo de partícula indexado en la variable global "p1" a la forma de la sprite indexada en "spr_Coins", y luego establece varias otras propiedades de las partículas.
