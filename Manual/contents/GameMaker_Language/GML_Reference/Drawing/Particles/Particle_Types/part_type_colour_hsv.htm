part_type_colour_hsv

Con esta función puede establecer un rango de tono, saturación y valor para todas las partículas del tipo dado. Usted proporciona un valor mínimo y un valor máximo para cada uno de los tres componentes y las partículas creadas tendrán un color aleatorio basado en el rango de parámetros dado. De esta manera puede crear partículas del mismo tono pero con diferentes saturaciones, o de diferentes tonos pero con el mismo valor (luminosidad), etc. Todos los valores deben estar entre 0 y 255.

Sintaxis:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
hmin Real The minimum the final colour's hue component can be.
hmax Real The maximum the final colour's hue component can be.
smin Real The minimum the final colour's saturation component can be.
smax Real The maximum the final colour's saturation component can be.
vmin Real The minimum the final colour's value component can be.
vmax Real The maximum the final colour's value component can be.

Devuelve:

N/A

Ejemplo:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

El código anterior establece que cada partícula emitida del tipo de partícula indexada en la variable global "Estrellas" tenga diferentes colores y saturaciones, pero el mismo valor (luminosidad).
