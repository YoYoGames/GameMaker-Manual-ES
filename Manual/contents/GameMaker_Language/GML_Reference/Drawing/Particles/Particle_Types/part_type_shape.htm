tipo_de_parte_forma

Con esta función se puede establecer la forma de sprite a utilizar para el tipo de partícula, siendo aceptables las siguientes constantes:

Constante de forma de la partícula
Constante Descripción
pt_shape_pixel Un píxel de 1x1. (Esta es la configuración por defecto).
pt_shape_disk Un círculo lleno.
pt_shape_square Un cuadrado lleno.
pt_shape_line Una línea horizontal de 8px de ancho.
pt_shape_star Una estrella de cinco puntas rellena.
pt_shape_circle Un círculo con un contorno de 3px.
pt_shape_ring Un círculo con un brillo interior (parece una burbuja).
pt_shape_sphere Un círculo con un resplandor exterior ' sólido en el centro, brillando hacia afuera.
pt_shape_flare Un punto que brilla con fuerza (parece una estrella real en la noche).
pt_shape_spark Un efecto de chispa ' como una estrella con múltiples puntos que se desvanecen.
pt_shape_explosion Una nube de humo cuadrada ' requiere múltiples colores para parecerse a una explosión.
pt_shape_cloud Una nube delgada, requiere un aumento de escala y múltiples partículas para parecerse a una nube.
pt_shape_smoke Una versión suave del efecto de explosión. Utiliza varios para crear una nube de humo.
pt_shape_snow Una forma genérica de copo de nieve.

Sintaxis:

part_type_shape(ind, shape);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to set.
shape Constante de forma de la partícula The shape to choose.

Devuelve:

N/A

Ejemplo:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

El código anterior establece el tipo de partícula indexado en la variable global "p1" a la forma de un cuadrado, y luego establece varias otras propiedades de la partícula.
