part_type_colour_rgb

Con esta función puede establecer la mezcla de colores rojo, verde y azul para todas las partículas creadas del tipo dado. Usted proporciona un valor mínimo y un valor máximo para cada uno de los tres componentes y las partículas creadas tendrán un color aleatorio basado en el rango de parámetros dado. Todos los valores deben estar entre 0 y 255.

Sintaxis:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
rmin Real The minimum the final colour's red component can be.
rmax Real The maximum the final colour's red component can be.
gmin Real The minimum the final colour's green component can be.
gmax Real The maximum the final colour's green component can be.
bmin Real The minimum the final colour's blue component can be.
bmax Real The maximum the final colour's blue component can be.

Devuelve:

N/A

Ejemplo:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

El código anterior establece que cada partícula emitida del tipo de partícula indexada en la variable global "Blood_Part" sea sólo de diferentes tonos de rojo.
