tipo_parte_color2

Esta función puede utilizarse para establecer un gradiente de dos colores para cada partícula creada del tipo dado. El primer color es con el que empezarán todas las partículas, y el segundo color es con el que terminará la partícula, y se producirá un suave cambio de gradiente en el color a lo largo de la vida de las partículas de un color a otro.

Sintaxis:

part_type_colour2(ind, colour1, colour2);

Argumento Tipo Descripción
ind Tipo de partícula ID The index of the particle type to change.
colour1 Color The colour the particle will start at.
colour2 Color The colour the particle will end at.

Devuelve:

N/A

Ejemplo:

part_type_colour2( part_Health, c_red, c_white);

El código anterior hará que todas las partículas creadas del tipo indexado en la variable "part_Health" tengan una mezcla de color de rojo a blanco durante su vida.
