part_emitter_create

Esta función se utiliza para crear un nuevo emisor de partículas en el sistema de partículas dado. La función devolverá el valor del índice del nuevo emisor, que debe almacenarse en una variable y utilizarse en todas las funciones posteriores que hagan referencia al emisor, y el propio emisor debe destruirse cuando ya no se utilice para evitar fugas de memoria (esto puede conseguirse utilizando las funciones específicas de destrucción del emisor o destruyendo todo el sistema de partículas al que pertenece el emisor).

NOTA Los IDs de los emisores devueltos por esta función se reutilizan, por ejemplo, si primero creas tres emisores de partículas tendrán asignados los IDs 0, 1 y 2. Si luego destruyes el segundo emisor (que tiene un ID de 1) usando part_emitter_destroy y luego creas otro emisor, entonces este emisor reutilizará el ID de 1. Como el ID del emisor 2 está en uso, el siguiente emisor obtendrá un ID de 3, y así sucesivamente.

Sintaxis:

part_emitter_create(ps);

Argumento Tipo Descripción
ps ID del sistema de partículas The particle system to create the emitter in.

Devuelve:

Particle Emitter ID

Ejemplo:

p_emit1 = part_emitter_create(Sname);

Esto creará un nuevo emisor de partículas y almacenará su índice en la variable "p_emit".
