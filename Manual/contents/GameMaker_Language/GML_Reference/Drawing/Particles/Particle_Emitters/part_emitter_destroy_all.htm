part_emitter_destroy_all

Esta función eliminará todos los emisores definidos del sistema dado y los borrará de la memoria (esto también detendrá cualquier partícula producida por el emisor dado, pero NO los elimina de la room). Esta función siempre debe ser llamada cuando los emisores ya no son necesarios para el sistema para evitar fugas de memoria y errores.

Sintaxis:

part_emitter_destroy_all( ps );

ArgumentoTipo Descripción
ps The particle system to destroy all emitters from.

Devuelve:

Ejemplo:

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

El código anterior comprueba la variable global incorporada "lives" y si es 0, destruye todos los emisores de partículas y luego cambia room.
