part_emitter_stream

Esta función permite configurar un emisor para que emita un tipo específico de partículas y se utiliza normalmente el evento create, ya que es un código único que configura el emisor para generar el número de partículas especificado en cada paso del juego a partir de entonces. Las partículas se emiten siguiendo la distribución, la forma y la posición establecidas por la función part_emitter_region(). Puedes establecer el número de partículas que se emiten a cero y llamar a esta función de nuevo para "apagar" el flujo de partículas, y la función también aceptará números negativos para la cantidad, en cuyo caso el emisor producirá partículas basadas en el azar. Por ejemplo, si tienes el número de partículas establecido en -5, hay una probabilidad de 1:5 de que se produzca una partícula en cada paso. Si necesitas que las partículas aparezcan todas a la vez en lugar de cada paso, deberás utilizar la función part_emitter_burst().

Sintaxis:

part_emitter_stream(ps, ind, parttype, number);

Argumento Tipo Descripción
ps ID del sistema de partículas The particle system that the emitter is in.
ind ID del emisor de partículas The index of the emitter to burst from.
parttype Tipo de partícula ID The index (type) of the particles to be created.
number Real The number of particles to create per step.

Devuelve:

N/A

Ejemplo:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

El código anterior transmitirá 1 partícula en cada paso del juego hasta que el emisor sea destruido o el flujo se ponga a 0.
