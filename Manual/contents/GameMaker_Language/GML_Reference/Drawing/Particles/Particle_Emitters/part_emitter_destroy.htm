part_emitter_destroy

Esta función eliminará el emisor especificado del sistema dado y lo borrará de la memoria (esto también detendrá cualquier partícula producida por el emisor dado, pero NO las elimina de la room). Esta función siempre debe ser llamada cuando el emisor dado ya no es necesario para el sistema para evitar fugas de memoria y errores.

Sintaxis:

part_emitter_destroy( ps, ind );

Argumento Tipo Descripción
ps ID del sistema de partículas The particle system to destroy the emitter from.
ind ID del emisor de partículas The index of the emitter to destroy.

Devuelve:

N/A

Ejemplo:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit);
}

El código anterior comprobará si el emisor de partículas indexado en la variable "p_emit" existe en el sistema de partículas dado y si lo hace se destruye.
