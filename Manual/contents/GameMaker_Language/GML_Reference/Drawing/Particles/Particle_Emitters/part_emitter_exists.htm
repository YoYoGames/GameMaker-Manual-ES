part_emitter_exists

Con esta función puedes comprobar si el emisor de partículas indexado existe en el sistema dado o no. Tenga en cuenta que si la variable que se está comprobando es una variable no inicializada (a la que un emisor de partículas tendría asignado su índice) esto arrojará un error.

Sintaxis:

part_emitter_exists(ps, ind);

Argumento Tipo Descripción
ps ID del sistema de partículas The particle system to check for an emitter.
ind ID del emisor de partículas The index of the emitter to search for.

Devuelve:

Boolean

Ejemplo:

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

El código anterior buscará el emisor indexado en la variable "permiso" y si existe, hará estallar 30 partículas de él.
