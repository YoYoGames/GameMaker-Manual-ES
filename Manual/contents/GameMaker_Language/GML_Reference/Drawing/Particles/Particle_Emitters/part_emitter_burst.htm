part_emitter_burst

Esta función permite configurar un emisor para que reviente un tipo específico de partícula y se utiliza típicamente en alarmas y eventos de destrucción ya que es un código único que crea el número de partículas especificado de una sola vez siguiendo la distribución, forma y posición establecida por la función part_emitter_region(). Si necesita que las partículas aparezcan en cada paso, debería utilizar la función part_emitter_stream() en lugar de llamar a esta función en cada paso.

Sintaxis:

part_emitter_burst(ps, ind, parttype, number);

Argumento Tipo Descripción
ps ID del sistema de partículas The particle system that the emitter is in.
ind ID del emisor de partículas The index of the emitter to burst from.
parttype Tipo de partícula ID The index (type) of the particles to be created.
number Real The number of particles to create.

Devuelve:

N/A

Ejemplo:

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

El código anterior hará estallar un número aleatorio de partículas entre 30 y 60 del emisor elegido.
