video_set_volume

Esta función cambia el volumen del audio del vídeo que se está cargando. Toma un valor entre 0 y 1, donde 0 es silencio y 1 es el volumen máximo.

Asegúrese de que esto sólo se llama después de una llamada a video_open() pero antes de una llamada a video_close(), de lo contrario no hará nada (ya que no habrá un vídeo cargado).

En algunas plataformas, esta función sólo funcionará después de que el vídeo se haya cargado completamente, por lo que hay que utilizar Async Callbacks para saber cuándo está listo el vídeo y luego cambiar el volumen.

Sintaxis:

video_set_volume(value);

Argumento Tipo Descripción
value Real Volume value between 0 and 1

Devuelve:

N/A

Ejemplo:

my_video = video_open("splash.mp4");
video_set_volume(0.5);

El código anterior carga splash.mp4 desde los Archivos Incluidos del juego, y cambia su volumen a 0,5 (50% de su volumen máximo).
