video_enable_loop

Esta función habilita o deshabilita la reproducción en bucle del vídeo cargado actualmente. Establezca el argumento enable en true para activar el bucle, y en false para desactivarlo.

Por defecto, el bucle está desactivado.

Sintaxis:

video_enable_loop(enable);

Argumento Tipo Descripción
enable Booleano Whether to enable (true) o desactivar(falso) el bucle

Devuelve:

N/A

Ejemplo:

video_open("Loading.mp4");
video_enable_loop(true);

El código anterior carga un vídeo y activa el bucle en él.
