video_open

Esta función abre un archivo de vídeo utilizando el path especificado. El path puede referirse a un archivo en los Archivos Incluidos de su proyecto, a un archivo presente localmente en el dispositivo del usuario o a una URL para cargar un archivo desde Internet (que puede no funcionar en todas las plataformas).

Como sólo se puede cargar un vídeo a la vez, esta función carga el vídeo especificado en la única "ranura de vídeo" disponible, que luego se manipula a través de las otras funciones de Vídeo.

Existen callbacks asíncronos asociados a la reproducción de vídeo. Consulte Devoluciones de llamada asíncronas para obtener más información.

Sintaxis:

video_open(path);

Argumento Tipo Descripción
path Cadena Path to the video file

Devuelve:

N/A

Ejemplo:

my_video = video_open("splash.mp4");

El código anterior carga splash.mp4 desde los Archivos Incluidos del juego.
