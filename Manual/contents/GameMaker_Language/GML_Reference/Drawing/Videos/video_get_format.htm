video_get_format

Esta función devuelve el formato de color del vídeo cargado actualmente. Puede ser cualquiera de las siguientes constantes:

Formato de vídeo constante
Constante Descripción
video_format_rgba La superficie de vídeo utiliza el modelo de color RGBA
video_format_yuv La superficie de vídeo utiliza el modelo de color YUV

Windows, macOS, Opera GX, Android, iOS, y HTML5 utilizan el modelo RGBA para los vídeos. Todas las demás plataformas utilizan el modelo YUV.

Como los métodos de dibujo para los vídeos RGBA y YUV son diferentes, el valor de retorno de esta función puede utilizarse para ejecutar un código diferente basado en el formato del vídeo que se reproduce. Consulte Dibujar vídeo para ver un ejemplo.

Sintaxis:

video_get_format();

Devuelve:

Video Format Constant
