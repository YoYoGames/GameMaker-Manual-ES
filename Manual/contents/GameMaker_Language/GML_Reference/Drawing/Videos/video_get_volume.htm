video_get_volume

Esta función devuelve el volumen del vídeo cargado actualmente, que es un valor entre 0 y 1. Puede cambiar el volumen del vídeo cargado utilizando video_set_volume().

Sintaxis:

video_get_volume();

Devuelve:

Real

Ejemplo:

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

El código anterior se ejecutaría en un evento Step. Obtiene el volumen del vídeo que se está reproduciendo en ese momento, y si es inferior a 1, lo aumentará en 0,02, incrementando gradualmente el volumen hasta llegar al nivel máximo.
