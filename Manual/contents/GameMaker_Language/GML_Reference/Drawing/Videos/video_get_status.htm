video_get_status

Esta función devuelve el estado del vídeo cargado actualmente. El estado puede ser cualquiera de las siguientes constantes:

Constante de estado de vídeo
Constante Descripción
video_status_closed No hay ningún vídeo cargado actualmente, o el vídeo se cerró con video_close()
video_status_preparing El vídeo se está preparando y aún no ha empezado a reproducirse
video_status_playing El vídeo se está reproduciendo actualmente
video_status_paused El vídeo está en pausa (ver video_pause())

Sintaxis:

video_get_status();

Devuelve:

Video Status Constant

Ejemplo:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

El código anterior obtiene el estado del vídeo y luego comprueba si el jugador ha pulsado Espacio. En ese caso, pausa el vídeo si se está reproduciendo, y lo reanuda si está en pausa.
