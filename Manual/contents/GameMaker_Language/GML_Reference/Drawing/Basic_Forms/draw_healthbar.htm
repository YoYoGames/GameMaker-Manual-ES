dibujar_barra_de_salud

Con esta función puede dibujar una barra de color para mostrar un valor constante. Aunque la función utiliza la palabra "barra de salud" puedes utilizarla para mostrar cualquier cosa que desees siempre que la cantidad a mostrar sea un valor porcentual entre 0 y 100 (un porcentaje de cualquier valor puede ser calculado con la fórmula (Valor_CURRENTE / Valor_MÁXIMO) * 100), así que, por ejemplo, puedes utilizarla para mostrar la potencia, la salud, el maná, el tiempo o cualquier otra cosa que se beneficie de la visualización de una barra.

Puede establecer varias cosas diferentes que cambian el aspecto visual de la barra y que se ilustran en la imagen siguiente (tenga en cuenta que los colores utilizados aquí son c_green para el valor máximo y c_red para el valor mínimo, pero puede utilizar los colores que desee):

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Argumento Tipo Descripción
x1 Real The x coordinate of the left of the healthbar
y1 Real The y coordinate of the top of the healthbar
x2 Real The x coordinate of the right of the healthbar
y2 Real The y coordinate of the bottom of the healthbar
amount Real The variable which defines total health (between 0 and 100)
backcol Color The colour of the background for the bar
mincol Color The colour of the bar when at no health (0)
maxcol Color The colour of the bar when at full health (100)
direction Real Where the bar is "anchored" (0 = left, 1 = right, 2 = top, 3 = bottom)
showback Booleano Whether the bar's background is drawn (true) o no (false). Si es falso, el argumento backcol no se tiene en cuenta.
showborder Booleano Whether the bar's elements have a 1px wide black border (true) o no (false).

Devuelve:

N/A

Ejemplo:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

El código anterior utiliza el valor porcentual que se obtiene de las variables "hp" y "hp_max" para dibujar una barra de salud roja/verde estándar.
