draw_set_circle_precision

Para optimizar el rendimiento al dibujar círculos, GameMaker básicamente dibuja una forma poligonal con suficientes lados para que parezca circular. Sin embargo, dependiendo de lo grande o pequeño que necesite que sean sus círculos, puede encontrar que cambiar este valor puede ayudar a aumentar el rendimiento de su juego, o hacer que los círculos se vean mejor. El valor de precisión que puedes introducir en esta función es el número de lados que tiene el polígono del círculo, y este número puede estar entre 4 y 64, pero debe ser un número divisible por 4, con un valor por defecto de 24.

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex Textura, Color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_set_circle_precision(precision);

ArgumentoTipo Descripción
precision The precision of all circles drawn after this function is called.

Devuelve:

Ejemplo:

draw_set_circle_precision(12);

El código anterior dibujará todos los círculos utilizando un polígono de 12 lados.
