draw_point

Con esta función puedes dibujar un solo píxel en cualquier lugar de la pantalla, utilizando el color de dibujo y el alfa actualmente establecidos.

Sintaxis:

draw_point(x, y);

ArgumentoTipo Descripción
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

Devuelve:

Ejemplo:

draw_set_colour(c_yellow);
draw_point(100,100);

Esto dibujará un píxel amarillo en la posición (100,100).
