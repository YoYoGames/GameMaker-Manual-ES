dibujar_ancho_de_línea_color

Con esta función puede dibujar una línea de una anchura determinada con el color mezclado entre el color 1 en el primer punto y el color 2 en el segundo punto. La configuración del color anulará el color base establecido con la función draw_set_colour(). Tenga en cuenta que la línea que se dibuja puede necesitar diferentes valores (+/-1 en las x, y) para ser dibujada con las dimensiones deseadas debido a las diferencias entre las distintas plataformas soportadas.

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_line_width_colour(x1, y1, x2, y2, w, col1, col2);

ArgumentoTipo Descripción
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
w The width in pixels of the line.
col1 The colour of the start of the line.
col2 The colour of the end of the line.

Devuelve:

Ejemplo:

draw_line_width_colour(50, 50, 300, 50, 4, c_red, c_blue);

Esto dibujaría una línea horizontal desde (50,50) hasta (300,50), de cuatro píxeles de ancho, con un suave gradiente de rojo a azul.
