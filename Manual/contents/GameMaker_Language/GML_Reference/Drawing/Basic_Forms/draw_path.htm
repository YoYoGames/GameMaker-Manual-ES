dibujar_trayectoria

Con esta función puedes hacer que GameMaker dibuje un path en la pantalla. El path se dibujará como una simple línea, y puede ser relativo a la instancia que lo llama o en la posición absoluta en la que fue creado en el editor de path o a través del código. Esta función es extremadamente útil cuando debugging dinámico paths (por ejemplo, los creados para instancias con la función mp_grid_path() ).

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_path(path, x, y, absolute);

ArgumentoTipo Descripción
path The path to draw
x The x coordinate of where the path is drawn
y The y coordinate of where the path is drawn
absolute Whether the path is drawn at the absolute position (true) o la posición relativa (false)

Devuelve:

Ejemplo:

if mp_grid_path(grid, path, x, y, obj_Player.x, obj_Player.y, 1)
{
    draw_path(path, x, y, false);
}

El código anterior utilizará la función mp_grid_path para generar un path y almacenarlo en la variable "path". Si el path se crea con éxito, se dibuja en la pantalla en una posición relativa a la instancia que ejecuta el código.
