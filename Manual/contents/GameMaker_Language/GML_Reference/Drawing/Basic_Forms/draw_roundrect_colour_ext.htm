dibujar_redondo_color_ext

Con esta función puedes dibujar un contorno de un rectángulo redondeado o un rectángulo redondeado relleno donde la posición (x1,y1) es la esquina superior izquierda y la posición (x2,y2) es la esquina inferior derecha. Si el rectángulo está relleno, los argumentos de color se utilizarán para generar un gradiente de color desde el centro hasta los bordes, donde el color 1 es el color del centro y el color 2 el color de los bordes. También debe suministrar valores de radio para los ejes x e y (en píxeles) y las esquinas se redondearán en estas cantidades. Puede definir la precisión del dibujo de las esquinas con la función draw_set_circle_precision(). Tenga en cuenta que el rectángulo que se dibuja puede necesitar valores diferentes (+/-1 en la x, la y, o la anchura o la altura) para que se dibuje con las dimensiones deseadas debido a las diferencias entre las distintas plataformas soportadas.

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline);

Argumento Tipo Descripción
x1 Real The x coordinate of the left of the rounded rectangle.
y1 Real The y coordinate of the top of the rounded rectangle.
x2 Real The x coordinate of the right of the rounded rectangle.
y2 Real The y coordinate of the bottom of the rounded rectangle.
xrad Real The radius of the curve along the x axis from the rectangle corners.
yrad Real The radius of the curve along the y axis from the rectangle corners.
col1 Color The center colour.
col2 Color The outside edge colour.
outline Booleano Whether the rectangle is an outline (true) or filled in (false).

Devuelve:

N/A

Ejemplo:

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0);

Esto dibujaría un cuadrado relleno con esquinas redondeadas, cuyo radio está definido por la distancia de la instancia que hace el dibujo al ratón. Este valor también se utiliza para calcular el color de fusión del centro.
