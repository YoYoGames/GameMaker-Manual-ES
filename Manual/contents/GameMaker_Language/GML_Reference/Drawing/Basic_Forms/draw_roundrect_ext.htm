draw_roundrect_ext

Con esta función puede dibujar un contorno de un rectángulo redondeado o un rectángulo redondeado relleno donde la posición (x1,y1) es la esquina superior izquierda y la posición (x2,y2) es la esquina inferior derecha. También debe suministrar los valores del radio para los ejes x e y (en píxeles) y las esquinas se redondearán en estas cantidades. Puedes definir la precisión del dibujo de las esquinas con la función draw_set_circle_precision(). Tenga en cuenta que el rectángulo que se dibuja puede necesitar diferentes valores (+/-1 en la x, la y, o la anchura o la altura) para que se dibuje con las dimensiones deseadas debido a las diferencias entre las distintas plataformas soportadas.

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_roundrect_ext(x1, y1, x2, y2, xrad, yrad, outline);

Argumento Tipo Descripción
x1 Real The x coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle).
y1 Real The y coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle).
x2 Real The x coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle).
y2 Real The y coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle).
xrad Real The radius of the curve along the x axis from the rectangle corners.
yrad Real The radius of the curve along the y axis from the rectangle corners.
outline Booleano Whether the rounded rectangle is drawn filled (false) or as a one pixel wide outline (true).

Devuelve:

N/A

Ejemplo:

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
draw_set_colour(c_lime);
draw_roundrect_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, 0);

Esto dibujará un rectángulo redondeado relleno con las esquinas redondeadas por una cantidad basada en la posición del ratón en relación con la instancia que hace el dibujo.
