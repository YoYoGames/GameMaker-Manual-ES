draw_line

Con esta función puedes dibujar una línea de 1 píxel de ancho entre dos puntos cualquiera del juego room. Tenga en cuenta que la línea que se dibuja puede necesitar diferentes valores (+/-1 en la x, y) para ser dibujada con las dimensiones deseadas debido a las diferencias entre las distintas plataformas soportadas.

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_line(x1, y1, x2, y2);

ArgumentoTipo Descripción
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.

Devuelve:

Ejemplo:

draw_set_colour(c_lime);
draw_line(50,50,150,50);

Esto dibujará una línea horizontal de color verde claro desde el punto (50,50) hasta el punto (150,50).
