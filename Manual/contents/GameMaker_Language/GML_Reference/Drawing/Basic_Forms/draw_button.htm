botón_de_dibujo

Esta función dibujará un "botón" rectangular muy simple utilizando el color de dibujo y el alfa actualmente seleccionados, donde el argumento arriba define cómo se ve el efecto de borde biselado, como se muestra en la imagen de abajo:

NOTA: Si desea dibujar una forma utilizando un shader, debe tener en cuenta que la mayoría de los shaders esperan las siguientes entradas: vertex textura, color. Sin embargo, cuando se utiliza esta función, sólo se pasan datos de vertex y de color, por lo que el shader puede no dibujar nada (o dibujar algo pero no correctamente). Si necesitas dibujar formas de esta manera, entonces el shader debe ser personalizado teniendo esto en cuenta.

Sintaxis:

draw_button(x1, y1, x2, y2, up);

Argumento Tipo Descripción
x1 Real The x coordinate of the left of the button
y1 Real The y coordinate of the top of the button
x2 Real The x coordinate of the right of the button
y2 Real The y coordinate of the bottom of the button
up Booleano Whether the button is up (true) o hacia abajo (false)

Devuelve:

N/A

Ejemplo:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Esto dibujará un botón que aparecerá pulsado si se mantiene pulsado el botón izquierdo del ratón.
