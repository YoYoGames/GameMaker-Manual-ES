Texto

Hay una serie de funciones disponibles para dibujar texto de diversas formas. Hay que tener en cuenta que algunas de estas funciones cambian los colores y tamaños del texto de forma dinámica cuando se dibuja, pero cuando se trabaja con HTML5 o con dispositivos, estas funciones pueden ralentizar tu juego o tener un aspecto pobre, por lo que puede ser mejor tener recursos font de varios tamaños en lugar de utilizar el dibujo a escala, o tener recursos font grandes y luego dibujarlos a escala en lugar de hacerlo a escala. Además, si necesitas colorear grandes bloques de texto es mejor utilizar las funciones draw_set_colour() y draw_set_alpha() junto con la función simple draw_text, en lugar de las funciones extendidas que tienen un componente de color.

Las siguientes funciones sirven para establecer el font que se utilizará al dibujar, así como el posicionamiento del texto cuando se dibuje:

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

Estas funciones pueden utilizarse para dibujar texto en la pantalla, ya sea de forma simple o con transformaciones y color:

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

También existe la siguiente función que es específica para dibujar las puntuaciones máximas creadas con el sistema de puntuaciones máximas incorporado:

  • draw_highscore
