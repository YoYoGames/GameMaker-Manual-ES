dibujar_texto_texto

Esta función dibujará el texto de forma similar a draw_text() sólo que ahora puede establecer el espacio entre cada línea de texto -en caso de que el texto ocupe más de una línea- y limitar el ancho (en píxeles) del string por línea, de modo que si alguna línea excede este valor, GameMaker dividirá automáticamente el texto a la siguiente línea en el espacio en blanco disponible más cercano (si el texto no tiene espacios en blanco, entonces sobrepasará este valor máximo de de ancho). Tenga en cuenta que cualquier espacio en blanco colocado al principio de string será eliminado antes de ser analizado para el dibujo debido a esto. También tenga en cuenta que un valor de -1 para el argumento de la separación de la línea será por defecto una separación basada en la altura del carácter "M" en el font elegido.

Sintaxis:

draw_text_ext(x, y, string, sep, w);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

Devuelve:

Ejemplo:

draw_text_ext(100, 50, keyboard_string, 3, 300);

El código anterior dibujará cualquier texto que el usuario escriba en el teclado, dividiéndolo en nuevas líneas cada vez que la longitud de string para esa línea supere los 300 píxeles. el código también mantendrá una separación de 3 píxeles entre líneas si esto ocurre.
