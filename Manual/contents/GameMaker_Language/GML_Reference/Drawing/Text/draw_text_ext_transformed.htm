dibujar_texto_transformado

Esta función es una combinación de la función base draw_text() con la función draw_text_ext() y draw_text_transformed() que permite escalar y girar el texto manteniendo un espacio entre líneas y una anchura máxima por línea. Tenga en cuenta que el argumento "anchura" se basa en una escala de 1, por lo que si la escala es diferente, este valor debe cambiarse proporcionalmente. Por ejemplo, si el ancho base para un salto de línea es 300 y usted establece la escala a 2, entonces el texto aparecerá mal, sobrepasando el ancho dado. En su lugar, debería haber establecido el ancho a 150 para compensar la escala.

Sintaxis:

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.

Devuelve:

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

El código anterior dibujará el texto dado en el centro de room, con una longitud máxima de string de 300 píxeles, un espacio entre cada línea de 10 píxeles, girando en redondo y escalado al doble de su tamaño original.
