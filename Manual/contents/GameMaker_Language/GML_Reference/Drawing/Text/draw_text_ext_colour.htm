dibujar_texto_ext_color

Esta función es una combinación de la función base draw_text() con la función draw_text_ext() y draw_text_colour() que le permite definir los colores del gradiente para el texto, así como el espacio entre líneas y la anchura máxima por línea, todo junto.

NOTA La mezcla de gradientes no está disponible para el objetivo HTML5 a menos que WebGL esté habilitado, aunque todavía puedes establecer los colores de mezcla y se mezclará el font con el primer color dado. Sin embargo, todas las mezclas de este modo crean un duplicado de font que se almacena en cache y se utiliza cuando se requiere, lo que está lejos de ser óptimo y si se utilizan múltiples cambios de color, se ralentizará el rendimiento de los juegos. Puedes establecer el tamaño de font cache para intentar limitar esto si es necesario utilizando la función font_set_cache_size().

Sintaxis:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Argumento Tipo Descripción
x Real The x coordinate of the drawn string.
y Real The y coordinate of the drawn string.
string Cadena The string to draw.
sep Real The distance in pixels between lines of text.
w Real The maximum width in pixels of the string before a line break.
c1 Color The colour for the top left of the drawn text.
c2 Color The colour for the top right of the drawn text.
c3 Color The colour for the bottom right of the drawn text.
c4 Color The colour for the bottom left of the drawn text.
alpha Real The alpha for the text.

Devuelve:

N/A

Ejemplo:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

El código anterior dibujará cualquier texto que el usuario escriba en el teclado, dividiéndolo en nuevas líneas cada vez que la longitud de string para esa línea supere los 300 píxeles. el código también mantendrá una separación de 5 píxeles entre líneas si esto ocurre. Cada línea del texto se coloreará utilizando un gradiente azul, con azul claro en la parte superior y azul oscuro en la inferior.
