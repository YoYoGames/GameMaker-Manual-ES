dibujar_texto_transformado_color

Esta función es una combinación de la función base draw_text() con la función draw_text_ext(), draw_text_transformed() y draw_text_colour() . Como tal, es la más general de todas las funciones para dibujar texto y ofrece una gran flexibilidad a la hora de dibujar. Tenga en cuenta que el argumento "ancho" se basa en una escala de 1, por lo que si la escala es diferente, este valor debe cambiarse proporcionalmente. Por ejemplo, si el ancho base para un salto de línea es 300 y usted establece la escala en 2, entonces el texto aparecerá mal, sobrepasando el ancho dado. En su lugar, debería establecer el ancho a 150 para compensar la escala.

NOTA: La mezcla de gradientes no está disponible para el objetivo de HTML5 a menos que WebGL esté habilitado, aunque todavía puede establecer los colores de mezcla y se mezclará el font con el primer color dado. Sin embargo, toda mezcla de esta manera crea un duplicado font que se almacena en cache y se utiliza cuando es necesario, lo que está lejos de ser óptimo y, si utilizas múltiples cambios de color, ralentizará el rendimiento de tus juegos. Puedes establecer el tamaño de font cache para intentar limitar esto si es necesario utilizando la función font_set_cache_size().

Sintaxis:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Devuelve:

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Esto dibujará el texto dado con un gradiente de azul a azul oscuro hacia abajo, en la posición (200,200) en el room, girado para estar al revés y a la mitad del tamaño de punto normal del font. Habrá 20 píxeles entre cada línea si hay más de una línea dibujada y cada línea tendrá un ancho máximo de 200 píxeles. una línea dibujada y cada línea tendrá una anchura máxima de 200 píxeles.
