draw_text

Con esta función puede dibujar cualquier string en cualquier posición dentro de room (para dibujar números reales debe utilizar la función string() para convertirlos en texto). Para combinar strings se puede utilizar + (ver ejemplo más abajo) y también se puede utilizar \n dentro de un string para añadir un salto de línea para que se dibuje en varias líneas (para obtener información sobre cómo formatear correctamente un string y qué caracteres de escape se pueden utilizar, por favor vea aquí). El color del texto y el alfa se rigen por los valores actuales de alfa y color base establecidos por draw_set_alpha() y draw_set_colour().

NOTA: La posición real del texto estará influenciada por los valores de alineación establecidos por draw_set_halign() y draw_set_valign().

Sintaxis:

draw_text(x, y, string);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

Devuelve:

Ejemplo:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

El código anterior dibujará un string en la posición x/y de la instancia, que utilizará el string almacenado en la variable global "Nombre" y lo dividirá en dos líneas.
