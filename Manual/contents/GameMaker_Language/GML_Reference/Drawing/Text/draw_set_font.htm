draw_set_font

Esta función establecerá el font que se utilizará para todo el dibujo de texto posterior. Este font debe haber sido añadido en los activos de fuente del juego o haber sido creado usando el font_add() o font_add_sprite().

Sintaxis:

draw_set_font(font);

ArgumentoTipo Descripción
font The name of the font to use.

Devuelve:

Ejemplo:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

El código anterior dibujará el texto dado utilizando el font indexado en la variable "fnt_Game" y coloreado en azul.
