draw_get_font

Esta función obtendrá el font actualmente asignado para dibujar el texto. La función devolverá -1 si no se ha establecido ningún font, o el valor de ID (un entero positivo) para el font asset asignado.

Sintaxis:

draw_get_font();

Devuelve:

Font Asset or -1

Ejemplo:

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

El código anterior obtiene el font aplicado actualmente y ejecuta una sentencia switch sobre él, aplicando un valor de desplazamiento Y diferente dependiendo del font. Luego utiliza ese valor de desplazamiento mientras dibuja algún texto.
