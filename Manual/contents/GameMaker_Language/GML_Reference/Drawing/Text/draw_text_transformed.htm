draw_text_transformed

Esta función dibujará el texto de forma similar a draw_text() sólo que ahora puede elegir escalar el texto a lo largo del eje horizontal o vertical (efectivamente estirándolo o encogiéndolo) y también hacer que GameMaker lo dibuje en un ángulo (donde 0 es normal y cada grado por encima de 0 gira el texto en sentido antihorario).

Sintaxis:

draw_text_transformed(x, y, string, xscale, yscale, angle);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale (default 1).
yscale The vertical scale(default 1).
angle The angle of the text.

Devuelve:

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

El código anterior dibujará el texto dado en el centro de la página room, girando alrededor y escalado al doble de su tamaño original.
