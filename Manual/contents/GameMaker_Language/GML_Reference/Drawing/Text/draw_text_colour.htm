dibujar_texto_color

Esta función dibujará el texto de forma similar a draw_text() sólo que ahora puede elegir los colores a utilizar para colorear el texto así como el valor alfa, y estos nuevos valores se utilizarán en lugar de del color base de dibujo y del alfa.

NOTA: La mezcla de gradientes no está disponible para el objetivo de HTML5 a menos que WebGL esté habilitado, aunque todavía puede establecer los colores de mezcla y se mezclará el font con el primer color dado. Sin embargo, toda mezcla de esta manera crea un duplicado font que se almacena en cache y se utiliza cuando se requiere, lo que está lejos de ser óptimo y si utilizas múltiples cambios de color, ralentizará el rendimiento de tus juegos. Puedes establecer el tamaño de font cache para intentar limitar esto si es necesario utilizando la función font_set_cache_size().

Sintaxis:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Devuelve:

Ejemplo:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

El código anterior dibujará dos secciones de texto en la misma línea, con el primer texto dibujado en blanco (ya que ese es el color base de dibujo) y el segundo texto dibujado con un gradiente de verde lima a verde normal.
