dibujar_texto_transformado_color

Esta función es una combinación de la función base draw_text() con la función draw_text_transformed() y draw_text_colour() que permite escalar y rotar el texto, así como colorearlo con un relleno de degradado y cambiar su valor alfa, ignorando los ajustes de alfa y color base para el dibujo.

NOTA La mezcla de gradientes no está disponible para el objetivo HTML5 a menos que WebGL esté habilitado, aunque todavía puede establecer los colores de mezcla y se mezclará el font con el primer color dado. Sin embargo, toda mezcla de esta manera crea un duplicado font que se almacena en cache y se utiliza cuando es necesario, lo que está lejos de ser óptimo y si se utilizan múltiples cambios de color, se ralentizará el rendimiento de los juegos. Puedes establecer el tamaño de font cache para intentar limitar esto si es necesario utilizando la función font_set_cache_size().

Sintaxis:

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentoTipo Descripción
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Devuelve:

Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

El código anterior dibujará el texto dado en el centro de la página room, girando alrededor y escalado al doble de su tamaño original, con un gradiente de color que va del amarillo al rojo, así como un alfa de 0,5.
