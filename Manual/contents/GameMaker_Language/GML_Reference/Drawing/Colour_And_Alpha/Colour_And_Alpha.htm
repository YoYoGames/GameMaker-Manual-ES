Color y Alfa

GameMaker tiene una amplia selección de funciones para manipular el color y el alfa de lo que se dibuja en la pantalla. Usando estas funciones puedes separar los colores en sus partes componentes, crear nuevos colores y manipular el alfa de dibujo para cambiar cómo se dibujan las diferentes cosas en tu juego.

Para mezclar imágenes utilizando modos de mezcla especiales y otras funciones específicas de la GPU, consulte la sección sobre el Control de la GPU.

Colores hexagonales

Se pueden crear colores a partir de valores hexadecimales utilizando previamente el símbolo $, y estos valores también se pueden utilizar con varias funciones de color para obtener los valores de los componentes por separado. El valor hexadecimal de un color se divide en tres partes, con el formato: Azul/Verde/Rojo(o $BBGGRR), así que, por ejemplo, para definir un color azul claro/acuático se podría utilizar:

col = $FFCC11;

Aquí, FF es el componente azul, CC es el componente verde y 11 es el componente rojo.

También puede utilizar "colores CSS" que comienzan con un signo de almohadilla(#). Esto utiliza un formato Rojo/Verde/Azul(o #RRGGBB) que se utiliza más ampliamente y es compatible con varias otras aplicaciones.

col = #11CCFF;

Este es el mismo color que se muestra en el bloque de código anterior, sólo los valores RR y BB se intercambian para que coincida con el formato.

NOTA GML también soporta el formato $AABBGGRR (con el valor hexadecimal alfa al principio) sin embargo la mayoría de las funciones de color de GML sólo podrán hacer uso de la parte de color e ignorar el alfa.

Funciones

Las siguientes funciones se pueden utilizar para obtener las tonalidades de los componentes, el tono compuesto, la saturación y la luminosidad de un color seleccionado, así como el alfa y otras propiedades:

  • colour_get_blue
  • colour_get_green
  • colour_get_red
  • colour_get_hue
  • colour_get_saturation
  • colour_get_value
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Estas funciones pueden utilizarse para crear colores a partir de los valores de entrada brutos de los componentes de color:

  • make_colour_hsv
  • make_colour_rgb
  • merge_colour

Las siguientes funciones se pueden utilizar para establecer las diferentes opciones para dibujar en la pantalla, incluyendo el alfa, el color y la mezcla:

  • draw_clear
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
