colour_get_red

Esta función devuelve la cantidad de rojo utilizada para hacer el color dado, con el valor entre 0 y 255, donde 0 es ningún rojo y 255 es todo rojo. La siguiente imagen lo ilustra:

Sintaxis:

colour_get_red(col);

ArgumentoTipo Descripción
col The colour to check

Devuelve:

Ejemplo:

r_comp = colour_get_red(c_teal);

El código anterior obtendrá el componente rojo de la constante de color "c_teal" y almacenará el valor en la variable "r_comp".
