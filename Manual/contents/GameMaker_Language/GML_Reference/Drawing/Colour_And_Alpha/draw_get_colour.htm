draw_get_colour

Esta función devuelve el color de dibujo actual que se utiliza para dibujar formas, texto, primitivas y modelos 3D sin textura. Se puede establecer con la función draw_set_colour() función.

Sintaxis:

draw_get_colour()

Devuelve:

Colour Constant

Ejemplo:

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

El código anterior almacena el color de dibujo actual en una variable local, y cambia el color de dibujo basándose en una variable de instancia. Después de dibujar algún texto, restablece el color al valor almacenado en la variable local.
