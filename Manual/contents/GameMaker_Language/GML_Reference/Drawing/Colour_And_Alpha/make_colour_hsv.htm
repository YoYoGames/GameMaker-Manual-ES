make_colour_hsv

GameMaker proporciona esta función (así como otras) para permitir al usuario crear sus propios colores. Esta función en particular toma tres componentes, el tono, la saturación y el valor (también conocido como "luminosidad") para crear el color deseado. Estos valores se toman entre 0 y 255, por lo que se pueden crear 16.777.216 (256*256*256) colores con esto. Abajo puedes ver una imagen de cómo se ven estos componentes cuando se separan:

La imagen de la izquierda es un desglose de los componentes individuales de la función y, a continuación, a la derecha, una ilustración de cómo el cambio de estos componentes afecta al color final. Como puedes ver, incluso sin cambiar el tono (se queda en 0, rojo), podemos cambiar drásticamente el color con los otros dos ajustes, con la saturación haciendo que el tono de color pase de gris (0) a un tono puro (255), y con el valor haciendo que pase de más oscuro (0) a más claro (255).

Sintaxis:

make_colour_hsv(hue, sat, val);

ArgumentoTipo Descripción
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

Devuelve:

Colour Constant

Ejemplo:

col = make_colour_hsv(100, 145, 255);

El código anterior utiliza la función para crear un color y almacenar su valor en la variable "col" para su uso posterior.
