draw_set_alpha

Con esta función puedes establecer el alfa base de dibujo para el juego. Este valor puede establecerse de 0 a 1, siendo 0 totalmente transparente y 1 totalmente opaco (el valor por defecto), y afectará a todo el dibujo posterior, incluyendo fondos, sprites, fonts, primitivas y 3D.

Tenga en cuenta que si está dibujando cualquiera de los assets y suministrando un argumento alfa como parte de la función (por ejemplo, cuando se utiliza draw_sprite_ext()), entonces el valor alfa establecido aquí será por lo que al dibujar un sprite usando draw_sprite_ext() con un argumento alfa de 1 se dibujará el sprite completamente opaco sin importar el alfa establecido con draw_set_alpha() (esto funcionará igual al usar draw_self() y estableciendo el image_alpha de la instancia). También hay que tener en cuenta que cuando una instancia está dibujando por defecto (es decir, tiene un sprite pero no un evento de dibujo), entonces la variable image_alpha se ajustará al alfa establecido con draw_set_alpha(), de manera que un image_alpha de 1 y un alfa de dibujo de 0.5 dibujará el sprite a alfa 0.5, mientras que un image_alpha de 0.25 y un alfa de dibujo de 0,5 dibujará el sprite a 0,25 alfa.

Sintaxis:

draw_set_alpha(alpha);

ArgumentoTipo Descripción
alpha The alpha to set (between 0 and 1)

Devuelve:

Ejemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

El código anterior dibujará un texto en la posición especificada con un efecto de sombra creado por la modificación del alfa y el color del dibujo.
