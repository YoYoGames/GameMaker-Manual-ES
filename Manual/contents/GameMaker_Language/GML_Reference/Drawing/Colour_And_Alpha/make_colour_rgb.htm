make_colour_rgb

GameMaker proporciona esta función (así como otras) para permitir al usuario hacer sus propios colores. Esta función en particular toma tres componentes, el rojo, el verde y el azul del color que desea hacer. Estos valores se toman entre 0 y 255, por lo que puedes hacer 16.777.216 (256*256*256) colores con esto. A continuación puedes ver una imagen de cómo se ven estos componentes cuando se separan:

La imagen de la izquierda es un desglose de los componentes individuales de la función, y a continuación, a la derecha, una ilustración de cómo el cambio de estos componentes afecta al color final.

Sintaxis:

make_colour_rgb(red, green, blue);

ArgumentoTipo Descripción
red The red component of the colour
green The green component of the colour
blue The blue component of the colour

Devuelve:

Colour Constant

Ejemplo:

col = make_colour_rgb(100, 145, 255);

El código anterior utiliza la función para crear un color y almacenar su valor en la variable "col" para su uso posterior.
