draw_get_alpha

Esta función devuelve el valor actual del alfa de dibujo, que oscilará entre 0 (totalmente transparente) y 1 (totalmente opaco). El alfa de dibujo afecta a la transparencia de todas las funciones de dibujo, y se puede establecer con la función draw_set_alpha().

Sintaxis:

draw_get_alpha()

Devuelve:

Ejemplo:

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

El código anterior almacena el alfa de dibujo actual en una variable local, y cambia el alfa de dibujo basado en una variable de instancia. Después de dibujar algún texto, restablece el alfa al valor almacenado en la variable local.
