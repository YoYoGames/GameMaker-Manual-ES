fusionar_color

Con esta función se pueden tomar dos colores y fusionarlos para obtener un nuevo color. La cantidad de cada uno de los colores componentes puede definirse cambiando el argumento "cantidad", donde un valor de 0 devolverá el primer color (col1), un valor de 1 devolverá el segundo color (col2) y un valor intermedio devolverá la mezcla correspondiente. Por ejemplo, un valor de 0,5 mezclará los dos colores por igual. La siguiente imagen ilustra el funcionamiento de la mezcla de los colores rojo y azul:

Sintaxis:

merge_colour(col1, col2, amount);

ArgumentoTipo Descripción
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

Devuelve:

Colour Constant

Ejemplo:

col = merge_colour(c_lime, c_orange, 0.5);

El código anterior utiliza la función para crear un color mediante la fusión de lima y naranja 50/50 y luego almacena su valor en la variable "col" para su uso posterior.
