color_get_hue

Esta función devolverá el tono del color dado. Se trata del tono de color "puro" que forma parte del método de tono, saturación y valor (luminosidad) para definir un color. La siguiente imagen ilustra cómo este valor se corresponde con la escala HSV del color:

Sintaxis:

colour_get_hue(col);

ArgumentoTipo Descripción
col The colour to check

Devuelve:

Ejemplo:

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

El código anterior obtiene el tono utilizado para hacer la constante de color "c_teal" y luego lo fusiona con el blanco en una proporción de 50:50, almacenando el color resultante en la variable "col".
