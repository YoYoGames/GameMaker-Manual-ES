draw_clear_alpha

Esta función se puede utilizar para limpiar toda la pantalla con un color determinado y el componente alfa del destino se establece en el valor que se ha establecido - esta función no hace ninguna mezcla mientras funciona pero cualquier operación de mezcla posterior puede Esta función no hace ninguna mezcla, pero cualquier operación de mezcla posterior puede ser configurada para utilizar el alfa de destino que se ha establecido. Esto es sólo para su uso en el evento de dibujo de una instancia (no se mostrará si se utiliza en cualquier otro evento), y también puede ser muy útil para limpiar las superficies cuando son recién creadas.

Vale la pena señalar que cuando se utiliza el módulo HTML5, se puede utilizar esta función para hacer que el fondo del lienzo sea transparente, lo que permitirá que cualquier imagen que se utilice en el HTML del anfitrión se vea con el juego corriendo sobre ella. Para conseguir este efecto primero debes ir al editor room y eliminar cualquier capa de fondo del editor de la sala, luego desmarcar la opción Borrar fondo de la vista que se encuentra en las propiedades de la vista. Finalmente, deberá hacer un nuevo object, colocarlo en el primer room del juego, y asignarle una profundidad muy alta (por ejemplo, 10000). Luego, en el evento Pre Draw utilice esta función para establecer el alfa del lienzo al color y la transparencia que que desees que tenga.

Sintaxis:

draw_clear_alpha(col, alpha);

ArgumentoTipo Descripción
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

Devuelve:

Ejemplo:

draw_clear_alpha(c_red, 0.5);

Esto limpiará la pantalla con un 50% (0,5) de transparencia con el color rojo.
