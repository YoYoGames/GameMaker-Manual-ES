colour_get_saturation

Esta función devolverá la saturación del color dado. Esta es la cantidad del tono de color que se mezcla en el color final y es parte del método de tono, saturación y valor (luminosidad) para definir un color. La siguiente imagen ilustra cómo este valor corresponde a la escala HSV del color:

Sintaxis:

colour_get_saturation(col);

ArgumentoTipo Descripción
col The colour to check

Devuelve:

Ejemplo:

col = make_colour_hsv(random(255), colour_get_sat(c_teal), 255);

El código anterior obtiene la saturación utilizada para hacer la constante de color "c_teal" y luego utiliza ese valor para establecer un color aleatorio que tenga la misma saturación, almacenando el color resultante en la variable "col".
