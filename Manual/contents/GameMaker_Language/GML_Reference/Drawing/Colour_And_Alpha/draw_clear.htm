draw_clear

Esta función se puede utilizar para limpiar toda la pantalla (sin mezcla alfa) con el color dado, y sólo se puede utilizar en el evento de dibujo de una instancia (no se mostrará si se utiliza en cualquier otro evento). También puede ser útil para limpiar las superficies cuando son recién creadas.

Sintaxis:

draw_clear(col);

ArgumentoTipo Descripción
col The colour with which the screen will be cleared

Devuelve:

Ejemplo:

draw_clear(c_blue);

Esto limpiará la pantalla con el color azul.
