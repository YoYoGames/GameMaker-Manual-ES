draw_set_colour

Con esta función puedes establecer el color de dibujo base para el juego.

Esto afectará al dibujo de fonts, formas, primitivas y 3D, sin embargo no afectará a sprites (dibujado manualmente o por una instancia). Si cualquier gráfico afectado se dibuja con sus propios valores de color, este valor será ignorado.

Sintaxis:

draw_set_colour(col);

Argumento Tipo Descripción
col Color The colour to set for drawing.

Devuelve:

N/A

Ejemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

El código anterior dibujará un texto en la posición especificada con un efecto de sombra creado por la modificación del alfa y el color del dibujo.
