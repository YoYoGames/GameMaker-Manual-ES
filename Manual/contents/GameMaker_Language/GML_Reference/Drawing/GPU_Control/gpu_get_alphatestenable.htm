gpu_get_alphatestenable

Con esta función puede comprobar si la prueba alfa está activada (devuelve true) o no (devuelve false). Para más información sobre la prueba alfa, véase la función gpu_set_alphatestref().

Sintaxis:

gpu_get_alphatestenable();

Devuelve:

Ejemplo:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

El código anterior comprobará si la prueba de alfa está activada y, si no lo está, switch en la prueba de alfa y establecerá el umbral de la prueba en 128 (sólo se dibujarán los píxeles con un alfa superior a 0,5).
