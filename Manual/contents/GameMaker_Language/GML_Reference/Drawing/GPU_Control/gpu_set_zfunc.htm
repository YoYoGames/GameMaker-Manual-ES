gpu_set_zfunc

Esta función puede utilizarse para establecer el modo de comparación de las pruebas del búfer z (véase gpu_set_ztestenable() para más información). Los valores disponibles para su uso son cualquiera de las siguientes constantes (el valor por defecto es cmpfunc_lessequal):

Constante Descripción
cmpfunc_never Nunca
cmpfunc_less Menos
cmpfunc_equal Equal
cmpfunc_lessequal Menos o igual
cmpfunc_greater Mayor
cmpfunc_notequal No es igual
cmpfunc_greaterequal Mayor o igual
cmpfunc_always Siempre

Sintaxis:

gpu_set_zfunc(cmp_func);

Argumento Tipo Descripción
cmp_func Función de comparación Constante The comparison mode to use (see list above)

Devuelve:

N/A

Ejemplo:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

El código anterior activa la prueba del búfer z y establece su modo de comparación antes de dibujar un fondo sprite y luego lo desactiva de nuevo para continuar dibujando.
