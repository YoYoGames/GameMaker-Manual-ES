gpu_get_blendmode_ext_sepalpha

Esta función puede utilizarse para recuperar el modo de fusión extendido que se está utilizando para dibujar, separando los modos de fusión en dos partes: el componente RGB y el componente Alfa. La función devuelve un array 1D de 4 elementos con los siguientes elementos:

  • [0] = Modo de fusión de la fuente menos el componente alfa (por defecto es bm_src_alpha (As, As, As))
  • [1] = Modo de fusión de destino menos el componente alfa (por defecto es bm_inv_src_alpha (1-As, 1-As, 1-As))
  • [2] = Componente alfa de la fuente (por defecto es bm_src_alpha (As))
  • [3] = Componente alfa de destino (por defecto es bm_inv_src_alpha (1-As))

Los valores de la matriz serán una de las siguientes constantes ("s" denota un valor tomado del origen mientras que "d" denota un valor del destino):

Constante Factor de mezcla (rojo, verde, azul, alfa)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Ad, Ad, Ad, Ad)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) donde f = min(As, 1-Ad)

Al crear los modos de mezcla extendidos de esta manera, los componentes RGB se tomarán de un modo de mezcla y el componente alfa se tomará de otro, de modo que -por ejemplo- si se utiliza bm_zero (0, 0, 0, 0) como modo de mezcla RGB y bm_one como modo de mezcla alfa, se obtendrá una mezcla compuesta de (0,0,0,1).

Tenga en cuenta que puede cambiar estos valores directamente dentro del array y pasar el array completo a la función como un único argumento (como se muestra en el ejemplo siguiente).

Sintaxis:

gpu_get_blendmode_ext_sepalpha();

Devuelve:

(4 elements only; see above for constants)

Ejemplo:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

El código anterior crea una matriz local y obtiene el modo de fusión actual extendido con valores alfa separados. Este array es entonces manipulado y devuelto para establecer el modo de mezcla.
