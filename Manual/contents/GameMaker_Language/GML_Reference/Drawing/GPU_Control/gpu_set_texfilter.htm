gpu_set_texfilter

Esta función se puede utilizar para establecer el filtrado de la textura (interpolación lineal) de todas las imágenes dibujadas en la pantalla del juego. Cuando se activa (true) todas las texturas se suavizan cuando se dibujan (esto incluye sprites ya que también se consideran texturas), lo que significa que cuando se escalan o se mueven si no hay una relación de píxeles de 1:1 entonces habrá una "mancha" a través de varios píxeles que puede hacer que las imágenes aparezcan borrosas dependiendo del estilo de arte utilizado. Si esta opción está desactivada (false), las imágenes se dibujarán basándose en el píxel más cercano cuando se escalen o muevan, lo que puede dar lugar a imágenes "en bloque". El valor por defecto es false, y esto también se puede cambiar en las Opciones Globales del Juego para las plataformas individuales de destino.

NOTA: Este ajuste anulará cualquier interpolación de etapa de textura establecida para shaders utilizando la función gpu_set_texfilter_ext().

NOTA: En el objetivo HTML5, esta función sólo funcionará con WebGL activado.

Sintaxis:

gpu_set_texfilter(enable);

ArgumentoTipo Descripción
enable Enable or disable texture filtering (true / false)

Devuelve:

Ejemplo:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

El código anterior comprueba si el filtrado de texturas está activado o desactivado y lo cambia en consecuencia.
