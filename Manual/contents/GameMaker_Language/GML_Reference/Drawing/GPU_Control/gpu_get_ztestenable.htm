gpu_get_ztestenable

Esta función puede utilizarse para obtener si la prueba z está activada (la función devuelve true) o no (la función devuelve false). El valor por defecto es que la prueba z está desactivada, por lo que la función devolverá false.

Sintaxis:

gpu_get_ztestenable();

Devuelve:

Ejemplo:

if gpu_get_ztestenable() == false
{
    gpu_set_ztestenable(true);
}

El código anterior comprueba si la prueba z está activada o no y, si está desactivada, se vuelve a activar.
