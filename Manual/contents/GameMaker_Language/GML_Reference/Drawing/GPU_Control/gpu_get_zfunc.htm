gpu_get_zfunc

Esta función puede utilizarse para recuperar el modo de comparación de z. El valor devuelto será una de las siguientes constantes (el valor por defecto es cmpfunc_lessequal):

Función de comparación Constante
Constante Descripción
cmpfunc_never Nunca
cmpfunc_less Menos
cmpfunc_equal Equal
cmpfunc_lessequal Menos o igual
cmpfunc_greater Mayor
cmpfunc_notequal No es igual
cmpfunc_greaterequal Mayor o igual
cmpfunc_always Siempre

Sintaxis:

gpu_get_zfunc();

Devuelve:

Comparison Function Constant (see table above)

Ejemplo:

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

El código anterior comprueba si el método de prueba z está establecido en cmpfunc_greater y si no es así, se establece en esa constante.
