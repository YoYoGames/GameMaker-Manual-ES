gpu_get_blendmode

Esta función se puede utilizar para recuperar el modo de mezcla actual que se está utilizando para dibujar. El valor devuelto será una de las siguientes constantes (el valor por defecto es bm_normal):

Modo de mezcla constante
Constante Descripción Modo de fusión ampliado
bm_normal Mezcla normal (el modo de mezcla por defecto). (bm_src_alpha, bm_inv_src_alpha)
bm_add Mezcla aditiva. Se añaden los valores de luminosidad de las áreas luminosas. (bm_src_alpha, bm_one)
bm_subtract Mezcla sustractiva. Se restan los valores de luminosidad de las zonas claras. (bm_zero, bm_inv_src_colour)
bm_max Mezcla máxima. Similar a la mezcla aditiva. (bm_src_alpha, bm_inv_src_colour)

Sintaxis:

gpu_get_blendmode();

Devuelve:

Blend Mode Constant (see above for constants)

Ejemplo:

if gpu_get_blendmode() != bm_normal
{
    gpu_set_blendmode(bm_normal);
}

El código anterior obtiene el modo de mezcla actual y si no es bm_normal se establece en esa constante.
