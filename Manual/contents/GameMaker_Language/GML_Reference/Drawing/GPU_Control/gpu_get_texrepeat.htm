gpu_get_texrepeat

Con esta función puedes comprobar si la repetición de textura está activada (devuelve true) o no (devuelve false). Para más información sobre la repetición de texturas, véase la función gpu_set_texrepeat().

Sintaxis:

gpu_get_texrepeat();

Devuelve:

Ejemplo:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

El código anterior comprueba si la repetición de la textura está activada o desactivada y la cambia en consecuencia.
