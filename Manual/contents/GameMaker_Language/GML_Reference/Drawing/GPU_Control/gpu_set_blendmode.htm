gpu_set_blendmode

Cuando GameMaker va a dibujar un píxel hay un color de origen (el color del píxel que vamos a dibujar) y un color de destino (el color que ya está en el píxel que estamos dibujando), por lo que al determinar el nuevo color del píxel, los colores de origen y destino se calculan de acuerdo con el modo de mezcla elegido. Cada componente de los colores se almacena como un valor de punto flotante entre 0 y 1, y el nuevo color se calcula multiplicando cada componente del color de origen por algún factor y multiplicando cada componente del color de destino por algún otro factor y luego sumando los resultados componente por componente.

Esta función permite establecer el modo de mezcla de cuatro maneras básicas utilizando las siguientes constantes:

Constante Descripción Modo de fusión ampliado
bm_normal Mezcla normal (el modo de mezcla por defecto). (bm_src_alpha, bm_inv_src_alpha)
bm_add Mezcla aditiva. Se añaden los valores de luminosidad de las áreas luminosas. (bm_src_alpha, bm_one)
bm_subtract Mezcla sustractiva. Se restan los valores de luminosidad de las zonas claras. (bm_zero, bm_inv_src_colour)
bm_max Mezcla máxima. Similar a la mezcla aditiva. (bm_src_alpha, bm_inv_src_colour)

Como puede ver en la tabla anterior, estos cuatro modos de fusión son en realidad compuestos de modos de fusión extendidos que se pueden encontrar en la página que describe gpu_set_blendmode_ext().

Para ayudarte a sacar el máximo provecho de los modos de fusión y entender cómo funcionan y cómo afectan a la imagen final que se dibuja en la pantalla, te recomendamos que leas la siguiente guía:

  • Guía de uso de Blendmodes

Sintaxis:

gpu_set_blendmode(mode);

Argumento Tipo Descripción
mode Modo de mezcla constante The blend mode to use (see the table above)

Devuelve:

N/A

Ejemplo:

gpu_set_blendmode(bm_add);
draw_circle_colour(100, 100, 50, c_white, c_black, 0);
gpu_set_blendmode(bm_normal);

Esto convertirá el negro en transparencia, creando un efecto de "resplandor" al ser el blanco fuerte en el exterior y debilitarse gradualmente a partir del centro del círculo. El modo de fusión se restablece después de dibujar el círculo para asegurar que la fusión aditiva no se aplique también a todo lo que viene después.
