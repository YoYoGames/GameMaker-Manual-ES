gpu_set_fog

Esta función se puede utilizar para activar o desactivar el dibujo de la niebla. La niebla se puede utilizar en los juegos 3D para hacer que las instancias en la distancia se vean desvanecidas o incluso invisibles, lo que ayuda a crear una atmósfera, así como a enmascarar el hecho de que no se están dibujando instancias que están lejos.

Se establece si está activada (true) o desactivada (false), el color que la niebla debe utilizar para la mezcla, así como las distancias de dibujo inicial y final. Los valores de distancia son esencialmente valores de profundidad en píxeles, y son relativos a la posición de la cámara.

Por ejemplo, una posición de 0 es la más cercana a la cámara, y con cada incremento se aleja más. Por defecto, la cámara de una vista se sitúa a una profundidad de -16000, por lo que si quisieras que la niebla empezara en la profundidad 0 y terminara en la profundidad 1000, establecerías la distancia inicial en 16000 y la final en 17000.

La función puede tomar cuatro argumentos individuales (dados arriba), o un solo array con la siguiente estructura (el código de ejemplo que sigue muestra este método):

  • [0] = alternancia habilitada (un booleano, ya sea true o false), por defecto false
  • [1] = color (real), por defecto c_black
  • [2] = distancia inicial (real), por defecto 0
  • [3] = distancia final (real), por defecto 1

Sintaxis:

gpu_set_fog(enable, colour, start, end);

Argumento Tipo Descripción
enable Booleano Enable or disable fog
colour Color The fog colour
start Real The distance to start applying fog from (relative to the camera)
end Real The distance to end the fog (relative to the camera)

Devuelve:

N/A

Ejemplo:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

El código anterior obtiene la configuración actual de la niebla y luego establece el elemento de color de la matriz a c_red antes de establecer la niebla de nuevo utilizando la matriz modificada.
