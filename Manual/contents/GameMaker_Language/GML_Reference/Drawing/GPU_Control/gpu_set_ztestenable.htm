gpu_set_ztestenable

Esta función se puede utilizar para activar o desactivar la prueba del búfer z, lo que afecta a cómo se dibujarán las cosas (en general, sólo es útil cuando se trabaja con proyectos 3D). Esencialmente, por defecto cuando la prueba z está desactivada y tienes dos objects dibujando en el mismo espacio, ambos objects se renderizarán sin tener en cuenta si uno de ellos sobredibujará al otro, resultando en llamadas de dibujo innecesarias. Si switch lo activa, el búfer z se comprueba para ver si un object será "visible" y no se dibuja si no lo es. no lo es. Ten en cuenta que este es el comportamiento por defecto, pero puedes cambiarlo modificando el tipo de comparación utilizado para la comprobación del z-buffer (ver la función gpu_set_zfunc().

Por defecto, la prueba del búfer z está desactivada (false).

Sintaxis:

gpu_set_ztestenable(enable);

ArgumentoTipo Descripción
enable Enable or disable z-buffer testing (true o false).

Devuelve:

Ejemplo:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

El código anterior activa la prueba del búfer z para dibujar un fondo sprite y luego la desactiva para seguir dibujando.
