gpu_get_blendmode_ext

Esta función se puede utilizar para recuperar el modo de mezcla extendido que se está utilizando para dibujar. La función devuelve un array 1D de 2 elementos con los siguientes elementos:

  • [0] = Fuente Constante del factor de mezcla (por defecto es bm_src_alpha)
  • [1] = Destino Constante del factor de mezcla (por defecto es bm_inv_src_alpha)

Los valores de la matriz serán una de las siguientes constantes ("s" denota un valor tomado del origen mientras que "d" denota un valor del destino):

Factor de mezcla constante
Constante Factor de mezcla (rojo, verde, azul, alfa)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Ad, Ad, Ad, Ad)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) donde f = min(As, 1-Ad)

Tenga en cuenta que puede cambiar estos valores y pasar la matriz completa a la función gpu_set_blendmode_ext() como argumento único.

Sintaxis:

gpu_get_blendmode_ext();

Devuelve:

Array (2 elements only; see above for constants)

Ejemplo:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

El código anterior obtiene el modo de fusión extendido actual, modifica la fuente y luego establece el modo de fusión extendido de nuevo.
