gpu_get_alphatestref

Puede utilizar esta función para encontrar el valor actual de la referencia de prueba alfa (por defecto es 0, pero puede utilizar gpu_set_alphatestref() para establecer este valor a algo distinto de esto).

Sintaxis:

gpu_get_alphatestref();

Devuelve:

(from 0 - 255)

Ejemplo:

if gpu_get_alphatestenable()
{
    if (gpu_get_alphatestref() < 254)
    {
        gpu_set_alphatestref(254);
    }
}

El código anterior comprueba si la prueba alfa está activada, y si lo está, entonces comprueba el valor de referencia de la prueba alfa actual y lo establece si es menor que 254.
