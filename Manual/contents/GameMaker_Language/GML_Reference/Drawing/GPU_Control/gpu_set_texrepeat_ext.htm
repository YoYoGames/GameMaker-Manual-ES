gpu_set_texrepeat_ext

Esta función se puede utilizar para establecer si una sola "ranura" de muestreo repite la textura dada cuando se utiliza Shaders en GameMaker. Si se establece en true se repetirá la textura si las coordenadas uv están fuera del rango 0-1, mientras que un ajuste de false significará que no se repite. El caso de uso probable para estas funciones es para repetir una textura en 3D, pero para que funcione y no tire de las imágenes del resto de la página de la textura, el sprite utilizado tendrá que ser marcado como estar en una "página de textura separada" en el Editor Sprite.

NOTA Este ajuste será anulado por el valor establecido al llamar a la función gpu_set_texrepeat().

Sintaxis:

gpu_set_texrepeat_ext(sampler_id, enable);

ArgumentoTipo Descripción
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Devuelve:

Ejemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

El código anterior comprueba si el filtrado de textura está activado o desactivado para un ID de muestreador específico (almacenado en una variable local) y lo cambia en consecuencia.
