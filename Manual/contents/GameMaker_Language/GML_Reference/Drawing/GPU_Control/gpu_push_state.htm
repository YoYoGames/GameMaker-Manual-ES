gpu_push_state

Con esta función puedes empujar el estado actual de la GPU a una pila para usarlo más tarde. Por lo general, esto se utiliza si se quiere "guardar" el estado actual de la GPU (cosas como el modo de mezcla, la escritura alfa, el culling, etc ... serán empujados a la pila), a continuación, dibujar algo con diferentes ajustes, y luego restablecer la pila de la GPU a lo que era antes (llamando a gpu_pop_state()).

Sintaxis:

gpu_push_state();

Devuelve:

Ejemplo:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

El código anterior "guardará" el estado actual de la GPU en la pila, luego cambiará ciertos ajustes de la GPU y dibujará un grupo de instancias antes de restablecer el estado de la GPU a lo que era anteriormente.
