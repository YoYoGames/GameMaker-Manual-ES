gpu_get_cullmode

Esta función se puede utilizar para recuperar el modo de selección de la cara posterior. El valor devuelto será una de las siguientes constantes (el valor por defecto es cull_noculling):

Modo de sacrificio constante
Constante Descripción
cull_noculling No se hará ningún sacrificio
cull_clockwise Todos los triángulos en el sentido de las agujas del reloj serán eliminados
cull_counterclockwise Todos los triángulos en sentido contrario a las agujas del reloj serán eliminados

Sintaxis:

gpu_get_cullmode();

Devuelve:

Culling Mode Constant (see above for constants)

Ejemplo:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

El código anterior obtiene el modo de sacrificio actual y si no es cull_clockwise se establece en esa constante.
