gpu_set_cullmode

Esta función se puede utilizar para establecer el modo de eliminación de la cara posterior. Un polígono (primitivo) tiene una cara frontal y otra trasera. Se dice que el lado frontal es el lado donde los vértices están definidos en orden contrario a las agujas del reloj. Normalmente se dibujan ambos lados, pero si haces una forma cerrada (como una pirámide) entonces esto es un desperdicio de poder de procesamiento porque el lado posterior del triángulo nunca puede ser visto. Es en estos casos en los que puedes switch en la selección de la cara posterior, ya que esto ahorra aproximadamente la mitad del tiempo de dibujo, aunque significa que tienes la tarea de definir tus polígonos de la manera correcta para evitar problemas.

El valor del modo de sacrificio utilizado debe ser una de las siguientes constantes (el valor por defecto es cull_noculling):

Constante Descripción
cull_noculling No se hará ningún sacrificio
cull_clockwise Todos los triángulos en el sentido de las agujas del reloj serán eliminados
cull_counterclockwise Todos los triángulos en sentido contrario a las agujas del reloj serán eliminados

Sintaxis:

gpu_set_cullmode(cullmode);

Argumento Tipo Descripción
cullmode Modo de sacrificio constante The cull mode to use (see list above)

Devuelve:

N/A

Ejemplo:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

El código anterior obtiene el modo de sacrificio actual y si no es cull_clockwise se establece en esa constante.
