gpu_set_blendenable

Esta función se puede utilizar para activar y desactivar la mezcla alfa. Básicamente, si se establece en false, todas las imágenes que se dibujen serán 100% opacas, lo que significa que cualquier área transparente o semitransparente de sprite o del fondo será visible como un color sólido. serán visibles como un color sólido. Es una buena idea tener la mezcla alfa desactivada siempre que sea posible (especialmente cuando se desarrolla para dispositivos móviles) ya que esto aumenta en gran medida la velocidad de dibujo.

Sintaxis:

gpu_set_blendenable(enable);

ArgumentoTipo Descripción
enable Enable or disable alpha blending value (true o false).

Devuelve:

Ejemplo:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

El código anterior desactiva la mezcla alfa para dibujar un fondo sprite y luego la vuelve a activar para seguir dibujando.
