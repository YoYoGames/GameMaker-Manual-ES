gpu_set_alphatestenable

Esta función activará o desactivará la prueba alfa para tu juego (por defecto está desactivada). Al activar la prueba alfa, puede utilizar la función complementaria gpu_set_alphatestref() para establecer el valor de "corte" en el que todos los valores de alfa se pondrán a 0. La imagen de abajo muestra la diferencia que se puede ver cuando la prueba de alfa está activada o desactivada:

NOTA: Esta función puede afectar negativamente al rendimiento de los dispositivos iOS y Android.

Sintaxis:

gpu_set_alphatestenable(enable);

ArgumentoTipo Descripción
enable Enable or disable alpha testing (true / false)

Devuelve:

Ejemplo:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

El código anterior comprobará si la prueba de alfa está activada y, si no lo está, switch en la prueba de alfa y establecerá el umbral de la prueba en 128 (sólo se dibujarán los píxeles con un alfa superior a 0,5).
