gpu_set_texrepeat

Esta función puede utilizarse para indicar a GameMaker si las texturas deben repetirse (true) o no (false) cuando se utilizan para modelos y primitivas. Normalmente las anchuras y alturas de las texturas están en el rango 0 - 1, pero si se especifica un valor mayor que 1 por defecto el resto no se dibuja (toda la textura se dibujará con una sección "vacía" correspondiente al porcentaje sobre 1 que se utilice. Así, un valor de 2 para el ancho o el alto de una textura dibujaría la textura en la mitad del espacio y dejaría la otra mitad vacía). Sin embargo, al usar esta función para establecer la repetición a true la textura se repite automáticamente sobre todos los modelos y primitivas. El caso probable de uso de estas funciones es para repetir una textura en 3D, pero para que funcione y no saque imágenes del resto de la página de texturas, el sprite utilizado tendrá que ser marcado como en una "Página de Textura Separada" en el Editor Sprite.

NOTA: Sprites y los fondos se dibujan siempre sin repetirse, por lo que una vez que se dibuja un sprite o un fondo este valor se restablece a false. Este comportamiento por defecto no afectará a la configuración del escenario de la textura (es decir, llamar a la función lo hará, pero el ajuste interno a false no lo hará).

NOTA: Este ajuste anulará cualquier repetición de etapa de textura establecida para shaders cuando se utilice la función gpu_set_texrepeat_ext().

Sintaxis:

gpu_set_texrepeat(enable);

ArgumentoTipo Descripción
enable Enable or disable texture filtering (true / false)

Devuelve:

Ejemplo:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

El código anterior comprueba si la repetición de la textura está activada o desactivada y la cambia en consecuencia.
