gpu_get_texrepeat_ext

Con esta función puedes comprobar si la repetición de la textura está activada (devuelve true) o no (devuelve false) para una textura de muestreo dada de shader.

Sintaxis:

gpu_get_texrepeat_ext(sampler_id);

ArgumentoTipo Descripción
sampler_id The sampler id from the shader.

Devuelve:

Ejemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (!gpu_get_texrepeat_ext(s_tex))
{
    gpu_set_texrepeat_ext(s_tex, true);
}

El código anterior comprueba si el filtrado de la textura está desactivado para un ID de muestreador específico (almacenado en una variable local) y lo activa si no lo está.
