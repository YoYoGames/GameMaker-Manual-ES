gpu_set_colourwriteenable

Con esta función puede switch activar o desactivar los canales de color y el canal alfa para todo el dibujo posterior. Por ejemplo, puede utilizar esta función para crear superficies sólo con el canal alfa (switch desactivar la escritura RGB antes de utilizar la superficie), o para crear efectos especiales mientras se dibuja en la pantalla. El valor por defecto para cada uno de los componentes es true, y puede ser suministrado como cuatro argumentos únicos o como un array 1D de 4 elementos en el que se encuentran los siguientes elementos que serán true (habilitado) o false (deshabilitado):

  • [0] = Canal rojo activado/desactivado
  • [1] = Canal verde activado/desactivado
  • [2] = Canal azul activado/desactivado
  • [3] = Canal alfa activado/desactivado

Sintaxis:

gpu_set_colorwriteenable(red, green, blue, alpha);

Argumento Tipo Descripción
red Booleano Enable/disable the red channel
green Booleano Enable/disable the green channel
blue Booleano Enable/disable the blue channel
alpha Booleano Enable/disable the alpha channel

Devuelve:

N/A

Ejemplo:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

El código anterior obtiene los valores de escritura del color actual y luego establece el componente alfa a false.
