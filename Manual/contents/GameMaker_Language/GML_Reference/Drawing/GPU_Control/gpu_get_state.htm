gpu_get_state

Esta función obtendrá el estado actual de la GPU, devolviéndolo como un Mapa DS. Este puede ser manipulado o incluso guardado, y puedes devolver este mapa a la GPU usando la función gpu_set_state().

Sintaxis:

gpu_get_state();

Devuelve:

DS Map ID

Ejemplo:

gpu_map = gpu_get_state();

El código anterior almacena el estado actual de la GPU en una variable.
