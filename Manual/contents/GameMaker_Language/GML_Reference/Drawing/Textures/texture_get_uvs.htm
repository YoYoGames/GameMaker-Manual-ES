texture_get_uvs

Esta función devuelve un array 1D con 4 elementos que representan las coordenadas UV para el puntero de textura dado, rellenando el array con los siguientes valores:

  • [0] = izquierda
  • [1] = arriba
  • [2] = derecho
  • [3] = fondo
  • [4] = cantidad de píxeles que el compilador de asset ha recortado del lado izquierdo de sprites (sprite assets solamente)
  • [5] = cantidad de píxeles que el compilador de asset ha recortado de la parte superior de sprites (sprite assets solamente)
  • [6] = porcentaje normalizado de datos de píxeles del ancho original de sprites que se ha guardado en la página de la textura (sprite assets solamente)
  • [7] = porcentaje normalizado de datos de píxeles de la altura original de sprites que se ha guardado en la página de la textura (sprite assets solamente)

Este valor se puede utilizar en otras funciones de dibujo, especialmente en las funciones generales de 3D y algunas de las primitivas de 2D, así como en las funciones de Shader. Si necesita el UVS para un sprite entonces puede utilizar el sprite_get_uvs() y para un font, font_get_uvs().

Sintaxis:

texture_get_uvs(texid)

Argumento Tipo Descripción
texid Textura The texture pointer to get the UVS for

Devuelve:

Array (4 - 8 elements)

Ejemplo:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

El código anterior primero recupera la textura para la superficie almacenada en surf_back, y luego obtiene las coordenadas UV para esa textura. A continuación, recupera las coordenadas UV izquierda, superior, derecha e inferior de la matriz devuelta y las almacena en variables locales.
