texturegroup_get_fonts

Con esta función puedes recuperar el índice font de cada uno de los fonts asignados a las páginas de textura dentro del grupo de texturas dado. Se suministra el ID del grupo de texturas string (tal y como se define en el Editor de grupos de texturas) y la función devolverá una matriz 1D en la que cada entrada contiene el índice font de un recurso font. Si la función falla, es decir, si se proporciona un grupo no válido o si el grupo no tiene ninguna textura asignada, la matriz estará vacía (longitud 0).

Sintaxis:

texturegroup_get_fonts(tex_id);

Argumento Tipo Descripción
tex Cadena The name of the texture group to check (a string)

Devuelve:

Array

Ejemplo:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
}

El código anterior recuperará los índices de font para el grupo de texturas "MainMenu", y luego mostrará esos IDs en la ventana de salida de la consola.
