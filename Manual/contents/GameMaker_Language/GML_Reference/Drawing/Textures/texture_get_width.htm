texture_get_width

Devuelve el ancho de la textura con el id dado, que siempre es un valor dentro del rango 0 - 1. Esto puede ser usado cuando se mapean texturas a modelos o primitivas.

Sintaxis:

texture_get_width(tex);

Argumento Tipo Descripción
tex Puntero de textura The texture page asset pointer to use

Devuelve:

Real

Ejemplo:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

El código anterior obtendrá el ancho de la textura tomada de un sprite asset.
