texture_get_texel_height

Un texel, o elemento de textura, es la unidad fundamental del espacio de textura utilizado en los gráficos por ordenador. Las texturas se representan mediante matrices de texels, al igual que las imágenes se representan mediante matrices de píxeles, y esta función devuelve la altura de un solo texel de la página de textura de la imagen asset utilizada.

Sintaxis:

texture_get_texel_height(tex);

Argumento Tipo Descripción
tex Puntero de textura The texture page asset pointer to use

Devuelve:

Real

Ejemplo:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

El código anterior obtendrá el ancho y el alto del texel de la textura tomada de un sprite asset.
