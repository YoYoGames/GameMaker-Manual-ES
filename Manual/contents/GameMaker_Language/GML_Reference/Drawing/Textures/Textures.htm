Texturas

Las siguientes funciones están todas relacionadas con las "texturas", es decir, las imágenes que se almacenan en la VRAM para ser utilizadas como sprites u otras cosas dentro de tu juego. El primer conjunto de funciones disponibles se utiliza para recuperar información sobre imágenes específicas y su posición o tamaño en la Página de Textura, así como para establecer ciertas características de la textura:

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

GameMaker también tiene una serie de funciones relacionadas con la gestión de la VRAM asociada a las texturas individuales, así como a los grupos de texturas completos:

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

Por último, tenemos las siguientes funciones que se utilizan principalmente para depurar el proyecto y asegurar un uso eficiente de la memoria de la textura:

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
