draw_texture_flush

Con esta función puedes eliminar todas las texturas de la memoria de vídeo, y se volverán a cargar en el primer uso. Esta es la única forma efectiva de gestionar la memoria de vídeo cuando tienes varias páginas de texturas para un juego, y deberías vaciar la memoria de texturas entre niveles en tu juego y organizar los gráficos usando la función de grupo de texturas para asegurar que se utiliza el mínimo número de texturas.

Vale la pena señalar que , por defecto, en todos los objetivos, las páginas de textura sólo se cargarán a medida que se requieran y se pueden utilizar las diversas funciones de prefetch y flush para sprites (que se encuentran aquí) para gestionar las cosas, así como esta función.

Sintaxis:

draw_texture_flush();

Devuelve:

Ejemplo:

draw_texture_flush();

El código anterior vacía la memoria de vídeo de datos de textura, y probablemente se colocaría en el evento de creación de la primera instancia de un object colocado en el room.
