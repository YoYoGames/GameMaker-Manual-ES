textura_de_lanzamiento

Esta función puede usarse para "limpiar" una página de textura o un grupo de páginas de textura, es decir, eliminarlas de la VRAM cuando ya no sean necesarias. Puedes suministrar el ID de la página de textura (tal y como lo devuelven las funciones de texturegroup_* ) para vaciar una sola página, o puedes suministrar el nombre de un grupo de texturas (tal y como se define en el Editor de Grupos de Texturas) para vaciar todas las páginas de textura del grupo.

Sintaxis:

texture_flush(tex_id);

Argumento Tipo Descripción
tex_id Puntero o cadena detextura The texture page pointer or a texture group name (a string)

Devuelve:

N/A

Ejemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_flush(_tex_array[i]);
}

El código anterior hará que se vacíen todas las páginas de texturas bajo el grupo de texturas "MainMenu".
