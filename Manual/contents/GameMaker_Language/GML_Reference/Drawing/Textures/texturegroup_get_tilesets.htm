texturegroup_get_tilesets

Con esta función puedes recuperar el índice del tileset de cada uno de los tilesets asignados a las páginas de textura dentro del grupo de texturas dado. Usted suministra el ID del grupo de texturas string (como se define en el Editor de Grupos de Texturas) y la función devolverá un array 1D donde cada entrada contiene el índice del tileset para un recurso del tileset. Si la función falla, es decir, si se proporciona un grupo no válido o si el grupo no tiene ninguna textura asignada, la matriz estará vacía (longitud 0).

Sintaxis:

texturegroup_get_tilesets(tex_id);

Argumento Tipo Descripción
tex Cadena The name of the texture group to check (a string)

Devuelve:

Array

Ejemplo:

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
}

El código anterior recuperará los índices del tileset para el grupo de texturas "MainMenu", y luego mostrará esos IDs en la ventana de salida de la consola.
