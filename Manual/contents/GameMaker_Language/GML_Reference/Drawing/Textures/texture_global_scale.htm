textura_escala_global

Esta función permite controlar el escalado de las páginas de texturas al cargarlas desde el archivo WAD que forma parte del ejecutable final del juego. El valor de entrada debe ser un valor de potencia de dos y funcionará así:

  • Para configurar las páginas de textura para que sean 1:1 utilice texture_global_scale(1)
  • Para las medias texturas todo el tiempo utiliza texture_global_scale(2)
  • Para descuartizarlos utiliza texture_global_scale(4), etc...

De esta manera puedes controlar mejor el uso de la memoria de la página de texturas en plataformas con problemas de poca memoria. Ten en cuenta que las páginas de texturas se crean bajo demanda desde el WAD, por lo que puedes llamar a esto al inicio del juego para cargar todas las texturas a escala, o puedes puede llamarlo en momentos específicos para cargar texturas específicas, en cuyo caso probablemente querrá llamar a draw_texture_flush() antes de llamar a esta función, y luego utilizar las funciones de pre-fetch para obtener la correcta páginas de textura en la memoria.

Sintaxis:

texture_global_scale(pow2integer);

ArgumentoTipo Descripción
pow2integer The scale factor to use (1, no scale, 2, half scale, 4, quarter scale, etc...)

Devuelve:

Ejemplo:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

El código anterior vaciará todas las texturas de la memoria, luego establecerá el escalado de texturas a 2 (para que las páginas de texturas tengan la mitad de tamaño) y finalmente precargará la página de texturas que contiene el sprite "spr_Trees".
