texture_debug_messages

Esta función se puede utilizar para activar o desactivar los mensajes de depuración de la textura. Cuando está habilitada (establecida a true), se enviará información adicional sobre el uso de la página de textura a la ventana de salida. Establezca a false para desactivar esta salida de nuevo.

Sintaxis:

texture_debug_messages(enable);

ArgumentoTipo Descripción
enable Enable or disable the texture debug messages

Devuelve:

Ejemplo:

if global.debug
{
    texture_debug_messages(true);
}

El código anterior comprobará el valor de una variable global y si se establece en true entonces los mensajes de depuración de la textura están habilitados.
