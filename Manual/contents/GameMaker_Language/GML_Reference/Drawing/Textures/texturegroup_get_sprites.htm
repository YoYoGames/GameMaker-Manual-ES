texturegroup_get_sprites

Con esta función puedes recuperar el índice sprite de cada uno de los sprites asignados a las páginas de textura dentro del grupo de texturas dado. Se suministra el ID del grupo de texturas string (tal y como se define en el Editor de Grupos de Texturas) y la función devolverá una matriz 1D en la que cada entrada contiene el índice sprite de un recurso sprite. Si la función falla, es decir, si se proporciona un grupo no válido o si el grupo no tiene ninguna textura asignada, la matriz estará vacía (longitud 0).

Sintaxis:

texturegroup_get_sprites(tex_id);

Argumento Tipo Descripción
tex Cadena The name of the texture group to check (a string)

Devuelve:

Array

Ejemplo:

var _tex_array = texturegroup_get_sprites( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i]));
}

El código anterior recuperará los índices de sprite para el grupo de texturas "MainMenu", y luego mostrará esos IDs en la ventana de salida de la consola.
