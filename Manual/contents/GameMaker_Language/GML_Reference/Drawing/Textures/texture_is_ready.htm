textura_está_lista

Esta función puede utilizarse para comprobar si una página de textura específica ha sido desempaquetada y está lista para su uso, o si un grupo de páginas de textura ha sido desempaquetado y está listo para su uso. Se suministra el ID de la página de textura única (tal y como lo devuelven las funciones de texturegroup_* ) o el ID del grupo de texturas string (tal y como se define en el editor de grupos de texturas), y la función devolverá true si se han desempaquetado, o false en caso contrario.

Sintaxis:

texture_is_ready(tex_id);

Argumento Tipo Descripción
tex_id Puntero o cadena detextura The texture page pointer or a texture group name (a string)

Devuelve:

Boolean

Ejemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

El código anterior recuperará los IDs de las páginas de texturas para el grupo de texturas "MainMenu", luego comprobará si están desempaquetadas, y si no lo están, serán pregrabadas en la VRAM.
