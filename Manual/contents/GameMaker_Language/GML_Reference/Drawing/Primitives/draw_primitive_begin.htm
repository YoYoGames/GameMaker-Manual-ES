draw_primitive_begin

Esta función debe ser llamada antes de poder definir cualquier primitiva. Hay 6 tipos de primitivas que se pueden definir como las siguientes constantes:

Constante de tipo primitivo
Constante Descripción
pr_pointlist Una lista de puntos
pr_linelist Una lista de líneas
pr_linestrip Una tira de líneas
pr_trianglelist Una lista de triángulos
pr_trianglestrip Una franja triangular
pr_trianglefan Un ventilador triangular

La siguiente imagen ilustra básicamente el aspecto que deben tener y también el orden en el que debes definir los vértices:

Ten en cuenta que en algunas plataformas (Windows, UWP, XBox) el tipo pr_trianglefan no está soportado de forma nativa y por eso GameMaker hace una conversión cuando se compila el juego para que funcione. Esto significa que en esas plataformas el tipo pr_trianglefan será mucho más lento de usar que los otros. Esto También tenga en cuenta que para utilizar esta función en HTML5, debe habilitar WebGL en las Opciones de Juego.

Sintaxis:

draw_primitive_begin(kind)

Argumento Tipo Descripción
kind Constante de tipo primitivo The kind of primitive you are going to draw.

Devuelve:

N/A

Ejemplo:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

El código anterior dibujará tres cuartos de un círculo hecho con primitivas.
