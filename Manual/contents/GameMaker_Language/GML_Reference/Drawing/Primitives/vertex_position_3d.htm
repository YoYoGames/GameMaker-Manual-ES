posición_vértice_3d

Esta función añadirá datos de posición 3D a la vertex que se está definiendo para la primitiva personalizada. Usted proporciona el buffer para escribir los datos, así como la posición x, y y z para el dibujo.

Sintaxis:

vertex_position_3d(buffer, x, y, z);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to write the information to.
x Real The x position.
y Real The y position.
z Real The z position.

Devuelve:

N/A

Ejemplo:

vertex_position_3d(buff, x - 100, y - 125, 0);

El código anterior fijará la posición del vertex que se está definiendo.
