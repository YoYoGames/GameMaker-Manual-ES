dibujar_textura_color

Esta función define la posición de una textura vertex para una primitiva. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar (Ver draw_primitive_begin() para más información), el orden con el que añadas los vértices a la misma, la posición de los puntos inicial y final que des para la muestra de la textura y los valores de color y alfa que hayas establecido. Para mantener la apariencia de la textura cambiando sólo el alfa, un valor de -1 (o c_white) para el argumento del color. Para terminar y dibujar la primitiva debes llamar a draw_primitive_end().

NOTA Para que una textura se repita debe ser una potencia de dos en tamaño, es decir: 32x32, 128x128, etc...

Sintaxis:

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

ArgumentoTipo Descripción
x The x coordinate of the vertex.
y The y coordinate of the vertex.
xtex The x coordinate within the texture.
ytex The y coordinate within the texture.
col The colour to blend with the texture at this vertex (-1 or c_white para no mezclar).
alpha The alpha to draw this vertex with (0-1).

Devuelve:

Ejemplo:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

El código anterior dibujará una tira de triángulos de 4 vertex (formando un rectángulo) texturizados con la textura mantenida en la variable "tex", y toda la textura se utilizará para cubrir la primitiva completada, y se mezclará con cuatro colores diferentes.
