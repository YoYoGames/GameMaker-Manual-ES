vertex_create_buffer_ext

Al igual que con la función vertex_create_buffer()esta función creará un nuevo vertex buffer. Se trata de un crecimiento especial buffer creado por GameMaker que está preformateado para su uso cuando se construyan primitivas para su uso con shaders. Puede especificar un tamaño inicial de partida para el buffer (en bytes) y devolverá un índice para el buffer que deberá ser utilizado en todas las llamadas posteriores al buffer.

Sintaxis:

vertex_create_buffer_ext(size);

ArgumentoTipo Descripción
size The initial size of the buffer (in bytes).

Devuelve:

Vertex Buffer ID

Ejemplo:

v_buff = vertex_create_buffer_ext(1024 * 1024);

El código anterior creará un nuevo vertex buffer, inicialmente de 1MB de tamaño, y almacenará su manejador en la variable "v_buff".
