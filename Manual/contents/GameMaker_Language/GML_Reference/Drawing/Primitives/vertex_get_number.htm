vertex_get_number

Con esta función se puede averiguar el número de vértices individuales definidos en cualquier vertex buffer.

Sintaxis:

vertex_get_number(buffer);

Argumento Tipo Descripción
buffer Vértice Buffer ID The vertex buffer to check.

Devuelve:

Real

Ejemplo:

v_num = vertex_get_number(v_buffer);

El código anterior almacenará el número de vértices almacenados en el vertex buffer dado a una variable.
