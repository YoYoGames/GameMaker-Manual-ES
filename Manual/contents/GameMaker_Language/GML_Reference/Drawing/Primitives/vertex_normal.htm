vertex_normal

Esta función añadirá datos de la normal de superficie a la vertex que se está definiendo para la primitiva personalizada. Usted proporciona el buffer para escribir los datos, así como las partes de los componentes x, y y z de la normal.

Sintaxis:

vertex_normal(buffer, nx, ny, nz);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to write the information to.
nx Real The x component of the surface normal.
ny Real The y component of the surface normal.
nz Real The z component of the surface normal.

Devuelve:

N/A

Ejemplo:

vertex_normal(buff, 0, 1, 1);

El código anterior establecerá la normal de la superficie de la vertex que se está definiendo.
