vertex_end

Con esta función se termina la construcción de la primitiva personalizada. Una vez que llame a este comando, la primitiva puede ser utilizada en la función vertex_submit() para su uso en un shader o puede congelar el buffer (haciendo que el vertex buffer utilizado sea de sólo lectura y mucho más rápido) usando vertex_freeze().

Sintaxis:

vertex_end(buffer);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to end writing to.

Devuelve:

N/A

Ejemplo:

vertex_end(buff);

El código anterior finalizará la definición de una primitiva personalizada.
