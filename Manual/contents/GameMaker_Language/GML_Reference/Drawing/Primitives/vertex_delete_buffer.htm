vertex_delete_buffer

Esta función puede utilizarse para eliminar de la memoria del sistema un vertex buffer previamente creado (véase vertex_create_buffer()) de la memoria del sistema. Esto también eliminará todos los datos de vertex para cualquier primitiva personalizada que contenga.

Sintaxis:

vertex_delete_buffer(buffer);

Argumento Tipo Descripción
buffer Vértice Buffer ID The vertex buffer to be removed.

Devuelve:

N/A

Ejemplo:

vertex_delete_buffer(v_buff);

El código anterior borrará de la memoria los buffer indexados en la variable "v_buff".
