vertex_texcoord

Esta función establecerá las coordenadas de la textura a utilizar para el vertex que se está definiendo actualmente para la primitiva personalizada. Usted proporciona el buffer para escribir los datos, así como la posición UV dentro de la textura a utilizar.

Sintaxis:

vertex_texcoord(buffer, u, v);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

Devuelve:

N/A

Ejemplo:

vertex_texcoord(buff, 0, 0);

El código anterior establecerá los valores UV de la vertex que se está definiendo.
