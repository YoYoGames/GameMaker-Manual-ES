vertex_format_add_colour

Indique a GameMaker que acepte los datos de color como parte del nuevo formato vertex que se está creando.

Sintaxis:

vertex_format_add_colour();

Devuelve:

Ejemplo:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

El código anterior creará un nuevo formato vertex con sólo los valores de color y posición y luego almacenará el id del formato en la variable "mi_formato".
