vertex_format_add_texcoord

Dile a GameMaker que acepte los datos de posición de las texturas (u y v) como parte del nuevo formato vertex que se está creando.

Sintaxis:

vertex_format_add_texcoord();

Devuelve:

Ejemplo:

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

El código anterior creará un nuevo formato vertex con sólo los valores de textura y posición y luego almacenará el id del formato en la variable "mi_formato".
