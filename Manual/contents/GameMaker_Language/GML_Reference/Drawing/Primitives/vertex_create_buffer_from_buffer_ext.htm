vertex_create_buffer_from_buffer_ext

Al igual que la función vertex_create_buffer(), esta función creará un nuevo vertex buffer, sólo que ahora los datos de vertex que almacena se copian del buffer regular que se especifica como fuente. El buffer está preformateado con el formato vertex para la construcción de primitivas para su uso con (por ejemplo) shaders, y también se puede suministrar un desplazamiento dentro de la fuente buffer desde la que copiar y el número de vértices que debe tener el buffer final. Tenga en cuenta que si el número de vértices no coincide con los que se están copiando puede obtener datos corruptos vertex.

Sintaxis:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

Argumento Tipo Descripción
buffer ID del búfer The buffer to create the vertex buffer from.
format Constante de tipo primitivo The primitive vertex format to use.
src_offset Real The offset within the the source buffer to copy from.
vert_num Real The number of vertices the buffer should have.

Devuelve:

Vertex Buffer ID

Ejemplo:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

El código anterior creará un nuevo formato vertex y luego creará un nuevo vertex buffer a partir de un buffer regular previamente creado, aplicándole el formato personalizado vertex con un desplazamiento de 0. La función indica al nuevo vertex buffer que debe esperar 512 vértices.
