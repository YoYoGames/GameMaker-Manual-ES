vertex_create_buffer_from_buffer

Al igual que la función vertex_create_buffer(), esta función creará un nuevo vertex buffer, sólo que ahora los datos de vertex que almacena se copian del buffer regular que se especifica como fuente. El buffer creado es un crecimiento especial buffer que está preformateado con el formato vertex para construir primitivas para su uso con (por ejemplo) shaders. Esta función requiere que se suministre el puntero a un regular creado previamente buffer, y un formato vertex que debe aplicarse a los datos copiados.

Sintaxis:

vertex_create_buffer_from_buffer(buffer, format);

Argumento Tipo Descripción
buffer ID del búfer The buffer to create the vertex buffer from.
format Constante de tipo primitivo The primitive vertex format to use.

Devuelve:

Vertex Buffer ID

Ejemplo:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

El código anterior creará un nuevo formato vertex y luego creará un nuevo vertex buffer a partir de un buffer regular previamente creado, aplicándole el formato personalizado vertex.
