vertex_argb

Esta función establecerá los valores ARGB para la vertex que se está definiendo para la primitiva personalizada. Usted suministra el buffer para escribir los datos así como los valores rojo, verde, azul y alfa que desea utilizar como un único entero sin signo de 32 bits - la muestra alfa en los 8 bits más altos, seguida por la muestra roja, la muestra verde y finalmente la muestra azul en los 8 bits más bajos. Puede utilizar la notación hexadecimal ($AARRGGBB) un número real o utilizar cualquiera de las funciones de make_colour_*() para definir el valor del color.

Sintaxis:

vertex_argb(buffer, argb);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to write the information to.
argb Color The ARGB value to set.

Devuelve:

N/A

Ejemplo:

vertex_argb(buff, $FFFFFFFF);

El código anterior establecerá los valores ARGB de la vertex que se está definiendo como blanco.
