color_vértice

Si su formato definido vertex toma un valor de color, puede utilizar esta función para añadir esos datos al vertex que se está definiendo para la primitiva actual. La función necesita un buffer para almacenar los datos y tomará una constante de color, o un valor hexadecimal (utilizando el formato estándar de GameMaker de BGR, por ejemplo: $FF0000 para el azul), así como un valor alfa de 0 (transparente) a 1 (totalmente opaco).

Sintaxis:

vertex_colour(buffer, colour, alpha);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to write the information to.
colour Color The colour for this vertex (can be a constant or a hex value).
alpha Real The alpha value for the vertex (from 0 to 1).

Devuelve:

N/A

Ejemplo:

vertex_colour(b, c_white, 1);

El código anterior establecerá el vertex que se está definiendo como blanco con un valor alfa de 1.
