vertex_format_delete

Esta función debe llamarse siempre que se termine de utilizar cualquier formato vectorial creado. Usted proporciona el valor del ID del formato (devuelto por la función vector_format_end()), y esta función liberará la memoria asociada a él. Tenga en cuenta que si intenta utilizar este formato de nuevo después de llamar a esta función, obtendrá un error.

Sintaxis:

vertex_format_delete(formatID);

Argumento Tipo Descripción
formatID ID de formato de vértice The vertex format to delete

Devuelve:

N/A

Ejemplo:

vertex_format_delete(myFormat);

El código anterior eliminará de la memoria el formato vertex creado en la variable "myFormat".
