vertex_format_add_normal

Indique a GameMaker que acepte los datos de las normales de superficie (nx, ny y nz) como parte del nuevo formato vertex que se está creando.

Sintaxis:

vertex_format_add_normal();

Devuelve:

Ejemplo:

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_normal();
my_format = vertex_format_end();

El código anterior creará un nuevo formato vertex con sólo los valores de la textura y la normal de la superficie y luego almacenará el id del formato en la variable "mi_formato".
