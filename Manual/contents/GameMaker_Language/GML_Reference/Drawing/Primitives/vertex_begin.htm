inicio_de_vértices

Con esta función se inicia la definición de una primitiva personalizada. Se asigna un buffer para escribir la primitiva, y el formato vertex a utilizar (previamente definido mediante las funciones de formato vertex ). A continuación, se definen los puntos necesarios para cada vertex de la primitiva antes de llamar a vertex_end() para finalizar la creación de la primitiva.

Sintaxis:

vertex_begin(buffer, format);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to be written to.
format ID de formato de vértice The vertex format to use.

Devuelve:

N/A

Ejemplo:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

El código anterior definirá un nuevo formato vertex, creará un nuevo buffer e iniciará el proceso de definición de una nueva primitiva.
