vertex_get_buffer_size

Con esta función se puede obtener el tamaño del vertex buffer dado en bytes.

Sintaxis:

vertex_get_buffer_size(buffer);

Argumento Tipo Descripción
buffer Vértice Buffer ID The buffer to get the size of.

Devuelve:

Real

Ejemplo:

bufferBytes = vertex_get_buffer_size(buff);

El código anterior obtendrá el número de bytes utilizados por el vertex buffer dado y almacenará el valor en la variable "bufferBytes".
