dibujar_vértice_color

Esta función define la posición de un vertex para una primitiva, con su propio color y ajuste alfa. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar y del orden con el que se añadan los vértices a la misma (ver draw_primitive_begin() para más información) y los vértices con diferentes colores y alfas se mezclarán suavemente de uno a otro. Para terminar y dibujar la primitiva debes llamar a draw_primitive_end().

Sintaxis:

draw_vertex_colour(x, y, col, alpha)

ArgumentoTipo Descripción
x The x coordinate of the vertex.
y The y coordinate of the vertex.
col The colour to draw this vertex with.
alpha The alpha to draw this vertex with (0-1).

Devuelve:

Ejemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

El código anterior dibujará un triángulo semitransparente con cada vertex coloreado de un color diferente.
