vertex_create_buffer

Con esta función puede crear un nuevo vertex buffer. Este es un crecimiento especial buffer creado por GameMaker que está preformateado para su uso cuando se construyen primitivas (para su uso con shaders, por ejemplo). La función devolverá un índice para el buffer que deberá ser utilizado en todas las llamadas posteriores al mismo.

Cuando se utiliza un vertex buffer creado con esta función, simplemente se llama a vertex_begin() para empezar a asignarle los datos de vertex para empezar a definir su primitiva personalizada, que luego se mantendrá en el buffer listo para ser enviado al shader. El buffer puede ser reutilizado cuando sea necesario (a menos que haya utilizado la función vertex_freeze() ), con cada llamada de vertex_begin() borrando los datos anteriores de buffer, listos para aceptar los nuevos datos.

Sintaxis:

vertex_create_buffer();

Devuelve:

Vertex Buffer ID

Ejemplo:

v_buff = vertex_create_buffer();

El código anterior creará un nuevo vertex buffer y almacenará su mango en la variable "v_buff".
