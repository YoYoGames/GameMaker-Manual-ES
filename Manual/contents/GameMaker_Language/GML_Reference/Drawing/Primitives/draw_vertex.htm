dibujar_vértice

Esta función define la posición de un vertex para una primitiva. El aspecto final de la primitiva dependerá del tipo de primitiva elegida para dibujar y del orden con el que se añadan los vértices a la misma. Ver draw_primitive_begin() para más información. Para finalizar y dibujar la primitiva debes llamar a draw_primitive_end().

Sintaxis:

draw_vertex(x, y)

ArgumentoTipo Descripción
x The x coordinate of the vertex.
y The y coordinate of the vertex.

Devuelve:

Ejemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

El código anterior dibujará una primitiva triangular simple.
