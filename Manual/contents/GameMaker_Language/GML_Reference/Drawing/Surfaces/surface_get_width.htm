superficie_obtener_ancho

Esta función simplemente devuelve el ancho, en píxeles, de la superficie indexada. Hay que tener en cuenta que si se llama a esta función para comprobar la application_surface inmediatamente después de haber cambiado su tamaño usando surface_resize() no devolverá el nuevo valor ya que el cambio necesita uno o dos pasos para ser procesado completamente. Después de esperar un paso debería devolver el nuevo tamaño correctamente.

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de texturas. Debe comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarlas directamente.

Sintaxis:

surface_get_width(surface_id);

ArgumentoTipo Descripción
surface_id The ID of the surface to get the width of.

Devuelve:

Ejemplo:

sw = surface_get_width(surf);

El código anterior almacenará el ancho de la superficie indexada en la variable "surf" en la variable "sw".
