superficie_obtener_objetivo

Con esta función puede obtener el ID de la superficie actual a la que se dirige el dibujo. Si ha creado esa superficie con surface_create()entonces este será el número entero positivo que es el ID único para esa superficie

Si no se establece ninguna superficie personalizada, pero la application_surface por defecto sigue activada, la función suele devolver 0. Sin embargo, si has estado manipulando la application_surface previamente, puede que quieras comprobar que el ID que obtuviste no coincide con el valor actual de la variable application_surface, en caso de que ésta ya no sea 0.

Si no se establece ninguna superficie personalizada y también se ha desactivado el application_surface por defecto, la función devolverá -1.

Sintaxis:

surface_get_target();

Devuelve:

Surface ID or -1 (if no target surface is set)

Ejemplo:

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

El código anterior comprobará si el objetivo de renderizado actual es una superficie o no, y si lo es, restablece el objetivo y luego dibuja una superficie.
