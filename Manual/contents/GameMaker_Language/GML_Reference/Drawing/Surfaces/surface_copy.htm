copia_de_superficie

Esta función simplemente toma la imagen de una superficie y la copia en otra en la posición local especificada dentro de esa superficie (donde la posición (0,0) es la esquina superior izquierda de la superficie de destino). Si la superficie de destino ya tiene información, ésta será sobrescrita por la copia, y la función no cambia la superficie de origen de ninguna manera.

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de texturas. Debe comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarlas directamente.

Sintaxis:

surface_copy(destination, x, y, source);

Argumento Tipo Descripción
destination Identificación de la superficie The ID of the surface to copy the other surface to.
x Real The x position to copy to.
y Real The y position to copy to.
source Identificación de la superficie The ID of surface to be copied.

Devuelve:

N/A

Ejemplo:

if view_current == 0
{
    surface_copy(surf, 0, 0, temp_surf);
}
else
{
    draw_surface(surf, 0, 0);
}

El código anterior comprobará la vista actual que se está dibujando y si es la vista[0] copia la superficie indexada en la variable "temp_surf" en la superficie indexada en la variable "surf". Si la vista actual es otra que la vista[0] la superficie "surf" se dibuja en la pantalla.
