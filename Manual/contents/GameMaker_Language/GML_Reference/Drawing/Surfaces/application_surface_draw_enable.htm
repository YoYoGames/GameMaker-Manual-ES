application_surface_draw_enable

Puede utilizar esta función para activar o desactivar el dibujo automático de la superficie de la aplicación. Por defecto está activada, pero en muchos casos querrá hacerse cargo de cuándo y cómo se dibuja la superficie (cuando se utiliza shaders por ejemplo), en cuyo caso utilizarás esta función para ponerla en falso y así poder dibujarla tú mismo cuando y como lo requieras. Es importante tener en cuenta que cuando switch desactiva el dibujo automático y dibuja la superficie de la aplicación usted mismo, puede ver ciertos problemas con el componente alfa de sprites y la propia superficie. Esto se debe a que GameMaker dibujará la superficie de la aplicación sin mezcla al fa cuando el dibujo automático está activado. Si usted switch apagar el dibujo automático entonces usted necesita manejar esto usted mismo, utilizando el siguiente código (por ejemplo):

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Sintaxis:

application_surface_draw_enable(flag);

ArgumentoTipo Descripción
flag The flag will be either true (activado, el valor por defecto) o false (desactivado).

Devuelve:

Ejemplo:

application_surface_draw_enable(false);

El código anterior switch desactivará el dibujo automático de la superficie de la aplicación.
