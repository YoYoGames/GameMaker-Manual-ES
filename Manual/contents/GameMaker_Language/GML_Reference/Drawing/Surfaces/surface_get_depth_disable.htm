superficie_obtener_profundidad_desactivar

Esta función comprueba si la generación automática de profundidad buffer para las superficies está activada. Normalmente todas las superficies tienen profundidad buffers así que si dibujas 3D objects en ellas entonces las ordenará correctamente por profundidad, sin embargo asignar profundidad buffers esencialmente duplica el tamaño de las superficies, lo que podría ser una sobrecarga excesiva e innecesaria especialmente si tu juego es muy intensivo en memoria o predominantemente 2D. En estos casos puedes comprobarlo usando esta función y desactivar la profundidad buffer para las superficies si es necesario usando la función surface_depth_disable().

Sintaxis:

surface_get_depth_disable();

Devuelve:

Ejemplo:

if (!surface_exists(surf))
{
    if (surface_get_depth_disable() == false)
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

El código anterior comprobará si la superficie dada existe, y si no existe, entonces comprueba el estado actual de la profundidad de la superficie buffer y si está activada, la desactivará en su lugar, antes de crear finalmente la superficie.
