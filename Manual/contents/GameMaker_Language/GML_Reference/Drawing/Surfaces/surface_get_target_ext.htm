superficie_obtener_objetivo_ext

Esta función recuperará el ID de la superficie asignada a uno de los 4 objetivos de renderización disponibles para las superficies. Se suministra el índice del objetivo de renderizado a comprobar, y la función devolverá -1 si no hay ninguna superficie asignada, o un valor entero >= 0, que representa el ID de la superficie asignada (como devuelve la función surface_create()).

Sintaxis:

surface_get_target_ext(index);

ArgumentoTipo Descripción
index The render target index to check (from 0 to 3).

Devuelve:

Surface ID or -1 (if no target surface is set)

Ejemplo:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

El código anterior comprobará primero si el objetivo de renderización 0 de shader se ha establecido en una superficie, y si no, se le asigna una.
