surface_set_target_ext

Esta función se utiliza con las funciones de sombreado y establece el MRT (0 - 3) para el nivel nativo shaders (DX9, DX11, OpenGL).

NOTA: Cuando se trabaja con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que están almacenadas en la memoria de texturas. Debe comprobar SIEMPRE que una superficie existe utilizando surface_exists() antes de referenciarlas directamente.

Sintaxis:

surface_set_target_ext(index, surface_id);

ArgumentoTipo Descripción
index The render target index to use (from 0 to 3).
surface_id The ID of the surface to use.

Devuelve:

Ejemplo:

surface_set_target_ext(0, surf);

El código anterior establecerá el objetivo de renderizado 0 al ID de la superficie indexada en la variable "surf".
