surface_depth_disable

Esta función desactiva la generación automática de profundidad buffer para todas las superficies que se creen después de haberla desactivado. Normalmente todas las superficies tienen profundidad buffers por lo que si dibujas en ellas 3D objects entonces las ordenará adecuadamente por profundidad para para mantener la paridad funcional entre el renderizado a las superficies y el renderizado a la parte posterior buffer, sin embargo asignar la profundidad buffers esencialmente duplica la cantidad de memoria requerida para las superficies, lo que podría ser una sobrecarga excesiva e innecesaria especialmente si tu juego es muy intensivo en memoria, y para los juegos 2D generalmente no son necesarios.

Para utilizar esta función, deberá llamarla antes de crear una superficie, y podrá volver a activarla después (si es necesario). Todas las superficies creadas cuando la profundidad buffer está deshabilitada ya no utilizarán una profundidad buffer cuando se dibujen en ella, lo que hace que Esto resulta en que las cosas que se dibujan en la superficie tienen paridad de profundidad. En la imagen anterior, estamos dibujando un rectángulo y un triángulo utilizando un vertex buffer en 3D. Las coordenadas de coordenadas del triángulo rojo lo colocan detrás del rectángulo verde, pero el código para dibujar el triángulo se llama después del código para el rectángulo, y cuando la profundidad buffer está activada, el triángulo se sigue dibujando detrás del rectángulo ya que se respeta la profundidad 3D. Sin embargo, cuando deshabilitamos la profundidad buffer para la superficie, sólo se respeta el orden del código y entonces el triángulo se dibuja sobre el rectángulo.

NOTA: Para preservar adecuadamente la profundidad en 3D cuando se utilizan superficies, no sólo debe tener activada la profundidad buffer, sino que también debe tener activada la escritura y la prueba z. Consulte las funciones gpu_set_ztestenable() y gpu_set_zwriteenable() para más información.

NOTA: Si tu juego es 100% 2D o sabes que no necesitas una profundidad de superficie buffer, entonces debería estar bien simplemente llamar a esta función una vez al inicio del juego para desactivar el buffer.

Sintaxis:

surface_depth_disable(disable);

ArgumentoTipo Descripción
disable If set to true el búfer de profundidad se desactivará para las superficies creadas.

Devuelve:

Ejemplo:

var _depth_setting = surface_get_depth_disable();
surface_depth_disable(false);
surface_with_depth = surface_create(room_width, room_height);
surface_depth_disable(true);
surface_without_depth = surface_create(room_width, room_height);
surface_depth_disable(_depth_setting);

El código anterior almacena la configuración actual de la profundidad buffer en una variable y luego crea dos superficies, una con el búfer de profundidad activado y otra sin él, antes de restaurar la configuración original de nuevo.
