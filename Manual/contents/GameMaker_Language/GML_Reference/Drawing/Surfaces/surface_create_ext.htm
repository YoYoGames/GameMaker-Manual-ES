surface_create_ext

Esta función le permite adjuntar una superficie a un elemento de lienzo que ya existe en su página web, lo que significa que puede dividir eficazmente partes de su juego para ser dibujadas en varios lugares diferentes dentro de la página. Para ello, debes haber definido correctamente el elemento lienzo dentro de la página *.html de tu juego utilizando los tamaños y nombres correctos que corresponden a las superficies que deseas crear. Así, tendrías tu lienzo "principal", y luego tus elementos de lienzo de superficie secundaria, que serán asignados usando esta función a las superficies correctas. La siguiente imagen es un ejemplo de cómo se configuraría una página con tres elementos de lienzo:

El diseño de la página puede ser complicado, especialmente si deseas que todos los elementos se alineen correctamente, pero una vez que la dura tarea de crear el diseño se ha completado, puedes añadir este archivo html como el archivo de página por defecto para el juego utilizando la pestaña HTML5 de las Opciones de Juego. Lo siguiente que debes hacer es configurar tu room y las vistas, ya que cada superficie tendrá que estar asociada a una vista específica para "capturar" las imágenes del juego (ver la variable view view_surface_id[0...7]). La imagen de abajo muestra cómo el juego room para el ejemplo del lienzo anterior sería establecido:

Por último, se utilizaría esta función para crear las superficies, siendo el nombre el mismo que el utilizado para los elementos del lienzo y el tamaño correspondiente al tamaño de ese mismo lienzo. La función devolverá el índice de la superficie, que deberá almacenarse en una variable para futuras llamadas a la función. Cuando la superficie se crea por primera vez, puede contener "ruido", ya que básicamente es sólo un área de memoria que se pone a un lado para el propósito (y que la memoria todavía puede contener información), por lo que es posible que desee borrar la superficie antes de su uso con una función como draw_clear_alpha().

NOTA: Esta función sólo está disponible para su uso con el módulo HTML5.

Sintaxis:

surface_create_ext(name, w, h);

ArgumentoTipo Descripción
name The name of the canvas element to link the surface to.
w The width of the surface to be created.
h The height of the surface to be created.

Devuelve:

Surface ID

Ejemplo:

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

El código anterior crea dos superficies de diferentes tamaños, asignando cada una a un elemento del lienzo diferente, y luego esas superficies se asignan a dos vistas para que se capture la parte correcta de la room.
