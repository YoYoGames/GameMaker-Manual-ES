is_infinity

Esta función devuelve si una variable dada es infinity (un número infinito) o no, devolviendo true si lo es, y false si no lo es.

Sintaxis:

is_infinity(n);

Argumento Tipo Descripción
n Variable The argument to check.

Devuelve:

Boolean

Ejemplo:

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

El código anterior comprueba la variable global "value" para ver si es infinita o no y muestra un mensaje de depuración si lo es.
