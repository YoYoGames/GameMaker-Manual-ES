array_resize

Con esta función puede redimensionar una dimensión de matriz existente a un nuevo tamaño. Se suministra el array a redimensionar así como el nuevo número de índices para el array, y la función redimensionará ese array. Tenga en cuenta que esta función está diseñada para redimensionar un array a una longitud menor, ya que puede redimensionar hacia arriba simplemente estableciendo un nuevo índice en el array. Dicho esto, si la utilizas para aumentar el tamaño de un array, los nuevos índices se establecerán en el valor por defecto de 0.

Esta función también se puede utilizar para matrices multidimensionales, siempre que se especifique qué dimensión se quiere redimensionar al suministrar el índice de la matriz, siguiendo este patrón:

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

El código indica que sólo se redimensiona la primera ranura de la segunda dimensión, ya que las ranuras de cualquier dimensión de la matriz no están interconectadas y pueden tener diferentes tamaños; por ejemplo:

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

En el código anterior, la longitud de la segunda dimensión del array es 10 en la primera ranura, pero 22 en la segunda.

Sintaxis:

array_resize(array_index, new_size);

ArgumentoTipo Descripción
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

Devuelve:

Ejemplo:

if array_length(a) > 10
{
    array_resize(a, 10);
}

El código anterior comprueba la longitud de un array y si tiene más de 10 índices, se redimensiona.
