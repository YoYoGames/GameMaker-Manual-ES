is_int32

Esta función devuelve si una variable dada es un entero de 32 bits o no. En algunos casos se quiere comprobar y ver qué data type contiene una variable en GameMaker y es entonces cuando se utilizaría esta función. Devuelve true o false dependiendo de si el valor es un int 32 o no.

Sintaxis:

is_int32(n);

Argumento Tipo Descripción
n Variable The argument to check.

Devuelve:

Boolean

Ejemplo:

if !is_int32(val)
{
    show_debug_message("Not a 32 bit integer!");
}

El código anterior comprueba la variable "val" para ver si contiene un int32 y si no lo es entonces muestra un mensaje en la consola de depuración.
