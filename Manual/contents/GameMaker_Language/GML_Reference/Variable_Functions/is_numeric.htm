es_numérico

Esta función devuelve si una variable dada es un valor numérico (real, int32, int64 o booleano) o no. En algunos casos se quiere comprobar si una variable en GameMaker tiene algún valor numérico, y es entonces cuando se utilizaría esta función. La función devolverá true si la entrada dada es numérica, y false en caso contrario.

Sintaxis:

is_numeric(n);

Argumento Tipo Descripción
n Variable The input to check.

Devuelve:

Boolean

Ejemplo:

if is_numeric(val)
{
    current_val += val;
}

El código anterior comprueba la variable "val" para ver si es un valor numérico y si lo es lo añade a la variable dada.
