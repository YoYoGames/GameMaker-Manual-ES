variable_global_get

Con esta función puede obtener el valor de una variable global con nombre. Usted proporciona el nombre de la variable global para obtener el valor de una cadena (ver el código de ejemplo a continuación) y la función devolverá el valor de la variable global o undefined si la variable no existe.

¡IMPORTANTE! Si la variable global que está obteniendo no existe entonces la función devolverá la palabra clave undefined y puede obtener errores que detendrán el funcionamiento del juego, así que si tiene dudas utilice la función variable_global_exists primero.

Sintaxis:

variable_global_get(name);

ArgumentoTipo Descripción
name The name of the global variable to get (as a string)

Devuelve:

Variable (any data type) or undefined (if the named variable does not exist)

Ejemplo:

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

El código anterior comprobará si existe una variable global y, si lo hace, se mostrará en la consola.
