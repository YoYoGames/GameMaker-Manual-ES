array_get

Con esta función puedes recuperar el valor de un índice en un array. La función requiere que proporciones una variable que contenga el array y el índice para obtener el valor dentro de ese array. Tenga en cuenta que si el índice de la matriz está fuera de los límites, el juego se bloqueará con un error. Esta función también se puede utilizar para matrices multidimensionales, siempre y cuando se especifique qué dimensión se quiere obtener cuando se proporciona el índice de la matriz, siguiendo este patrón:

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

Sintaxis:

array_get(variable, index);

Argumento Tipo Descripción
variable Matriz The variable that holds the array.
index Real The index of the array element to get the value from.

Devuelve:

Variable (Any valid data type that an array can hold)

Ejemplo:

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

El código anterior mostrará en la consola los 10 primeros elementos de lo dado.
