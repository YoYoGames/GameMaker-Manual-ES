array_create

Con esta función puedes crear un array 1D de un tamaño determinado. Le dices a la función la longitud del array a crear, y te devolverá el "handle" del array que puedes asignar a una variable. Las matrices creadas de esta manera tendrán cada entrada inicializada a 0 a menos que se especifique un valor de inicialización (opcional). Si se proporciona el valor extra para inicializar el array, entonces todos los índices dentro del nuevo array se establecerán a ese valor en lugar de 0, pero tenga en cuenta que la función tendrá una mayor sobrecarga de rendimiento en este caso.

Sintaxis:

array_create(size, [value]);

Argumento Tipo Descripción
size Real The size of the array to create.
value Variable OPTIONAL El valor a utilizar para inicializar todos los índices del array.

Devuelve:

Array

Ejemplo:

instance_array = array_create(100, noone);

El código anterior creará un nuevo array de 100 entradas, cada una de ellas con la palabra clave noone, y lo asignará a la variable "instance_array".
