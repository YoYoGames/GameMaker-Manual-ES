is_string

Esta función devuelve si una variable dada es un número string o no. En algunos casos se quiere comprobar si una variable en GameMaker contiene un string y no un real y es entonces cuando se utilizaría esta función. No devuelve el string sino true o false, por lo que un valor de, por ejemplo, "pez" para n devolverá true, pero un valor de 200 para n devolverá false.

Sintaxis:

is_string(n);

Argumento Tipo Descripción
n Variable The argument to check.

Devuelve:

Boolean

Ejemplo:

if is_string(val)
{
    name = "Player: " + val;
}

El código anterior comprueba la variable "val" para ver si contiene un string y si lo hace lo añade en otro string que luego se asigna a la variable "name".
