variable_instance_exists

Con esta función puede comprobar si una variable de ámbito de instancia existe o no. Debe suministrar el valor único del ID de la instancia (que puede encontrarse en las Propiedades de la Instancia en el editor room, o que se devuelve cuando se llama a la función instance_create_layer()) así como el nombre de la variable a comprobar como una cadena (véase el código de ejemplo más abajo). La función devolverá true si existe una variable con el nombre dado para la instancia y false en caso contrario.

Sintaxis:

variable_instance_exists(instance_id, name);

Argumento Tipo Descripción
instance_id ID de la instancia The unique ID value of the instance to check
name Cadena The name of the variable to check for

Devuelve:

Boolean

Ejemplo:

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

El código anterior comprobará si la variable llamada "escudos" existe en la instancia que ejecuta el código y, si no existe, se crea y se inicializa a 0.
