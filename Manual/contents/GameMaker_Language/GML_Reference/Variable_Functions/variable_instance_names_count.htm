variable_instance_names_count

Con esta función puede encontrar el número total de variables definidas para una instancia. Se proporciona el ID de la instancia a comprobar, y la función devolverá un valor entero para el número de variables encontradas, o (si no existe ninguna instancia del ID dado) -1.

Sintaxis:

variable_instance_names_count(instance_id);

Argumento Tipo Descripción
instance_id ID de la instancia The unique ID value of the instance to check.

Devuelve:

Real

Ejemplo:

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

El código anterior recuperará el número de variables en la instancia dada y mostrará un mensaje de depuración en la salida de la consola con ese valor.
