array_push

Con esta función se puede empujar un valor (o valores) al final de un array sin tener que conocer la longitud del mismo. La función requiere que proporcione una variable que contenga la matriz, así como al menos un valor para empujar, aunque puede proporcionar opcionalmente otros argumentos y todos ellos serán empujados en la matriz en orden consecutivo.

Sintaxis:

array_push(variable, value, [value], [value], [etc...]);

Argumento Tipo Descripción
variable Matriz The variable that holds the array.
value Variable The value to push onto the end of the array
[value], [value], [etc...] Variable OPTIONAL Otros valores que se introducen en la matriz

Devuelve:

N/A

Ejemplo:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

El código anterior empujará cuatro valores al final de la matriz dada.
