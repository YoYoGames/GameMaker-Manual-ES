method_get_index

Con esta función se puede recuperar el índice de Script para el script donde se definió el método. Si el método no fue definido en un script entonces la función devolverá -1, de lo contrario devolverá el valor del índice para el script.

Sintaxis:

method_get_index(method);

Argumento Tipo Descripción
method Método The method variable to check

Devuelve:

Script Asset or Real

Ejemplo:

var _index = method_get_index(light_setup);
if _index != -1
{
    show_debug_message(script_get_name(_index));
}

El código anterior comprobará el índice script para el método "light_setup" y si no es -1 mostrará el nombre script en la consola.
