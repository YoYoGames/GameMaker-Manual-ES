es_estructura

Esta función comprueba si el valor suministrado es un struct. Devuelve true si lo es, en caso contrario devuelve false.

Tenga en cuenta que las variables de los métodos también devolverán true, y las instancias de los objetos devolverán false.

Sintaxis:

is_struct(val);

Argumento Tipo Descripción
val Variable The value to check.

Devuelve:

Boolean

Ejemplo:

if is_struct(a)
{
    delete(a);
}

El código anterior comprueba una variable para ver si es un struct, y si la función devuelve true, se elimina el struct.
