Funciones variables

Las funciones de esta página están diseñadas para tratar con todas las diferentes variables y tipos de variables cuando se utiliza el GameMaker Language en sus juegos. Para más información sobre las variables, consulta la página de Variables y Ámbito de las Variables.

Están disponibles las siguientes funciones variables:

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variable_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

Hay algunas funciones asociadas al uso de Arrays también. Estas están diseñadas para darte flexibilidad en tu código, y te permitirán crear arrays más funcionales y dinámicos cuando hagas tus juegos. Estas funciones son:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(obsoleto)
  • array_length_2d(obsoleto)
  • array_height_2d(obsoleto)

Las siguientes funciones están relacionadas con el uso de las Variables del Método:

  • method
  • method_get_self
  • method_get_index

Por último, también dispone de las siguientes funciones para determinar el tipo de variable (para más información sobre data types, consulte aquí, y vea aquí para las tablas de tipos):

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
