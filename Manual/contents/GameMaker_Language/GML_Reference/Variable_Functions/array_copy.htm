array_copy

Con esta función puedes copiar todo o parte de un array en otro array en cualquier posición. Para ello, es necesario indicar las matrices de origen y de destino (ambas deben haber sido creadas previamente), así como una posición dentro de la matriz de origen para copiar y una posición dentro de la matriz de destino para copiar. Por último, hay que especificar la longitud del array (o la longitud de la parte que se desea) a copiar. Si los datos que se copian superan la longitud de la matriz de destino, ésta se ampliará para aceptar los datos.

Esta función también se puede utilizar para matrices multidimensionales, siempre y cuando se especifique qué dimensión se quiere copiar al suministrar el índice de la matriz, siguiendo este patrón:

// Copy to the first dimension of an array
// from the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy to the third dimension of an array
// from the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc.

Sintaxis:

array_copy(dest, dest_index, src, src_index, length);

Argumento Tipo Descripción
dest Matriz The ID of the array to copy to.
dest_index Real The index within the array to copy to.
src Matriz The ID of the array to copy from.
src_index Real The index with the array to start copying from.
length Real The length (number of array indices) to copy.

Devuelve:

N/A

Ejemplo:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}
