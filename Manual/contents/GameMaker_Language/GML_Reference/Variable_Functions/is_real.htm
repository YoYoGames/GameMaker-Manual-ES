es_real

Esta función devuelve si una variable dada es un número real (simple, doble o entero) o no. En algunos casos se quiere comprobar si una variable contiene un número real, y es entonces cuando se utilizaría esta función. No devuelve el número real sino true o false, por lo que un valor de, por ejemplo, "pez" devolvería false, sin embargo un valor de 200 devolvería true.

NOTA: Esta función devolverá false para cualquier enum valores ya que están almacenados como int64s. La función correcta para comprobar que data type es is_int64().

Sintaxis:

is_real(n);

Argumento Tipo Descripción
n Variable The argument to check.

Devuelve:

Boolean

Ejemplo:

if is_real(val)
{
    score += val;
}

El código anterior comprueba la variable "val" para ver si es un número real y, si lo es, lo añade a la puntuación.
