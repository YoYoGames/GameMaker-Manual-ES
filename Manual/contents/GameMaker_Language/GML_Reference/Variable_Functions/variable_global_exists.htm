variable_global_exists

Con esta función puede comprobar si una variable de ámbito global existe o no. Se suministra el nombre de la variable global a comprobar como una cadena (véase el código de ejemplo a continuación) y la función devolverá true si existe una variable global con el nombre dado o false en caso contrario.

Sintaxis:

variable_global_exists(name);

Argumento Tipo Descripción
name Cadena The name of the global variable to check for (as a string)

Devuelve:

Boolean

Ejemplo:

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

El código anterior comprobará si la variable global llamada "enemy_num" existe y si no existe se crea y se le asigna un valor.
