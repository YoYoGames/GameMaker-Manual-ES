de

Esta función puede utilizarse para obtener el nombre de la función constructora que se utilizó para crear un struct. El propio struct debería haberse creado utilizando la función new junto con el propio constructor. Se suministra la variable con la referencia struct a comprobar y la función devolverá un string con el nombre del constructor o undefined.

Tenga en cuenta que si pasa a la función un literal de struct (es decir: un struct que fue creado sin usar una función constructora) entonces simplemente devolverá el "struct" string .

Esta función también puede utilizarse para comprobar si una referencia struct es una referencia débil o no, en cuyo caso la función devolverá el string "weakref" en lugar del nombre de la función que creó el struct. Para más información, véase la función weak_ref_create().

El nombre del constructor devuelto incluirá texto adicional si la función del constructor no fue creada en un script. Si fue creada dentro de un object, su nombre incluirá el object y el evento donde fue creada. Por ejemplo, un constructor llamado test_constructor creado en el evento Create de Object1 será devuelto como "test_constructor_gml_Object_Object1_Create_0". Para evitar esto, crea tus constructores en un evento script.

Sintaxis:

instanceof(struct_id);

Argumento Tipo Descripción
struct Struct The struct reference to use.

Devuelve:

String or undefined

Ejemplo ampliado:

En este ejemplo debemos definir primero la función que se utilizará como constructor de nuestro struct. La siguiente función está definida en un activo de script:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Esta función puede utilizarse junto con el operador new para crear un struct y rellenarlo con las variables establecidas con los valores de los argumentos utilizados en la función:

mystruct = new init_struct(10, 100, "Hello World");

Entonces podemos obtener el nombre de la función constructora que se utilizó así:

var _name = instanceof(mystruct);
if is_string(_name)
{
    show_debug_message(_name);
}

Esto imprimirá el string "init_struct" en el registro de salida, que es el nombre de la función constructora tal como se define en su script.
