longitud_de_arreglo

Con esta función puede obtener la longitud (número de entradas) de una dimensión de matriz. Se suministra el valor del índice del array y la función devolverá un valor entero que representa el número de entradas que contiene el array. Esta función también se puede utilizar para matrices multidimensionales, siempre y cuando se especifique de qué dimensión se quiere obtener la longitud al suministrar el índice de la matriz, siguiendo este patrón:

// Returns the first dimension of the array
val = array_length(my_array);

// Returns the second dimension of the array
val = array_length(my_array[0]);

// Returns the third dimension of the array
val = array_length(my_array[0][0]);

// etc...

Sintaxis:

array_length(array_index);

ArgumentoTipo Descripción
array_index The index of the array to check.

Devuelve:

Ejemplo:

for (var i = 0; i < array_length(a); ++i)
{
    a[i] = -1;
}

El código anterior loop a través de una matriz y establecer cada entrada a -1.
