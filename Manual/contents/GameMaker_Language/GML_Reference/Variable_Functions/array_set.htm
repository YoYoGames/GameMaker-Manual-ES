array_set

Con esta función puedes establecer el valor de un índice en un array a un valor. La función requiere que proporcione una variable que contenga el array, así como el índice a establecer y el valor a establecer. Esta función también se puede utilizar para matrices multidimensionales, siempre que se especifique qué dimensión se quiere establecer cuando se proporciona el índice de la matriz, siguiendo este patrón:

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

Sintaxis:

array_set(variable, index, value);

Argumento Tipo Descripción
variable Matriz The variable that holds the array.
index Real The index of the array element to set the value for.
value Variable The value to set.

Devuelve:

N/A

Ejemplo:

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

El código anterior establecerá los 10 primeros elementos de la matriz dada con un valor.
