is_method

Esta función puede utilizarse para comprobar si una variable es una variable de método (devolverá true) o no (en cuyo caso devolverá false).

Sintaxis:

is_method(n);

Argumento Tipo Descripción
n Variable The variable to check.

Devuelve:

Boolean

Ejemplo:

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

El código anterior comprueba una variable para ver si es un método, y si la función devuelve true, entonces se emite un mensaje de depuración en la consola.
