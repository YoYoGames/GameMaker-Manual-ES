int64

Esta función intentará convertir un valor dado en un entero de 64 bits con signo, donde el valor debe ser un real, un string, un int64, un int32, o un ptr. Cualquier otra cosa hará que el juego se bloquee con un mensaje de error. Puedes comprobar si una variable contiene un int64 utilizando la función is_int64().

Sintaxis:

int64(val);

Argumento Tipo Descripción
val Real, Cadena, o Puntero The value to convert.

Devuelve:

int64 (signed 64-bit integer)

Ejemplo:

steam_handle = int64(global.fileReadString);

El código anterior convierte el valor de la variable global en un entero de 64 bits.
