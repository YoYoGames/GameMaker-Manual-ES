struct_set

Con esta función se puede establecer el valor de una variable dada en un struct. Se suministra la referencia struct así como el nombre de la variable a establecer el valor como una cadena (ver el código de ejemplo más abajo), y finalmente el valor a establecer (puede ser cualquier tipo de dato válido). Si la variable no existe todavía en struct, se creará y se le asignará el valor.

 

Sintaxis:

struct_set(struct, name, val)

Argumento Tipo Descripción
struct Struct The struct reference to set
name String The name of the variable to set (as a string)
val Any The value to set the variable to

 

Devuelve:

N/A

 

Ejemplo:

if !struct_exists(mystruct, "shields")
{
    struct_set(mystruct, "shields", 0);
}

El código anterior comprobará si la variable dada existe en el struct dado y si no existe entonces se crea y se pone a 0.
