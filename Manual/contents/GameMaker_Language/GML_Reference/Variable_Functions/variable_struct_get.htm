variable_struct_get

Con esta función se puede obtener el valor de una variable con nombre dentro de un struct. Se suministra la referencia struct así como el nombre de la variable para obtener el valor en forma de cadena (véase el código de ejemplo a continuación). La función devolverá el valor de la variable o undefined si la variable nombrada no existe.

¡IMPORTANTE! Si la variable que está obteniendo no existe, la función devolverá la palabra clave undefined y puede obtener errores que detendrán el funcionamiento del juego, así que si tiene dudas utilice la función variable_struct_exists() primero.

Sintaxis:

variable_struct_get(struct, name);

ArgumentoTipo Descripción
struct The struct reference to use
name The name of the variable to get (as a string)

Devuelve:

Variable (any data type) or undefined (if the named variable does not exist)

Ejemplo:

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

El código anterior comprobará si existe una variable en la dirección struct y, si existe, el valor que contiene se recupera y se almacena en una variable local. Si no existe, la variable local se pone a -1.
