array_pop

Esta función eliminará el último elemento de la matriz dada y devolverá su valor. Si el array está vacío, se devuelve undefined.

Sintaxis:

array_pop(array);

ArgumentoTipo Descripción
array The array to remove the last value from.

Devuelve:

Variable or undefined

Ejemplo:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

El código anterior sacará el último valor de la matriz dada, y luego lo dibujará junto con algún texto en la pantalla. La entrada recuperada también se elimina de la matriz de puntuación.
