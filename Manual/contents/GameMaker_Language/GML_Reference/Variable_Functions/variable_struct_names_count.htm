variable_struct_names_count

Con esta función se puede encontrar el número total de variables definidas para un struct. Se suministra el ID de struct a comprobar, y la función devolverá un valor entero para el número de variables encontradas, o (si no existe ningún struct del ID dado) -1.

Sintaxis:

variable_struct_names_count(struct_id);

ArgumentoTipo Descripción
struct_id The unique ID value of the struct to check.

Devuelve:

Ejemplo:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

El código anterior recuperará el número de variables en el struct dado y mostrará un mensaje de depuración en la salida de la consola con este valor.
