array_equals

Con esta función puedes comprobar si dos matrices son iguales (equivalentes o iguales). Se dan las dos matrices a comprobar, y la función devolverá true si son iguales (o equivalentes o iguales) o false si no lo son. Tenga en cuenta que esto no es lo mismo que comprobar si dos matrices son iguales utilizando ==, que no comprobará si las dos matrices contienen valores equivalentes, sino sólo si las matrices están referenciando la misma matriz inicial. Por ejemplo:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Tenga en cuenta que hay algunas constantes que pueden no ser iguales a sí mismas, lo que puede hacer que esta función falle. He aquí un ejemplo:

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Consulte la tabla de igualdad para obtener más información.

Sintaxis:

array_equals(var1, var2);

Argumento Tipo Descripción
var1 Matriz The first array.
var2 Matriz The second array.

Devuelve:

Boolean

Ejemplo:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

El código anterior comprobará dos matrices para ver si contienen valores equivalentes, y si no lo hacen entonces el código copiará todo el contenido de la matriz "inventory_array" a la matriz "item_array".
