variable_struct_exists

Con esta función puede comprobar si una variable existe o no dentro de la dirección struct. Se suministra la referencia struct que se va a utilizar, así como el nombre de la variable que se va a comprobar en forma de cadena (véase el código de ejemplo siguiente). La función devolverá true si existe una variable con el nombre dado para el struct y false en caso contrario.

Sintaxis:

variable_struct_exists(struct, name);

Argumento Tipo Descripción
struct Struct The struct reference to check
name Cadena The name of the struct variable to check for (as a string)

Devuelve:

Boolean

Ejemplo:

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

El código anterior comprobará si la variable llamada "shields" existe en el struct dado y si no existe entonces se crea y se inicializa a 0.
