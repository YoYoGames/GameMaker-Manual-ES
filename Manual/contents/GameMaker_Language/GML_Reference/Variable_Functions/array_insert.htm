array_insert

Con esta función puede insertar un valor (o varios) en una matriz en cualquier posición. La función requiere que proporcione una variable que contenga el array, el índice (posición) en el array a insertar, así como al menos un valor a insertar, aunque opcionalmente puede proporcionar más argumentos y todos ellos se insertarán en el array en orden consecutivo desde el índice dado.

Tenga en cuenta que si el índice dado está fuera de la longitud de la matriz, los valores se añadirán en el índice dado, y cualquier ranura vacía en la matriz entre su último valor y los valores insertados se establecerá un valor por defecto de 0.

Sintaxis:

array_insert(variable, index, value, [value], [value], [etc...]);

Argumento Tipo Descripción
variable Matriz The variable that holds the array.
index Real The index (position) in the array to insert the value(s) 
value Variable The value to insert
[value], [value], [etc...] Variable OPTIONAL Otros valores a insertar en el array

Devuelve:

N/A

Ejemplo:

array_insert(score_array, score_rank, current_score);

El código anterior insertará la puntuación actual en la matriz de puntuación, en el índice definido por la variable score_rank.
