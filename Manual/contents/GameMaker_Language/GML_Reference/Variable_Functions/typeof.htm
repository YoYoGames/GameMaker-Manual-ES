tipo de

Esta función devuelve la data type de cualquier variable dada como string. Los posibles valores de retorno se enumeran en la tabla siguiente:

Cadena Descripción
"number" La variable contiene un número real (de punto flotante) - esto puede incluir NaN y infinity
"string" La variable contiene una cadena
"array" La variable hace referencia a un array
"bool" La variable contiene un booleano (true / false)
"int32" La variable contiene un entero de 32 bits
"int64" La variable contiene un entero de 64 bits
"ptr" La variable contiene un puntero
"undefined" La variable no está definida
"null" La variable contiene un valor nulo (esto no debería verse normalmente)
"vec3" La variable contiene un vector de 3 valores
"vec4" La variable contiene un vector de 4 valores
"method" La variable contiene una referencia a una función
"struct" La variable contiene una referencia struct
"ref" La variable contiene una referencia de instancia
"unknown" El valor es desconocido. Esto nunca debería verse y significa que algo ha ido mal en el nivel más básico como una sobreescritura de memoria

Tenga en cuenta que hay casos en los que esta función puede no devolver el valor correcto para un método. Considere las siguientes dos definiciones de funciones:

a = function()
{
    // something
}

function b()
{
    // Something
}

Técnicamente, ambos se consideran métodos, ya que están vinculando una función a una variable de ámbito de instancia, sin embargo, llamar a typeof() sobre la función b devolverá "número" y no "método", mientras que llamarlo sobre a devolverá "método". Esto se debe a que a los métodos creados como el de b se les asignan índices de script (que son valores enteros), ya que esta es la forma en que el compilador reconoce las funciones de script, incluso si la función no fue definida en un script.

Sintaxis:

typeof(variable);

Argumento Tipo Descripción
variable Cualquier The variable to get the data type of.

Devuelve:

String (see table above)

Ejemplo:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

El código anterior obtiene la data type contenida en la variable global dada y devuelve la string a una variable local que se utiliza para enviar un mensaje a la consola.
