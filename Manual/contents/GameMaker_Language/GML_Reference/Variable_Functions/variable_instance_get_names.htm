variable_instance_get_names

Con esta función puedes recuperar un array poblado con los nombres de las variables de instancia para una instancia, o las variables globales para un juego. Cuando pasas un valor de ID de instancia, cada entrada en el array será un string del nombre de la variable que corresponde a una variable de ámbito de instancia que ha sido creada en la instancia. Sin embargo, si pasas la palabra clave global, cada entrada del array será un string del nombre de la variable que corresponde a una variable de ámbito global.

Sintaxis:

variable_instance_get_names(instance_id/global);

Argumento Tipo Descripción
instance_id/global ID de la instancia o global The unique ID value of the instance to check or the keyword global

Devuelve:

Array (each entry is a string)

Ejemplo:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

El código anterior recuperará un array de todas las variables de ámbito de instancia para la instancia que ejecuta el bloque de código y luego las mostrará junto con sus valores en la salida de depuración.
