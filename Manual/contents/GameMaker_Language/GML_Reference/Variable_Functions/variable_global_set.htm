variable_global_set

Con esta función puede establecer el valor de una variable global determinada. Se suministra el nombre de la variable global para establecer el valor como una cadena (ver el código de ejemplo a continuación), y luego el valor a establecer (puede ser cualquier tipo de datos válido). Si la variable global no existe ya en el juego, se creará y se le asignará el valor.

Sintaxis:

variable_global_set(name, val);

Argumento Tipo Descripción
name Cadena The name of the global variable to set (as a string)
val Variable The value to set the global variable to

Devuelve:

N/A

Ejemplo:

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

El código anterior comprobará si existe una variable global y, si no existe, se crea y se pone a 0.
