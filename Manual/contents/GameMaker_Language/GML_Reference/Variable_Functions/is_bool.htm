is_bool

Esta función devuelve si una variable dada es un booleano (true ior false) o no. En algunos casos se quiere comprobar y ver si una variable en GameMaker tiene un valor booleano, y es entonces cuando se utilizaría esta función.

Sintaxis:

is_bool(n);

Argumento Tipo Descripción
n Variable The argument to check.

Devuelve:

Boolean

Ejemplo:

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

El código anterior comprueba la variable "val" para ver si es un número real y, si lo es, lo utiliza para establecer una variable global.
