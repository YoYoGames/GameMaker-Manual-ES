variable_instance_get

Con esta función puede obtener el valor de una variable con nombre dada. Debe suministrar el valor único del ID de la instancia (que puede encontrarse en las Propiedades de la Instancia en el editor room, o que se devuelve cuando se llama a la función instance_create_layer()), así como el nombre de la variable para obtener el valor de una cadena (véase el código de ejemplo a continuación). La función devolverá el valor de la variable, o undefined si la variable no existe.

¡IMPORTANTE! Si la variable que está obteniendo no existe, la función devolverá la palabra clave undefined y puede obtener errores que detendrán el funcionamiento del juego, así que si tiene dudas utilice la función variable_instance_exists primero.

Sintaxis:

variable_instance_get(instance_id, name);

Argumento Tipo Descripción
instance_id ID de la instancia The unique ID value of the instance to use
name Cadena The name of the variable to get (as a string)

Devuelve:

Variable (any data type) or undefined (if the named variable does not exist)

Ejemplo:

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

El código anterior comprobará si una variable existe y, si lo hace, el valor que contiene se recupera y se almacena en una variable local. Si no existe, la variable local se pone a -1.
