is_array

Esta función puede utilizarse para comprobar si una variable contiene una matriz (devolverá true) o no (en cuyo caso devolverá false).

Sintaxis:

is_array(n);

Argumento Tipo Descripción
n Variable The variable to check.

Devuelve:

Boolean

Ejemplo:

if is_array(a)
{
    a = -1;
}

El código anterior comprueba una variable para ver si es un array, y si la función devuelve true, el array se borra poniendo la variable a -1.
