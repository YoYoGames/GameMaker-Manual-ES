variable_struct_get_names

Con esta función se puede recuperar un array poblado con los nombres de las variables de un struct. Se pasa la referencia a struct para comprobarla, y cada entrada del array será una cadena de los nombres de las variables que contiene el struct.

Sintaxis:

variable_struct_get_names(struct);

ArgumentoTipo Descripción
struct The struct reference to check.

Devuelve:

(each entry is a string)

Ejemplo:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
}

El código anterior recuperará una matriz de los nombres de las variables para el struct dado y luego los mostrará junto con sus valores en la salida de depuración.
