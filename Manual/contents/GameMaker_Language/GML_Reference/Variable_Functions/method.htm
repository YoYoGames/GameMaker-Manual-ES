método

Con esta función se puede vincular una función (o método) existente a la instancia dada o struct, creando una nueva variable de método que puede ser utilizada posteriormente.

Se suministra el ID de la instancia que se va a utilizar (una instancia que esté activa en room, y no un índice de object ) o una referencia a struct, así como el ID de la función (o la referencia al método) que se quiere vincular. La función devolverá un nuevo método que puede llamarse desde la variable a la que se asigna (véase el ejemplo de código siguiente).

El método devuelto estará "ligado" a la instancia dada o struct, lo que significa que siempre se ejecutará en el ámbito de esa instancia/estructura.

Puede enlazar funciones incorporadas así como funciones/métodos definidos por el usuario, y también puede proporcionar undefined como argumento de instancia/estructura, lo que significa que el ámbito actual self se utilizará para la vinculación.

Sintaxis:

method(struct_ref_or_instance_id, func);

Argumento Tipo Descripción
struct_ref_or_instance_id ID delaestructura o de la instancia The unique reference or ID value of the struct or instance to use (can be self o undefined)
func Función o métododel script The ID of the function (or method reference) to use

Devuelve:

Method

Ejemplo:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if (instance_exists(_inst))
{
    enemy_func = method(_inst, enemy_ai);
}

El código anterior comprobará si existe una instancia enemiga en la posición del ratón. Si existe, el método enemy_ai se vincula a la instancia enemiga y se devuelve como una nueva variable de método "enemy_func".
