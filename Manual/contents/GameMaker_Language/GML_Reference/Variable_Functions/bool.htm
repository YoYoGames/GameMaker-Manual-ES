bool

Esta función intentará convertir un valor dado en un booleano data type, donde el valor será devuelto como true si es mayor o igual a 0.5, y false en caso contrario.

Sintaxis:

bool(n);

Argumento Tipo Descripción
n Real The value to convert.

Devuelve:

Boolean

Ejemplo:

if !is_bool(val)
{
    val = bool(val);
}

El código anterior comprueba la variable "val" para ver si es un booleano y si no lo es entonces se convierte en uno.
