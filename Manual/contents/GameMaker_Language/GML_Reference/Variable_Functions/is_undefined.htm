is_undefined

Esta función devuelve si una variable dada está definida o no. En algunos casos se quiere comprobar y ver qué data type contiene una variable en GameMaker y es entonces cuando se utilizaría esta función. Devuelve true o false dependiendo de si el valor está definido o no.

NOTA: Esta función no puede utilizarse para verificar la existencia de una variable. Utilice variable_instance_exists() o variable_global_exists() en su lugar.

Sintaxis:

is_undefined(n);

Argumento Tipo Descripción
n Variable The argument to check.

Devuelve:

Boolean

Ejemplo:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
    show_debug_message("Map entry does not exist!");
}

El código anterior comprueba la variable "val" para ver si es undefined o no y muestra un mensaje de depuración si lo es.
