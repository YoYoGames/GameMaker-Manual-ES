string_replace

Puede utilizar esta función para analizar un string en busca de una parte específica, que puede ser sustituida por el nuevo string que haya especificado.

Sintaxis:

string_replace(str, substr, newstr);

ArgumentoTipo Descripción
str The string to be copied.
substr The substring within the string to be replaced.
newstr The new substring to replace the previous one.

Devuelve:

Ejemplo:

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

Esto establecerá str2 como str1, pero con su instancia de "Tierra" reemplazada por "Mundo", resultando en str2 ser "Hola Mundo".
