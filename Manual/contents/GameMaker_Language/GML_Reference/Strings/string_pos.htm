string_pos

Esta función devolverá la posición del carácter de una instancia de una subcadena dentro de un string, buscando desde el principio del string (para buscar desde el final, utilice la función string_last_pos()). La función devolverá 0 si no se encuentra, o la posición del primer carácter de la subcadena si se encuentra. Tenga en cuenta que para el soporte de legado strings se indexan desde 1, por lo que 1 es la primera posición en el string, no 0 como podría esperarse. Un uso de esto es para filtrar palabras que puedan ser consideradas ofensivas, o para encontrar el lugar correcto para insertar algún texto en otro string.

Sintaxis:

string_pos(substr, str);

ArgumentoTipo Descripción
substr The substring to look for in the string.
str The string.

Devuelve:

Ejemplo:

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

El código anterior busca una coma en la string almacenada en la variable "text", y si la encuentra inserta la subcadena "name" en esa posición.
