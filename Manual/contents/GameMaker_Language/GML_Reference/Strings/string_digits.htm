dígitos_de_cadena

Puede utilizar esta función para analizar un determinado string y obtener cualquier número de él. Por ejemplo, digamos que tiene este texto - "Tengo 81 años". Con esta función obtendrías un retorno string de "81".

Sintaxis:

string_digits(string);

ArgumentoTipo Descripción
string The string to get the digits from.

Devuelve:

Ejemplo:

var t_str = string_digits(input_str);
age = real(t_str);

El código anterior tomará la entrada string, la despojará de todos los caracteres que no sean números y luego establecerá la variable age para mantener el valor numérico real del retorno string (así, si la entrada string fuera - por ejemplo - "Tengo 18 años", la función devolvería "18").
