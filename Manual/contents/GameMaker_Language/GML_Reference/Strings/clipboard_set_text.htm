clipboard_set_text

Esta función establece el portapapeles para que contenga el string definido. Se puede establecer en un string vacío "" para limpiar el portapapeles de texto.

Sintaxis:

clipboard_set_text(string);

Argumento Tipo Descripción
string Cadena The text (as a string) to set the clipboard to hold.

Devuelve:

N/A

Ejemplo:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

El código anterior comprueba si hay texto en el portapapeles y, si lo contiene, lo lee como string en la variable "str". Por último, el portapapeles se borra poniéndolo en un string vacío.
