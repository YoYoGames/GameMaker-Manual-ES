string_set_byte_at

Esta función establece un byte directamente en un string (basado en el formato UTF8) y devuelve una copia del string con los cambios.

NOTA Esta función es increíblemente lenta, así que considere cuidadosamente si es necesaria y dónde la utiliza.

Sintaxis:

string_set_byte_at(str, pos, byte);

ArgumentoTipo Descripción
str The string to change the byte of.
pos The position within the string (starting at 1) to change the byte of.
byte The new byte value.

Devuelve:

Ejemplo:

str = string_set_byte_at("hello", 2, 97);

El código anterior cambiaría el valor del byte de la segunda letra en el string, y así establecería la variable str para mantener "hallo".
