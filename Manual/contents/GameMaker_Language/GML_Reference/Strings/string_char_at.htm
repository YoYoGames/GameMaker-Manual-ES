string_char_at

Puede utilizar esta función para devolver un carácter específico en una posición específica dentro de un string, con el índice comenzando en 1 para el primer carácter. Si no se encuentra ningún carácter o el string es más corto que el valor del índice dado, se devuelve un string "" vacío, sin embargo si el índice dado es igual o menor que 0, entonces se devuelve el primer carácter del string.

Sintaxis:

string_char_at(str, index);

ArgumentoTipo Descripción
str The string to check.
index The position to get the character from.

Devuelve:

Ejemplo:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

De este modo, str2 se ajustará al séptimo carácter ("H" contando como el primero) en string str1, en este caso "W".
