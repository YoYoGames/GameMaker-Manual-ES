cadena_ord_at

Puede utilizar esta función para devolver un código de carácter específico en una posición concreta dentro de un string, con el índice empezando en 1 para el primer carácter. Si no se encuentra ningún carácter o el string es más corto que el valor dado al índice, se devuelve -1.

Sintaxis:

string_ord_at(str, index);

ArgumentoTipo Descripción
str The string to check.
index The position to get the character code from.

Devuelve:

Ejemplo:

str = "Hello World";
char_code = string_ord_at(str, 7);

Esto obtendrá el código del séptimo carácter (donde "H" cuenta como el primero) en string "str" y lo almacenará en la variable "char_code".
