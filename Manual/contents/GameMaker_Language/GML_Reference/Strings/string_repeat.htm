string_repeat

Esta función simplemente devuelve el mismo string repetido un número determinado de veces sobre sí mismo.

Sintaxis:

string_repeat(str, count);

ArgumentoTipo Descripción
str The string to repeat.
count The number of times to repeat the string.

Devuelve:

Ejemplo:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

El código anterior establecerá str2 como "Hello WorldHello WorldHello World".
