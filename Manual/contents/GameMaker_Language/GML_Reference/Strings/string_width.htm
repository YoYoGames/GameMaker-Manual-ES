string_width

Esta función devolverá la anchura (en píxeles) de la entrada string, teniendo en cuenta los saltos de línea que pueda tener el texto. Es muy útil para calcular las distancias entre los elementos del texto en función de la anchura total de las letras que componen la cadena tal y como se dibujaría con draw_text() utilizando el font actualmente definido.

Sintaxis:

string_width(string);

ArgumentoTipo Descripción
string The string to measure the width of.

Devuelve:

Ejemplo:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

El código anterior obtendrá la anchura del string dado y luego dibujará dos líneas de texto, utilizando la anchura devuelta de string como separador.
