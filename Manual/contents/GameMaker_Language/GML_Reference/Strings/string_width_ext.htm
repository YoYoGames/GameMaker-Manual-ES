ancho_de_cadena_ext

Esta función devolverá la anchura máxima (en píxeles) de la entrada string, teniendo en cuenta la separación entre líneas y la anchura del salto de línea (que se define como el número de píxeles que puede ocupar el string antes de que se inserte un salto de línea). Es muy útil para calcular las distancias entre los elementos de texto en función de la anchura máxima de un string que se divide en varias líneas como se dibujaría con draw_text_ext() utilizando la definición actual de font. La separación y la anchura pueden establecerse en -1 para obtener el espaciado por defecto.

Sintaxis:

string_width_ext(string, sep, w);

ArgumentoTipo Descripción
string The string to measure the width of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Devuelve:

Ejemplo:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

El código anterior obtendrá la anchura del string dado, teniendo en cuenta la separación de líneas y la anchura del salto de línea, y luego dibujará dos líneas de texto, utilizando la anchura total devuelta de string como separador.
