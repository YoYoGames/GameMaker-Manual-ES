string_byte_length

Esta función devuelve el número de bytes en un string. Debido a que se mantienen como UTF8, esto no será igual al número de caracteres en el string.

Sintaxis:

string_byte_length(string);

Argumento Tipo Descripción
string Cadena The string to measure the number of bytes of.

Devuelve:

Real

Ejemplo:

bytesize = string_byte_length("Hello World");

Esto establecería bytesize al número de bytes en "Hello World".
