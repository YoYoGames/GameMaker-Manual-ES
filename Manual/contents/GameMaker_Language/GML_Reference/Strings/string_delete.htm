string_delete

Esta función se utiliza para eliminar una parte del string dado. Se suministra el string de entrada, la posición inicial dentro de ese string para eliminar caracteres (el índice comienza en 1) y el número de caracteres a eliminar. La función devolverá un nuevo string sin esa parte.

Sintaxis:

string_delete(str, index, count);

ArgumentoTipo Descripción
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Devuelve:

String

Ejemplo:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Esto establecerá str2 como "Hello World", ya que elimina la "o" extra de "Hello" (que está presente en la posición 5).
