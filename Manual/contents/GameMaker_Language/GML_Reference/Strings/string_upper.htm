cadena_superior

Con esta función puede forzar que un string contenga sólo caracteres en mayúsculas.

NOTA Esta función sólo detecta el alfabeto inglés de 26 letras de la A a la Z.

Sintaxis:

string_upper(string);

ArgumentoTipo Descripción
string The string to convert to uppercase.

Devuelve:

Ejemplo:

str1 = "Hello World";
str2 = string_upper(str1);

El código anterior establecerá la variable str2 como "HELLO WORLD".
