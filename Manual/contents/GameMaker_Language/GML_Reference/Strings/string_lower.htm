string_lower

Con esta función puede forzar que un string contenga sólo caracteres en minúsculas.

NOTA Esta función sólo detecta el alfabeto inglés de 26 letras de la A a la Z.

Sintaxis:

string_lower(string);

ArgumentoTipo Descripción
string The string to convert to lowercase.

Devuelve:

Ejemplo:

str1 = "Hello World";
str2 = string_lower(str1);

El código anterior establecerá str2 como "hola mundo".
