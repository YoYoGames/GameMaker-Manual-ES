letras_de_cadena

Con esta función puede eliminar todos los caracteres que no se clasifican como letras. Esto es útil para evitar que la gente introduzca caracteres no deseados en una entrada de texto (como introducir "#" para forzar una nueva línea).

NOTA Esta función sólo detecta y devuelve el alfabeto inglés de 26 letras de la A a la Z (mayúsculas o minúsculas).

Sintaxis:

string_letters(string);

ArgumentoTipo Descripción
string The string to convert to letters.

Devuelve:

Ejemplo:

username = string_letters(username);

El código anterior establecerá la variable "username" para que sólo contenga los caracteres de la letra del string original (si el string original era "MacSw33ny" el string devuelto sería "MacSwny").
