chr

Esta función devuelve un string que contiene el carácter que se relaciona con el código Unicode de entrada para su visualización. Este carácter depende de la página de códigos del juego de caracteres del dibujo actual fonts y si no se establece font, utilizará la página de códigos por defecto para la máquina.

Sintaxis:

chr(val);

ArgumentoTipo Descripción
val The Unicode code value to get the string from

Devuelve:

Ejemplo:

mystring = chr(53) + chr(48);

Esto establecería mystring en "50" (un string, no un entero) ya que chr(53) es "5" y chr(48) es "0".
