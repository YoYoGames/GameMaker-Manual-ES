formato_de_cadena

Convierte un número real en un string usando su propio formato, donde puede elegir cuántos "lugares" se guardan en el string y cuántos decimales se guardan también. Ambos pueden ser muy útiles, algunos juegos prefieren mostrar una puntuación como un número fijo de dígitos, mientras que el control sobre los lugares decimales puede ser bueno para una alta precisión que los dos lugares decimales de string() no pueden proporcionar. Si el número de lugares especificados es mayor que el valor a mostrar y/o el número más los lugares decimales que se han especificado es menor que el total de lugares, entonces se añadirán espacios antes del valor para compensar la diferencia (ver el ejemplo de abajo). Se añadirán ceros a la derecha del punto decimal si el valor dado es menor que el total y el número de decimales a incluir. El formato por defecto es sin espacios extra a la izquierda, y sólo dos decimales a la derecha, por ejemplo "265,73".

Sintaxis:

string_format(val, tot, dec);

Argumento Tipo Descripción
val Real The real number to be turned into a string.
tot Real The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec Real The number of decimal places to be included.

Devuelve:

String

Ejemplo:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

Esto establecerá str1 a " 1234", str2 a "3.1415926535" y str3 a " 3.14159".
