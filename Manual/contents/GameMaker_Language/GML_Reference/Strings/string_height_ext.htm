string_height_ext

Esta función devolverá la altura (en píxeles) de la entrada string, teniendo en cuenta la separación entre líneas y el ancho del salto de línea (que se define como el número de píxeles que puede ocupar el string antes de que se inserte un salto de línea). Es muy útil para calcular las distancias entre los elementos de texto en función de la más alta de las letras que componen el string así como la propia longitud de string cuando se reparte en varias líneas como se dibujaría con draw_text_ext() utilizando el font actualmente definido. La separación y la anchura pueden establecerse en -1 para obtener el espaciado por defecto.

Sintaxis:

string_height_ext(string, sep, w);

ArgumentoTipo Descripción
string The string to measure the height of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Devuelve:

Ejemplo:

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

El código anterior obtendrá la altura del string dado, teniendo en cuenta la separación entre líneas y el ancho del salto de línea, y luego dibujará dos líneas de texto, utilizando la altura total devuelta de string como separador.
