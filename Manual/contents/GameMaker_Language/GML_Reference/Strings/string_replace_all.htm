string_replace_all

Puede utilizar esta función para analizar un string en busca de partes específicas, que pueden ser sustituidas por el nuevo string que ha especificado en todos los lugares en los que aparecen.

Sintaxis:

string_replace_all(str, substr, newstr);

ArgumentoTipo Descripción
str The string to be copied.
substr The substring within the string to be replaced in all instances.
newstr The new substring to replace the previous one(s).

Devuelve:

Ejemplo:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Esto establecerá str2 como str1, pero con sus instancias de "x" reemplazadas por "l", resultando en str2 ser "Hello World".
