cuenta_de_cadenas

Esta función devolverá la cantidad de veces que la subcadena dada aparece dentro de un string específico. De este modo, puede comprobar cuántas veces se repite una sola letra o una frase en una sección del texto almacenado.

Sintaxis:

string_count(substr, str);

ArgumentoTipo Descripción
substr The substring to check the string for.
str The string to check.

Devuelve:

Ejemplo:

str1 = "Hello World";
ocount = string_count( "o", str1 );

Esto establecerá la variable ocount al número de "o" en str1, en este caso 2.
