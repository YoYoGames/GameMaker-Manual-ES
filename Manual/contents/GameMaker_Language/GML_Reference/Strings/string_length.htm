longitud_de_cadena

Esta función devuelve el número de caracteres que componen un determinado string. Puede ser útil para cosas como calcular cuándo limitar la longitud de los caracteres de una entrada de texto personalizada (por ejemplo: limitar el nombre de un jugador a 10 caracteres). Recuerde que esto es diferente a string_width() en que mide el número de caracteres en string, no su anchura como se dibuja en la pantalla en píxeles.

Sintaxis:

string_length(string);

Argumento Tipo Descripción
string Cadena The string to measure the number of characters of.

Devuelve:

Real

Ejemplo:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

Esto comprobará si el string del nombre es mayor de diez caracteres y si lo es, copiará y utilizará sólo los diez primeros caracteres.
