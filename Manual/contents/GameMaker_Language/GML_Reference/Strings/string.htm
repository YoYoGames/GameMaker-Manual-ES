cadena

Con esta función puede convertir cualquier número real en un string. Si el número real es un entero, se guardará sin decimales, de lo contrario, se guardará con dos decimales. Si necesita más decimales, utilice la función string_format(). También tenga en cuenta que el uso de esta función en una variable que almacena una matriz, una estructura de datos o una estructura convertirá el contenido de estas variables en un string que puede ser enviado a la consola o guardado en un archivo para debugging.

Sintaxis:

string(val);

Argumento Tipo Descripción
val Real The real number to be turned into a string.

Devuelve:

String

Ejemplo:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

El código anterior utiliza la función string para dibujar tanto números reales como strings juntos (draw sólo acepta un string o un real, pero no ambos).
