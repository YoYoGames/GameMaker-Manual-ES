string_hash_to_newline

Esta función devuelve un string en el que el símbolo "#" se ha convertido en una nueva línea. Así que un string formateado como:

"Hello#World"

Se imprimiría en la pantalla como:

Hello
World

¡IMPORTANTE! Esta función se proporciona para la compatibilidad de importación entre GameMaker y las versiones anteriores del software y, como tal, esta función no debe utilizarse excepto cuando sea necesario para replicar la funcionalidad obsoleta. En su lugar debería utilizar los caracteres de escape para marcar una nueva línea. Para más información sobre la compatibilidad de importación, consulte aquí.

Sintaxis:

string_hash_to_newline(string);

ArgumentoTipo Descripción
string The string to convert over multiple lines

Devuelve:

Ejemplo:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

El código anterior convierte el string con el símbolo del hash en un string dividido en dos líneas y luego lo imprime en la pantalla.
