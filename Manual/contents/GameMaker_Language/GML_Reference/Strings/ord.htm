ord

Esta función toma una entrada de un solo carácter string y devuelve el valor Unicode (UTF8) para ese carácter. Tenga en cuenta que cuando se utiliza con las funciones keyboard_check* la entrada string sólo puede tener un carácter y sólo puede ser un número del 0 al 9 o un carácter romano en mayúsculas de la A a la Z.

Sintaxis:

ord(string);

Argumento Tipo Descripción
string Cadena The string with which to find the Unicode value.

Devuelve:

Real

Ejemplo:

if keyboard_check(ord("W"))
{
    y -= 4;
}

Esto moverá la instancia de llamada cuatro píxeles hacia arriba si se mantiene pulsada la tecla W.
