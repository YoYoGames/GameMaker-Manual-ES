string_height

Esta función devolverá la altura (en píxeles) de la entrada string, teniendo en cuenta la separación entre líneas y los saltos de línea que pueda tener el texto. Es muy útil para calcular las distancias entre los elementos del texto basándose en la más alta de las letras que componen el string tal y como se dibujaría con draw_text() utilizando el font actualmente definido.

Sintaxis:

string_height(string);

ArgumentoTipo Descripción
string The string to measure the height of.

Devuelve:

Ejemplo:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

El código anterior obtendrá la altura del string dado y luego dibujará dos líneas de texto, usando la altura devuelta del string como separador.
