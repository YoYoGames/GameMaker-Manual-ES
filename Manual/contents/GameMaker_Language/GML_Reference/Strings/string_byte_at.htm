string_byte_at

Devuelve el valor del byte crudo como un valor real en una posición dada en el string dado.

Sintaxis:

string_byte_at(str, index);

ArgumentoTipo Descripción
str The string to check.
index The position to get the byte from.

Devuelve:

Ejemplo:

newbyte = string_byte_at("Hello World", 5);

Esto establecerá en newbyte el valor en bytes de la sexta letra de "Hello World".
