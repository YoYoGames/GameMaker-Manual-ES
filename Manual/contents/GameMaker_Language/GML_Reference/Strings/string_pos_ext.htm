string_pos_ext

Esta función devolverá la posición del carácter de una instancia de una subcadena dentro de un string, buscando hacia adelante a través del string desde la posición dada como posición inicial. La función devolverá 0 si la búsqueda string no se encuentra, o la posición del primer carácter de la búsqueda string si se encuentra. Tenga en cuenta que para el soporte de legado strings se indexan desde 1, por lo que 1 es la primera posición en el string, no 0 como podría esperarse.

Sintaxis:

string_pos_ext(substr, str, start_pos);

ArgumentoTipo Descripción
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

Devuelve:

Ejemplo:

if (string_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_pos_ext(",", text, 20));
}

El código anterior busca en la string almacenada en la variable "text" una coma después del 20º carácter, y si la encuentra inserta la subcadena "name" en esa posición.
