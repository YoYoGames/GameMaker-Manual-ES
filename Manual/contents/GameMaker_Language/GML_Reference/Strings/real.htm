real

Esta función puede utilizarse para convertir un string dado en un número real. Cuando se utiliza esta función, se tienen en cuenta los números, los signos menos, los puntos decimales y las partes exponenciales en el string, mientras que otros caracteres (como las letras) provocarán un error de error. Si sabe, o sospecha, que un string puede tener otros caracteres, puede utilizar string_digits() para eliminar todos los caracteres no numéricos, antes de utilizar esta función para convertir la cadena resultante en un número real.

Sintaxis:

real(string);

ArgumentoTipo Descripción
string The string to be converted to a real value.

Devuelve:

Ejemplo:

var t_str = string_digits(input_str);
age = real(t_str);

El código anterior tomará la entrada string, la despojará de todos los caracteres que no sean números y luego establecerá la variable "edad" para mantener el valor numérico real del retorno string.
