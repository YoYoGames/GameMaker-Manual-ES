string_insert

Con esta función puede crear un nuevo string compuesto por dos strings, donde uno ha sido insertado en el otro en una posición determinada. Puede ser útil, por ejemplo, para añadir un nombre de usuario en un texto predefinido y así hacer que el jugador de su juego se sienta más involucrado en la acción. Tenga en cuenta al calcular la posición para insertar en que strings están indexados desde 1, por lo que el primer carácter en un string es 1 y no 0 como se puede esperar.

Sintaxis:

string_insert(substr, str, index);

ArgumentoTipo Descripción
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

Devuelve:

Ejemplo:

str2 = string_insert(username, "Hello, , how are you?", 8);

Esto insertará el string en la variable "nombre de usuario" en la frase dada, con el resultado de string con este aspecto: "Hola, NOMBRE, ¿cómo estás?".
