letrasdigitoscadenas

Esta función devolverá una copia de un string dado con todo menos sus letras y dígitos eliminados, lo que significa que puede utilizarse para eliminar cualquier carácter no deseado (como "#" o "?") de, por ejemplo, un nombre de usuario o una contraseña.

NOTA Esta función sólo devuelve el alfabeto inglés de 26 letras de la A a la Z (mayúsculas o minúsculas) y los números del 0 al 9.

Sintaxis:

string_lettersdigits(string);

ArgumentoTipo Descripción
string The string to convert to letters and digits.

Devuelve:

Ejemplo:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
    draw_text(32,32, "Invalid Password! Only Letters and numbers please!");
}

El código anterior comprobará la longitud de un string con la longitud del mismo string, pero con todos los símbolos eliminados. Si no son iguales, se dibuja un mensaje en la pantalla.
