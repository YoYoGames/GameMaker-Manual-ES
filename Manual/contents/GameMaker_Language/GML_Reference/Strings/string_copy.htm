copiar_cadenas

Con esta función puede seleccionar fácilmente un número de caracteres de un string para copiarlo en otro. El primer carácter de un string siempre se indexa como 1 y no como 0, como es de esperar, por lo que para copiar (por ejemplo) los cinco primeros caracteres de string se tendría string_copy(str, 1, 5). A continuación se ofrece otro ejemplo.

Sintaxis:

string_copy(str, index, count);

ArgumentoTipo Descripción
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Devuelve:

Ejemplo:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

El código anterior permitirá al jugador introducir un string a través del teclado que se almacena en la variable name. Esta variable se comprueba para ver si excede la longitud máxima de 15 caracteres y si lo hace, la cadena_del_teclado se sustituye por una copia de 15 caracteres del name string. Esto limita efectivamente la entrada del jugador a 15 caracteres.
