ansi_char

Esta función devuelve un string que contiene el carácter con el valor crudo BYTE establecido. Esto no se mostrará, ni debe mostrarse, pero se guardará correctamente en el disco para su uso en la codificación.

Sintaxis:

ansi_char(val);

ArgumentoTipo Descripción
val The raw byte value.

Devuelve:

(Single character)

Ejemplo:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

El código anterior crea un string a partir de datos de bytes sin procesar y lo escribe en un archivo (previamente abierto).
