clipboard_get_text

Esta función devolverá un string del texto contenido en el portapapeles. si no hay texto almacenado devolverá un string vacío "".

Sintaxis:

clipboard_get_text();

Devuelve:

String

Ejemplo:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

El código anterior comprueba si hay texto en el portapapeles y, si lo contiene, lo lee como string en la variable "str". Por último, el portapapeles se borra poniéndolo en un string vacío.
