mostrar_pregunta

Esta función crea un cuadro de mensaje emergente con dos botones para "Sí" y "No". Devuelve true o false dependiendo de cuál de los dos botones pulse el usuario.

NOTA Esta función es para uso de depuración sólo en el objetivo Windows, pero es obsoleta en todos los demás objetivos.

Sintaxis:

show_question(str);

Argumento Tipo Descripción
str Cadena The string to show in the pop-up question.

Devuelve:

Boolean

Ejemplo:

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

El código anterior comprobará la puntuación y si es superior a 500, preguntará al usuario si desea continuar o no y si se pulsa el botón "sí" pasará a otro room, pero si se selecciona el botón "no" terminará el juego.
