show_debug_message

Esta función creará un mensaje de depuración personalizado que se mostrará en la ventana del compilador en runtime. Tenga en cuenta que el mensaje puede ser un string o un número real, pero si necesita ambos entonces el número tendrá que ser convertido a string primero usando la función string() (véase el ejemplo siguiente) y que si el número tiene más de dos decimales, deberá utilizar string_format() para mostrarlos, ya que por defecto los decimales se redondean a los dos más cercanos (así que "1.2468" se mostraría como "1.25" en la ventana de salida).

Los mensajes de depuración mostrados con esta función se mostrarán en la Ventana de Salida del Compilador en la parte inferior de IDE así como en la Vista Gráfica de debugger cuando se ejecute el juego en Modo Depuración. Si sólo quieres ver los mensajes en el modo de depuración, entonces probablemente deberías usar debug_event() en su lugar.

Sintaxis:

show_debug_message(string);

ArgumentoTipo Descripción
string The custom debug message to show.

Devuelve:

Ejemplo:

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

El código anterior comprueba si existe una instancia y, si no es así, se muestra un mensaje de depuración en la ventana de compilación y se termina el juego.
