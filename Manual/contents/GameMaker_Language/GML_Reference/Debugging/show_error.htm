mostrar_error

Esta función mostrará un string personalizado como mensaje de error.

IMPORTANTE El segundo argumento sólo existe para la compatibilidad con proyectos anteriores y no tendrá ningún efecto en el diálogo de error mostrado.

NOTA Esta función es sólo para uso de depuración.

Sintaxis:

show_error(str, abort);

Argumento Tipo Descripción
str Cadena The string to show in the pop-up message.
abort Booleano UNUSED Si el error debe abortar el juego (true) o permitir que el jugador lo ignore (false).

Devuelve:

N/A

Ejemplo:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

El código anterior comprobará si el room actual es rm_Dungeon, y si no lo es, mostrará un mensaje de error.
