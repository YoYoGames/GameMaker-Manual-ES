mostrar_mensaje

Esta función crea un cuadro de mensaje emergente que muestra el string dado y un botón "Ok" para cerrarlo.

NOTA Esta función es para uso de depuración sólo en el objetivo Windows, pero es obsoleta en todos los demás objetivos.

Sintaxis:

show_message(str);

Argumento Tipo Descripción
str Cadena The string to show in the pop-up message.

Devuelve:

N/A

Ejemplo:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

El código anterior loop a través de los valores almacenados en la matriz "inv" y añadirlos a la variable "tot" antes de mostrar un mensaje con el total.
