fps

En GameMaker hay dos formas principales que se pueden utilizar para indicar la velocidad a la que se ejecuta el juego. El game speed (como se especifica en las Opciones de Juego) y el fps (cuadros por segundo). Estos valores se confunden a menudo, pero básicamente uno es el número de pasos del juego que GameMaker se supone que está completando en un segundo, mientras que el otro (los fps) es el número de pasos de la CPU que GameMaker está realmente completando en un segundo hasta un valor máximo de la propia velocidad de room . Para obtener los verdaderos fps, es decir, el número real de pasos de cpu por paso de juego, utilice la variable fps_real variable.

Esta variable de sólo lectura devuelve los fps actuales como un valor entero. Tenga en cuenta que la función sólo se actualizará una vez cada paso de su juego y por lo tanto puede parecer que "salta" de un valor a otro, pero esto es bastante normal.

Sintaxis:

fps

Devuelve:

Ejemplo:

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

El código anterior comprobará si el juego está en modo de depuración y si lo está mostrará los fps actuales en la pantalla.
