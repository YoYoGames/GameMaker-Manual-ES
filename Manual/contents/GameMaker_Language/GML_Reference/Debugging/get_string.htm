get_string

Esto crea una ventana emergente que muestra un mensaje estándar, con un botón llamado "Ok", que pide al usuario que introduzca un string. La función devolverá la entrada string, o el valor por defecto si no se ha introducido nada.

NOTA Esta función es para uso de depuración sólo en el objetivo Windows, pero es obsoleta en todos los demás objetivos.

Sintaxis:

get_string(str, def);

Argumento Tipo Descripción
str Cadena The string to show in the pop-up message.
def Cadena The default string in the text box.

Devuelve:

String

Ejemplo:

global.test_name = get_string("Test highscore name:", "Anonymous");

El código anterior pedirá al usuario que dé un nombre que se almacenará en la variable global "nombre_de_prueba". Si no se introduce nada y el usuario se limita a pulsar "Ok", se devolverá el valor por defecto, "Anónimo".
