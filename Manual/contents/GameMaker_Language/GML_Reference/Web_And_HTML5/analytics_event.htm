analytics_event

Esta función enviará el texto especificado al proveedor de analítica que haya configurado a través de las Opciones de Juego HTML5. Esta función se puede utilizar para crear una analítica personalizada para rastrear algo fuera del ámbito del proveedor que se está utilizando.

Sintaxis:

analytics_event(string);

ArgumentoTipo Descripción
string A string to send to the provider

Devuelve:

Ejemplo:

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

El código anterior comprobará una variable para ver si es true y, si lo es, se activará un evento especial de análisis con el string especificado.
