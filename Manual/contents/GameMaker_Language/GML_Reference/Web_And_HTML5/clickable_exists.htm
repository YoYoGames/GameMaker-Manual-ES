clickable_exists

Esta función devuelve si se ha creado un icono DOM clicable con el índice especificado existe o no. Tenga en cuenta que el valor utilizado para la comprobación debe haber sido inicializado previamente o de lo contrario obtendrá un error que hará que GameMaker se cierre.

Sintaxis:

clickable_exists(index);

Argumento Tipo Descripción
index ID de clicks The index of the clickable to check.

Devuelve:

Boolean

Ejemplo:

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

El código anterior comprueba la variable "home_but" para ver si ya existe, y si no lo hace crea un icono DOM clicable en la posición (32, 32) de la página en la que se está ejecutando el lienzo del juego. El icono utiliza el sprite referenciado desde la página de la textura como "spr_MS_Home" y cuando se hace clic en el icono se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana.
