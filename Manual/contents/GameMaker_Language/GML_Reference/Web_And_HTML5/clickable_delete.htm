clickable_delete

Esta función debe utilizarse para eliminar de la ventana de juego un icono clicable creado previamente con clickable_add() de la ventana del juego.

Sintaxis:

clickable_delete(index);

Argumento Tipo Descripción
index ID de clicks Index of the clickable icon to remove.

Devuelve:

N/A

Ejemplo:

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

El código anterior elimina el icono clicable indexado en la variable global "Help_Icon" de la ventana del juego, si existe.
