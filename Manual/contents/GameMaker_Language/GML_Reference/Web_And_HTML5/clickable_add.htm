clickable_add

Con esta función se puede crear un icono personalizado, sobre el que se puede hacer clic, que actúa como un "botón" fuera del lienzo del juego, pero dentro de la propia ventana del navegador. La función devolverá un valor de ID que debe ser almacenado en una variable para ser utilizado en todas las funciones de botón posteriores que hacen referencia al nuevo clickable. Esta función es muy útil para evitar que el navegador cree una ventana emergente al hacer clic en el botón, ya que todo se hace a través del propio GameMaker.

Primero debe establecer la posición, que es relativa al (0,0) de la ventana del navegador (esquina superior izquierda) y luego dar la entrada de la página de textura para el sprite que desea que el icono del botón tenga (ver: sprite_get_tpe()). A continuación, puede especificar una URL para abrir o un script (desde el navegador Asset de GameMaker) para abrir o ejecutar. El parámetro "target" que se especifica es el mismo que el valor "name" estándar de JavaScript cuando se utiliza el método open() (tenga en cuenta que todo lo que no sea '_self' puede dar lugar a que el navegador se bloquee, o pregunte al usuario si desea permitirlo) y el argumento "params" es el mismo que el parámetro "specs" de JavaScript para controlar qué propiedades debe mostrar la nueva ventana/pestaña (puede que no todos los navegadores soporten todas las características). Si está utilizando el icono para ejecutar un script interno, estos dos últimos argumentos pueden dejarse como strings "" vacíos.

Los objetivos válidos son:

Objetivo Descripción
En blanco Abre el documento vinculado en una nueva ventana o pestaña.
_self Abre el documento vinculado en el mismo marco en el que se ha hecho clic (es el predeterminado).
_padre Abre el documento vinculado en el marco principal.
Arriba Abre el documento vinculado en el cuerpo completo de la ventana.

Los parámetros válidos son:

Parámetros Descripción
'height=[px]' La altura de la ventana, siendo el valor mínimo 100.
'width=[px]' La anchura de la ventana, siendo el valor mínimo 100.
'left=[px]' La posición izquierda de la ventana.
'top=[px]' La posición superior de la ventana (sólo en IE).
'location=[boolean]' Mostrar o no el campo de la dirección (por defecto es 1).
'menubar=[boolean]' Mostrar o no la barra de menús (por defecto es 1).
'resizable=[boolean]' Si la ventana es redimensionable o no (por defecto es 1).
'scrollbars=[boolean]' Si se muestran o no las barras de desplazamiento (por defecto es 1).
'status=[boolean]' Añadir o no una barra de estado (por defecto es 1).
'titlebar=[boolean]' Mostrar o no la barra de título. Esto se ignora a menos que la aplicación que llama sea una aplicación HTML o un cuadro de diálogo de confianza (por defecto es 1);
'toolbar=[boolean]' Mostrar o no la barra de herramientas del navegador (por defecto, sí).


También vale la pena señalar que si el argumento de la URL que pasas es el nombre de una función de script (es decir, no comienza con "http://") Y el script se llama gmcallback_* entonces puedes obtener alimentación directa en un evento html5. Los nombres de las funciones de script de GameMaker no están ofuscados si comienzan con gmcallback_.

Sintaxis:

clickable_add(x, y, tpe, url, target, params);

Argumento Tipo Descripción
x Real The x position within the window.
y Real The y position within the window.
tpe Entrada de la página de texturas The texture page entry for the sprite to be used.
url Cadena The URL (website address) to link to, or the script to be run.
target Cadena This is the target area to open the URL in (see description).
params Cadena The various parameters for the icon (see description).

Devuelve:

Clickable ID

Ejemplo:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

El código anterior crea un icono DOM clicable en la posición (32, 32) de la página en la que se está ejecutando el lienzo del juego. El icono utiliza el sprite referenciado desde la página de la textura como "spr_MS_Home" y cuando se hace clic en el icono se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana.
