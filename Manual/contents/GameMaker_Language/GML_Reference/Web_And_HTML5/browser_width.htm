browser_width

Esta variable contiene la anchura (en píxeles) del navegador en el que se está ejecutando el juego. La siguiente imagen ilustra cómo esta variable se relaciona con el resto de las funciones de posicionamiento disponibles a través de las Funciones de Ventana.

Sintaxis:

browser_width

Devuelve:

Ejemplo:

if (browser_height > window_get_height()) || (browser_width > window_get_width())
{
    var xx, yy;
    if (browser_width > window_get_width())
    {
        xx = (browser_width - window_get_width()) / 2;
    }
    else
    {
        xx = 0;
    }
    if (browser_height > window_get_height())
    {
        yy = (browser_height - window_get_height()) / 2;
    }
    else
    {
        yy = 0;
    }
    window_set_position(xx, yy);
}

El código anterior comprueba la anchura y la altura del navegador para ver si alguna de ellas es mayor que la anchura y la altura de la ventana actual. Si alguno de ellos lo es, entonces calcula una nueva posición para la ventana del juego dentro del navegador.
