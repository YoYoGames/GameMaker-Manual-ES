clickable_change_ext

Con esta función se puede cambiar el sprite y la posición de un icono clicable creado previamente con clickable_add(). Tenga en cuenta que la posición se basa en la posición (0,0) de la ventana, no del lienzo, y que el sprite debe ser referenciado directamente desde la página de la textura (ver sprite_get_tpe()). Esta función también permite cambiar la escala de la sprite utilizada (como multiplicador, de modo que 1 es el valor por defecto, 0,5 sería la mitad y 2 el doble) y el valor alfa de 0 (totalmente transparente) a 1 (totalmente opaco) para el icono final en la pantalla.

Sintaxis:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Argumento Tipo Descripción
index ID de clicks The index of the clickable icon to change.
tpe Entrada de la página de texturas The texture page entry for the sprite to be used
x Real The new x position within the window.
y Real The new y position within the window.
scale Real The scale of the icon (default 1).
alpha Real The image alpha of the icon (default 1).

Devuelve:

N/A

Ejemplo:

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

El código anterior cambiará el índice de la imagen y la posición del icono clicable indexado en la variable "global.Help" dependiendo de la actual room, cambiando también la escala y el alfa del icono.
