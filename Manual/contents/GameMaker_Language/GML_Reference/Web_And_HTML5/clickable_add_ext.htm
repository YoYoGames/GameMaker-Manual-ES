clickable_add_ext

Con esta función se puede crear un icono personalizado, sobre el que se puede hacer clic, que actúa como un "botón" fuera del lienzo del juego, pero dentro de la propia ventana del navegador. La función devolverá un valor de ID que debe ser almacenado en una variable para ser usado en todas las funciones de botón posteriores que hagan referencia al nuevo clicable. Esta función es muy útil para evitar que el navegador cree una ventana emergente al hacer clic en el botón, ya que todo se hace a través del propio GameMaker.

Esta función funciona exactamente igual que la función clickable_add() donde debe establecer la posición relativa al (0,0) de la ventana del navegador, dar la entrada de la página de textura para el sprite que desea que tenga el icono del botón (ver: sprite_get_tpe()), especificar una URL para abrir o un script (desde el navegador Asset de GameMaker) para abrir o ejecutar y luego establecer el objetivo y los parámetros a utilizar. Sin embargo, esta función también tiene dos opciones adicionales que le permiten establecer el alfa del icono de 0 (totalmente transparente) a 1 (totalmente opaco) y la escala del icono, que es un valor multiplicador aplicado al sprite utilizado. Así, un valor de 1 es la escala por defecto, mientras que (por ejemplo) 0,5 sería la mitad de la escala y 2 sería el doble.

También vale la pena señalar que si el argumento de la URL que pasas es el nombre de una función de script (es decir, no comienza con "http://") Y el script se llama gmcallback_* entonces puedes obtener alimentación directa en un evento html5. Los nombres de las funciones de script de GameMaker no están ofuscados si comienzan con gmcallback_.

Sintaxis:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argumento Tipo Descripción
x Real The x position within the window.
y Real The y position within the window.
tpe Entrada de la página de texturas The texture page entry for the sprite to be used.
url Cadena The URL (website address) to link to, or the script to be run.
target Cadena This is the target area to open the URL in (see description).
params Cadena The various parameters for the icon (see description).
alpha Real The image alpha of the icon (default 1).
scale Real The scale of the icon (default 1).

Devuelve:

Clickable ID

Ejemplo:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

El código anterior crea un icono DOM clicable en la posición (32, 32) de la página en la que se está ejecutando el lienzo del juego. El icono utiliza el sprite referenciado desde la página de la textura como "spr_MS_Home" y cuando se hace clic en el icono se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana. el botón también se escalará al doble del tamaño original de sprite y la mitad de un alfa de imagen de 1.
