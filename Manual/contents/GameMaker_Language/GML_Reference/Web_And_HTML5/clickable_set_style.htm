clickable_set_style

Esta función le permite establecer las propiedades de estilo CSS para el botón dado a través de los pares clave/valor en el archivo DS Map. Es necesario haber creado previamente el elemento botón (con clickable_add()) y el DS Map y suministrar los índices almacenados de cada uno como argumentos.

Sintaxis:

clickable_set_style(index, map)

Argumento Tipo Descripción
index ID de clicks The index of the clickable icon to style.
map ID del mapa DS The index of the DS Map para establecer el estilo de.

Devuelve:

N/A

Ejemplo:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

El código anterior cambiará el estilo del botón pulsable con el índice almacenado en la variable "botón".
