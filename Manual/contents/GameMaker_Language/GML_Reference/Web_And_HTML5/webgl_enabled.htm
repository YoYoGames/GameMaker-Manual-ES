webgl_enabled

Esta variable de sólo lectura devolverá si WebGL está habilitado (true) o no (false) para tu juego. Sólo funcionará para aquellos juegos que se ejecuten a través de un navegador (es decir, HTML5), y para todas las demás plataformas devolverá true.

Sintaxis:

webgl_enabled

Devuelve:

Ejemplo:

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

El código anterior comprueba el WebGL flag y luego establece la variable global "quality" en consecuencia.
