browser_input_capture

Con esta función puedes establecer si la ventana del navegador debe capturar toda la entrada (establécela en false) o si el juego debe capturar la entrada (establécela en true). Ten en cuenta que esta función es sólo para uso con el módulo HTML5.

Sintaxis:

browser_input_capture(enable);

Argumento Tipo Descripción
enable Booleano Enable or disable game input capture

Devuelve:

N/A

Ejemplo:

browser_input_capture(true);

El código anterior configura el juego para capturar todas las entradas del navegador en la plataforma de destino HTML5.
