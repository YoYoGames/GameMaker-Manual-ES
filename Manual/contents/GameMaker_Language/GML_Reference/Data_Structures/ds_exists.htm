ds_exists

Con esta función se puede comprobar si existe una estructura de datos del tipo dado. Se suministra el valor del "índice" (tal y como se mantiene en una variable) y el "tipo" del DS, que puede ser cualquiera de las constantes enumeradas a continuación, y la función devolverá true si la estructura de datos existe y false en caso contrario.

Tipo de DS Constante
Constante Descripción
ds_type_map Una estructura de datos de mapa
ds_type_list Una estructura de datos de lista
ds_type_stack Una estructura de datos de pila
ds_type_grid Una estructura de datos en forma de cuadrícula
ds_type_queue Una estructura de datos de cola
ds_type_priority Una estructura de datos prioritarios

NOTA No se puede utilizar esta función para comprobar el tipo de una estructura de datos, ya que el mismo número de índice puede ser utilizado por múltiples data structures de diferentes tipos.

Sintaxis:

ds_exists(ind, type);

Argumento Tipo Descripción
ind Cualquier DS ID The variable index to check for the data structure
type Tipo de DS Constante The type of data structure to check for (see the list of constants above)

Devuelve:

Boolean

Ejemplo:

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

El código anterior comprueba la variable (previamente inicializada) "ai_grid" para ver si indexa una estructura de datos de tipo grid DS, y si no lo hace entonces crea una y almacena su índice en la variable.
