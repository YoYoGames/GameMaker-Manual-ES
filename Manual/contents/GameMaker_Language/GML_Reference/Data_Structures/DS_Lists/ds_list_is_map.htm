ds_list_is_map

Con esta función se puede comprobar si un mapa DS está almacenado en la posición dada dentro de una lista DS. Si la posición dada contiene un ID de mapa DS, entonces la función devolverá true de lo contrario devolverá false.

Tenga en cuenta que esto sólo detectará los mapas marcados manualmente mediante la función ds_list_mark_as_map() .

Sintaxis:

ds_list_is_map(id, pos);

ArgumentoTipo Descripción
id The id of the list to check.
pos The position within the list to check.

Devuelve:

Ejemplo:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if ds_list_is_map(ships, i)
    {
        ds_map_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

El código anterior loops a través de una lista DS y comprueba si alguna de las entradas contiene IDs de mapas. Si lo hacen, entonces estos mapas son destruidos, y luego la lista principal es destruida después de que el loop haya terminado.
