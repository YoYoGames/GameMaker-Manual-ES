ds_list_find_index

Con esta función se puede buscar un valor en la lista dada y se devolverá la posición dentro de la lista de ese valor. Tenga en cuenta que si hay más de una entrada en la lista con el mismo valor, se puede devolver la posición de cualquiera de ellas, y que si el valor no existe, se devolverá -1.

Sintaxis:

ds_list_find_index(id, val);

Argumento Tipo Descripción
id ID de la lista DS The id of the list to use.
val Variable The value to find.

Devuelve:

Real

Ejemplo:

pos = ds_list_find_index(list, "Player1");

El código anterior comprueba la lista indexada en la variable "list" para el valor "Player1" y almacena la posición devuelta en la variable "pos".
