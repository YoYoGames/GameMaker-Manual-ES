ds_list_delete

Con esta función puede eliminar el valor almacenado en una posición específica dentro de la lista.

Sintaxis:

ds_list_delete(id, pos);

ArgumentoTipo Descripción
id The id of the list to change.
pos Where in the list to delete the value.

Devuelve:

Ejemplo:

if (ds_list_size(sc_list) > 10)
{
    while (ds_list_size(sc_list) > 10)
    {
        ds_list_delete(sc_list, 0);
    }
}

El código anterior comprueba el tamaño de una lista DS y si es mayor de diez, loops a través de la lista eliminando el valor superior (posición 0) hasta que la lista tenga sólo 10 entradas.
