ds_list_create

Esta función creará una nueva estructura de datos de la lista y devolverá el valor del índice. Este valor debe almacenarse en una variable y utilizarse en todas las llamadas posteriores a la lista. Tenga en cuenta que si intenta acceder a un valor de la lista antes de que se haya rellenado, es decir, si la lista está "vacía", el valor de retorno puede ser 0, ya que internamente las primeras entradas de la lista se establecen cuando se crean para minimizar los problemas de rendimiento al añadir inicialmente elementos a la lista (aunque la función ds_list_size() devolverá 0 en una lista recién creada). Si desea asegurarse de que la lista está "realmente" vacía al crearla, deberá llamar a ds_list_clear() después de crear la lista, lo que significará que cualquier valor devuelto para las ranuras de la lista sin rellenar será undefined.

¡IMPORTANTE! Cuando se crea una estructura de datos, el valor del índice para identificarla es un valor entero que comienza en 0. Esto significa que data structures de diferentes tipos puede tener el mismo valor de índice, por lo que en caso de duda se debe utilizar la función ds_exists() antes de acceder a ellos. También hay que tener en cuenta que los índices se reutilizan, por lo que el valor del índice de una estructura de datos destruida puede ser utilizado por otra recién creada posteriormente.

Sintaxis:

ds_list_create();

Devuelve:

DS List ID

Ejemplo:

list = ds_list_create();

Esto creará una nueva lista y asignará su índice a la variable de instancia "lista".
