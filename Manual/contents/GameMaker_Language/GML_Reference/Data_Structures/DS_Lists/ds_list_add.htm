ds_list_add

Esta función puede utilizarse para añadir un nuevo valor (real o string) a la lista, que se añadirá al final. La función puede recibir más argumentos opcionales (tantos como se necesiten), lo que permite añadir varios valores consecutivos a la lista en una sola llamada.

Sintaxis:

ds_list_add(id, val1 [, val2, ... max_val]);

Argumento Tipo Descripción
id ID de la lista DS The id of the list to add to.
val Variable The value to add to the list.
[val2, ... max_val] Variable Optional values to be added to the list.

Devuelve:

N/A

Ejemplo:

ds_list_add(sc_list, score);

El código anterior añadirá el valor almacenado en la variable "score" a la lista indexada en la variable "sc_list".
