ds_list_find_value

Con esta función se puede comprobar la posición de la lista dada y se devolverá el valor que tenga la lista en esa posición. Tenga en cuenta que si da una posición que está fuera del tamaño de la lista dada (por ejemplo: la posición 11 en una lista de 10 valores) entonces la función puede devolver undefined o 0. Esto se debe a que cuando se crea la lista, internamente las primeras entradas de la lista se ponen a 0 para minimizar los problemas de rendimiento cuando se añaden inicialmente elementos a la lista (aunque la función ds_list_size() devolverá 0 en una lista recién creada). Si desea asegurarse de que la lista está "realmente" vacía al crearla, debe llamar a ds_list_clear() después de crear la lista, lo que significará que cualquier valor devuelto para las ranuras de la lista sin rellenar será undefined, lo que puede comprobar de forma consistente utilizando la función is_undefined() función.

Sintaxis:

ds_list_find_value(id, pos);

Argumento Tipo Descripción
id ID de la lista DS The id of the list to use.
pos Real The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

Devuelve:

Variable or undefined

Ejemplo:

val = ds_list_find_value(list, ds_list_size(list) - 1);

El código anterior comprueba la lista indexada en la variable "list" en la última posición de la lista y almacena el valor devuelto en la variable "val".
