ds_list_clear

Con esta función puedes borrar todos los datos de la estructura de datos de la lista dada. Esto NO destruye la estructura de datos (para eso debe usar ds_list_destroy()), sólo borra todos los datos de la misma y deja la lista vacía (con tamaño cero). Tenga en cuenta que borrar una lista desreferenciará cualquier data structures almacenado en ella dando una fuga de memoria, por lo que tendría que ir a través de la lista y destruir todos los elementos de la estructura de datos manualmente antes de borrarla para evitar esto. El único caso en el que esto no es necesario es cuando has marcado algún elemento de la lista como otra lista DS o como un mapa DS, en cuyo caso estos elementos serán destruidos (¡no borrados!) y su memoria se limpiará automáticamente cuando se borre el padre.

Sintaxis:

ds_list_clear(id);

ArgumentoTipo Descripción
id The id of the data structure to clear.

Devuelve:

Ejemplo:

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y si lo ha hecho borra la lista DS indexada en la variable "lista_de_comandos", establece una alarma y reinicia la variable a 0.
