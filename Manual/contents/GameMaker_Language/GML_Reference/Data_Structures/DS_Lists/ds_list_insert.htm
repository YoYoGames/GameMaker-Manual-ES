ds_list_insert

Esta función añadirá el valor dado a la lista en la posición dada. Si la lista contiene más valores después de la posición dada, su posición se desplazará uno hacia arriba para hacer room haciendo la lista más grande en uno.

Sintaxis:

ds_list_insert(id, pos, val);

Argumento Tipo Descripción
id ID de la lista DS The id of the list to add to.
pos Real The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Variable The value to add to the list.

Devuelve:

N/A

Ejemplo:

ds_list_insert(list, 9, score);

El código anterior añadirá el valor almacenado en la variable "score" en la décima posición de la lista indexada por la variable "list" (las listas se cuentan desde 0, por lo que una lista de diez valores se numera de 0 a 9).
