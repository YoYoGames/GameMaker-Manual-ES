ds_list_is_list

Con esta función se puede comprobar si otra lista DS está almacenada en la posición dada dentro de una lista DS. Si la posición dada contiene un ID de lista DS, entonces la función devolverá true de lo contrario devolverá false.

Tenga en cuenta que esto sólo detectará las listas que fueron marcadas manualmente utilizando la función ds_list_mark_as_list() .

Sintaxis:

ds_list_is_list(id, pos);

ArgumentoTipo Descripción
id The id of the list to check.
pos The position within the list to check.

Devuelve:

Ejemplo:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_list(ships, i))
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

El código anterior loops a través de una lista DS y comprueba si alguna de las entradas contiene otros ID de lista. Si lo hacen, entonces estas listas se destruyen, y luego la lista principal se destruye después de que el loop haya terminado.
