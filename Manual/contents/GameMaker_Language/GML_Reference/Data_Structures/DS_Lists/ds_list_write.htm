ds_list_write

Esta función devuelve un string que puede ser almacenado o transferido a otra estructura de datos utilizando la función ds_list_read() función.

NOTA: El string devuelto no es un string legible, sino un volcado del contenido de la estructura de datos.

Sintaxis:

ds_list_write(id);

ArgumentoTipo Descripción
id The id of the data structure to write.

Devuelve:

Ejemplo:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

El código anterior abre un archivo ini y luego escribe un string que contiene la información almacenada en la lista DS indexada en la variable "list". A continuación se borra la lista y se cierra el archivo ini.
