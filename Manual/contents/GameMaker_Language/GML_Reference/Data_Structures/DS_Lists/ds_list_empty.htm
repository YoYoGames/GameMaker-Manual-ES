ds_list_empty

Con esta función puede comprobar la lista DS dada para ver si está vacía (devuelve true) o no (devuelve false).

Sintaxis:

ds_list_empty(id);

Argumento Tipo Descripción
id ID de la lista DS The id of the data structure to check.

Devuelve:

Ejemplo:

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y si lo ha hecho borra la lista DS indexada en la variable "lista_de_comandos", establece una alarma y reinicia la variable a 0.
