ds_list_sort

Con esta función puede ordenar todos los valores de una lista, ya sea en orden ascendente o descendente. Si la lista contiene strings, éstos se ordenarán alfabéticamente, basándose en el alfabeto inglés de 26 letras.

Sintaxis:

ds_list_sort(id, ascend);

ArgumentoTipo Descripción
id The id of the list to sort.
ascend Whether the values should be ascending (true) o descendente (false).

Devuelve:

Ejemplo:

if newgame
{
    ds_list_sort(name_list, true);
}

El código anterior ordenará la lista indexada en la variable "name_list" si la variable "newgame" está marcada como true.
