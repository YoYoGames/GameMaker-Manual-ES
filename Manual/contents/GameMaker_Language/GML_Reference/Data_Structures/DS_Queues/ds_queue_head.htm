ds_queue_head

Esta función sólo leerá el primer valor de la queue (el que está "en la cabeza"). No pondrá en cola el valor, lo que significa que todavía puede ser leído en el futuro por esta función o por la función ds_queue_dequeue(). Si el queue está vacío, la función devolverá la constante undefined, de lo contrario devolverá el valor real o string contenido en el queue.

Sintaxis:

ds_queue_head(id);

ArgumentoTipo Descripción
id The id of the data structure to read from.

Devuelve:

(Data type value stored in the queue)

Ejemplo:

num = ds_queue_head(control_queue);

El código anterior leerá el valor de la queue indexado en la variable "control_queue" y almacenará el valor de retorno en la variable "num".
