ds_queue_copy

Esta función puede utilizarse para copiar el contenido de un queue en otro. Tenga en cuenta que esto NO elimina el contenido del queue original, ni destruye el queue original. Cuando se utiliza esta función, el queue que se está copiando debe haber sido creado previamente y si contenía algún elemento antes de la copia, éste se borrará primero (lo que significa que esta información se perderá).

Sintaxis:

ds_queue_copy(id, source);

Argumento Tipo Descripción
id DS Queue ID The id of the new queue.
source DS Queue ID The original queue to copy from.

Devuelve:

N/A

Ejemplo:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

La función anterior crea una nueva instancia y luego en esa instancia crea una nueva cola DS y copia el contenido de la queue en la instancia que ejecuta el bloque de código, en la instancia recién creada queue.
