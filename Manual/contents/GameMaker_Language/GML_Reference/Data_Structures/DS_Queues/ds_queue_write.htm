ds_queue_write

Esta función devuelve un string que puede ser almacenado o transferido a otra estructura de datos utilizando la función ds_queue_read() función.

NOTA: El string devuelto no es un string legible, sino un volcado del contenido de la estructura de datos

Sintaxis:

ds_queue_write(id);

ArgumentoTipo Descripción
id The id of the data structure to write.

Devuelve:

Ejemplo:

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

El código anterior abre un archivo ini y luego escribe un string que contiene la información almacenada en la cola DS indexada en la variable "queue". A continuación se borra el queue y se cierra el archivo ini.
