ds_queue_dequeue

Esta función dequeue el valor de la cabeza del DS queue, eliminándolo de la queue y devolviendo el valor para ser almacenado en una variable. Si el queue está vacío entonces la función devolverá la constante undefined, de lo contrario devolverá el valor contenido en el queue.

Sintaxis:

ds_queue_dequeue(id);

ArgumentoTipo Descripción
id The id of the queue to dequeue from.

Devuelve:

Variable (Data type value stored in the queue) or undefined

Ejemplo:

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

El código anterior comprueba la cola DS indexada en la variable "move_queue" para ver si está vacía, y si no lo está, entonces dequeues los dos valores de la cabeza de la queue y los utiliza para establecer una dirección para el movimiento.
