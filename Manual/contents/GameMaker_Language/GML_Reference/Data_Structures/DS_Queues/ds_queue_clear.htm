ds_queue_clear

Con esta función puede borrar todos los datos de la estructura de datos queue dada. Esto NO destruye la estructura de datos (para eso debes usar ds_queue_destroy()), sólo borra todos los datos de la misma y devuelve un queue vacío.

Sintaxis:

ds_queue_clear(id);

ArgumentoTipo Descripción
id The id of the data structure to clear.

Devuelve:

Ejemplo:

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

El código anterior comprueba una variable para ver si ha alcanzado un valor específico y si lo ha hecho borra la cola DS indexada en la variable "command_queue", establece una alarma y reinicia la variable a 0.
