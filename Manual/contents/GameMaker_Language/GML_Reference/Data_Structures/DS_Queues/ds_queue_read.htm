ds_queue_read

Con esta función puede recrear una cola DS guardada (una que haya sido escrita previamente como string usando ds_queue_write()). Primero debes crear una nueva cola DS en la que leer el string, y si la cola DS ya existe y tiene información guardada en ella, ésta se borrará antes de la lectura. Esta función es de vital importancia a la hora de crear mecanismos de guardado/carga para tu juego.

Tenga en cuenta que si el DS especificado string fue escrito por el GameMaker: Studio 1.2.x runtime (o anterior), debe especificar el argumento opcional "legacy", estableciéndolo en true ya que el formato string cambió después de eso.

Sintaxis:

ds_queue_read(id, str [, legacy]);

Argumento Tipo Descripción
id DS Queue ID The id of the data structure to read into.
str Cadena The string to read from.
legacy Booleano OPTIONAL Puede ser true o false u omitirse por completo.

Devuelve:

N/A

Ejemplo:

queue = ds_queue_create();
ini_open("save.ini");
var str = ini_read_string("Queues", "0", "");
if (str != "")
{
    ds_queue_read(queue, str);
}
ini_close();

El código anterior crea un queue y almacena el índice en la variable "queue". A continuación, abre un archivo ini y lee un string de él, comprobando que el string no se devuelva primero como vacío. Este string se lee entonces en el recién creado DS queue.
