Colas de prioridad DS

Una prioridad DS queue es una estructura de datos especial similar a un DS queue, sólo que aquí el queue de los valores está ordenado por las prioridades (siempre números reales) que se han asignado a los valores. Esta estructura puede ser muy útil para construir tablas de clasificación o listas de información en las que se puede ponderar la importancia de cada entrada.

NOTA: Como todos los recursos dinámicos, data structures ocupa memoria y, por tanto, debe destruirse siempre que ya no se necesite para evitar fugas de memoria que ralenticen y acaben por colapsar el juego.

NOTA: Si necesita comprobar si una estructura de datos existe, puede utilizar la función ds_exists() función.

Existen las siguientes funciones que se ocupan de las colas de prioridad DS:

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
