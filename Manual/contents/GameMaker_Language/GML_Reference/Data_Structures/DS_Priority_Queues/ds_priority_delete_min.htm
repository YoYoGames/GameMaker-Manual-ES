ds_priority_delete_min

Esta función devolverá el valor que tenga la menor prioridad en el queue y luego eliminará el valor (y la prioridad) de la estructura de datos. Si más de un valor tiene la misma prioridad, entonces cualquiera de ellos podría ser devuelto en cualquier orden, pero todos los demás valores con la misma prioridad seguirán estando en el queue.

Sintaxis:

ds_priority_delete_min(id);

ArgumentoTipo Descripción
id The id of the priority queue to use.

Devuelve:

(Data type stored in the priority)

Ejemplo:

if ai_move
{
    script_execute(ds_priority_delete_min(ai_priority));
}

El código anterior comprueba una variable de instancia y si devuelve true ejecutará una función script indexada en la prioridad queue con el valor de prioridad más bajo y luego eliminará ese script de la queue.
