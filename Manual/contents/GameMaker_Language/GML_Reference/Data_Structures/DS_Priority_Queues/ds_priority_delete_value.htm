ds_priority_delete_value

Esta función simplemente borrará el valor dado, junto con su prioridad, de la prioridad indexada queue.

Sintaxis:

ds_priority_delete_value(id,val);

ArgumentoTipo Descripción
id The id of the priority queue to use.
val The value to delete from the priority queue.

Devuelve:

Ejemplo:

if (ai_move == false)
{
    ds_priority_delete_value(ai_priority, AI_Move);
}

El código anterior comprueba una variable de instancia y si devuelve false eliminará la función indexada script de la prioridad queue.
