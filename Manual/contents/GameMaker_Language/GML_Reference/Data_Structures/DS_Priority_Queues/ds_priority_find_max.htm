ds_priority_find_max

Con esta función se puede encontrar el valor almacenado en la prioridad queue con la prioridad más alta, y si más de un valor tiene la misma prioridad entonces cualquiera de ellos podría ser devuelto en cualquier orden. Sin embargo, a diferencia de ds_priority_delete_max()esta función no eliminará el valor de queue.

Sintaxis:

ds_priority_find_max(id);

ArgumentoTipo Descripción
id The id of the priority queue to use.

Devuelve:

Real or String

Ejemplo:

if ai_move
{
    script_execute(ds_priority_find_max(ai_priority));
}

El código anterior comprueba una variable de instancia y si devuelve true ejecutará una función script indexada en la prioridad queue con el valor de prioridad más alto.
