ds_priority_write

Esta función devuelve un string que puede ser almacenado o transferido a otra estructura de datos utilizando la función ds_priority_read() función.

NOTA: El string devuelto no es un string legible, sino un volcado del contenido de la estructura de datos

Sintaxis:

ds_priority_write(id);

ArgumentoTipo Descripción
id The id of the data structure to check.

Devuelve:

Ejemplo:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

El código anterior abre un archivo ini y luego escribe un string que contiene la información almacenada en la prioridad DS queue indexada en la variable "p_queue". A continuación, se borra la prioridad queue y se cierra el archivo ini.
