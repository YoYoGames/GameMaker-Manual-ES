ds_priority_add

Con esta función se puede añadir un valor (ya sea un número real o un string) a una prioridad queue, asignándole al mismo tiempo un valor de prioridad.

Sintaxis:

ds_priority_add(id, val, priority);

Argumento Tipo Descripción
id ID de prioridad DS The id of the priority queue to add to.
val Variable The value to add to the priority queue.
priority Real The priority of the value to add.

Devuelve:

N/A

Ejemplo:

ds_priority_add(ai_priority, AI_Search, 5);

El código anterior añade una función script a la prioridad queue indexada en la variable "ai_priority" y le asigna una prioridad de 5.
