ds_priority_copy

Esta función puede utilizarse para copiar el contenido de una prioridad queue en otra. Tenga en cuenta que esto NO elimina el contenido de la prioridad original queue, ni destruye la prioridad original queue. Cuando se utiliza esta función, la prioridad queue a la que se está copiando debe haber sido creada previamente y si contenía algún elemento antes de la copia, éste se borrará primero (lo que significa que esta información se perderá).

Sintaxis:

ds_priority_copy(id, source);

ArgumentoTipo Descripción
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

Devuelve:

Ejemplo:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    p_queue = ds_priority_create();
    ds_priority_copy(p_queue, other.p_queue);
}

La función anterior crea una nueva instancia y luego en esa instancia crea una nueva prioridad DS queue y copia el contenido de la prioridad queue en la instancia que ejecuta el bloque de código, en la prioridad de la instancia recién creada queue.
