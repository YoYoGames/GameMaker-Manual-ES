ds_map_find_first

Esta función devuelve la primera clave almacenada en el mapa DS. ¡Esta no es la primera clave en el orden en que las agregó! Los mapas DS no se almacenan de forma lineal, para eso usa la lista DS, así que todo lo que hace es encontrar la primera clave tal y como la almacena el ordenador. Esto puede ser útil si tienes que iterar a través del mapa DS buscando algo, pero debe ser evitado si es posible ya que puede ser lento.

Tenga en cuenta que esta función devolverá undefined si el mapa DS dado está vacío.

Sintaxis:

ds_map_find_first(id);

ArgumentoTipo Descripción
id The id of the map to use.

Devuelve:

Variable or undefined

Ejemplo:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

El código anterior crea algunas variables temporales y luego obtiene el tamaño del mapa DS y encuentra la primera clave almacenada por el ordenador en el mapa. A continuación, utiliza un for loop para recorrer el mapa DS en busca del valor de la clave "oro". Si lo encuentra, saca el loop.
