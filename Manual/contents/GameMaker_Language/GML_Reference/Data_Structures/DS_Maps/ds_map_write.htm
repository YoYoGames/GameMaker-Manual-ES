ds_map_write

Esta función convertirá los datos del mapa DS del mapa especificado en el formato string que puede ser escrito en un archivo *.ini o *.txt para facilitar su almacenamiento. Este string puede ser leído posteriormente en un nuevo ds_map usando ds_map_read().

NOTA: El string devuelto no es un string legible, sino un volcado del contenido de la estructura de datos.

Sintaxis:

ds_map_write(id);

ArgumentoTipo Descripción
id The id of the data structure to use

Devuelve:

Ejemplo:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

El código anterior abre un archivo ini listo para ser escrito. A continuación, utiliza ds_map_write() para generar un string que se almacena en la variable temporal "t_string". Finalmente, escribe ese string en el archivo ini antes de cerrarlo.
