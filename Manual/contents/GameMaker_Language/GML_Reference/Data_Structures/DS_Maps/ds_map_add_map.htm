ds_map_add_map

Con esta función puede asignar un mapa DS (previamente creado) a una clave dentro del mapa DS dado . Esta función está diseñada para crear mapas compatibles con JSON que luego codificarás con json_encode() y sólo debe utilizarse junto con esa función. Si un mapa DS tiene otro mapa añadido de esta manera, entonces destruir el mapa padre también destruirá los mapas contenidos y liberará su memoria, y llamar a ds_map_clear() en el mapa padre también destruirá y limpiará cualquier mapa marcado.

Sintaxis:

ds_map_add_map(id, key, value)

Argumento Tipo Descripción
id ID del mapa DS The id of the map to use.
key Cadena The key for the added map.
value ID del mapa DS The id of the map to add.

Devuelve:

N/A

Ejemplo:

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

El código anterior creará dos mapas DS, y luego rellenará uno con una lista y un array antes de añadirlo al segundo, que luego se codifica en un JSON string. A continuación, el mapa se destruye para eliminarlo de la memoria, así como cualquier otro mapa o lista que contenga.
