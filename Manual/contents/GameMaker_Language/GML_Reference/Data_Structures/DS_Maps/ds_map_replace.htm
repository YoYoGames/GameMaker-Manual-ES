ds_map_replace

Con esta función se puede cambiar el valor de la clave dada dentro del mapa de un DS. Se suministra el índice del mapa (devuelto por la función ds_map_create()) y luego la clave a reemplazar - ya sea un string o un entero - y el valor con el que se reemplazará el valor de la clave. Si la clave dada no existe, se creará para usted, y si existe, el valor actual será reemplazado por el nuevo valor. valor. La función devolverá true si la clave existe y el valor es reemplazado, y false si la clave no existe y se ha creado una nueva clave con el valor.

Sintaxis:

ds_map_replace( id, key, val );

ArgumentoTipo Descripción
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

Devuelve:

Ejemplo:

ds_map_replace(inventory, "torso", 55);

El código anterior busca en el mapa DS la clave "torso" y cuando la encuentra (o se crea si no existe) se sustituye el valor actual por el especificado.
