ds_map_clear

Esta función borrará el mapa DS (previamente creado) de todos los pares clave/valor, pero no borrará el propio mapa DS de la memoria. Para ello debe utilizar la función ds_map_destroy(). Tenga en cuenta que borrar un mapa desreferenciará cualquier data structures almacenado en el mapa dando lugar a una fuga de memoria, por lo que tendría que ir a través de él y destruir todos los elementos de la estructura de datos manualmente antes de borrar el mapa para evitar esto. El único caso en el que esto no es necesario es cuando se han marcado elementos en el mapa como una lista DS o como otro mapa DS, en cuyo caso estos elementos serán destruidos (¡no borrados!) y su memoria se limpiará automáticamente cuando se borre el mapa padre.

Sintaxis:

ds_map_clear(id);

ArgumentoTipo Descripción
id The id of the data structure to clear.

Devuelve:

Ejemplo:

if global.new_game
{
    ds_map_clear(inventory);
}

La función anterior comprobará una variable global para ver si es verdadera y, si lo es, entonces el mapa DS indexado en la variable de instancia "inventario" será limpiado de todos los pares clave/valor (pero no es destruido).
