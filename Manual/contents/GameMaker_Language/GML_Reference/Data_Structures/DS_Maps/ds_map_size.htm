ds_map_size

Con esta función se puede encontrar cuántos pares clave/valor contiene el mapa DS (previamente creado).

Sintaxis:

ds_map_size(id);

ArgumentoTipo Descripción
id The id of the data structure to check.

Devuelve:

Ejemplo:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

El código anterior comprobará el tamaño del mapa DS (es decir, el número de pares clave/valor) y si es superior a 49 establece la variable "full" en true.
