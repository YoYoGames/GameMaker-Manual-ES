ds_map_keys_to_array

Con esta función puedes recuperar todas las claves que contiene un mapa DS. Se suministra el ID del mapa DS del que se quieren obtener las claves (tal y como lo devuelve ds_map_create()) y la función devolverá un array en el que cada entrada del array es una clave del mapa DS. La función tiene un segundo argumento opcional en el que puede suministrar un array que haya creado, en cuyo caso los datos de las claves del mapa se añadirán a cualquier dato existente en el array. Tenga en cuenta que la función modificará el array suministrado directamente, pero también devolverá una referencia al mismo (o una referencia a un nuevo array si no se suministra ninguno).

Sintaxis:

ds_map_keys_to_array(id, [array])

Argumento Tipo Descripción
id ID del mapa DS The id of the map to use.
[array] Matriz OPTIONAL El array al que añadir las claves del mapa DS.

Devuelve:

Array

Ejemplo:

map_keys = ds_map_keys_to_array(inventory);

El código anterior recupera las claves de un mapa DS y luego las almacena como una matriz para su uso futuro.
