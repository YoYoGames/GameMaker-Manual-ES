ds_map_replace_map

Con esta función se puede reemplazar un DS Map que ha sido almacenado en la "clave" dada por otro mapa que ha sido creado previamente. Esta función está diseñada para crear mapas compatibles con JSON que luego codificarás con json_encode() y sólo debe utilizarse junto con esa función.

Sintaxis:

ds_map_replace_map(id, key, value)

Argumento Tipo Descripción
id ID del mapa DS The id of the ds_map to use.
key Cadena The key to replace.
value ID del mapa DS The id of the ds_map to use to replace the one previously stored in the given key.

Devuelve:

N/A

Ejemplo:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

El código anterior creará un DS Map y lo poblará con un array y un DS List antes de reemplazar un mapa previamente almacenado en el DS Map "j_map".
