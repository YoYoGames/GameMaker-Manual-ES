ds_map_secure_save_buffer

Esta función guardará un mapa DS previamente creado en un buffer. Usted suministra el valor del ID del mapa DS (como lo devuelve la función ds_map_create()) y el ID del buffer en el que se va a escribir (devuelto por la función buffer_create()). Tenga en cuenta que si el mapa DS que se está guardando contiene un array, éste se convertirá en una lista DS cuando se guarde.

¡IMPORTANTE! El archivo de mapa DS guardado de forma segura sólo se puede cargar en el dispositivo que lo creó, y si intenta cargar un archivo que se guardó en un dispositivo diferente, no funcionará.

NOTA: Esta función no es compatible con HTML5 y UWP.

Sintaxis:

ds_map_secure_save_buffer(filename);

Argumento Tipo Descripción
id ID del mapa DS The DS map ID value.
buffer ID del búfer The buffer to save to.

Devuelve:

N/A

Ejemplo:

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

El código anterior creará un buffer y un mapa DS, luego rellenará el mapa con algunos valores y lo escribirá en el buffer antes de borrar el mapa.
