ds_map_add

Esta función debe utilizarse para añadir conjuntos de pares clave/valor en el mapa DS especificado. Puede comprobar esta función para ver si ha tenido éxito o no (devolverá true en caso de éxito o false en caso contrario), ya que puede fallar si ya existe la misma clave en el mapa DS o si se especifica un mapa DS inexistente como el ID del mapa a añadir. Las claves y los valores que proporcione pueden estar formados por cualquier combinación de data types, por lo que todos los siguientes - y más - son aceptables (aunque, en la práctica, lo más normal es que utilices un string para la clave):

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

También se puede añadir a un mapa utilizando el accesorio "?", como se muestra a continuación:

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

NOTA: A diferencia de otros data structures en GameMaker esta tecla no irá al inicio (ni al final) del mapa de la DS, sino que simplemente entrará en algún lugar del mapa de la DS.

Sintaxis:

ds_map_add(id, key, val);

ArgumentoTipo Descripción
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

Devuelve:

Ejemplo:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Esto creará un nuevo mapa, asignando su id a la variable "inventario". A continuación, añade dos nuevas claves al mapa, "hp potion" y "gold", y establece sus valores iniciales como 1 y 100.
