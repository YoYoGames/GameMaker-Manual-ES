ds_map_read

Esta función tomará un string que ha sido creado previamente por la función ds_map_write() y lo leerá en un mapa DS previamente creado. Si el mapa en el que se está leyendo string contiene algún par clave/valor, éste se borrará primero antes de reconstruir el mapa guardado.

Tenga en cuenta que si el DS especificado string fue escrito por el GameMaker: Studio 1.2.x runtime (o anterior), debe especificar el argumento opcional "legacy", estableciéndolo en true ya que el formato string cambió después de eso.

Sintaxis:

ds_map_read(id, str, [legacy]);

Argumento Tipo Descripción
id ID del mapa DS The id of the data structure to read the string into
str Cadena The string to read
legacy Booleano OPTIONAL Puede ser true o false u omitirse por completo.

Devuelve:

N/A

Ejemplo:

inventory = ds_map_create();
ini_open("map.ini");
var _t_string = ini_read_string("Saved", "0", "");
if (_t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

El código anterior crea un nuevo mapa DS y almacena su índice de identificación en la variable "inventario". Luego abre un archivo ini y lee un string de ese archivo en la variable temporal "t_string". Finalmente, comprueba que el string es válido (no el valor ini por defecto de "") y si lo es, lee el string en el mapa DS recién creado antes de cerrar el ini de nuevo.
