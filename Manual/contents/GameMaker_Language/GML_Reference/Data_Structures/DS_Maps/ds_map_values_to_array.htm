ds_map_values_to_array

Con esta función puede recuperar todos los valores que contiene un mapa DS para cada clave del mapa. Se suministra el ID del mapa DS para obtener las claves (tal y como lo devuelve ds_map_create()) y la función devolverá un array en el que cada entrada del array es un valor del mapa DS (por ejemplo, si el mapa tiene 10 claves, entonces se devolverá un array de 10 entradas con cada entrada correspondiente a un valor de una clave). La función tiene un segundo argumento opcional en el que puede suministrar un array que haya creado, en cuyo caso los datos del valor de la clave del mapa se añadirán a cualquier dato existente en el array. Tenga en cuenta que la función modificará la matriz suministrada directamente, pero también devolverá una referencia a ella (o una referencia a una nueva matriz si no se suministra ninguna).

Sintaxis:

ds_map_values_to_array(id, [array])

Argumento Tipo Descripción
id ID del mapa DS The id of the map to use.
[array] Matriz OPTIONAL La matriz a la que se añaden los valores del mapa DS.

Devuelve:

Array

Ejemplo:

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

Lo anterior recupera todos los valores de un mapa DS para cada clave del mapa y los almacena en una variable local como un array. A continuación, loops recorre la matriz y añade cada uno de los valores a una variable local que se utiliza para mostrar el total de todos los valores.
