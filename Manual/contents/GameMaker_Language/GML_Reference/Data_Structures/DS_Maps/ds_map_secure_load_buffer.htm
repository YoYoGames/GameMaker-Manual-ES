ds_map_secure_load_buffer

Esta función cargará un mapa DS guardado de forma segura desde un buffer. Debe haber cargado previamente el buffer en la memoria (usando buffer_load()) y luego pasarlo a esta función devolverá un mapa DS poblado con el contenido de buffer. Tenga en cuenta que el buffer debe haber sido creado usando la función ds_map_secure_save_buffer() para que esto funcione correctamente, y también tenga en cuenta que si el mapa DS que se está cargando contenía un array, éste se convertirá en una lista DS en su lugar en la carga.

¡IMPORTANTE! Una de las características de un archivo guardado de forma segura es que está bloqueado para el dispositivo en el que fue creado, por lo que no se puede cargar un archivo guardado en un dispositivo en un proyecto que se ejecuta en otro dispositivo.

NOTA: Esta función no es compatible con HTML5 y UWP.

Sintaxis:

ds_map_secure_load_buffer(buffer);

Argumento Tipo Descripción
buffer ID del búfer The buffer ID of the buffer to load the map data from.

Devuelve:

DS Map ID

Ejemplo:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

El código anterior cargará un mapa DS guardado de forma segura desde un buffer y almacenará su valor de índice en una variable para su uso futuro.
