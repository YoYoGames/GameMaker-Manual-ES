ds_map_add_list

Con esta función puedes asignar una lista DS (previamente creada) a una clave dentro del mapa DS dado. Esta función está diseñada para crear mapas compatibles con JSON que luego codificarás con json_encode() y sólo debe usarse junto con esa funcionalidad. Si un mapa DS tiene una lista añadida de esta manera, destruir el mapa padre también destruirá las listas contenidas y liberará su memoria, y llamar a ds_map_clear() en el mapa padre también destruirá y limpiará cualquier lista marcada.

Sintaxis:

ds_map_add_list(id, key, value)

Argumento Tipo Descripción
id ID del mapa DS The id of the map to use.
key Cadena The key for the added list.
value ID de la lista DS The id of the list to add.

Devuelve:

N/A

Ejemplo:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

El código anterior creará una lista y la rellenará con los distintos valores de las variables globales. Esta lista es entonces "anidada" dentro de un mapa DS, y el mapa es entonces codificado en un JSON string, antes de que el mapa sea destruido, eliminándolo, y cualquier lista que contenga, de la memoria.
