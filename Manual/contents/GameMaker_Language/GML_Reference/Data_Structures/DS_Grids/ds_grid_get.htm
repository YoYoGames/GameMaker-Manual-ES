ds_grid_get

Esta función se puede utilizar para obtener el valor de cualquier celda dentro de la cuadrícula DS dada. Si se pasan coordenadas de cuadrícula no válidas a la función, el valor devuelto será undefined y se mostrará un error en la ventana de salida.

Sintaxis:

ds_grid_get(index, x, y);

ArgumentoTipo Descripción
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

Devuelve:

Variable

Ejemplo:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

El código anterior selecciona una celda aleatoria de la cuadrícula DS indexada en la variable "grid" y almacena su valor en la variable "val".
