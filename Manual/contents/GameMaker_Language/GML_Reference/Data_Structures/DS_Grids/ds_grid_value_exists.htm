ds_grid_value_exists

Con esta función se puede comprobar si un valor específico (real o string) está presente dentro de un área rectangular de una cuadrícula DS dada. Si está presente la función devolverá true de lo contrario devolverá false.

Sintaxis:

ds_grid_value_exists(index, x1, y1, x2, y2, val);

ArgumentoTipo Descripción
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to find.

Devuelve:

Ejemplo:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
    xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
    ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

El código anterior comprueba una cuadrícula DS en busca de un valor específico dentro de una región rectangular. Si lo encuentra, entonces almacena la posición x e y del valor en dos variables para su uso posterior.
