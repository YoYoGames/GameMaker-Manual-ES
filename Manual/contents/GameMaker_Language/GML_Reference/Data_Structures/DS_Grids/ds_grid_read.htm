ds_grid_read

Esta función puede utilizarse para convertir una string que haya sido creada previamente por la función ds_grid_write() en una rejilla DS. La rejilla DS debe haber sido creada previamente (véase el ejemplo siguiente).

Tenga en cuenta que si el DS especificado string fue escrito por el GameMaker: Studio 1.2.x runtime (o anterior), debe especificar el argumento opcional "legacy", estableciéndolo en true ya que el formato string cambió después de eso.

Sintaxis:

ds_grid_read(index, string [, legacy]);

Argumento Tipo Descripción
index DS Grid ID The index of the grid to read.
string Cadena The string to read into the DS grid.
legacy Booleano OPTIONAL Puede ser true o false u omitirse por completo.

Devuelve:

N/A

Ejemplo:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

El código anterior crea una cuadrícula DS basada en el tamaño de la room (cada cuadrado de 32x32 píxeles representa una celda de la cuadrícula) y luego lee un conjunto de datos de la cuadrícula previamente guardados de un archivo ini en la nueva cuadrícula DS.
