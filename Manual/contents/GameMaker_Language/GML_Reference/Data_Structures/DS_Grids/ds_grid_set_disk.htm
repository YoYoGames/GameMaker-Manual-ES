ds_grid_set_disk

Con esta función puede fijar una región circular de una cuadrícula a un valor determinado. Debe proporcionar una celda de la rejilla inicial (como coordenadas de los ejes x e y), así como el radio del disco que debe fijarse y el valor al que desea fijar el disco, como se muestra en la siguiente ilustración:

Sintaxis:

ds_grid_set_disk(index, xm, ym, r, val);

ArgumentoTipo Descripción
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to set with the cells within the disk.

Devuelve:

Ejemplo:

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

El código anterior establecerá una región circular con un radio de 5 celdas en la cuadrícula DS indexada en la variable "grid" con un valor de -4.
