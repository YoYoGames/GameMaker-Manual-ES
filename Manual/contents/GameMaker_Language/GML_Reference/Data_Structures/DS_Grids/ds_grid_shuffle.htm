ds_grid_shuffle

Esta función puede utilizarse para aleatorizar las posiciones de todos los valores en todas las celdas de una cuadrícula.

NOTA Esta función barajará las celdas de la rejilla en las mismas posiciones cada vez que el juego se ejecute de nuevo debido a que GameMaker genera la misma semilla aleatoria inicial cada vez para hacer que el código de debugging sea una tarea mucho más fácil. Para evitar este comportamiento utilice randomise() al inicio de su juego. Esto sólo es cierto cuando las pruebas y debugging el juego, como el ejecutable final no mostrará este comportamiento y será aleatorio cada vez que se juegue.

Sintaxis:

ds_grid_shuffle(index);

ArgumentoTipo Descripción
index The index of the grid to shuffle.

Devuelve:

Ejemplo:

ds_grid_shuffle(grid)

Esto tomaría todos los valores de la cuadrícula DS indexados en la variable "grid" y los barajaría para obtener un orden diferente.
