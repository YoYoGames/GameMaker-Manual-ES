ds_grid_set_grid_region

Esta función puede utilizarse para copiar el contenido de un área rectangular de celdas de la rejilla de una rejilla DS (previamente definida) a otra, o puede utilizarse para copiar una región desde dentro de la misma rejilla. Las siguientes imágenes ilustran las dos formas en que esta función puede ser utilizada:

Sintaxis:

ds_grid_set_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

ArgumentoTipo Descripción
index The index of the destination grid.
source The index of the source grid.
x1 The left position of the region of cells to copy from the source grid.
y1 The top position of the region of cells to copy from the source grid.
x2 The right position of the region of cells to copy from the source grid.
y2 The bottom position of the region of cells to copy from the source grid.
xpos The x position on the destination grid to copy the source region to.
ypos The y position on the destination grid to copy the source region to.

Devuelve:

Ejemplo:

ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 10, 10)

Esto copiaría la región de celdas de (0,0) a (5,5) de la cuadrícula indexada en la variable "t_grid" y las copiaría en la posición (10,10) de la cuadrícula indexada en la variable "grid".
