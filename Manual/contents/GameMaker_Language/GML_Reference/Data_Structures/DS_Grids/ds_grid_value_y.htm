ds_grid_value_y

Con esta función puede obtener la coordenada y (dentro de la región de la cuadrícula rectangular dada) del valor que se busca. Se da el índice de la cuadrícula DS (devuelto por ds_grid_create()) junto con las posiciones x/y de las esquinas superior izquierda e inferior derecha de la región de la cuadrícula a buscar, antes de suministrar el valor a buscar. Si el valor buscado no existe, la función devolverá -1.

Sintaxis:

ds_grid_value_y(index, x1, y1, x2, y2, val);

ArgumentoTipo Descripción
index The index of the grid.
x1 The x position of the left of the region in the grid, from 0 to (grid width - 1).
y1 The y position of the top of the region in the grid, from 0 to (grid height - 1).
x2 The x position of the right of the region in the grid, from 0 to (grid width - 1).
y2 The y position of the bottom of the region in the grid, from 0 to (grid height - 1)
val The value to find.

Devuelve:

Ejemplo:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
    xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
    ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

El código anterior comprueba una cuadrícula ds_grid en busca de un valor específico dentro de una región rectangular. si se encuentra, entonces almacena la posición x e y del valor en dos variables para su uso posterior.
