ds_grid_get_disk_min

Esta función puede utilizarse para encontrar el valor mínimo de todas las celdas que se encuentran dentro del área de disco definida de una cuadrícula, como se muestra en la imagen siguiente:

Sintaxis:

ds_grid_get_disk_min(index, xm, ym, r);

ArgumentoTipo Descripción
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.

Devuelve:

Real or String

Ejemplo:

val = ds_grid_get_disk_min(grid, 5, 5, 2)

El código anterior establecerá la variable "val" al valor mínimo contenido en el disco dado de la rejilla DS indexada en la variable "grid".
