ds_grid_write

Esta función puede utilizarse para convertir el ds_grid dado en un string, que puede almacenarse en un archivo externo (por ejemplo). Puede leer el string devuelto por esta función de nuevo en un ds_grid utilizando la función ds_grid_read().

NOTA El string devuelto no es un string legible, sino un volcado del contenido de la estructura de datos.

Sintaxis:

ds_grid_write(index);

ArgumentoTipo Descripción
index The index of the grid to write.

Devuelve:

Ejemplo:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

El código anterior abrirá un archivo ini (creándolo si no existe ya) y luego escribirá el ds_grid dado como string en ese archivo.
