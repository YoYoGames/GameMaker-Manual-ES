ds_grid_add_disk

Esta función puede utilizarse para añadir un valor dado (real o string) a todos los valores de las celdas que se encuentran dentro del área de disco definida de una cuadrícula. El valor a añadir debe ser del mismo tipo que el que se encuentra en las celdas de la cuadrícula, es decir, no se puede añadir una cadena a un real o viceversa, y para strings esto corresponde a la concatenación.

Sintaxis:

ds_grid_add_disk(index, xm, ym, r, val);

ArgumentoTipo Descripción
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to add to the cells within the disk.

Devuelve:

Ejemplo:

ds_grid_add_disk(grid, 7, 6, 5, 2)

Esto sumaría 2 a todos los valores contenidos en las celdas dentro del área de disco definida de la cuadrícula DS referenciada por la variable "grid".
