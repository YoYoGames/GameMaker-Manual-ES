ds_grid_clear

Esta función se puede utilizar para borrar una cuadrícula DS dada a un valor específico. Todas las celdas de la cuadrícula contendrán entonces este valor, que puede ser un número real o un string. La imagen siguiente ilustra cómo funciona:

Sintaxis:

ds_grid_clear(index, val);

ArgumentoTipo Descripción
index This index of the grid to clear.
val The new value for all grid cells.

Devuelve:

Ejemplo:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

El código anterior redimensionará la cuadrícula DS indexada en la variable global "Grid" y luego la borrará para que cada celda tenga el valor -1.
