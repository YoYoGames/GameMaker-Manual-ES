ds_stack_write

Esta función devuelve un string que puede ser almacenado o transferido a otra estructura de datos utilizando la función ds_stack_read() función.

NOTA: El string devuelto no es un string legible, sino un volcado del contenido de la estructura de datos

Sintaxis:

ds_stack_write(id);

ArgumentoTipo Descripción
id The id of the data structure to write.

Devuelve:

Ejemplo:

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

El código anterior abre un archivo ini y luego escribe un string que contiene la información almacenada en la pila DS indexada en la variable "stack". A continuación, se borra la pila y se cierra el archivo ini.
