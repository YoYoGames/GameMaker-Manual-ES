ds_stack_destroy

Esta función eliminará de la memoria la estructura de datos de la pila dada, liberando los recursos que estaba utilizando y eliminando todos los valores que contenía. Esta función debe usarse siempre cuando se haya terminado de usar la pila de DS para evitar fugas de memoria que puedan ralentizar y bloquear el juego.

¡IMPORTANTE! Cuando se crea una estructura de datos, el valor del índice para identificarla es un valor entero que comienza en 0. Esto significa que data structures de diferentes tipos puede tener el mismo valor de índice, por lo que en caso de duda se debe utilizar la función ds_exists() antes de acceder a ellos. También hay que tener en cuenta que los índices se reutilizan, por lo que el valor del índice de una estructura de datos destruida puede ser utilizado por otra recién creada posteriormente, por lo que se recomienda establecer siempre la variable que contenía el índice DS a -1 después de destruirla.

Sintaxis:

ds_stack_destroy(id);

ArgumentoTipo Descripción
id The id of the data structure to remove.

Devuelve:

Ejemplo:

if (lives == 0)
{
    ds_stack_destroy(AI_stack);
    AI_stack = -1;
    room_goto(rm_Menu);
}

El código anterior comprobará el valor de la variable global incorporada "lives" y si es 0, destruye la pila DS indexada en la variable "AI_stack" y luego cambia rooms.
