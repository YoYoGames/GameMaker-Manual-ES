ds_stack_top

Esta función sólo leerá el primer valor de la pila (el que está "encima"). No hará saltar el valor, lo que significa que todavía puede ser leído en el futuro por esta función o por la función ds_stack_pop(). Si la pila está vacía, la función devolverá la constante undefined, de lo contrario devolverá el valor real o string contenido en la pila.

Sintaxis:

ds_stack_top(id);

ArgumentoTipo Descripción
id The id of the data structure to read from.

Devuelve:

Variable (Data type value that is stored in the stack) or undefined

Ejemplo:

num = ds_stack_top(control_stack);

El código anterior leerá el valor de la pila indexado en la variable "control_stack" y almacenará el valor de retorno en la variable "num".
