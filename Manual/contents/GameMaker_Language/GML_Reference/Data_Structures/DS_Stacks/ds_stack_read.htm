ds_stack_read

Con esta función puedes recrear una pila DS guardada (una que haya sido escrita previamente como string usando ds_stack_write()). Primero debes crear una nueva pila DS en la que leer el string, y si la pila DS ya existe y tiene información almacenada en ella, ésta se borrará antes de la lectura. Esta función es de vital importancia a la hora de crear mecanismos de guardado/carga para tu juego.

Tenga en cuenta que si el DS especificado string fue escrito por el GameMaker: Studio 1.2.x runtime (o anterior), debe especificar el argumento opcional "legacy", estableciéndolo en true ya que el formato string cambió después de eso.

Sintaxis:

ds_stack_read(id, str [, legacy]);

Argumento Tipo Descripción
id ID de pila DS The id of the data structure to read into.
str Cadena The string to read from.
legacy Booleano OPTIONAL Puede ser true o false u omitirse por completo.

Devuelve:

N/A

Ejemplo:

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
{
    ds_stack_read(stack, str);
}
ini_close();

El código anterior crea una pila y almacena el índice en la variable "pila". A continuación, abre un archivo ini y lee un string de él, comprobando que el string no se devuelva primero como vacío. Este string se lee entonces en la pila DS recién creada.
