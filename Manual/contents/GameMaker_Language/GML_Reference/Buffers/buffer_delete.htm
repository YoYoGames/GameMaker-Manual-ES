buffer_delete

Con esta función se puede borrar de la memoria un buffer creado previamente con la función buffer_create() de la memoria, liberando los recursos utilizados para crearlo y eliminando los datos que pueda contener actualmente.

NOTA: Es importante eliminar siempre los recursos creados dinámicamente de la memoria cuando ya no los necesite para evitar fugas de memoria.

¡IMPORTANTE! Cuando se crea un buffer, el valor del índice utilizado para identificarlo es un valor entero que comienza en 0. Estos índices son reutilizados por GameMaker, por lo que un valor de índice destruido de buffer puede ser utilizado por uno recién creado posteriormente, por lo que se recomienda establecer cualquier variable que contenga un índice de buffer a -1 después de haber destruido el buffer.

Sintaxis:

buffer_delete(buffer)

ArgumentoTipo Descripción
buffer The index of the buffer to delete.

Devuelve:

Ejemplo:

buffer_delete(player_buffer);
player_buffer = -1;

El código anterior borrará el buffer previamente creado con el id almacenado en la variable "player_buffer" y luego establece la variable a -1.
