buffer_async_group_option

Con esta función puede establecer algunas opciones específicas de la plataforma para el grupo buffer que se está guardando/cargando. Las opciones disponibles son las siguientes:

  • "subtítulo" o "slottitle" - El valor de esta opción sería un string, que se mostrará al usuario cuando gestione sus datos guardados en el SO. Esto sólo es importante cuando se guardan los datos, no cuando se cargan de nuevo.
  • "showdialog" - El valor de esta opción es un booleano true o false. Si se establece en true, cuando se guarda o carga GameMaker mostrará la interfaz de usuario del sistema, de lo contrario se hará un quicksave/quickload sin mostrar la interfaz de usuario. Normalmente sólo se necesita esto si usted está apoyando múltiples ranuras de guardado para permitir al usuario elegir una ranura, pero si sólo admite una ranura por usuario, establezca esto a false.
  • "savepadindex" - Para esta opción se especificaría el índice del pad del jugador que está guardando o cargando y el sistema escribirá datos en la carpeta de guardado de este jugador y los leerá.
  • "saveslotsize" - Esta opción requiere que se especifique el tamaño real en bytes que se quiere guardar (así se puede hacer un buffer_seek() y buffer_tell() para obtenerlo, por ejemplo). Tenga en cuenta que no es obligatorio suministrar este valor, ya que todos los guardados tienen preasignado un espacio mínimo, que suele variar con la plataforma de destino.
  • "ps_create_backup" - (Sólo para PlayStation) - Si se establece en true se asegura que los datos guardados se respaldan cuando se desmonta el directorio de guardado (es decir, cuando se llama a la función buffer_async_group_end() ). Consulta la documentación de la consola para obtener más información.

¡IMPORTANTE! Esta función sólo es válida actualmente para los módulos de destino PSVita, PS4 y XBox One. En todos los demás objetivos no hará nada.

Sintaxis:

buffer_async_group_option(option, value);

Argumento Tipo Descripción
option Cadena The option to set.
value Real o Cadena The value to set (can be a real or string, depending on the option).

Devuelve:

N/A

Ejemplo:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

El código anterior inicia un grupo buffer y luego establece las opciones del grupo antes de establecer 4 archivos para guardar de forma asíncrona. A continuación, se finaliza la definición del grupo (momento en el que comenzará el guardado).
