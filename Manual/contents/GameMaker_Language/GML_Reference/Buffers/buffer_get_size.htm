buffer_get_size

Con esta función se puede obtener el tamaño del buffer dado en bytes.

Sintaxis:

buffer_get_size(buffer);

Argumento Tipo Descripción
buffer ID del búfer The index of the buffer to get the size of.

Devuelve:

Real

Ejemplo:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

El código anterior creará un nuevo buffer y almacenará su índice en la variable local "_temp", siendo el tamaño de este nuevo buffer el mismo que el del buffer indexado en la variable "player_data".
