buffer_set_used_size

Esta función es principalmente para su uso dentro de extensions, y le permite establecer el tamaño "usado" del buffer dado, que es el número de bytes que se han escrito en él.

Cuando se escriben datos en un buffer desde un extension, GameMaker no sabe qué parte del buffer fue llenada por el código extension y no es capaz de leer esos datos. Esta función puede ser llamada por el extension para decirle al motor cuántos bytes de datos se escribieron en el buffer, para que el motor pueda leer esos datos.

Sintaxis:

buffer_set_used_size(buffer, size);

ArgumentoTipo Descripción
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

Devuelve:

Ejemplo:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Las funciones anteriores se llamarían en un extension para escribir algunos datos en un GameMaker buffer (a través de su dirección de memoria), y luego establecer el número de bytes que se escribieron en él para que el motor pueda leer esos datos.
