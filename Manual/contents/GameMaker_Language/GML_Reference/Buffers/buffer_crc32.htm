buffer_crc32

Esta función toma los datos de entrada de un buffer y devuelve una suma de comprobación crc32. Se especifica el ID de buffer del buffer que se va a utilizar, luego un valor de desplazamiento (en bytes) para donde comenzar, y luego un tamaño (de nuevo en bytes) para la región a ser hash, y la función devolverá un valor entero de 32 bits para esa región.

Sintaxis:

buffer_crc32(buffer, offset, size);

ArgumentoTipo Descripción
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Devuelve:

Ejemplo:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

El código anterior creará un valor hash de suma de comprobación crc32 para los datos completos almacenados en el buffer indexados por la variable "buff", y almacenará el valor hash entero devuelto en la variable "check_val".
