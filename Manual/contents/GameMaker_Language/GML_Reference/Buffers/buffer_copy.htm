búfer_copia

Esta función puede utilizarse para copiar un segmento (o todos) de los datos almacenados en un buffer a otro. Cuando se utilizan dos buffers y se copia de uno a otro, ambos buffers deben haber sido creados previamente utilizando la función buffer_create() (por ejemplo), y se puede especificar un desplazamiento de datos (en bytes) para el punto de inicio de los datos que se van a copiar desde el buffer de origen en relación con el inicio del buffer, así como otro desplazamiento de datos para definir la posición en la que se van a copiar los datos en en el destino buffer.

NOTA: No se puede copiar en el mismo buffer.

Sintaxis:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

ArgumentoTipo Descripción
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

Devuelve:

Ejemplo:

buffer_copy(buff1, 0, 2048, buff2, 2048);

El código anterior copiará los datos almacenados en el buffer indexado en la variable "buff1", y luego los pegará en el buffer indexado en la variable "buff2", pero desplazado en 2048 bytes.
