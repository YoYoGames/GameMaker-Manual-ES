relleno_del_buffer

Esta función puede utilizarse para rellenar un buffer previamente creado con un data type y un valor dados. Los datos con los que se rellena el buffer deben estar de acuerdo con el argumento "tipo" de esta función, lo que significa que no se puede intentar rellenar con un string y utilizar el tipo entero de 16 bits sin signo, por ejemplo. Las constantes de tipo son las mismas que las utilizadas por las funciones buffer_read() y buffer_write() . El "tamaño" es el tamaño del buffer (en bytes) que se desea rellenar, mientras que el "offset" es el valor de desplazamiento (también en bytes) desde el inicio del buffer para comenzar el relleno.

Sintaxis:

buffer_fill(buffer, offset, type, value, size);

ArgumentoTipo Descripción
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Devuelve:

Ejemplo:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

El código anterior encuentra el inicio de la buffer con el id almacenado en la variable "buff" y escribe en ella una serie de valores enteros de 16 bits con signo.
