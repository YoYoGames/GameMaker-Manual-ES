buffer_save_ext

Con esta función puede guardar parte del contenido de un buffer en un archivo, listo para ser leído de nuevo en la memoria mediante la función buffer_load() función. El "offset" define la posición inicial dentro del buffer para guardar (en bytes), y el "tamaño" es el tamaño del área de buffer que se guardará a partir de ese offset (también en bytes).

Sintaxis:

buffer_save_ext(buffer, filename, offset, size);

ArgumentoTipo Descripción
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Devuelve:

Ejemplo:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Guarda en un archivo parte del contenido actual de la página buffer con el id almacenado en la variable "buff".
