buffer_seek

Esta función se puede utilizar para moverse a través de un buffer, encontrando el inicio, el final, o una posición relativa a la que se utilizó por última vez al leer o escribir datos. El valor "offset" es el desplazamiento (en bytes) que se debe añadir a la posición de búsqueda dada, por ejemplo, si la base es relativa y el offset es 4, entonces la posición de buffer se moverá a lo largo de 4 bytes desde su posición actual. Tenga en cuenta lo siguiente:

  • Puede utilizar valores negativos para el desplazamiento para buscar en el buffer, así como valores positivos.
  • Si el buffer es del tipo "wrap" y se desplaza más allá del final del buffer, la posición de búsqueda también se desplaza.
  • Si el buffer no es del tipo "wrap", la búsqueda se sujetará al principio o al final del buffer, incluso cuando el desplazamiento lleve la búsqueda fuera de los límites de buffer.

Se aceptan las siguientes constantes como argumento "base" para buscar:

Constante de búsqueda en el búfer
Constante Descripción
buffer_seek_start El inicio del buffer
buffer_seek_relative Una posición relativa a la posición actual de lectura/escritura
buffer_seek_end El final del buffer

Sintaxis:

buffer_seek(buffer, base, offset);

Argumento Tipo Descripción
buffer ID del búfer The index of the buffer to use.
base Real The base position to seek.
offset Real The data offset value.

Devuelve:

N/A

Ejemplo:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

El código anterior encuentra el inicio de la buffer con el id almacenado en la variable "buff" y escribe en ella una serie de valores enteros de 16 bits con signo.
