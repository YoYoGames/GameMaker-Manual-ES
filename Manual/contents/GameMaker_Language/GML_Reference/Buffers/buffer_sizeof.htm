buffer_sizeof

Esta función devolverá el tamaño (en bytes) de la Constante de Buffer Data Type dada (listada aquí).

Sintaxis:

buffer_sizeof(type);

Argumento Tipo Descripción
type Tampón Data Type Constante The type of data that is to be checked (see the list of constants here).

Devuelve:

Real

Ejemplo:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

El código anterior calcula primero el tamaño del buffer a crear multiplicando el data type de 8 bits sin signo por 12 (ya que usaremos el buffer para contener 12 piezas de datos), y luego usa este valor para establecer un buffer fijo con el tamaño correcto.
