buffer_get_alignment

Con esta función puede obtener la alineación de un byte para el ID de buffer dado.

Sintaxis:

buffer_get_alignment(buffer);

ArgumentoTipo Descripción
buffer The index of the buffer to check.

Devuelve:

Ejemplo:

alignment = buffer_get_alignment(buff);

El código anterior obtendrá la alineación de la buffer a partir del valor indexado en la variable "buff" y la almacenará en una variable.
