búfer_copia_de_vértice_buffer

Esta función puede utilizarse para copiar algunos (o todos) los datos de vertex almacenados en un vertex buffer en un buffer regular previamente creado. Cuando se copia de un vertex buffer a un buffer regular con esta función, ambos buffers deben haber sido creados previamente (usando los botones vertex_create_buffer() y buffer_create() por ejemplo). Puede especificar el rango de datos de vertex que desea copiar en el buffer, donde el inicio vertex puede estar en cualquier lugar entre 0 y el número de vértices -1, y puede dar el número de vértices a partir de ese punto para copiar. Puedes utilizar la función vertex_get_number() en vertex buffer para obtener el número total de vértices almacenados. Por último, se da el índice de buffer para copiar los datos de vertex, así como un desplazamiento de datos para definir la posición a la que copiar los datos de vertex en el destino buffer.

Sintaxis:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argumento Tipo Descripción
vertex_buffer Vértice Buffer ID The index of the vertex buffer to copy from.
start_vertex Real The starting vertex.
num_vertices Real The total number of vertices to use.
dest_buffer ID del búfer The index of the buffer to copy to.
dest_offset Real The offset position to copy the data to (in bytes).

Devuelve:

N/A

Ejemplo:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

El código anterior copiará los datos de vertex almacenados en el vertex buffer indexado en la variable "model_buffer", y luego los pegará en el buffer indexado en la variable "player_buffer".
