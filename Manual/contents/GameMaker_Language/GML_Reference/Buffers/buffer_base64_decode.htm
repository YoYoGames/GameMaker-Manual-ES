buffer_base64_decode

Con esta función puede decodificar un string codificado en base64 (creado con la función buffer_base64_encode() ) en un buffer. Esta función creará el buffer (como un búfer de "crecimiento" alineado de 1 byte) y devolverá el índice único para el buffer, que deberá utilizarse en todas las llamadas posteriores a la función.

NOTA: Es importante que elimines de la memoria cualquier recurso creado dinámicamente como éste cuando ya no lo necesites para evitar fugas de memoria, así que cuando termines con el buffer que has creado debes liberarlo de nuevo usando buffer_delete().

Sintaxis:

buffer_base64_decode(string);

ArgumentoTipo Descripción
string The base64 encoded string to decode

Devuelve:

Buffer ID

Ejemplo:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

El código anterior abrirá un archivo ini y luego leerá un string de él en la función de decodificación. Esta función devolverá un índice buffer, que se almacena en la variable "buff", que contiene los datos previamente codificados y guardados. A continuación se cierra el archivo ini.
