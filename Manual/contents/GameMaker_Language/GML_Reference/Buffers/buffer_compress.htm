buffer_compress

Con esta función puede comprimir parte (o la totalidad) de un buffer utilizando la compresión zlib. Se suministra el ID del buffer a comprimir (devuelto por buffer_create()), el desplazamiento dentro de buffer a utilizar en bytes, y el tamaño de los datos de buffer a comprimir (también en bytes). La función devolverá un nuevo valor de ID de buffer para el buffer comprimido, o un valor menor que 0 si ha fallado por alguna razón. Esta función no alterará el buffer original.

Sintaxis:

buffer_compress(buffer, offset, size);

ArgumentoTipo Descripción
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

Devuelve:

Buffer ID

Ejemplo:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

El código anterior creará un buffer y lo rellenará con los datos de un string. Este buffer se comprime y se guarda, y tanto el origen como el buffers comprimido se eliminan
