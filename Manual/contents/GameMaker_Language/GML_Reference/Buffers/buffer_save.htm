guardar_buffer

Con esta función puede guardar el contenido de un buffer en un archivo, listo para ser leído de nuevo en la memoria mediante la función buffer_load() función.

NOTA: En HTML5 el contenido de buffer se guardará como strings codificado en base64 cuando se utilice esta función.

Sintaxis:

buffer_save(buffer, filename);

ArgumentoTipo Descripción
buffer The index of the buffer to save.
filename The name of the file to save as.

Devuelve:

Ejemplo:

buffer_save(buff, "Player_Save.sav");

Guarda en un archivo el contenido actual de la página buffer con el id almacenado en la variable "buff".
