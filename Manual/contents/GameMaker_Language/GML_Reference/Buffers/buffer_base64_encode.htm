buffer_base64_encode

Esta función convertirá los datos del buffer dado en un formato base64 codificado string. Este es un esquema de codificación comúnmente utilizado para cualquier medio que necesite ser almacenado o transferido a través de Internet como texto, y hace que la salida ilegible para el ojo humano. Para utilizarlo es necesario especificar un buffer ya creado, el valor de offset (que es el punto dentro del buffer en el que se desea empezar a codificar) así como el tamaño, en bytes, de la memoria buffer a codificar.

Sintaxis:

buffer_base64_encode(buffer, offset, size);

ArgumentoTipo Descripción
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Devuelve:

Ejemplo:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

El código anterior creará la codificación de los datos completos almacenados en el buffer indexado por la variable "buff", y almacenará el string devuelto en la variable local "b_str".
