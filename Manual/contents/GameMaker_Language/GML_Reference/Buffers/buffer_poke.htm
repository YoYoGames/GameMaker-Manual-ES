búfer_punto

Con la función buffer_write() se pueden escribir datos en el buffer en la posición actual de "búsqueda", y cada dato avanza esta posición en los bytes que se escriben o leen. Sin embargo, puede ser necesario que cambie un dato dado sin querer cambiar la posición de búsqueda actual, y es entonces cuando usaría esta función. Simplemente suministre a la función un índice buffer, y luego la posición de desplazamiento desde el inicio de buffer (en bytes) dentro de ese buffer para escribir, así como el data type y el valor a escribir.

Sintaxis:

buffer_poke(buffer, offset, type, value);

Argumento Tipo Descripción
buffer ID del búfer The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to write the given data to.
type Tampón Data Type Constante The type of data that is to be written to the buffer (see the list of constants here).
value Variable The data to add to the buffer, in accordance with the type specified.

Devuelve:

N/A

Ejemplo:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

El código anterior añadirá el valor del componente azul del color utilizado para la mezcla de la imagen en el buffer indexado en la variable "buff", en la tercera posición del buffer y como valor de 8 bits sin signo.
