buffer_peek

Con la función buffer_read() se pueden leer datos de la buffer dada en la posición de "búsqueda" actual, y cada pieza de datos se lee avanzando esta posición por los bytes que se leen o escriben. Sin embargo, puede ser necesario leer un dato determinado sin querer cambiar la posición de búsqueda actual, y es entonces cuando se utiliza esta función. Simplemente se suministra a la función un id de buffer, y luego una posición de desplazamiento (desde el inicio de buffer ) dentro de ese buffer para leer, así como el data type que se desea leer.

NOTA: ¡Usar el data type incorrecto para los datos que se están leyendo dará lugar a valores erróneos!

Si la función tiene éxito, se devolverá un valor (Real, string, booleano, etc...) dependiendo del data type, mientras que un fallo devolverá simplemente 0.

Sintaxis:

buffer_peek(buffer, offset, type);

Argumento Tipo Descripción
buffer ID del búfer The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to read the given data from.
type Tampón Data Type Constante The type of data that is to be read from the buffer (see the list of constants here).

Devuelve:

Real, Boolean, String or 0 (if it fails)

Ejemplo:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

El código anterior obtendrá tres valores de tres posiciones diferentes dentro de la buffer indexados en la variable "buff" y luego utilizará esos valores para establecer la mezcla de imágenes de la instancia.
