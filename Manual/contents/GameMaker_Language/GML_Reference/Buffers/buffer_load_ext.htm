buffer_load_ext

Esta función cargará los datos de buffer que fueron guardados previamente usando las funciones buffer_save() en un buffer existente. Se da el ID del buffer creado previamente para cargarlo, luego el archivo buffer guardado para cargarlo y, finalmente, el desplazamiento desde el inicio del buffer (en bytes) en el que se desea cargar los datos.

Por favor, lea la página buffer_load() para ver las notas específicas de la plataforma.

Sintaxis:

buffer_load_ext(buffer, filename, offset);

ArgumentoTipo Descripción
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

Devuelve:

Ejemplo:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

El código anterior obtendrá primero la posición del final del buffer indexado en la variable "player_buffer" y luego cargará los datos del dado en esa posición (tenga en cuenta que este ejemplo sólo funcionará con los tipos "grow" o "wrap" buffer ).
