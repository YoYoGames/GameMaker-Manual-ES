buffer_get_address

Con esta función puede obtener el puntero a la dirección buffer en bruto y alineada. Esto es principalmente para su uso con extensions ya que puede pasar este valor a través de ellos, permitiéndoles acceder a los datos de buffer. Tenga en cuenta que hay algunas cosas que no se pueden hacer con el buffer cuando se accede desde una extensión:

  • No se puede liberar la memoria.
  • No se puede cambiar el tamaño de la memoria.
  • No puede escribir más allá de la dirección buffer más el tamaño de buffer (puede utilizar la función buffer_get_size() para esto) o obtendrá un error de fuera de límites.

Sintaxis:

buffer_get_address(buffer);

ArgumentoTipo Descripción
buffer The index of the buffer to use.

Devuelve:

Ejemplo:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

El código anterior obtendrá la dirección de memoria del buffer indexado en la variable "buff_model" y luego obtendrá la dirección de memoria para el final del buffer usando la función buffer_get_size(), almacenando ambos valores en variables locales para para su uso posterior.
