buffer_sha1

En criptografía, SHA-1 es una función de hash criptográfico diseñada por la Agencia de Seguridad Nacional de los Estados Unidos y se emplea en varias aplicaciones y protocolos ampliamente utilizados, como el popular Git, donde se utiliza para comprobar los cambios en los archivos. Esta función toma los datos de entrada de un buffer y devuelve un compendio de mensajes de 160 bits en formato ASCII. De esta manera se puede generar una clave segura que puede ser almacenada y utilizada para comprobar la integridad de la información que se envía a (o se recibe de) un server externo (por ejemplo). Al aplicar esto a buffers utilizando esta función, debe especificar el id de buffer del buffer a utilizar, luego un valor de desplazamiento (en bytes) para donde comenzar, y luego un tamaño (de nuevo en bytes) para la región a la que se aplicará el hash.

NOTA: SHA-1 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

buffer_sha1(buffer, offset, size);

ArgumentoTipo Descripción
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Devuelve:

Ejemplo:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

El código anterior creará un hash sha1 para los datos completos almacenados en buffer indexados por la variable "buff", y almacenará el hash devuelto en la variable "check_string".
