redimensionamiento_del_buffer

Con esta función puede redimensionar un determinado buffer para que tenga el tamaño (en bytes) que usted especifique.

Sintaxis:

buffer_resize(buffer, newsize);

ArgumentoTipo Descripción
buffer The index of the buffer to change the size of.
newsize The new size of the buffer (in bytes).

Devuelve:

Ejemplo:

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

El código anterior comprobará el tamaño del buffer indexado en la variable "buff" y si es menor que el valor dado, se redimensiona el buffer.
