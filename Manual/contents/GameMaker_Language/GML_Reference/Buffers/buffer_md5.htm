buffer_md5

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se utiliza habitualmente para comprobar la integridad de los datos. Esta función tomará los datos de entrada de un determinado buffer (o parte del buffer) y devolverá el hash MD5 de 32 caracteres hexadecimales que es único para esos datos. De esta manera se puede generar una clave segura que se puede almacenar y utilizar para comprobar la integridad de la información que se envía a (o se recibe de) un server externo (por ejemplo). Al aplicar esto a buffers utilizando esta función debe especificar el id de buffer del buffer a utilizar, luego un valor de desplazamiento (en bytes) para saber dónde comenzar, y luego un tamaño (de nuevo en bytes) para la región que se va a hashear.

NOTA: MD5 no es completamente seguro y puede romperse. Consulte esta página para obtener más información.

Sintaxis:

buffer_md5(buffer, offset, size);

ArgumentoTipo Descripción
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Devuelve:

Ejemplo:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

El código anterior creará un hash md5 para los datos completos almacenados en buffer indexados por la variable "buff", y almacenará el hash devuelto en la variable "check_string".
