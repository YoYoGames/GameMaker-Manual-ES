buffer_base64_decode_ext

Con esta función puedes decodificar un string codificado en base64 (creado con la función buffer_base64_encode() en un buffer. A diferencia de la función buffer_base64_decode(), esta función no creará un buffer por usted, sino que ya debería haber creado el buffer (ver buffer_create()), cuyo identificador deberá utilizar con esta función. El "offset" es la posición dentro de buffer para decodificar el string dado (en bytes).

Sintaxis:

buffer_base64_decode_ext(buffer, string, offset);

ArgumentoTipo Descripción
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

Devuelve:

Ejemplo:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

El código anterior creará un buffer y almacenará el id único para él en la variable "buff", luego abrirá un archivo ini y leerá un string de él en la variable local "str". Este string es entonces decodificado en el recién creado buffer antes de cerrar el archivo ini de nuevo.
