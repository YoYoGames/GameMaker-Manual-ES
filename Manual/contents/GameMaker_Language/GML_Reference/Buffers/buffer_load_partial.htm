buffer_load_partial

Esta función cargará algunos de los datos de buffer que se guardaron previamente utilizando las funciones buffer_save() en un buffer existente. Se da el id del buffer previamente creado para cargarlo, luego el archivo buffer guardado para cargarlo, y luego el desplazamiento desde el inicio del buffer (en bytes) desde el que se desea cargar los datos. Los siguientes argumentos sirven para establecer la longitud de los datos de buffer (en bytes) desde el punto de desplazamiento inicial que se desea cargar y el punto de desplazamiento al que cargar los datos en el buffer (de nuevo, en bytes).

Por favor, lea la página buffer_load() para ver las notas específicas de la plataforma.

Sintaxis:

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

ArgumentoTipo Descripción
buffer The index of the buffer to load into (destination).
filename The name of the file to load from (source).
offset The offset within the destination buffer to load to (in bytes).
src_len The length of the part of the source buffer to load (in bytes).
dest_offset The offset where to start putting the partial data in the destination buffer (in bytes).

Devuelve:

N/A

Ejemplo:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

El código anterior creará un nuevo "grow" buffer y luego cargará en él una parte de los datos guardados en el archivo "save.dat".
