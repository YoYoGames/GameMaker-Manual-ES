buffer_decompress

Con esta función puede descomprimir un buffer previamente comprimido utilizando la compresión zlib. Usted suministra el ID de buffer para descomprimir, y la función devolverá un nuevo ID de buffer que contiene los datos sin comprimir. Si la descompresión ha fallado (por ejemplo, está suministrando un buffer que no ha sido comprimido) entonces la función devolverá un valor menor que 0.

Sintaxis:

buffer_decompress(buffer);

ArgumentoTipo Descripción
buffer The index of the buffer to decompress.

Devuelve:

Buffer ID

Ejemplo:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

El código anterior cargará primero un buffer guardado, luego lo descomprimirá y finalmente leerá los datos de string del buffer descomprimido en una variable global.
