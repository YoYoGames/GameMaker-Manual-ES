network_create_socket

Esta función se utiliza para crear un nuevo socket de cliente para que su juego se comunique a través de la red. Debes definir el tipo de socket (ver la lista de constantes más abajo) y la función devolverá un id único para ese socket, que debe ser usado en todas las llamadas posteriores a la función para ese socket, o un valor menor que 0 si la conexión falla.

SUGERENCIA Puedes utilizar la IP "127.0.0.1" para conectarte de nuevo al mismo dispositivo que está ejecutando el juego.

Sintaxis:

network_create_socket(type);

Argumento Tipo Descripción
type Tipo de enchufe Constante The type of socket connection to create (see the constants listed above).

Devuelve:

Network Socket ID

Ejemplo:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

El código anterior comprobará si el juego se está ejecutando en un navegador o no y creará un nuevo socket TCP o Web antes de intentar conectarse a través de él a la dirección IP dada en el puerto dado.
