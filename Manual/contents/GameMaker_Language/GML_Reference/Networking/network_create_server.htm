network_create_server

Esta función se utiliza para crear una nueva red server para su juego, utilizando uno de los protocolos de conexión permitidos (vea las constantes listadas abajo). Usted suministra el tipo de server, luego le da un puerto a utilizar, y finalmente el número de conexiones máximas que deben ser permitidas en cualquier momento a la server (tenga en cuenta que este número depende de usted, pero demasiados clientes conectados saturarán la red o la CPU del dispositivo no será capaz de manejar el procesamiento de ese número de jugadores, así que úselo con cuidado). La función devuelve un identificador único que debe ser usado almacenado en una variable y utilizado para identificar el server en todas las funciones de red posteriores, o un valor menor que 0 si la conexión falla.

Sintaxis:

network_create_server(type, port, max_client);

Argumento Tipo Descripción
type Tipo de enchufe Constante The type of server to create (see the constants listed above).
port Real The port that the server will use.
max_client Real The maximum number of clients that can connect at once.

Devuelve:

Network Socket ID

Ejemplo:

var port = 6510;
server = network_create_server(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

El código anterior intentará crear un server usando TCP a través del puerto 6510. Si ese puerto no está disponible, entonces loop a través de los puertos para encontrar uno que sí lo esté.
