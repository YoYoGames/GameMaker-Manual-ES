network_send_packet

Con esta función puedes enviar un "paquete" de datos a través de la red. La función toma el id del socket para conectarse y luego debe suministrar el id del buffer que contiene los datos a ser enviados (para más información sobre buffers ver Referencia - Buffers) y finalmente el tamaño (en bytes) del paquete de datos. Los paquetes enviados con esta función están formateados de tal manera que el juego GameMaker que recibe los datos puede "dividir" los paquetes correctamente, y la función devolverá el número de bytes de datos enviados, o un número menor que 0 si el envío ha fallado. Cabe destacar que el tamaño final de los datos enviados que devuelve esta función también incluirá la información de la cabecera GameMaker, que son 12 bytes adicionales.

Sintaxis:

network_send_packet(socket, bufferid, size);

ArgumentoTipo Descripción
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Devuelve:

Real

Ejemplo:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

La información anterior carga un dato de buffer previamente guardado en memoria y devuelve el id de buffer que se almacena en la variable "buff". Este buffer completo se envía entonces como un paquete a través de la red utilizando el socket identificado por la variable "sock".
