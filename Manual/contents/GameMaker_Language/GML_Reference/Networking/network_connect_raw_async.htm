network_connect_raw_async

Con esta función puedes enviar una petición para conectarte a un server. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que le des la dirección IP a la que conectarte a (un string) así como el puerto a través del cual conectarse, y si la conexión falla se devolverá un valor inferior a 0. La diferencia entre esta función y network_connect_async()es que esta función puede conectarse a cualquier server y no hace nada con los datos en bruto, lo que significa que tienes que implementar los protocolos tú mismo en el extremo de server. Tenga en cuenta que esta función es asíncrona, generando un evento de red asíncrono del tipo network_type_non_blocking_connect.

Sintaxis:

network_connect_raw_async(socket, url, port);

ArgumentoTipo Descripción
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Devuelve:

Ejemplo:

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

El código anterior creará un nuevo socket TCP y luego intentará crear una conexión asíncrona "cruda" a través de él a la URL dada en el puerto 6510.
