network_connect_async

Con esta función puedes enviar una petición para conectarte a un server. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que le des la dirección IP a la que conectarte a (un string) así como el puerto a conectar, y si la conexión falla se devolverá un valor inferior a 0. La conexión utiliza un protocolo especial que asegura que sólo los juegos de GameMaker se conectan entre sí, sin embargo, si usted necesita conectarse a un server que no es un juego de GameMaker, puede utilizar network_connect_async_raw(). Tenga en cuenta que esta función es asíncrona, generando un evento de red asíncrono del tipo network_type_non_blocking_connect.

Sintaxis:

network_connect_async(socket, url, port);

ArgumentoTipo Descripción
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Devuelve:

Ejemplo:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

El código anterior creará un nuevo socket TCP y luego intentará hacer una conexión asíncrona a través de él a la dirección IP dada en el puerto 6510.
