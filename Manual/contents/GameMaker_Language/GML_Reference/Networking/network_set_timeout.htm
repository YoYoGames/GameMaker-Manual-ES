network_set_timeout

Con esta función puedes establecer el tiempo de espera para leer y escribir datos a/desde un server a través del socket dado. Ten en cuenta que el tiempo de espera no genera ningún tipo de evento, por lo que tendrás que ocuparte tú mismo de los tiempos de espera utilizando alarmas (por ejemplo). Tenga en cuenta que este valor sólo afecta al envío y recepción de datos, y si desea cambiar el valor del tiempo de espera de la conexión, deberá utilizar la función network_set_config().

Sintaxis:

network_set_timeout(socket, read_timeout, write_timeout);

ArgumentoTipo Descripción
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

Devuelve:

Ejemplo:

network_set_timeout(sock, 3000, 3000);

El código anterior establecerá el tiempo de espera para leer y escribir datos a través del socket indexado en la variable "sock" a 3 segundos.
