network_create_socket_ext

Esta función se utiliza para crear un nuevo socket de cliente para que su juego se comunique a través de la red. Debes definir el tipo de socket (ver la lista de constantes más abajo) y dar un puerto a utilizar, y la función devolverá un id único que debe ser usado en todas las llamadas posteriores a la función para ese socket, o un valor menor a 0 si la conexión falla.

Sintaxis:

network_create_socket_ext(protocol, port);

Argumento Tipo Descripción
protocol Tipo de enchufe Constante The network protocol to use
port Real The port to use

Devuelve:

Network Socket ID

Ejemplo:

client = network_create_socket_ext(network_socket_udp, 6510);

El código anterior creará un nuevo socket UDP en el puerto 6510.
