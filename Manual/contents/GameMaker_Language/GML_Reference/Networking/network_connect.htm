conexión_de_red

Con esta función se puede enviar una petición para conectarse a un server. La función toma el id del socket para conectarse (ver network_create_socket()) y requiere que le des la dirección IP a la que conectarse (un string) así como el puerto a través del cual conectarse, y si la conexión falla se devolverá un valor inferior a 0. La conexión utiliza un protocolo especial que asegura que sólo los juegos de GameMaker se conectan entre sí, sin embargo, si usted necesita conectarse a un server que no es un juego de GameMaker, puede utilizar network_connect_raw(). Tenga en cuenta que por defecto la función es sincrónica, lo que significa que su juego puede parecer que se "cuelga" mientras se realiza la conexión. Puede establecer un valor de tiempo de espera para la conexión, o alternativamente hacer la creación asíncrona, utilizando la función network_set_config(), o alternativamente utilizar la función network_connect_async() en su lugar.

NOTA No se puede utilizar esta función en HTML5. Para WebSockets, utilice la función Async.

Sintaxis:

network_connect(socket, url, port);

Argumento Tipo Descripción
socket ID de la toma de red The id of the socket to use.
url Cadena The URL or IP to connect to (a string).
port Real The port to connect to.

Devuelve:

Real or Network Socket ID

Ejemplo:

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

El código anterior creará un nuevo socket TCP e intentará conectarse a través de él a la dirección IP indicada en el puerto 6510.
