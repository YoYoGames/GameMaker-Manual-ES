time_source_get_children

Para la Fuente de Tiempo dada, esta función devuelve un array que contiene los ID de la fuente de tiempode sus hijos.

Sintaxis:

time_source_get_children(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para que los hijos de

Devuelve:

Array of Time Source ID

Ejemplo:

var _children = time_source_get_children(time_source);
var _count = array_length(_children);

for (var i = 0; i < _count; i ++)
{
    time_source_destroy(_children[i]);
}

time_source_destroy(time_source);

Este código recorre todos los hijos de una Fuente del Tiempo y los destruye uno a uno.

Al final, destruye la Fuente del Tiempo matriz.
