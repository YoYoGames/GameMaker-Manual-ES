time_source_reconfigure

Esta función se utiliza para modificar las propiedades principales de una Fuente de Tiempo, sin tener que crear una completamente nueva.

Se especifica una Fuente de Tiempo existente, y luego se establecen las propiedades que también se establecen en time_source_create()excepto el padre. Lea esa página para obtener información detallada sobre estas propiedades.

La Fuente de Tiempo especificada se restablecerá y desactivará, y tendrá que ser iniciada de nuevo.

Sintaxis:

time_source_reconfigure(id, period, units, callback, [args, repetitions, expiry_type]);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para reconfigurar
period Real The period that the Time Source se ejecuta para, en las unidades dadas
units Fuente de Tiempo Unidad Constante The units that the given period is in
callback Método The method to call when the Time Source expira
args Matriz OPTIONAL Un array que contiene los argumentos a pasar al método
repetitions Real OPTIONAL El número de veces que la fuente de tiempo debe repetirse, o -1 para una repetición indefinida
expiry_type Constante de caducidad de la fuente de tiempo OPTIONAL Si la fuente de tiempo expira en la trama más cercana a su vencimiento, o en la siguiente trama

Devuelve:

N/A

Ejemplo:

function change_spawn_delay(new_delay)
{
    time_source_reconfigure(obj_game.spawn_time_source, new_delay, time_source_units_frames, obj_game.spawn_method, [], -1);
    time_source_start(obj_game.spawn_time_source);
}

Esto crea una nueva función que cambia el retardo de aparición de los enemigos en el juego.

Asumiendo que el juego usa una Fuente de Tiempo llamada obj_game.spawn_time_source para desovar enemigos, esa Fuente de Tiempo necesitará ser actualizada una vez que el retraso de desovar cambie.

Esta función hace exactamente eso, reconfigurando la Fuente de Tiempo con el nuevo retardo y luego iniciándola de nuevo.
