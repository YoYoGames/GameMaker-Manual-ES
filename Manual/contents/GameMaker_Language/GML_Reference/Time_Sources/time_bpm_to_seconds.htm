tiempo_bpm_a_segundos

Esta función toma un valor de pulsaciones por minuto y devuelve la duración de cada pulsación en segundos.

Esto puede ser usado cuando se crea una Fuente de Tiempo para usar un valor BPM para la Fuente de Tiempo. Es importante que dicha Fuente de Tiempo utilice segundos como unidad.

Sintaxis:

time_bpm_to_seconds(bpm);

Argumento Tipo Descripción
bpm Real The beats-per-minute value to convert into seconds-per-beat

Devuelve:

Real

Ejemplo:

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

Este código convierte un valor de 90 BPM en segundos, y lo utiliza para crear una Fuente de Tiempo que se ejecuta indefinidamente.

En cada latido, imprime el mensaje "¡Beat!" en el registro de salida.
