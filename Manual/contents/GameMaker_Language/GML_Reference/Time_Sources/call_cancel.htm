llamar_cancelar

Esta función cancela una función de devolución de llamada que fue iniciada anteriormente por call_later.

Sintaxis:

call_cancel(handle);

Argumento Tipo Descripción
handle ID de la fuente de tiempo The handle to a Time Source devuelto por call_later.

Devuelve:

N/A

Ejemplo:

call_cancel(handle);

Volver: call_later
Siguiente: time_source_create
