inicio_de_la_fuente_de_tiempo

Esta función inicia la Fuente de Tiempo dada, cambiando su estado a time_source_state_active.

La Fuente de Tiempo especificada puede ser una nueva Fuente de Tiempo que nunca fue iniciada, fue pausada, detenida o expiró sin repeticiones.

Esta función "reiniciará" la Fuente de Tiempo dada, restableciendo su tiempo de expiración y las repeticiones restantes a los valores que fueron configurados para ella.

Sintaxis:

time_source_start(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para empezar

Devuelve:

N/A

Ejemplo:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

En este ejemplo, queremos que la instancia se destruya a sí misma 300 fotogramas después.

El código crea primero un método que simplemente llama a instance_destroy().

Luego crea una Fuente de Tiempo, heredando de la Fuente de Tiempo del Juego. Establece su periodo a 300 fotogramas.

Por último, pone en marcha la Fuente del Tiempo.
