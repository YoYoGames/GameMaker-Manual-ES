Puede utilizar la Fuente de Tiempo Global o la Fuente de Tiempo del Juego como padre para su Fuente de Tiempo personalizada. Ambas fuentesde tiempo están disponibles globalmente.

La Fuente de Tiempo Global se ejecuta fuera del juego principal loop, mientras que la Fuente de Tiempo del Juego se ejecuta como parte del juego loop.

La elección entre Global/Juego no afecta a si las Fuentes de Tiempose ven afectadas por el framerate de tu juego; eso dependerá de la unidad que utilices.

Pida

Lasfuentes de tiempoque heredan de la fuente de tiempo global se procesan antes que la fuente de tiempo del juego.

Estados

La fuente de tiempo de juego tiene estados, lo que significa que puede ser paused. La Fuente de Tiempo Global no tiene estados y no puede ser pausada.

Puedes utilizar la Fuente de Tiempo Global para los temporizadores que deben ejecutarse independientemente del estado del juego, y la Fuente de Tiempo del Juego para los temporizadores que están vinculados a tu juego principal loop. Puedes entonces pausar todos tus temporizadores relacionados con el juego pausando la Fuente de Tiempo del Juego.
