time_source_get_time_remaining

Esta función devuelve el tiempo restante hasta que la Fuente de Tiempo dada expira y ejecuta su método de devolución de llamada.

Este valor de tiempo se expresará en las unidades establecidas para la Fuente de Tiempo (segundos o fotogramas).

Sintaxis:

time_source_get_time_remaining(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para obtener información para

Devuelve:

Real
