reanudación_de_la_fuente_de_tiempo

Esta función reanuda la Fuente de Tiempo dada y sus hijos. La Fuente de Tiempo dada debe haber sido pausada previamente usando time_source_pause().

Sintaxis:

time_source_resume(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para reanudar

Devuelve:

N/A

Ejemplo:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_resume(time_source);
    }
}

Cuando se pulsa la tecla Espacio, este código obtendrá el estado de la Fuente de Tiempo.

Cuando el estado esté activo, pondrá en pausa la Fuente de Tiempo, y cuando esté en pausa, la reanudará.
