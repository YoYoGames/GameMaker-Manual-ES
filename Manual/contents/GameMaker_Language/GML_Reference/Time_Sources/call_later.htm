llamar_más tarde

Esta función activa una función de devolución de llamada después del tiempo especificado o después de un cierto intervalo. Para ello, crea una fuente de tiempo (véase Fuentes de tiempo).

Para cancelar la función de devolución de llamada, véase call_cancel.

IMPORTANTE Aunque esta función devuelve un ID de fuente de tiempo real, este ID no puede utilizarse con las funciones de fuente de tiempo, ya que las fuentes de tiempo utilizadas por call_later y call_cancel son "invisibles" - no aparecerán como hijos de ninguna fuente de tiempo incorporada y no pueden utilizarse con otras funciones de fuente de tiempo. Sin embargo, se incluyen en el recuento de fuentes de tiempo devuelto por debug_event("ResourceCounts").

Argumentos

A continuación se detallan los argumentos de esta función:

Periodo

Se trata de un periodo de tiempo que puede expresarse en segundos o en fotogramas, dependiendo de la unidad especificada en el siguiente argumento.

Cuando se utilizan fotogramas como unidad, el periodo debe ser un número entero. Los valores que no son enteros se redondean hacia abajo (por defecto), con la excepción de los valores inferiores a 1, que se redondean a 1.

Unidad

Es la unidad en la que se expresa el periodo, y puede ser time_source_units_seconds o time_source_units_frames.

Ver: Unidades de la fuente del tiempo

Puede utilizar un valor de pulsaciones por minuto utilizando time_bpm_to_seconds().

Llamada de retorno

Debe especificar una "devolución de llamada" method que se llama cuando la Fuente de Tiempo expira.

Bucle

Si la fuente de tiempo debe loop.

Sintaxis:

call_later(period, unit, callback, [loop=false]);

Argumento Tipo Descripción
period Real The period that the Time Source se ejecuta para, en las unidades dadas
units Fuente de tiempo Unidad Constante The units that the given period is in
callback Método o Función de script The method or script function to call after the specified time
loop Booleano OPTIONAL Si se hace un bucle con la fuente de tiempo (por defecto es falso)

Devuelve:

Time Source ID

Ejemplo 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

En este ejemplo, creamos una función de devolución de llamada que muestra un mensaje de depuración después de que hayan pasado cinco segundos. Después de eso, la fuente de tiempo subyacente se destruye.

Ejemplo 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

En este ejemplo, creamos una función de devolución de llamada que crea un nuevo enemigo en una posición aleatoria en el room. Esta función se ejecuta cada 10 segundos pasándola a call_later. El último argumento se establece como true para que la función siga siendo llamada cada 10 segundos.

Utilice call_cancel con el manejador devuelto por call_later para detener la Fuente de Tiempo y las llamadas a la devolución de llamada.
