time_source_reset

Esta función restablece la Fuente de Tiempo dada, reiniciando su temporizador e información de repetición.

Después de llamar a esta función, tendrá que iniciar de nuevo la Fuente de Tiempo para que se inicie la cuenta atrás.

Sintaxis:

time_source_reset(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para restablecer

Devuelve:

N/A

Ejemplo:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

Este código reiniciará la Fuente de Tiempo dada cada vez que se inicie un nuevo room, y luego iniciará la Fuente de Tiempo de nuevo.
