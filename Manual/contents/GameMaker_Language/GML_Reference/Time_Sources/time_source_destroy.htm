time_source_destroy

Esta función destruirá la Fuente de Tiempo dada.

Opcionalmente puede especificar si todas las Fuentes de Tiempo hijas deben ser eliminadas también.

Sintaxis:

time_source_destroy(id, [destroy_tree]);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para destruir
destroy_tree Booleano OPTIONAL Si se destruyen también las Fuentes de Tiempo hijas, es falso por defecto

Devuelve:

N/A

Ejemplo:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

El código del evento Create crea una nueva Fuente de Tiempo que expira una vez por segundo y se repite indefinidamente.

El código del evento Clean Up destruye esa Fuente de Tiempo, ya que no será necesaria después de que la instancia sea destruida.
