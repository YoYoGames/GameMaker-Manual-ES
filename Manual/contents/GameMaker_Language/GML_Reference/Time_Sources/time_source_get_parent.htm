time_source_get_parent

Esta función devuelve el ID de la fuente de tiempo del padre de la Fuente de Tiempo dada.

El valor devuelto puede ser el ID de una Fuente de Tiempo personalizada, o una de las Fuentes de Tiempo incorporadas, dependiendo de cómo se haya creado la Fuente de Tiempo dada.

Sintaxis:

time_source_get_parent(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para obtener el padre de

Devuelve:

Time Source ID

Ejemplo:

var _parent = time_source_get_parent(time_source);

Este código obtiene el padre de una Fuente de Tiempo y lo almacena en una variable local.
