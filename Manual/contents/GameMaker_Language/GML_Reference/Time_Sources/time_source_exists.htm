fuente_de_tiempo_existe

Esta función devuelve si el ID dado es una fuente de tiempo válida.

Si la Fuente de Tiempo fue destruida, esto devolverá falso, sin embargo si la Fuente de Tiempo fue detenida o simplemente expiró, continuará existiendo, lo que significa que esta función devuelve verdadero.

Sintaxis:

time_source_exists(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para comprobar

Devuelve:

Boolean

Ejemplo:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

Este código comprueba si existe una Fuente de Tiempo. Si existe, comprueba si se ha pulsado la tecla Espacio.

En ese caso, destruye la Fuente del Tiempo.
