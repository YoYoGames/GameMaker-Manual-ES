tiempo_fuente_parada

Esta función detiene la Fuente de Tiempo dada y reinicia su temporizador. No se puede detener una Fuente de Tiempo que no se ha iniciado, o que está en un estado finalizado.

Sintaxis:

time_source_stop(id);

Argumento Tipo Descripción
id ID de la fuente de tiempo The Time Source para detener

Devuelve:

N/A

Ejemplo:

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

El código anterior detendrá una Fuente de Tiempo cuando no esté ya detenida.
