Fuentes de tiempo

Una "fuente de tiempo" es un temporizador personalizado que usted crea. Se ejecuta durante un período de tiempo determinado, y al final, expira.

Una Fuente de Tiempo (TS) tiene la capacidad de llamar a un método personalizado ("callback") al expirar.

También es capaz de repetirse una cierta cantidad de veces, o para siempre. Cada repetición de una fuente de tiempo se conoce como "repetición".

¿Cómo utilizar las fuentes de tiempo?

Crear e iniciar una Fuente de Tiempo que llame a un método personalizado:

  • Llame a time_source_create() / Crear fuente de tiempo para crear una fuente de tiempo
    • Especifique si este inherits desde la Fuente de Tiempo Global, la Fuente de Tiempo del Juego o una Fuente de Tiempo personalizada
    • Especifique el período de tiempo que se ejecuta, y las unidades utilizadas (segundos o cuadros)
    • Especifique un método personalizado o una función script, con una matriz de argumentos para pasar a ese método
    • Opcionalmente, especifique el número de veces que debe repetirse
      • Por defecto sólo se ejecuta una vez
  • Llame a time_source_start() / Iniciar fuente de tiempo para iniciar esa fuente de tiempo inmediatamente, o llamarla más tarde cuando necesite que se inicie

Esto hará que la Fuente de Tiempo se ejecute, y después del período especificado, expirará y llamará a su método especificado.

Fuentes de tiempo incorporadas

Una Fuente de Tiempo puede inherit desde cualquiera de las dos Fuentes de Tiempoincorporadas, o desde una Fuente de Tiempo personalizada.

Temporizadores simples

Aunque la configuración de una fuente de tiempo no requiere muchas líneas de código, puede ser conveniente configurar y gestionar rápidamente un temporizador utilizando una sola función.

Las siguientes dos funciones proporcionan una forma más sencilla y abreviada de crear una Fuente de Tiempo y detenerla:

  • llamar_más tarde
  • llamar_cancelar

NOTA No se pueden utilizar las funciones de la fuente de tiempo con las identificaciones de la fuente de tiempo utilizadas por estas funciones. Consulte call_later para obtener más información.

Lista de funciones

Las siguientes funciones se utilizan para crear y modificar las Fuentes de Tiempo:

  • time_source_create
  • time_source_destroy
  • inicio_de_la_fuente_de_tiempo
  • tiempo_fuente_parada
  • pausa_de_tiempo
  • reanudación_de_la_fuente_de_tiempo
  • time_source_reconfigure
  • time_source_reset
  • time_source_get_children
  • time_source_get_parent
  • time_source_get_period
  • time_source_get_reps_completed
  • time_source_get_reps_remaining
  • time_source_get_state
  • time_source_get_time_remaining
  • time_source_get_units
  • fuente_de_tiempo_existe

Cada función get_* devolverá undefined si la Fuente de Tiempo especificada no existe.

Las siguientes funciones de ayuda se proporcionan para convertir períodos de tiempo:

  • tiempo_segundos_a_bpm
  • tiempo_bpm_a_segundos
