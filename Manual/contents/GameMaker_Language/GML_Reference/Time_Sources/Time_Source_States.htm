Estados de la fuente del tiempo

Constante de estado de la fuente de tiempo
Constante Descripción Valor
time_source_state_initial La Fuente del Tiempo aún no se ha puesto en marcha 0
time_source_state_active La Fuente del Tiempo se ha puesto en marcha y está en cuenta atrás 1
time_source_state_paused La fuente del tiempo es paused 2
time_source_state_stopped La Fuente de Tiempo se detuvo o expiró completamente 3

Una fuente de tiempo puede tener un estado, que puede ser cualquiera de las constantes anteriores. El estado por defecto para una Fuente de Tiempo recién creada es time_source_state_initial.

El estado se puede cambiar con time_source_start(), time_source_stop()y time_source_pause().

El estado de una fuente de tiempo se devuelve mediante time_source_get_state().
