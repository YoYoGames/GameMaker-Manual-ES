tiempo_segundos_a_bpm

Esta función toma la duración de un latido en segundos, y devuelve un valor de latidos por minuto.

Sintaxis:

time_seconds_to_bpm(seconds);

Argumento Tipo Descripción
seconds Real The seconds-per-beat value to convert into beats-per-minute

Devuelve:

Real

Ejemplo:

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

Esta función obtiene el período de una Fuente de Tiempo, lo convierte a BPM e imprime ese valor en el registro de salida.

Este código sigue el ejemplo de time_bpm_to_seconds().
