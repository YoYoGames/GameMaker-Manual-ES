conjunto_de_matriz

Esta función toma un array de 16 valores, correspondientes a un tipo de matriz 4x4 dado, donde los elementos [0 - 3] serían la fila 1, los elementos [4 -7] serían la fila 2, etc... (véase la imagen de la página principal). Se puede crear una matriz de este tipo utilizando los botones matrix_build() o matrix_get() o simplemente construyendo la matriz usted mismo y pasándola a la función. Los tipos de matrices disponibles son vista, proyección y mundo, para los que se utilizaría una de las siguientes constantes:

Constante Descripción
matrix_view La matriz de vista actual
matrix_projection La matriz de proyección actual
matrix_world La matriz mundial actual

Sintaxis:

matrix_set(type, matrix);

Argumento Tipo Descripción
type Tipo de matriz Constante The type of matrix to get the values of (see the constants listed above)
matrix Matriz The matrix data as an array

Devuelve:

N/A

Ejemplo:

matrix_set(matrix_world, m_array);

El código anterior establecerá los valores de la matriz del mundo actual a los almacenados en la matriz del array "m_array".
