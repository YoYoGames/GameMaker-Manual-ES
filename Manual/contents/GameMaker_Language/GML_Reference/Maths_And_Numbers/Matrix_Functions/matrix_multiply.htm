matrix_multiply

Con esta función puedes multiplicar dos matrices juntas para crear una nueva matriz de transformación. La función devolverá el índice de la nueva matriz, que debe almacenarse en una variable para su uso futuro.

NOTA: No puedes usar una constante de matriz como argumento con esta función, así que si deseas multiplicar la matriz de la vista (por ejemplo) con una matriz personalizada que hayas construido, primero debes llamar a matrix_get() y asignar los valores de la matriz de la vista a una variable de matriz, luego multiplicarla por su matriz personalizada, y luego establecer la matriz elegida (vista, proyección o mundo).

Sintaxis:

matrix_multiply(matrix1, matrix2);

ArgumentoTipo Descripción
matrix1 The first matrix index to use.
matrix2 The second matrix index to use.

Devuelve:

Matrix Array

Ejemplo:

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

El código anterior obtendrá la matriz de la vista actual, luego la multiplicará por una matriz personalizada y luego utilizará el índice de la matriz resultante para establecer la matriz de la vista de nuevo.
