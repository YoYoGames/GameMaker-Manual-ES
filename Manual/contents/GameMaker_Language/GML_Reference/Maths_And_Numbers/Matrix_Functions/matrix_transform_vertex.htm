matrix_transform_vertex

Esta función transforma un vector por una matriz. Se suministra el ID de la matriz de transformación (devuelto por la función matrix build()), así como los valores x, y y z del vector a transformar. La función devolverá una matriz de 3 elementos donde:

  • array[0] = x
  • array[1] = y
  • array[2] = z.

Sintaxis:

matrix_transform_vertex(matrix, x, y, z);

ArgumentoTipo Descripción
matrix The matrix to use
x The x component of the transform vector
y The y component of the transform vector
z The z component of the transform vector

Devuelve:

(3 elements)

Ejemplo:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

El código anterior transforma los valores dados utilizando la matriz almacenada en la variable "t_matrix" y los almacena en la matriz "verts".
