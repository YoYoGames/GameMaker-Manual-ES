matrix_build

Esta función se puede utilizar para crear su propia matriz personalizada y devolverá un array de matrices, que debe ser almacenado en una variable para su futura referencia y uso. Acepta valores de traslación, rotación y escala tridimensionales(x, y, z) y los utiliza para construir una matriz.

La matriz contiene 16 valores en total, donde los 4 elementos iniciales son fila/columna 1los siguientes 4 elementos son fila/columna 2 y así sucesivamente, como parte de una matriz 4x4. El hecho de que la matriz esté ordenada por filas o por columnas depende de la plataforma de destino, ya que el renderizador de gráficos utilizado por una plataforma puede utilizar matrices de fila mayor o de columna mayor.

NOTA Cuando se construye una nueva matriz de esta manera, el orden de operación es YXZ.

Sintaxis:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

ArgumentoTipo Descripción
x The x component of the translation vector.
y The y component of the translation vector.
z The z component of the translation vector.
xrotation The angle to rotate around the x-axis (in degrees °).
yrotation The angle to rotate around the y-axis (in degrees °).
zrotation The angle to rotate around the z-axis (in degrees °).
xscale The x scale amount.
yscale The y scale amount.
zscale The z scale amount.

Devuelve:

Matrix Array

Ejemplo:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

El código anterior construirá una nueva matriz transformada y almacenará el índice de la matriz resultante en la variable "t_matrix".
