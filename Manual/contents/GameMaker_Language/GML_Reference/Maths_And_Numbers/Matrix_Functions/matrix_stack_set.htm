matrix_stack_set

Esta función sobrescribe la parte superior actual de la pila de matrices con la matriz especificada.

Sintaxis:

matrix_stack_set(matrix);

ArgumentoTipo Descripción
matrix The matrix index to use.

Devuelve:

Ejemplo:

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

El código anterior construirá una nueva matriz y almacenará el índice de la matriz resultante en la variable "m" antes de reemplazar la parte superior de la pila de matrices con ella.
