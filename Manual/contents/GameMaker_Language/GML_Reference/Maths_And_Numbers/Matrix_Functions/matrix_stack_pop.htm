matrix_stack_pop

Esta función elimina la matriz que está en la parte superior de la pila de matrices actual.

Sintaxis:

matrix_stack_pop();

Devuelve:

Ejemplo:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

El código anterior construirá dos nuevas matrices y las pondrá en la pila de matrices. La matriz mundial se coloca en la parte superior de la pila, se dibuja un sprite y las transformaciones se sacan de la pila.
