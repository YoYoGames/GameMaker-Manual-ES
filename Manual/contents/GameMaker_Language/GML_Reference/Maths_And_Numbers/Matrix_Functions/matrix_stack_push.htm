matrix_stack_push

Esta función se utiliza para empujar una nueva matriz a la parte superior de la pila de matrices actual. Primero multiplicará la matriz dada por la matriz que se encuentra actualmente en la parte superior de la pila, y luego empujará la matriz resultante a la pila.

Esta función es útil para aplicar múltiples transformaciones matriciales a tus visuales sin tener que multiplicar manualmente varias matrices juntas.

Por ejemplo, puedes tener una matriz base empujada en la pila que dibuje gráficos en una determinada área rectangular (digamos, una ventana). Después de dibujar algunos gráficos usando esa matriz, puedes poner otra matriz en la pila para dibujar gráficos dentro de una subárea (como un botón en la ventana). Después de dibujar algo en esa subárea, puedes llamar a matrix_stack_pop() para eliminar su matriz de la pila y continuar dibujando visuales en el área de la ventana principal.

Sintaxis:

matrix_stack_push(matrix);

ArgumentoTipo Descripción
matrix The matrix to push to the stack

Devuelve:

Ejemplo:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

El código anterior construirá dos nuevas matrices y las pondrá en la pila de matrices. La matriz mundial se coloca en la parte superior de la pila, se dibuja un sprite y las transformaciones se sacan de la pila.
