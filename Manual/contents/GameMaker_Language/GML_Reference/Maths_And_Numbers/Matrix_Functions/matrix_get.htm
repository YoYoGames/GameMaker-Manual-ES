matrix_get

Esta función devuelve un array de 16 valores, correspondientes al tipo de matriz 4x4 dado, donde la fila 1 son los elementos [0 - 3], la fila 2 son los elementos [4 -7], etc... (ver la imagen en la página principal). Las matrices disponibles son la vista, la proyección y el mundo, para las que se utilizaría una de las siguientes constantes:

Tipo de matriz Constante
Constante Descripción
matrix_view La matriz de vista actual
matrix_projection La matriz de proyección actual
matrix_world La matriz mundial actual

Sintaxis:

matrix_get(type);

Argumento Tipo Descripción
type Tipo de matriz Constante The type of matrix to get (see the constants listed above)

Devuelve:

Matrix Array

Ejemplo:

v_array = matrix_get(matrix_view);

El código anterior obtendrá los valores de la matriz de la vista actual y rellenará el array "v_array" con ellos.
