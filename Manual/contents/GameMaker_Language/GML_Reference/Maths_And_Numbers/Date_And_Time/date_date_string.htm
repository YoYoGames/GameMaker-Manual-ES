cadena_de_fecha

Con esta función puede crear un string que contenga la fecha dada, formateada como día/mes/año.

Sintaxis:

date_date_string(date);

ArgumentoTipo Descripción
date The date to use.

Devuelve:

Ejemplo:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

Esto haría que "str" contuviera un string formateado de la fecha y la hora actuales que muestra el sistema.
