date_compare_time

Con esta función puede comprobar dos fechas y horas para ver cuál tiene el componente de tiempo anterior o posterior a la otra. La función devuelve -1 si datetime1 es anterior, 1 si datetime1 es posterior y 0 si son iguales, e ignora la fecha, por lo que literalmente sólo se ve cuál de las horas es más lejana a su día dado.

Sintaxis:

date_compare_time(datetime1, datetime2);

ArgumentoTipo Descripción
datetime1 The first datetime.
datetime2 The datetime to compare the first one to.

Devuelve:

Ejemplo:

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Esto establecería "d" al valor correspondiente dependiendo de cuál de las horas sea la más temprana. Básicamente, si la hora actual es posterior a las 11:04, devolvería 1. Si es anterior, -1. Si la hora es las 11:04, devolvería 0. Si es anterior, -1. Si la hora es las 11:04, devolvería 0.
