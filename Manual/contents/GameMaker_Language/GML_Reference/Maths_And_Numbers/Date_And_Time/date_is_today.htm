fecha_es_hoy

Esta función devolverá true si el valor de la fecha dada es el día que se está comprobando (es decir, hoy), o false en caso contrario. Esta función puede ser muy útil para cosas como los Huevos de Pascua en tus juegos, o para desbloquear contenido de temporada. Ten en cuenta que esta función se verá afectada por la zona horaria establecida (por defecto es la hora local) que puedes cambiar usando la función date_set_timezone() función.

Sintaxis:

date_is_today(date);

ArgumentoTipo Descripción
date The datetime to use.

Devuelve:

Ejemplo:

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

El código anterior comprobará la fecha almacenada en la variable global "Halloween" para ver si es hoy, y si lo es establece otra variable global con un nuevo valor.
