hora_actual

Esta variable de sólo lectura devolverá la hora que corresponde al momento actual basada en la zona horaria por defecto del sistema (es decir, la hora local). Puede cambiar la zona horaria base a utilizar con la función date_set_timezone().

Sintaxis:

current_hour;

Devuelve:

Ejemplo:

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

El código anterior dibujaría la hora internacional actual en la pantalla.
