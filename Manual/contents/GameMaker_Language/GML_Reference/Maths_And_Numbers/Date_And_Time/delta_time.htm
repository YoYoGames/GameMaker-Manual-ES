tiempo_delta

Esta variable devuelve el tiempo delta de la trama, que es la diferencia de tiempo entre la trama anterior y la actual. Este valor está en microsegundos, donde 1 microsegundo es la 1.000.000ª parte de un segundo.

El propósito del delta timing es eliminar los efectos del lag o la lentitud de los ordenadores que tratan de manejar gráficos complejos o mucho código. Es un valor que se puede añadir a la velocidad de objects para que finalmente se muevan a la misma velocidad, independientemente del lag. Esto se consigue en GameMaker utilizando la variable delta_time en cada paso ya que mide el tiempo que ha pasado entre un paso y el siguiente en microsegundos. Por lo tanto, la variable delta_time se puede utilizar para calcular cuánto más rápido (por ejemplo) tiene que moverse un personaje del juego para compensar un pico de lag en el juego.

Sintaxis:

delta_time

Devuelve:

Real (integer)

Ejemplo:

var _dt = delta_time / 1000000;
speed = spd * _dt;

El código anterior establecerá la velocidad de la instancia utilizando el tiempo delta para corregir el retraso. La variable "_dt" se utilizaría para mantener el valor del tiempo delta previamente calculado, de modo que el ratio pueda utilizarse para multiplicar el valor de la velocidad base (mantenido en la variable "spd") y así obtener una velocidad consistente para la instancia.
