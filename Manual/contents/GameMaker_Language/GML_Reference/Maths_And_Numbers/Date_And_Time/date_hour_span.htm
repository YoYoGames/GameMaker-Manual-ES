date_hour_span

Con esta función puede obtener el número de horas entre dos fechas. Este valor es siempre positivo, y las horas incompletas se devolverán como una fracción.

Sintaxis:

date_hour_span(date1, date2);

ArgumentoTipo Descripción
date1 The first datetime.
date2 The datetime to compare the first one to.

Devuelve:

Ejemplo:

diff = date_hour_span(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Esto establecería "diff" al número de horas entre el 15 de septiembre de 2011, 11:04.0 y la fecha y hora actuales.
