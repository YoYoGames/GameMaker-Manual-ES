date_set_timezone

Establece la zona horaria base que se utilizará para el resto de las funciones de fecha y hora. Esta zona horaria puede ser local (tal y como la establece el sistema) o UTC, y se utilizaría una de las siguientes constantes para definir cuál se utiliza (por defecto es la hora local):

Constante Descripción
timezone_local utilizar la zona horaria local establecida por el sistema
timezone_utc utilizar el Tiempo Universal Coordinado

Sintaxis:

date_set_timezone(timezone);

Argumento Tipo Descripción
timezone Constante de la zona horaria The time zone to use for the base time.

Devuelve:

N/A

Ejemplo:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

Este código comprueba la configuración de la zona horaria base del juego y, si no es UTC, la cambia.
