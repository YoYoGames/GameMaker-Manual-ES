clamp

Con esta función puede mantener un valor de entrada entre un rango especificado.

Sintaxis:

clamp(val, min, max)

ArgumentoTipo Descripción
val The value to clamp.
min The minimum value to clamp between.
max The maximum value to clamp between.

Devuelve:

Ejemplo:

speed = clamp(speed, 1, 10);

El código anterior fijará la velocidad para que nunca caiga por debajo de 1 o pase de 10.
