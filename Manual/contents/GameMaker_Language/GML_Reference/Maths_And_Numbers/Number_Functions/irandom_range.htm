rango_irandom

Esta función es similar a random_range() sólo que con valores enteros como entrada. Se suministra el valor bajo para el rango así como el valor alto, y la función devolverá un valor entero aleatorio dentro de (y incluyendo) el rango dado. Por ejemplo, irandom_range(10, 35) devolverá un entero entre 10 y 35 inclusive.

Al igual que con la función irandom() se pueden utilizar números reales, en cuyo caso se redondearán al entero más cercano EG: irandom_range(6.2,9.9) dará un valor entre 6 y 9.

NOTA Esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido a que GameMaker: Studio genera la misma semilla aleatoria inicial cada vez para hacer que el código de debugging sea una tarea mucho más fácil. Para evitar este comportamiento utilice randomise() al inicio de su juego.

Sintaxis:

irandom_range(n1, n2);

ArgumentoTipo Descripción
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

Devuelve:

Ejemplo:

score += irandom_range(500, 600);

Esto añadirá un valor entero entre 500 y 600 (inclusive) a la puntuación total.
