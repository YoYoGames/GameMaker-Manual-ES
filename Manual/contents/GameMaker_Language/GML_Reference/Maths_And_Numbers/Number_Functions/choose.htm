A veces quiere especificar algo más que números para una selección aleatoria, o los números que quiere no están en ningún orden real o dentro de algún rango establecido. En estos casos, utilizarías choose() para generar un resultado aleatorio. Por ejemplo, digamos que quiere crear un object con un sprite aleatorio al principio, entonces podría usar esta función para establecer el índice sprite a uno de un conjunto de sprites dados. Tenga en cuenta que puede tener tantos como necesite (tenga en cuenta que más argumentos significarán que la función será más lenta de analizar).

NOTA Esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido a que GameMaker genera la misma semilla aleatoria inicial cada vez para hacer que el código de debugging sea una tarea mucho más fácil. Para evitar este comportamiento utilice randomise() al inicio de su juego.

Sintaxis:

choose(val0, val1, val2... max_val);

Argumento Tipo Descripción
val0... max_val Variable Any type of value(s).

Devuelve:

Variable (One of the given arguments)

Ejemplo:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

El código anterior utiliza choose para establecer una serie de propiedades para la instancia.
