sqr

Multiplica un número por sí mismo y devuelve el cuadrado de ese número. Por ejemplo: sqr(5) devolvería 25 ya que 5*5=25.

Sintaxis:

sqr(val);

ArgumentoTipo Descripción
val The number to square.

Devuelve:

Ejemplo:

score += sqr(dmg);

El código anterior añadirá el cuadrado de la variable "dmg" a la puntuación.
