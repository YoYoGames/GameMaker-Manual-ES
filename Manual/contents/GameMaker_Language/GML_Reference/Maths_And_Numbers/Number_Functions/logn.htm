logn

Esta función es similar a las funciones log2(n) y log10(n), sólo que se estipula el valor de la base logarítmica. Por ejemplo, logn(5,25) devolverá cuántos 5 necesitamos multiplicar para obtener 25 (que es 2).

Sintaxis:

logn(n, val);

ArgumentoTipo Descripción
n The log base.
val The input value.

Devuelve:

Ejemplo:

logval = logn(5, num);

El código anterior obtiene el log5 del valor almacenado en "num".
