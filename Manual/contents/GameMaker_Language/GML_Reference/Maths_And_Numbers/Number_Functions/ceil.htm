ceil

Esta función toma cualquier número real y lo redondea al entero más cercano. Hay que tener cuidado con esta función ya que un error común es utilizarla para redondear un valor aleatorio y esperar que siempre sea mayor que 1, es decir:

int = ceil(random(5));

Ahora bien, es de esperar que este código dé siempre un número entero entre 1 y 5, pero no siempre es así, ya que hay una posibilidad muy pequeña de que la función aleatoria devuelva 0, y el redondeo de 0 sigue dando 0. Esta es una posibilidad remota posibilidad, pero debe tenerse en cuenta cuando se utiliza esta función.

Sintaxis:

ceil(val);

ArgumentoTipo Descripción
x The number to change.

Devuelve:

Ejemplo:

val = ceil( 3.4 );

Esto pondrá a Val en 4.
