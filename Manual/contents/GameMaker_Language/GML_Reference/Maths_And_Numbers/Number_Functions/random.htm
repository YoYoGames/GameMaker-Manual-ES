al azar

Esta función es buena para las probabilidades en las que no es necesario devolver un entero (número entero). Por ejemplo, random(100) devolverá un valor entre 0 y 99, ¡pero ese valor puede ser 22,56473! También puede utilizar números reales y no enteros en esta función como esta - random(0.5), que devolverá un valor entre 0 y 0.4999999.

NOTA: Esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido a que GameMaker genera la misma semilla aleatoria inicial cada vez para hacer que el código de debugging sea una tarea mucho más fácil. Para evitar este comportamiento utilice randomise() al inicio de su juego.

Sintaxis:

random(n);

ArgumentoTipo Descripción
n The upper range from which the random number will be selected.

Devuelve:

Ejemplo:

if (random(10) >= 9)
{
    score += 100;
}

Esto producirá aproximadamente una posibilidad entre diez de sumar 100 a la puntuación.
