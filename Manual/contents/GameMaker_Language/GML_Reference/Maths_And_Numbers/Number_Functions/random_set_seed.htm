random_set_seed

Para generar un número aleatorio GameMaker comienza con un número semilla aleatorio. Con esta función puedes establecer esa semilla a un valor conocido y así "forzar" que el resultado de todos los eventos aleatorios posteriores sea el mismo cada vez que se ejecute el programa. Por ejemplo, esta función puede utilizarse junto con random_get_seed() para crear contenidos generados procedimentalmente y guardar los resultados sin tener que hacer enormes partidas guardadas (sólo se guarda la semilla, no hace falta nada más). Si necesitas resultados verdaderamente aleatorios para todo, deberías usar la función randomise() función.

NOTA Aunque esta semilla dará resultados consistentes en cada plataforma de destino, los resultados pueden variar entre plataformas debido a la forma diferente en que funciona cada objetivo.

Sintaxis:

random_set_seed(val);

ArgumentoTipo Descripción
val The seed to set.

Devuelve:

N/A

Ejemplo:

if debug
{
    random_set_seed(1);
}

El código anterior establece la semilla aleatoria a 1 sólo si la variable "debug" es verdadera.
