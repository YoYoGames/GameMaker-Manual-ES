poder

Esto devolverá el valor de un número multiplicado por sí mismo "n" veces. Por ejemplo, power(5,3) multiplicará 5 por sí mismo 3 veces y devolverá 125, que es lo mismo que decir 5*5*5=125. Tenga en cuenta que el valor "x" (el número a cambiar) no puede ser un valor negativo.

Sintaxis:

power(x, n);

ArgumentoTipo Descripción
x The number to change.
n How many times to multiply x by itself.

Devuelve:

Ejemplo:

score += power(dmg, 3);

Esto añadirá el valor de la variable "dmg" a la potencia de 3 a la variable de puntuación.
