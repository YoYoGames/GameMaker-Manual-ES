math_set_epsilon

Epsilon es un valor que se utiliza para determinar si dos números sujetos a un error de redondeo están lo suficientemente cerca como para ser considerados "iguales". Es útil cuando se trata de matemáticas en coma flotante, ya que puede reducir los "errores de redondeo" que hacen que ciertas operaciones devuelvan valores que parecen incorrectos o contrarios a lo que esperamos. Por ejemplo, puede que hayamos sumado un valor al índice de la imagen de un sprite y esperemos que el resultado sea un único entero, pero debido a la naturaleza de las matemáticas en coma flotante, el valor final real acaba siendo algo así como 5,0000002, por lo que cuando tenemos la siguiente comprobación:

if image_index == 5
{
    //do something
}

El código no se comporta como se esperaba y devuelve false. Sin embargo, si establecemos el valor de épsilon en 0,000001, el valor de image_index se redondeará al número real más cercano que esté a +/- 0,000001 del valor original, haciendo que la comparación anterior devuelva verdadero.

El valor épsilon se utilizará para todos los operadores enteros siguientes:

  • < : Menos de
  • > : Mayor que
  • == : Igual a
  • <= : Menor o igual que
  • >= : Mayor o igual que
  • != : No es igual a

Tenga en cuenta que al establecer un valor de épsilon de 0 se desactivará todo el redondeo, y al utilizar un valor de 1 se producirá un error.

Sintaxis:

math_set_epsilon(epsilon);

Argumento Tipo Descripción
epsilon Real The new epsilon value (from 0 to 0.999999999).

Devuelve:

Real

Ejemplo:

math_set_epsilon(0.0001);

Esto establecerá el valor de épsilon para todas las operaciones posteriores de punto flotante.
