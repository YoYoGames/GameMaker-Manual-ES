signo

Esta función devuelve si un número es positivo, negativo o ninguno y devuelve 1, -1, 0 respectivamente. Por ejemplo - sign(458) devolverá 1, sign(-5) devolverá -1 y sign(0) devolverá 0.

Sintaxis:

sign(n);

ArgumentoTipo Descripción
n The number to get the sign of.

Devuelve:

Ejemplo:

y += sign(y - mouse_y);

El código anterior añadirá 1, -1 o 0 a y dependiendo del resultado de y - mouse_y.
