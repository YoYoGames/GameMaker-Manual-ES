darccos

Devuelve el coseno inverso de x, de forma que si dcos(val)=n, darccos(n)=val, y el número resultante estará entre 180 y 0.

NOTA: Esto sólo aceptará un número entre -1 y 1 (cualquier otra cosa arrojará un error).

NOTA: Esta función devuelve un valor en grados, no en radianes.

Sintaxis:

darccos(val)

ArgumentoTipo Descripción
x The value to return the inverse cosine of.

Devuelve:

Ejemplo:

val = arccos(-1);

Esto pondrá "val" a 180.

Atrás: Ángulos y distancia
