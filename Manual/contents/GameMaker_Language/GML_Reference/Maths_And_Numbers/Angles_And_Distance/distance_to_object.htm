distancia_al_objeto

Esta función calcula la distancia desde el borde del cuadro delimitador de la instancia que llama hasta el borde más cercano de la instancia más próxima de la object especificada. El object puede ser un índice de object o un ID de instancia específico, así como el palabra clave y la distancia se devuelve en píxeles. Tenga en cuenta que si alguno de los objects no tiene sprite o no tiene definida ninguna máscara, los resultados serán incorrectos.

Sintaxis:

distance_to_object(obj);

Argumento Tipo Descripción
obj Activo del objeto o ID de la instancia The object to check for.

Devuelve:

Real

Ejemplo:

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

El código anterior comprobará la distancia al jugador object y si es menor que el valor almacenado en la variable "range" la variable "canshoot" se pone a true.
