distancia_al_punto

Esta función calcula la distancia desde el borde del cuadro delimitador de la instancia que llama hasta la posición x/y especificada en room, con el valor de retorno en píxeles. Tenga en cuenta que si el object de llamada no tiene sprite o no tiene definida ninguna máscara, los resultados serán incorrectos.

Sintaxis:

distance_to_point(x, y);

ArgumentoTipo Descripción
x The x position to check.
y The y position to check.

Devuelve:

Ejemplo:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    can_shoot = true;
}

El código anterior comprobará la distancia a la posición x/y del jugador object y si es menor que el valor almacenado en la variable "range" la variable "canshoot" se pone a true.
