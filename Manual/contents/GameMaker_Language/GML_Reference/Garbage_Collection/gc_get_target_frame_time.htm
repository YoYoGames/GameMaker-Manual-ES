gc_get_target_frame_time

Con esta función puedes recuperar el valor actual del marco objetivo para el recolector de basura. El valor devuelto está en microsegundos (donde 1.000.000 de microsegundos equivale a un segundo) y el tiempo de cuadro objetivo por defecto es de 100 microsegundos. Si desea cambiar este valor, debe utilizar la función gc_target_frame_time().

Sintaxis:

gc_get_target_frame_time(time);

Devuelve:

Ejemplo:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

El código anterior comprueba el objetivo de tiempo del marco actual para el recolector de basura y si no es 50 microsegundos entonces se establece en este valor.
