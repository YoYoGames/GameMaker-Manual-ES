gc_target_frame_time

Con esta función puedes indicar al recolector de basura que no debe pasar más del tiempo especificado ejecutando cada trama. La función toma un valor de tiempo especificado en microsegundos (donde 1.000.000 de microsegundos equivale a un segundo) y el tiempo de trama objetivo por defecto es de 100 microsegundos.

Tenga en cuenta que esto es simplemente un ya que algunas operaciones deben ejecutarse completamente en una sola trama y a veces pueden tardar más que el tiempo objetivo. También tenga en cuenta queaumentar el tiempo objetivo hará que el recolector de basura responda mejor a los cambios rápidos en el uso de la memoria y hará que la memoria se libere más rápidamente, aunque en la práctica es poco probable que esto sea necesario en la mayoría de los casos. Ten en cuenta también que si estableces el tiempo de fotogramas objetivo en 0, el recolector no se detendrá por completo, sino que seguirá realizando una cantidad mínima de trabajo en cada fotograma. Para desactivar completamente el recolector, utilice la función gc_enable() .

Sintaxis:

gc_target_frame_time(time);

ArgumentoTipo Descripción
time The target time - in microseconds -  that the garbage collector should work each frame

Devuelve:

N/A

Ejemplo:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

El código anterior comprueba el objetivo de tiempo de cuadro actual para el recolector de basura y si no es 50 microsegundos entonces se establece en este valor.
