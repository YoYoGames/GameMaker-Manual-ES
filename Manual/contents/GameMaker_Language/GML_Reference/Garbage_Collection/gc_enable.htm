gc_enable

Con esta función puedes activar o desactivar el recolector de basura. Llamar a la función con true como argumento lo habilita y usar false lo deshabilita (no se recomienda). Está habilitado por defecto.

Sintaxis:

gc_enable(enable);

ArgumentoTipo Descripción
enable enable (true) o desactivar (false) el recolector de basura.

Devuelve:

Ejemplo:

if (global.debug == true)
{
    gc_enable(false);
}

El código anterior desactiva la recolección de basura si la variable global dada es true.
