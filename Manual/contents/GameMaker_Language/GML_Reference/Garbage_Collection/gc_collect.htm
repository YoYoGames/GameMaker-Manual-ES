gc_collect

Con esta función puedes activar el recolector de basura, forzándolo a ejecutarse al final del marco actual (paso). Cabe destacar que no es necesario que el recolector de basura esté activo para que esto funcione. Llamar a esta función después de desactivar el recolector de basura (usando la función gc_enable()) activará el recolector de basura durante un fotograma en el que todos los objects que han sido marcados para su recolección serán eliminados de la memoria antes de que el recolector de basura se desactive de nuevo.

Sintaxis:

gc_collect();

Devuelve:

Ejemplo:

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

El código anterior comprueba una variable global y una tecla que se está pulsando y si son true entonces se activa la recolección de basura para el final de la trama (paso).
