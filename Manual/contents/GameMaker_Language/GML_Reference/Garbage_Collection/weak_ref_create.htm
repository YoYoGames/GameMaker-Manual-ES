weak_ref_create

Con esta función se puede crear una referencia débil a un struct que luego se puede utilizar para comprobar si el struct sigue "vivo" (referenciado) o no en el juego. Usted proporciona la referencia al struct que desea rastrear, y la función devolverá otro struct que es una referencia débil a ese struct. Ten en cuenta que puedes comprobar si una referencia es "fuerte" o "débil" utilizando la función instanceof(), ya que una referencia fuerte devolverá o bien "struct" o el nombre de la función constructora que creó el struct, o bien "weakref" si se trata de una referencia débil. También hay que tener en cuenta que la referencia débil struct tendrá una variable "ref" a la que se puede acceder para obtener la referencia fuerte al struct en cuestión, a menos que se haya recogido la basura, en cuyo caso devolverá undefined.

Sintaxis:

weak_ref_create(struct_to_track);

Argumento Tipo Descripción
struct_to_track Struct The struct that you want to create a weak reference for

Devuelve:

Struct Weak Reference

Ejemplo:

inventory_ref = weak_ref_create(inventory);

El código anterior crea una referencia débil a un struct y lo almacena en una variable de instancia para su uso posterior.
