weak_ref_any_alive

Con esta función puedes comprobar las referencias débiles a varios structs para ver si siguen "vivos" o no. Proporcionas un array de referencias débiles para comprobar (cada referencia débil debe haber sido creada usando la función weak_ref_create()), y la función devolverá true si cualquier de los structs todavía están siendo referenciados en alguna parte o false si no lo están y han sido recolectados por la basura. Tenga en cuenta que si proporciona un array en el que alguno de los valores no es una referencia débil, la función devolverá undefined.

Tenga en cuenta que la función también toma dos argumentos opcionales, donde el primero le permite suministrar un índice inicial en la matriz para comprobar desde, así como un argumento para establecer el número de posiciones (longitud) desde ese índice para comprobar. Si se especifica, sólo se comprobarán los índices del array dentro de esos parámetros.

Sintaxis:

weak_ref_any_alive(array, [index], [length]);

Argumento Tipo Descripción
array Conjunto de referencias débiles de Struct Array containing weak references to the structs that you want to check.
[index] Real OPTIONAL El índice de la matriz desde el que se inicia la comprobación
[length] Real OPTIONAL El número de posiciones, a partir del índice dado, que se deben comprobar

Devuelve:

Boolean (or undefined)

Ejemplo:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

El código anterior comprueba un array de referencias débiles y si alguna sigue viva se destruye una instancia.
