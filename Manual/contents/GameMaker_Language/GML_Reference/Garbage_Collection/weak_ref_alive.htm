weak_ref_alive

Con esta función puede comprobar la referencia débil a un struct para ver si sigue "vivo" o no. Usted proporciona la referencia débil a comprobar (tal como la devuelve la función weak_ref_create()), y la función devolverá true si el struct sigue siendo referenciado en alguna parte o false si no lo es y ha sido recolectado por la basura. Tenga en cuenta que si suministra un valor que no es una referencia débil, la función devolverá undefined.

Sintaxis:

weak_ref_alive(weak_ref);

Argumento Tipo Descripción
weak_ref Referencia débil de la estructura The weak reference to the struct you want to check.

Devuelve:

Boolean (or undefined)

Ejemplo:

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

El código anterior comprueba una referencia débil a un struct y, si sigue vivo, la variable que lo referencia se pone a -1, desreferenciando el struct y permitiendo su recogida de basura.
