inicio / fin

GameMaker Language le permite utilizar las palabras clave begin y end en lugar de las llaves más habituales {} al crear bloques de código. El siguiente ejemplo de código muestra cómo funciona:

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

Tenga en cuenta que el uso de estas palabras clave no es típico y se proporciona como parte del lenguaje más para el apoyo de legado que para otra cosa, y en cualquier momento en el futuro pueden ser obsoletos.
