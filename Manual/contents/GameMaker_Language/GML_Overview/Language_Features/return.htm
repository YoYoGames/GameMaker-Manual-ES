devolver

La declaración return tiene la siguiente sintaxis:

return (<expression>)

La sentencia return sólo se utiliza en las funciones y métodos de los scripts, y sirve para devolver un valor de la función que se utilizará en otras llamadas a código o funciones. Hay que tener en cuenta que la ejecución de la función termina en la sentencia return , lo que significa que cualquier código que venga después de la llamada a return no se ejecutará. Aquí hay un ejemplo corto script llamado sqr_calc que calcula el cuadrado de cualquier valor que se le pase, e incluye la captura de errores en caso de que el parámetro que se le pase no sea un número real:

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if (!is_real(value))
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

Para llamar a una función desde un trozo de código, basta con utilizarla de la misma manera que cuando se llaman funciones de runtime - es decir, escribir el nombre de la función con los valores de los parámetros entre paréntesis. Así, la función anterior se llamaría así:

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

Aquí la variable val será o bien 0 (porque la variable amount no es un número real) o bien el total de value * value.
