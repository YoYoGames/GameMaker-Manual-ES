El operador new puede ser utilizado para crear un struct con una función que es llamada con los argumentos dados. La función puede entonces rellenar el struct con las variables de los argumentos - al igual que el evento de creación de una instancia establecerá las variables iniciales para la instancia - y luego el operador new devolverá el struct. Antes de utilizar este operador es importante tener en cuenta que la función dada debe ser marcada como una función constructor de lo contrario el operador new no creará el struct (el ejemplo de código a continuación muestra esto, y para más información, por favor consulte la página sobre Estructuras).

NOTA Puede comprobar un struct para saber qué función se utilizó para crearlo utilizando la función runtime instanceof().

Este operador tiene la siguiente sintaxis:

<variable> = new <function> (<argument>, <argument>, ...);

Cuando se utiliza el operador new junto con una función, devolverá una referencia struct que se almacena en la variable dada, permitiéndole acceder a la struct y a cualquier variable que haya sido creada dentro de ella por la función.

El siguiente ejemplo muestra una función que ha sido definida para ser utilizada como constructor:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Esta función se puede utilizar junto con el operador new para crear un struct y rellenarlo con las variables ajustadas a los valores de los argumentos utilizados en la función, así:

mystruct = new init_struct(10, 100, "Hello World");

Es importante tener en cuenta que llamar a new en cualquier función que no haya sido marcada como constructor provocará una excepción en tiempo de ejecución.
