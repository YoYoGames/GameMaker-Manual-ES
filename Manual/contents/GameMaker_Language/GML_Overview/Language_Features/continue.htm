continuar

La declaración continue tiene la siguiente sintaxis básica:

continue;

Si se utiliza dentro de una sentencia que forme un loop (for, repeat, while o do / until), terminará inmediatamente la iteración actual y saltará al principio de loop comenzando una nueva iteración y omitiendo cualquier código que venga después de continue dentro de loop. También puede utilizarse dentro de la sentencia with donde hará que el código salte a la siguiente instancia y se ejecute de nuevo. Tenga en cuenta que si continue se utiliza fuera de cualquiera de estos contextos dará un error.

A continuación se muestra un ejemplo de uso en un bucle de for:

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

A continuación se muestra un ejemplo de uso en un bucle de while:

var file = file_text_open_read("Game_Data.txt");
var _num = 0;

while (!file_text_eof(file))
{
    var _s = file_text_readln(file);
    if (_s == "")
    {
        continue;
    }
    str[num++] = _s;

file_text_close(file);

A continuación se muestra un ejemplo de uso en un bucle de do / until:

do
{
    var _x = random(room_width);
    var _y = random(room_height);

    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }

    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}
until (instance_count(obj_Enemy) >= 10);

Por último, un ejemplo de uso en una declaración de with:

with (obj_Enemy_Parent)
{
    if (object_index == obj_Enemy_InDestructible)
    {
        continue;
    }

    hp -= 100;

    if (hp <= 0)
    {
        instance_destroy();
    }
}
