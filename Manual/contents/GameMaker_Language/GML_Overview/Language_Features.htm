Características lingüísticas

El GameMaker Language (GML) tiene una serie de características que forman parte de la estructura esencial de todos los programas escritos con el lenguaje. Cada una de ellas se utilizará en cualquier código que escribas varias veces, por lo que debes saber cuáles son y cómo funcionan antes de empezar a programar un proyecto.

  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
