Visión generaldel código GML

Esta sección del manual contiene toda la información necesaria para entender y utilizar Código GML. Las funciones de runtime en este lenguaje se pueden utilizar para crear sus juegos y se añaden en objects desde el Editor de Objetosaunque también se puede utilizar junto con GMLVisual. A continuación puedes ver una imagen típica de un object con el editor de código abierto en un evento:

Cada evento tiene su propia pestaña en el editor y puede añadir, editar o eliminar código de ellos en cualquier momento (para más información sobre los eventos, consulte Eventos de objetos). El código en sí debe tener una estructura básica y puede contener índices de recursos, variables, funciones, expresiones, palabras clave, etc., todo lo cual se explica en las secciones siguientes. Si es un principiante en la programación o en la realización de la switch desde GML Visual, se recomienda que empiece con la página sobre Estructura básica del código y luego lea todas las demás páginas de esta sección, probando el código de cada una dentro de GameMaker mismo.

Puede utilizar Feather para completar el código de forma inteligente e informar de los errores en su GML Code scripts.
