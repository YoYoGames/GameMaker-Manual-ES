El GameMaker Language (también llamado simplemente GML) es el lenguaje de scripting propio de GameMaker. Este lenguaje está estructurado para permitir a los usuarios crear sus juegos de forma intuitiva y flexible, al tiempo que ofrece toda la potencia de cualquier otro lenguaje de programación importante. También es la base de GML Visual y puede utilizarse junto con éste si es necesario.

La sección de GML Code del manual se divide en los siguientes apartados:

  • Resumen de GML Code
