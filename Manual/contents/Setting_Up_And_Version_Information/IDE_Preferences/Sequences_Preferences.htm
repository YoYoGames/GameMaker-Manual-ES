Secuencias Preferencias

Las Preferencias de Secuencias se utilizan para definir ciertas propiedades para el funcionamiento del Editor de Secuencias y sus elementos. Las opciones principales son:

  • Mostrar el marco del lienzo en las secuencias recién creadas: Cuando se activa, la guía inteligente del marco del lienzo se mostrará en el lienzo del Editor Sequence para todas las nuevas sequences. Esta opción puede ser activada/desactivada según sea necesario desde las propiedades del lienzo en el propio editor también. El valor por defecto es activado.
  • Ancho de cuadrodel lienzo en las secuencias recién creadas: Aquí puede establecer el ancho de marco por defecto para todas las nuevas sequences. El valor por defecto es 1366px.
  • La altura del marcodel lienzo en las secuencias recién creadas: Aquí puede establecer la altura del marco por defecto para todas las nuevas sequences. El valor por defecto es 768px.
  • Pausa al hacer clic en el lienzo: Cuando esta opción está marcada, al hacer clic en cualquier parte del lienzo del Editor Sequence se pausará la reproducción de sequence. Esta opción está activada por defecto.
  • Retraso de la selección de superposición en milisegundos: Controla el tiempo que hay que esperar para que aparezca la caja de herramientas de elementos superpuestos en el lienzo del editor Sequence. El valor por defecto es 1000ms.
  • Traducción path margen de inserción de puntos: En el lienzo del Editor Sequence, cuando hayas añadido un asset y luego hayas creado una pista de parámetros de posición, habrá líneas de puntos dibujadas para demostrar el path que tomará el asset al moverse. Si colocas el ratón cerca de esta línea, entonces puedes añadir un nuevo punto de pista de parámetros a la misma, y esta opción establece la distancia desde la línea de traslación que el ratón necesita estar antes de que se muestre el marcador de punto. El valor por defecto es 6px.
  • Crear automáticamente pistas de parámetros por defecto: Cuando está activada, esta opción añadirá automáticamente pistas de parámetros para la posición, la escala, la rotación y el origen a cualquier pista nueva creada en el panel de pistas de sequence. Por defecto está desactivada.
  • Estirarautomáticamente los activos recién creados: Esta opción estirará todas las claves de asset recién creadas para que coincidan con la longitud de sequence. Por defecto está desactivada.
  • Altura de la Pista de Curvas de Animación en la hoja dopw: Esta opción se puede utilizar para establecer cuántas líneas de pista de parámetros que utilizan curvas ocuparán en la hoja dopw cuando en la vista keyframe. El valor por defecto es 2, pero también se puede seleccionar 1 o 3.
  • Texto por defecto para una nueva Pista de Texto: El texto string utilizado cuando se crea una nueva Pista de Texto en un Sequence.
  • Color de selección del inspector: El color utilizado para resaltar la pista seleccionada en la lista de selección múltiple del inspector.

Confirmar diálogos

Las preferencias de los Diálogos de Confirmación se utilizan para cambiar los mensajes que se muestran en el editor sequence cuando se realizan ciertas acciones. Los mensajes que se pueden cambiar son:

  • Elección automática para incrustar la curva editada: Si intenta incrustar una animation curve que ha editado, se le mostrará un mensaje por defecto para confirmar la acción. Sin embargo, si se ajusta esta opción a "Sí" no se mostrará el mensaje y se incrustará cada vez. Si se establece en "No", se evitará que se incruste cada vez.
  • Elección automática para convertir la curva en llaves: Si intenta convertir una animation curve en keyframes, se le mostrará un mensaje por defecto preguntándole si desea continuar. Sin embargo, si se ajusta esta opción a "Ok" no se mostrará el mensaje y simplemente no se permitirá convertir la curva.

Guías inteligentes

Aquí puede seleccionar el comportamiento de las guías inteligentes cuando se trabaja en el Editor Sequences. Las opciones disponibles son:

  • Habilitar guías inteligentes: Aquí puede activar o desactivar las guías inteligentes en el lienzo del Editor Sequences. Esto está habilitado por defecto.
  • Habilitar Reglas: Aquí puede activar o desactivar las guías inteligentes en el lienzo del Editor Sequences. Esto está habilitado por defecto.
  • Ajustar reglas a los bordes: Esta opción permitirá que las reglas se "ajusten" a los bordes de assets en el lienzo sequences (y viceversa). Por defecto está activada.
  • Distancia de ajuste: Es la distancia (en píxeles) que debe tener un elemento respecto a una regla (o una regla respecto a un elemento) para que se "ajuste" a la posición. El valor por defecto es 6px.
