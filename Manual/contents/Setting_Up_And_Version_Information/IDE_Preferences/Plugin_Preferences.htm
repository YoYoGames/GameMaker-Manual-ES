Preferencias de los plugins

Las Preferencias de los Plugins se utilizan para controlar cualquier plugin extra que se haya añadido a GameMaker IDE. Por defecto tiene una sección dedicada al plugin de Control de Fuentes para integrar los proyectos de GameMaker con Git.

Control de la fuente (Git)

Esta sección trata de las preferencias necesarias para que el plugin Git SCM funcione con GameMaker.

  • Ruta del ejecutable de Git: Esta es la ubicación del ejecutable de Git que GameMaker utiliza para todos los comandos de Source Control. Como se muestra en la imagen anterior, estará ubicado en la carpeta cmd de su instalación Git.

Debajo de esto puede configurar la herramienta de fusión y la herramienta de diferenciación para Source Control con estos ajustes(opcional):

  • Fusión (Ubicación de la herramienta): Aquí se especifica el archivo completo path a la herramienta de fusión Git.
  • Fusión (Opciones de la herramienta): En este campo puede añadir cualquier comando extra que se ejecute cada vez que utilice la herramienta de fusión.
  • Diff (Ubicación de la herramienta): Aquí se especifica el archivo completo path a la herramienta de diferencias Git.
  • Diff (Opciones de la herramienta): En este campo puedes añadir cualquier comando extra que se ejecute cada vez que utilices la herramienta diff.
