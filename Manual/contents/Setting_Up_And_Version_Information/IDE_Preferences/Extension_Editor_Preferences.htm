Extension Preferencias del editor

Las preferencias deExtension Editor cubren algunas opciones relacionadas con los mensajes que se que se le muestran al utilizarlo, con los detalles exactos que se describen a continuación:

  • Respuesta automática al borrado de archivos: La configuración por defecto aquí es mostrar un mensaje preguntando si está seguro de que desea eliminar, pero puede establecer esto en "Aceptar" para que todas las eliminaciones se completen sin este mensaje como si hubiera ya hubiera hecho clic en "Aceptar".
  • Respuesta automática a la eliminación de archivos proxy: Como en el caso anterior, esto controla el mensaje que se le muestra al eliminar cualquier archivo proxy.
