Preferencias de la pantalla de inicio

Las preferencias de la pantalla de inicio le permiten cambiar el comportamiento de la página de inicio. Las opciones principales son:

  • Mostrar Tour: Permite seleccionar si se muestra el recorrido introductorio al iniciar el IDE. Esto se desactiva automáticamente la primera vez que se salta el tour.
  • Mostrar asistente de configuración: Le permite seleccionar si el Asistente de Configuración se muestra cuando se inicia el IDE. El Asistente de Configuración le lleva esencialmente a través de los pasos necesarios para crear un nuevo proyecto.
  • Estilo de la lista de proyectos recientes: Permite cambiar la visualización de la lista de Proyectos Recientes en la Página de Inicio entre "Lista" y "Cuadrícula".
