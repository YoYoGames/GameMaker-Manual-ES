Preferencias de ahorro de energía

Las preferencias de ahorro de energía aquí afectarán a la cantidad de energía de la CPU y de la GPU que se utilizará para generar el GameMaker IDE (esencialmente, esto reducirá la velocidad de fotogramas base con la que se ejecuta el programa y, por tanto, el consumo de energía).

Las opciones aquí son:

  • Velocidad de fotogramas de la alimentación de CA (%): El modificador de framerate base IDE cuando se conecta a una entrada de alimentación. Por defecto es el 100%.
  • Velocidad de fotogramas de la batería (%): El modificador de framerate base de IDE cuando se utiliza la batería para alimentar la máquina. Por defecto es el 35%.
  • Background framerate (%): El modificador de framerate base de IDE cuando GameMaker no está enfocado (independientemente de la alimentación). Por defecto es el 15%.
