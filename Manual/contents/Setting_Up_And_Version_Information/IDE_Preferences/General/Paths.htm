Preferencias de la ruta

Las preferencias de ruta controlan los diferentes archivos paths requeridos por los elementos en el GameMaker IDE así como por cualquier editor externo que se requiera. Existen las siguientes opciones para el IDE:

  • Directorio temporal: La ubicación para guardar todas las carpetas temporales. Por defecto en Windows es

    \Users\<Username>\AppData\Local\Temp\GameMaker

    Y en macOS es:

    /var/folders/<Path_Hash>/GameMaker

  • Directorio del IDE cache: La ubicación para guardar el IDE cache. Por defecto en Windows es

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    Y en macOS es:

    /Users/<Username>/.config/GameMaker/Cache

  • Directorio de activos cache: La ubicación para guardar el asset cache para cada proyecto. Por defecto en Windows es

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    Y en macOS es:

    /Users/<Username>/.config/GameMaker/Cache

  • Ubicación de "Mis Proyectos": La ubicación donde GameMaker creará inicialmente los nuevos proyectos. Por defecto en Windows es

    \Users\<Username>\Documents\GameMaker

    Y en macOS es

    /Users/<Username>/GameMaker

  • Borrar automáticamente el directorio temporal al cerrar: Marcando esta opción forzará a GameMaker a borrar automáticamente la carpeta temporal que crea por proyecto para compilar, etc... Esta opción está activada por defecto, y al desmarcarla switch la desactivará (lo que significa que tendrá que eliminar manualmente los archivos temporales más tarde).
  • Borrar automáticamente asset cache al cerrar: Si esta opción está marcada, la carpeta asset del compilador cache se eliminará cuando salga de GameMaker. Esta opción está desactivada por defecto, y si la activas significará que cada vez que cargues y ejecutes cualquier proyecto el cache tendrá que ser reconstruido (lo que puede llevar tiempo dependiendo del tamaño del juego).
  • Borrar automáticamente IDE cache al cerrar: Si esta opción está marcada, la carpeta IDE del compilador cache se eliminará cuando salga de GameMaker. Esta opción está desactivada por defecto, y si la activas significará que cada vez que inicies GameMaker el cache tendrá que ser reconstruido.
  • Borrar carpeta temporal: Al hacer clic en este botón se eliminará la carpeta temporal del proyecto.
  • Borrar Asset Cache: Al hacer clic en este botón se eliminará el compilador asset cache para el proyecto.
  • Borrar la caché de IDE: Al hacer clic en este botón se borrará la caché de IDE

Existen las siguientes opciones para configurar paths a los editores externos:

  • Ruta de acceso al editor/visualizador externo para archivos SWF: Si está trabajando con el formato SWF sprites, puede establecer esto a la path de su visor/editor preferido y cuando haga clic en el botón Editar Imagen en el Editor Sprite entonces se abrirá el programa dado. El valor por defecto aquí es no tener path.
  • Ruta de acceso al editor/visualizador externo para los archivos Spine: Si está trabajando con el formato Spine sprites , puede establecer esto en el path de su visor/editor preferido y cuando haga clic en el botón Editar Imagen en el Editor Sprite entonces se abrirá el programa dado. El valor por defecto aquí es no tener path.
