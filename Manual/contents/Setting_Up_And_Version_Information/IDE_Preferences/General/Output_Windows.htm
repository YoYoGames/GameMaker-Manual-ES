Preferencias de la ventana de salida

Las preferencias de Ventana de salida controlan las diferentes opciones de color y formato de las diferentes ventanas de salida en IDE. Cada una de las diferentes ventanas tiene el mismo conjunto de opciones, que se enumeran a continuación:

  • Color: El color por defecto del texto que se escribe en la ventana.
  • Font Nombre: El nombre del font a utilizar (los fonts disponibles dependerán de los instalados en su sistema).
  • Estilo: El estilo de font a utilizar, como negrita o cursiva (tenga en cuenta que no todas las ventanas permitirán los mismos estilos y también dependerá del font utilizado).
  • Tamaño: El tamaño a establecer en font para la ventana de salida dada.
  • Anti-Aliasing: Aquí puede establecer el modo de antialiasing para el renderizado de las ventanas de salida font. Puede optar por dejar que el IDE decida seleccionando Usar por defecto, o puede forzar su activación o desactivación seleccionando cualquiera de estas opciones. Off seleccionando cualquiera de estas opciones.
