Confirmar las preferencias del diálogo

Las preferencias del diálogo de confirmación se ocupan de algunas de las ventanas emergentes de confirmación generales que aparecerán al realizar ciertas acciones en GameMaker IDE. Las opciones son:

  • Respuesta automática al cierre de una workspace mediante un atajo de teclado: El atajo de teclado mencionado anteriormente puede utilizarse para cerrar cualquier pestaña abierta, pero a veces puede cerrar accidentalmente una pestaña vacía que necesite. Para evitarlo, puede puede configurar esta opción como "Mostrar mensaje" (la opción por defecto) para preguntar si realmente quiere cerrar esa pestaña vacía. Las otras opciones son "No", para simplemente no hacer nada si se usa el acceso directo, o "Bien" para cerrar la pestaña sin ningún mensaje.
  • Respuesta automática al reinicio de una compilación en curso: Cuando se reinicia una compilación y otra anterior sigue en ejecución, se le preguntará si desea detener primero la compilación actual. Al no marcar este mensaje se suprime ese mensaje y obligará a GameMaker a asumir que usted ha hecho clic en "Okay" como respuesta. Esta opción está configurada como "Mostrar mensaje" por defecto.
  • Respuesta automática a la advertencia cuando se intenta depurar YYC: El objetivo YYC compila en código nativo y, como tal, no puede ser depurado utilizando el debugger incorporado. Cuando intente utilizar el debugger en el objetivo YYC, recibirá una advertencia informándole de esto, y del hecho de que el proyecto será compilado usando la VM en su lugar. Esta opción está configurada por defecto como "Show Message", pero si se configura como "OK" se suprimirá esta advertencia.
  • Respuesta automática al restablecimiento de la disposición del proyecto: Cuando seleccione restablecer el diseño del proyecto (desde el menú Diseños), se le mostrará, por defecto, un mensaje de advertencia sobre cómo esto cerrará cualquier ventana de recursos abierta. Sin embargo, puede suprimir este mensaje y restablecer automáticamente el diseño seleccionando "Vale", o no hacer nada seleccionando "No". El valor por defecto es tener "Mostrar Mensaje".
