Recientes Windows Preferencias

Las preferencias aquí contienen opciones sobre cómo se mostrarán los elementos en la lista de Recientes Windows:

  • Mover las ventanas enfocadas a la parte superior: Cuando esta opción está activada y se hace clic en cualquier elemento de la lista de ventanas recientes, el elemento seleccionado se moverá a la parte superior de la lista. Por defecto, esta opción está desactivada.
  • Mostrar sólo las ventanas actualmente abiertas: Si se activa esta opción, la lista de Windows recientes eliminará los elementos que ya no estén abiertos en workspace (es decir, si se cierra cualquier ventana, también se eliminará de la lista). Por defecto, esta opción está desactivada.
  • Número máximo de ventanas a mostrar: Establece el número máximo de ventanas abiertas anteriormente que se mostrarán dentro de la lista de Recientes Windows. El valor predeterminado es 10.
