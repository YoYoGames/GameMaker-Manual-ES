Preferencias de ayuda

El manual de ayuda está diseñado para funcionar como una página web y, aunque los archivos pueden estar almacenados localmente (dependiendo de su configuración), requiere el uso de un navegador externo para verlos. Los siguientes ajustes relacionados con el manual de ayuda se pueden cambiar aquí:

  • Idioma del manual: Establece el idioma del manual. El valor por defecto es "Inglés".
  • Utilizar el manual en línea: Active esta opción para utilizar el sitio web del manual en línea cada vez que se abra el manual, o desactívela para ejecutar el manual localmente. Esta opción está desactivada por defecto.
  • Puerto del manual de ayuda: server Establece el número de puerto para la micro web manual que se utilizará, si "Usar manual en línea" está desactivado. El valor por defecto es 51290.
