Preferencias de recursos

Las preferencias aquí contienen opciones sobre cómo se dibujarán ciertos recursos en sus proyectos:

  • Color de dibujo inicial por defecto para los nuevos proyectos: Cuando utilice un font o dibuje una forma utilizando las funciones de GameMaker o las acciones de GML Visual sin especificar un color, el color establecido aquí es el que se utilizará. El color por defecto es el blanco ( ARGB: $FFFFFFFF).
  • Color inicial de la ventana por defecto para los nuevos proyectos: Esta opción se utiliza para establecer el color de fondo de la página HTML para el objetivo HTML5, así como para establecer el color que se muestra fuera de rooms en todos los objetivos (incluyendo HTML5). El color por defecto es el negro (ARGB: $000000FF).
