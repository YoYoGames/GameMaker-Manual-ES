Preferencias del editor de fuentes

Las Preferencias del Editor de Fuentes se utilizan para definir cómo se comporta el Editor de Fuentes. Las diferentes opciones disponibles se enumeran a continuación junto con una descripción de lo que hace cada una.

  • Actualizar Fonts importado desde proyectos de GameMaker 1: Cuando esto está marcado, GameMaker intentará actualizar fonts y regenerar su textura. Esto ocurrirá sólo si:
    • Esta preferencia está activada
    • El font está disponible en la máquina
    • GameMaker puede localizar una cara coincidente en la familia font (esta es la causa más probable de la inconsistencia en la importación)
    Tenga en cuenta que también ocurrirá si el font de GameMaker 1 no tiene una textura font en caché. Si un legado font no puede ser actualizado, entonces GameMaker por defecto a utilizar el caché font y el editor de Font ya no le permitirá editar el font, mostrando una opción para hacer una copia de seguridad y actualizar a otra:

    Al hacer clic en esta opción, podrá editar el font, añadiendo el font que falta o estableciendo otro, y el copia de seguridad de font se almacena en la carpeta font asset, como <font name>.yy.gms1 y <font name>.png.gms1. Para restaurar estos archivos, borre manualmente los archivos <font name>.yy y sustitúyalos por estos eliminando el .gms1 extension de los nombres de los archivos. ¡Importante! Todos los assets almacenan sus nombres en el archivo .yy, por lo que debe editarlos y poner el nombre correcto del recurso en el campo <name> también (si es diferente).
  • Elección automática para el diálogo de regeneración de contenido fallido: Cuando se va a regenerar una textura de font asset, se muestra una advertencia. Si se ajusta esta opción a "Aceptar", la font se regenera sin que se muestre la advertencia. advertencia.
  • Elección automática para el diálogo de regeneración fallida de font: Cuando la regeneración de una textura de font asset falla, se muestra una advertencia. Si se ajusta esta opción a "OK" se omitirá mostrar el mensaje.
  • Elección automática para actualizar las fuentes heredadas importadas: Cuando se importa un proyecto heredado, puede aparecer una advertencia sobre las fuentes heredadas fonts y la oferta de actualización. Si se establece esta opción en "Aceptar", se autorizará la actualización automáticamente sin mostrar este mensaje.
