Preferencias del editor de curvas de animación

Las Preferencias de las Curvas de Animación se utilizan para definir ciertas propiedades para cuando se trabaja con el Editor de Curvas de Animación. Las opciones disponibles son:

  • Respuesta automática al borrado de curvas: Esta opción puede utilizarse para controlar el mensaje emergente que se muestra cuando se borra una curva del editor de curvas de animación. La acción por defecto aquí es mostrar un mensaje de advertencia, pero puede establecer pero puede establecerlo en "Aceptar" y el diálogo no se mostrará.
