Preferencias del editor de splines

Las Preferencias del Editor de Splines se utilizan principalmente para definir el aspecto de la suavidad de paths en el Editor dePath , aunque afectarán a otros aspectos del IDE (como el Editor de formas físicas Editor). Existen las siguientes opciones:

  • Insertar path precisión del punto (px): Establece el valor de precisión (en píxeles) para insertar puntos en un path, es decir, el ratón debe estar dentro de este rango para insertar el punto en lugar de añadir uno nuevo. El valor por defecto es 20px.
  • Anchura de los puntos de anclaje: Establece la anchura de los puntos de anclaje, en píxeles. El valor predeterminado es 8px;
  • Altura delos puntos de anclaje: Establece la altura de los puntos de anclaje, en píxeles. El valor predeterminado es 8px;
  • Path color: El color que se utilizará para mostrar el path al editarlo. Por defecto es ( ARGB) #FFFFFFFF.
  • Color del segmento resaltado de path : El color que se utilizará para resaltar cualquier segmento seleccionado de path. Por defecto es (ARGB) #FFFFFFFF.
  • Color inválido de path : El color que se utilizará para resaltar un path que no es válido. Por defecto es (ARGB) #FF0000FF.
  • Color de los puntos de anclaje: El color que se utilizará para dibujar los puntos de anclaje de path. Por defecto es (ARGB) #FFC0C0.
  • Color de los puntos de anclajeinválidos: El color que se utilizará para dibujar los puntos de anclaje de path. Por defecto es (ARGB) #FF800000.
  • Color del punto de anclaje seleccionado: El color que se utilizará para dibujar los puntos de anclaje de path que han sido resaltados (seleccionados). Por defecto es (ARGB) #FF00A4FF.
  • Color del punto de anclaje no válido seleccionado: El color que se utilizará para dibujar los puntos de anclaje resaltados pero no válidos. Por defecto es (ARGB) #FFA400FF.
  • Color del punto de anclaje al pasar el ratón: El color que se utilizará para dibujar los puntos de anclaje que estén activos. Por defecto es (ARGB) #FFFFFFFF.
  • Color del punto de anclaje inválido al pasar el ratón: El color que se utilizará para dibujar los puntos de anclaje que están activos pero no son válidos. Por defecto es (ARGB) #FF0000FF.
  • Color de superposición de la máscara: Establece el color de superposición de las máscaras si están habilitadas (ver arriba). Por defecto es (ARGB) #FF404040.
