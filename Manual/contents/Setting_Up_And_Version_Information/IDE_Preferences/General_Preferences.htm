Preferencias generales

Las Preferencias Generales se utilizan para definir el comportamiento general del IDE de GameMaker. Las más generales están listadas en la página principal, mientras que muchas otras están divididas en múltiples subcategorías, explicadas más adelante en esta página.

  • Idioma del IDE: Esta opción le permite seleccionar el idioma para el IDE de la lista de opciones suministradas. Por defecto es el inglés.
  • Fuente IDE: Esta opción le permite cambiar el font utilizado en todo el IDE. Por defecto es Open Sans.
  • Velocidad máxima del editor de Game Maker : Esta opción le permite establecer la máxima velocidad de fotogramas a la que se actualizará el editor de GameMaker IDE, permitiéndole ajustar la velocidad de fotogramas para que coincida con la velocidad de actualización de su monitor. El valor por defecto es 60.
  • Piel del IDE: Aquí puede seleccionar entre los diferentes skins que puede utilizar el IDE. Por defecto GameMaker viene con dos skins, uno claro y otro oscuro, y la configuración inicial utiliza el skin oscuro.
  • Dispositivo de audiopor defecto: Aquí puede elegir el dispositivo de audio por defecto para el uso de GameMaker IDE. Puede actualizar la lista haciendo clic en el botón Actualizar .
  • Respuesta automática a la confirmación de salida: Esta opción permite establecer si se debe mostrar un mensaje de confirmación al salir de GameMaker. Por defecto está configurado como "mostrar mensaje", pero puede configurarse para omitir el mensaje como si se hubiera hecho clic en OK.
  • Respuesta automática al diálogo de cambio de sistema Font: Esta opción permite establecer si se debe mostrar un mensaje de confirmación cuando se ha añadido, eliminado o editado un sistema font. Por defecto está configurado como "mostrar mensaje", pero puede configurarse para omitir el mensaje como si se hubiera pulsado OK.
  • Habilitar notificaciones: Al marcar esta opción, IDE mostrará notificaciones sobre nuevos runtimes, etc... Esta opción está activada por defecto.
  • Tiempo de espera de la notificación (ms): Establece el tiempo (en milisegundos) que se muestra una notificación. El valor predeterminado es 6000.
  • Logout on exit: Si esta opción está marcada, IDE cerrará automáticamente la sesión del usuario cuando GameMaker se cierre. Esto está dirigido principalmente a los usuarios de Educación, ya que - cuando se marca - esto significa que cada vez que el programa se cierra el usuario se cierra la sesión y así liberar el asiento de la sesión para otros usuarios a utilizar. Esta opción está desactivada por defecto.
  • Mostrar pantalla de bienvenida al inicio: Esta opción está activada por defecto y simplemente le dice a GameMaker que muestre una breve pantalla de bienvenida al iniciar el programa. Si se desmarca esta opción, no se mostrará la pantalla de bienvenida.
  • Permitir que las pestañas vacías se cierren mediante un atajo de teclado: Puedes cerrar cualquier pestaña abierta utilizando el atajo de teclado / + W, pero si desactivas esta opción ese atajo ya no funcionará. Esta opción está activada por defecto.
  • Abrir nuevos recursos para su edición al arrastrar archivos al IDE: Ciertos recursos (como sprites o sonidos) pueden ser arrastrados al Workspace principal en el GameMaker IDE y serán detectados y añadidos como un nuevo recurso en el Navegador Asset. Por defecto, el editor de recursos correspondiente también se abrirá para su edición cuando esto ocurra, pero al desmarcar esta opción se evitará que se abran los editores y deberá abrirlos manualmente.
  • Abrir nuevos recursos para su edición en el momento de la creación: Si esta opción está marcada, cada vez que cree un nuevo recurso en el navegador Asset, se abrirá automáticamente el editor de recursos correspondiente. Esta opción está marcada por defecto y si la desmarcas tendrás que abrir manualmente todos los recursos recién creados.
  • Recargar automáticamente los archivos modificados: Si esta opción está marcada, IDE escaneará y comprobará automáticamente los archivos para ver si alguno de ellos ha sido modificado fuera de GameMaker y los recargará si es así.
  • Desactive las animaciones de transición de IDE: El GameMaker IDE tiene una serie de transiciones animadas para cosas como abrir ventanas, etc... sin embargo no son del gusto de todos y puede experimentar un aumento de rendimiento si se desactivan en máquinas más antiguas o poco potentes si están desactivadas. Puedes desactivarlas aquí, y están activadas por defecto.
  • Optar por las encuestas de opinión: Cuando esta opción está marcada, le estás diciendo a GameMaker que deseas participar en encuestas de retroalimentación, lo que significa que ocasionalmente mientras trabajas se mostrará un mensaje sobre el área en la que estás trabajando y se solicitará una calificación y retroalimentación. Esta opción está activada por defecto.
  • Habilitar la anulación de los PPP: Esta opción le permite anular la configuración de PPP por defecto con la suya propia. Primero debe habilitar la opción y luego establecer el valor de Anulación de DPi (el valor por defecto es 96) o establecer el Porcentaje del DPi nativo (valor por defecto 100%), y una vez cambiado se le pedirá que reinicie el IDE, lo que puede requerir que guarde cualquier trabajo que se esté realizando en el proyecto actualmente abierto. Tenga en cuenta que la DPi tiene un porcentaje mínimo del 50% y un máximo del 500%, y si pasa el ratón por encima del icono de información podrá ver el ajuste máximo recomendado para la pantalla que se está utilizando:

    Tenga en cuenta que si supera el ajuste de DPi recomendado para el monitor, se le mostrará una notificación que le permitirá restablecer el DPi.
  • Allow empty tabs to be closed via the keyboard shortcut: You can close any open Tabs using the keyboard shortcut  /  + W, but if you disable this option that shortcut will no longer work. This is enabled by default.

  • Recursos
