Preferencias del editor de objetos

Las Preferencias del Editor de Objetos se utilizan para definir ciertas propiedades de la ventana del Editor de Objetos. La sección principal contiene la siguiente opción:

  • Método de clic para abrir el evento: Esta opción controla si es necesario realizar un doble clic o un solo clic en un evento de object para abrirlo. Por defecto está configurado como doble clic.
  • Alfa de los eventos heredados: Esta opción permite cambiar el alfa con el que se dibujarán los eventos ¡heredados en la lista de eventos de la página object. Por defecto está fijado en 0,5 (los valores deben estar entre 0 y 1).
  • Formato de la etiqueta de eventos: Aquí puede habilitar o deshabilitar los diferentes textos posibles que se pueden aplicar a la etiqueta del evento en la lista de eventos del editor object. Por defecto sólo está habilitada la descripción del evento.
  • Añadir contenido de eventos por defecto: Por defecto esta opción está activada y lo único que hace es añadir un simple comentario en el editor de código de todos los nuevos eventos para recordarte que puedes añadir tu código allí.
  • Contenido del evento por defecto: Aquí puedes editar los comentarios que se añaden por defecto en la ventana de código para los nuevos eventos.

También existe la siguiente subcategoría para el editor Object:

Confirmar diálogos

Esta sección trata de los diálogos que se pueden mostrar mientras se utiliza el Editor Object, con la siguiente opción disponible:

  • Respuesta automática a la conversión de GML Visual a Lenguaje Gamemaker: Puede utilizar el botón derecho del ratón dentro de GML Visual workspace para seleccionar la opción de convertir todo GML Visual para ese evento en código GML. La acción por defecto aquí es mostrar un mensaje de advertencia preguntando si quieres continuar, pero puedes ponerlo en "Okay" y el diálogo no se mostrará antes de la conversión.
  • Respuesta automática al exceder el límite de la física vertex : Esta opción se puede utilizar para controlar el mensaje emergente que se muestra cuando se excede el número permitido de vertex en el editor de formas de la física. La acción por defecto aquí es mostrar un mensaje de advertencia, pero puedes establecerlo en "Okay" y el diálogo no se mostrará.
  • Elección automática al borrar un evento: Esta opción se puede utilizar para controlar el mensaje emergente que se muestra cuando se borra un evento en la lista de eventos del editor Object. La acción por defecto aquí es mostrar un mensaje de advertencia, pero puedes establecerlo en "Okay" y el diálogo no se mostrará.
