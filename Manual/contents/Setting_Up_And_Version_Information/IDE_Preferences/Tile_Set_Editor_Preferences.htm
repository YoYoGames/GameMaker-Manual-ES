Tile Set Preferencias del editor

Las Preferencias del EditorTile Set tienen la siguiente opción para el Auto Tiling:

Color de fondo de Auto tile: Esta opción establece el color de fondo dentro del editor de tile set en la sección de creación de Auto Tiles. El color por defecto es (rgba) #808080FF.

Las preferencias de Tile Set Editor también tiene una sub-sección de Diálogos de Confirmación :

La opción disponible aquí controla el mensaje que aparece cuando se cambia el tamaño de la celda del tile set y esto afectará a la forma en que se renderizan las capas del mapa de azulejos de room (resultando en capas recortadas). Por defecto se le pedirá que continúe o cancele la operación, y esta opción le permite suprimir este mensaje seleccionando "Okay" que seguirá adelante con el cambio de celdas de tile set sin tener en cuenta el impacto en las capas del mapa de azulejos de room. El valor por defecto es "Mostrar mensaje".
