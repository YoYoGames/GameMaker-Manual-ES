Preferencias de la plataforma

Esta sección del manual cubre las diferentes preferencias específicas de la plataforma que necesitan ser configuradas antes de que puedas compilar tus juegos a ciertos objetivos. Estos generalmente sólo necesitan ser configurados una vez para empezar (aunque las futuras actualizaciones de los tiempos de ejecución pueden requerir que usted cambie ciertas cosas, pero estos serán mencionados en las Notas de la versión) y se utilizan para apuntar a cualquier SDKque se requiere y establecer las opciones básicas de compilación para la forma en que desea GameMaker para compilar sus proyectos de juego para cualquier plataforma determinada. Tenga en cuenta que las plataformas disponibles dependerán de la Licencia que usted tiene y no todos ellos pueden estar disponibles.

A continuación se muestra una lista de todas las plataformas de destino disponibles para las licencias estándar:

  • Android
  • HTML5
  • Opera GX
  • iOS
  • tvOS
  • macOS
  • Windows
  • Windows UWP

Tenga en cuenta que, además de configurar estas preferencias, también tendrá que configurar los dispositivos a los que apuntan algunas de estas plataformas, de lo contrario no podrá compilar ni probar sus proyectos. Consulte la sección sobre compilación y la sección sobre el administrador de dispositivos para obtener más información.

El manual no muestra las preferencias de las diferentes consolas (Xbox, PlayStation y Nintendo Switch) debido a las diferentes restricciones legales que mantienen las diferentes compañías. Si tienes una licencia de consola, puedes obtener esta información en el servicio de asistencia de YoYo Games, pero primero debes contactar con YoYo Games con el correo electrónico de identificación de tu consola y solicitar permiso para ver las diferentes secciones.
