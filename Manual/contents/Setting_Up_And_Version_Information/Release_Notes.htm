Notas de publicación

Cada versión de GameMaker viene con su propio conjunto de notas de la versión. Estas contienen información detallada sobre los cambios que se han realizado, así como una lista completa de todos los errores que se han corregido. Se trata de una herramienta inestimable para decidir si se debe actualizar la versión actual, o cuando se encuentra algo que puede ser un error después de un lanzamiento.

Las notas de la versión están divididas en dos conjuntos diferentes debido a que tanto el IDE como el propio Runtimes pueden actualizarse de forma independiente. Puede encontrar las notas de la versión actual de IDE en el siguiente enlace:
