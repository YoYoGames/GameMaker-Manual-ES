Guía de instalación

GameMaker está diseñado para su uso en ordenadores de sobremesa macOS y Windows con las especificaciones mínimas mencionadas aquí.

Puedes descargar el instalador de GameMaker desde la página de tu cuenta YoYo en el sitio web de YoYo Games. Una vez instalado, la primera vez que ejecutes GameMaker es posible que se te pida una actualización antes de poder utilizar el producto. Una vez instaladas las actualizaciones, se te pedirá que proporciones los datos de la licencia de tu cuenta de YoYo Games (ver aquí para más información) antes de poder utilizar el producto, lo que significa que la primera vez que utilices GameMaker deberás tener una conexión a Internet. Tras el inicio de sesión inicial, debería poder utilizar el producto sin conexión, aunque periódicamente se le pedirá que se conecte para verificar la cuenta de inicio de sesión.

Hay que tener en cuenta que, dependiendo del tipo de licencia que tengas, es posible que tengas que instalar componentes adicionales o seguir pasos adicionales para configurar y utilizar GameMaker para plataformas específicas. Para obtener información completa sobre lo que requiere cada plataforma, consulte este artículo en el servicio de ayuda de YoYo Games: GameMaker Required SDKs.

Versiones estables y beta

La versión que descargues de la página web YoYo Games debería ser la versión estable actual de GameMaker, pero también puede haber otra versión " Beta" disponible que se utiliza para mostrar o probar nuevas funcionalidades antes de pasar a la versión estable final. Los detalles de la versión completa son los siguientes:

  • Estable: Esta versión es la que utilizarías para el desarrollo general y se recomienda su uso con cualquier producto comercial. Puede carecer de ciertas características disponibles en las versiones Beta, pero esto se debe a que no han sido completamente probadas o requieren trabajo adicional.
  • Beta: La versión Beta de GameMaker es para aquellos que desean probar nuevas características antes de que se añadan a la versión estable, o para aquellos que necesitan una solución para un problema. No se recomienda utilizar la versión Beta para la producción final del juego, a menos que contenga una corrección importante de un error que esté retrasando su proyecto. Puedes obtener la versión Beta de GameMaker desde la página de "Descargas" de tu cuenta YoYo, y puede ser instalada y ejecutada junto a la versión estable. Por favor, ten en cuenta que si estás ejecutando GameMaker a través de Steam , no tendrás acceso a la versión Beta del producto a través de Steam, aunque podrás descargar y utilizar la versión de escritorio.
