Guía de actualización

GameMaker IDE buscará periódicamente actualizaciones y las descargará automáticamente. Algunas de estas actualizaciones serán opcionales y puede elegir no instalarlas si no lo desea o cree que algunos de los cambios pueden romper un proyecto que está a punto de publicar. Sin embargo, hay algunas actualizaciones que incluyen correcciones o mejoras importantes que no se pueden omitir y GameMaker forzará una actualización en estos casos.

GameMaker es más que un simple IDE y tiene un número de diferentes herramientas runtime que utiliza detrás de las escenas para construir sus juegos para las diferentes plataformas disponibles. Estas herramientas son principalmente los diferentes ejecutores y compiladores que cada y compiladores que requiere cada plataforma, y en realidad puedes tener varios de cada uno en cualquier momento. Esto significa que puedes "elegir" qué versión del compilador y qué versión del ejecutor utilizar si has encontrado algún problema con el actual o si está apuntando a una versión o plataforma muy específica. Las herramientas se actualizarán automáticamente cuando haya una nueva versión disponible, pero puede utilizar las Preferencias de las Fuentes de Ejecu ción para seleccionar las que realmente desea utilizar. Para más información sobre las actualizaciones de runtime, consulte Preferencias - Runtime Feeds.
