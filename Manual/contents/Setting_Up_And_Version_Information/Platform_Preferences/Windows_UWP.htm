Preferencias de Windows UWP

Las preferencias de Windows UWP tienen las siguientes opciones:

  • Ubicación de Visual Studio: Esta opción sirve para dirigir GameMaker a la instalación de Microsoft Visual Studio en el PC de compilación.
  • Ganar la ubicación de las herramientas UWP: Esta opción sirve para dirigir GameMaker a la instalación de las herramientas UWP path en el PC de compilación.
  • Arquitectura del paquete por defecto: Por defecto, se muestra una ventana de diálogo para elegir entre las arquitecturas x86, x64 y ARM al compilar un ejecutable UWP. Puede establecer esta opción a una de esas tres, que se seleccionará por defecto cada vez que se decida compilar un ejecutable.
  • Respuesta automática a la creación de un paquete de tienda: En la ventana de diálogo que se muestra al compilar, puede elegir crear un paquete de tienda. Puede establecer esta opción en Sí o No , que se utilizará por defecto cada vez que elija compilar un ejecutable.
