Steam Preferencias

El SDK de Steam se utiliza para las construcciones de Windows, macOS y Ubuntu (Linux) que se van a distribuir a través de la plataforma Steam. Esta sección contiene las opciones de configuración necesarias para que funcione en sus proyectos:

  • Path A Steam SDK: El sistema path a donde ha instalado el SDK Steam. Una vez que haya configurado este path al SDK, tendrá que activar Steam en las Opciones de Juego y luego para cada plataforma añadir el App ID.
