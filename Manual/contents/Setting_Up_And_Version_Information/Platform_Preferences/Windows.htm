Preferencias de Windows

Las preferencias de Windows tienen las siguientes opciones:

  • Ubicación de Visual Studio: Esta opción sirve para dirigir GameMaker a la instalación de Microsoft Visual Studio en el PC de compilación.
  • Tipo de empaquetado por defecto: Al construir un paquete ejecutable final de Windows, puedes elegir entre crear un instalador (usando NSIS) o un archivo comprimido ZIP que tendrá el juego y cualquier archivo adicional dentro de él. Por defecto se te mostrará un diálogo para elegir entre estas dos opciones al compilar, pero puedes establecer esta preferencia para usar siempre una u otra. Tenga en cuenta que cuando esta opción se establece en Mostrar diálogo, en la ventana de diálogo real que aparece al compilar, al activar "Recordar la opción de empaquetado" se establecerá esta preferencia a la opción que selecciones.
