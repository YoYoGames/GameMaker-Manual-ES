Preferencias de Opera GX

Las preferencias de Opera GX tienen las siguientes opciones:

  • Ubicacióndel SDK de Emscripten: Esta es la ubicación del SDK de Emscripten, necesario para las construcciones de YYC hechas para Opera GX. Lee " Setting Up For Opera GX " para obtener información sobre la configuración del SDK.
  • Puerto web predeterminado Server : Permite cambiar el puerto predeterminado para la web de Opera GX server. El valor por defecto es 51264.
  • Dirección webpor defecto Server: Aquí puede cambiar la dirección web por defecto server para la web del Opera GX server. El valor por defecto es 127.0.0.1.
  • Web por defecto Server IPs permitidas: Aquí puede añadir cualquier IP permitida para la web server y se establecerá por defecto para todos los dispositivos añadidos a través del Target Manager.
