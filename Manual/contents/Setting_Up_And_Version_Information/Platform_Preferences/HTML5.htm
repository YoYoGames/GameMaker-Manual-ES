Preferencias de HTML5

Las preferencias de HTML5 tienen las siguientes opciones:

  • Tipo de empaquetado por defecto: Cuando construyas un paquete ejecutable HTML5 final, puedes elegir entre Empaquetar como archivos sueltos - que simplemente generará todos los archivos necesarios en una carpeta - o Empaquetar como Zip - que creará un archivo comprimido ZIP que tendrá el juego y cualquier archivo adicional dentro. Por defecto se te dará a elegir cuál de estas dos opciones quieres escoger al compilar, pero puedes establecer esta preferencia para usar siempre una u otra. Ten en cuenta que cuando esta opción está configurada para mostrar el diálogo, en la ventana de diálogo que aparece al compilar, elegir "Recordar la opción de empaquetado" establecerá esta preferencia a la opción que selecciones.
  • Puerto web predeterminado Server : Permite cambiar el puerto predeterminado para la web HTML5 server. El valor por defecto es 51264.
  • Dirección webpor defecto Server: Aquí puede cambiar la dirección web por defecto server para la web HTML5 server. El valor por defecto es 127.0.0.1.
  • Web por defecto Server IPs permitidas: Aquí puede añadir cualquier IP permitida para la web server y se establecerá por defecto para todos los dispositivos añadidos a través del Target Manager.

IMPORTANTE Cuando se utiliza un dispositivo de destino concreto (por ejemplo, un navegador), se utilizan los valores de IP y puerto establecidos para ese dispositivo concreto en lugar de estas preferencias. Consulte el Administrador de dispositivos para ver una lista de sus dispositivos con sus detalles.
