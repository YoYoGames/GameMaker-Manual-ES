Obtener la posición a lo largo de la ruta

Esta acción obtendrá la posición de una instancia a lo largo de un path. El valor devuelto se normaliza de 0 - 1, por lo que si se obtiene un resultado de, por ejemplo, 0,5, la instancia se encuentra exactamente en la mitad del path. Tenga en cuenta que debe haber llamado a la acción Start Following Path para asignar un path a la instancia para que esta acción devuelva un valor distinto de 0. El valor devuelto se almacenará en la variable de destino que usted dé, o puede flag la variable como una variable local temporal para crearla y asignar el valor.
