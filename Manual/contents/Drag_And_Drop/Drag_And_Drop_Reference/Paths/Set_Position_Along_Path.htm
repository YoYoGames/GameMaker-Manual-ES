Fijar la posición a lo largo de la ruta

Esta acción fijará la posición de una instancia a lo largo de un path. El valor se normaliza de 0 a 1, por lo que si se fija, por ejemplo, en 0,5, la instancia se moverá exactamente a la mitad del path. Tenga en cuenta que debe haber llamado a la acción Comenzar a seguir la ruta para asignar un path a la instancia para que esto tenga algún efecto.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Position La posición (de 0 a 1) a lo largo de path para colocar la instancia

Ejemplo:

El código del bloque de acción anterior inicia la instancia a lo largo de un path y luego genera un valor decimal aleatorio entre 0 y 1. Este valor se almacena en una variable local temporal y se utiliza para establecer la posición de la instancia a lo largo de path.
