Dejar de seguir el camino

Esta acción terminará el movimiento a lo largo de la path actual que la instancia está siguiendo.

Sintaxis de la acción:

Ejemplo:

El código del bloque de acción anterior comprueba el valor de una alarma y si es menor que 0 se comprueba un Switch se comprueba. Dependiendo del string almacenado en la variable de instancia que se pasa al switch, la instancia detendrá el path actual en el que se encuentra o iniciará un nuevo path antes de volver a activar la alarma.
