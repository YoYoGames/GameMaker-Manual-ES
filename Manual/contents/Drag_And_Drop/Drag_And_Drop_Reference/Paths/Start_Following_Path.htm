Empezar a seguir el camino

Esta acción iniciará una instancia siguiendo un path que ha sido creado en el Navegador Asset. Esta acción le indicará a la instancia qué path debe seguir, a qué velocidad debe seguir el path (medida en píxeles por fotograma de juego), cómo debe comportarse cuando llegue al final del y si debe seguir la posición absoluta o relativa del . comportarse cuando llegue al final de path, y si debe seguir la posición absoluta o relativa de path. Al establecer la velocidad puedes utilizar valores negativos para indicar que la instancia debe seguir el path en sentido inverso y cabe destacar que la velocidad que establezcas aquí puede ser modificada por el modificador de velocidad establecido en el editor de path para cada punto de path, así que si estableces la velocidad de path a 2 y en path uno de los puntos tiene un modificador de velocidad de 200%, entonces la velocidad real en ese punto será de 4 píxeles por fotograma del juego.

Una vez que haya establecido la velocidad inicial de path, puede establecer cómo debe comportarse la instancia cuando llegue al final de path desde las siguientes opciones (tenga en cuenta que cuando una instancia llega al final de path también se activará un evento Otro - Path Terminado):

  • Detener: Finaliza la path, deteniendo la instancia en el último punto de path.
  • Reiniciar: Continúa la path desde el principio, saltando de nuevo a la posición inicial si la path no está marcada como cerrada en el editor Path.
  • Bucle: Continúa desde la posición actual para ejecutar el path de nuevo.
  • Retroceder: Vuelve a retroceder a lo largo de la path (se consigue invirtiendo la velocidad de seguimiento de la path ), y cuando llegue al inicio de la path, volverá a retroceder, etc.

Por último, puede flag el path como Absoluto o no. Por defecto, la instancia comenzará a seguir el path desde su posición actual (la posición relativa), pero si flag la casilla de verificación Absoluto entonces comenzará a seguir el path exactamente como usted lo diseñó y colocó en el Editor Path (la posición absoluta). Sintaxis de la acción:

Argumentos:

Argumento Descripción
Path El path asset para empezar a seguir
Speed La velocidad (en píxeles por fotograma del juego) para seguir la trayectoria
On End La respuesta al llegar al final del camino
Absolute Si se inicia en posición absoluta o relativa
