Configurar Path Velocidad de seguimiento

Esta acción establecerá la velocidad, en píxeles por fotograma del juego, a la que una instancia debe seguir un path (ten en cuenta que debes haber llamado a la acción Empezar a seguir la trayectoria para asignar un path a la instancia para que esto tenga para que esto tenga algún efecto). Puedes utilizar valores negativos para indicar que la instancia debe seguir el path en sentido inverso y cabe destacar que la velocidad que establezcas aquí puede ser modificada por el modificador de velocidad establecido en el editor de path para cada punto de path, por lo que si Si estableces una velocidad de path a 2 y en path uno de los puntos tiene un modificador de velocidad del 200%, la velocidad real de píxeles por fotograma de juego en ese punto será de 4.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Speed La velocidad (en píxeles por fotograma del juego) a la que la instancia debe seguir la trayectoria

Ejemplo:

El código del bloque de acción anterior inicia una instancia siguiendo un path, luego genera un valor aleatorio entre 2 y 4 antes de utilizar este valor para establecer la velocidad a la que se mueve la instancia a lo largo de path.
