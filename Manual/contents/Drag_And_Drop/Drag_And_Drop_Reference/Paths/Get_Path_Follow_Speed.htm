Obtenga Path Velocidad de seguimiento

Esta acción obtendrá la velocidad, en píxeles por fotograma del juego, que una instancia está configurada para seguir un path (ten en cuenta que debes haber llamado a la acción Empezar a seguir la trayectoria para asignar un path a la instancia para que esto devuelva un valor distinto de 0). El valor devuelto se almacenará en la variable de destino que le des, o puedes flag la variable como una variable local temporal para crearla y asignarle el valor.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Target La variable de destino para mantener la velocidad de seguimiento devuelta path

Ejemplo:

El código del bloque de acción anterior obtiene la velocidad actual de la instancia a lo largo de la ruta asignada path y si es menor que 4, incrementa la velocidad de path en 0,1.
