Cargar el juego

Esta acción cargará una partida que haya sido previamente guardada mediante la acción Guardar partida. Usted da el nombre del archivo a cargar (como string, e incluyendo el extension) y el juego se cargará y continuará corriendo desde el punto guardado. Tenga en cuenta que esto no está diseñado como un sistema universal de guardar/cargar, y tratar de cargar un juego guardado de una ejecución anterior del proyecto puede dar errores (especialmente si ha utilizado cosas como Estructuras de Datos o Partículas), y como tal sólo debe tratar de cargar los juegos que se han guardado en el progreso de un solo juego (para cosas como puntos de control). También tenga en cuenta que el archivo no se cargará hasta el final del evento actual o script, por lo que cualquier acción después de que la acción de carga se llama todavía se llevará a cabo. Para una aproximación más completa a la carga de los datos de las partidas guardadas, ver las Acciones de Archivo.
