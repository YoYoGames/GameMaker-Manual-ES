Si el audio está en pausa

Esta acción se puede utilizar para comprobar si una instancia de cualquier recurso de sonido está actualmente en pausa en su juego. Se da el recurso de sonido a comprobar desde el explorador asset y la función devolverá true if it is paused and false si no lo está, aunque si compruebas el "no" flag, esto se invierte y la función devolverá true si el sonido no está en pausa y false en caso contrario.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el "if" se evalúa como true, mientras que cualquier otra acción que se lance en otro lugar se ejecutará después del bloque "if".
