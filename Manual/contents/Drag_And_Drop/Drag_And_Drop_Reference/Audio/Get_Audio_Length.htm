Obtener la longitud del audio

Esta acción puede utilizarse para obtener la duración de un sonido en tu juego. Debes suministrar el recurso de sonido para obtener la longitud y una variable de destino para devolver el valor también (o puedes flag el objetivo como una variable temporal para que la acción cree la variable para usarla hasta el final del evento o script). La duración se devolverá a la variable de destino (que puede hacerse temporal) en segundos y puede ser un valor decimal.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sound El recurso de sonido para obtener la longitud de
Target La variable de destino para almacenar el valor de retorno

Ejemplo:

El código del bloque de acción anterior obtiene la duración de la reproducción del recurso sonoro dado y la almacena en una variable local temporal. A continuación, se utiliza para generar un valor aleatorio entre 0 y la longitud del sonido, que se almacena en otra variable local temporal. Este nuevo valor aleatorio se utiliza para establecer la posición de inicio del y el sonido se reproduce.
