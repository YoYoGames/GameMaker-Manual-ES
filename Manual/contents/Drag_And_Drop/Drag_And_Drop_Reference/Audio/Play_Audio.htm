Reproducir audio

Esta acción hará que se reproduzca el sonido dado. Seleccione el sonido en el explorador de Asset y, a continuación, establezca si desea que el sonido se reproduzca en bucle o no (un sonido en bucle se reproducirá una y otra vez hasta que se detenga) estableciendo el "Bucle" flag.

Además, puedes ajustar la ganancia, el offset (la posición de la pista desde la que se empieza a reproducir) y el tono con el que se reproduce el sonido.

El valor final de la ganancia y el tono del sonido que se está reproduciendo es el producto de los valores de ganancia y tono establecidos para el sonido asset. El valor de offset es el valor proporcionado en el campo "Offset" o el valor de offset del activo sonoro que se está reproduciendo cuando el campo se deja vacío (véase el ejemplo 2).

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sound El sonido asset para jugar
Loop Si loop el sonido o no (desactivado por defecto)
Target La variable de destino para almacenar el valor de retorno
Gain Valor del multiplicador de ganancia (por defecto 1). El multiplicador de ganancia establecido aquí se multiplica con el multiplicador de ganancia del activo de sonido
Offset El tiempo (en segundos) para establecer el punto de inicio (por defecto 0). Los valores más largos que la longitud del sonido dado se ignoran.
El desplazamiento del activo de sonido (posición de la pista) se utiliza cuando este campo se deja en blanco.
Pitch El multiplicador de tono (por defecto 1). El multiplicador de ganancia establecido aquí se multiplica con el multiplicador de ganancia del activo de sonido

Ejemplo 1:

El código del bloque de acción anterior comprueba si se pulsa el botón del ratón y, si se detecta, comprueba una variable global para ver si no es true. Si no es true, entonces se reproduce un sonido (y se pone en bucle) y la variable global se pone a true, de lo contrario el sonido se detiene y la variable global se pone a false.
El sonido se reproduce con los ajustes por defecto: un multiplicador de ganancia y de tono de 1 y un desplazamiento de 0 (el desplazamiento por defecto del sonido asset).

Ejemplo 2:

El código del bloque de acción anterior establece primero la posición de la pista del sonido asset "snd_MusicBoss" a 20 segundos. A continuación, comprueba si la variable de instancia "skip_intro" está establecida en true. Si lo está, comienza a reproducir "snd_MusicBoss" a los 20 segundos, el valor de la posición de la pista establecido anteriormente (esto se debe a que el campo "Offset" se deja en blanco, por lo que se utiliza el offset del activo ). Los primeros 20 segundos del audio se saltan.
Si la variable de instancia "skip_intro" se establece en false, también comienza a reproducirse "snd_MusicBoss" pero, como el "Offset" se ha establecido en 0, el sonido comenzará a reproducirse desde el principio. La posición de la pista de 20 segundos definida anteriormente para el activo de sonido se ignora.
