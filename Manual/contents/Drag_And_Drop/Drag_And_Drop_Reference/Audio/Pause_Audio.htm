Pausa de audio

Esta acción se puede utilizar para pausar el sonido elegido si se está reproduciendo en ese momento. Tenga en cuenta que si ha reproducido varias instancias del mismo sonido, todas ellas se pondrán en pausa, y esto no pone en pausa ningún sonido reproducido después de que la acción haya sido acción. Puede iniciar la reproducción de un sonido en pausa utilizando la acción Reanudar audio.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sound El recurso sonoro para la pausa

Ejemplo:

El código del bloque de acción anterior comprobará una variable global y si se evalúa como true entonces el sonido dado se pausa, de lo contrario se reanuda.
