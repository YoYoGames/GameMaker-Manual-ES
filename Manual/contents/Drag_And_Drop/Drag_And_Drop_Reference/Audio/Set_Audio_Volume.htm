Ajustar el volumen de audio

Con esta acción puede ajustar el volumen de un sonido determinado. Se selecciona el sonido en el explorador de asset y luego se establece el valor del volumen. El volumen puede estar entre 0 (silencio) y 1 (volumen completo) y la escala es lineal, de modo que un valor de 0,5 sería la mitad del volumen. Tenga en cuenta que esta acción afectará a todas las instancias del sonido que se estén reproduciendo actualmente en el room, y afectará a todas las instancias futuras del sonido reproducido.

El campo "Tiempo" controla el tiempo que tarda el cambio de volumen en surtir efecto. Un valor de 0 resulta en un cambio instantáneo, mientras que un valor mayor cambiará gradualmente el volumen a lo largo de la cantidad dada de milisegundos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sound El recurso de sonido para ajustar el volumen de
Volume El nuevo volumen del sonido (de 0 a 1, por defecto 1)
Time La cantidad de tiempo durante el cual se desvanece el volumen (en milisegundos).

Ejemplo:

El código del bloque de acción anterior comprueba una variable global para ver si es true o falso. Si es true entonces se recupera el volumen del sonido dado y se almacena en una variable local temporal. El valor se comprueba para ver si no es igual a 0,5, y si no lo es, entonces el volumen del sonido se establece para desvanecerse a 0,5 en 1000 milisegundos (1 segundo). Si la variable global se evalúa como false, entonces el volumen del sonido se establece en 1. Como el valor del tiempo es 0, el cambio de volumen es inmediato.
