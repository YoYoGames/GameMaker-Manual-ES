Obtener volumen de audio

Esta acción devuelve el volumen del sonido dado. Eliges el sonido en el explorador de asset y le das una variable de destino a la que devolver el valor del volumen (puedes flag que es una variable local temporal). El valor devuelto estará entre 0 y 1, donde 0 es silencio y 1 es volumen máximo.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sound El recurso de sonido para obtener el volumen de
Target La variable de destino para almacenar el volumen devuelto

Ejemplo:

El código del bloque de acción anterior comprueba una variable global para ver si es true o falso. Si es true entonces se recupera el volumen del sonido dado y se almacena en una variable local temporal. A continuación, se comprueba si el valor no es igual a 0,5, y si no lo es, el volumen del sonido se establece en 0,5. Si la variable global se evalúa como false, entonces el volumen del sonido se fija en 1.
