Reanudar todo el audio

Esta acción se puede utilizar para reanudar todos los sonidos previamente pausados mediante Pausa de Audio o Pausa de Todo el Audio. No tendrá efecto en aquellos sonidos que no hayan sido pausados previamente.

Ejemplo:

El código del bloque de acción anterior comprobará una variable global y si se evalúa como true entonces todo el sonido se pone en pausa, de lo contrario todo se reanuda.
