Obtenga el volumen maestro

Esta acción devuelve el volumen maestro del sistema de sonido. Se da una variable de destino para devolver el valor del volumen (puede flag como una variable local temporal). El valor devuelto estará entre 0 y 1, donde 0 es silencio y 1 es volumen completo. volumen.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Target La variable de destino para almacenar el volumen devuelto

Ejemplo:

El código del bloque de acción anterior comprueba una variable global para ver si es true o falso. Si es true entonces se recupera el volumen maestro del sistema de sonido y se almacena en una variable local temporal. El valor se comprueba entonces para ver si no es igual a 0,5, y si no lo es, entonces el volumen maestro del sistema de sonido se ajusta a 0,5. Si la variable global se evalúa como false, entonces el volumen maestro se establece en 1.
