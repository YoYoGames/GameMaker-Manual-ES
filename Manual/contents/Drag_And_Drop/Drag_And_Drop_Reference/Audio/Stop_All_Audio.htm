Detener todo el audio

Esta acción puede utilizarse para detener todo el audio que se esté reproduciendo en ese momento (tenga en cuenta que aunque el audio esté marcado como pausado, se detendrá con esta función y no podrá reanudarlo más tarde).

Sintaxis de la acción:

Ejemplo:

El código del bloque de acción anterior comprueba si se pulsa el botón del ratón y, si se detecta, comprueba una variable global para ver si no es verdadera. Si no es cierto, entonces se reproduce un sonido y la variable global se establece en true, de lo contrario todo el sonido se detiene y la variable global se establece en false.
