Obtener la altura del teclado virtual

Esta acción devolverá la altura actual en píxeles del teclado virtual, en función del tamaño de la pantalla. Si el teclado no es visible, se devolverá 0.

Sintaxis de la acción:

Ejemplo:

El código del bloque de acción anterior comprueba si el teclado virtual del sistema operativo es visible, y si lo es obtiene la altura del teclado (en píxeles de pantalla) y luego dibuja algún texto por encima con la entrada del teclado.
