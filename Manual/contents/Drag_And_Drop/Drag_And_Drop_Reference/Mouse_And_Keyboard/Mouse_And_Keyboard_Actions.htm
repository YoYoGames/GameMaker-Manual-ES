Biblioteca de acciones con el ratón y el teclado

El Ratón y Teclado es donde puede encontrar las acciones necesarias para sondear y procesar la entrada de los dispositivos de teclado o ratón. Las acciones disponibles se enumeran a continuación:

Si se pulsa el ratón
Si el ratón baja
Si el ratón se libera
Si se pulsa la tecla
Si se pulsa la tecla
Si se libera la llave

El resto de las acciones aquí se refieren al teclado virtual. Si necesita texto o entrada numérica en su juego en un dispositivo de pantalla táctil, entonces puede utilizar las acciones de teclado virtual proporcionadas para utilizar el teclado OS del dispositivo. Estas acciones llamarán u ocultarán el teclado del SO y aceptarán la entrada de éste igual que un teclado físico, por lo que las acciones regulares del teclado de GameMaker deberían funcionar con los teclados virtuales (por ejemplo, la acción "If Key Released"), y cualquier entrada del teclado virtual será almacenada en las diferentes variables del teclado, como keyboard_string o keyboard_last_charetc...).

Al abrir o cerrar el teclado virtual se generarán Eventos Asíncronos del Sistema, en los que el mapa async_load  DS se poblará con los siguientes pares clave/valor:

  • "event_type" - el tipo de evento del sistema recibido, que será "virtual keyboard status" para los teclados virtuales.
  • "screen_height" - la altura del teclado virtual (en píxeles). Será 0 si el teclado es invisible.
  • "keyboard_status" - el estado actual del teclado, devuelto como una de las siguientes cadenas:
    • "oculto"
    • "oculto"
    • "mostrando"
    • "visible"

Las acciones del teclado virtual disponibles son:

Mostrar el teclado virtual
Ocultar el teclado virtual
Obtener la altura del teclado virtual
Si se muestra el teclado virtual
