Si se pulsa la tecla

Con esta acción puedes sondear el teclado para ver si alguna tecla está siendo presionada. La acción devolverá true cada vez que se mantenga pulsada la tecla del teclado, o false en caso contrario, aunque si se pulsa el modificador not se puede comprobar si la tecla no está siendo ayudada, es decir: la acción devolverá true mientras no haya ninguna tecla pulsada y false si la hay. Si sólo necesita comprobar si se ha pulsado una sola tecla, utilice la acción Si se ha pulsado la tecla.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Cualquier acción que se lance en otro lugar se realizará después del bloque "if" del bloque "if".

Argumentos:

Argumento Descripción

Not Negar la comprobación (verdadero se convierte en falso y viceversa)
