Si se muestra el teclado virtual

Esta acción se puede utilizar para comprobar si el teclado virtual del sistema operativo se está mostrando o no. La acción evaluará a true cuando el teclado esté abierto o abriéndose, o a false si el teclado virtual está cerrado o cerrándose. Si se habilita el modificador not, esta acción se invertirá y se podrá almacenar el valor devuelto true / false en una variable de destino, que puede ser marcada como temp (local).

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el bloque "if" evalúa a true, mientras que cualquier acción lanzada en otro lugar se ejecutará después del bloque "if".
