Si el ratón baja

Con esta acción se puede sondear el ratón para ver si el botón seleccionado se mantiene pulsado en ese momento. La acción devolverá true cada vez que el botón se mantenga pulsado, o false en caso contrario, aunque si se pulsa el modificador not se puede comprobar si el botón no se mantiene pulsado, es decir, la acción devolverá true mientras no haya ningún botón pulsado y false si lo hay. Si sólo necesita comprobar si se ha pulsado un solo botón, utilice la acción Si se ha pulsado el ratón.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el "if" se evalúa como true, mientras que cualquier acción lanzada en otro lugar se realizará después del bloque "if".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Mouse Button El nombre del botón del ratón a comprobar (ver esta página para información sobre las constantes del botón del ratón).
Not Negar la comprobación (verdadero se convierte en falso y viceversa)

Ejemplo:

El código del bloque de acción anterior sondea el estado del botón del ratón a cada paso y, si se mantiene pulsado, comprueba la pulsación inicial del ratón. Si la comprobación es true en la pulsación inicial, la velocidad de la instancia se establece, entonces, mientras el botón está abajo, la mezcla sprite se cambia a rojo. Si el ratón no se mantiene pulsado, se comprueba cuando se suelta el ratón para restablecer el color de la mezcla a blanco y establecer la velocidad a 0.
