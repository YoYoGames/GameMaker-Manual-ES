Buffer de carga

Con esta acción se puede cargar un buffer que fue previamente guardado con la acción Guardar Buffer. La acción creará un nuevo buffer, cargará el archivo de datos en él, y devolverá el valor de ID único para identificar el buffer a la variable. Usted proporciona el nombre del archivo a cargar (como string) y una variable para almacenar el ID devuelto de buffer. Tenga en cuenta que si flag la variable como una variable local temporal, debe asegurarse de eliminar el buffer que esta función crea antes de el final de la acción script o del evento, de lo contrario perderá la referencia a la misma y obtendrá una fuga de memoria cuando la variable sea descartada.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Filename El nombre (como string) del archivo a abrir
Target La variable de destino para mantener el valor de identificación único para el buffer creado en la carga

Ejemplo:

El código del bloque de acción anterior creará una variable para mantener los datos de buffer y luego comprobará si se ha hecho un buffer archivo guardado previamente. Si el archivo existe, entonces se carga en la variable buffer que creamos, pero si no existe, se crea, se escribe en él y luego se guarda listo para cargarlo la próxima vez.
