Borrar archivo

Con esta acción se puede eliminar un archivo que ha sido creado previamente. Para ello, debe indicar el nombre del archivo (en string) que desea eliminar. Tenga en cuenta que no puede eliminar ningún archivo que esté incluido en el paquete del juego, sólo los que han sido creados usando las acciones Cerrar Ini o Guardar Buffer (ver la sección sobre El Sistema de Archivos para más información).

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Filename El nombre (como string) del archivo a eliminar

Ejemplo:

El código del bloque de acción anterior comprobará si el archivo "checkpoint.sav" existe, y si existe, comprueba si el archivo "checkpoint_OLD.sav" existe. Si ese archivo también existe, entonces se elimina, y luego el archivo original "checkpoint.sav" se renombra a "checkpoint_OLD.sav". Por último, se guarda un buffer se guarda como "checkpoint.sav" (esencialmente estamos haciendo una copia de seguridad de un archivo buffer guardado cada vez que lo guardamos, de modo que siempre hay un archivo guardado "actual" y otro "antiguo").
