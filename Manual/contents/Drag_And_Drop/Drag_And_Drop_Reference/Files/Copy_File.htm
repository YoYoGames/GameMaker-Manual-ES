Copiar archivo

Con esta acción puedes copiar cualquier archivo que haya sido guardado por tu juego o que haya sido incluido como parte de los Archivos Incluidos. Usted da el nombre del archivo a copiar (como string) y luego el nombre del archivo a crear como copia del archivo original (también como string e incluyendo el archivo extension). El archivo dado será duplicado y se le dará el nuevo nombre.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Filename El nombre (como string) del archivo a copiar
Copy To El nombre (como string) del archivo que se va a crear como copia

Ejemplo:

El código del bloque de acción anterior comprobará si un archivo ini no existe y si no existe entonces se crea copiando otro archivo.
