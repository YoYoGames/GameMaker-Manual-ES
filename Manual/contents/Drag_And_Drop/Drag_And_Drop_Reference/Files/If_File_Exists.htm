Si el archivo existe

Con esta acción puede comprobar si existe un archivo con el nombre dado. Se selecciona el archivo que se desea comprobar (como un sting e incluyendo el extension) y la función devolverá true si existe, y false si no existe. Tenga en cuenta que esto devolverá true si el archivo existe en el paquete del juego como un Archivo Incluido o si el archivo ha sido guardado fuera mientras el juego se estaba ejecutando (ver la sección sobre El Sistema de Archivos para más información).

El código del bloque de acción anterior creará una variable para mantener los datos de buffer y luego comprobará si un archivo guardado en buffer ha sido hecho previamente. Si el archivo existe, entonces se carga en la variable buffer que creamos, pero si no existe, se crea, se escribe en él y luego se guarda listo para cargarlo la próxima vez.
