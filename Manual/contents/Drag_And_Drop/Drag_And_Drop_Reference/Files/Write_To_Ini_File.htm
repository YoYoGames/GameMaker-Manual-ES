Escribir en un archivo Ini

Con esta acción puede escribir un valor numérico o un string en un archivo Ini. Previamente debe haber abierto el fichero Ini para su lectura/escritura mediante la acción Abrir fichero Ini, y luego con esta acción se da el tipo de datos a escribir (ya sea un número real o un string encerrado en ""), así como la cabecera de la sección, la clave dentro de esa sección y el valor de la clave. Para ayudar a visualizar la estructura de un archivo Ini, vea esta imagen:

Como puede ver, los archivos Ini se dividen en secciones - donde cada sección tiene su propio nombre de cabecera - y luego cada sección se divide en varios pares clave/valor (como una estructura de datos de mapa). Tenga en cuenta que puede crear archivos Ini usted mismo utilizando cualquier software de procesamiento de texto (como el Bloc de notas en Windows) y luego incluirlos como un Archivo Incluido para leer y escribir en su juego.

¡IMPORTANTE! Tratar de leer datos de un archivo Ini que no ha sido abierto previamente usando la acción de Abrir Archivo Ini causará un error grave y bloqueará su juego.

¡IMPORTANTE! Esto sólo escribe los datos en la memoria y para escribir finalmente los datos en el archivo en el disco, debe llamar a Cerrar Archivo Ini, de lo contrario los datos no se guardarán.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Type El tipo de datos a escribir, ya sea un string (entre comillas ""), o un real number
Section La sección del archivo Ini en la que escribir (como string)
Key La clave dentro de la sección para escribir (como string)
Value El valor a escribir, ya sea un string o un número real basado en el argumento "tipo" (arriba)

Ejemplo:

El código del bloque de acción anterior abrirá un ini y luego escribirá datos en tres claves diferentes bajo la misma cabecera de sección (una string y dos números reales) antes de volver a cerrar el archivo ini (escribiendo así los datos en el disco).
