Guardar el búfer

Con esta acción puede guardar un archivo que contenga los datos de un buffer. Suministre el ID único de buffer (tal como se le devolvió cuando creó el buffer con Crear Buffer) y dé un nombre al archivo (como string, con el extension de su elección) y el buffer se escribirá en este archivo, listo para ser cargado de nuevo utilizando Load Buffer.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El valor de identificación único de buffer del buffer para guardar
Filename El nombre (como string) del archivo a guardar

Ejemplo:

El código del bloque de acción anterior creará una variable para mantener los datos de buffer y luego comprobará si se ha hecho un buffer archivo guardado previamente. Si el archivo existe, entonces se carga en la variable buffer que creamos, pero si no existe, se crea, se escribe en él y luego se guarda listo para cargarlo la próxima vez.
