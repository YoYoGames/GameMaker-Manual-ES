Dibujar Sprite

Esta acción dibujará un sprite en algún lugar de room. Usted especifica el sprite, la posición (ya sea una posición absoluta dentro de room, o una posición relativa a la instancia que hace el dibujo) y el marco de animación del sprite, si tiene múltiples cuadros (tenga en cuenta que los cuadros de la imagen se numeran de 0 en adelante). Si tiene un sprite animado y quiere que dibuje los fotogramas como si estuviera asignado al object, entonces utilice -1 para el índice de fotogramas o la variable de instancia incorporada image_index, pero tenga en cuenta que si la instancia no tiene asignado sprite entonces no se animará, y si el sprite asignado tiene un número de fotogramas diferente al que se está dibujando entonces el sprite se dibujará con fotogramas extra o se perderá.

IMPORTANTE: Esta acción no dibujará el sprite transformado, aunque haya cambiado la mezcla o la escala de la imagen. Para ello vea la acción Dibujar Sprite transformado.

NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sprite El sprite para dibujar
x La posición x a dibujar
y La posición y para dibujar
Frame Index El índice del cuadro a dibujar

Ejemplo:

El código del bloque de acción anterior dibuja una "sombra" sprite en la misma posición relativa que la y luego dibuja la instancia sprite encima.
