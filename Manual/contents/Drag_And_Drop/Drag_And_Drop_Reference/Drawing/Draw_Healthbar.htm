Dibujar la barra de salud

Con esta acción puede dibujar una barra de color para mostrar una representación gráfica de un valor. La cantidad a mostrar es un valor porcentual y debe estar dentro del rango de 0 - 100, sin embargo puedes utilizar valores mayores con la fórmula (CURRENT_Value / MAXIMUM_value) * 100, por lo que, por ejemplo, puedes usar esto para mostrar el poder, la salud, el maná, el tiempo o cualquier otra cosa que se beneficiaría de la visualización de una barra. Aparte del valor a dibujar puedes establecer el ancla de dirección para dibujar la barra de salud también (de izquierda a derecha o de derecha a izquierda y de arriba a abajo o de abajo a arriba), así como la posición, ya sea dentro de room o en relación con la instancia que llama a la acción. Además de la posición, también se puede establecer el color que se mezclará desde cada una de las cuatro esquinas de la barra de salud, creando gradientes de rojo a verde (por ejemplo, pero puede utilizar cualquier color). Sintaxis de la acción:

Argumentos:

Argumento Descripción
Direction La dirección para dibujar el contenido de la barra de salud
Value El valor (entre 0 y 100) a utilizar para la barra de salud
Left La posición de la parte izquierda de la barra de salud
Top La posición de la parte superior de la barra de salud
Right La posición de la parte derecha de la barra de salud
Bottom La posición de la parte inferior de la barra de salud
Background El color a dibujar como fondo en la barra de salud
Outline El color que se utilizará para delinear la barra de salud
Min Colour El color a mezclar a partir del valor mínimo
Max Colour El color a mezclar a partir del valor máximo
