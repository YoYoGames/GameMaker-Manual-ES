Biblioteca de acciones de dibujo

La biblioteca de acciones de dibujo es donde se encuentran las acciones necesarias para dibujar sprites, texto o formas, así como para establecer ciertas propiedades de dibujo. La mayoría de estas acciones son sólo para su uso en los diferentes Eventos de Dibujo de un object, y pueden no funcionar si se utilizan fuera de del evento de dibujo. Las excepciones a esto son las acciones Set, que pueden ser añadidas a cualquier evento y afectarán a todo el dibujo para todas las instancias posteriores.

Es importante notar que si usted agrega cualquier acción en el Evento de Dibujo principal de un object, entonces no dibujará el sprite que ha sido asignado a la instancia a menos que usted explícitamente le diga a GameMaker que lo dibuje, usando una acción como Draw Self. Básicamente, GameMaker dibujará por defecto cualquier sprite asignado a una instancia, sólo si no hay nada más en el Evento Draw.
