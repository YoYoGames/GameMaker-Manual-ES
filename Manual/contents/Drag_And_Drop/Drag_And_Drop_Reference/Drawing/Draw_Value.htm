Valor del sorteo

Con esta acción puede dibujar un título de string y/o una variable o valor en la pantalla. El argumento de la leyenda es un string que desea añadir antes del valor, y puede dejarse en blanco si es necesario, pero si desea incluir texto debe ir entre comillas. entre comillas, y el valor puede ser una variable, un valor fijo u otro string -en cuyo caso también debe ir entre comillas- pero no puede dejarse en blanco. A continuación, indique la posición en la que debe dibujarse el string, ya sea como posición absoluta o como una posición relativa a la instancia que dibuja el texto. Tenga en cuenta que el aspecto del texto cuando se dibuje dependerá de la font utilizada, el color utilizado y la alineación. Todas estas cosas se pueden cambiar antes de dibujar, como se muestra en el ejemplo siguiente.

NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Caption La leyenda a dibujar (a string), puede dejarse en blanco
Value El valor a dibujar (real, variable o string)
x La posición x a dibujar
y La posición y para dibujar
