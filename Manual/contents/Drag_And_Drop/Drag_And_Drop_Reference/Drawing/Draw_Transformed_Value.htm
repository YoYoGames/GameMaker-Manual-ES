Dibujar el valor transformado

Con esta acción puede dibujar en la pantalla un título de string y/o una variable o un valor. El argumento "caption" es un string que desea añadir antes del valor, y puede dejarse en blanco si es necesario, y el valor puede ser una variable, un valor fijo o otro string - en cuyo caso debe ir entre comillas, y también puede dejarse en blanco si es necesario. A continuación, indique la posición en la que debe dibujarse el string, ya sea como posición absoluta o como posición relativa a la instancia que dibuja el texto. Además, puede establecer la escala del texto, estirándolo a lo largo del eje horizontal o vertical, así como la rotación (los ángulos para rotar el texto son en sentido contrario a las agujas del reloj, por lo que si se establece a 90° se girará a la izquierda y -90° a la derecha). La escala se realiza por el factor dado, por lo que una escala de 0,5 reducirá el ancho del texto a la mitad y una escala de 2 duplicará el ancho, siendo 1 el valor por defecto para no escalar. Tenga en cuenta que el aspecto del texto dependerá de el font utilizado, el color utilizado y la alineación. Todas estas cosas pueden cambiarse antes de dibujar, como se muestra en el ejemplo siguiente.

NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Caption La leyenda a dibujar (a string), puede dejarse en blanco
Value El valor a dibujar (real, variable o string)
x La posición x a dibujar
y La posición y para dibujar
xscale El factor de escala x (horizontal), por defecto es 1
yscale El factor de escala y (vertical), por defecto es 1
Rotation El ángulo de rotación para dibujar el texto

Ejemplo:

El código del bloque de acción anterior establece un color, un font y luego alinea el texto para que se dibuje desde el centro antes de dibujar realmente el valor de "current_time" junto con la leyenda "TIME: ", escalada al doble de tamaño y "vibrando" ligeramente debido al cambio de ángulo aleatorio.
