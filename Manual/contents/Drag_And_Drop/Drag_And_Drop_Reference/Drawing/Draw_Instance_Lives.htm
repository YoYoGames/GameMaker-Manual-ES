Dibujar las vidas de las instancias

Esta acción dibujará un sprite para representar cada una de las vidas que la instancia tiene actualmente. El número a dibujar se toma del valor actual de la variable local de la instancia lives, que es una variable especial que se crea automáticamente dentro de la instancia si se utiliza alguna de las acciones que la requieren. Usted da el sprite a dibujar para las vidas y el orden de la pila, que puede ser Fila (horizontalmente, de izquierda a derecha), o columna (verticalmente, de arriba a abajo), así como la posición. La posición puede ser una posición absoluta dentro de room, o una relativa a la posición de la instancia que hace el dibujo, y el espacio entre imágenes se basará en la anchura o la altura de sprite. Tenga en cuenta que esto simplemente dibuja una imagen estática - la imagen inicial (fotograma 0) del sprite dado - y cualquier otro fotograma será ignorado, al igual que cualquier transformación que se haya añadido a la instancia mediante el cambio de las variables de instancia (como image_xscale o image_blend). El texto del pie de foto se dibujará precediendo al valor de la vida y, si es texto, deberá ir envuelto con "", y ambos serán formateados utilizando el dibujo actual font, el color y la alineación.

NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Caption El string para dibujar junto con el valor de la puntuación (precedente y puede dejarse en blanco)
Stack Order El orden en el que se dibujará (fila o columna)
x La posición x para dibujar las vidas en
y La posición y para dibujar las vidas en
