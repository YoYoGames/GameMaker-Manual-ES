Establecer la fuente

Con esta acción puede establecer el font que se utilizará para todas las acciones posteriores de dibujar texto. Al hacer clic en el cuadro de diálogo se abrirá la siguiente ventana en la que podrá seleccionar el font que desea utilizar:

Tenga en cuenta que esto establecerá el font a utilizar para todas las llamadas de texto de dibujo posteriores, y por lo que se puede llamar una vez al principio del juego - en cualquier caso, no sólo en el evento de dibujo - y todas las acciones de dibujo de texto utilizarán el conjunto font. Sin embargo, si necesitas múltiples fonts en tu juego, entonces tendrás que llamar a esta acción antes de dibujar cualquier texto con el font requerido.
