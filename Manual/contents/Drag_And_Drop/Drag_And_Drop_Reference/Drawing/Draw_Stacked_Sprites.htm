Dibujar sprites apilados

Esta acción dibujará en sprite un número determinado de sprites uno tras otro en una posición determinada dentro de room. Usted da el sprite a dibujar y el orden de la pila, que puede ser de Fila (horizontalmente, de izquierda a derecha), o de columna (verticalmente, de arriba a abajo), así como el número de sprites a dibujar y la posición. La posición puede ser una posición absoluta dentro de room, o una relativa a la posición de la instancia que realiza el dibujo, y el espacio entre las imágenes se basará en la anchura o la altura de sprite. Tenga en cuenta que esto simplemente dibuja una imagen estática - la imagen inicial única (fotograma 0) del sprite dado - y cualquier otro fotograma será ignorado, al igual que cualquier transformación que se haya añadido mediante el cambio de las variables de instancia (como image_xscale o image_blend).

NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sprite El sprite para dibujar
Stack Order El orden en el que se dibujará (fila o columna)
Number El número de sprites a dibujar
x La posición x a dibujar
y La posición y para dibujar

Ejemplo:

El código del bloque de acción anterior obtiene el número de instancias de object "obj_Player" y luego lo utiliza para dibujar un número de marcadores sprites en la pantalla.
