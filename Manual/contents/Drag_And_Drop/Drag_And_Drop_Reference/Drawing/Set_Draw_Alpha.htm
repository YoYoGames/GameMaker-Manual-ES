Establecer Alfa de Dibujo

Con esta acción puedes establecer el alfa a utilizar para cualquier forma (rectángulo/elipse/línea) o para cualquier texto que se dibuje posteriormente. Tenga en cuenta que esto no tiene que ser llamado antes de cada acción de dibujo, y se puede establecer una sola vez al comienzo del juego. Alfa es el ajuste que controla cómo de "transparentes" serán dibujadas las cosas y se basa en una escala de 0 a 1, donde 0 es totalmente transparente y 1 es totalmente opaco.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Alpha El valor alfa a utilizar (de 0 a 1, por defecto es 1)

Ejemplo:

El código del bloque de acción anterior establece el alfa de dibujo a un valor aleatorio entre 0 y 1.
