Dibujar uno mismo

Cuando añades cualquier tipo de acción o código al Evento de Dibujo de un object, le estás diciendo a GameMaker que te haces cargo y que le dirás qué dibujar a partir de ahora (por defecto una instancia dibujará el sprite que se le asigne en el editor object cuando no hay acciones en el Evento de Dibujo principal). Esto significa que cualquier sprite asignado ya no se dibujará a menos que lo digas explícitamente, que es cuando usarías esta acción. Esto simplemente dibuja el sprite asignado para el object con cualquier transformación que se haya establecido en eventos anteriores (como escalado, mezcla o cambios en el alfa) en la sub-imagen actual.

NOTA: Esta acción es sólo para su uso en los diversos eventos de sorteo, y no dibujará nada si se utiliza en otro lugar.

Sintaxis de la acción:

Ejemplo:

El código del bloque de acción anterior dibuja una "sombra" sprite en la misma posición relativa que la y luego dibuja la instancia sprite encima.
