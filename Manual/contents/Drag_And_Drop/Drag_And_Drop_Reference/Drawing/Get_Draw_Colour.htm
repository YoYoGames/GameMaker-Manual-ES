Obtener el color del dibujo

Con esta acción puedes obtener el color que se está utilizando para cualquier forma (rectángulo/elipse/línea) o para cualquier texto que se dibuje. La acción devolverá el color a la variable de destino que especifique, que puede haber sido creada previamente o puede ser una nueva temporal (si marca la casilla "Temp").

Ejemplo:

El bloque de acción anterior obtiene el color de dibujo actual y si no está establecido en $ffffff (blanco) se establece en ese color.
