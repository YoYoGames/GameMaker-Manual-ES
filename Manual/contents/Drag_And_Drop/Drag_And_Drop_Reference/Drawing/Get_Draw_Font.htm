Obtenga Draw Font

Con esta acción se puede obtener el valor del ID del font actualmente asignado para el dibujo. El valor devuelto será o bien -1 para ningún font asignado, o bien un valor >= 0 que represente una constante font del Navegador Asset. El valor devuelto se almacenará en la variable de destino que especifique, que puede haber sido creada previamente o puede ser una nueva temporal (si marca la casilla "Temp").
