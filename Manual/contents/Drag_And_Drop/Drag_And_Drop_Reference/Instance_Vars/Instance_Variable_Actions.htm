Variable de instancia Biblioteca de acciones

Las acciones de las Variables de Instancia están diseñadas para que puedas empezar a crear un juego y se utilizan para crear y mantener variables de ámbito de instancia específicas para score, lives y health (puedes encontrar más información sobre el ámbito de las variables scope aquí). Estas acciones crearán estas variables por ti en cualquier momento que se utilicen y están pensadas más para la comodidad o para los principiantes, y si necesitas un control más Si necesitas un control más complejo sobre los valores del juego, deberías crear tus propias variables usando las acciones apropiadas (ver aquí).

Existen las siguientes acciones para utilizar estas Variables de Instancia:
