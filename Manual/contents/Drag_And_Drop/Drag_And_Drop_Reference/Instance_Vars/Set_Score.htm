Puntuación del conjunto

Con esta acción puede crear o establecer una variable de instancia para la puntuación. Usted suministra el valor para establecer la variable de puntuación o para añadirla a la variable de puntuación (si marca la casilla relativa, el valor se añadirá al valor actual de "puntuación" actual). Tenga en cuenta que, a diferencia de las variables de instancia normales, el nombre de la variable "puntuación" se almacena internamente, por lo que para recuperar el valor de la puntuación debe utilizar la acción Obtener puntuación.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Score La cantidad a la que se debe fijar la puntuación (o a la que se debe añadir si está marcada la opción relativa )
