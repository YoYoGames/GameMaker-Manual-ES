Set Salud

Con esta acción puedes crear o establecer una variable de instancia para la salud. Usted proporciona el valor para establecer la variable de salud o para añadirla a la variable de salud (si marca la casilla relativa, el valor se añadirá al valor actual de "salud" actual). Tenga en cuenta que si desea utilizar la acción de la barra de salud de la instancia, este valor debe estar entre 0 y 100, de lo contrario puede ser cualquier cantidad que usted requiera. También tenga en cuenta que, a diferencia de las variables de instancia las variables de instancia regulares, el nombre de la variable "salud" se almacena internamente, por lo que para recuperar el valor de la puntuación es necesario utilizar la acción Obtener salud.
