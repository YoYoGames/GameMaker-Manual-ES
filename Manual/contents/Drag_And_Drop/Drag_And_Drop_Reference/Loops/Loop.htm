Bucle

La acción Loop (bucle ) será loop infinitamente hasta que se llame a una acción Break ( pausa ). Se debe tener cuidado con esta acción para asegurar que independientemente de las acciones colocadas dentro de loop, siempre se llamará a un Break, de lo contrario el juego causará un error de "no responde" en el sistema operativo de destino y dejará de funcionar. Usted agrega esta acción en el código de acción workspace y luego agrega las acciones a realizar dentro de loop, de la misma manera que agregaría una acción a un "if", es decir, colocándola al lado de la acción en lugar de debajo de ella:
