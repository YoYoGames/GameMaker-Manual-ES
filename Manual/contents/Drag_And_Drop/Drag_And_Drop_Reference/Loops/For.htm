Para

La acción For se utiliza para realizar loops que sólo termina cuando un valor alcanza la condición dada. Se da un valor inicial como variable (que puede ser marcada como una variable local temporal - esto es lo que normalmente se quiere), luego se da la condición para comprobar la variable (por ejemplo, si es menor que un determinado valor) y, por último, la cantidad para incrementar la variable en cada iteración de loop. Así, el flujo general es el siguiente:

  • Se inicializa una variable con un valor
  • Se establece la condición para comprobar esta variable con
  • Se establece cuánto debe incrementarse la variable en cada iteración del bucle
  • A continuación, se añaden acciones en el loop, y éstas se ejecutarán en cada iteración hasta que la condición devuelva false o se encuentre una acción de ruptura.

Cuando se añaden acciones a realizar dentro de loop, éstas se añaden de forma similar a como se añadiría una acción a un "si", es decir, soltándola al lado de la acción en lugar de debajo de ella:

Todas las acciones añadidas al lado se incluirán en el For y se realizarán antes de que el evento o script continúe con las acciones añadidas debajo. Tenga en cuenta que puede terminar cualquier loop prematuramente si llama a la acción Break, y cuando se utiliza For loops no tiene que incrementar y puede hacer que disminuya también, siempre y cuando la condición que utilices sea apropiada. Por ejemplo, establezca la variable inicial en 10, establezca la condición para comprobar si es menor que 0 y luego establezca el valor de incremento en -1 y el loop iterará 10 veces, contando hacia atrás desde 10 hasta 0.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Initialise Inicializar la variable que se utilizará para el recuento a un valor (se puede marcar como una variable local temporal)
Condition La condición a comprobar para romper el bucle
Increment La cantidad a incrementar (o disminuir) la variable de recuento inicial en cada iteración del bucle
