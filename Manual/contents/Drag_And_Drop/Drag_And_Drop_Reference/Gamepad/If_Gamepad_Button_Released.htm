Si se suelta el botón del Gamepad

Esta acción comprueba cuándo se ha soltado un botón del gamepad para el índice del gamepad dado. Devolverá true en el único tick de juego en el que se suelte el botón y false en el resto de ocasiones. Si activas el modificador not, esta acción se invertirá y comprobará si el botón no se ha soltado, devolviendo false en el tick de juego en el que se ha soltado y true en el resto de ocasiones.

Si el botón que se está comprobando es un botón analógico, no se detectará el desbloqueo hasta que el valor del botón haya descendido por debajo del umbral establecido mediante la acción Establecer umbral del botón del Gamepad.

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el bloque "if" evalúa a true mientras que cualquier acción que se haya lanzado en otro lugar se ejecutará después del bloque "if".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Gamepad El índice del gamepad.
Button El nombre del botón a comprobar.
Not Negar la comprobación (verdadero se convierte en falso y viceversa)

Ejemplo:

El código del bloque de acción anterior sondea el estado del botón del gamepad 0 a cada paso y si se mantiene un botón pulsado, comprueba la pulsación inicial del botón. Si la comprobación es true en la pulsación inicial, se reproduce un sonido, entonces, mientras el botón se mantiene pulsado, la velocidad de la instancia se establece en 10. Si el botón no se mantiene Si el botón no se mantiene pulsado, se realiza una comprobación al soltar el botón para volver a poner la velocidad a 0.
