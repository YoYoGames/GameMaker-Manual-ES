Por defecto

Esta acción creará un nuevo caso por defecto dentro de un interruptor. El caso por defecto es el que se realizará si el valor de entrada no coincide con ninguna otra constante de caso. Las acciones que se añaden a un caso por defecto deben ser a un lado de la acción, de la misma forma que se añadiría una acción a un "si":

El código del bloque de acción anterior obtiene el valor almacenado en una variable global y luego lo compara con los diferentes casos posibles. Si alguno de los casos coincide con el valor, entonces se ejecutará la acción correspondiente a ese caso, de lo contrario se evaluará el siguiente caso. Si ningún caso coincide con el valor introducido, se ejecutará el caso por defecto en su lugar.
