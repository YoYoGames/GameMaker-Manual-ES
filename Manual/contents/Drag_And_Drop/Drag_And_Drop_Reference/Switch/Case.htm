Caso

Esta acción creará un nuevo Caso dentro de un Interruptor. Usted da el valor para el caso y será comparado con el valor de entrada para el Switch, y si coinciden las acciones agregadas bajo el Caso serán ejecutadas. Las acciones que se añaden a un Caso para que se realicen sólo en ese Caso deben dejarse a un lado de la acción, del mismo modo que se añadiría una acción a un "si":

Tenga en cuenta que un valor Case debe ser una constante, que puede ser cualquier data type (real, string, etc...), siempre y cuando se evalúe a un solo valor en la compilación - por lo que no se pueden utilizar expresiones que dependan de variables. También tenga en cuenta que Case no puede ser utilizado fuera de un Switch y que intentar hacerlo le dará un error.

Argumentos:

Argumento Descripción
Constant El valor constante que debe evaluarse con respecto al valor de entrada Switch

Ejemplo:

El código del bloque de acción anterior obtiene el valor almacenado en una variable y luego lo compara con los diferentes casos posibles. Si alguno de los casos coincide con el valor, entonces se realizará la acción correspondiente a ese caso, de lo contrario se evaluará el siguiente caso. Después de que se haya ejecutado el switch, se establece una alarma a 30.
