Establecer Timeline Estado

Esta acción puede utilizarse para iniciar, detener, pausar o reanudar una timeline. La timeline debe haberse configurado primero para la instancia utilizando la acción Establecer línea de tiempo de la instancia, y luego puede utilizar esta acción en cualquier momento para cambiar su estado de ejecución entre las siguientes opciones

Opción Descripción
Inicie Inicie el timeline asignado a la instancia (si el timeline estaba en pausa, esto pondrá la posición de inicio en el momento 0 de nuevo).
Curriculum vitae Reanudar un timeline en pausa (si el timeline estaba parado y no en pausa, esto no hará nada).
En pausa Poner en pausa el timeline asignado a la instancia (mientras esté en pausa los momentos en un timeline no avanzarán con los fotogramas del juego).
Stop Detener la timeline asignada a la instancia.

Sintaxis:

Argumentos:

Argumento Descripción
State El estado en el que se encuentra timeline (véase la tabla anterior)

El código del bloque de acción anterior comprobará el valor mantenido en una variable global y si es true entonces se pausa el timeline, y si es false entonces se reanuda el timeline.
