Establecer la línea de tiempo de la instancia

Esta acción asigna un timeline del Navegador Asset a la instancia. Se da el timeline para asignar , y luego se puede elegir que el timeline loop cuando llegue al último momento (el siguiente fotograma después de que se ejecute el último momento volverá a loop el primer momento del timeline de nuevo), así como si desea iniciar el timeline inmediatamente o no. Si no se inicia el timeline inmediatamente se puede utilizar la acción Set Timeline State para iniciarlo más tarde en otro evento o script.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Timeline El recurso timeline para asignar a una instancia
Loop Bandera para tener la timeline loop después del último momento (por defecto está marcada)
Start Timeline Marca para que el timeline se inicie inmediatamente o no (por defecto está marcada)

Ejemplo:

El código del bloque de acción anterior asignará un timeline a la instancia que llama las acciones, así como establecer el momento inicial a 10 y luego la velocidad timeline a 2.
