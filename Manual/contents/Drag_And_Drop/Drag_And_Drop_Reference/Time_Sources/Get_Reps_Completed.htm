Obtenga Reps completadas

Esta acción obtiene el número de repeticiones que la Fuente de Tiempo dada ha completado, desde su último reinicio.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo para obtener las repeticiones completadas para
Target La variable para almacenar el valor devuelto
Temp Si la variable debe ser creada como una variable temporal
