Si existe una fuente de tiempo

Esta acción permite controlar otras acciones en función de la existencia o no de la Fuente de Tiempo dada.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo para comprobar
Not Habilite esta opción para ejecutar las acciones conectadas si la fuente de tiempo no existe

Ejemplo:


Esto comprueba si existe una Fuente de Tiempo, y si lo hace, la destruye.
