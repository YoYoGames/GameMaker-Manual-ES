Si [Fuente de tiempo] Estado

Esta acción le permite controlar otras acciones basadas en el estado de la Fuente de Tiempo dada.

Comprueba si el Constante de estado de la fuente de tiempo dado en "Estado" es (o no) el estado de la Fuente de Tiempo dada en "Fuente de Tiempo".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo para comprobar el estado de
Not Active esta opción para invertir la condición
State El Constante de estado de la fuente de tiempo para comprobar

Este bloque de código se ejecutaría idealmente en un evento de pulsación de tecla.

Comprueba si el estado de una Fuente de Tiempo es "activo", y en ese caso, pausa esa Fuente de Tiempo.

Sin embargo, si el estado es "pausado", se inicia de nuevo la Fuente de Tiempo.
