Reconfigurar la fuente de tiempo

Esta acción le permite cambiar la configuración de una Fuente de Tiempo existente, especificada en el campo "Fuente de Tiempo".

Los ajustes suministrados aquí son los mismos que los de Crear Fuente de Tiempo, excepto el padre.

Esto reiniciará y desactivará la Fuente de Tiempo para que tengas que iniciarla de nuevo.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo para reconfigurar
Period La duración del período de la Fuente de Tiempo, el tiempo que tarda en expirar
Units Las unidades en las que se expresa el periodo (segundos o fotogramas)
Callback La dirección method para llamar cuando la Fuente de Tiempo expira
Arguments OPCIONAL Una matriz que contiene los argumentos que se pasarán a la función de devolución de llamada
Repetitions OPCIONAL Cuántas veces debe ejecutarse la fuente de tiempo en total, o -1 para una repetición indefinida
Expiry Type OPCIONAL El tipo de caducidad de la fuente de tiempo
