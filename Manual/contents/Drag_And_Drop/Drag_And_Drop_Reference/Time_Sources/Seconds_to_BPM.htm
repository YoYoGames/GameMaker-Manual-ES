Segundos a BPM

Esta acción toma la duración de un latido en segundos, y devuelve un valor de latidos por minuto.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Value El valor de segundos por latido
Target La variable para almacenar el valor de pulsaciones por minuto devuelto
Temp Si la variable debe ser creada como una variable temporal
