Si [Fuente de tiempo] Padre

Esta acción le permite controlar otras acciones basadas en el padre de la Fuente de Tiempo dada.

Comprueba si la fuente de tiempo indicada en "Parent" es (o no) el padre de la fuente de tiempo indicada en "Time Source".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo para comprobar el padre de
Not Active esta opción para invertir la condición
Parent El ID de la fuente de tiempo para comparar con el padre de la Fuente de Tiempopreviamente dada
