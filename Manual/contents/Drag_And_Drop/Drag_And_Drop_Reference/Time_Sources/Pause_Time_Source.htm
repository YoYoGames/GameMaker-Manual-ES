Tiempo de pausa Fuente

Esta acción pone en pausa la Fuente de Tiempo dada.

Puede reanudarlo más tarde utilizando la fuente de tiempo de inicio.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo a la pausa

Ejemplo:


Este bloque de código se ejecutaría idealmente en un evento de pulsación de tecla.

Comprueba si el estado de una Fuente de Tiempo es "activo", y en ese caso, pausa esa Fuente de Tiempo.

Sin embargo, si el estado es "pausado", reanuda la Fuente de Tiempo.
