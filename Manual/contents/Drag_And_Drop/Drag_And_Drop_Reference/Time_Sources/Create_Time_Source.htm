Crear fuente de tiempo

Esta acción crea una nueva Fuente de Tiempo, y la almacena en la variable Destino. Lea Fuentes de Tiempo para una visión general.

Esto se basa en la función time_source_create() función. Lea su página para obtener información detallada sobre los argumentos.

Para activar una fuente de tiempo después de haberla creado, es necesario utilizar la función Iniciar fuente de tiempo.

Debes destruir una Fuente de Tiempo usando Destruir Fuente de Tiempo cuando ya no la necesites.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Target La variable donde se almacenará la nueva fuente de tiempo
Parent La fuente de tiempo principal: una fuente de tiempo integrada o una personalizada
Period La duración del período de la Fuente de Tiempo, el tiempo que tarda en expirar
Units Las unidades en las que se expresa el periodo (segundos o fotogramas)
Callback El method o función de script a la que llamar cuando la Fuente de Tiempo expira
Arguments OPCIONAL Una matriz que contiene los argumentos que se pasarán a la función de devolución de llamada
Repetitions OPCIONAL Cuántas veces debe ejecutarse la fuente de tiempo en total, o -1 para una repetición indefinida
Expiry Type OPCIONAL El tipo de caducidad de la fuente de tiempo

Ejemplo:

Este bloque de código hace lo siguiente:

  • Crea una nueva función, con la opción "Temp" activada. Esto crea una variable local del método.
    • Este método toma un argumento, msg, que imprime en el registro de salida.
  • Luego crea una nueva Fuente de Tiempo con una longitud de período de 1 segundo.
    • Especifica la variable callback_method como "Callback", y un array con un argumento para la función.
    • Esta Fuente de Tiempo está configurada para repetirse indefinidamente, ya que se especifica -1 en "Repeticiones".
  • A continuación, se pone en marcha la fuente de tiempo.

Esta fuente de tiempo ejecutará su llamada de retorno cada segundo, imprimiendo "¡Ha pasado un segundo!" en el registro de salida.
