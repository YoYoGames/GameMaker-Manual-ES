BPM a segundos

Esta acción toma un valor de pulsaciones por minuto y devuelve la duración de cada pulsación en segundos.

Esto puede ser usado cuando se crea una Fuente de Tiempo para usar un valor BPM para la Fuente de Tiempo. Es importante que dicha Fuente de Tiempo utilice segundos como unidad.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Value El valor de las pulsaciones por minuto
Target La variable para almacenar el valor de segundos por latido devuelto
Temp Si la variable debe ser creada como una variable temporal
