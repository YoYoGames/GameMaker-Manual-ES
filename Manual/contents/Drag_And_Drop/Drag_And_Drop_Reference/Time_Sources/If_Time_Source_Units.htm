Si las unidades de [Fuente de tiempo]

Esta acción le permite controlar otras acciones basadas en el tipo de unidad de la Fuente de Tiempo dada.

Comprueba si el Constante de la unidad de la fuente de tiempo dada en "Unidades" es (o no) el tipo de unidad de la Fuente de Tiempo dada en "Fuente de Tiempo".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Time Source El ID de la fuente de tiempo para comprobar el estado de
Not Active esta opción para invertir la condición
Units El Constante de la unidad de la fuente de tiempo para comprobarlo
