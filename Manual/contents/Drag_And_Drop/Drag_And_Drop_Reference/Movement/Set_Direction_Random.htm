Establecer dirección aleatoria

Esta acción se utiliza para establecer la dirección de movimiento de la instancia a una o más direcciones aleatorias elegidas entre los 8 puntos cardinales. Puede hacer clic en una o más de las 8 flechas de dirección dentro de la acción y ésta elegirá una de ellas al azar como dirección para la instancia. Si no selecciona ninguna de las flechas, la acción no tendrá ningún efecto sobre la dirección actual de la instancia.

El código del bloque de acción anterior elige entre true o false y almacena el resultado en una variable temporal (local). Esta variable se comprueba entonces y si es true se elige un conjunto de direcciones aleatorias y si es false se elige otro conjunto.
