Saltar al inicio

Esta acción se utiliza para mover una instancia desde su posición actual a la posición en la que fue creada originalmente. La posición original donde se crea una instancia se almacena en dos variables de instancia llamadas xstart y ystart. Estas variables pueden utilizarse en cualquier otra acción y también pueden establecerse para cambiar la posición inicial registrada.
