Dirección del punto de ajuste

Esta acción se utiliza para establecer una dirección a partir de un vector formado por la posición actual de la instancia y la posición dada. Se proporciona la posición x e y a la que hay que "apuntar" -puede marcarse como relativa a la posición de la instancia- y la dirección desde la instancia hasta ese punto se utilizará para la instancia.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
x El componente x del vector de dirección.
y La componente y del vector de dirección.

Ejemplo:

El código del bloque de acción anterior comprueba si existe una instancia del objeto "obj_Player" existe en el room y si lo hace entonces establece la dirección de la instancia para que apunte hacia la posición x/y de "obj_Player".
