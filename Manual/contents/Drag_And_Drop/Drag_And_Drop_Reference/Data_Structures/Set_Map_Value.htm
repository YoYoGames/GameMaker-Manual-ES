Establecer el valor del mapa

Con esta acción puedes añadir un par clave/valor en la estructura de datos del mapa. Se suministra la variable que contiene el índice del mapa (devuelta por la acción Crear mapa) y luego se da una "clave" (que es el identificador dentro del mapa para un valor), y luego el "valor" que se asociará a esa clave. Ambas claves pueden ser valores enteros (como 0, 200, 36712, etc...) o strings (como "Nombre", "Escudo", etc...), pero el valor puede ser cualquier válido data type.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Map El índice (almacenado en una variable) del mapa a establecer
Key La clave que hay que establecer (entero o string)
Value El valor a almacenar con la clave dada

El código del bloque de acción anterior crea una nueva estructura de datos de mapa y luego establece tres entradas clave/valor en ella.
