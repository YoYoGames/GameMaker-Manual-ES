Añadir a la lista

Esta acción puede utilizarse para añadir un nuevo valor (de cualquier data type) a la lista, y este valor se añadirá al final. Se suministra la variable que almacena el índice de la lista (devuelta por la acción Crear lista) y el valor a almacenar.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) de la lista a añadir
Value El valor a añadir en la lista

Ejemplo:

El código del bloque de acción anterior crea una variable de ámbito global y luego crea una nueva estructura de datos de lista, asignando su valor de índice a la variable global. El ámbito se cambia entonces para que todas las instancias de object "obj_Enemy_Parent" añadan su valor de ID de instancia único en la lista.
