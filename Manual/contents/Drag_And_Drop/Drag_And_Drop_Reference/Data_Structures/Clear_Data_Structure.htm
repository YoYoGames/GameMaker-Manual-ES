Borrar la estructura de datos

Esta acción borrará todos los valores de la estructura de datos dada. Primero debe seleccionar el tipo de estructura de datos que desea borrar y luego dar el índice de la estructura de datos (el índice es el valor que se devuelve y se almacena en una variable cuando se creó la estructura de datos originalmente). Una estructura de datos borrada se considera "vacía" (es decir, no tiene entradas), excepto las cuadrículas. En el caso de las cuadrículas, esta acción simplemente borrará todas las celdas de la cuadrícula a 0.

NOTA: Esto no libera la estructura de datos de la memoria y sólo la limpia de cualquier información. Para liberar la memoria asignada a la estructura de datos debe utilizar la acción Liberar estructura de datos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Structure El tipo de estructura de datos que está borrando (lista, mapa, cuadrícula o pila)
Variable La variable que contiene el índice de la estructura de datos que hay que borrar
