Rejilla transparente

Con esta acción puede borrar una estructura de datos de cuadrícula a un solo valor (cada celda dentro de la cuadrícula se establecerá al valor dado). Se suministra la variable que contiene el índice de la rejilla (devuelta por la acción Crear rejilla) y luego el valor para borrar la cuadrícula.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Grid El índice (almacenado en una variable) de la rejilla que hay que borrar
Value El valor al que se borrará la cuadrícula

Ejemplo:

El código del bloque de acción anterior borra la estructura de datos de la cuadrícula dada para mantener la palabra clave noone en cada celda si una variable global es menor o igual a 0.
