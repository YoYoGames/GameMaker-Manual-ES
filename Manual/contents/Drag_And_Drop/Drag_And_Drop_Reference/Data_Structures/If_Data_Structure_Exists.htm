Si existe una estructura de datos

Con esta acción puedes comprobar una variable para ver si indexa una estructura de datos o no. Primero tienes que elegir el tipo de estructura de datos que quieres comprobar (ya sea una Lista, un Mapa, una Cuadrícula o una Pila) y luego suministrar la variable que contiene la estructura de datos (como la devuelta por las acciones Crear Lista, Crear Mapa, Crear Cuadrícula y Crear Pila). La acción devolverá true si la estructura de datos existe, y false si no existe. Ten en cuenta que puedes flag "not" en la acción para comprobar si una variable no contiene un índice de estructura de datos.
