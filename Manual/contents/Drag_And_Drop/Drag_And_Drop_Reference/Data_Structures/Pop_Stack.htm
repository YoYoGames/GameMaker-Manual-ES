Pop Stack

Con esta acción puedes sacar un valor de la parte superior de una estructura de datos de la pila. Se suministra la variable que contiene el índice de la pila (devuelta por la acción Crear Pila) y luego se da la variable de destino que debe contener la variable devuelta (que puede ser marcada como una variable local temporal).

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Stack El índice (almacenado en una variable) de la pila a utilizar
Target La variable de destino para mantener el valor devuelto

Ejemplo:

El código del bloque de acción anterior comprueba si la pila dada está vacía, y si no lo está, quita el valor superior y lo almacena en una variable temporal. el valor superior y lo almacena en una variable temporal. Esto se utiliza entonces para establecer una alarma, mientras que si la pila está vacía, se introducen 10 valores aleatorios en la pila.
