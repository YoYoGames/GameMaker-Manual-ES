Obtener el valor del mapa

Con esta acción puede recuperar el valor asociado a una clave dada en una estructura de datos de mapa. Se suministra la variable que contiene el índice del mapa (tal y como lo devuelve la acción Crear mapa) y luego se da una "clave" (que es el identificador dentro del mapa para un valor), y también suministrar una variable de destino para mantener el valor devuelto (que puede ser marcado como una variable local temporal).

NOTA: Si da una clave que no existe en el mapa, la acción devolverá undefined. Puede comprobarlo utilizando la acción If Undefined.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Map El índice (almacenado en una variable) del mapa para obtener el valor de
Key La clave para conseguir (real o string)
Target La variable de destino para mantener el valor del mapa devuelto

Ejemplo:

El código del bloque de acción anterior comprueba si hay una colisión en la posición de la instancia y si se encuentra una si se encuentra una, se busca en un mapa para obtener un valor de la clave "daño", y este valor se utiliza para cambiar la variable de instancia "hp".
