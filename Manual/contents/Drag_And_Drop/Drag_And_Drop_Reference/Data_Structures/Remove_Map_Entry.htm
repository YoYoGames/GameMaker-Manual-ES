Eliminar la entrada del mapa

Con esta acción puede eliminar una clave (y su correspondiente valor) de una estructura de datos de mapa. Se suministra la variable que contiene el índice del mapa (tal y como lo devuelve la acción Crear mapa) y luego se da una "clave" (que es el identificador dentro del mapa para un valor), y la acción eliminará esa clave del mapa. Tenga en cuenta que esta acción no devuelve nada, y si necesita el valor de la clave dada debe utilizar la acción Obtener valor del mapa antes de eliminarlo.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Map El índice (almacenado en una variable) del mapa para eliminar el valor de
Key La clave a utilizar (real o string)

Ejemplo:

El código del bloque de acción anterior recupera el valor de una clave específica de la estructura de datos del mapa y luego comprueba el valor para ver si es menor o igual a 0. Si lo es, la clave se elimina del mapa.
