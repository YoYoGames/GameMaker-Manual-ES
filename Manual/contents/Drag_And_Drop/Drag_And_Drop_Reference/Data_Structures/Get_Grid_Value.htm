Obtener el valor de la rejilla

Con esta acción puede establecer un valor en una celda específica dentro de la estructura de datos de la cuadrícula. Se suministra la variable que contiene el índice de la rejilla (devuelta por la acción Crear rejilla) y luego se da la posición x e y (la columna y la fila) para la celda que se va a establecer, y finalmente el valor para esa celda. También debe establecer una variable de destino (que puede ser marcada como una variable local temporal) para almacenar el valor de la celda devuelta, y el valor puede ser cualquier data type válido. Tenga en cuenta que la posición x e y de la celda deben ser enteros dentro de los límites de la cuadrícula.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Grid El índice (almacenado en una variable) de la cuadrícula a obtener
x La posición de la celda a lo largo del eje x para obtener (la columna)
y La posición de la celda a lo largo del eje y para obtener (la fila)
Target La variable de destino que se utilizará para almacenar el valor devuelto

Ejemplo:

El código del bloque de acción anterior obtiene el valor de la cuadrícula almacenado en una variable global que corresponde a la posición de la instancia dentro de room. Este valor se comprueba y si es un valor de ID de instancia, esa instancia se destruye y la celda de la cuadrícula se establece en la palabra clave noone.
