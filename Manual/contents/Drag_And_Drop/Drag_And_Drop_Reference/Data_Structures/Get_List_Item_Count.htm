Obtener el recuento de elementos de la lista

Puede utilizar esta acción para saber cuántas entradas hay en una lista. Se suministra la variable que contiene el índice de la lista (devuelto por la acción Crear lista) y la variable de destino para almacenar el número devuelto de elementos de la lista (que puede marcarse como una variable local temporal que se utilizará hasta el final del script o del evento).

Sintaxis de la acción:

Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) para obtener el recuento de elementos
Target La variable de destino para almacenar el recuento de artículos devueltos

Ejemplo:

El código del bloque de acción anterior obtiene el tamaño de la lista referenciada en la variable global loop y luego utiliza un for loop para recorrerla y destruir las instancias con los IDs almacenados en la lista. Una vez finalizado el loop, la lista se libera de la memoria.
