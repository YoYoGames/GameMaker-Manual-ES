Obtenga el índice Tile en Pixel

Esta acción se utiliza para obtener un índice tile en una capa determinada. La capa se especifica a partir del nombre de la capa dado (un string, tal y como se utiliza para nombrar la capa en el Editor Room ), y luego se da una posición x e y para encontrar el tile del que obtener el índice. El azulejo será el que la posición dada cae en el room, y el índice será devuelto a la variable de destino, donde 0 es ningún tile, y un número entero positivo es el índice del tile. La posición x y y puede ser marcada como relativa a la posición de la instancia que llama, y la variable de destino puede ser marcada como una variable temporal (local) que puede ser utilizada hasta el final del evento.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa para establecer el tile en
x La posición x a comprobar
y La posición y a comprobar
Target La variable de destino para almacenar el índice tile

Ejemplo:

El código del bloque de acción anterior comprueba si el ratón pulsa el botón izquierdo y si se detecta uno, obtiene el índice de la celda tile en la que se encuentra la posición del ratón. Este índice de tile se comprueba entonces y si no es 0, se pone a 0.
