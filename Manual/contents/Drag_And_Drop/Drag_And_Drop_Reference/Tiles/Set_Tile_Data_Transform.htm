Establecer Tile Transformación de datos

Con esta acción puede cambiar una de las tres transformaciones de tile en los datos de tile. Usted suministra la variable que contiene los datos de tile para el tile, y luego selecciona la transformación a cambiar. Al hacer clic en la casilla "Aplicar" se aplicará esa transformación y desmarcarla la eliminará. Los datos de tile son simplemente un valor que refleja el índice de tile junto con las transformaciones aplicadas, así como cualquier máscara personalizada de tile que haya utilizado, y esta acción le permite cambiar tres de esos valores: Voltear, reflejar y rotar. Tenga en cuenta que normalmente querrá obtener los datos de tile primero usando Obtener datos de Tile en la celda o Obtener datos de Tile en el píxel, luego aplicar la(s) transformación(es) requerida(s) y luego establecer el y luego establecer el tile en el mapa tile de nuevo usando Set Tile Data In Cell o Set Tile Data At Pixel, ya que esta acción sólo establece un valor pero no lo aplica al tile del que fue tomado. Para más información sobre los datos de tile, consulte la sección GML sobre las funciones del mapa de mosaicos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Tile Los datos de tile para el cambio de tile
Transform La transformación a aplicar/eliminar

Ejemplo:

El código del bloque de acción anterior obtiene los datos de tile en la posición del ratón y luego comprueba si a tile se le ha aplicado una transformación de rotación. Si lo ha hecho, la transformación se restablece y los datos de tile vuelven a situar a tile en la posición.
