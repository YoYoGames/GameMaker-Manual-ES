Set Tile Set

Con esta acción puede configurar una capa para que utilice conjuntos de azulejos diferentes a los creados en el Navegador Asset. Se da el nombre de la capa (un string, como se define en el Editor Room ), y luego el recurso tile set a utilizar, y todos los tiles en el room de esa capa se dibujarán con el nuevo tile set.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa en la que se va a establecer el nuevo tile set.
Tileset El tile set a utilizar para la capa.

Ejemplo:

El código del bloque de acción anterior comprueba si el tiles en la capa "Floor_Tiles" está usando el tile set "tl_PalaceRuins", y si no es así, entonces se establece para usarlo.
