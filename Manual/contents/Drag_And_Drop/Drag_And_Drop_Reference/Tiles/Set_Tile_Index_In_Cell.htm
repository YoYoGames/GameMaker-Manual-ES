Establecer Tile Índice en la celda

Con esta acción puede establecer un tile en una capa dada a un nuevo índice. La capa se especifica a partir del nombre de la capa dado (un string, como se usa para nombrar la capa en el Editor Room ), y luego se da la fila y la columna de la celda dentro de la capa para obtener el tile a establecer. Por último, se proporciona el índice tile para establecer el tile (tiles se indexa de izquierda a derecha, de arriba a abajo, con el 0 en la parte superior izquierda). Tenga en cuenta que siempre puede poner el índice tile a 0 para borrar el tile.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa para establecer el tile en
Column La columna (de izquierda a derecha) de la celda
Row La fila (de arriba a abajo) de la celda
Tile El índice de tile para establecer la celda
