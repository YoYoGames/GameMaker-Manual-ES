Establecer Tile Datos en la celda

Con esta acción puede establecer los datos de tile para una celda de tile en una capa de mapa de azulejos. Los datos de tile son simplemente un valor que refleja el índice de tile junto con los valores de voltear, reflejar y rotar, así como cualquier máscara personalizada de tile que haya utilizado. Con esta acción, primero se suministra la capa a la que se apunta utilizando el nombre de la capa (un string) tal y como se define en el editor room, y luego la columna y la fila de la capa del mapa tile para la que se establecen los datos de mosaico para. La "celda" es el área de la rejilla del mapa tile que contiene el tile para el que desea establecer los datos, así que si su tiles es de 16x16, por ejemplo, y el room es de 1024x768 el mapa tile tendrá 64 columnas y 48 filas.

NOTA: La capa seleccionada debe haber sido definida como una capa de mapa Tile en el editor room, de lo contrario puede obtener errores.

El argumento final de la acción es el propio dato de la baldosa. Normalmente se recuperan los datos de tile para la celda del mapa de tile utilizando la acción Obtener datos de Tile en la celda y luego se manipulan utilizando la acción Establecer transformación de datos de Tile antes de establecerlos de nuevo con esta acción. Para más información sobre los datos de tile, por favor vea la sección GML sobre las funciones del mapa de mosaico.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa con el mapa tile para apuntar
Columns La posición de la columna de la celda a lo largo del eje horizontal para establecer los datos del mosaico para
Row La posición de la fila de celdas a lo largo del eje vertical para establecer los datos del mosaico
Data Los datos de tile que se utilizarán para fijar la celda

Ejemplo:

El código del bloque de acción anterior loops a través de cada celda tile en el room, recupera los datos de tile para la celda, los refleja, y luego establece la celda de nuevo.
