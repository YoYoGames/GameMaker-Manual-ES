Establecer Tile Datos en el píxel

Con esta acción puede establecer los datos de tile para una celda de tile que se encuentra en una posición específica en una capa de mapa de azulejos. Los datos de tile son simplemente un valor que refleja el índice de tile junto con los valores de valores de giro, espejo y rotación, así como cualquier máscara personalizada de tile que haya utilizado. Con esta acción, primero se suministra la capa a la que se apunta utilizando el nombre de la capa (un string) tal y como se define en el editor room, y luego los ejes x e y en la sala para los que se van a fijar los datos de tile. Tenga en cuenta que los datos de tile se establecerán para la "celda" del mapa tile en la que se encuentra la posición dada. Por ejemplo, si sus tile set tiles son de 16x16, entonces la capa del mapa tile contendrá tiles en celdas de 16x16 píxeles, así que si utiliza esta acción para apuntar a la posición room (8, 8), estará cambiando el tile en la celda (0, 0) del mapa tile.

NOTA: La capa seleccionada debe haber sido definida como una capa de mapa Tile en el editor room, de lo contrario puede obtener errores.

El argumento final de la acción son los propios datos de tile. Normalmente se recuperan los datos de tile para la celda del mapa de tile utilizando la acción Obtener datos de Tile en el píxel y luego se manipulan utilizando la acción Establecer transformación de datos de Tile antes de establecerlos de nuevo con esta acción. Para más información sobre los datos de tile, consulte la sección GML sobre las funciones del mapa de mosaicos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa con el mapa tile para apuntar
X La posición a lo largo del eje horizontal (X) para establecer los datos del mosaico para
Y La posición a lo largo del eje vertical (Y) para establecer los datos del mosaico
Data Los datos del mosaico que se utilizarán para fijar la celda en la posición

Ejemplo:

El código del bloque de acción anterior comprueba si hay un clic del ratón y cuando se recibe uno recupera los datos de la celda que coincide con la posición del ratón. Estos datos se modifican y se devuelven a la capa del mapa tile, girando el tile en esa celda.
