Obtener la variable Tilemap

Con esta acción puede recuperar cualquiera de un número determinado de variables para el elemento de mapa de azulejos en una capa. Cuando se crea una capa tile en el editor room, esta capa contiene un elemento de mapa de azulejos que se rellena con tiles de un conjunto de azulejos. Este elemento de mapa tile tiene ciertos valores asociados, como una posición de desplazamiento, el conjunto de baldosas que se utiliza, la anchura, la altura, etc. La lista completa de valores que puede obtener es:

  • Coordenada X: La posición a lo largo del eje x en la capa donde se ha colocado el elemento del mapa tile.
  • Coordenada Y: La posición a lo largo del eje y en la capa donde se ha colocado el elemento del mapa tile.
  • Columnas: La anchura (definida como el número de columnas de las celdas de tile ) del elemento del mapa tile.
  • Filas: La altura (definida como el número de filas de las celdas de tile ) del elemento del mapa tile.
  • Ancho de la baldosa: El ancho de una sola celda de tile.
  • Altura de labaldosa: La altura de una sola celda de tile.
  • Tile Set Resource: El recurso de conjunto de baldosas que se ha asignado al elemento de mapa tile para su dibujo.
  • Fotograma actual: El fotograma actual que se está dibujando para la animación tiles.
  • Máscara: Los datos de la máscara para el elemento del mapa tile.
  • Máscara global: Los datos de la máscara global para todos los elementos del mapa tile.

Cuando se utiliza esta acción, se suministra el nombre de la capa (un string, como se define en el Editor de Room ) para obtener los datos del elemento de mapa tile, y luego el tipo de datos que se desea recuperar (como se muestra en la lista anterior). El valor devuelto se almacenará entonces en la variable de destino que puede haber sido creada previamente o puede ser una nueva temporal (si marca la casilla "Temp"). Tenga en cuenta que puede recuperar valores adicionales haciendo clic en el icono más junto a la acción, y seleccionando otra variable y dando otra variable para almacenar el valor devuelto.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa de la que se obtienen los datos
Variable La variable para recuperar el valor de (como se indica arriba)
