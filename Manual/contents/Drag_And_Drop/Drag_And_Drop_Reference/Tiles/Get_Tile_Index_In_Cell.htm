Obtenga el índice Tile en la celda

Con esta acción se puede obtener un índice tile en una capa determinada. La capa se especifica a partir del nombre de la capa dado (un string, como el utilizado para nombrar la capa en el Editor Room ), y luego se da la fila y la columna de la celda dentro de la capa para obtener el índice de la baldosa del azulejo. Por último, se proporciona la variable de destino en la que se almacenará el índice devuelto de tile, donde 0 es que no hay tile, y un número entero positivo es el índice de tile. Tenga en cuenta que la variable de destino puede marcarse como una variable temporal (local) que puede utilizarse hasta el final del evento.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa de la que hay que obtener el tile
Column La columna (de izquierda a derecha) de la celda
Row La fila (de arriba a abajo) de la celda
Target La variable de destino para el valor de retorno

Ejemplo:

El código del bloque de acción anterior crea un loop para recorrer la fila superior de la capa room dada, y mientras lo hace comprueba si el tile indexado en la celda para la columna está a 0. Si no lo está el índice tile para esa celda se pone a 0 y la variable temporal (local) se incrementa antes de que el bucle haga otra serie de comprobaciones.
