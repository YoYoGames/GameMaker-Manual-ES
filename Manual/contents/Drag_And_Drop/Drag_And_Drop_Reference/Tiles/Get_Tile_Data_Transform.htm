Obtenga Tile Transformación de datos

Con esta acción podrá averiguar si los datos de tile de un tile han sido transformados de una de las tres maneras siguientes: volteados, reflejados o girados. Los datos de tile son simplemente un valor que refleja el índice de tile junto con las transformaciones aplicadas, y usted normalmente querrá obtener los datos de tile primero usando Obtener datos de Tile en la celda o Obtener datos de Tile en el píxel antes de usar esta acción. La acción devolverá true si si la transformación elegida se ha aplicado a los datos del mosaico, o false en caso contrario, y el valor devuelto se almacenará en la variable de destino que puede haber sido creada previamente o puede ser una nueva temporal (si marca la casilla "Temp" ). Para más información sobre los datos de tile, consulte la sección GML sobre las funciones del mapa de azulejos.

Argumentos:
