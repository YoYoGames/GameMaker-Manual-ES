Obtenga datos de Tile en Pixel

Con esta acción puede obtener los datos de tile para una celda de tile en una capa de mapa de azulejos desde una posición. Los datos de tile son simplemente un valor que refleja el índice de tile junto con los valores de volteo, espejo y rotación, así como cualquier máscara personalizada de tile que haya utilizado. Con esta acción, primero se suministra la capa a la que se apunta utilizando el nombre de la capa (un string) tal y como se define en el editor de room, y luego los ejes x e y en el room para obtener los datos del mosaico. El valor de los datos devueltos en tile coincidirá con cualquier celda del mapa de azulejos que se encuentre en ese punto. Por ejemplo, si tu conjunto de baldosas tiles es de 16x16, entonces la capa del mapa de baldosas tendrá tiles en celdas de 16x16 píxeles, así que si usas esta acción para apuntar a la posición room (8, 8), obtendrás los datos del mosaico para la celda (0, 0) del mapa de mosaicos.

NOTA: La capa seleccionada debe haber sido definida como una capa de mapa Tile en el editor room, de lo contrario puede obtener errores.

El valor devuelto se almacenará en la variable de destino que puede haber sido creada previamente o puede ser una nueva temporal (si marca la casilla "Temp"). Para obtener más información sobre los datos de tile, consulte la sección GML sobre las funciones del mapa de azulejos.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa con el mapa de azulejos a la que apuntar
X La posición a lo largo del eje horizontal (X) para establecer los datos del mosaico para
Y La posición a lo largo del eje vertical (Y) para establecer los datos del mosaico

Ejemplo:

El código del bloque de acción anterior comprueba si hay un clic del ratón y cuando se recibe uno recupera los datos de la celda que coincide con la posición del ratón. Estos datos se modifican y se devuelven a la capa del mapa de baldosas, girando la tile en esa celda.
