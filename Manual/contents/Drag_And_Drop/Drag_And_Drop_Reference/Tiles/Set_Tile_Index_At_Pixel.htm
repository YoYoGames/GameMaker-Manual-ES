Establecer Tile Índice en el píxel

Con esta acción se puede fijar un tile en una capa dada a un nuevo índice. La capa se especifica a partir del nombre de la capa dado (un string, como se usa para nombrar la capa en el Editor Room ), y luego se da una posición x e y para obtener el tile a establecer. La posición x e La posición x e y puede ser marcada como relativa a la posición de la instancia que llama, y el tile que se establece será el que la posición dada cae en el room y el tile set utilizado será el asignado a la capa en el editor de la sala editor. Por último, se suministra el índice tile para establecer el tile (tiles se indexan de izquierda a derecha de arriba a abajo, con el 0 en la parte superior izquierda). Tenga en cuenta que siempre puede poner el índice tile a 0 para borrar el tile.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Layer La capa para establecer el tile en
x La posición x a comprobar
y La posición y a comprobar
Tile El nuevo índice de la tile a utilizar

Ejemplo:

El código del bloque de acción anterior comprueba si el ratón pulsa el botón izquierdo y si se detecta una, obtiene el índice de la celda de tile en la que se encuentra la posición del ratón. Este índice de tile se comprueba entonces y si no es 0, se pone a 0.
