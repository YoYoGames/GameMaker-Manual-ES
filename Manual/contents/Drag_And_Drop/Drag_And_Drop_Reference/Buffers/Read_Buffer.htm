Buffer de lectura

Esta acción puede utilizarse para leer datos de un buffer creado previamente. Se suministra el valor de identificación único de buffer (devuelto por la acción Crear Buffer) y se da una variable para almacenar el valor devuelto. Si flag la variable como una variable local temporal, se creará para ti y se descartará de nuevo al final del evento o de la acción script. Todos los datos se devuelven como una cadena, así que si has almacenado algún número real tendrás que usar una de las acciones de tipo de datos para convertirlo de string a valor. Tenga en cuenta también que buffers se lee y se escribe secuencialmente, por lo que la lectura de un buffer moverá la posición de "búsqueda" a la siguiente entrada, lo que significa que para leer varias piezas de datos simplemente se llama a esta acción el número de veces necesario y se almacena cada dato devuelto en una variable.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El índice buffer (almacenado en una variable)
Target La variable de destino que contendrá los datos devueltos
