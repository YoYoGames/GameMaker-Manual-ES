Obtenga Buffer Tamaño

Esta acción puede utilizarse para recuperar el tamaño de un buffer, en bytes. Se suministra el valor de identificación único de buffer devuelto por la acción Crear Buffer y luego se da una variable de destino para devolver el valor del tamaño. La variable de destino puede marcarse como una variable local temporal, en cuyo caso se creará para usted y se eliminará de la memoria al final de la acción actual script o del evento.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El índice buffer (almacenado en una variable)
Target La variable de destino para mantener el valor devuelto

Ejemplo:

El código del bloque de acción anterior obtiene el tamaño del buffer y luego utiliza ese valor para establecer la posición de lectura/escritura al final de los datos almacenados actualmente.
