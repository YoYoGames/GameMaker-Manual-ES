Crear un búfer

Esta acción se puede utilizar para asignar una parte de la memoria como buffer en su juego, con la función que devuelve el valor único de identificación del búfer que debe ser almacenado en una variable y utilizado para todas las llamadas de función posteriores a la buffer. Usted puede proporcionar una variable global, una variable de instancia o hacer que la acción cree una variable local temporal para almacenar el ID de buffer. Tenga en cuenta que puede crear buffers adicionales haciendo clic en el icono más junto a la acción, y seleccionando otra variable para guardar el ID de buffer.

NOTA: Es importante que elimines de la memoria cualquier recurso creado dinámicamente como éste cuando ya no lo necesites para evitar fugas de memoria, así que cuando termines con el buffer que has creado debes liberarlo de nuevo utilizando la acción Borrar Buffer (en el evento Otros - Limpiar, por ejemplo). Esto es especialmente importante de recordar cuando se utiliza una variable local temporal ya que si no eliminas la buffer de nuevo antes de que termine el evento o la acción script entonces ya no podrás acceder a ella y puedes tener problemas más adelante.
