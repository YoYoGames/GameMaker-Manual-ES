Buffer de escritura

Esta acción puede usarse para escribir datos en un buffer. Se suministra el ID único de buffer en el que escribir (devuelto por la acción Crear Buffer) y luego el valor a escribir, que puede ser un real o un string, pero tenga en cuenta que todos los valores se almacenan como cadenas, por lo que cuando se lee de la buffer (utilizando el Buffer de Lectura) puede ser necesario utilizar una de las Acciones de Tipo de Datos para convertirlo de una cadena al valor original. Tenga en cuenta también que buffers se lee y se escribe secuencialmente y por lo tanto escribir en un buffer moverá la posición de "búsqueda" a lo largo de uno, lo que significa que para escribir varias piezas de datos simplemente llame a esta acción el número de veces necesario.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El índice buffer (almacenado en una variable)
Value El valor a añadir en el buffer
