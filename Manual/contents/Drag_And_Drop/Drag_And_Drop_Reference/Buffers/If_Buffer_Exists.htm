Si Buffer existe

Esta acción puede utilizarse para comprobar si un ID buffer dado es válido y el buffer existe realmente en la memoria. Si el ID de buffer es válido y existe, la acción devolverá true de lo contrario devolverá false. Puede flag "no" para que la acción compruebe si un buffer no existe.
