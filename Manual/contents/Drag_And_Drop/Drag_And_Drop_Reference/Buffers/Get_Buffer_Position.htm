Obtener Buffer Posición

Esta acción se puede utilizar para recuperar la posición actual de lectura/escritura en el buffer. Cuando se crea un buffer, la posición de lectura/escritura se establece en 0 bytes, y a medida que se añaden datos al buffer esta posición se moverá secuencialmente, lo que significa que puede utilizar esta acción para obtener el número de bytes en el buffer que se establece en cualquier momento. Se suministra el valor de identificación único de buffer devuelto por la acción Create Buffer y luego se da una variable de destino para devolver el valor de posición. La variable de destino se puede marcar como una variable local temporal, en cuyo caso se creará por ti y se eliminará de la memoria de nuevo al final de la acción actual script o evento.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El índice buffer (almacenado en una variable)
Position La posición de lectura/escritura (en bytes) dentro del buffer
