Si el final del búfer

Esta acción se puede utilizar para comprobar si la posición de lectura/escritura ha llegado al final de buffer. Se suministra el valor de ID único de buffer devuelto por la acción Crear Buffer y entonces la acción devolverá true si la posición de lectura/escritura ha llegado al final de todos los datos disponibles en buffer, o false si no es así. Puede flag "no" para que la acción compruebe si la posición de lectura/escritura no está al final del buffer.
