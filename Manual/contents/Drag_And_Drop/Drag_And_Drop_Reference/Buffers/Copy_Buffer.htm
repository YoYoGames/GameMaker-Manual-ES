Buffer de copia

Esta acción puede utilizarse para copiar todos los datos almacenados en un buffer a otro. Cuando se utilizan dos buffers y se copia de uno a otro, ambos buffers deben haber sido creados previamente utilizando la acción Crear Buffer, y suministrar el ID único de buffer devuelto como argumentos de origen y destino.

NOTA: No se puede copiar en el mismo buffer.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El índice buffer (almacenado en una variable) para copiar desde
Copy To El índice buffer (almacenado en una variable) para copiar en
