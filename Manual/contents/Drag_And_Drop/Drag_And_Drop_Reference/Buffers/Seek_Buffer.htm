Buffer de búsqueda

Esta acción se puede utilizar para desplazarse (buscar) a una posición dentro de un buffer. Se suministra el valor de ID único de buffer devuelto por la acción Create Buffer y, a continuación, el valor de desplazamiento en bytes al que hay que desplazarse. El valor de desplazamiento se aplicará desde el inicio de buffer, por lo que al llamar a esta función se moverá la posición de lectura/escritura al inicio de buffer y luego se moverá el número de bytes dado y se posicionará la nueva posición de lectura/escritura allí. Sin embargo, si se marca el desplazamiento como relativo, moverá la posición de lectura/escritura hacia adelante (o hacia atrás si el desplazamiento es un valor negativo) desde la posición actual.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Buffer El índice buffer (almacenado en una variable)
offset El desplazamiento (en bytes) relativo a la posición de búsqueda actual
