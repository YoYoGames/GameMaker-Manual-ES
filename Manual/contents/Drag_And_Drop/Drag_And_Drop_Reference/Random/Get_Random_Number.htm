Obtener número aleatorio

Esta acción generará un número aleatorio dentro de los límites de un conjunto de valores mínimos y máximos. Puede elegir el tipo de número a generar (ya sea un decimal o un entero) y luego establecer el rango dentro del cual se generará el número suministrando un valor mínimo y máximo. El valor aleatorio generado será devuelto a la variable de destino (que puede ser marcada como una variable local temporal).

NOTA: Al generar un entero aleatorio la acción devolverá un valor entero entre el valor mínimo y el máximo , ambos inclusive, por lo que un rango de 5 - 10 podría devolver 5 o 10, así como cualquier entero entre ambos. Sin embargo, al generar un decimal aleatorio no devolverá nunca el valor máximo, por lo que el rango de 5 a 10 puede devolver 5 pero no devolverá 10 (aunque podría devolver 9.999999999999).

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Type El tipo de números a utilizar (enteros o decimales)
Minimum El número mínimo para el rango aleatorio
Maximum El número máximo para el rango aleatorio
Target La variable de destino para almacenar el valor devuelto
