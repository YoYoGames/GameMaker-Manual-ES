Aleatorizar

Esta acción establece la "semilla" para el generador de números aleatorios a un valor aleatorio. Cuando se utilizan las funciones de números aleatorios en GameMaker la semilla inicial es siempre la misma, ya que esto hace que el seguimiento de los errores y debugging mucho más fácil (compilado juegos compilados no tienen este comportamiento). Si desea probar con el verdadero azar, debe llamar a esta acción una vez al inicio del juego.
