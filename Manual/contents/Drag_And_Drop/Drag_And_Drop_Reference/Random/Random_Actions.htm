Obtener número aleatorio
Aleatorizar
Elija

Vale la pena señalar que GameMaker utilizará la misma semilla al generar números aleatorios cada vez que ejecute el juego desde el IDE. Esto es para facilitar debugging y significa que obtendrá los mismos resultados iniciales al generar valores aleatorios, pero esto no ocurrirá cuando haya compilado el juego en un ejecutable (se generará una nueva semilla cada vez que se ejecute el juego). Si no desea este comportamiento al hacer pruebas, entonces debe llamar a la acción Randomise una vez al inicio del juego, antes de generar cualquier valor aleatorio.
