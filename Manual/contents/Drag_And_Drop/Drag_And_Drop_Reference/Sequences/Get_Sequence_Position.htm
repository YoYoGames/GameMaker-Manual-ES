Obtener Sequence Posición

Con esta acción se puede recuperar la posición actual del elemento sequence dado en room. Se proporciona el ID del elemento sequence (devuelto por la acción Crear secuencia), y esta acción devolverá su posición X e Y en variables de destino. Tenga en cuenta que las variables de destino pueden ser marcadas como variables temporales (locales), que serán creadas para la acción y pueden ser utilizadas en todas las acciones posteriores para el evento en el que se encuentra la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence para obtener la posición.
Target X La variable de destino para mantener la posición x.
Target Y La variable de destino para mantener la posición y.

Ejemplo:

El código del bloque de acción anterior recupera la posición actual del elemento sequence ID almacenada en la variable "my_seq". Estos valores se utilizan entonces para fijar la posición de sequence, moviéndola 10 píxeles a la derecha.
