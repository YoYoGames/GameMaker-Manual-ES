Secuencia de juego

Con esta acción se puede iniciar la reproducción de un sequence (previamente pausado). Se suministra el ID del elemento sequence (devuelto por la acción Crear secuencia), y la acción iniciará su reproducción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence a reproducir.

Ejemplo:

El código del bloque de acción anterior comprueba si una variable global es true. Si lo es, entonces el elemento sequence al que se hace referencia en la variable "my_seq" se pone en pausa, de lo contrario se pone en reproducción.
