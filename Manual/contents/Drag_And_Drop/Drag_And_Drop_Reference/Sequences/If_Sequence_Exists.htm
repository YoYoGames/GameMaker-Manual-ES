Si Sequence existe

Esta acción se puede utilizar para comprobar si el elemento sequence dado existe en el juego room. Se da el ID del elemento, tal y como lo devuelve la acción Crear Secuencia, y si el elemento sequence existe en el room entonces la acción devolverá true de lo contrario devolverá false.

Si flag el argumento "Not", la acción comprobará si no existe tal elemento sequence y si no se encuentra ninguno devolverá true.

Tenga en cuenta que para añadir acciones en el bloque "if", se deben dejar caer al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el bloque "if" se evalúa como true, mientras que las acciones que se lancen en otro lugar se ejecutarán después del bloque "if".
