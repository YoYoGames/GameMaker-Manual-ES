Secuencia de destrucción

Puede llamar a esta acción siempre que desee "destruir" un elemento creado previamente en sequence. Se le da el ID del elemento, tal y como lo devuelve la acción Crear Sequence Elemento, y esta acción lo destruirá, eliminándolo del room.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence a destruir.

Ejemplo:

El código del bloque de acción anterior comprueba si el elemento sequence almacenado en la variable "my_seq" existe y, si lo hace, lo destruye.
