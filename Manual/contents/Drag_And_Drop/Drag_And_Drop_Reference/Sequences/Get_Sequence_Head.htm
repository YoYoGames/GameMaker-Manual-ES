Consiga Sequence Head

Con esta acción puede recuperar la posición actual del cabezal de reproducción del elemento sequence dado en el room. Suministre el ID del elemento sequence (tal y como lo devuelve la acción Crear Secuencia), y esta acción devolverá el fotograma actual en el que se encuentra el cabezal de reproducción de la secuencia, en la variable de destino. Para obtener información sobre el cabezal de reproducción, véase Uso de la hoja de cálculo.

Tenga en cuenta que la variable de destino puede ser marcada como temporal (local), que se creará para la acción y puede ser utilizada en todas las acciones posteriores para el evento en el que se encuentra la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence para obtener la posición de la cabeza.
Target La variable objetivo para mantener la posición de la cabeza.

Ejemplo:

El bloque de acción anterior recupera la posición actual de la cabeza del elemento sequence ID almacenado en la variable "seq_id". Ese valor se utiliza entonces para fijar la posición de la cabeza del mismo elemento sequence, adelantándolo 4 cuadros.
