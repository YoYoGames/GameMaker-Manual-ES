Crear secuencia

Con esta acción puede crear un elemento sequence de cualquier sequence desde el Navegador Asset. Usted da el nombre del sequence y una posición x/y dentro del room donde debe ser creado, así como la capa en la que debe ser creado (esta capa debe ser una capa de activos). Puede marcar la opción "relativa" flags para posicionar el sequence en relación con la instancia que ejecuta los bloques de código de acción. Cada vez que se crea un sequence en el juego de esta manera, se le da un valor de ID de elemento único con el que se puede identificar el sequence en acciones posteriores, por lo que esta acción permite definir una variable "objetivo" para mantener este valor de ID (como se muestra en el ejemplo siguiente). Tenga en cuenta que el objetivo puede marcarse como una variable temporal (local), que se creará para la acción y podrá utilizarse en todas las acciones posteriores para el evento en el que se encuentre la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sequence El nombre de la sequence para crear un elemento.
x La posición x en el room para crear el sequence (puede ser relativa a la instancia que llama).
y La posición y en el room para crear el sequence (puede ser relativa a la instancia que llama).
layer El nombre (a string) de la capa asset para crear el sequence en.
Target La variable a la que se dirige el valor de retorno.
