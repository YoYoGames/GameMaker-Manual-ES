Obtener Sequence Longitud

Con esta acción se puede recuperar la longitud (en fotogramas) del elemento sequence dado en room. Se suministra el ID del elemento sequence (devuelto por la acción Crear secuencia), y esta acción devolverá su longitud en la variable de destino. Este será el número de fotogramas que el Sequence ejecuta.

Tenga en cuenta que la variable de destino puede ser marcada como temporal (local), que se creará para la acción y puede ser utilizada en todas las acciones posteriores para el evento en el que se encuentra la acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence para obtener la longitud.
Target La variable de destino para mantener la longitud.

Ejemplo:

El bloque de acción anterior recupera la longitud del ID del elemento sequence almacenado en la variable "seq_id". Ese valor se utiliza entonces para fijar la posición de la cabeza del mismo elemento sequence en 10 fotogramas antes de su final.
