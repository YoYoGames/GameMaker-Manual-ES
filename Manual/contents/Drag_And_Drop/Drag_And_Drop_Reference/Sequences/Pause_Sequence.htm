Secuencia de juego

Con esta acción puede pausar la reproducción de un elemento sequence. Usted suministra el ID del elemento sequence (tal y como lo devuelve la acción Crear Secuencia), y la acción pausará su reproducción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence a pausar.

Ejemplo:

El código del bloque de acción anterior comprueba si una variable global es true. Si lo es, entonces el elemento sequence al que se hace referencia en la variable "my_seq" se pone en pausa, de lo contrario se pone en reproducción.
