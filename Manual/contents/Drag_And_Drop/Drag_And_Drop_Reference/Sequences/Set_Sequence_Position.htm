Establecer Sequence Posición

Con esta acción se puede establecer la posición del elemento sequence en room. Se proporciona el ID del elemento sequence (devuelto por la acción Crear secuencia), y luego la nueva posición X e Y. Si usted Si marca la casilla "relativa", los valores dados para X e Y se añadirán a la posición actual de sequence, de lo contrario serán una posición absoluta.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Variable El ID del elemento sequence para fijar la posición.
X La nueva posición x en el room para el sequence.
Y La nueva posición y en el room para el sequence.

El código del bloque de acción anterior recupera la posición actual del elemento sequence ID almacenada en la variable "my_seq". Estos valores se utilizan entonces para fijar la posición de sequence, moviéndola 10 píxeles a la derecha.
