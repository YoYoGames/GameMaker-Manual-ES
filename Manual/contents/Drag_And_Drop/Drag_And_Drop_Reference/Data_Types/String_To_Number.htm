Cadena a número

Esta acción puede convertir un string en un número. Usted proporciona el string para cambiar y luego da una variable de destino para devolver el número real de nuevo a (esta variable puede ser marcada como una variable local temporal). Tenga en cuenta que si el string tiene algún carácter que no sean números y un punto decimal, no se convertirá, aunque si el string comienza con un número, éste se convertirá y se excluirá el resto del string. Es importante tener en cuenta que cuando se pasan decimales a esta acción acción, el string devuelto se redondeará a dos decimales solamente. A continuación se muestra una tabla de valores de entrada y lo que se espera que se devuelva

Cadena Valor devuelto
"123" 123
"123 Hello World" 123
"Hello World" 0 (por defecto, si el string no se puede convertir, se devolverá 0)
"Hello World 123" 0
"123.45678" 123.46

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Value El string para convertir
Target La variable de destino para mantener el valor devuelto

Ejemplo:

El código del bloque de acción anterior inicializa una variable global y luego carga un archivo con un buffer guardado en él. Se leen dos strings del buffer y se almacenan en una variable local temporal antes de convertir uno de ellos en un número real y almacenarlo en otra variable local temporal. A continuación, se asignan a variables globales, y el buffer es eliminado.
