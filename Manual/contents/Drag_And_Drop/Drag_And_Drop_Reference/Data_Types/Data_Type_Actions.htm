Tipos de datos Biblioteca de acciones

GameMaker tiene un número de diferentes data types, como enteros, strings, punteros, etc... y algunos de ellos pueden ser convertidos a otros. Esta sección del manual muestra las diferentes acciones disponibles para tratar y convertir diferentes data types.

NOTA: puede encontrar más información sobre los diferentes data types aquí.

La biblioteca de Tipos tiene las siguientes acciones:

Decimal a entero
Cadena a número
Número a cadena
