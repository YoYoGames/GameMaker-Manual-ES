Número a cadena

Esta acción convertirá un número real en un string. Si el número no es un valor entero y tiene decimales, éstos serán redondeados a los dos lugares más cercanos, así, por ejemplo, el número 123.45678 devolvería el string "123.46".

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Value El número real a convertir
Target La variable de destino para mantener el valor devuelto

Ejemplo:

El código del bloque de acción anterior toma un valor decimal y lo convierte en un entero, almacenando el valor devuelto en una variable local temporal. Este valor se convierte entonces en un string y se dibuja en la pantalla.
