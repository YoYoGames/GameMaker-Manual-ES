Ejecutar script

Esta acción puede utilizarse para llamar a una función de script o a una función definida por el usuario. En general, esta acción sólo se utilizaría para el soporte de legado y no debería usarse realmente en proyectos actuales o nuevos. Para utilizar la acción se le da a script asset el nombre de la variable de la función definida por el usuario a la que se va a llamar, y luego se le dan los argumentos que necesita, es decir, los valores (strings, números enteros, índices de recursos, etc.) que desea introducir en el script o función para que pueda utilizarlos. Puede añadir los argumentos que necesite haciendo clic en el icono del signo más junto a la acción.

Algunas funciones de scripts y definidas por el usuario también pueden devolver un valor para su uso posterior en la lista de bloques de acción, por lo que se puede especificar la variable de destino que debe almacenar este valor de retorno. Esta puede haber sido creada previamente o puede ser una nueva variable temporal local (si marca la casilla "Temp"). Tanto el campo de destino como el de argumento pueden dejarse en blanco si no son necesarios.

Tenga en cuenta que, por lo general, es mejor utilizar la acción Llamada a la Función para llamar a funciones definidas por el usuario, y no debería necesitar utilizar esta acción, o llamar a un script asset directamente en el código.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Script El nombre de la función script o definida por el usuario a la que se va a llamar.
Argument0 ... Argument3 Los diferentes argumentos (valores) que se van a pasar a la script o a la función (los argumentos no utilizados pueden dejarse en blanco)
Target El nombre de la variable a la que se dirigen los valores devueltos (puede dejarse en blanco)
