Si la expresión

Esta acción se utiliza para comprobar que una expresión es true o false, y puede comprobar aquí cualquier expresión lógica siempre que se evalúe a uno u otro valor. Por ejemplo, se puede utilizar cualquiera de las siguientes expresiones:

x < (room_width / 2)
a == b
++val >= 100

Si flag el argumento "No", entonces las expresiones anteriores serán negadas, por ejemplo "==" se convierte en "not=". También tenga en cuenta que la expresión no está limitada a valores numéricos y puede comprobar strings, punteros, IDs de recursos o cualquier otra cosa que una función pueda devolver o utilizar (vea la sección sobre tipos de datos para más información), pero no puede comparar dos data types diferentes sin obtener un error - por ejemplo si intenta utilizar un string y un entero en la expresión obtendrá un error. Para obtener más información sobre las expresiones, consulte aquí.

NOTA IMPORTANTE: Debido a problemas de precisión en coma flotante, comprobar si dos valores son exactamente iguales puede devolver false, ya que uno puede ser exactamente 1, mientras que el otro puede ser 1.00000000000001. Esto puede evitarse utilizando la acción Decimal a Entero antes de la comprobación o utilizando las expresiones "mayor que (>)" o "menor que (<)".
