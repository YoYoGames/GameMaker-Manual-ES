Nueva Estructura

Esta acción sólose utiliza en cuando se crea structs utilizando una función que ha sido marcada como función constructora (ver Declarar una nueva función para más información). Esta acción toma el método de una función previamente definida, así como un número de argumentos, que puede ampliar para añadir más si es necesario utilizando el icono de la izquierda. Estos argumentos deben corresponder a las entradas requeridas por la función, y se utilizarán para rellenar el struct que se está creando. El struct será devuelto a la variable de destino, que puede ser marcada como una variable local temporal o no.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Function El nombre de la función script o definida por el usuario a la que hay que llamar.
Argument0 ... Argument3 Los diferentes argumentos (valores) que se van a pasar a la script o a la función (los argumentos no utilizados pueden dejarse en blanco)
Target El nombre de la variable a la que se dirigen los valores devueltos (puede dejarse en blanco)
