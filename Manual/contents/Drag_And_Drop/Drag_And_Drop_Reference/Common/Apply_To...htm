Aplicar a...

Esta acción se puede utilizar para cambiar el Ámbito de Acción para todas las acciones posteriores que se encadenen a ésta. Al igual que con una if si se colocan más acciones acciones a la derecha de la acción Aplicar a... entonces se ejecutarán desde el ámbito object o instancia. El ámbito se elige en el menú emergente que se muestra al hacer clic en la pequeña flecha situada en la parte superior derecha de la acción, y de este modo se puede indicar un conjunto diferente de objects - o incluso una sola instancia si tienes el ID de la instancia - para ejecutar múltiples acciones antes de continuar con el código de la actual object.

Tenga en cuenta que para añadir acciones en el bloque "Se aplica a", se deben dejar caer al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora desde la object o la instancia a la que se dirige el bloque "Se aplica a", mientras que cualquier acción lanzada en otro lugar se ejecutará después del bloque "Se aplica a".
