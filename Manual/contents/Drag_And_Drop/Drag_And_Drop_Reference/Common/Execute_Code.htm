Ejecutar código

Esta acción simplemente añade un bloque de editor de código en la lista de GML Visual donde puede añadir cualquier código válido de GML que necesite para el object. Tenga en cuenta que todo el código de la acción se ejecutará antes de que se ejecute la siguiente acción de la lista.
