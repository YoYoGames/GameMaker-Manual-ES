Establecer la forma de las partículas

Esta acción puede utilizarse para establecer la forma incorporada en sprite que una partícula debe utilizar al dibujar. Se proporciona el valor de identificación único del tipo que se va a definir (tal y como se devolvió cuando se creó el tipo con la acción Crear tipo de partícula) y la forma a utilizar.
