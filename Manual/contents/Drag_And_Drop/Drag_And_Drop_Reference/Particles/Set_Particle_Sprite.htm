Establecer Sprite de Partículas

Esta acción puede utilizarse para establecer el sprite que una partícula puede utilizar al dibujar. Se proporciona el valor de identificación único del tipo que se va a establecer en sprite (tal y como se devolvió cuando se creó el tipo con la acción Crear tipo de partícula) y el recurso sprite a utilizar. Si el sprite tiene sub-imágenes, entonces la partícula se animará a través de los cuadros a la velocidad establecida en el Editor de Imágenes para el sprite. También tenga en cuenta que si usted establece un valor de color para la partícula (utilizando la acción Establecer Color de la Partícula ) entonces este color se mezclará con el sprite utilizado, al igual que el establecimiento de la mezcla de la imagen en una instancia se mezclará un color con la instancia sprite (utilice el blanco para no mezclar).

NOTA: Si utiliza esta acción, no debe utilizar la acción Establecer forma de partícula, ya que sólo puede tener una sprite o una forma, pero no ambas.

Argumentos:

Type El valor de identificación del tipo para establecer el sprite de
Sprite El sprite para asignar al tipo de partícula
