Establecer el tamaño de las partículas

Esta acción se utiliza para establecer el tamaño de la partícula, que puede ser estático o configurado para aumentar/disminuir en el tiempo. Se suministra el valor de identificación único del tipo para establecer el tamaño (como se devuelve cuando se crea el tipo con la acción Crear tipo de partícula) y luego dar un rango de tamaños iniciales de partida, a continuación, la cantidad de aumento y, finalmente, el "meneo". El tamaño con el que se crea una partícula es un valor escalar, donde 1 es el mismo tamaño que la imagen base de la partícula, 0,5 sería la mitad del tamaño de la imagen base y un valor de 2 sería el doble. Si los tamaños mínimo y máximo no son iguales, el tamaño inicial de cualquier partícula creada será un valor aleatorio entre ellos.

La opción de aumentar el tamaño de una partícula te permite introducir un valor que se añadirá al valor escalar base en cada fotograma de juego de la vida de la partícula. Por lo tanto, estableciendo este valor a, por ejemplo, 0,1, se añadirá 0,1 al valor de la escala en cada fotograma del juego. Tenga en cuenta que puede utilizar valores negativos aquí también para hacer que la imagen de las partículas se reduzca también. Ten en cuenta que el mínimo decremento de escala que se acepta es -0.01 y el mínimo incremento es 0.01.

Por último, puedes establecer la opción "wiggle". Se trata de un valor que se generará aleatoriamente entre 0 y el valor que usted dé y que luego se sumará o restará al tamaño en cada paso de la vida de las partículas, haciendo que la imagen dibujada fluctúe en tamaño. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede ser incluso negativo, estando el rango máximo entre -20 y 20.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Type El valor de identificación único para el tipo de partícula para establecer el tamaño de
Min Size La escala mínima para el tipo de partícula (por defecto 0,1)
Max Size La escala máxima para el tipo de partícula (por defecto 1)
Increase El valor que se añade al tamaño del tipo de partícula en cada fotograma del juego (puede ser negativo, mínimo +/-0,01)
Wiggle El valor máximo a sumar o restar aleatoriamente del tamaño de las partículas por paso (+/-20).
