Fijar las partículas secundarias

Esta acción le permite utilizar un segundo tipo de partícula junto con la partícula primaria. Debes dar el valor de ID del tipo de partícula primaria (que debe ser el mismo que el valor usado cuando creaste la partícula usando la acción Crear Tipo de Partícula ) y luego puedes elegir que emita un tipo de partícula secundaria cada fotograma de juego que exista y/o al final de la vida de cada partícula primaria. Debe dar otro valor de ID de partícula para la "Secundaria" y luego un número que se creará en el "Recuento".

Si el Evento seleccionado es "Por Paso", entonces cada fotograma del juego en el que exista una partícula primaria en la pantalla, ésta emitirá el número dado de partículas secundarias - por ejemplo, podrías tener un cohete con partículas de "llama", y cada partícula de llama llama emite una partícula secundaria "humo". Si el evento se establece como "Al morir", entonces las partículas secundarias sólo se emitirán cuando una partícula del tipo primario llegue al final de su vida útil - por ejemplo, podrías tener una partícula "explosión" que emita una partícula "polvo" al final de su vida.

Es importante tener en cuenta que no se puede utilizar el mismo ID de tipo de partícula como la partícula primaria como para las partículas secundarias, y que puede encontrar muy rápidamente problemas de rendimiento cuando se utiliza esta acción, ya que es fácil de crear miles de partículas muy rápidamente de esta manera - así que tenga cuidado de limitar la vida útil y el número de partículas que se crean por los tipos de partículas primarias y secundarias. También hay que tener en cuenta que se puede asignar una partícula secundaria a cualquier tipo de partícula y así se pueden crear "cadenas de partículas" en las que una partícula crea otra a cada paso y luego esta partícula crea otra cuando llega al final de su vida útil, etc. Aunque, de nuevo, hay que tener cuidado al encadenar partículas de esta manera para asegurarse de que que el rendimiento no se vea afectado. También puedes llamar a esta acción dos veces para un solo tipo de partícula y hacer que fluya en el paso así como que estalle al morir.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Type ID El valor de la ID del tipo de partícula para dar la partícula secundaria a
Event Cuándo crear la partícula secundaria, ya sea "Por paso" o "Al morir"
Secondary El valor de identificación del tipo de partícula que se emitirá en el evento secundario elegido
Count El número de partículas secundarias que se crearán en el Evento secundario elegido
