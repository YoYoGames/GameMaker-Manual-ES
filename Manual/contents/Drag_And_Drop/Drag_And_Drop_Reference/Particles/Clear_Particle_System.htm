Sistema de partículas transparentes

Con esta acción puedes borrar el sistema de partículas dado a su estado por defecto, eliminando todos los emisores (tampoco es necesario llamar a la acción Destruir Emisor de Partículas ya que esto se hace automáticamente) y borrar todas las partículas de la pantalla. Se suministra el valor de ID único para el sistema que se va a borrar, tal y como se devolvió cuando se creó el sistema con la acción Crear Sistema de Partículas. Hay que tener cuidado al utilizar esta acción ya que destruye todos los emisores asociados al sistema de partículas - esto significa que cualquier instancia en el room que utilice un emisor puede dar un error ya que ese emisor ya no existe, así que asegúrese de que la acción sólo se llama cuando no vayas a crear ninguna partícula nueva a partir de ese emisor (para más información sobre los emisores de partículas consulta Crear emisor de partículas).

NOTA: Esta acción sólo borrará las partículas visibles en el room, pero no borra las propiedades de las partículas ni las elimina de la memoria. Para ello debe utilizar la acción Destruir tipo de partícula.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
System El valor de identificación único para el sistema que se va a destruir
