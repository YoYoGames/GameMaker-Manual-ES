Establecer el color de las partículas

Esta acción establecerá el color o los colores que se utilizarán cuando se cree su sprite y durante su vida útil. Primero se especifica el valor del ID del tipo de partícula (que debe ser el mismo que el valor devuelto al crear la partícula mediante la acción Crear tipo de partícula ) y luego se pueden establecer los colores a utilizar.

Puedes elegir tres colores para usar, y el sprite se mezclará entre ellos durante el tiempo de vida de la partícula. Por ejemplo, si una partícula tiene colores rojo, verde y amarillo, y un tiempo de vida de 60 fotogramas, entonces cuando se crea (fotograma 0) será rojo, luego se desvanecerá hacia el verde a la mitad de su vida (fotograma 30), antes de desvanecerse en amarillo al final de su vida (fotograma 60). Si está utilizando un sprite personalizado en lugar de una forma de partícula, cualquier color que no sea blanco se mezclará con el sprite utilizado.

NOTA: El componente alfa del color no se utiliza, sólo los componentes RGB del selector de color. Para establecer el alfa utilice la acción Establecer Alfa de Partículas.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Type El valor de identificación único para el tipo de partícula para establecer el color de
Start El color inicial a utilizar
Middle El color intermedio a utilizar
End El color final a utilizar

El código del bloque de acción anterior creará un nuevo tipo de partícula y asignará su valor único de ID a una variable global. A continuación, procede a establecer todas las propiedades del tipo de partícula.
