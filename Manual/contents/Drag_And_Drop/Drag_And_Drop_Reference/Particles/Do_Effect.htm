Efecto Do

Esta acción creará un efecto de partículas incorporado, ya sea por encima o por debajo de las capas que se están dibujando en su juego room. Se elige el tipo de efecto que se va a crear en la lista desplegable (que se muestra en la imagen de abajo), y luego se establece la posición (que puede ser relativa a la posición de la instancia) y selecciona un tamaño - pequeño, mediano o grande - para el efecto. También puede establecer el color que se utilizará -el valor alfa se ignorará- y decidir si quiere que el efecto se cree debajo de todas las capas en el room, o por encima de todas las capas en el room. Tenga en cuenta que si elige utilizar los efectos de lluvia o nieve, los valores de posición y tamaño no se tendrán en cuenta, ya que el efecto está diseñado para ser visible en toda la pantalla.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Type El tipo de efecto de partículas que se va a crear (ver imagen superior)
Where Si las partículas deben crearse por encima o por debajo de las capas de room
Size El tamaño del efecto (pequeño, mediano o grande)
X La posición x dentro de room para crear el efecto (en píxeles)
Y La posición y dentro de room para crear el efecto (en píxeles)
Colour El color del efecto (se ignorará el alfa)

Ejemplo:

El código del bloque de acción anterior comprobará si hay un clic del ratón y si se detecta uno creará un pequeño efecto en la posición del ratón.
