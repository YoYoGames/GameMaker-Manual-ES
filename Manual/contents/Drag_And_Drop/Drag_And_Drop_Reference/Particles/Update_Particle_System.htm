Actualizar el sistema de partículas

Esta acción forzará al sistema de partículas dado a actualizar la posición, el color, el alfa, etc... Usted suministra el valor de ID único para el sistema a actualizar (como se devuelve cuando usted creó el sistema con la acción Crear Sistema de Partículas), y cada vez que llames a la acción el sistema dado se actualizará un fotograma del juego, lo que significa que puedes - por ejemplo - llamar a esta acción dentro de una repetición loop y hacer que el sistema de partículas se actualice varias veces en un solo cuadro de juego (útil para cuando quieras que un nivel tenga algún efecto que requiera que las partículas estén presentes al principio, como la nieve o la lluvia). También puedes usar esta acción cuando el sistema de partículas ha sido pausado usando el Sistema de Partículas Pausado para actualizar manualmente las partículas en el sistema.
