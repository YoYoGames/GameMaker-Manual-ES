Fijar el Alfa de la Instancia

Este bloque de acción establece el valor de image_alpha para la instancia. El alfa de la imagen es el valor de "transparencia" para dibujar cualquier sprite asignado a la instancia y tiene un valor entre 0 y 1, donde 0 sería totalmente transparente y 1 sería totalmente opaco. Tenga en cuenta que si el sprite tiene píxeles transparentes o semitransparentes, su transparencia se escalará por la cantidad, por lo que si tiene algunos píxeles transparentes al 50% en el sprite y establece el alfa de la instancia en 0,5, el alfa final para los píxeles transparentes será del 25%. Puede establecer este valor para que sea una cantidad relativa

Por favor, ten en cuenta que para que los cambios en esta variable sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker dibujará por defecto el sprite) o ser dibujado usando la acción Draw Self .

Sintaxis de la acción:

Argumentos:

Argumento Descripción
alpha El valor alfa a establecer (de 0 a 1, por defecto es 1).

Ejemplo:

El código del bloque de acción anterior establece un nuevo sprite, así como una serie de otras propiedades para la visualización de ese sprite, incluyendo la configuración del alfa de la imagen a 0,5.
