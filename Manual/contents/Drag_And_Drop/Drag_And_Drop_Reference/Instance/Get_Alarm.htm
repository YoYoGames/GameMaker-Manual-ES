Obtenga la alarma

Este bloque de acción recupera el valor de una cuenta atrás de alarma y lo almacena en la variable de destino dada. Si usted flag la variable de destino como "Temp" entonces esto creará una nueva variable temporal (local) para mantener el valor de retorno para usted, como se muestra en el ejemplo siguiente.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Alarm La alarma que hay que conseguir.
Target La variable de destino que se utilizará para almacenar el valor de la alarma.

Ejemplo:

El código del bloque de acción anterior crea una variable local (temporal) y almacena el valor actual de la alarma[0]. A continuación se comprueba y si es menor que 0, la alarma se pone a 60.
