Convocatoria de evento para padres

Este bloque de acción ejecutará el evento del padre object del que la instancia es hija antes de continuar ejecutando las acciones en la instancia actual. Cuando se asigna un padre object a una instancia, todos los eventos de ese object son "heredados" por el hijo. por el hijo. Sin embargo, si añades un evento al hijo object, éste anula el evento del padre. Usando este bloque de acción, puedes forzar a un hijo object a ejecutar tanto el evento padre como el evento de la propia instancia. Para más información sobre los padres, consulte la sección del Editor de Objetos.
