Establecer la rotación de la instancia

Este bloque de acción establece la image_angle (rotación) de la instancia. El ángulo se mide en grados, siendo el derecho 0°, el superior 90°, el izquierdo 180° y el inferior 270°. Si se establece esta variable a 0, se restablece la instancia, lo que significa que dibujar cualquier sprite asignado a como fue definido en el editor sprite. Tenga en cuenta que para que los cambios en esta acción sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker dibujará por defecto el sprite) o ser dibujada usando la acción Draw Self.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Angle El nuevo ángulo a utilizar (de 0 a 360).

Ejemplo:

El código del bloque de acción anterior establece un nuevo sprite así como una serie de otras propiedades para la visualización de ese sprite, incluyendo la rotación de 180°.
