Establecer la velocidad de la animación

Este bloque de acción establece el image_speed (velocidad de animación) para el sprite asignado a la instancia. Se establece el número ideal de fotogramas por segundo en el Editor de Sprite y luego se puede utilizar este valor para modificarlo. La velocidad de animación por defecto está fijada en 1, lo que significa que el juego intentará mantener el número de fotogramas por segundo que hayas definido para el sprite, pero puedes utilizar valores superiores a 1 para animar más rápido (pueden saltarse fotogramas), o valores decimales para animar más lento (los fotogramas se mostrarán en varios pasos) o incluso valores negativos para invertir la velocidad de la animación.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Speed El nuevo valor de la velocidad del modificador.

Ejemplo:

El código del bloque de acción anterior establece un nuevo sprite, así como una serie de otras propiedades de cómo se mostrará ese sprite, incluyendo la configuración para que se anime a la mitad de la velocidad definida.
