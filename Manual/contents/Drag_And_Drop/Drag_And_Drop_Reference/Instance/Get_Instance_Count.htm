Obtener el recuento de instancias

Con esta acción puede encontrar el número de instancias de un determinado object actualmente activas dentro de room. Puede seleccionar cualquier object de la lista de recursos o puede utilizar la palabra clave especial all para obtener el número de todas las instancias activas. También es necesario suministrar una variable de destino para devolver el valor y poder utilizarlo en otras acciones. Si flag la variable de destino como "Temp" entonces esto creará una nueva variable temporal (local) para mantener el valor de retorno, como se muestra en el siguiente ejemplo.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Object El nombre de la página object que hay que comprobar.
Target La variable a la que se dirige el valor de retorno.
