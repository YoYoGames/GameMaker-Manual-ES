Establecer el color de la instancia

Este bloque de acción establece el color de image_blend para "mezclarlo" con la instancia sprite. El valor por defecto es el código hexadecimal $FFFFFFFF (que representa el color blanco usando el formato ARGB ) y esto dibujará el sprite sin mezcla y con alfa completo. Cualquier otro valor (incluyendo las constantes de color internas como c_red, o c_aqua) mezclará el color especificado con el original sprite así como establecerá el componente image_alpha para la instancia, pero sólo cuando la casilla Usar Alfa esté marcada. Esto sobrescribirá cualquier valor alfa establecido previamente usando la acción Establecer Alfa de la Instancia.

Por favor, tenga en cuenta que para que los cambios en esta acción sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker dibujará por defecto el sprite) o ser dibujado usando la acción Draw Self. Es importante importante tener en cuenta también que se debe intentar limitar la mezcla en la plataforma HTML5 (a menos que se utilice WebGL), ya que cada sprite mezclado tiene que ser almacenado en caché por separado y, por lo tanto, tener muchos sprites mezclados puede afectar negativamente al rendimiento.

Argumentos:

Argumento Descripción
Colour El nuevo color de mezcla a utilizar (al hacer clic en la muestra de color se abrirá la ventana del selector de color).

El código del bloque de acción anterior establece un nuevo sprite así como una serie de otras propiedades de cómo se mostrará ese sprite, incluyendo el ajuste del color de la mezcla a rosa.
