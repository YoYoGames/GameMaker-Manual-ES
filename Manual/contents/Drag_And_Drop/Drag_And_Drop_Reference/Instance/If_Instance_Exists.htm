Si la instancia existe

Esta acción se puede utilizar para comprobar si existe una instancia de un object específico en el juego room, o si existe una instancia específica en sí misma. Usted da el object desde el explorador asset, y si alguna instancia existe en el room entonces la acción devolverá true de lo contrario devolverá false. Tenga en cuenta que puede utilizar aquí una variable con el ID único de la instancia específica (por ejemplo, como el devuelto por la acción Crear Object Instancia) o incluso el ID de la instancia devuelto por el editor de room al añadir la instancia.

Si flag el argumento "No", entonces la acción comprobará si no existe ninguna instancia y si no se encuentra ninguna devolverá true y si incluso existe una devolverá false.
