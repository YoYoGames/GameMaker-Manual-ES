Establecer Sprite

Este bloque de acción establece el sprite_index de la instancia a otro sprite del Navegador Asset. Se selecciona el sprite al que se va a cambiar, y luego se da la imagen (cuadro de animación) que se va a mostrar. Ten en cuenta que si no quieres que el sprite se anime después de establecer el marco, tendrá que utilizar la acción de Establecer Velocidad de Animación y establecerla en 0, o si quiere que el sprite continúe animando entonces establezca el valor para usar la variable incorporada image_index en su lugar.

Al cambiar el sprite también cambiará la máscara de colisión de la instancia, a menos que también haya especificado un mask_index separado en el Editor de Objetos. Tenga en cuenta que para que los cambios en esta acción sean visibles, la instancia no debe tener ningún evento de dibujo (y así GameMaker dibujará por defecto el sprite) o ser dibujado usando la acción Draw Self.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Sprite El nuevo sprite a utilizar (-1 si quiere eliminar el sprite).
Frame El fotograma de la animación que se mostrará inicialmente (si no se utilizan fotogramas adicionales, se utilizará 0).
