Llamar a un evento de usuario

Con esta acción puede llamar a cualquiera de los 12 Eventos de Usuario disponibles. Un Evento de Usuario es uno al que puede añadir scripts o código de acción y luego llamarlo en cualquier momento usando esta acción, haciéndolos esencialmente eventos personalizados object.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Event El número del evento de usuario a llamar (0 - 11).

Ejemplo:

El código del bloque de acción anterior comprueba una variable y luego llama al Evento de Usuario [0] o al Evento de Usuario [1] dependiendo de si la comprobación es true o false.
