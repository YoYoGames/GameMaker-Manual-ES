Crear una instancia de Object

Con esta acción puede crear una instancia de cualquier object desde el Navegador Asset. Se da el nombre del object y una posición x/y dentro del room donde se va a crear, así como el nombre de la capa en la que se va a crear la instancia (debe ser una Capa de Instancia). Puedes marcar la opción "relativa" flags para posicionar la instancia en relación con la instancia que ejecuta los bloques de código de acción. Cada vez que se crea una instancia en el juego, se le da un valor de identificación único con el que se puede identificar la instancia en acciones posteriores, por lo que esta acción le permite definir una variable "objetivo" para mantener este valor de ID (como se muestra en el ejemplo siguiente). Tenga en cuenta que el objetivo puede ser marcado como una variable temporal (local), que será que se creará para la acción y se podrá utilizar en todas las acciones posteriores.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Object El nombre del object para crear una instancia.
x La posición x en el room para crear la instancia (puede ser relativa a la instancia que llama).
y La posición y en el room para crear la instancia (puede ser relativa a la instancia que llama).
Layer El nombre (un string) de la capa de instancia para crear la instancia.
Target La variable a la que se dirige el valor de retorno.
