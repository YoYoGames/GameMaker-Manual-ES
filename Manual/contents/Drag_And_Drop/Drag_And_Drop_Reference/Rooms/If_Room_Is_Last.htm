Si Room es el último

Esta acción comprobará si el room actual es el último room que aparece en el Gestor de Salas. La acción devolverá true si el room actual es el último en el Gestor Room y false si no lo es. Puede flag la casilla de verificación "no" para convertir esto en "Si Room NO es el último".

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el "if" se evalúa como true, mientras que cualquier acción lanzadas en otro lugar se ejecutarán después del bloque "if".

El código del bloque de acción anterior comprueba si el room actual no es el último que aparece en el listado en el Gestor de Room. Si no lo es, el juego irá al siguiente room en el Navegador Asset, de lo contrario irá a un room específico.
