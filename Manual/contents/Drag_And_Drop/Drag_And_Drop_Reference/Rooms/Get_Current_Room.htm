Obtenga la sala actual

Esta acción devolverá el valor del índice único de room para el actual room. Es necesario proporcionar una variable de destino para mantener el valor devuelto, y esta variable puede ser marcada como una variable local temporal.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Target La variable de destino para almacenar el índice devuelto room

Ejemplo:

El código del bloque de acción anterior obtiene el valor del índice actual de room y luego utiliza una acción Switch para comprobar el room y establecer una variable global en función del resultado.
