Ir a la habitación

Esta acción terminará el room actual y pasará al room especificado por el índice de entrada. El room debe existir o de lo contrario se dará un error, y si se usa el mismo room que el actual room tendrá el mismo efecto que usar Reiniciar Sala. Tenga en cuenta que el room no cambiará hasta el final del evento o script donde se llamó a la acción, por lo que cualquier acción posterior a ésta seguirá ejecutándose.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Room El room para ir a

Ejemplo:

El código del bloque de acción anterior recupera una variable global y luego comprueba si el valor que contiene es menor o igual a 0. Si lo es, el juego pasará a otro room.
