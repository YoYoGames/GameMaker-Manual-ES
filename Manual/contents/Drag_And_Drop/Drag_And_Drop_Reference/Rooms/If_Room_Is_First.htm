Si Room es lo primero

Esta acción comprobará si el room actual es el primer room que aparece en el Gestor de Salas. La acción devolverá true si el actual room es el primero en el Gestor Room y false si no lo es. Puede flag la casilla de verificación "no" para convertir esto en "Si Room NO es el primero".

Tenga en cuenta que para añadir acciones en el bloque "if", éstas deben colocarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones se ejecutarán ahora si el "if" se evalúa como true, mientras que cualquier acción lanzadas en otro lugar se ejecutarán después del bloque "if".
