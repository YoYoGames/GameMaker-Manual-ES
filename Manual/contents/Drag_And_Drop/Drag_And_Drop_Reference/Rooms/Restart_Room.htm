Sala de reinicio

Esta acción reiniciará el room actual. Tenga en cuenta que el room no se reiniciará hasta el final del evento o script donde la acción fue llamada, por lo que cualquier código después de esta acción todavía se ejecutará.

Ejemplo:

El código del bloque de acción anterior comprueba si se ha pulsado una tecla y, si se detecta una, se reinicia room.
