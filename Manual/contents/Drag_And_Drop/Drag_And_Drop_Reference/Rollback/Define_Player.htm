Definir jugador (Rollback)

Esta acción indica al sistema de Rollback el object que debe ser utilizado para los jugadores. Esto es opcional, y cuando se utiliza, el sistema creará automáticamente instancias para los jugadores que se conectan, y destruirá instancias para los jugadores que se desconectan.

Se puede especificar la capa en la que se crearán las instancias de los jugadores. Si no se especifica una capa, los jugadores se crearán en la capa "Instancias ", que debe existir en la página room.

Para obtener información sobre la creación de un juego con o sin esta función, consulte Definir un objeto jugador.

Si se utiliza, esta función debe ejecutarse antes de la acción de unión/inicio.

Cada instancia creada con esta función obtiene las variables de instancia listadas en esta página.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Object La dirección object que se utilizará para las instancias de los jugadores
Layer OPCIONAL El nombre de la capa donde se crearán las instancias de los jugadores. "Instancias" por defecto (cuando no se especifica ningún valor).
