Únase al juego de retroceso

Esta acción intenta unirse a un juego. Debes estar en una URL que contenga el ID de la room virtual para unirte.

Si se unió un room, se ejecutarán todas las acciones adjuntas a esta acción.

Cuando un nuevo jugador se une a una partida en la que ya estás presente, se activa el evento Rollback. Consulta los eventos de retroceso para obtener más información.
