Si se pulsa la tecla (Rollback)

Esta acción comprueba si la entrada dada acaba de ser pulsada por el jugador actual.

Permite adjuntar acciones a la derecha, que se ejecutan sólo cuando se encuentra la entrada dada.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Input El nombre de la entrada a comprobar
Not Habilite esta opción para comprobar si la entrada no se ha pulsado
