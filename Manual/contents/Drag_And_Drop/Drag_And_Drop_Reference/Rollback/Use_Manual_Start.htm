Utilizar el arranque manual (Rollback)

Por defecto, una partida multijugador comienza automáticamente cuando todos los jugadores se han unido.

Al llamar a esta acción se desactiva ese comportamiento de inicio automático, por lo que el juego espera a que lo inicies manualmente.

Si se utiliza, esta acción debe ejecutarse antes de la acción de unión/creación.

Sintaxis de la acción:

Ejemplo:

El bloque anterior permite el inicio manual, y luego se une o crea una nueva partida multijugador.
