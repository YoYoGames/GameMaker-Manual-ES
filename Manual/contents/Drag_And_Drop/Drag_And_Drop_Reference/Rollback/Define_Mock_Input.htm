Definir la entrada simulada (Rollback)

Esta acción puede utilizarse durante la prueba de sincronización para asignar una entrada temporal "simulada" al reproductor especificado. Esto anula los controles definidos para el reproductor y cualquier entrada aleatoria de la prueba de sincronización.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Player ID El número de jugador que recibe este conjunto de entradas falsas
Input El nombre de la entrada a definir
Value La constante de entrada a asignar (ver Controles personalizados)

Ejemplo:

La acción anterior define las entradas simuladas fuego, izquierda, derecha, arriba y abajo para el jugador 1.
