Obtener información sobre el retroceso

Esta función devuelve un struct que contiene información sobre un jugador. Sólo debe utilizarse cuando se gestionen manualmente las instancias de los jugadores. Si estás usando Define Player, entonces cada instancia de jugador recibirá automáticamente esta información como variables.

Para obtener información sobre el contenido del struct devuelto, véase: rollback_get_info

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Player ID El ID del jugador del que se quiere obtener información
Target La variable donde se debe almacenar el struct devuelto
