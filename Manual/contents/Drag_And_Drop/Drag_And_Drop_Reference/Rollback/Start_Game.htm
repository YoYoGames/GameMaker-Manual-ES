Iniciar el juego de retroceso

Esta acción fuerza el inicio de la partida multijugador actual, antes de que todos los jugadores se hayan unido. Primero debes crear una partida multijugador usando Crear partida de retroceso.

El sistema normalmente espera a que todos los jugadores se unan antes de iniciar automáticamente la partida. Esta acción puede utilizarse para iniciar la partida antes de que todos los jugadores se hayan unido.

Puedes llamar a Usar Inicio Manual para hacer que el Rollback espere esta llamada incluso después de que todos los jugadores se hayan unido.

La ejecución de esta acción activará el evento Rollback Start.

Sintaxis de la acción:

Atrás: Retroceso de las acciones multijugador
