Definir la entrada (Rollback)

Esta función se utiliza para definir entradas personalizadas para todos los jugadores. Toma un par Entrada/Valor, donde la Entrada es el nombre de la entrada, y el Valor es la constante de entrada asignada a ella.

Para saber cómo asignar entradas personalizadas, consulte Controles personalizados.

Si se utiliza, esta función debe ejecutarse antes de la acción de unión/creación.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Input El nombre de la entrada a definir
Value La constante de entrada a asignar (ver Controles personalizados)

Ejemplo:

La acción anterior define las entradas fuego, izquierda, derecha, arriba y abajo para todos los jugadores.
