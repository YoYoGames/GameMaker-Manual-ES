Si se produce el evento de retroceso

Esta acción se utiliza en el evento Rollback Event, para comprobar qué tipo de evento se disparó.

Permite adjuntar acciones a la derecha, que se ejecutan sólo cuando se dispara el evento dado.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Event El evento Rollback a comprobar
Not Habilita esta opción para comprobar si el evento dado no se ha disparado (lo que significa que la condición será verdadera para todos los demás eventos que se disparen)
