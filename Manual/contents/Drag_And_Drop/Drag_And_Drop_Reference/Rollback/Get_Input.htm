Obtener entrada (Rollback)

Esta acción devuelve un struct que contiene los valores de entrada de un jugador. Toma un argumento opcional que especifica el ID del jugador para el que se devolverán las entradas.

Se recomienda utilizar las opciones Si tecla abajo (retroceso), Si tecla pulsada (retroceso) y Si tecla arriba (retroceso) para comprobar las entradas en el modo multijugador.

Cuando se utiliza en una instancia de jugador que fue creada por Define Player (Rollback), no es necesario especificar un ID de jugador, por lo que se puede especificar simplemente player_id.

Cuando se utiliza en otras instancias, o en instancias de jugadores creadas manualmente, se debe especificar un ID de jugador.

Para obtener información sobre las entradas que se devuelven y cómo se pueden modificar, consulte Definición de entradas.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Player ID El ID del jugador para el que hay que obtener información
Target La variable donde se debe almacenar el struct devuelto
