Establecer las preferencias del jugador (Rollback)

Cuando juegas una partida utilizando la Prueba de Sincronización, todos los jugadores "remotos" reciben valores aleatorios para sus entradas definidas, como una forma básica de prueba. Puedes desactivar o volver a activar este comportamiento mediante esta acción.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Enabled Activar o desactivar la entrada aleatoria para los jugadores remotos durante las pruebas
