Mostrar mensaje de depuración

Con esta acción puede enviar un mensaje de depuración a la salida del compilador. Este mensaje puede ser un string o un número real, pero si desea utilizar ambos, el número debe ser convertido a una cadena (ver la acción Número a Cadena). Tenga en cuenta que puede enviar múltiples mensajes utilizando esta acción haciendo clic en el icono para expandir la acción con más campos de salida.

Sintaxis de la acción:

Argumentos:

Argumento Descripción
Message El mensaje a mostrar en la ventana de salida

Ejemplo:

El código del bloque de acción anterior comprobará una variable y, si devuelve true, se mostrarán un par de mensajes de depuración en la ventana de salida.
