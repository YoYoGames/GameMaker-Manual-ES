{GML_Visual}

GML Visual es el método de scripting visual para programar con el Lenguaje GameMaker (GML). Consiste en el uso de bloques de acción para construir la lógica del juego de una manera visual e intuitiva, y es una herramienta ideal para las personas que están aprendiendo a programar, o para las personas que tienen una inclinación más visual.

  • Resumen de GML Visual
