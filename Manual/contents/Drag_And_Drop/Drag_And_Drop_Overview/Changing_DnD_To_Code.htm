Cambiando GML Visual to Code

Otra característica de GML Visual es que puede convertirlo en GML Code, y de esta manera ver qué es exactamente lo que ocurre entre bastidores cuando se utilizan las acciones. Sin embargo, antes de convertir las acciones en código, primero puede obtener una vista previa utilizando el botón derecho del ratón opción de menú Vista previa en vivo:

Lo que abrirá una nueva ventana:

A medida que añada, cambie o elimine acciones de GML Visual, la vista previa en vivo de GML se actualizará para mostrarle el código real que se está creando "entre bastidores". El código en la vista previa en vivo no puede editarse directamente, pero puede seleccionar secciones y copiarlas para pegarlas en GML scripts o en Acciones de Código (por ejemplo).

Si decide que quiere convertir el GML Visual en código después de ver la vista previa, puede hacerlo de nuevo pulsando el botón derecho del ratón en cualquier evento workspace con acciones y seleccionando Convertir a código GML .

La primera vez que lo haga recibirá un mensaje de advertencia diciendo que se trata de una conversión de un solo sentido, ya que puede convertir las acciones en código, pero no puede volver a convertirlas en acciones individuales más adelante. Al hacer clic en "Aceptar" se realizará la conversión por usted.

El código resultante utilizará {} para delimitar las acciones individuales, y podrá ver claramente qué acciones se relacionan con qué funciones o declaraciones de variables dentro del código. Si el GML Visual es más complejo, el código también lo será, pero se aplicarán las mismas reglas generales y el código se estructurará secuencialmente de la misma manera que se ha escrito el GML Visual. ¡Cuando se aprende a programar usando GML Visual, esto puede ser una herramienta importante para pasar a usar GML en una etapa posterior, pero no es de ninguna manera obligatorio y todavía puedes hacer grandes juegos usando GML Visual! También vale la pena señalar que, aunque el proceso de conversión es unidireccional, después de convertir las acciones en código, puedes volver a utilizar GML Visual utilizando el menú del botón derecho del editor de código y seleccionando "Convertir a GML Visual".
