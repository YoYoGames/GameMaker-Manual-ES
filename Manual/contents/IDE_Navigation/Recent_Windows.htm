Ventanas recientes

La lista de Ventanas Recientes se puede abrir desde el menú de Windows, y mostrará una breve lista (por defecto tiene un máximo de 10 elementos) de las ventanas que has abierto anteriormente. Al principio esta lista estará vacía, pero a medida que abra diferentes ventanas en IDE (como una sprite o una script o una object, etc...) la lista se irá poblando. Si necesita volver a cualquiera de estas ventanas abiertas previamente, simplemente haga clic con el botón izquierdo del ratón en el elemento de la lista y el workspace se desplazará a la ventana. Si hace clic con el botón derecho del ratón en un elemento, se abrirá el siguiente menú: Aquí puede elegir eliminar un elemento (o elementos, ya que puede utilizar + o / + para seleccionar varios) de la lista o puede hacer que se muevan a la parte superior de la lista. El menú RMB también tiene una opción adicional que permite borrar la lista por completo. Vale la pena señalar que la lista de Windows recientes puede acoplarse a los lados de IDE y también puedes ajustar ciertas preferencias para cambiar su comportamiento.
