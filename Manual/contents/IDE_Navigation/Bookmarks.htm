Marcadores

Otro método para navegar por workspace es utilizar los Marcadores. Puede añadir hasta 9 "marcadores" a las ventanas de IDE o a las secciones de código/GML Visual en los eventos de scripts o object, y luego puede saltar rápidamente entre ellos utilizando la combinación de teclas correcta.

Para añadir un marcador, simplemente seleccione la ventana o línea de código que desea etiquetar y luego + + NumberKey[1 - 9] en WIndows o + + NumberKey[1 - 9] en Mac y esto establecerá el marcador, etiquetando el elemento en el IDE como sigue:

En la imagen anterior hemos asignado los marcadores numerados 1, 2 y 4 a diferentes partes de IDE, y ahora podemos navegar fuera de estas ventanas o incluso cerrarlas del todo, ya que el marcador nos permitirá volver directamente a él en cualquier momento posterior. Esto se hace pulsando / + NumberKey[1 - 9], y si el elemento marcado se ha cerrado se abrirá, de lo contrario el workspace se desplazará al marcador seleccionado.

IMPORTANTE Los marcadores no se guardan con el proyecto ni cuando se cierra IDE. Sólo son válidos para la sesión en la que fueron creados.

Si desea eliminar un marcador, sólo tiene que seleccionar la ventana o la línea de código marcada y, a continuación, repetir los comandos + / + NumberKey[1 - 9] que utilizó para crearlo.
