El menú Marketplace

Además de crear tu propio assets y el código de tus proyectos, también tienes acceso a la página web de GameMaker Marketplace, donde puedes descargarte assets, tanto gratuitos como de pago, para añadirlos a tus proyectos. Puedes encontrar más información sobre Marketplace en la sección sobre el mercado de GameMaker, y las opciones de este menú se indican a continuación:

  • Vaya al Mercado - Abra la ventana Marketplace donde puede buscar assets.
  • Mi biblioteca - Vea su biblioteca de contenidos actual en Marketplace.
  • Mi historial de compras - Esta opción abrirá un navegador y le llevará a la página de la cuenta del usuario conectado para que pueda ver el historial de compras de esa cuenta.
  • Actualizar paquete existente: esta opción abrirá el asistente de actualización de paquetes para que pueda actualizar un paquete existente de Marketplace que haya creado previamente.
  • Crear Paquete - Con esta opción puede abrir el Asistente para Crear Paquete para crear un nuevo paquete o tutorial de asset para el Marketplace.
