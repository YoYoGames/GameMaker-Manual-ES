El menú de diseños

Este menú está relacionado con la posibilidad de guardar y cargar diferentes diseños de IDE. Puede configurar el IDE como desee, con ciertos elementos (como el Navegador Asset o la salida Windows) dispuestos de forma diferente a la predeterminada. Esto le permite configurar GameMaker para que se adapte a su flujo de trabajo y con las opciones de este menú puede guardar los diseños del proyecto actual como un archivo *.xml que luego puede ser cargado en cualquier proyecto para establecer los mismos diseños. Las opciones de este menú son las siguientes:

  • Cargar diseño - Esto abrirá un explorador de archivos donde puede seleccionar un archivo *.xml de diseño previamente guardado para cargarlo.
  • Guardar diseño - Esto abrirá un explorador de archivos donde puede nombrar y guardar el diseño actual de IDE como un archivo *.xml.
  • Restablecer diseño - Esto restablecerá el diseño actual de IDE a la configuración predeterminada.
