El menú Archivo

En file menu se puede elegir cambiar, guardar o importar proyectos, así como establecer las preferencias de GameMaker IDE. Las opciones disponibles son:

  • Nuevo Proyecto - Esto es para crear un nuevo proyecto desde cero en GameMaker. Se abrirá el diálogo "Guardar" y le pedirá que elija una ubicación para guardar el nuevo proyecto. Una vez que haya dado un nombre al proyecto y una ubicación para guardarlo, será llevado a un nuevo espacio de trabajo en blanco para empezar a trabajar en él. Los archivos de proyecto se guardan con el archivo *.yyp extension .
  • Abrir Proyecto - Esto abrirá un explorador de archivos para que usted seleccione un archivo de proyecto creado previamente en *.yyz para cargarlo. Por favor, tenga en cuenta que al importar, si el proyecto fue creado con una versión inferior a la 2.3.0, recibirá el siguiente mensaje de advertencia: Si elige continuar, se abrirá un explorador de archivos y podrá dar al proyecto un nombre actualizado y una ubicación para guardarlo, de modo que el proyecto original no se sobreescriba. Una vez que el proyecto haya sido convertido al nuevo formato 2.3.0+, se abrirá en el IDE y obtendrá un mensaje diciendo que la conversión ha sido exitosa. Deberías proceder a comprobar el informe de compatibilidad que ha sido generado por el proceso de conversión y ver si hay algo que tengas que arreglar antes de intentar ejecutar o compilar el proyecto. Tenga en cuenta que si se repiten los errores al intentar abrir un proyecto realizado con una versión anterior a la 2.3.0, deberá ponerse en contacto con YoYo Games (utilizando la opción "Informar de un error" del menú de ayuda), facilitando el archivo original *.yyp que está intentando convertir.
  • Importar Proyecto - Aquí puede importar proyectos que hayan sido previamente exportados como un archivo *.yyz (ver "Exportar Proyecto", más abajo), o - si tiene un proyecto hecho con el legado GameMaker: Studio 1.4 - puede seleccionar esta opción para importar el archivo *.gmx o el *.gmz de esa línea de productos. Al igual que al abrir un proyecto, si el *.yyz fue creado con una versión de GameMaker anterior a la actualización 2.3.0, será necesario convertirlo, ya que el formato de los archivos de proyecto cambió con esa actualización.

NOTA: Para importar un proyecto anterior de GameMaker a GameMaker debe haber sido creado con la versión 1.4, e incluso entonces puede que no funcione directamente al importarlo. Muchas cosas son diferentes entre GameMaker y GameMaker: Studio 1.4, por lo que tus proyectos más antiguos pueden necesitar algunos ajustes para que funcionen. Por favor, consulte el artículo Porting A GM:S 1.4 Game To GameMaker en la Base de Conocimiento de GameMaker y la sección del manual que cubre Funciones Obsoletas y Scripts de Compatibilidad.

¡IMPORTANTE! La importación de un proyecto *.yyz que contenga recursos font puede requerir que se "regenere" el fonts eliminándolos/añadiéndolos, o cambiando el font en el editor font y volviéndolo a establecer. Esto se debe a que la información existente de font es retenida por el *.yyz cuando se exporta, y, al importarlo posteriormente o en otra máquina, el proyecto puede no tener ya instalado el fonts requerido (o puede tener un font diferente con el mismo nombre). Esto sólo debe hacerse si se observan problemas con la representación de font después de la importación.

  • Proyectos recientes - Puede utilizar esta opción para ver los proyectos utilizados recientemente. Al seleccionarla, se reabrirá la página de inicio (en caso de que la haya cerrado) y podrá seleccionar un proyecto utilizado recientemente de la lista que se encuentra allí.
  • Guardar proyecto - Esto guardará su juego en la misma ubicación que definió cuando creó el proyecto (o desde donde lo abrió).
  • Guardar proyecto como - Igual que el anterior, sólo que se especifica la ubicación y el nuevo nombre del proyecto.
  • Exportar Proyecto - Esto le permite exportar su proyecto ya sea como un archivo o como una plantilla:
    • YYZ - Esto exportará el proyecto como un único archivo *.yyz y es muy útil para enviar copias a la gente o para guardar copias de seguridad.
    • Como Plantilla - Esto le permitirá exportar su proyecto como una plantilla, que podrá utilizar posteriormente como base para otros proyectos en La Página de Inicio. Al seleccionar esta opción se le mostrará el siguiente mensaje:
      Primero debe hacer clic en "Editar información de la plantilla" para asegurarse de que la información de su plantilla es correcta, y luego seleccionar "Exportar" cuando abra este aviso por segunda vez para guardarlo como plantilla. A continuación, la verá en el menú Plantilla de la pantalla de inicio.
  • Preferencias - Seleccionando esto te llevará a la Ventana de Preferencias donde puedes configurar el aspecto y la sensación de GameMaker.
  • Nuevo IDE - Al seleccionar esta opción, se abrirá una nueva instancia de IDE para su uso (se abrirá en la página de inicio).
  • Mi cuenta - Esta opción abrirá su navegador web por defecto y le llevará a la página de su cuenta YoYo.
  • Cerrar la sesión - Al seleccionar esta opción, se cerrará la sesión de GameMaker. Tendrá que volver a iniciar la sesión con la misma licencia (o cualquier otra) para poder utilizar el producto más adelante.

¡IMPORTANTE! Si has estado utilizando una licencia gratuita para GameMaker y luego compras una de las otras licencias, es posible que tengas que cerrar la sesión en IDE y luego reiniciar GameMaker y volver a iniciar la sesión para asegurarte de que la nueva licencia se aplica correctamente (aunque esto debería ocurrir automáticamente).

  • Salir - Salir de GameMaker al escritorio.
