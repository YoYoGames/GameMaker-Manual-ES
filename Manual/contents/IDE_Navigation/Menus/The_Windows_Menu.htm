El menú Windows

Este menú sirve para abrir las ventanas generales acopladas de IDE. Dado que las ventanas acopladas pueden reorganizarse y su contenido puede eliminarse como ventanas independientes o incluso cerrarse por completo, esta ventana le permite reabrirlas si las necesita para algo. Las ventanas que se pueden abrir desde aquí son la ventana del Navegador de Activos y todas las diferentes ventanas de Salida (Salida, Búsqueda, Source Control, Breakpoints y Errores de compilación/sintaxis).

A continuación puede abrir la ventana del Inspector. Esta ventana se utiliza para inspeccionar las propiedades del elemento IDE actualmente seleccionado, y estas propiedades pueden ser editadas según sea necesario.

Después de la opción Inspector tiene la lista de Ventanas Recientes, y también está la opción de abrir la ventana de Variables de Entorno donde puede ver los diferentes paths a los que apuntan las distintas variables de entorno utilizadas en las Preferencias. de entorno utilizadas en las Preferencias. Estas variables no pueden ser editadas (ni su nombre ni su ruta/configuración) pero puede seleccionar y copiar/pegar cualquiera de ellas para usarlas en otro lugar.

Por último, tiene la opción de abrir las ventanas del tutorial. Esto reabrirá cualquier ventana que esté relacionada con un proyecto tutorial que haya abierto, pero sólo si el proyecto es uno que fue creado como parte de un tutorial (si no, esta opción no estará disponible). Para obtener información sobre la creación de sus propios proyectos de tutoriales, consulte la sección sobre el Mercado (ya que todos los tutoriales se crean, alojan y descargan desde allí).
